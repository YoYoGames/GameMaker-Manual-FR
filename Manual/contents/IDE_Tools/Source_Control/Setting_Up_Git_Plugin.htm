Configuration du plugin Git

Avant de pouvoir utiliser le plugin Git, nous devons configurer l'exécutable Git à utiliser en allant dans la section Plugins - Source Control (Git) des Préférences:

Le chemin de l'exécutable Git devrait être rempli automatiquement s'il est trouvé par GameMaker, mais si le champ est vide, vous pouvez cliquer sur le bouton et sélectionner manuellement le fichier git.exe inclus dans le dossier cmd de votre installation Git, qui se trouve généralement dans C:\Program Files\Git\.

Pour le reste des options de Git, consultez la page des préférences du plugin.
