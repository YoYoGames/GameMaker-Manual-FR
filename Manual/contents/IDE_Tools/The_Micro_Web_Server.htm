Le serveur Micro Web

La fonction principale de ce petit outil est de se connecter et de servir des fichiers à certaines des plateformes cibles disponibles pour GameMaker (comme la cible HTML5). Il sert également des fichiers pour le manuel GameMaker lorsqu'il est utilisé hors ligne.

Lorsque vous exécutez un jeu sur une cible qui nécessite le Micro Web Server, vous verrez une nouvelle fenêtre s'ouvrir pendant que le jeu est servi à l'appareil choisi :

REMARQUE La première fois que vous exécutez un jeu sur une cible qui nécessite le Micro Web Server, vous pouvez recevoir une alerte de pare-feu. Si cela se produit, vous devez autoriser le serveur Web à communiquer à travers le pare-feu.

Lorsque le Micro Web Server fonctionne, vous pouvez access votre jeu à partir de n'importe quel appareil local (et même à distance, si la redirection de port est correctement configurée). Il suffit d'envoyer l'adresse IP locale de votre ordinateur à access via le port indiqué dans cette fenêtre, et vous pourrez jouer à votre jeu à partir d'un autre appareil, comme un téléphone portable ou un autre ordinateur.

Par exemple, disons que l'adresse IP locale de votre ordinateur est 192.168.0.110(lisez ceci pour savoir comment la trouver), et que le port affiché dans la fenêtre du Micro Web Server est 51264. Après avoir lancé le jeu sur votre ordinateur, vous vous connecteriez à 192.168.0.110:51264 à partir d'un navigateur sur votre mobile, et vous joueriez votre jeu.

Le serveur Web affiche un certain nombre d'informations de débogage relatives aux fichiers qui ont été servis, et dispose d'un bouton pour les effacer en bas. À côté du bouton "Clear" se trouve le chemin d'accès au fichier JS racine du projet, tandis qu'en dessous se trouve l'option Network Speed. Cette option est réglée sur "No Limit" par défaut, mais si vous cliquez dessus, une liste de différentes options de réseau s'affiche :

En modifiant ce paramètre, vous modifiez également l'option Taille du paquet vers la droite. Vous pouvez également modifier ce paramètre manuellement, auquel cas la vitesse du réseau sera réglée sur "Personnalisé". Si vous choisissez une valeur autre que la valeur maximale, la vitesse de transfert du réseau sera réduite et vous pourrez tester des jeux dans un environnement à faible bande passante.

Vous pouvez également voir la liste des IP autorisées ainsi que le numéro de port utilisé, mais ceux-ci ne peuvent pas être modifiés dans le Micro Web Server. Ils doivent être définis dans les paramètres du gestionnaire de périphériques pour le périphérique cible donné, ou dans les préférences HTML5 si vous utilisez le périphérique "par défaut".
