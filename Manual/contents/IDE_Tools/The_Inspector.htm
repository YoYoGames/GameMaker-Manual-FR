L'inspecteur

La fenêtre de l'inspecteur montre toutes les propriétés associées aux éléments sélectionnés dans le site IDE.

Vous pouvez sélectionner un ou plusieurs éléments dans n'importe quel éditeur, et afficher ou modifier ses différentes propriétés :

L'inspecteur peut être ouvert à partir du menu Windows en haut de la fenêtre GameMaker:

Sélectionnez cette option pour ouvrir une nouvelle fenêtre d'inspecteur. Vous pouvez en avoir plusieurs ouvertes en même temps.

Après avoir ouvert une fenêtre de l'inspecteur, vous pouvez l'ancrer n'importe où sur le site IDE ou l'utiliser comme une fenêtre flottante :

Utilisation de l'inspecteur

L'inspecteur vous permet de visualiser et de modifier les actifs sans avoir à vous fier uniquement aux espaces de travail.

Par exemple, vous avez l'habitude d'éditer votre code d'événement en plein écran, et vous retournez fréquemment dans un espace de travail pour éditer les propriétés de cet objet.

Avec l'inspecteur, vous pouvez visualiser les détails d'un objet tout en restant en plein écran :

Il est ainsi possible de modifier les propriétés d'un objet et de supprimer/changer les événements tout en éditant son code en plein écran, mais aussi de mettre en place des Parenting, des définitions de variables, etc.

Bien que cet exemple particulier ne concerne que les objets, l'inspecteur peut faire beaucoup plus et vous permettre de modifier de nombreux types d'actifs différents (par exemple Sprites, les polices, les courbes d'animation, etc.)

Éléments de l'éditeur

Outre les actifs, vous pouvez également inspecter des éléments dans un éditeur, tels que les couches d'une pièce, les pistes d'une séquence, etc.

L'inspecteur vous permet d'appliquer un filtre ou un effet à un calque de pièce, ce que vous ne pouvez pas faire via l'éditeur de pièce lui-même :

REMARQUE Lorsque vous appliquez un filtre/effet à un calque non-FX à l'aide de l'inspecteur, il sera toujours appliqué comme un effet "monocouche", c'est-à-dire que son effet ne sera appliqué que sur ce calque particulier et non sur les autres calques inférieurs.

Le bouton "Ouvrir l'éditeur

Si vous avez sélectionné un actif, vous pouvez cliquer sur "Ouvrir l'éditeur" dans l'inspecteur pour ouvrir son éditeur d'actifs.

Par exemple, si vous faites cela avec un objet, vous ouvrirez l'éditeur d'objets, où vous pourrez modifier ses propriétés, ajouter des événements, etc.

Cependant, lorsque vous modifiez quelque chose à l'intérieur d'une pièce, comme une instance ou un calque, l'inspecteur ne se concentre que sur les éditeurs de l'éditeur de pièce.

Comme indiqué ci-dessus, sélectionner une instance dans une pièce et cliquer sur "Ouvrir l'éditeur" dans l'inspecteur ouvrira son éditeur d'instance.

De même, en sélectionnant un calque de chemin et en cliquant sur "Ouvrir l'éditeur", le focus sera déplacé vers le panneau des calques où vous pourrez modifier les propriétés du calque, au lieu d'ouvrir l'éditeur d'actifs de chemin.

Multi-Select

L'inspecteur vous permet d'afficher et de modifier les propriétés de plusieurs actifs et éléments (tels que les couches de salle et les pistes de séquence) en même temps.

Sélectionnez les actifs ou les éléments souhaités en maintenant la touche CTRL / CMD enfoncée et en cliquant avec le bouton gauche de la souris. Les éléments sélectionnés s'afficheront dans une liste "multi-sélection" dans l'inspecteur :

Appuyez sur le X à côté d'un élément pour le supprimer de votre sélection.

Modification des valeurs

Si les éléments que vous avez sélectionnés partagent la même valeur pour une propriété, cette valeur partagée apparaîtra dans ce champ.

Cependant, si les éléments ont des valeurs différentes pour une propriété, l'éditeur affichera simplement - à la place de cette valeur.

Vous pouvez modifier n'importe lequel de ces champs. La modification d'une valeur s'appliquera à tous les éléments sélectionnés.

Sélection dans la liste

Vous pouvez sélectionner n'importe quel élément de votre liste multi-sélection pour afficher et modifier ses propriétés, sans avoir à réinitialiser votre sélection.

Cliquez sur la zone vide de la liste pour revenir à la vue multi-sélection.

Notez que vous ne pouvez pas sélectionner plusieurs éléments dans cette liste.

Mise en évidence de la sélection

Lorsqu'un élément est sélectionné dans la liste multi-sélection, il apparaît dans l'éditeur avec un contour rouge :

Vous pouvez modifier la couleur de ce contour dans les préférences de la pièce et les préférences de la séquence.

Inspecteurs multiples

Vous pouvez ouvrir plusieurs fenêtres de l'inspecteur à la fois, et chacune d'elles peut être liée à un élément spécifique dans le site IDE.

Pour vous assurer que chaque fois que vous cliquez sur un élément, tous les inspecteurs ouverts ne changent pas pour afficher la nouvelle sélection, vous devez d'abord verrouiller chaque fenêtre d'inspecteur une par une.

Pour ce faire, vous devez ouvrir un inspecteur, sélectionner un élément IDE à inspecter, puis cliquer sur le bouton de verrouillage de l'inspecteur pour le verrouiller sur l'élément (voir l'image ci-dessous). Une fois que vous avez verrouillé un inspecteur, vous pouvez en ouvrir un autre et répéter le processus.

Notez qu'une fenêtre d'inspecteur verrouillée ne bloque pas l'édition des valeurs de l'élément inspecté. Elle verrouille simplement l'inspecteur sur l'élément sélectionné et vous pouvez continuer à le modifier à tout moment, même si l'élément d'origine n'est pas l'objet principal du site IDE.
