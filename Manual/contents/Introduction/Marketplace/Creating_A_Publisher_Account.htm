Création d'un compte éditeur

Avant de pouvoir utiliser le marché GameMaker pour acheter ou vendre des ressources, vous devez avoir créé un compte YoYo. Si vous souhaitez vendre des ensembles de ressources, vous devez également avoir créé un compte d'éditeur Marketplace.

Les étapes à suivre pour vous inscrire en tant qu'éditeur sont les suivantes :

  • Connectez-vous à votre compte YoYo.
  • Une fois que vous êtes dans le tableau de bord de votre compte, visitez la page d'enregistrement de l'éditeur. Celle-ci se trouve dans la section Éditeur du tableau de bord de votre compte YoYo, où vous devez cliquer sur la section Vendre vos actifs:
  • Sur la page suivante, cliquez sur le bouton " Register", puis saisissez quelques informations de base sur votre identité d'éditeur, comme votre nom d'éditeur, votre adresse électronique et les types de ressources que vous envisagez de télécharger.

Vous devez lire et accepter le contrat d'éditeur Marketplace avant de pouvoir finaliser l'enregistrement de votre compte d'éditeur. Lorsque vous l'aurez fait, votre demande sera complète, mais notez que toutes les inscriptions doivent être vérifiées par YoYo Games, de sorte que votre nouvel éditeur ne sera pas disponible immédiatement. Lorsque votre inscription sera vérifiée, vous en serez informé par l'intermédiaire de l'adresse électronique que vous avez fournie (veuillez noter que nous fonctionnons selon les horaires de bureau habituels du Royaume-Uni, de sorte que les demandes présentées le week-end, etc.)

Une fois que vous aurez soumis l'inscription, vous serez dirigé vers votre compte d'éditeur, où vous pourrez voir les détails déjà donnés et les modifier, ainsi que télécharger un avatar :

Vous devez maintenant vous rendre dans la section Détails du paiement et y remplir le formulaire :

Veuillez noter que :

  • Actuellement, le site Marketplace ne propose que le paiement par PayPal. Lors de la création d'un compte d'éditeur, il vous sera demandé de fournir une adresse électronique valide pour un compte PayPal, sinon vous ne pourrez pas être payé.
  • Pour les utilisateurs du Royaume-Uni/de l'UE, un numéro de TVA valide doit être défini si cela s'applique à votre organisation (pour les utilisateurs hors du Royaume-Uni, vous devez faire précéder votre numéro de TVA du code à deux chiffres de votre pays).

Si vous prévoyez de télécharger uniquement des ressources gratuites sur le site Marketplace, vous pouvez ignorer les champs PayPal et TVA. Vous pouvez ensuite cliquer sur le bouton Soumettre pour continuer.

Une fois que votre inscription a été vérifiée, vous pouvez utiliser le reste des options de la Console de l'éditeur. Il s'agit de la "maison" de toutes vos opérations de publication de ressources, et vous pouvez voir un menu d'éléments sur la gauche de la page. Chacune d'entre elles vous montrera différents détails sur vos activités de publication :

  • Actifs: Cette page vous donne un aperçu rapide de vos ressources (sprites, audio, projets, etc.), et vous permet d'afficher et de modifier leurs détails, d'examiner vos ventes, les statistiques de téléchargement, et aussi de publier une nouvelle version de votre ressource. Vous pouvez également ajouter de nouvelles ressources que vous avez créées pour les publier. Pour plus d'informations sur la publication des ressources, veuillez consulter la page : Référencer une ressource.
  • Services: Vous pouvez voir ici les services que vous offrez aux autres utilisateurs de Marketplace, ainsi que modifier leurs détails et mettre à jour ou ajouter de nouveaux services. Pour plus d'informations sur les services, consultez la page : Lister un service.
  • Détails du compte: Mettez à jour votre profil d'éditeur public sur la boutique et ajoutez les détails de paiement via les détails du compte (voir ci-dessus pour plus d'informations sur la création d'un compte de paiement).
  • Accès des utilisateurs: Si vous travaillez au sein d'une équipe, vous pouvez ajouter plusieurs utilisateurs à votre compte éditeur et définir les parties de la console éditeur auxquelles ils ont accès access. Le compte qui enregistre le compte éditeur est le propriétaire, avec un accès complet à toutes les parties de la console du développeur ( access ). Le propriétaire peut ensuite ajouter des comptes d'utilisateur et gérer les parties de la Console du développeur auxquelles ils ont accès ( access ). Il suffit d'indiquer l'adresse électronique de la personne que vous souhaitez ajouter, puis de cocher les parties de l'identité de l'éditeur auxquelles vous souhaitez qu'elle ait accès à access.
  • Analytique: La console du développeur vous donne accès à access aux ventes quotidiennes et aux rapports et graphiques de téléchargement vous permettant de suivre les performances de vos actifs. Vous pouvez filtrer vos rapports en fonction de plusieurs critères, notamment le caractère payant ou gratuit, la catégorie, le pays et la date.
  • Achats: Vous pouvez voir ici une liste de tous les achats qui ont été effectués sur vos biens (les achats gratuits seront affichés, mais avec un prix de 0,00 $).
  • Conditions de l'auteur: Voici les conditions générales actuelles de publication sur le site Marketplace. Vous devez les accepter avant de pouvoir publier quoi que ce soit, même des ressources gratuites.

Notez qu'une fois que vous avez configuré et vérifié un compte d'éditeur, vous pouvez retourner à votre tableau de bord de compte YoYo et créer d'autres comptes si nécessaire. Ainsi, vous pouvez créer deux comptes distincts, l'un vendant des ressources artistiques et l'autre des ressources de code (par exemple).
