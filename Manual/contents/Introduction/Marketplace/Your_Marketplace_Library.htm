Votre bibliothèque Marketplace

La fenêtre Bibliothèque vous montre les packages que vous avez achetés et vous donne la possibilité de les télécharger et/ou de les ajouter à votre projet.Sur la gauche se trouvent les différents filtres de catégories que vous pouvez utiliser pour voir ce que vous avez dans chacune d'elles, puis au milieu vous avez la liste des actifs de votre bibliothèque. Chacune d'entre elles peut être sélectionnée afin d'obtenir une vue d'ensemble de ce qu'elle fait (à droite) et toutes comportent des icônes permettant d'afficher, de télécharger ou d'installer le paquet de ressources dans un projet. Notez que vous disposez de deux filtres supplémentaires en haut de la fenêtre principale, qui vous permettent d'afficher uniquement les ressources créées spécifiquement pour GameMaker, ainsi qu'un filtre pour afficher uniquement les ressources que vous avez déjà téléchargées sur votre machine. Vous pouvez également rafraîchir la liste des ressources affichées à tout moment en cliquant sur le bouton Rafraîchir .

Pour ajouter un package de ressources à votre projet actuel, vous devez d'abord le télécharger en cliquant sur le bouton Télécharger . Une fois le téléchargement terminé, vous pouvez cliquer sur le bouton Importer pour l'ajouter à votre projet. L'ajout d'un package de cette manière créera une extension dans votre projet actuel, et vous demandera de sélectionner parmi les différentes ressources du package d'actifs celles que vous souhaitez ajouter (ou vous pouvez les importer toutes). Vous pouvez également sélectionner l'option Importer toutes les ressources dans un nouveau projet. Au lieu d'ajouter les ressources à votre projet actuel, GameMaker vous demandera de donner un nouveau nom de projet et d'enregistrer les ressources dans ce projet. Notez que si vous accédez directement à la bibliothèque depuis la page de démarrage, vous constaterez que la plupart des ressources ne peuvent pas être ajoutées individuellement et que vous serez obligé d'utiliser le bouton Ajouter tout. Cela est dû au fait que vous n'avez pas de projet dans lequel ajouter les ressources et que GameMaker doit donc créer un nouveau projet dans lequel vous pourrez ajouter toutes les ressources du package (lorsque vous cliquez sur le bouton Importer, vous serez invité à enregistrer le nouveau projet sous le nom de votre choix).Une fois les ressources ajoutées à votre navigateur de ressources, vous pouvez les utiliser dans votre jeu comme vous le feriez avec des ressources que vous avez créées vous-même. Veuillez noter que les ressources téléchargées à partir de Marketplace sont destinées à un usage individuel et ne peuvent pas être partagées avec des tiers - bien que vous puissiez les sauvegarder et les copier à d'autres endroits pour plus de sécurité - sauf si la ressource possède un CLUF personnalisé qui stipule le contraire.

REMARQUE: Les personnes qui créent des packs d'actifs pour le site Marketplace travaillent très dur pour créer une ressource de qualité que vous pouvez utiliser, alors pensez à évaluer les actifs que vous achetez lorsque vous les avez testés ou si vous les utilisez. Cela encourage le développeur et constitue également un guide utile pour les autres utilisateurs quant à la qualité des ressources disponibles. Vous pouvez facilement évaluer les ressources que vous avez téléchargées à partir de la fenêtre Bibliothèque.
