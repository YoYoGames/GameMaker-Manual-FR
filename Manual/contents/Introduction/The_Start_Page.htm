La page d'accueil

Lorsque vous ouvrez GameMaker pour la première fois, il vous est demandé de vous connecter à votre compte YoYo ou Opera (ou d'en créer un si ce n'est déjà fait - voir ici). Une fois enregistré et connecté, l'IDE principal de GameMaker s'ouvre sur la page de démarrage :

Lorsque vous démarrez GameMaker pour la première fois, vous pourrez voir une description de chaque élément de l'écran de démarrage en le survolant.

Ici, vous pouvez cliquer sur le bouton "SKIP to Setup Wizard" pour commencer à créer votre premier projet à l'aide de l'assistant de configuration, qui vous guidera à travers les étapes nécessaires à la création de votre premier jeu dans GameMaker.

Vous pouvez également cliquer sur "Fermer" pour fermer l'assistant et continuer à utiliser GameMaker manuellement, et éventuellement activer la case "Ne plus me montrer" si vous ne souhaitez pas voir l'assistant lorsque vous redémarrez GameMaker.

Éléments de la page d'accueil

La page d'accueil contient les éléments suivants :

  • Barre de menu de l'IDE: Elle vous montre les menus que vous pouvez utiliser dans l'IDE ; de plus amples informations sont données dans une section ci-dessous.
  • Options de projet: La section Projets est l'endroit où vous pouvez créer, ouvrir ou importer des projets. Ceci est expliqué plus en détail plus bas dans cette page.
  • Projets récents: Vous pouvez voir ici une liste de projets précédents que vous pouvez ouvrir. Vous pouvez également voir le nom complet du projet et son chemin d'accès, et cliquer sur le bouton gauche de la souris pour ouvrir le projet. Vous pouvez également basculer l'affichage de cette section entre le mode tuile et le mode liste en appuyant sur les boutons situés dans le coin supérieur droit.
  • Détails de la version: Cette section fournit des détails sur la version actuelle de IDE utilisée, ainsi que sur la version actuelle du Runtime. Vous serez également informé de tout changement disponible pour le IDE ou le Runtime dans cette section.
  • Détails de l'utilisateur: Cette rubrique vous indique le nom d'utilisateur de votre compte YoYo et la dernière fois que vous vous êtes connecté.
  • Ressources utiles: Cette page présente différentes tuiles sur lesquelles vous pouvez cliquer pour accéder à access aux ressources officielles des didacticiels de GameMaker.

Menus de l'IDE

En haut, vous trouverez les menus généraux de IDE qui sont expliqués dans les sections suivantes du manuel :

  • Le menu Fichier
  • Le menu d'édition
  • Le menu de construction
  • Le menu Windows
  • Le menu Outils
  • Le menu Marketplace
  • Le menu des mises en page
  • Le menu d'aide

Notez qu'il y aura également des options de menu spécifiques au contexte dans la barre de menu supérieure, en fonction de la fenêtre sur laquelle vous vous trouvez actuellement. Par exemple, si vous avez la fenêtre de l'éditeur d'images en point de mire, vous aurez des éléments de menu supplémentaires ici pour "Images", "Vue" et "Effets". Ces éléments de menu supplémentaires sont expliqués dans les sections du manuel correspondant à l'espace de travail ou à la fenêtre qui les a générés.

Démarrer un nouveau projet

Vous pouvez cliquer sur le bouton Nouveau pour créer un nouveau projet, sur le bouton Ouvrir pour ouvrir un projet existant ou sur le bouton Importer pour ouvrir un fichier de projet GameMaker compressé YYZ ou un fichier de projet GameMaker : Studio 1.4 GMX. Les boutons Ouvrir et Importer ouvrent l'explorateur de fichiers pour vous permettre de naviguer jusqu'au fichier de projet souhaité.

REMARQUE: GameMaker n'est pas complètement rétrocompatible avec GameMaker : Studio 1.4, mais les projets 1.4 importés devraient toujours fonctionner, car les fonctionnalités obsolètes ont été recréées automatiquement pour vous à l'aide de scripts de compatibilité. Pour plus de détails sur les problèmes possibles et les modifications apportées à GML, veuillez consulter l'article du Centre d'aide intitulé Portage d'un projet GMS 1.4 vers GameMaker, ainsi que la section du manuel consacrée aux fonctions obsolètes.

Pour créer un nouveau projet, il suffit de cliquer sur le bouton " Nouveau", ce qui ouvre le menu "Modèle" :

Sélectionnez un modèle (ou commencez par un projet vierge), choisissez un nom et un emplacement de sauvegarde pour votre projet, puis cliquez sur "C'est parti !".
