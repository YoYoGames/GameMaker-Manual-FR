Compilateur YoYo

Le compilateur YoYo (YYC ) est un compilateur spécial pour créer des paquets exécutables qui utilisent le code machine au lieu du code interprété et d'un runner (ce qu'utilise la compilation de GameMaker VM). Le code compilé est plus rapide à exécuter, mais la compilation de l'exécutable prend plus de temps.

Licence

Votre licence GameMaker peut inclure ou non le compilateur YoYo. Pour le savoir, consultez la page du Centre d'aide intitulée Quel abonnement me convient ?

Configuration du SDK

Pour que le compilateur YoYo puisse compiler pour la plate-forme cible, vous devez avoir installé les SDKcorrects.

Pour un aperçu général à ce sujet, consultez la page suivante du Centre d'aide : SDK requis par GameMaker.

Des informations sur les SDK requis par plateforme sont également disponibles dans le centre d'aide :

  • Opéra GX
  • Windows
  • macOS
  • Ubuntu
  • Android
  • iOS (y compris iPadOS)
  • tvOS

GameMaker doit connaître l'emplacement des fichiers nécessaires à la construction d'un exécutable sur la plate-forme sélectionnée. Si cela n'a pas été configuré correctement, une erreur de compilation est affichée.

Pour vérifier que les chemins ont été définis correctement pour la plate-forme sélectionnée, vous pouvez vérifier les préférences de la plate-forme.
