Options de jeu d'Opera GX

Les options de jeu d'Opera GX vous permettent de modifier les propriétés de votre jeu. Les sections suivantes sont proposées :

Général

Cette fenêtre montre l'état de votre connexion à Opera GX. Vous pouvez cliquer sur l'icône Refresh pour vous connecter en utilisant votre compte Opera.

Vous disposez alors des options suivantes :

  • Sélectionnez un jeu: Dans cette liste, vous pouvez soit sélectionner un jeu existant à mettre à jour, soit choisir de télécharger un nouveau jeu.
  • Nom du jeu: Il s'agit du nom (titre) de votre jeu sur GX.games.

    IMPORTANT Pour que le téléchargement vers GX.games réussisse, il est important que vous n'utilisiez que les caractères autorisés dans le nom du jeu (qui, par défaut, est défini comme le nom de votre projet GameMaker, veillez donc à choisir un nom de projet valide ou à modifier manuellement le nom du jeu par la suite).

    Si vous téléchargez un jeu dont le titre comporte des caractères interdits, vous obtiendrez cette erreur : Données incorrectes. Vérifiez la sortie

    Les caractères autorisés sont :
    • a-z, A-Z (avec un support des schémas locaux tels que å)
    • 0-9
    • espaces blancs
    • : - ! ? . , ' & ( )
  • Version: Il s'agit de la version du jeu qui a été téléchargée pour la dernière fois. Vous ne pouvez modifier ce champ qu'avant que votre jeu ait été téléchargé sur GX.games.
  • Version suivante : Il s'agit du numéro de version utilisé pour votre prochain chargement sur GX.games. Vous ne pouvez le modifier qu'après avoir téléchargé votre jeu sur GX.games. Vous n'avez pas besoin de le modifier manuellement pour chaque téléchargement, car GameMaker incrémentera automatiquement le dernier chiffre du numéro de version pour vous.
  • S'inscrire sur GX.games: Cliquez sur ce bouton pour enregistrer une nouvelle entrée sur GX.games. Ceci n'est valable que lorsque "Select game" est réglé sur "New game".
  • Sélectionnez le groupe: Avant de télécharger un nouveau jeu, vous pouvez sélectionner le studio auquel le jeu appartiendra.
  • Modifier le jeu sur Opera: Cela ouvre GX.games DevCloud et vous amène à la page " Game details " de votre jeu. Vous ne pouvez cliquer sur cette page que lorsque vous êtes connecté et que votre jeu est téléchargé.
  • URL de partage interne: Cette option ouvre la version private de votre jeu, uniquement si elle a été activée sur GX.games DevCloud.
  • URL de partage public: Cette option ouvre la version public de votre jeu, uniquement si elle a été activée sur GX.games DevCloud.

    Ces deux URL peuvent être partagées avec d'autres personnes pour leur permettre de jouer à votre jeu.

Graphiques

Ici, vous pouvez modifier les détails suivants relatifs à l'affichage de votre jeu :

  • Afficher le curseur: Contrôle si le curseur de la souris est visible au démarrage de votre jeu. Utile pour les jeux qui ne nécessitent pas de saisie de la souris ou qui utilisent un sprite de souris personnalisé.
  • Interpoler les couleurs entre les pixels: Active l'interpolation linéaire, qui a pour effet de "lisser" les pixels. Pour des graphiques de pixels nets, il est préférable de désactiver cette fonction, mais si vous avez de beaux mélanges alpha et des graphiques aux bords lissés, il est préférable de l'activer. Cette option est activée par défaut.
  • Mise à l'échelle: Votre jeu peut être configuré pour mettre automatiquement à l'échelle le canevas de dessin afin de conserver le rapport d'aspect dans le navigateur, ou vous pouvez choisir de le faire fonctionner à pleine échelle. L'option "Pleine échelle" n'affiche pas le jeu en plein écran dans le navigateur, mais étire plutôt ce qui est dessiné pour l'adapter à la taille de la toile, telle que définie par la première pièce du jeu. Par défaut, le rapport d'aspect est conservé.

Enfin, il est possible de définir la taille de la page de texture. La taille par défaut (et la plus compatible) est de 2048x2048, mais vous pouvez choisir entre 256x256 et 8192x8192. Il y a également un bouton marqué Prévisualisation qui génère les pages de texture pour cette plate-forme et ouvre une fenêtre pour que vous puissiez voir à quoi elles ressemblent. Cela peut être très utile si vous souhaitez voir comment les pages de texture sont structurées et éviter d'avoir des pages de texture plus grandes (ou plus petites) que nécessaire. Pour plus d'informations sur les pages de texture, voir ici.
