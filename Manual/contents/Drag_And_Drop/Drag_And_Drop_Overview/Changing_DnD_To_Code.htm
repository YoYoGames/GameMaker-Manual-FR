Changer GML Visual en Code

Une autre caractéristique de GML Visual est que vous pouvez le convertir en GML Code, et de cette façon voir ce qui se passe exactement dans les coulisses lorsque vous utilisez des actions. Toutefois, avant de transformer les actions en code, vous pouvez d'abord les prévisualiser en utilisant le bouton droit de la souris et l'option de menu Live Preview:

Ce qui ouvrira une nouvelle fenêtre :

Au fur et à mesure que vous ajoutez, modifiez ou supprimez des actions visuelles GML, l'aperçu en direct GML se met à jour pour vous montrer le code réel qui est créé "en coulisses". Le code dans l'aperçu en direct ne peut pas être modifié directement, mais vous pouvez sélectionner des sections et les copier pour les coller dans des scripts GML ou des actions de code (par exemple).

Si vous décidez de convertir le visuel GML en code après avoir vu l'aperçu, vous pouvez le faire à nouveau en cliquant sur le bouton droit de la souris dans n'importe quel espace de travail d'événement avec actions et en sélectionnant Convertir en code GML .

La première fois, vous recevrez un message d'avertissement indiquant qu'il s'agit d'une conversion à sens unique, car vous pouvez convertir des actions en code, mais vous ne pouvez pas les reconvertir en actions individuelles par la suite. En cliquant sur "OK", la conversion sera effectuée pour vous.

Le code résultant utilisera {} pour délimiter les actions individuelles, et vous pourrez clairement voir quelles actions se rapportent à quelles fonctions ou déclarations de variables dans le code. Si le visuel GML est plus complexe, le code le sera aussi, mais les mêmes règles générales s'appliquent et le code sera structuré séquentiellement exactement de la même manière que vous avez écrit le visuel GML. Lorsque vous apprenez à programmer à l'aide de GML Visual, cela peut être un outil important pour passer à l'utilisation de GML à un stade ultérieur, mais ce n'est en aucun cas obligatoire et vous pouvez toujours créer de superbes jeux en utilisant GML Visual ! Il est également intéressant de noter que, bien que le processus de conversion soit à sens unique, après avoir converti des actions en code, vous pouvez revenir à l'utilisation de GML Visual en utilisant le menu de clic droit dans l'éditeur de code et en sélectionnant "Convert To GML Visual".
