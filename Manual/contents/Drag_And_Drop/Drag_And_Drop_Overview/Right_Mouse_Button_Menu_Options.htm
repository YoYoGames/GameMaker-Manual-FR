Options de menu du bouton droit de la souris

Lorsque vous travaillez sur votre code GML Visual, vous pouvez cliquer avec le bouton droit de la souris sur l'action, sur l'espace de travail, ou sur le texte dans la liste restreinte à gauche pour ouvrir le menu du bouton droit de la souris :

Ce menu comporte les options suivantes :

  • Couper / Copier / Coller: Ceci vous permettra de couper, copier ou coller toute action sélectionnée dans l'espace de travail. Notez que vous pouvez utiliser + en dehors des actions et les faire glisser, ou utiliser / + sur des actions individuelles, pour en sélectionner plusieurs et appliquer la commande à chacune d'entre elles.
  • Commentaire / Supprimer le commentaire: Cette option vous permet d'ajouter et de supprimer les commentaires de vos blocs de code GML Visual. Lorsque vous sélectionnez Commentaire, un éditeur de texte s'ouvrira à droite du bloc sélectionné de GML Visual, et là vous pourrez ajouter le texte que vous souhaitez, qu'il s'agisse d'un rappel pour modifier quelque chose, d'un commentaire utile pour un tutoriel ou même d'instructions pour vos coéquipiers :

Les deux options suivantes sont uniquement disponibles lorsque le menu RMB est ouvert sur un nœud GML Visual :

  • Panoramique: Lorsque vous sélectionnez cette option sur n'importe quel nœud, l'espace de travail effectuera un panoramique pour faire correspondre ce nœud au centre.
  • Afficher l'aide... : Cette option ouvre le manuel dans un nouvel espace de travail sur la page correspondant au nœud sélectionné. Vous pouvez également obtenir les mêmes résultats en utilisant sur le nœud.
