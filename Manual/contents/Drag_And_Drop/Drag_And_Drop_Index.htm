GML Visual

GML Visual est la méthode de script visuel pour programmer avec le GameMaker Language (GML). Elle consiste à utiliser des blocs d'action pour construire la logique du jeu de manière visuelle et intuitive. C'est un outil idéal pour les personnes qui apprennent la programmation ou pour celles qui ont un penchant plus visuel.

La section GML Visual du manuel est divisée en plusieurs parties :
