Connectez-vous au Gamepad

Vous pouvez utiliser cette action pour vérifier et voir si un gamepad "slot" donné a un gamepad connecté. Vous fournissez l'index du gamepad (c'est le numéro du "slot" qu'un gamepad occupe) et la valeur renvoyée sera stockée dans la variable cible que vous fournissez pour une utilisation future (elle peut être marquée comme une variable locale temporaire que vous créez pour cet événement ou ce script).

Syntaxe de l'action :

Arguments :

Arguments Description
Gamepad L'index du gamepad à vérifier.
Target La variable cible dans laquelle stocker la valeur retournée.
