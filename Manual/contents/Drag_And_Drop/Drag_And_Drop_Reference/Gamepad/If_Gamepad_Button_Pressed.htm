Si le bouton du Gamepad est pressé

Avec cette action, vous pouvez interroger le gamepad pour voir si un bouton est actuellement pressé. L'action renverra true à chaque étape où le bouton du gamepad est enfoncé (et maintenu), ou false sinon, bien que si vous cliquez sur le modificateur not vous pouvez alors vérifier si le bouton n'est pas enfoncé, c'est-à-dire que l'action renverra true si aucun bouton n'est enfoncé et false s'il l'est. Si vous n'avez besoin de vérifier qu'une seule pression sur un bouton, alors utilisez l'action if gamepad button down.

Si le bouton vérifié est un bouton analogique, la pression ne sera pas détectée tant que la valeur du bouton n'aura pas dépassé le seuil défini à l'aide de l'action Set Gamepad Button Threshold.
