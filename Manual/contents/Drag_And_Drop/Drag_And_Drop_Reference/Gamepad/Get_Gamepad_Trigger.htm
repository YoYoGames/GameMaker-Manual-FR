Obtenir la gâchette du Gamepad

Vous pouvez utiliser cette action pour obtenir la valeur des différents triggers d'un gamepad donné. Vous fournissez l'index du gamepad (c'est le numéro du "slot" qu'un gamepad occupe) et sélectionnez le trigger à vérifier (gauche ou droite). La valeur renvoyée sera comprise entre 0 et 1, où 0 correspond à une pression nulle sur le trigger et 1 à une pression totale. Cette valeur retournée sera stockée dans la variable cible que vous fournissez pour une utilisation future. Notez que cette fonction ne prend pas en compte le réglage du seuil.

Syntaxe de l'action :

Arguments :

Arguments Description
Gamepad L'index du gamepad.
Trigger Le déclencheur à vérifier (gauche/droite)
Target La variable cible dans laquelle stocker la valeur retournée.

Le code du bloc d'action ci-dessus crée une variable locale (temporaire) et l'utilise pour stocker la valeur actuelle du déclencheur gauche. la valeur actuelle du trigger gauche. Si la valeur du trigger est supérieure à 0,5, une instance est créée à la position de l'instance appelante.
