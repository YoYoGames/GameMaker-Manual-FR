Si le bouton du Gamepad est enfoncé

Avec cette action, vous pouvez interroger un gamepad pour savoir si un bouton donné est actuellement maintenu enfoncé. L'action retournera true à chaque étape où le bouton est maintenu enfoncé, ou false sinon, bien que si vous cliquez sur le modificateur not vous pouvez alors vérifier si le bouton n'est pas maintenu enfoncé, c'est-à-dire que l'action renverra true si aucun bouton n'est enfoncé et false s'il l'est. Si vous n'avez besoin de vérifier qu'une seule pression sur un bouton, utilisez l'action If Gamepad Button Pressed.

Si le bouton vérifié est un bouton analogique, il ne sera pas considéré comme maintenu enfoncé tant que la valeur du bouton n'aura pas dépassé le seuil défini à l'aide de l'action Set Gamepad Button Threshold.

Ces actions seront maintenant exécutées si le bloc "if" donne la valeur true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if".

Syntaxe de l'action :

Arguments :

Arguments Description
Gamepad L'index du gamepad.
Button Le nom du bouton à vérifier.
Not Négation de la vérification (vrai devient faux et vice versa)
