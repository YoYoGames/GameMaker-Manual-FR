Définir le seuil des boutons de la manette de jeu

Cette action peut être utilisée pour définir le "seuil" des boutons analogiques du gamepad. Vous spécifiez l'index du gamepad à définir, puis vous définissez une valeur de 0 à 1. Si le montant de l'entrée du bouton analogique est inférieur à la valeur donnée, le bouton du gamepad est considéré comme étant à 0. Notez qu'il s'agit d'un paramètre global qui affectera tous les boutons analogiques connectés au gamepad. Cette valeur sera utilisée dans toutes les vérifications d'appui, de pression et de relâchement pour le gamepad donné, mais sera ignorée par l'action Get Gamepad Trigger.

Arguments :

Gamepad L'index du gamepad.
