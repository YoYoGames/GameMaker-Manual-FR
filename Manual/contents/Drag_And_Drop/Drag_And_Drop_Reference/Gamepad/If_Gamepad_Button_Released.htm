Si le bouton du Gamepad est relâché

Cette action vérifie quand un bouton de manette de jeu a été relâché pour l'index de manette de jeu donné. Elle renvoie true lors de l'unique tic-tac du jeu où le bouton est relâché et false dans tous les autres cas. Si vous activez le modificateur not, cette action sera inversée, et vérifiera si le bouton n'a pas été relâché, renvoyant false au tick de jeu où il a été relâché et true à tous les autres moments.

Si le bouton vérifié est un bouton analogique, le relâchement ne sera pas détecté tant que la valeur du bouton ne sera pas passée sous le seuil défini à l'aide de l'action Set Gamepad Button Threshold.

Ces actions seront maintenant exécutées si le bloc "if" donne la valeur true tandis que toute action déposée ailleurs sera exécutée après le bloc "if".

Arguments :

Gamepad L'index du gamepad.
