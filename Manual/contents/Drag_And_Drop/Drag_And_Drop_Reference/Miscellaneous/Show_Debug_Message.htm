Afficher le message de débogage

Avec cette action, vous pouvez envoyer un message de débogage à la sortie du compilateur. Ce message peut être une chaîne ou un nombre réel, mais si vous souhaitez utiliser les deux, le nombre doit être converti en chaîne (voir l'action Number To String). Notez que vous pouvez envoyer plusieurs messages en utilisant cette action en cliquant sur l'icône pour développer l'action avec des champs de sortie supplémentaires.
