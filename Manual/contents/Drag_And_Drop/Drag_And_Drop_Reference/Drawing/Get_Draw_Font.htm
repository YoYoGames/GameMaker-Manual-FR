Obtenir la police Draw

Cette action vous permet d'obtenir la valeur d'identification de la police actuellement affectée au dessin. La valeur renvoyée sera soit -1 si aucune police n'est attribuée, soit une valeur >= 0 qui représente une constante de police du navigateur d'actifs. La valeur de retour sera stockée dans la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou peut être une nouvelle variable temporaire (si vous cochez la case "Temp").
