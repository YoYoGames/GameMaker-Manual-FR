Obtenir l'alignement du texte

Cette action récupère l'alignement de la police pour toutes les actions actuelles de dessin de texte. Vous pouvez choisir de récupérer l'alignement horizontal ou vertical à vérifier, et l'action renverra l'une des valeurs constantes suivantes, qui représentent un type différent d'alignement horizontal ou vertical: :. type d'alignement horizontal ou vertical :

Le texte est aligné horizontalement sur la gauche
Le texte est aligné horizontalement au centre
Le texte est aligné horizontalement à droite
Le texte est aligné verticalement en haut
Le texte est aligné verticalement au milieu
Le texte est aligné verticalement sur le bas

La constante de retour sera stockée dans la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou être une nouvelle variable temporaire (si vous cochez la case "Temp").
