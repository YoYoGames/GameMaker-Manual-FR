Dessiner soi-même

Lorsque vous ajoutez un type d'action ou de code à l'événement Draw d'un objet, vous dites à GameMaker que vous prenez le contrôle et que vous lui direz ce qu'il doit dessiner à partir de maintenant (par défaut, une instance dessinera le sprite qui lui est affecté dans l'éditeur d'objets lorsqu'il n'y a pas d'action dans le Draw Event principal). Cela signifie que tout sprite assigné ne sera plus dessiné à moins que vous ne le disiez explicitement, ce qui est le cas lorsque vous utilisez cette action. Cette action dessine simplement l'adresse sprite assignée pour l'objet avec toutes les transformations qui ont pu être définies dans des événements précédents (comme la mise à l'échelle, le mélange ou les changements d'alpha) dans la sous-image actuelle.

REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs.
