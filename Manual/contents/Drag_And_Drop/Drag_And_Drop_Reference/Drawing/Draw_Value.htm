Valeur du tirage au sort

Avec cette action, vous pouvez dessiner une légende de chaîne et/ou une variable ou une valeur à l'écran. L'argument de la légende est une chaîne que vous souhaitez ajouter avant la valeur, et peut être laissé vide si nécessaire, mais si vous souhaitez inclure du texte, il doit être entouré de guillemets. entre guillemets, et la valeur peut être une variable, une valeur fixe ou une autre chaîne - dans ce cas, elle doit également être placée entre guillemets - mais elle ne peut pas être laissée vide. Vous indiquez ensuite la position où la chaîne de caractères doit être dessinée, soit en tant que position absolue ou une position relative à l'instance qui dessine le texte. Notez que l'aspect du texte une fois dessiné dépendra de la police utilisée, de la couleur utilisée et de l'alignement. Tous ces éléments peuvent être modifiés avant le dessin, comme le montre l'exemple ci-dessous.

REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs.

Syntaxe de l'action :

Arguments :

Arguments Description
Caption La légende à dessiner (une chaîne de caractères), peut être laissée vide.
Value La valeur à dessiner (réel, variable ou chaîne)
x La position x à dessiner
y La position y à dessiner
