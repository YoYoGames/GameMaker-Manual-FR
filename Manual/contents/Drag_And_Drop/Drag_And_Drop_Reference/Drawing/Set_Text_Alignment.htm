Définir l'alignement du texte

Cette action définira l'alignement de la police pour toutes les actions suivantes de dessin de texte. Vous pouvez définir l'alignement horizontal comme étant la gauche, le centre ou la droite, et l'alignement vertical comme étant le haut, le milieu ou le bas, et le texte sera aligné par rapport à la position x/y de l'action de dessin, comme le montrent les images ci-dessous :

Le texte est aligné horizontalement sur la gauche
Le texte est aligné horizontalement au centre
Le texte est aligné horizontalement à droite
Le texte est aligné verticalement en haut
Le texte est aligné verticalement au milieu
Le texte est aligné verticalement sur le bas

Notez que cette action définira l'alignement à utiliser pour tous les appels ultérieurs de texte à dessiner, et peut donc être appelée, par exemple, une fois au début du jeu - dans tous les cas, pas seulement l'événement de dessin - et toutes les actions de texte à dessiner utiliseront l'alignement défini. Cependant, si vous souhaitez que plusieurs actions de texte à dessiner soient alignées de différentes manières, vous devrez appeler cette action avant chaque action de texte à dessiner que vous souhaitez aligner différemment.

Syntaxe de l'action :

Arguments :

Arguments Description
H.Align La valeur de l'alignement horizontal (Gauche, Centre, Droite)
V.Align La valeur de l'alignement vertical (Haut, Milieu, Bas)
