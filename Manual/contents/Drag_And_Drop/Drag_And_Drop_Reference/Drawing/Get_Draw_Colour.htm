Obtenir la couleur du dessin

Avec cette action, vous pouvez obtenir la couleur utilisée pour toutes les formes (rectangle/ellipse/ligne) ou pour tout texte qui est dessiné. L'action renverra la couleur à la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou peut être une nouvelle variable temporaire (si vous cochez la case "Temp"). être une nouvelle variable temporaire (si vous cochez la case "Temp").
