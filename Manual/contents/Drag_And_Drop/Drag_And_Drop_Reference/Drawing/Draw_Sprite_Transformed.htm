Dessiner Sprite Transformé

Cette fonction dessine le site sprite de manière similaire à l'action Draw Sprite, mais avec des options supplémentaires permettant de modifier l'échelle, le mélange, la rotation et le cadre du site sprite dessiné. Le changement de ces valeurs ne modifie pas la ressource de quelque manière que ce soit (seulement la manière dont elle est dessinée), et vous pouvez utiliser n'importe quelle variable de propriété de sprite disponible au lieu de valeurs directes pour tous les arguments de la fonction. L'image ci-dessous illustre comment différentes valeurs affectent le dessin du sprite :

REMARQUE Le mélange de couleurs n'est recommandé pour la cible HTML5 que lorsque WebGL est activé, bien que vous puissiez toujours définir la couleur de mélange si elle n'est pas activée et le mélange se fera normalement sur sprite. Cependant, tous les mélanges de cette manière créent un doublon sprite qui est ensuite stocké dans le site cache et utilisé si nécessaire. C'est loin d'être optimal et si vous utilisez de multiples changements de couleur, cela ralentira les performances de votre jeu, à moins que vous n'activiez la fonction WebGL. la taille de la police cache pour essayer de limiter ce phénomène si nécessaire en utilisant la fonction sprite_set_cache_size().

REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs.

Syntaxe de l'action :

Arguments :

Arguments Description
Sprite Le site sprite à dessiner
Frame L'image sprite à dessiner (utilisez la variable intégrée image_index pour l'image actuelle).
X La position x à laquelle dessiner dans la pièce
Y La position y pour dessiner dans la pièce
Scale X Le facteur d'échelle pour l'axe X (utilisez la variable intégrée image_xscale pour l'échelle X actuelle)
Scale Y Le facteur d'échelle pour l'axe Y (utilisez la variable intégrée image_yscale pour l'échelle Y actuelle)
Rotation L'angle de dessin pour le sprite (de 0 à 360, rotation dans le sens inverse des aiguilles d'une montre où 0 est à droite - utilisez la variable intégrée image_angle pour la rotation actuelle)
Colour La couleur à mélanger avec le site sprite (la valeur par défaut est le blanc, et vous pouvez utiliser la variable intégrée image_blend pour la couleur actuelle).
