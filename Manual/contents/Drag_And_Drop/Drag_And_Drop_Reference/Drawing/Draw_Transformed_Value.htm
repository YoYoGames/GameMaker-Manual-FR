Dessiner la valeur transformée

Avec cette action, vous pouvez dessiner une légende de chaîne et/ou une variable ou une valeur à l'écran. L'argument de la légende est une chaîne que vous souhaitez ajouter avant la valeur, et peut être laissé vide si nécessaire, et la valeur peut être une variable, une valeur fixe, ou une autre chaîne - dans ce cas, elle doit être placée entre guillemets, et peut également être laissée en blanc si nécessaire. Vous indiquez ensuite la position où la chaîne de caractères doit être dessinée, soit en tant que position absolue, soit en tant que position relative à l'instance dessinant le texte. De plus, vous pouvez définir l'échelle du texte, en l'étirant le long de l'axe horizontal ou vertical, ainsi que la rotation (les angles de rotation du texte sont dans le sens inverse des aiguilles d'une montre, donc une rotation de 90° le fera pivoter vers la gauche et de -90° vers la droite). La mise à l'échelle est effectuée par le facteur indiqué, ainsi une échelle de 0,5 réduira de moitié la largeur du texte et une échelle de 2 la doublera, 1 étant la valeur par défaut pour l'absence de mise à l'échelle. Notez que l'aspect du texte lorsqu'il est dessiné dépend des facteurs suivants de la police utilisée, de la couleur utilisée et de l'alignement. Tous ces éléments peuvent être modifiés avant le dessin, comme le montre l'exemple ci-dessous.

REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs.

Syntaxe de l'action :

Arguments :

Arguments Description
Caption La légende à dessiner (une chaîne de caractères), peut être laissée vide.
Value La valeur à dessiner (réel, variable ou chaîne)
x La position x à dessiner
y La position y à dessiner
xscale Le facteur d'échelle x (horizontal), la valeur par défaut est 1
yscale Le facteur d'échelle y (vertical), la valeur par défaut est 1
Rotation L'angle de rotation pour dessiner le texte
