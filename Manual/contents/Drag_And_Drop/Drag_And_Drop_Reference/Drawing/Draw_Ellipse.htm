Dessiner une ellipse

Cette action va dessiner une ellipse à une position donnée dans la pièce. Vous indiquez la position supérieure gauche et la position inférieure droite de la zone dans laquelle l'ellipse doit s'insérer, et l'ellipse sera dessinée entre ces deux positions. Les positions peuvent être une position absolue dans la room, ou une position relative à l'instance appelant l'action, et vous pouvez également définir si l'ellipse peut être remplie ou délimitée en cochant la case de remplissage en bas.

REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs.

Arguments :

Arguments Description
Left La position gauche pour commencer à dessiner à partir de
Top La meilleure position pour commencer à dessiner
Right La bonne position pour terminer le dessin
Bottom La position inférieure pour terminer le dessin de
