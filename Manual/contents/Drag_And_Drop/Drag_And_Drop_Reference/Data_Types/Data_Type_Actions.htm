Types de données Bibliothèque d'action

GameMaker possède un certain nombre de types de données différents, comme les entiers, les chaînes de caractères, les pointeurs, etc... et certains d'entre eux peuvent être convertis en d'autres. Cette section du manuel présente les différentes actions disponibles pour traiter et convertir les différents types de données.

REMARQUE: vous pouvez en savoir plus sur les différents types de données ici.

La bibliothèque des types comporte les actions suivantes :

De décimal à entier
Chaîne en nombre
Nombre vers chaîne
