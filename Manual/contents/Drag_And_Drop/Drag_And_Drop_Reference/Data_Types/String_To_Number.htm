Chaîne en nombre

Cette action peut convertir une chaîne de caractères en un nombre. Vous fournissez la chaîne à modifier, puis une variable cible à laquelle renvoyer le nombre réel (cette variable peut être marquée comme une variable locale temporaire). Notez que si la chaîne de caractères contient des caractères autres que des chiffres et un point décimal, elle ne sera pas convertie, mais si la chaîne commence par un chiffre, celui-ci sera converti et le reste de la chaîne sera exclu. Il est important de noter que lorsque vous passez des décimales dans cette action la chaîne retournée sera arrondie à deux décimales seulement. Voici un tableau des valeurs d'entrée et de ce que vous pouvez vous attendre à recevoir en retour.

Chaîne de caractères Valeur retournée
"123" 123
"123 Hello World" 123
"Hello World" 0 (par défaut, si la chaîne ne peut pas être convertie, 0 sera renvoyé)
"Hello World 123" 0
"123.45678" 123.46
