Nombre vers chaîne

Cette action convertit un nombre réel en une chaîne de caractères. Si le nombre n'est pas un nombre entier et comporte des décimales, celles-ci seront arrondies aux deux chiffres les plus proches. Ainsi, par exemple, le nombre 123.45678 donnera la chaîne "123.46".

Syntaxe de l'action :

Arguments :

Arguments Description
Value Le nombre réel à convertir
Target La variable cible qui contient la valeur retournée

Exemple :

Le code du bloc d'action ci-dessus prend une valeur décimale et la convertit en un nombre entier, en stockant la valeur retournée dans une variable locale temporaire. Cette valeur est ensuite transformée en une chaîne de caractères et affichée à l'écran.
