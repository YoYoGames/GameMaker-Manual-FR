Obtenir la variable Tilemap

Cette action vous permet de récupérer l'une des variables d'un nombre donné de variables pour l'élément de carte de carreaux d'une couche. Lorsque vous créez une couche de tuiles dans l'éditeur de pièce, cette couche contient un élément de carte de tuiles qui est ensuite rempli de tuiles d'un jeu de tuiles. Cet élément de carte de tuiles est associé à certaines valeurs, comme une position de décalage, le jeu de tuiles utilisé, la largeur, la hauteur, etc... La liste complète des valeurs que vous pouvez obtenir est la suivante :

  • Coordonnée X: La position le long de l'axe x sur la couche où l'élément de carte à tuiles a été placé.
  • Coordonnée Y: La position le long de l'axe des y sur la couche où l'élément de carte de tuiles a été placé.
  • Colonnes: La largeur (définie comme le nombre de colonnes de cellules de tuiles) de l'élément de carte de tuiles.
  • Rows (rangées): La hauteur (définie comme le nombre de rangées de cellules de tuiles) de l'élément de carte de tuiles.
  • Largeur de la tuile: La largeur d'une seule cellule de la tuile.
  • Hauteur des tuiles: La hauteur d'une seule cellule de tuile.
  • Ressource de jeu de tuiles: La ressource de jeu de tuiles qui a été assignée à l'élément de carte de tuiles pour le dessin.
  • Cadre actuel: Le cadre actuel dessiné pour les tuiles animées.
  • Mask: Les données du masque pour l'élément de la carte des tuiles.
  • Global Mask: Les données du masque global pour tous les éléments de la carte des tuiles.

Lorsque vous utilisez cette action, vous devez fournir le nom de la couche (une chaîne, telle que définie dans l'éditeur de pièce) pour obtenir les données de l'élément de carte de tuiles, puis le type de données que vous souhaitez récupérer (comme indiqué dans la liste ci-dessus). La valeur renvoyée sera ensuite stockée dans la variable cible qui peut avoir été créée précédemment ou être une nouvelle variable temporaire (si vous cochez la case "Temp"). Notez que vous pouvez récupérer des valeurs supplémentaires en cliquant sur l'icône plus à côté de l'action, en sélectionnant une autre variable et en donnant une autre variable pour stocker la valeur retournée.
