Set Tile Set

Cette action vous permet de configurer un calque pour utiliser des jeux de tuiles différents de ceux créés dans le navigateur d'actifs. Vous donnez le nom de la couche (une chaîne de caractères, telle que définie dans l'éditeur de pièce), et ensuite la ressource de jeu de tuiles à utiliser, et toutes les tuiles de la pièce sur cette couche seront dessinées avec le nouveau jeu de tuiles.

Syntaxe de l'action :

Arguments :

Arguments Description
Layer Le calque sur lequel doit être placé le nouveau jeu de tuiles.
Tileset Le jeu de tuiles à utiliser pour la couche.
