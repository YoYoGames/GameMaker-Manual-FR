Transformer les données des tuiles

Avec cette action, vous pouvez modifier l'une des trois transformations de tuile dans les données de tuile. Vous fournissez la variable qui contient les données de la tuile, puis vous sélectionnez la transformation à modifier. En cliquant sur la case à cocher "Appliquer", vous appliquerez cette transformation, et en la décochant, vous la supprimerez. décochez-la pour la supprimer. Les données de la tuile sont simplement une valeur qui reflète l'index de la tuile avec les transformations appliquées ainsi que tout masque de tuile personnalisé que vous avez utilisé, et cette action vous permet de modifier trois de ces valeurs : Flip, mirror et rotate. Notez que vous voudrez normalement obtenir les données de la tuile d'abord en utilisant soit Obtenir les données de la tuile dans la cellule, soit Obtenir les données de la tuile au pixel. puis définir à nouveau la tuile dans la carte des tuiles en utilisant Set Tile Data In Cell ou Set Tile Data At Pixel, puisque cette action définit uniquement une valeur mais ne l'applique pas à la tuile dont elle provient. Pour plus d'informations sur les données de tuile, veuillez consulter la section GML sur les fonctions de carte de tuile.
