Si Fin De Tampon

Cette action peut être utilisée pour vérifier si la position de lecture/écriture a atteint la fin de buffer. Vous fournissez la valeur ID unique de buffer telle que renvoyée par l'action Create Buffer et l'action renverra true si la position de lecture/écriture a atteint la fin de toutes les données disponibles dans buffer, ou false sinon. Vous pouvez marquer "not" pour que l'action vérifie si la position de lecture/écriture n' est pas à la fin de buffer.
