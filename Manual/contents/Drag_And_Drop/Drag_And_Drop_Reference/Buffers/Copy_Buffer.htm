Tampon de copie

Cette action peut être utilisée pour copier toutes les données stockées dans un buffer vers un autre. Lorsque vous utilisez deux buffers et que vous copiez de l'un à l'autre, les deux buffers doivent avoir été préalablement créés à l'aide de l'action Create Buffer, et vous devez fournir l'ID unique de buffer retourné comme arguments de/à.

REMARQUE: Vous ne pouvez pas copier sur le même site buffer.

Syntaxe de l'action :

Arguments :

Arguments Description
Buffer L'index buffer (stocké dans une variable) à copier à partir de
Copy To L'index buffer (stocké dans une variable) à copier vers
