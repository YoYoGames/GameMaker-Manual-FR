Obtenir la position de Buffer

Cette action peut être utilisée pour récupérer la position actuelle de lecture/écriture sur buffer. Lorsque vous créez un buffer, la position de lecture/écriture est fixée à 0 octet, et au fur et à mesure que vous ajoutez des données sur buffer, cette position se déplace séquentiellement, ce qui signifie que vous pouvez utiliser cette action pour obtenir le nombre d'octets dans le buffer qu'il est défini à tout moment. Vous fournissez la valeur d'identification unique de buffer telle qu'elle est renvoyée par l'action Créer un tampon, puis vous donnez une variable cible à laquelle renvoyer la valeur de position. la valeur de la position. La variable cible peut être marquée comme étant une variable locale temporaire, auquel cas elle sera créée pour vous et supprimée de la mémoire à la fin du script ou de l'événement de l'action en cours.
