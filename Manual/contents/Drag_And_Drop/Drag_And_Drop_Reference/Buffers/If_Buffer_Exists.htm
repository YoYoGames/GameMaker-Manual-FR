Si Buffer existe

Cette action peut être utilisée pour vérifier si un ID donné buffer est valide et si le buffer existe réellement en mémoire. Si l'ID buffer est valide et existe, l'action retournera true sinon elle retournera false. Vous pouvez marquer "not" (pas) pour que l'action vérifie si un buffer n' existe pas.
