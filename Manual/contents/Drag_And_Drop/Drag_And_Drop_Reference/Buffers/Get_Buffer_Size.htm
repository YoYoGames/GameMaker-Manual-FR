Obtenir la taille de Buffer

Cette action peut être utilisée pour récupérer la taille d'un buffer, en octets. Vous fournissez la valeur unique de l'ID de buffer telle qu'elle est renvoyée par l'action Créer un tampon, puis vous donnez une variable cible à laquelle renvoyer la valeur de la taille. La variable cible peut être marquée comme étant une variable locale temporaire, auquel cas elle sera créée pour vous et supprimée de la mémoire à la fin du script ou de l'événement de l'action en cours.

Syntaxe de l'action :

Arguments :

Arguments Description
Buffer L'indice buffer (stocké dans une variable)
Target La variable cible qui contient la valeur retournée
