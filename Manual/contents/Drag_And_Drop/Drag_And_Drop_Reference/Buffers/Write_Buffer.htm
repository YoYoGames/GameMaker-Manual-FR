Tampon d'écriture

Cette action peut être utilisée pour écrire des données dans un buffer. Vous fournissez l'ID unique de buffer dans lequel écrire (tel que retourné par l'action Create Buffer) et ensuite la valeur à écrire, qui peut être un réel ou une chaîne, mais notez que toutes les valeurs sont stockées sous forme de chaînes de caractères, donc lorsque vous lisez à partir de buffer (en utilisant Read Buffer), vous pouvez avoir besoin d'utiliser l'une des actions de type de données pour le convertir d'une chaîne de caractères à la valeur originale. en valeur d'origine. Gardez également à l'esprit que buffers est lu et écrit de manière séquentielle et que l'écriture sur buffer déplacera la position de recherche d'une unité, ce qui signifie que pour écrire divers éléments de données, il suffit d'appeler cette action le nombre de fois nécessaire.
