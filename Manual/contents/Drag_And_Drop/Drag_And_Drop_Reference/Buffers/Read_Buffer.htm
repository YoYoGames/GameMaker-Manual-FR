Tampon de lecture

Cette action peut être utilisée pour lire les données d'un buffer précédemment créé. Vous fournissez la valeur d'identification unique de buffer (telle que renvoyée par l'action Créer un tampon) et donnez une variable pour stocker la valeur renvoyée. Si vous marquez la variable comme une variable locale temporaire, elle sera créée pour vous et supprimée à la fin de l'événement ou du script d'action. Toutes les données sont renvoyées sous forme de chaîne de caractères. Si vous avez stocké des nombres réels, vous devrez utiliser l'une des actions de type de données pour les convertir de la chaîne de caractères en valeur. Gardez également à l'esprit que buffers est lu et écrit de manière séquentielle et que la lecture d'un fichier buffer déplacera la position de recherche vers l'entrée suivante, ce qui signifie que pour lire divers éléments de données il suffit d'appeler cette action le nombre de fois nécessaire et de stocker chaque élément de données renvoyé dans une variable.
