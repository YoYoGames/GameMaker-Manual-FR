Tampon de recherche

Cette action peut être utilisée pour se déplacer (rechercher) vers une position à l'intérieur d'un buffer. Vous fournissez la valeur d'identification unique de buffer telle que retournée par l'action Create Buffer et ensuite la valeur de décalage en octets vers laquelle vous souhaitez vous déplacer. La valeur de décalage sera La valeur de décalage sera appliquée à partir du début du buffer, donc l'appel de cette fonction déplacera la position de lecture/écriture au début du buffer puis déplacera le nombre d'octets donné et positionnera la nouvelle position de lecture/écriture à cet endroit. Cependant, si vous marquez le décalage comme relatif, la position de lecture/écriture sera déplacée vers l'avant (ou vers l'arrière si le décalage est une valeur négative) à partir de la position actuelle.
