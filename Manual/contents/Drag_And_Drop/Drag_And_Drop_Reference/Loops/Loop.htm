Boucle

L'action Loop tournera en boucle à l'infini jusqu'à ce qu'une action Break soit appelée. Il faut faire attention à cette action pour s'assurer que, quelles que soient les actions placées dans la boucle, une pause sera toujours appelée, sinon sinon le jeu provoquera une erreur de type "not responding" sur le système d'exploitation cible et cessera de fonctionner. Vous ajoutez cette action dans l'espace de travail du code d'action, puis vous ajoutez les actions à exécuter dans la boucle, comme vous ajouteriez une action à un "if", c'est-à-dire en la plaçant sur le côté de l'action plutôt que sous celle-ci :
