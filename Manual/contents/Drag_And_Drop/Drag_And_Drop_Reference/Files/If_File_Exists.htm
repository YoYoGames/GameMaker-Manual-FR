Si le fichier existe

Avec cette action, vous pouvez vérifier si un fichier portant le nom donné existe. Vous sélectionnez le fichier à vérifier (sous forme de chaîne et en incluant l'extension) et la fonction renverra true s'il existe, et false s'il n'existe pas. Notez que cette fonction renverra true si le fichier existe dans le paquet de jeu en tant que fichier inclus ou si le fichier a été sauvegardé pendant que le jeu était en cours d'exécution (voir la section sur le système de fichiers pour plus d'informations).
