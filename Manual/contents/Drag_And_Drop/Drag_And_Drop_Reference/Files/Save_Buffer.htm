Sauvegarder le tampon

Cette action vous permet de sauvegarder un fichier contenant les données d'un buffer. Vous fournissez l'ID unique de buffer (tel que retourné lorsque vous avez créé le buffer avec Create Buffer) et donnez un nom au fichier (sous forme de chaîne de caractères, avec l'extension de votre choix) et le site buffer sera écrit dans ce fichier, prêt à être chargé à nouveau avec Load Buffer.

Syntaxe de l'action :

Arguments :

Arguments Description
Buffer La valeur unique de l'ID buffer du buffer à sauvegarder
Filename Le nom (sous forme de chaîne) du fichier à sauvegarder

Le code du bloc d'action ci-dessus crée une variable pour contenir les données de buffer et vérifie ensuite si un fichier de sauvegarde a été effectué précédemment. buffer fichier de sauvegarde a été créé précédemment. Si le fichier existe, il est chargé dans la variable buffer que nous avons créée, mais s'il n'existe pas, il est créé, écrit puis sauvegardé, prêt à être chargé la prochaine fois.
