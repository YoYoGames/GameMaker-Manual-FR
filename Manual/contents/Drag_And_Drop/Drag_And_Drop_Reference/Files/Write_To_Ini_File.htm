Ecrire dans un fichier Ini

Avec cette action, vous pouvez écrire une valeur numérique ou une chaîne de caractères dans un fichier Ini. Vous devez préalablement avoir ouvert le fichier Ini en lecture/écriture à l'aide de l'action Ouvrir un Fichier Ini, puis avec cette action vous donnez le type de données à écrire (soit un nombre réel, soit une chaîne de caractères entourée de ""), ainsi que l'en-tête de section, la clé dans cette section et la valeur de la clé. Pour aider à visualiser la structure d'un fichier Ini, voir cette image :

Comme vous pouvez le voir, les fichiers Ini sont divisés en sections - où chaque section a son propre nom d'en-tête - et puis chaque section est ensuite divisée en diverses paires clé/valeur (un peu comme une structure de données Map). Notez que vous pouvez créer vous-même des fichiers Ini à l'aide de n'importe quel logiciel de traitement de texte (comme le Bloc-notes sur Windows), puis les inclure dans votre jeu en tant que fichier inclus dans lequel vous pourrez lire et écrire.

IMPORTANT ! Essayer de lire les données d'un fichier Ini qui n'a pas été ouvert précédemment à l'aide de l'action Ouvrir un fichier Ini provoquera une erreur grave et fera planter votre jeu.

IMPORTANT ! Cette opération ne fait qu'écrire les données en mémoire. Pour écrire définitivement les données dans le fichier sur le disque, vous devez appeler Close Ini File, sinon les données ne seront pas enregistrées.

Syntaxe de l'action :

Arguments :

Arguments Description
Type Le type de données à écrire, soit un string (entre guillemets ""), soit un real number
Section La section du fichier Ini dans laquelle il faut écrire (sous forme de chaîne).
Key La clé de la section sur laquelle écrire (sous forme de chaîne).
Value La valeur à écrire, soit une chaîne de caractères, soit un nombre réel basé sur l'argument "type" (ci-dessus).
