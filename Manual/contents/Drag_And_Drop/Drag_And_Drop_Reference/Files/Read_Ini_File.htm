Lire le fichier Ini

Cette action vous permet de lire une donnée à partir d'un fichier Ini. Vous devez au préalable avoir ouvert le fichier Ini pour la lecture/écriture en utilisant l'action Ouvrir un fichier Ini, puis avec cette action vous donnez le type de données à lire (soit un nombre réel ou une chaîne de caractères entourée de ""), ainsi que l'en-tête de section et la clé dans cette section pour obtenir les données. Vous pouvez également fournir une valeur par défaut qui peut être un nombre réel ou une chaîne de caractères, et le type doit coïncider avec le paramètre "Type". coïncider avec le "Type" défini dans l'action. Cette action par défaut sera retournée par l'action si l'une des situations suivantes se produit :

  • Le fichier Ini que vous avez ouvert n'existe pas.
  • L'en-tête de section auquel vous faites référence n'existe pas
  • La clé dont vous essayez d'obtenir les données n'existe pas.

Dans l'une des circonstances ci-dessus, vous obtiendrez la valeur par défaut renvoyée plutôt qu'une valeur provenant du fichier Ini. Quelle que soit la valeur renvoyée, elle sera stockée dans la variable cible que vous spécifiez, qui peut également être créée à ce moment-là si elle est marquée comme une variable locale temporaire.

IMPORTANT ! Si vous essayez de lire les données d'un fichier Ini qui n'a pas été ouvert précédemment à l'aide de l'action Ouvrir un fichier Ini, cela provoquera une erreur grave et fera planter votre jeu.

Syntaxe de l'action :

Arguments :

Arguments Description
Type Le type de données à lire, soit un string (entre guillemets ""), soit un real number
Section La section du fichier Ini à lire (sous forme de chaîne)
Key La clé de la section à lire (sous forme de chaîne).
Default La valeur par défaut à retourner si le fichier, la section ou la clé n'existe pas.
Target La variable cible pour contenir la valeur de retour de l'action.
