Copier le fichier

Avec cette action, vous pouvez copier n'importe quel fichier qui a été sauvegardé par votre jeu ou qui fait partie des fichiers inclus. Vous donnez le nom du fichier à copier (sous forme de chaîne de caractères) et ensuite le nom du fichier à créer en tant que copie du fichier original (également sous forme de chaîne de caractères et avec l'extension de fichier). sous forme de chaîne de caractères et en incluant l'extension du fichier). Le fichier donné sera dupliqué et se verra attribuer le nouveau nom.

Syntaxe de l'action :

Arguments :

Arguments Description
Filename Le nom (sous forme de chaîne) du fichier à copier.
Copy To Le nom (sous forme de chaîne) du fichier à créer en tant que copie.
