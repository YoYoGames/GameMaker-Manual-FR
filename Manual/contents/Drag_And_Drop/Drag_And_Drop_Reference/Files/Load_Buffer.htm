Chargement de la mémoire tampon

Avec cette action, vous pouvez charger un buffer qui a été précédemment enregistré avec l'action Save Buffer. L'action créera un nouveau buffer, chargera le fichier de données dans celui-ci, et renverra la valeur ID unique pour identifier le buffer à la variable variable. Vous devez fournir le nom du fichier à charger (sous forme de chaîne de caractères) et une variable dans laquelle stocker l'ID retourné de buffer. Notez que si vous marquez la variable comme une variable locale temporaire, vous devez vous assurer de supprimer le buffer que cette fonction crée avant la avant la fin du script d'action ou de l'événement, sinon vous perdrez la référence à cette variable et obtiendrez une fuite de mémoire lorsque la variable sera supprimée.

Arguments :

Arguments Description
Filename Le nom (sous forme de chaîne) du fichier à ouvrir.
Target La variable cible qui contient la valeur d'identification unique pour le site buffer créé au chargement.
