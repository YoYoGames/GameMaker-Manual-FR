Renommer un fichier

Cette action vous permet de renommer n'importe quel fichier qui a été sauvegardé par votre jeu. Vous donnez le nom du fichier à modifier (sous forme de chaîne de caractères), puis le nouveau nom (également sous forme de chaîne de caractères et incluant l'extension du fichier), et le fichier sera renommé. Notez que vous ne pouvez pas renommer un fichier qui a été inclus dans votre jeu (voir la section sur le système de fichiers pour plus d'informations).

Arguments :

Arguments Description
Filename Le nom (sous forme de chaîne) du fichier à renommer
New Name Le nouveau nom (sous forme de chaîne) du fichier

Enfin, un tampon est sauvegardé en tant que "checkpoint.sav" (essentiellement, nous sauvegardons un fichier buffer à chaque fois que nous le sauvegardons, de sorte qu'il y a toujours une sauvegarde "actuelle" et un "ancien" fichier de sauvegarde).
