Appel de fonction

Cette action peut être utilisée pour appeler toute fonction GML ou toute fonction définie par l'utilisateur. Vous indiquez la fonction à appeler, puis vous fournissez tous les supplémentaires dont elle a besoin, c'est-à-dire toutes les valeurs (chaînes de caractères, entiers, indices de ressources, etc...) que vous souhaitez entrer dans la fonction pour qu'elle puisse les utiliser. Vous pouvez ajouter des arguments supplémentaires en cliquant sur l'icône plus à côté de Vous pouvez ajouter des arguments supplémentaires en cliquant sur l'icône plus à côté de l'action, et vous devez vous assurer que votre nombre d'arguments correspond à celui de la fonction utilisée (c'est-à-dire que si la fonction requiert 3 arguments, vous devez fournir 3 arguments). Certaines fonctions peuvent également renvoyer une valeur qui sera utilisée ultérieurement dans le bloc d'action. et vous pouvez donc également spécifier la variable cible qui doit stocker cette valeur de retour, qui peut avoir été créée précédemment ou être une nouvelle variable locale temporaire (si vous cochez la case "Temp"). Les champs cible et l'argument peuvent être laissés vides s'ils ne sont pas nécessaires.

Arguments Description

Argument0 ... ArgumentX Les différents arguments (valeurs) qui doivent être transmis à la fonction.
Target Le nom de la variable qui doit être ciblée pour toutes les valeurs retournées (peut être laissé vide).
