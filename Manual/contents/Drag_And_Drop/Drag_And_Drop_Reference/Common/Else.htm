Else

L'action "else" n' est jamais utilisée qu'en conjonction avec If Variable (ou toute autre action "if"), et permet de définir un ensemble d'actions qui doivent être exécutées si l'évaluation de "if" retourne renvoie false. Par exemple, vous pourriez avoir une action "if" qui vérifie si la variable de santé des joueurs est inférieure ou égale à 0, et si l'évaluation est vraie, vous détruisez l'instance.
