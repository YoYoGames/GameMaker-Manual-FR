Macro

Cette action vous permet de définir une ou plusieurs macros. Les macros sont simplement des valeurs constantes uniques - ou des expressions qui évaluent une valeur constante unique - qui peuvent ensuite être utilisées de manière similaire à une constante régulière. Vous pouvez également utiliser un appel de fonction dans cette action pour générer une valeur de retour unique, auquel cas la macro évaluera la fonction chaque fois qu'elle sera appelée. Notez que les macros ont une portée de global. Une fois déclarées, elles peuvent être utilisées par n'importe quelle instance, script ou bloc d'action de votre jeu. ou bloc d'action de votre jeu, mais elles ne peuvent pas être modifiées après avoir été définies. Notez que vous pouvez définir des macros supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action. Pour plus d'informations sur les macros, veuillez consulter cette page.
