Postuler à...

Cette action peut être utilisée pour changer la portée de l'action pour toutes les autres actions à enchaîner à celle-ci. Comme avec une if si vous déposez d'autres actions à droite de l'action Appliquer à..., elles seront exécutées à partir de l'objet ou de l'instance concerné.
