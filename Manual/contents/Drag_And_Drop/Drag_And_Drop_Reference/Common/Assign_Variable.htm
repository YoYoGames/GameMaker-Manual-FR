Attribuer une variable

Cette action est utilisée soit pour attribuer une valeur à une variable (en la créant dans le processus), soit pour modifier la valeur d'une variable existante. Vous donnez le nom de la variable et ensuite la valeur, qui peut être ajoutée par rapport à la valeur originale si nécessaire. nécessaire. Si vous donnez le nom d'une nouvelle variable (c'est-à-dire une variable qui n'a jamais été assignée auparavant dans l'instance), le programme créera la nouvelle variable et lui assignera la valeur que vous avez donnée, tandis que si vous donnez le nom d'une variable existante, il simplement changer la valeur de la nouvelle variable. Notez que si vous cochez la case Is Relative, la valeur sera ajoutée à la valeur de la variable existante (utilisez un nombre négatif pour la soustraire) et vous pouvez également utiliser les expressions et les variables créées précédemment comme valeur.

Il convient de noter que lorsque nous parlons de "valeurs", nous ne parlons pas uniquement de valeurs numériques, car une variable peut être une chaîne de caractères, un pointeur, un ID de ressource ou tout autre élément qu'une fonction peut retourner ou utiliser. Dans le cas d'une chaîne de caractères, la vérification de Relative va concaténera la valeur avec ce que la variable contenait précédemment, donc si votre variable était "Hello" et que vous lui attribuez "World" en utilisant la vérification relative, la valeur finale de la variable sera "Hello World". Il faut cependant faire attention, Il faut cependant veiller à ce que les valeurs de la variable soient toutes deux du même type de données, car si vous essayez d'ajouter, par exemple, un entier et une chaîne de caractères, une erreur de compilation se produira.

Notez que vous pouvez ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action, et en donnant un autre nom et une autre valeur à la variable. valeur. Pour des informations plus avancées sur les variables, veuillez consulter la section sur les variables et leur portée.
