Nouvelle structure

Cette action est uniquement utilisée lors de la création de structs en utilisant une fonction qui a été marquée comme une fonction constructeur (voir Déclarer une nouvelle fonction pour plus d'informations). Cette action prend la méthode de fonction pour une fonction précédemment définie, ainsi qu'un certain nombre d'arguments, que vous pouvez développer pour en ajouter d'autres si nécessaire en utilisant l'icône sur la gauche. Ces arguments doivent correspondre aux entrées requises par la fonction et seront utilisés pour remplir le site struct qui est créé. L'adresse struct sera renvoyée à la variable cible, qui peut être marquée comme une variable locale temporaire ou non.

Syntaxe de l'action :

Arguments :

Arguments Description
Function Le nom du script ou de la fonction définie par l'utilisateur à appeler.
Argument0 ... Argument3 Les différents arguments (valeurs) qui doivent être transmis au script ou à la fonction (les arguments non utilisés peuvent être laissés vides).
Target Le nom de la variable qui doit être ciblée pour toutes les valeurs retournées (peut être laissé vide).
