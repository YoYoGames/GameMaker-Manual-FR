Régler le volume principal

Cette action vous permet de régler le volume de l'ensemble du système sonore. La valeur du volume que vous donnez peut être comprise entre 0 (silencieux) et 1 (plein volume) et l'échelle est linéaire, de sorte qu'une valeur de 0,5 correspondrait à la moitié du volume. Notez que cette action affectera toutes les instances de tous les sons qui sont joués actuellement dans la pièce, et affectera toutes les instances futures de tous les sons joués. Cette action est également indépendante du volume défini pour chaque son individuel, donc si vous avez un son défini pour être joué à un volume de 0.5 et que vous réglez le volume principal sur 0,5, le son sera joué à un volume de 0,25.

Syntaxe de l'action :

Arguments :

Arguments Description
Volume Le nouveau volume maître pour tous les sons (de 0 à 1, par défaut 1)

Le code du bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si elle est true, le volume principal du système de sonorisation est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si elle ne l'est pas, alors le volume principal du système de sonorisation est réglé sur 0,5. Si la variable globale est égale à false, le volume principal est réglé à 1.
