Obtenir la longueur de l'audio

Cette action peut être utilisée pour récupérer la longueur d'un son dans votre jeu. Vous fournissez la ressource sonore pour obtenir la longueur et une variable cible pour retourner la valeur (ou vous pouvez marquer la cible comme une variable temporaire pour que l'action crée la variable qui sera utilisée jusqu'à la fin de l'événement ou du script). La durée sera renvoyée à la variable cible (qui peut être temporaire) en secondes et peut être une valeur décimale.
