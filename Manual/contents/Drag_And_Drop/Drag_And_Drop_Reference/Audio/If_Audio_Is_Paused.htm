Si l'audio est en pause

Cette action peut être utilisée pour vérifier si une instance d'une ressource sonore quelconque est actuellement en pause dans votre jeu. Vous donnez la ressource sonore à vérifier à partir de l'explorateur d'actifs et la fonction retournera true if it is paused and false si ce n'est pas le cas, bien que si vous vérifiez le drapeau "not", ceci est inversé et la fonction retournera true si le son n' est pas en pause et false sinon.

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous :

Ces actions seront maintenant exécutées si le bloc "if" donne la valeur true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if".
