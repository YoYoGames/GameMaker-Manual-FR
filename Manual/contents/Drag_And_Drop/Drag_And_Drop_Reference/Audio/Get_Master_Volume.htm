Obtenir le Master Volume

Cette action renvoie le volume principal du système de sonorisation. Vous donnez une variable cible à laquelle renvoyer la valeur du volume (vous pouvez la marquer comme étant une variable locale temporaire). La valeur retournée sera comprise entre 0 et 1, où 0 est silencieux et 1 est le plein volume. volume.

Syntaxe de l'action :

Arguments :

Arguments Description
Target La variable cible pour stocker le volume retourné

Exemple :

Le code du bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si elle est true, le volume principal du système de sonorisation est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si elle ne l'est pas, alors le volume principal du système de sonorisation est réglé sur 0,5. Si la variable globale est égale à false, le volume principal est réglé à 1.
