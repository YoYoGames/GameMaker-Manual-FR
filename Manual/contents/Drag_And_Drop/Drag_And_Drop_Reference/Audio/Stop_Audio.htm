Stop Audio

Cette action peut être utilisée pour arrêter la lecture d'un son. Vous sélectionnez le son dans l'explorateur d'actifs et toutes les instances de ce son s'arrêteront de jouer (ainsi, si vous avez défini le même son à jouer dans trois événements différents, ils s'arrêteront tous).
