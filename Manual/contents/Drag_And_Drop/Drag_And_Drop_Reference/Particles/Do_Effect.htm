Effet Do

Cette action créera un effet de particule intégré, soit au-dessus, soit au-dessous des couches dessinées dans votre salle de jeu. Vous choisissez le type d'effet à créer dans la liste déroulante (illustrée dans l'image ci-dessous), puis vous définissez la position (qui peut être relative à la position de l'instance) et sélectionnez une taille - petite, moyenne ou grande - pour l'effet. par rapport à la position de l'instance) et sélectionnez une taille - petite, moyenne ou grande - pour l'effet. Vous pouvez également définir la couleur à utiliser - la valeur alpha sera ignorée - et décider si vous souhaitez que l'effet soit créé sous tous les calques de la pièce ou au-dessus de tous les calques. la pièce, ou au-dessus de toutes les couches de la pièce. Notez que si vous choisissez d'utiliser les effets de pluie ou de neige, les valeurs de position et de taille ne seront pas prises en compte car l'effet est conçu pour être visible sur tout l'écran.

Arguments :

Arguments Description
Type Le type d'effet de particule à créer (voir image ci-dessus)
Where Si les particules doivent être créées au-dessus ou au-dessous des couches de la pièce.
Size La taille de l'effet (petit, moyen ou grand)
X La position x dans la pièce pour créer l'effet (en pixels)
Y La position y dans la pièce pour créer l'effet (en pixels)
Colour La couleur de l'effet (l'alpha sera ignoré)
