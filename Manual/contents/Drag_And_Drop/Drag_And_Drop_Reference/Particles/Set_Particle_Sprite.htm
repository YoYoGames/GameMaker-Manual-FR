Définir le sprite de particules

Cette action peut être utilisée pour définir le sprite qu'une particule peut utiliser lors du dessin. Vous fournissez la valeur d'identification unique du type dont vous voulez définir l'adresse sprite (telle que renvoyée lorsque vous avez créé le type avec l'action Créer un type de particule) et la ressource sprite à utiliser. Si le site sprite comporte des sous-images, la particule s'animera à travers les images à la vitesse définie dans l'éditeur d'images pour le site sprite. Notez également que si vous définissez une valeur de couleur pour la particule (à l'aide de l'action Définir la couleur de la particule ), cette couleur sera mélangée avec le site sprite utilisé, de la même manière que la définition de la fusion d'image sur une instance mélangera une couleur avec l'instance sprite (utilisez le blanc pour ne pas mélanger).

NOTE: Si vous utilisez cette action, vous ne devez pas utiliser l'action Set Particle Shape car vous ne pouvez avoir qu'une sprite ou une forme, mais pas les deux.

Syntaxe de l'action :

Arguments :

Arguments Description
Type La valeur d'identification du type à définir à l'adresse sprite.
Sprite Le sprite à attribuer au type de particule
