Créer un type de particule

Dans vos projets, vous pouvez avoir plusieurs types de particules différentes, chacune avec ses propres propriétés comme la direction et la couleur, etc... qui sont définies en utilisant les différentes actions de type de particule. Cependant, vous devez d'abord créer un type de particule en utilisant cette action, qui alloue essentiellement une petite quantité de mémoire pour contenir tous les détails de la particule afin que vous puissiez utiliser les particules à tout moment dans le futur. Vous donnez une variable pour contenir la valeur d'identification unique pour le type de particule, puis dans toutes les actions futures relatives à cette particule. toutes les actions futures relatives à cette particule, vous pouvez appeler la variable. Vous pouvez choisir de marquer cette variable comme étant une variable locale temporaire, mais cela signifie qu'elle sera supprimée à la fin du script d'action ou de l'événement. existera toujours, mais vous laissera sans aucun moyen de l'envoyer à access. Il faut donc être prudent lorsque vous l'utilisez (en général, les types de particules n'auront jamais besoin d'être stockés dans une variable locale temporaire, mais pour certaines choses, comme dessiner des particules sur une surface, cela peut être nécessaire), cela peut être nécessaire).

NOTE: Un type de particule, une fois créé, reste résident dans la mémoire du système pour la durée du jeu, et donc si vous devez le recréer à un moment donné (par exemple, si vous appelez l'action Restart Game ), vous devez alors détruire le type de particule actuel, sinon vous obtenez une fuite de mémoire. Cela se produit lorsqu'une ressource est ajoutée à la mémoire puis "déréférencée", ce qui signifie que GameMaker n'a plus aucun moyen d'interagir avec cette ressource. plus. Cela conduit à l'encombrement de la mémoire par des ressources inaccessibles qui peuvent à leur tour causer des problèmes dans votre jeu. Pour détruire un type de particule lorsqu'il n'est plus nécessaire, utilisez l'action Détruire le type de particule.

Vous pouvez également définir le mode de fusion à utiliser pour les particules. Par défaut, elles seront dessinées avec un mode de mélange "normal", ce qui signifie que les couleurs ne seront pas modifiées par ce qui a été dessiné sous la particule, cependant, lorsque l'option "Blend" (mélange) est activée, elles seront dessinées en utilisant le mode de fusion additif, ce qui signifie que les particules seront dessinées en modifiant les couleurs en fonction de ce qui a été dessiné sous la particule. Le mélange additif permet d'obtenir des effets très brillants et "lumineux". idéal pour les explosions, les lasers, le feu, etc., tandis que le mélange normal est meilleur pour les effets de pluie, de fumée et de débris.
