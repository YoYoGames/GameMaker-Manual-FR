Émettre des particules

Cette action est utilisée pour créer des particules à partir d'un émetteur. Vous fournissez l'ID du système de particules auquel appartient l'émetteur (tel que retourné par l'action Create Particle System), puis l'ID de l'émetteur réel (tel que retourné par l'action par l'action Create Particle Emitter), et enfin le type de particule à créer (tel que retourné par l'action Create Particle Type). Vous pouvez ensuite choisir d'envoyer les particules en "Burst" ou en "Stream" à partir de l'émetteur. ou "Stream" les particules de l'émetteur, ainsi que donner le nombre d'entre eux que vous voulez créer.

Lorsque vous choisissez de diffuser des particules, vous n'avez pas besoin de placer cette action dans un événement de trame de jeu comme l'événement Step ou l'événement Draw, mais vous pouvez simplement l'appeler une fois dans l'événement Create (par exemple). jusqu'à ce que vous appeliez à nouveau cette action pour l'arrêter (en fixant la valeur du compte à 0). Si vous choisissez d'éclater les particules, vous pouvez appeler cette action dans n'importe quel événement et les particules seront créées une fois.

Syntaxe de l'action :

Arguments :

Arguments Description
System L'ID du système dans lequel se trouve l'émetteur.
Emitter L'ID de l'émetteur à utiliser
Type L'ID du type de particule à créer
Event Le type d'événement pour les particules, soit "Burst" ou "Stream".
Count Le nombre de particules à faire éclater ou à diffuser.
