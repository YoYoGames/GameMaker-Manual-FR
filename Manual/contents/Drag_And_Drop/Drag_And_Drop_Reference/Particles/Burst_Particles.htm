Particules d'éclatement

Cette action vous permet d'éclater des particules à tout moment sans dépendre d'un émetteur. Vous fournissez la valeur d'identification unique du système dans lequel créer les particules (telle que retournée lorsque vous avez créé le système avec l'action Create Particle System) et l'ID unique du type de particule à créer (comme indiqué lorsque vous avez créé le type de particule avec l'action Créer un type de particule), ainsi que la position x/y à laquelle les créer et le nombre à créer. La position peut être une position absolue dans la pièce, ou vous pouvez la marquer comme étant une position relative à l'instance qui appelle l'action.

Vous pouvez appeler cette action à chaque image du jeu pour reproduire l'effet de flux d'un émetteur, ou vous pouvez l'appeler une fois dans un événement (comme l'événement Destroy) pour simplement faire éclater les particules.
