Définir la forme des particules

Cette action peut être utilisée pour définir la forme intégrée de sprite qu'une particule doit utiliser lors du dessin. Vous fournissez la valeur d'identification unique du type dont vous voulez définir la forme (telle qu'elle est retournée lorsque vous avez créé le type avec l'action Créer un type de particule) et la forme à utiliser. Les formes de particules sont simplement des sprites blancs qui ont été créés pour GameMaker afin d'être utilisés spécifiquement comme des particules et consistent en les formes suivantes :

NOTE: Si vous utilisez cette action, vous ne devez pas utiliser l'action Set Particle Sprite car vous ne pouvez avoir qu'un sprite ou une forme, mais pas les deux.

Type La valeur d'identification du type dont il faut définir la forme.
