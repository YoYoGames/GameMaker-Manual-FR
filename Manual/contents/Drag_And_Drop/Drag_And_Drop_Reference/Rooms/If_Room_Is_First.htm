Si la chambre est la première

Cette action vérifiera si la pièce actuelle est la première pièce listée dans le Room Manager. L'action retournera true si la pièce actuelle est la première dans le Room Manager et false si elle ne l'est pas. Vous pouvez cocher la case " pas" pour transformer cette action en "Si la pièce n'est PAS la première".

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous :
