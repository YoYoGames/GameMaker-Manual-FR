Sauvegarder le jeu

Cette action effectuera une sauvegarde de base du jeu dans un fichier donné. Vous donnez un nom de fichier (sous forme de chaîne et incluant l'extension) et l'action effectuera une sauvegarde limitée de l'état actuel du jeu. Il est important de comprendre que cette action est conçue pour prendre un "instantané" de l'état actuel du jeu et est plus utilisée comme un système de point de contrôle que comme un système de sauvegarde universel, puisqu'elle ne sauvegarde aucune des ressources dynamiques (comme les structures de données ou les particules). En tant que telle, cette action doit être utilisée avec précaution afin d'éviter les erreurs ou les fuites de mémoire. Pour une approche plus complète de la sauvegarde de vos données de jeu, consultez les actions de fichier.
