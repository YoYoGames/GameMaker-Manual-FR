BPM à Secondes

Cette action prend une valeur de battements par minute, et renvoie la durée de chaque battement en secondes.

Ceci peut être utilisé lors de la création d'une source de temps pour utiliser une valeur BPM pour la source de temps. Il est important qu'une telle source de temps utilise les secondes comme unité.

Syntaxe de l'action :

Arguments :

Arguments Description
Value La valeur des battements par minute
Target La variable pour stocker la valeur retournée des secondes par battement.
Temp Si la variable doit être créée comme une variable temporaire
