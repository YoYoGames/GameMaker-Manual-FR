Si les unités [Source de temps]

Cette action vous permet de contrôler d'autres actions basées sur le type d'unité de la source de temps donnée.

Elle vérifie si l'unité Constante d'unité de la source de temps indiquée dans "Unités" est (ou n'est pas) le type d'unité de la source de temps indiquée dans "Source de temps".

Syntaxe de l'action :

Arguments :

Arguments Description
Time Source Le site Identification de la source de temps pour vérifier l'état de
Not Activez cette option pour inverser la condition
Units Le site Constante d'unité de la source de temps à vérifier par rapport à

Retour : Actions de la source de temps
Suivant : Du BPM à la seconde
