Destroy Time Source

Cette action détruit la source de temps donnée.

Vous ne pouvez pas détruire une source de temps si elle a des enfants existants. Détruisez d'abord les sources de tempsenfants avant de détruire un parent(exemple).

Time Source Le site Identification de la source de temps de détruire
