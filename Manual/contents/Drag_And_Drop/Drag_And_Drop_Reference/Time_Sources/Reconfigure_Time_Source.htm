Reconfigurer la source de temps

Cette action vous permet de modifier les paramètres d'une source de temps existante, spécifiée dans le champ "Source de temps".

Les paramètres fournis ici sont les mêmes que ceux de Créer une source de temps, sauf pour le parent.

Cela réinitialisera et désactivera la source de temps et vous devrez la redémarrer.

Syntaxe de l'action :

Arguments :

Arguments Description
Time Source Le site Identification de la source de temps pour reconfigurer
Period La durée de la période de la source de temps, c'est-à-dire le temps qu'il faut pour qu'elle expire.
Units Les unités dans lesquelles la période est exprimée (secondes ou images).
Callback Le site method à appeler lorsque la source de temps expire
Arguments OPTIONNEL Un tableau contenant les arguments à passer dans la fonction de rappel.
Repetitions OPTIONNEL Combien de fois la source de temps doit être exécutée au total, ou -1 pour une répétition indéfinie.
Expiry Type OPTIONNEL Le type d'expiration de la source de temps.
