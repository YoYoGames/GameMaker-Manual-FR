Source du temps de pause

Cette action met en pause la source de temps donnée.

Vous pouvez le reprendre plus tard en utilisant Start Time Source.

Syntaxe de l'action :

Arguments :

Arguments Description
Time Source Le site Identification de la source de temps pour faire une pause

Exemple :


Ce bloc de code s'exécuterait idéalement dans un événement de type "Key Press".

Il vérifie si l'état d'une source de temps est "actif", et dans ce cas, il met en pause cette source de temps.

Toutefois, si l'état est "en pause", il reprend la source de temps.
