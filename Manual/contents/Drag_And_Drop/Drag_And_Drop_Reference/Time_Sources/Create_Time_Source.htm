Créer une source de temps

Cette action crée une nouvelle source de temps, et la stocke dans la variable cible. Lisez Sources de temps pour une vue d'ensemble.

Ceci est basé sur la time_source_create() . Lisez sa page pour des informations détaillées sur les arguments.

Vous devez utiliser Start Time Source pour activer une source de temps après sa création.

Vous devez détruire une source de temps à l'aide de la fonction Détruire la source de temps lorsque vous n'en avez plus besoin.

Syntaxe de l'action :

Arguments :

Arguments Description
Target La variable où la nouvelle source de temps sera stockée.
Parent La source de temps parente : soit une source de temps intégrée, soit une source de temps personnalisée.
Period La durée de la période de la source de temps, c'est-à-dire le temps qu'il faut pour qu'elle expire.
Units Les unités dans lesquelles la période est exprimée (secondes ou images).
Callback Le site method ou fonction de script à appeler lorsque la source de temps expire
Arguments OPTIONNEL Un tableau contenant les arguments à passer dans la fonction de rappel.
Repetitions OPTIONNEL Combien de fois la source de temps doit être exécutée au total, ou -1 pour une répétition indéfinie.
Expiry Type OPTIONNEL Le type d'expiration de la source de temps.

Exemple :

Ce bloc de code fait ce qui suit :

  • Elle crée une nouvelle fonction, avec l'option "Temp" activée. Elle crée une variable locale de méthode.
    • Cette méthode prend un argument, msg, qu'elle imprime dans le journal de sortie.
  • Elle crée ensuite une nouvelle source de temps avec une période de 1 seconde.
    • Il spécifie la variable callback_method comme "Callback", et un tableau avec un argument pour la fonction.
    • Cette source de temps est configurée pour se répéter indéfiniment, puisque -1 est spécifié dans "Repetitions".
  • La source de temps est alors lancée.

Cette source de temps exécutera son rappel toutes les secondes, en imprimant "Une seconde s'est écoulée !" dans le journal de sortie.
