Obtenir la variable de vue

Cette action vous permet d'obtenir la valeur de certaines variables intégrées liées à un port de vue donné. Vous sélectionnez la variable du port de vue à récupérer, puis vous donnez le port de vue à cibler (de 0 à 7) et enfin la variable cible qui doit contenir la valeur retournée (celle-ci peut être marquée comme étant une variable locale temporaire). (celle-ci peut être signalée comme étant une variable locale temporaire). Le port d'observation est la zone de l'écran qui sera utilisée pour dessiner une vue de caméra donnée (telle que définie dans l'éditeur de salle), et les différentes variables que vous êtes en mesure d'obtenir de cette façon sont décrites dans le tableau ci-dessous :

Variable Description
Camera La valeur renvoyée sera la valeur d'identification unique de la caméra à affecter au port (voir ici pour plus d'informations).
Visibility Cela renverra une valeur de true ou false où true signifie que le port de vue est visible (c'est-à-dire qu'il est dessiné à l'écran), et false signifie que rien n'est dessiné.
View Port X Coordinate Il s'agit de la position X du port de vue dans la fenêtre du jeu.
View Port Y Coordinate Il s'agit de la position Y du port de vue dans la fenêtre du jeu.
View Port Width Il s'agit de la largeur (en pixels) du port d'affichage.
View Port Height Il s'agit de la hauteur (en pixels) du port d'affichage.
View Port Surface ID Cela renvoie la valeur d'identification unique de la surface attribuée au port de vue ou -1 si aucune surface n'a été attribuée.
