Utiliser les préférences du lecteur (Rollback)

Lorsque vous jouez une partie en utilisant Sync Test, tous les joueurs "distants" reçoivent des valeurs aléatoires pour leurs entrées définies, comme un moyen de base de test. Vous pouvez désactiver ou réactiver ce comportement en utilisant cette action.

Syntaxe de l'action :

Arguments :

Arguments Description
Enabled Activez ou désactivez la saisie aléatoire pour les lecteurs distants pendant les tests.
