Définir le joueur (Rollback)

Cette action indique au système Rollback l'objet qui doit être utilisé pour les joueurs. Elle est facultative, et lorsqu'elle est utilisée, le système créera automatiquement des instances pour les joueurs qui se connectent, et détruira les instances pour les joueurs qui se déconnectent.

Vous pouvez spécifier la couche où les instances de joueurs seront créées. Si aucune couche n'est spécifiée, les joueurs seront créés sur la couche "Instances", qui doit exister dans la pièce.

Pour plus d'informations sur la création d'un jeu avec ou sans cette fonction, voir Définir un objet joueur.

Si elle est utilisée, cette fonction doit être exécutée avant l'action de jonction/départ.

Chaque instance créée avec cette fonction reçoit les variables d'instance listées sur cette page.

Syntaxe de l'action :

Arguments :

Arguments Description
Object L'objet à utiliser pour les instances de joueurs
Layer OPTIONNEL Le nom de la couche où les instances de lecteur seront créées. "Instances" par défaut (quand aucune valeur n'est spécifiée).
