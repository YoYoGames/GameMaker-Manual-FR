Si touche vers le haut (retour en arrière)

Cette action vérifie si l'entrée donnée vient d'être libérée pour le joueur actuel.

Il vous permet d'attacher des actions sur la droite, qui ne s'exécutent que lorsque l'entrée donnée est trouvée.

Syntaxe de l'action :

Arguments :

Arguments Description
Input Le nom de l'entrée à vérifier
Not Activez cette option pour vérifier si l'entrée n'a pas été libérée.

Retour : Actions multijoueurs Rollback
Suivant : Si l'événement Rollback a eu lieu
