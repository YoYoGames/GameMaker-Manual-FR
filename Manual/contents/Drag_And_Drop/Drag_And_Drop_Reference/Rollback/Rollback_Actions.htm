Actions multi-joueurs de Rollback

Les actions de retour en arrière sont utilisées pour créer des jeux multi-joueurs en utilisant le système de retour en arrière de GameMaker.

Pour une introduction, voir : Rollback Multiplayer

Pour une liste des variables utilisées avec le système Rollback, voir : Variables Rollback

Les actions suivantes sont prévues pour le multijoueur Rollback :

  • Créer un jeu
  • Rejoindre le jeu
  • Début du jeu
  • Leave Game
  • Définir l'entrée
  • Définir le délai de la trame d'entrée
  • Définir l'entrée fictive
  • Définir le joueur
  • Événements d'affichage
  • Obtenir des informations
  • Obtenir des informations
  • Si touche vers le bas (retour en arrière)
  • Si la touche est pressée (Rollback)
  • Si touche vers le haut (retour en arrière)
  • Si l'événement Rollback a eu lieu
  • Sync On Frame
  • Utiliser une entrée aléatoire
  • Utiliser le démarrage manuel
  • Envoyer un message
