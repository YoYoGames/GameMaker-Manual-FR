Obtenir l'entrée (Rollback)

Cette action renvoie un struct contenant les valeurs d'entrée pour un joueur. Elle prend un argument facultatif spécifiant l'ID du joueur pour lequel les entrées seront retournées.

Il est recommandé d'utiliser If Key Down (Rollback), If Key Pressed (Rollback) et If Key Up (Rollback) pour vérifier les entrées en mode multijoueur.

Lorsqu'il est utilisé dans une instance de joueur qui a été créée par Define Player (Rollback), il n'est pas nécessaire de spécifier un ID de joueur, vous pouvez donc simplement spécifier player_id.

Lorsqu'il est utilisé dans d'autres instances, ou dans des instances de joueur créées manuellement, vous devez spécifier un ID de joueur.

Pour plus d'informations sur les entrées qui sont renvoyées et sur la façon dont elles peuvent être modifiées, voir Définition des entrées.

Arguments :

Arguments Description
Player ID L'ID du joueur pour lequel il faut obtenir des données
Target La variable dans laquelle le site struct doit être stocké.
