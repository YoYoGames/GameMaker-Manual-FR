Envoyer le message (Rollback)

Cette action est utilisée pour envoyer un message de type chaîne aux autres joueurs.

Vous spécifiez une chaîne de caractères à envoyer, dont la taille maximale est de 128 octets.

Lorsque le deuxième argument n'est pas spécifié, le message est envoyé à tous les joueurs, y compris l'expéditeur.

Vous pouvez spécifier un ID de joueur dans le second argument qui est le joueur qui recevra le message.

Comment l'utiliser ?

Cette action est destinée à être utilisée "localement", et résulterait donc d'une entrée locale, et non d'une entrée de type Rollback.

Voir l'exemple d'utilisation en bas de page. Écoutez les messages reçus dans l'événement Rollback Event.

Syntaxe de l'action :

Arguments :

Arguments Description
Message Le message à envoyer, sous forme de chaîne.
To OPTIONNEL L'ID du joueur auquel le message sera envoyé. Ne pas spécifier cet argument, ou spécifier -1, envoie le message à tous les joueurs.

Exemple :

Si la touche entrée est pressée localement, la keyboard_string est envoyée à tous les joueurs. Cette chaîne de caractères est ensuite effacée afin qu'un nouveau message puisse être tapé.

Cela déclenchera l'événement Rollback Event lorsque le message sera reçu par tous les clients (y compris l'expéditeur).
