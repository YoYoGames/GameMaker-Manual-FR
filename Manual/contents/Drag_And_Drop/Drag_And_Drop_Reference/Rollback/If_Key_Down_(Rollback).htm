Si touche vers le bas (retour en arrière)

Cette action vérifie si l'entrée donnée est maintenue pour le joueur actuel.

Il vous permet d'attacher des actions sur la droite, qui ne s'exécutent que lorsque l'entrée donnée est trouvée.

Syntaxe de l'action :

Arguments :

Arguments Description
Input Le nom de l'entrée à vérifier
Not Activez cette option pour vérifier si l'entrée n' est pas maintenue.

Exemple :

Les actions ci-dessus déplacent le lecteur vers la gauche ou la droite en fonction de l'entrée.
