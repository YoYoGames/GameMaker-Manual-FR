Définir l'entrée (Rollback)

Cette fonction est utilisée pour définir des entrées personnalisées pour tous les joueurs. Elle prend une paire Entrée/Valeur, où l'Entrée est le nom de l'entrée, et la Valeur est la constante d'entrée qui lui est assignée.

Pour savoir comment attribuer des entrées personnalisées, voir Commandes personnalisées.

Si elle est utilisée, cette fonction doit être exécutée avant l'action de joindre/créer.

Syntaxe de l'action :

Arguments :

Arguments Description
Input Le nom de l'entrée à définir
Value La constante d'entrée à affecter (voir Contrôles personnalisés)
