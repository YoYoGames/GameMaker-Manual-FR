Rejoignez le jeu Rollback

Cette action tente de rejoindre une partie. Vous devez être sur une URL qui contient l'ID de la salle virtuelle à rejoindre.

Si une salle a été jointe, toutes les actions liées à cette action seront exécutées.

Lorsqu'un nouveau joueur rejoint une partie dans laquelle vous êtes déjà présent, l'événement Rollback Event est déclenché. Voir Événements de retour en arrière pour plus d'informations.

Not En activant cette option, toutes les actions jointes seront exécutées si une pièce n'a pas été jointe. Vous pouvez l'utiliser pour créer une nouvelle salle si aucune n'a été trouvée.
