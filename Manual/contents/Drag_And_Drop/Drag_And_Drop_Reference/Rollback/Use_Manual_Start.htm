Utiliser le démarrage manuel (Rollback)

Par défaut, une partie multijoueur commence automatiquement lorsque tous les joueurs se sont inscrits.

L'appel de cette action désactive ce comportement de démarrage automatique, de sorte que le jeu attend que vous le démarriez manuellement.

Si elle est utilisée, cette action doit être exécutée avant l'action "joindre/créer".
