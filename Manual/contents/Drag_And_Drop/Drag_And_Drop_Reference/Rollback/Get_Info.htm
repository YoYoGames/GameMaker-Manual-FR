Obtenir des informations sur le retour en arrière

Cette fonction renvoie un struct contenant des informations sur un joueur. Cette fonction ne doit être utilisée que pour gérer manuellement les instances de joueurs. Si vous utilisez Define Player, chaque instance de joueur recevra automatiquement ces informations sous forme de variables.

Pour plus d'informations sur le contenu du fichier retourné struct, voir : rollback_get_info

Syntaxe de l'action :

Arguments :

Arguments Description
Player ID L'ID du joueur pour lequel il faut obtenir des informations
Target La variable dans laquelle le site struct doit être stocké.

Retour : Actions multijoueurs Rollback
