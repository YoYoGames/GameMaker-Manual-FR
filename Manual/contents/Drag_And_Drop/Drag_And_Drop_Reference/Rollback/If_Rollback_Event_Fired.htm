Si l'événement Rollback a eu lieu

Cette action est utilisée dans l'événement Rollback Event, pour vérifier quel type d'événement a été déclenché.

Il vous permet d'attacher des actions sur la droite, qui ne s'exécutent que lorsque l'événement donné a été déclenché.

Syntaxe de l'action :

Arguments :

Arguments Description
Event L'événement Rollback à vérifier
Not Activez cette fonction pour vérifier si l'événement donné n'a pas été déclenché (ce qui signifie que la condition sera vraie pour tous les autres événements déclenchés).
