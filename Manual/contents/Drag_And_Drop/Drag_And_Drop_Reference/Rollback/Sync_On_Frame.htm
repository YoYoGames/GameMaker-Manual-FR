Sync On Frame (Rollback)

Cette action vérifie si tous les joueurs sont synchronisés sur la trame actuelle, ce qui signifie que leurs états ne sont pas basés sur des prédictions mais sur les données réelles reçues de leur part. Dans ce cas, elle renvoie true.

Si les joueurs ne sont pas synchronisés sur la trame actuelle, il renvoie false, et tente de synchroniser tous les joueurs.

Tenter de synchroniser des lecteurs avec cette action peut provoquer un gel. Vous devez vous assurer que cette action n'est pas appelée en permanence dans un événement Step.

Utilisez-le plutôt comme une bascule. Voir l'exemple au bas de la page.

La variable rollback_confirmed_frame est mise à jour automatiquement lorsque tous les joueurs sont synchronisés.
