Début du jeu de retour en arrière

Cette action force le démarrage de la partie multijoueur en cours, avant que tous les joueurs ne l'aient rejointe. Vous devez d'abord créer une partie multi-joueurs à l'aide de l'action Créer une partie de retour.

Le système attend normalement que tous les joueurs se joignent au jeu avant de le lancer automatiquement. Cette action peut être utilisée pour commencer la partie plus tôt, avant que tous les joueurs ne se soient joints.

Vous pouvez appeler Use Manual Start pour que Rollback attende cet appel même après que tous les joueurs se soient joints.

L'exécution de cette action déclenchera l'événement Rollback Start.

Syntaxe de l'action :

Retour : Actions multijoueurs Rollback
Suivant : Le jeu du retour en arrière
