Définir la chronologie de l'instance

Cette action assigne une timeline de l'Asset Browser à l'instance. Vous donnez la ligne de temps à assigner, puis vous pouvez choisir de mettre la ligne de temps en boucle lorsqu'elle atteint le dernier moment (la prochaine image après le dernier moment sera de nouveau en boucle vers le premier moment de la ligne de temps). le premier moment de la timeline) ainsi que de démarrer la timeline immédiatement ou non. Si vous ne démarrez pas la timeline immédiatement, vous pouvez utiliser l'action Set Timeline State pour la démarrer plus tard dans un autre événement ou script. événement ou un script.

Arguments :

Arguments Description
Timeline La ressource chronologique à affecter à une instance
Loop Flag pour avoir la timeline en boucle après le dernier moment (par défaut, flagged)
Start Timeline Drapeau pour que la timeline démarre immédiatement ou non (par défaut, drapeau).
