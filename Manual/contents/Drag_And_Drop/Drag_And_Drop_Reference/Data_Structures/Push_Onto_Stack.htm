Pousser sur la pile

Avec cette action, vous pouvez pousser une valeur sur une structure de données en pile. Vous fournissez la variable qui contient l'index de la pile (tel que retourné par l'action Create Stack) et ensuite vous donnez la valeur à pousser au sommet. Avec les structures de données les valeurs sont toujours ajoutées au "sommet" de la pile, et jamais à la fin ou au milieu (si vous avez besoin de cette fonctionnalité, utilisez une structure de données de type liste).

Syntaxe de l'action :

Arguments :

Arguments Description
Stack L'index (stocké dans une variable) de la pile à utiliser
Value La valeur à pousser sur la pile

Exemple :

Le code du bloc d'action ci-dessus crée une variable d'instance et une nouvelle structure de données de pile. L'index de la pile est stocké dans la nouvelle variable, puis une boucle est exécutée pour créer 10 instances et placer leurs valeurs d'identification uniques sur la pile.
