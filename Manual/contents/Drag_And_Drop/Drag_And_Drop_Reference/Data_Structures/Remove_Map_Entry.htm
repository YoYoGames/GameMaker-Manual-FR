Supprimer l'entrée de la carte

Cette action vous permet de supprimer une clé (et sa valeur correspondante) d'une structure de données de carte. Vous fournissez la variable qui contient l'index de la carte (tel que retourné par l'action Create Map) et ensuite vous donnez une "clé" (qui est l'identifiant d'une valeur dans la carte). est l'identifiant d'une valeur dans la carte), et l'action supprimera cette clé de la carte. Notez que cette action ne renvoie rien, et si vous avez besoin de la valeur de la clé donnée, vous devez utiliser l'action Get Map Value avant de la supprimer.

Syntaxe de l'action :

Arguments :

Arguments Description
Map L'index (stocké dans une variable) de la carte dont il faut retirer la valeur.
Key La clé à utiliser (réel ou chaîne)
