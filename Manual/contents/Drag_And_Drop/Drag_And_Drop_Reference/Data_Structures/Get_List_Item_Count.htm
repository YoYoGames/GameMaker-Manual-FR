Obtenir le nombre d'éléments de la liste

Vous pouvez utiliser cette action pour savoir combien d'entrées il y a dans une liste. Vous fournissez la variable qui contient l'index de la liste (tel que retourné par l'action Créer une liste) et la variable cible pour stocker le nombre retourné d'éléments de la liste (qui peut être marqué comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement). éléments de la liste (qui peut être marquée comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement).

Syntaxe de l'action :

Arguments :

Arguments Description
List L'index (stocké dans une variable) pour obtenir le nombre d'éléments à partir duquel on peut obtenir le nombre d'éléments.
Target La variable cible pour stocker le nombre d'éléments retournés

Exemple :

Le code du bloc d'action ci-dessus obtient la taille de la liste référencée dans la variable et utilise ensuite une boucle for pour la parcourir et détruire les instances dont les ID sont stockés dans la liste. Une fois la boucle terminée, la liste est libérée de la mémoire.
