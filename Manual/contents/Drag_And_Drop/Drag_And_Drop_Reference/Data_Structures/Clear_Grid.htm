Grille claire

Avec cette action, vous pouvez effacer une structure de données de grille à une valeur unique (chaque cellule de la grille sera définie à la valeur donnée). Vous fournissez la variable qui contient l'index de la grille (tel que retourné par l'action Create Grid) et ensuite vous donnez la valeur à effacer de la grille.

Syntaxe de l'action :

Arguments :

Arguments Description
Grid L'index (stocké dans une variable) de la grille à effacer.
Value La valeur à laquelle la grille doit être effacée
