Insérer dans une liste

Avec cette action, vous pouvez récupérer la position de l'index d'un élément dans une liste donnée. Vous fournissez la variable qui stocke l'index de la liste (tel que retourné par l'action Créer une liste) et la valeur dont vous voulez trouver la position de l'index dans la liste. dans la liste, ainsi qu'une variable cible pour stocker la position de l'indice retourné pour l'élément (qui peut être marquée comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement). Lorsque vous insérez une valeur dans une liste, tous les index Ainsi, si vous avez une liste contenant 4 éléments (positions d'index 0 à 3) et que vous placez un nouvel élément à la position d'index 2, l'élément précédent 2 deviendra l'élément 3 et l'élément précédent 3 deviendra l'élément 4. Notez que si la valeur que vous recherchez n'existe pas dans la liste, il n'y aura pas de position d'index de liste à retourner, et la valeur de retour sera donc -1.
