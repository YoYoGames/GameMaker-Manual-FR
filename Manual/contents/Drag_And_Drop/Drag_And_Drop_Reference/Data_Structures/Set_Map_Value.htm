Définir la valeur de la carte

Cette action vous permet d'ajouter une paire clé/valeur dans la structure de données de la carte. Vous fournissez la variable qui contient l'index de la carte (tel que retourné par l'action Create Map), puis vous donnez une "clé" (qui est l'identifiant dans la carte pour une valeur), puis la "valeur" à associer à cette clé. Les deux clés peuvent être des valeurs entières (comme 0, 200, 36712, etc...) ou des chaînes de caractères (comme "Name", "Shield", etc...), mais la valeur peut être n'importe quel type de données valide. type de données valide.

Syntaxe de l'action :

Arguments :

Arguments Description
Map L'index (stocké dans une variable) de la carte à définir
Key La clé à définir (entier ou chaîne)
Value La valeur à stocker avec la clé donnée
