Get List Item At

Avec cette action, vous pouvez récupérer la position de l'index d'un élément dans une liste donnée. Vous fournissez la variable qui stocke l'index de la liste (tel que retourné par l'action Créer une liste) et la position de l'index dans la liste pour obtenir la valeur de l'élément. Vous fournissez la variable qui stocke l'index de la liste (tel que retourné par l'action Créer une liste) et la position de l'index dans la liste pour obtenir la valeur de l'élément, ainsi qu'une variable cible pour stocker la valeur de l'élément retourné (qui peut être marquée comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement).

REMARQUE: Si vous donnez une position qui est en dehors de la taille de la liste donnée (par exemple, la position 11 dans une liste de 10 valeurs), l'action retournera undefined. Vous pouvez vérifier cela en utilisant l'action If Undefined.

Syntaxe de l'action :

Arguments :

Arguments Description
List L'index (stocké dans une variable) de la liste à retirer de la liste
Index La position de l'index dans la liste pour obtenir l'élément à partir de la liste.
Target La variable cible pour stocker la valeur retournée
