Obtenir la valeur de la carte

Cette action vous permet de récupérer la valeur associée à une clé donnée dans une structure de données de carte. Vous fournissez la variable qui contient l'index de la carte (tel que retourné par l'action Create Map) et ensuite vous donnez une "clé" (qui est l'identifiant d'une valeur dans la carte). est l'identifiant d'une valeur dans la carte), et fournissez également une variable cible pour contenir la valeur retournée (qui peut être marquée comme une variable locale temporaire).

REMARQUE: Si vous donnez une clé qui n'existe pas dans la carte, l'action retournera undefined. Vous pouvez vérifier cela en utilisant l'action If Undefined.

Syntaxe de l'action :

Arguments :

Arguments Description
Map L'index (stocké dans une variable) de la carte dont il faut récupérer la valeur
Key La clé à obtenir (réel ou chaîne)
Target La variable cible qui contient la valeur de la carte retournée
