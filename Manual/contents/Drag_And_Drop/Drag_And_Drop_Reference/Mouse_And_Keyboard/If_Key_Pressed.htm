Si la touche est pressée

Cette action vérifie l'appui initial sur une touche du clavier. Elle renvoie true lors du seul pas de jeu (image) où la touche est initialement enfoncée et false à tous les autres moments. Si vous activez le modificateur not, cette action sera inversée et vérifiera si la touche n'a pas été enfoncée, en renvoyant false à l'étape du jeu (image) où elle a été enfoncée et true à tous les autres moments. Si vous avez besoin de vérifier si une touche est maintenue enfoncée et non pas seulement l'événement d'appui initial, utilisez l'action If Key Down.
