Masquer le clavier virtuel

Cette action peut être utilisée pour masquer le clavier virtuel sur le périphérique exécutant le jeu. L'appel de cette action générera un événement asynchrone du système, dans lequel la carte DS async_load  sera remplie avec les paires clé/valeur suivantes :

  • "event_type" - il s'agira de "l'état du clavier virtuel" lorsqu'il sera déclenché par des actions du clavier virtuel.
  • "screen_height" - la hauteur du clavier virtuel (en pixels). Elle sera de 0 si le clavier est invisible.
  • "keyboard_status" - l'état actuel du clavier, renvoyé sous la forme d'une des chaînes suivantes :
    • "caché"
    • "caché"
    • "visible"
    • "visible"
