Si le clavier virtuel est affiché

Cette action peut être utilisée pour vérifier si le clavier virtuel de l'OS est affiché ou non. L'action sera évaluée à true si le clavier est ouvert ou en cours d'ouverture, ou à false si le clavier virtuel est fermé ou en cours de fermeture. Si vous activez le modificateur not, cette action sera inversée et vous pourrez stocker la valeur de retour true / false dans une variable cible, qui peut être marquée comme temp (locale).

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous :

Ces actions seront désormais exécutées si le bloc "if" donne la valeur true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if".
