Si la clé est libérée

Cette action vérifie si une touche du clavier a été relâchée. Elle renvoie true lors de l'unique étape du jeu (image) où la touche est relâchée et false à tous les autres moments. Si vous activez le modificateur not, cette action sera inversée, et vérifiera si la touche n'a pas été relâchée, renvoyant false sur le pas de jeu (frame) où elle a été relâchée et true à tous les autres moments.

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous :

Ces actions seront maintenant exécutées si le bloc "if" donne la valeur true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if".

Syntaxe de l'action :

Arguments :

Arguments Description
Key Le nom de la clé à vérifier.
Not Négation de la vérification (vrai devient faux et vice versa)

Si la vérification est true sur l'appui initial sur la touche, un son est joué, puis, pendant que la touche est maintenue enfoncée, le mélange sprite passe au rouge. Si la touche n'est pas maintenue Si la touche n'est pas maintenue enfoncée, un contrôle est effectué au moment du relâchement de la touche pour remettre la couleur de mélange en blanc.
