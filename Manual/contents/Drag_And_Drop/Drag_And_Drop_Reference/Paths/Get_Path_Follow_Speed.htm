Obtenir la vitesse de suivi du chemin

Cette action récupère la vitesse, en pixels par image de jeu, à laquelle une instance est configurée pour suivre un chemin (notez que vous devez avoir appelé l'action Start Following Path pour assigner un chemin à l'instance pour que cette action renvoie une valeur autre que 0). retourne une valeur différente de 0). La valeur retournée sera stockée dans la variable cible que vous donnez, ou vous pouvez marquer la variable comme une variable locale temporaire pour la créer et lui assigner la valeur.

Syntaxe de l'action :

Arguments :

Arguments Description
Target La variable cible pour contenir la vitesse de suivi du chemin retourné
