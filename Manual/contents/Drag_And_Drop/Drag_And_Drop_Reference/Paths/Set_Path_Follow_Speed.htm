Régler la vitesse de suivi de la trajectoire

Cette action va définir la vitesse, en pixels par image de jeu, à laquelle une instance doit suivre un chemin (notez que vous devez avoir appelé l'action Start Following Path pour assigner un chemin à l'instance pour que cela ait un effet). pour que cela ait un effet). Vous pouvez utiliser des valeurs négatives pour signifier que l'instance doit suivre le chemin en sens inverse et il est important de noter que la vitesse que vous définissez ici peut être modifiée par le modificateur de vitesse défini dans l'éditeur de chemin pour chaque point du chemin, donc si vous Ainsi, si vous définissez une vitesse de chemin de 2 et que dans le chemin l'un des points a un modificateur de vitesse de 200%, la vitesse réelle des pixels par image de jeu à ce point sera de 4.

Arguments :

Arguments Description
Speed La vitesse (en pixels par image de jeu) à laquelle l'instance doit suivre le chemin.
