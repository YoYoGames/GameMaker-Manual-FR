Définir la couleur de l'instance

Ce bloc d'action définit la couleur de image_blend pour la "fusion" avec l'instance sprite. La valeur par défaut est le code hexadécimal $FFFFFFFF (qui représente la couleur blanche au format ARGB ) et ceci dessinera le sprite sans mélange et avec un alpha complet. Toute autre valeur (y compris les constantes de couleur internes comme c_red ou c_aqua) permet de mélanger la couleur spécifiée avec la couleur originale sprite et de définir la composante image_alpha pour l'instance, mais uniquement lorsque la case Utiliser l'alpha est cochée. Cela écrasera toute valeur alpha définie précédemment à l'aide de l'action Set Instance Alpha.

Veuillez noter que pour que les modifications apportées à cette action soient visibles, l'instance ne doit pas avoir d'événement de dessin (et GameMaker dessinera donc par défaut le site sprite) ou être dessinée à l'aide de l'action Draw Self. Il est important de noter également que vous devez essayer de limiter les mélanges sur la plate-forme HTML5 (à moins d'utiliser WebGL), car chaque mélange sprite doit être mis en cache séparément et le fait d'avoir beaucoup de mélanges sprites peut nuire aux performances.
