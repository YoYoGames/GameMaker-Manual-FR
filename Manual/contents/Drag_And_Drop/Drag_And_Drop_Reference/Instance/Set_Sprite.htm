Définir le Sprite

Ce bloc d'action définit le sprite_index de l'instance à un autre sprite du navigateur d'actifs. Vous sélectionnez le sprite à modifier, puis vous indiquez l'image (cadre d'animation) à afficher. Notez que si vous ne souhaitez pas que le site sprite s'anime après avoir après avoir défini le cadre, vous devrez utiliser l'action Définir la vitesse d'animation et la définir sur 0, ou si vous voulez que le sprite continue à s'animer, définissez la valeur pour utiliser la variable intégrée image_index à la place.

La modification de sprite changera également le masque de collision de l'instance, sauf si vous avez également spécifié un mask_index distinct dans l'éditeur d'objets. Veuillez noter que pour que les modifications de cette action soient visibles, l'instance doit soit ne pas avoir d'événement de dessin (et donc GameMaker dessinera par défaut le sprite), soit être dessinée à l'aide de l'action Draw Self.

Syntaxe de l'action :

Arguments :

Arguments Description
Sprite Le nouveau sprite à utiliser (-1 si vous voulez supprimer le sprite).
Frame Le cadre d'animation à afficher initialement (si aucun cadre supplémentaire n'est utilisé, utilisez 0).
