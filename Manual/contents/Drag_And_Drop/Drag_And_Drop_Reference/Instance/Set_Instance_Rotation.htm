Définir la rotation de l'instance

Ce bloc d'action définit le image_angle (rotation) de l'instance. L'angle est mesuré en degrés, la droite correspondant à 0°, le haut à 90°, la gauche à 180° et le bas à 270°. Définir cette variable à 0 pour réinitialiser l'instance, ce qui signifie qu'elle va l'instance, ce qui signifie qu'elle dessinera tout sprite qui lui est attribué tel qu'il a été défini dans l'éditeur sprite. Veuillez noter que pour que les modifications de cette action soient visibles, l'instance ne doit pas avoir d'événement de dessin (et donc GameMaker dessinera par défaut le sprite) ou être dessinée avec l'action Draw Self. en utilisant l'action Draw Self.

Angle Le nouvel angle à utiliser (de 0 à 360).
