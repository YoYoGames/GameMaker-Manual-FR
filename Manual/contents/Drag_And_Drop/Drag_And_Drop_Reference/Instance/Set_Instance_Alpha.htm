Définir l'instance Alpha

Ce bloc d'action définit la valeur de image_alpha pour l'instance. L'alpha de l'image est la valeur de "transparence" pour dessiner toute sprite assignée à l'instance et a une valeur comprise entre 0 et 1, où 0 serait totalement transparent et 1 serait complètement opaque. Notez que si le site sprite comporte des pixels transparents ou semi-transparents, leur transparence sera mise à l'échelle par le montant. Ainsi, si vous avez des pixels transparents à 50 % dans le site sprite et que vous définissez l'alpha de l'instance à 0,5, l'alpha final des pixels transparents sera de 25 %. final des pixels transparents sera de 25 %. Vous pouvez définir cette valeur comme une quantité relative.

Veuillez noter que pour que les modifications de cette variable soient visibles, l'instance ne doit pas avoir d'événement de dessin (et GameMaker dessinera donc par défaut le site sprite) ou être dessinée à l'aide de l'action Draw Self .

Syntaxe de l'action :

Arguments :

Arguments Description
alpha La valeur alpha à définir (de 0 à 1, la valeur par défaut est 1).
