Régler le frottement

Cette action est utilisée pour définir la "friction" de l'instance. La friction signifie simplement qu'un montant spécifique sera déduit du vecteur vitesse à chaque frame de jeu, où vous spécifiez le montant à déduire. En général, les valeurs les plus petites fonctionnent le mieux, comme 0.2, mais le montant réel dépendra de la configuration de votre projet. Si vous mettez le drapeau relatif, alors le montant donné sera ajouté à la valeur de friction actuelle au lieu de la fixer.

Syntaxe de l'action :

Arguments :

Arguments Description
friction La friction à appliquer (valeurs positives uniquement)
