Jump To Start

Cette action est utilisée pour déplacer une instance de sa position actuelle à la position où elle a été créée à l'origine. La position originale où une instance est créée est stockée dans deux variables d'instance appelées xstart et ystart. Ces variables peuvent être utilisées dans toute autre action et peuvent également être définies pour changer la position de départ enregistrée.
