Créer une séquence

Avec cette action, vous pouvez créer un élément de séquence de n'importe quelle séquence à partir du navigateur d'actifs. Vous donnez le nom de la séquence et une position x/y dans la pièce où elle doit être créée ainsi que le calque sur lequel elle doit être créée (ce calque doit être un calque d'actif). Vous pouvez cocher la case "relative" flags pour positionner la séquence par rapport à l'instance qui exécute les blocs de code d'action. Chaque fois que vous créez une séquence dans le jeu de cette manière, une valeur d'identification unique lui est attribuée, avec laquelle vous pouvez ensuite identifier la séquence dans d'autres actions. Cette action vous permet donc de définir une variable "cible" pour contenir cette valeur d'identification (comme indiqué dans l'exemple ci-dessous). Notez que la cible peut être marquée comme une variable temporaire (locale), qui sera créée pour l'action et pourra être utilisée dans toutes les actions suivantes pour l'événement dans lequel l'action se trouve.

Syntaxe de l'action :

Arguments :

Arguments Description
Sequence Le nom de la séquence pour créer un élément de celle-ci.
x La position x dans la pièce pour créer la séquence (peut être relative à l'instance appelante).
y La position y dans la pièce pour créer la séquence (peut être relative à l'instance appelante).
layer Le nom (une chaîne) de la couche d'actifs sur laquelle créer la séquence.
Target La variable à cibler pour la valeur de retour.
