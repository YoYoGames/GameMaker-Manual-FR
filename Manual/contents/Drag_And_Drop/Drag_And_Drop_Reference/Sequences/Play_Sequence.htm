Séquence de lecture

Cette action vous permet de lancer la lecture d'une séquence (précédemment mise en pause). Vous fournissez l'ID de l'élément de séquence (tel que renvoyé par l'action Create Sequence), et l'action lancera la lecture.

Syntaxe de l'action :

Arguments :

Arguments Description
Variable L'ID de l'élément de séquence à lire.
