Obtenir la longueur de la séquence

Avec cette action, vous pouvez récupérer la longueur (en images) de l'élément de séquence donné dans la pièce. Vous fournissez l'ID de l'élément de séquence (tel que renvoyé par l'action Create Sequence), et cette action renverra sa longueur dans la variable cible. sa longueur dans la variable cible. Il s'agira du nombre d'images pendant lesquelles la séquence s'exécute.

Notez que la variable cible peut être marquée comme temporaire (locale), qui sera créée pour l'action et pourra être utilisée dans toutes les actions suivantes pour l'événement dans lequel l'action se trouve.

Syntaxe de l'action :

Arguments :

Arguments Description
Variable L'ID de l'élément de la séquence dont on veut obtenir la longueur.
Target La variable cible pour contenir la longueur.

Exemple :

Le bloc d'action ci-dessus récupère la longueur de l'ID de l'élément de séquence stocké dans la variable "seq_id". Cette valeur est ensuite utilisée pour définir la position de la tête du même élément de séquence à 10 images avant sa fin.
