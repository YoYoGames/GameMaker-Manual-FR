Séquence de lecture

Cette action vous permet de mettre en pause une séquence en cours de lecture. Vous fournissez l'ID de l'élément de la séquence (tel que renvoyé par l'action Create Sequence), et l'action interrompra la lecture.

Syntaxe de l'action :

Arguments :

Arguments Description
Variable L'ID de l'élément de séquence à mettre en pause.
