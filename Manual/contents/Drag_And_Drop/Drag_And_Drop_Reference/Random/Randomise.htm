Randomisez

Cette action définit la "graine" du générateur de nombres aléatoires à une valeur aléatoire. Lorsque vous utilisez les fonctions de génération de nombres aléatoires dans GameMaker, la graine initiale est toujours la même, car cela facilite grandement le repérage des erreurs et le site debugging (les jeux compilés n'ont pas ce comportement). compilés n'ont pas ce comportement). Si vous souhaitez tester avec un vrai aléatoire, vous devez appeler cette action une fois au début du jeu.
