Set Score

Avec cette action, vous pouvez créer ou définir une variable d'instance pour le score. Vous fournissez la valeur pour définir la variable score ou pour l'ajouter à la variable score (si vous cochez la case relative, la valeur sera ajoutée à la valeur "score" actuelle). actuelle). Notez que, contrairement aux variables d'instance ordinaires, le nom de la variable "score" est stocké en interne. Pour récupérer la valeur du score, vous devez donc utiliser l'action Get Score.
