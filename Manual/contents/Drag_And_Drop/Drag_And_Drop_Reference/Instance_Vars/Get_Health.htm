Obtenir la santé

Vous pouvez utiliser cette action pour récupérer la valeur de la variable d'instance créée pour la "santé" de l'objet. Vous fournissez la variable cible pour stocker la valeur "santé" retournée (qui peut être marquée comme une variable locale temporaire), mais notez que si la variable n'a pas été définie précédemment (en utilisant Set Health) alors la variable sera créée pour vous et l'action retournera 0 (contrairement aux variables d'instance régulières qui donneront une erreur si vous essayez si vous essayez d'aller sur access sans les avoir définies au préalable).
