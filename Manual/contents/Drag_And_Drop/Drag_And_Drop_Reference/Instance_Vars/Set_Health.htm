Set Health

Avec cette action, vous pouvez créer ou définir une variable d'instance pour la santé. Vous fournissez la valeur pour définir la variable de santé ou pour l'ajouter à la variable de santé (si vous cochez la case relative, la valeur sera ajoutée à la valeur actuelle "santé"). valeur actuelle de "health"). Notez que si vous souhaitez utiliser l'action Draw Instance Healthbar, cette valeur doit être comprise entre 0 et 100, sinon elle peut être n'importe quelle valeur. Notez également que, contrairement aux le nom de la variable "health" est stocké en interne. Pour récupérer la valeur du score, vous devez donc utiliser l'action Get Health.

Arguments :

Arguments Description
Health Le montant à fixer pour la santé (ou à ajouter à la santé si la case relative est cochée).
