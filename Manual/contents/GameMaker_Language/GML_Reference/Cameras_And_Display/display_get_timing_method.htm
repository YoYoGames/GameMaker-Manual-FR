afficher_get_méthode de chronométrage

Cette fonction peut être utilisée pour récupérer la méthode de timing à utiliser pour le rendu de votre jeu. La méthode peut être l'une des constantes énumérées ci-dessous.

Pour plus d'informations sur les différentes méthodes de chronométrage, veuillez consulter le site suivant display_set_timing_method().

Syntaxe :

display_get_timing_method();

Les retours :

Display Timing Method Constant

Méthode de temporisation de l'affichage Constante
Constant Description
tm_sleep La valeur de la marge de sommeil est la principale méthode de chronométrage
tm_countvsyncs La synchronisation Vsync est la principale méthode de synchronisation (par défaut pour toutes les plateformes prises en charge).

Exemple :

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

Le code ci-dessus vérifiera la méthode de synchronisation et si elle n'est pas définie sur tm_sleep, elle sera définie sur cette méthode et la marge de sommeil sera définie sur 20.
