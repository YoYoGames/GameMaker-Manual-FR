display_reset

Cette fonction réinitialise les paramètres d'affichage à ceux qui ont été définis au démarrage du jeu, mais vous permet également de modifier le niveau actuel d'anticrénelage plein écran utilisé et d'utiliser ou non la synchronisation verticale. Les niveaux d'anticrénelage disponibles sont 0, 2, 4 et 8, la valeur par défaut au démarrage étant 0, et le paramètre par défaut de la synchronisation verticale est false (désactivé). L'activation de la synchronisation verticale peut donner une expérience de jeu plus fluide, mais elle nécessitera également plus de puissance de traitement et son impact doit donc être considéré avec attention avant l'utilisation, et il en va de même pour l'anticrénelage où plus le nombre est élevé, plus le traitement est nécessaire.

Comme tous les périphériques cibles ne sont pas les mêmes, certains peuvent ne pas supporter l'anticrénelage 8x ou 4x par exemple, et il y a donc une variable en lecture seule disponible pour obtenir les différents niveaux d'AA que le périphérique exécutant le jeu peut afficher :

display_aa

Cette variable renverra une valeur entière basée sur le réglage des bits pour les différents niveaux. Ainsi, pour seulement 2xAA, cela donnera 2, pour 2x et 4x disponibilité, cela donnera 6. Pour 8 et 4, la valeur sera de 12. Pour les 3 niveaux (2,4 et 8), la valeur sera de 14.

Syntaxe :

display_reset(aa, vsync);

Arguments Type Description
aa Real The level of anti-aliasing filtering (0, 2, 4 or 8).
vsync Booléen Toggle vertical synchronisation to on (true) or off (false).

Les retours :

Real

Exemple :

if (display_aa > 12)
{
    display_reset(8, true);
}

Le code ci-dessus définira le niveau d'anticrénelage à 8 s'il est pris en charge et activera la synchronisation verticale.
