view_xport

Avec ce tableau intégré, vous pouvez obtenir ou définir la position x du port de vue donné. Le port d'affichage est la zone de l'écran où la vue est dessinée, et vous pouvez en avoir jusqu'à 8 actifs à tout moment (le tableau a des valeurs de 0 à 7 inclus pour donner 8 ports). ports). Par défaut, dans GameMaker, la fenêtre de jeu (ou le canevas d'arrière-plan) a la même taille que la pièce. Cependant, lorsque vous activez des ports de vue et des caméras, ce comportement change et la taille totale de la boîte englobante de tous les ports de vue est utilisée. Ainsi, si vous avez deux ports de vue différents à deux positions différentes, la surface totale qu'ils couvrent définit la taille de la fenêtre de jeu. L'image suivante illustre cela :

Syntaxe :

view_xport[0 .. 7];

Les retours :

Exemple :

view_xport[0] = 0;
view_yport[0] = 0;

Le code ci-dessus réinitialise la position du port de vue [0] à la position (0,0) (coin supérieur gauche) de l'écran.
