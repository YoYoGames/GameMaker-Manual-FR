caméra_get_view_height

Cette fonction peut être utilisée pour récupérer la hauteur (en pixels) de la caméra donnée. Notez que cette fonction n' est valable que pour les caméras créées à l'aide de la fonction camera_create_view() ou pour celles ajoutées dans l'éditeur éditeur.

Syntaxe :

camera_get_view_height(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Exemple :

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

Le code ci-dessus récupère la largeur et la hauteur de la caméra affectée au port de vue [0], puis définit sa position par rapport au centre.
