view_get_hport

Cette fonction peut être utilisée pour récupérer la hauteur du port de vue donné.

Syntaxe :

view_get_hport(view_port)

ArgumentsType Description
view_port The view port to target (0 - 7)

Les retours :

Real

Exemple :

if view_get_hport(0) != (display_get_height () / 2)
{
    view_set_hport(0, display_get_height() / 2);
}

Le code ci-dessus vérifie la hauteur du port de vue [0] et si elle n'est pas égale à la moitié de la hauteur de l'écran, elle est fixée à cette valeur.
