camera_set_update_script

Cette fonction peut être utilisée pour définir une fonction de script qui sera appelée à chaque image de jeu où la caméra est affectée à un port de vue visible et actif. Vous donnez la valeur unique de l'ID de la caméra (telle que renvoyée par les différentes fonctions de camera_create ) et l'ID de la fonction à appeler. L'ordre dans lequel les fonctions attachées aux caméras et le rendu réel de la vue de la caméra sont les suivants :

  • Les caméras de tous les ports de vue visibles et actifs ont leur fonction de mise à jour appelée
  • Ensuite, pour chaque vue :
    • La fonction de démarrage de la caméra pour cette vue est appelée.
    • Les événements de dessin sont exécutés pour cette vue
    • La fonction de fin de la caméra est appelée
  • Passez à la vue active visible suivante et répétez...

L'application d'une fonction de mise à jour personnalisée à une caméra remplace son comportement de mise à jour par défaut. Par exemple, si vous avez configuré votre caméra pour qu'elle suive un objet (par exemple, obj_player) lorsque vous appelez camera_create_view(), ou si vous l'avez configuré ainsi dans les propriétés de la pièce, elle cessera de suivre cet objet dès qu'un script de mise à jour personnalisé sera défini.

Syntaxe :

camera_set_update_script(camera_id, script)

Arguments Type Description
camera_id ID de la caméra The unique camera ID value returned when you created the camera.
script Fonction de script The script function to run each game frame

Les retours :

N/A

Exemple :

camera_set_update_script(view_camera[0], updateCamera);

Le code ci-dessus définit la fonction de mise à jour pour la caméra affectée au port de vue [0].
