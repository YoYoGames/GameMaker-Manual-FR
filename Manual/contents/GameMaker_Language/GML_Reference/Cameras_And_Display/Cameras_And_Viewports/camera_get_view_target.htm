camera_get_view_target

Cette fonction peut être utilisée pour récupérer la cible de suivi de la caméra donnée. La valeur de retour peut être un index d'objet, un ID d'instance ou -1 si aucune cible de suivi n'a été définie.

Syntaxe :

camera_get_view_target(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Object Asset, Instance ID or -1

Exemple :

var target = camera_get_view_target(view_camera[0]);
if (target != obj_Player)
{
    camera_set_view_target(view_camera[0], obj_player);
}

Le code ci-dessus récupère la cible de suivi de la caméra affectée au port de vue [0], puis vérifie si elle correspond à l'index d'objet "obj_Player". Si ce n'est pas le cas, la caméra de la vue est configurée pour suivre une instance de cet objet.
