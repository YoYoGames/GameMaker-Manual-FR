view_set_surface_id

Avec cette variable, vous pouvez définir le contenu d'un port de vue à dessiner sur une surface. Lorsque l'on travaille avec des surfaces, il est souvent nécessaire de capturer toute la région visible de l'écran sur la surface, et donc vous l'assignerez à un port de vue en utilisant cette fonction. Cela signifie que tout ce qui est affiché dans le port de vue choisi sera désormais dessiné sur la surface assignée et que le contenu de ce port de vue ne sera plus affiché :

  • Activez une nouvelle vue et dessinez la surface uniquement dans cette vue (en utilisant la commande view_current pour vérifier quelle vue est dessinée)
  • Dessinez la surface dans l'événement Draw GUI ou Post Draw d'une instance, puisque ces événements sont indépendants des vues.

Lorsque vous utilisez cette fonction, vous donnez l'indice du port de vue (de 0 à 7) et un indice de surface (soit l'indice application_surface ou la valeur d'index unique renvoyée par la fonction surface_create()) ou, si une surface a déjà été attribuée et que vous souhaitez la supprimer, une valeur de -1. Pour plus d'exemples sur l'attribution du port de vue à une surface, voir la variable view_surface_id.

Syntaxe :

view_set_surface_id(view_port, surf)

Arguments Type Description
view_port Real The view port to target (0 - 7)
surf ID de la surface The surface to apply to the view

Les retours :

Real

Exemple :

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

Le code ci-dessus vérifiera si une surface a été assignée au port de vue [0] et si ce n'est pas le cas, une surface sera assignée.
