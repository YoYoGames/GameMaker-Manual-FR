camera_set_proj_mat

Cette fonction va définir la matrice de projection pour une caméra donnée. Vous donnez la valeur unique de l'ID de la caméra (telle que retournée par les différentes fonctions) et la matrice de projection à appliquer. camera_create() ) et une matrice de projection à appliquer. Pour en savoir plus sur la création de matrices de projection, consultez la section Fonctions de matrice, en particulier matrix_build_projection_perspective() et matrix_build_projection_ortho().

Il y a certaines choses que vous devez garder à l'esprit lorsque vous utilisez cette fonction :

  • Si votre caméra effectue un suivi automatique des objets - c'est-à-dire qu'elle a été créée à l'aide de camera_create_view() avec un indice d'objet / ID d'instance qui n'est pas -1, ou si vous définissez une caméra dans l'éditeur de salle, ou si vous définissez la caméra par défaut - alors cette matrice sera écrasée à la prochaine image de jeu.
  • Si vous utilisez cette fonction pour passer à une projection en perspective (par opposition à la projection orthographique par défaut), le dessin automatique des tuiles sprites cessera de fonctionner et devra être géré manuellement. Cela inclut toutes les couches d'arrière-plan qui utilisent le tuilage horizontal ou vertical et les fonctions draw_sprite_tiled() et draw_sprite_tiled_ext(), qui ne dessineront le sprite qu'une seule fois au lieu de le tuiler si une projection en perspective est utilisée.

Syntaxe :

camera_set_proj_mat(camera_id, matrix)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Les retours :

Exemple :

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Le code ci-dessus crée une nouvelle caméra et l'affecte au port de vue [0]. Cette caméra a ensuite ses matrices de vue et de projection définies.
