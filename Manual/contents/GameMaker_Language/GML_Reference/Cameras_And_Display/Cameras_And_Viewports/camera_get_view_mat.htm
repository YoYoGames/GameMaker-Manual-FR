camera_get_view_mat

Cette fonction peut être utilisée pour récupérer la matrice de vue. La fonction renvoie le tableau matriciel qui peut ensuite être utilisé dans d'autres fonctions matricielles ou pour définir la matrice de vue d'une autre caméra (en utilisant la fonction camera_set_view_mat()). Notez que cette fonction est généralement destinée aux caméras créées à l'aide de la fonction camera_create()mais elle peut également être utilisée avec celles créées dans l'éditeur de salle (ou avec la fonction camera_create_view()) - dans ce cas, la matrice renvoyée ne sera valide que lorsque la caméra donnée sera utilisée dans un port de vue pour la première fois.

Syntaxe :

camera_get_view_mat(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Matrix Array

Exemple :

mat = camera_get_view_mat(camera_view[0]);

Le code ci-dessus obtient la matrice de vue pour la caméra affectée au port de vue [0].
