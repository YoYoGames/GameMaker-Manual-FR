vue_get_visible

Cette fonction peut être utilisée pour vérifier la visibilité du port de vue donné. La fonction renverra true s'il est visible et false s'il ne l'est pas.

Syntaxe :

view_get_visible(view_port)

ArgumentsType Description
view_port The view port to target (0 - 7)

Les retours :

Boolean

Exemple :

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

Le code ci-dessus vérifiera si le port de vue [0] est visible et s'il ne l'est pas, il sera défini comme visible.
