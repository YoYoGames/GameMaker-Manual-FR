vue_visible

Cette variable peut être utilisée pour savoir si un port de vue particulier est actuellement visible ou non. Vous pouvez également définir cette variable pour activer ou désactiver efficacement une vue en définissant la valeur à true (visible) ou false (invisible). Notez que même si un port de vue est défini comme visible, si les ports de vue ne sont pas activés (à l'aide de la variable intégrée view_enabled ou en les activant dans l'éditeur de salle), ils ne seront pas affichés à l'écran.

Syntaxe :

view_visible[0 ... 7];

Les retours :

(true: enabled, false: disabled)

Exemple :

if !view_visible[0]
{
    view_visible[0] = true;
}

Le code ci-dessus vérifie si view[0] est visible ou non et s'il ne l'est pas, il le rend visible.
