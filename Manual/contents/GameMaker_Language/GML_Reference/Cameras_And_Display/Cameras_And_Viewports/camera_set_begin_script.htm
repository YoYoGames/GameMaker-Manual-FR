camera_set_begin_script

Cette fonction peut être utilisée pour définir une fonction de script qui sera appelée au début de chaque image de jeu où la caméra est affectée à un port de vue visible et actif. Vous indiquez la valeur unique de l'ID de la caméra (telle que renvoyée par les différentes fonctions de camera_create() ) et l'ID de la fonction à appeler. L'ordre dans lequel les scripts sont attachés aux caméras et le rendu réel de la vue de la caméra sont les suivants :

  • Les caméras de tous les ports de vue visibles et actifs ont leur fonction de mise à jour appelée
  • Ensuite, pour chaque vue :
    • La fonction de démarrage de la caméra pour cette vue est appelée.
    • Les événements de dessin sont exécutés pour cette vue
    • La fonction de fin de la caméra est appelée
  • Passez à la vue active visible suivante et répétez...

Syntaxe :

camera_set_begin_script(camera_id, script)

Arguments Type Description
camera_id ID de la caméra The unique camera ID value returned when you created the camera.
script Fonction de script The script function to run each game frame

Les retours :

N/A

Exemple :

camera_set_begin_script(view_camera[0], beginCamera);

Le code ci-dessus définit la fonction de script de début pour la caméra affectée au port de vue [0].
