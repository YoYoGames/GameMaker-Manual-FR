camera_set_view_speed

Vous pouvez utiliser cette fonction pour mettre à jour la vitesse de la caméra de surveillance dans la pièce. Vous donnez la valeur unique de l'ID de la caméra (telle que retournée par les différentes fonctions de la fonction camera_create() fonctions), puis vous indiquez la vitesse x et y (horizontale et verticale) à laquelle la caméra doit se déplacer lorsqu'elle est réglée pour suivre une instance donnée. et verticale) à laquelle elle doit se déplacer lorsqu'elle est configurée pour suivre une instance donnée. La vitesse est calculée en pixels par pas et peut être définie à "-1" pour que la caméra se déplace instantanément, mais si la caméra n'est pas définie pour suivre une instance, les valeurs définies ici n'auront aucun effet visible. valeurs définies ici n'auront aucun effet visible.

Syntaxe :

camera_set_view_speed(camera_id, xspeed, yspeed)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera.
xspeed The speed (number of pixels per game frame) that the view should move on the horizontal (x) axis
yspeed The speed (number of pixels per game frame) that the view should move on the vertical (y) axis

Les retours :

Exemple :

camera_set_view_speed(view_camera[0], 5, 5);

Le code ci-dessus définira la vitesse de la caméra de vue pour la caméra affectée au port de vue [0].
