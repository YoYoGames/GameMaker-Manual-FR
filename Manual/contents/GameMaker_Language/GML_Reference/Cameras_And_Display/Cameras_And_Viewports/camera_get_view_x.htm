camera_get_view_x

Cette fonction peut être utilisée pour récupérer la position x de la vue pour une caméra donnée. Notez que cette fonction n' est valable que pour les caméras créées à l'aide de la fonction camera_create_view() ou pour celles ajoutées dans l'éditeur de pièces éditeur de salle.

Syntaxe :

camera_get_view_x(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Exemple :

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

Le code ci-dessus récupère la position de la caméra affectée au port de vue [0], puis dessine le texte par rapport à cette position.
