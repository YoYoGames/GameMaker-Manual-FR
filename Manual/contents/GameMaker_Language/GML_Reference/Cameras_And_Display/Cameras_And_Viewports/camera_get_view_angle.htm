camera_get_view_angle

Cette fonction peut être utilisée pour récupérer l'angle de la caméra donnée. La valeur de retour sera comprise entre 0 et 360. Notez que cette fonction n' est valable que pour les caméras créées à l'aide de la fonction camera_create_view() ou pour celles ajoutées dans l'éditeur de salle.

Syntaxe :

camera_get_view_angle(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Exemple :

var ang = camera_get_view_angle(view_camera[0]);
if ang != 0
{
    camera_set_view_angle(view_camera[0], 0);
}

Le code ci-dessus récupère l'angle de la caméra affectée au port de vue [0], puis vérifie s'il correspond à la valeur donnée. Si ce n'est pas le cas, la caméra de vue est réglée sur cet angle.
