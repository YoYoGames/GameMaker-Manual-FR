camera_get_view_border_y

Cette fonction peut être utilisée pour récupérer la valeur de la bordure pour l'objet/instance suivant de la caméra donnée le long de l'axe y (bordure verticale). La valeur de retour sera en pixels.

Syntaxe :

camera_get_view_border_y(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Exemple :

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
    camera_set_view_border(view_camera[0], 200, 200);
}

Le code ci-dessus récupère les valeurs xborder et yborder de la caméra affectée au port de vue [0], puis vérifie si elles correspondent à la valeur donnée. Si ce n'est pas le cas, la caméra de vue est définie sur la valeur donnée.
