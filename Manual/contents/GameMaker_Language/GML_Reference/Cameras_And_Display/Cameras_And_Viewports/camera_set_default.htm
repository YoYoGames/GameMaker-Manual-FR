camera_set_default

Cette fonction peut être utilisée pour définir la caméra par défaut afin d'utiliser une caméra personnalisée que vous avez préalablement créée à l'aide de l'une des fonctions camera_create() fonctions. Lorsque vous créez une pièce sans ports d'observation actifs ou de caméra, GameMaker utilise toujours une caméra pour montrer l'action dans le jeu. Cette caméra est appelée caméra par défaut et peut être définie et manipulée (et même détruite) comme n'importe quelle autre caméra.

Syntaxe :

camera_set_default(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Exemple :

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

Le code ci-dessus crée une nouvelle caméra et l'assigne comme caméra par défaut à utiliser.
