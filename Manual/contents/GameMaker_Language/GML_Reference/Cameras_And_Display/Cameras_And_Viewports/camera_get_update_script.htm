camera_get_update_script

Cette fonction peut être utilisée pour récupérer l'ID de la fonction de script assignée comme script de mise à jour pour la caméra donnée. Si aucun script n'est attribué, la fonction renvoie -1.

Syntaxe :

camera_get_update_script(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Script Function or -1 if no function is assigned

Exemple :

var scr = camera_get_update_script(camera_view[0]);
if (scr != scr_cutscene)
{
    camera_set_update_script(camera_view[0], scr_cutscene);
}

Le code ci-dessus vérifie la fonction de script assignée en tant que fonction de mise à jour pour la caméra assignée au port de vue [0] et si ce n'est pas "cutScene", il est défini sur ce script.
