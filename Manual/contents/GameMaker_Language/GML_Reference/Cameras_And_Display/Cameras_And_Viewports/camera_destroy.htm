camera_destroy

Avec cette fonction, vous pouvez détruire n'importe quelle caméra. Lorsque vous appelez la fonction, vous fournissez la valeur unique de l'ID de la caméra, que vous obtenez à partir des fonctions camera_create_* ou à partir de la matrice view_camera si vous utilisez l'éditeur de salle pour configurer le port de vue et la vue. Vous ne devez jamais détruire une caméra qui est actuellement assignée à une vue visible, à moins que vous n'assigniez une nouvelle caméra à cette vue au cours de la même étape, et vous devez toujours détruire les caméras que vous avez créées par code lorsqu'elles ne sont plus nécessaires à votre jeu afin d'éviter les fuites de mémoire, et vous pouvez également détruire les caméras par défaut si vous en avez assignées dans l'éditeur de salle, mais vous devez assigner une nouvelle caméra au port de vue (ou la désactiver) sinon vous obtiendrez des résultats étranges.

Syntaxe :

camera_destroy(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera.

Les retours :

N/A

Exemple :

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

Le code ci-dessus détruit la caméra actuellement affectée au port de vue [0] puis crée une nouvelle caméra et affecte son ID à ce port.
