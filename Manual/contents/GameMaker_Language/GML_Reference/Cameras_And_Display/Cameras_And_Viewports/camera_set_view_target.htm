camera_set_view_target

Vous pouvez utiliser cette fonction pour définir la cible de suivi de la caméra de surveillance dans la pièce. Vous donnez la valeur unique de l'ID de la caméra (telle que renvoyée par les différentes camera_create() ), puis l'instance cible ou l'ID de l'objet sur lequel vous souhaitez définir la vue de la caméra. Notez que si vous choisissez l'ID d'un objet et qu'il y a plus d'une instance de cet objet dans la pièce, GameMaker n'a aucun moyen de savoir quelle instance vous souhaitez suivre et cela peut donc être n'importe laquelle.

Syntaxe :

camera_set_view_target(camera_id, instance_id/object_id)

Arguments Type Description
camera_id ID de la caméra The unique camera ID value returned when you created the camera
instance_id/object_id ID de l'actif ou de l'instance del'objet Instance or object to have the camera target for following

Les retours :

N/A

Exemple :

camera_set_view_target(view_camera[0], id);

Le code ci-dessus définira la cible de la caméra de vue pour la caméra affectée au port de vue [0] à celle de l'instance qui exécute le code.
