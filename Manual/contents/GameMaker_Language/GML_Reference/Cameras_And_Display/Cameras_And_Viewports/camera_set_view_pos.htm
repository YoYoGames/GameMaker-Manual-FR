camera_set_view_pos

Vous pouvez utiliser cette fonction pour mettre à jour la position de la vue de la caméra dans la pièce. Vous donnez la valeur unique de l'ID de la caméra (telle qu'elle est renvoyée par les différentes fonctions de camera_create() ), puis vous indiquez les positions x et y sur lesquelles la caméra doit se positionner.

Syntaxe :

camera_set_view_pos(camera_id, x, y)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera.
x The x position to place the view at (in the room)
y The y position to place the view at (in the room)

Les retours :

Exemple :

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

Le code ci-dessus définira la position de la caméra de vue pour la caméra affectée au port de vue [0].
