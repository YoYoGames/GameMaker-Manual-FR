camera_get_proj_mat

Cette fonction peut être utilisée pour récupérer la matrice de projection. La fonction renvoie le tableau matriciel qui peut ensuite être utilisé dans d'autres fonctions matricielles ou pour définir la matrice de projection d'une autre caméra (en utilisant la fonction camera_set_proj_mat()).

Syntaxe :

camera_get_proj_mat(camera_id)

ArgumentsType Description
camera_id The unique camera ID value returned when you created the camera

Les retours :

Matrix Array

Exemple :

mat = camera_get_proj_mat(camera_view[0]);

Le code ci-dessus obtient la matrice de projection pour la caméra affectée au port de vue [0].
