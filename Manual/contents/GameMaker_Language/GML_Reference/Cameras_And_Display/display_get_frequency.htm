afficher_get_fréquence

Cette fonction renvoie la fréquence (ou taux de rafraîchissement) de l'écran sur lequel le jeu est joué. Elle renvoie une valeur réelle sous forme d'images par seconde. Par exemple, si votre écran est à 60 Hz, vous obtiendrez 60, s'il fonctionne à 144 Hz, vous obtiendrez 144, et ainsi de suite.

NOTE Cette fonction n'est pas prise en charge par HTML5.

Syntaxe :

display_get_frequency();

Les retours :

Exemple :

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Le code ci-dessus récupère la fréquence de l'écran, et exécute certaines conditions pour que le jeu fonctionne à 240, 120, 60 ou 30 fps. Cela signifie que si votre écran est à 90hz, le jeu fonctionnera à 60 fps, s'il est à 144hz, il fonctionnera à 120 fps, etc. Bien entendu, vous pouvez transmettre la fréquence de l'écran directement à game_set_speed() pour qu'elle soit utilisée comme fréquence d'images du jeu.
