gif_save_buffer

Cette fonction vous permet d'enregistrer une animation GIF. Vous fournissez l'index GIF (tel que renvoyé par la fonction gif_open()) et la fonction renvoie un grow aligné sur 1 octet buffer avec les données GIF. Notez que les données GIF finales auront été palettisées à l'aide de la palette Universal 884 (voir ici pour plus d'informations).

Syntaxe :

gif_save_buffer(gif_index);

ArgumentsType Description
gif_index The ID of gif to save

Les retours :

Buffer ID

Exemple :

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        global.capture_buff = gif_save_buffer(gif_image);
        count = 0;
        save_gif = false;
    }
    count++;
}

Le code ci-dessus créera un fichier image GIF avec 30 images prises sur la surface de l'application, puis l'enregistrera sur un site buffer.
