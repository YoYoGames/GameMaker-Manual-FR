window_get_width

Cette fonction permet d'obtenir la largeur actuelle (en pixels) de la fenêtre du jeu.

Syntaxe :

window_get_width();

Les retours :

Exemple :

ww = window_get_width();

Le code ci-dessus stocke la largeur actuelle de la fenêtre dans la variable "ww".

Retour : La fenêtre de jeu
