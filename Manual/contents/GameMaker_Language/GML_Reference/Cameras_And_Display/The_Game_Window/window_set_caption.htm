fenêtre_set_caption

Cette fonction vous permet de modifier ou de définir la légende de la fenêtre pour la pièce dans laquelle vous vous trouvez actuellement. Cette légende apparaît en haut de la fenêtre, à côté de l'icône du jeu, lorsque le jeu n'est pas en mode plein écran.

Syntaxe :

window_set_caption(caption);

ArgumentsType Description
caption The new caption.

Les retours :

Exemple :

if window_get_caption() != ""
{
    window_set_caption("");
}

Le code ci-dessus vérifie la légende de la fenêtre pour voir si elle contient du texte ou non, et si c'est le cas, il lui donne la valeur d'une chaîne vide afin qu'aucune légende ne soit affichée.
