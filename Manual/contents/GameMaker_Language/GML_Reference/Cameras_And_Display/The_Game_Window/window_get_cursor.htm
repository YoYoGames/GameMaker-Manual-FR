window_get_cursor

Avec cette fonction, vous pouvez obtenir le curseur actuel utilisé dans la fenêtre du jeu, la valeur de retour étant l'une des constantes énumérées ci-dessous (pour définir le curseur actuel utilisé, vous pouvez utiliser la fonction window_set_cursor() qui prend également ces constantes) :

Constante du curseur
Constant Curseur
cr_none
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

Syntaxe :

window_get_cursor();

Les retours :

Cursor Constant

Exemple :

if mouse_check_button_pressed(mb_left)
{
    if window_get_cursor() != cr_drag window_set_cursor(cr_drag);
}

Le code ci-dessus changera le curseur de la fenêtre en curseur de déplacement standard de Windows si le bouton gauche de la souris a été enfoncé et s'il n'a pas déjà été changé auparavant.
