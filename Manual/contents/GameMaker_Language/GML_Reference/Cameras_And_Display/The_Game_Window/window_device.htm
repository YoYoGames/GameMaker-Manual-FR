window_device

Cette fonction renvoie le pointeur de périphérique d3d actuel, que vous pouvez ensuite (par exemple) transmettre à une DLL ou à une Dylib sur Windows et macOS.

NOTE Cette fonction a été dépréciée dans GameMaker en faveur de la fonction os_get_info().

Syntaxe :

window_device();

Les retours :

Pointer

Exemple :

gfx_pointer = window_device();

Le code ci-dessus stockera le pointeur du périphérique d3d dans une variable.

Retour : La fenêtre de jeu
