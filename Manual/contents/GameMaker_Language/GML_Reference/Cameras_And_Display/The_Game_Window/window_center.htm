fenêtre_centre

Cette fonction permet de centrer la fenêtre de jeu dans l'écran lorsque le module cible est Windows, Ubuntu (Linux) ou macOS, ou de la centrer dans le navigateur si le module cible est HTML5. Cette fonction n'a aucun effet sur les autres appareils.

NOTE Si vous avez redimensionné la fenêtre de jeu au cours de l'étape actuelle (par exemple, en passant du plein écran au fenêtré ou en utilisant window_set_size()), et que vous souhaitez centrer la nouvelle fenêtre, cette fonction doit être appelée au moins une étape plus tard (par exemple, en utilisant une fonction alarm), sinon elle ne fonctionnera pas correctement.

Syntaxe :

window_center();

Les retours :

Exemple :

if !window_get_fullscreen()
{
    window_center();
}

Le code ci-dessus permet de centrer la fenêtre du jeu dans l'écran si celui-ci n'est pas en mode plein écran.
