fenêtre_set_position

Cette fonction permet de placer la fenêtre de jeu à un endroit précis de l'écran (sur macOS, Linux(Ubuntu) et Windows) ou du navigateur (HTML5).

REMARQUE: si votre jeu HTML5 utilise un index.html personnalisé et qu'il définit le canevas à une position fixe, cette fonction n'aura aucun effet sur la position de la fenêtre.

Syntaxe :

window_set_position(x, y);

ArgumentsType Description
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

Les retours :

Exemple :

window_set_position(0, 0);

Le code ci-dessus positionnera la fenêtre de jeu dans le coin supérieur gauche du navigateur ou de l'écran (selon le module cible utilisé).
