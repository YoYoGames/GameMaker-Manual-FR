window_set_rectangle

Cette fonction vous permet de définir la position de la fenêtre de jeu dans le navigateur (HTML5) ou l'affichage (Windows, Ubuntu (Linux) ou macOS) et de définir également l'échelle de la fenêtre. Pour plus d'informations sur la position et la taille de la fenêtre, voir window_set_position() et window_set_size().

Syntaxe :

window_set_rectangle(x, y, w, h);

ArgumentsType Description
x The new x coordinate of the window.
y The new y coordinate of the window.
w The new width of the window.
h The new height of the window.

Les retours :

Exemple :

window_set_rectangle(0, 0, display_get_width(), display_get_height());

Le code ci-dessus fait en sorte que la fenêtre de jeu occupe toute la zone d'affichage (soit le navigateur, soit l'écran, selon la plate-forme cible).
