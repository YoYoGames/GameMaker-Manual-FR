fenêtre_get_fullscreen

Cette fonction indique si la fenêtre de jeu est en mode plein écran (true) ou non (false).

Syntaxe :

window_get_fullscreen();

Les retours :

Exemple :

if window_get_fullscreen()
{
    draw_text(32, 32, "Fullscreen is ON");
}
else
{
    draw_text(32, 32, "Fullscreen is OFF");
}

Le code ci-dessus va vérifier si la fenêtre est en mode plein écran ou non et dessiner un message texte en fonction de la valeur retournée.
