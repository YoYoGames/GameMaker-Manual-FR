window_view_mouse_get_y

Cette fonction renvoie la position y de la souris par rapport à la vue sélectionnée.

REMARQUE: Pour les fonctions normales de la souris, voir la section sur la saisie de la souris.

Syntaxe :

window_view_mouse_get_y( id );

Arguments Type Description
id Real The id of the view to compare the mouse position to.

Les retours :

Real

Exemple :

if mouse_check_button_pressed(mb_left)
{
    var xx, yy;
    xx = window_view_mouse_get_x(0);
    yy = window_view_mouse_get_y(0);

    if xx > 0 && xx < 32 && yy > 0 && yy < 32
    {
        b_press[0] = true;
    }
}

Le code ci-dessus vérifie si un bouton de la souris a été pressé, et si c'est le cas, il obtient la position de la souris par rapport à la vue [0] et la compare pour voir si une variable doit être définie comme vraie.
