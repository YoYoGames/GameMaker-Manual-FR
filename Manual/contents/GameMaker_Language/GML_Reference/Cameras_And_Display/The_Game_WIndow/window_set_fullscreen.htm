fenêtre_set_fullscreen

Cette fonction vous permet de définir la fenêtre de jeu en plein écran (true) ou non (false). Veuillez noter que pour la cible macOS, vous devez avoir décoché l'option "Start In Fullscreen" et coché l'option "Allow the player to resize the game window" dans les Game Options, sinon cette fonction échouera. Notez également que cette fonction ne fonctionnera pas sur HTML5, sauf si elle est ajoutée en tant que callback "cliquable" (voir ici pour plus de détails).

Syntaxe :

window_set_fullscreen(full);

ArgumentsType Description
full Whether to set the screen to fullscreen (true) or not (false).

Les retours :

Exemple :

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

Le code ci-dessus vérifie si un bouton de la souris a été pressé et fait passer la fenêtre en plein écran si ce n'est pas déjà le cas, ou en mode fenêtré si c'est déjà le cas.
