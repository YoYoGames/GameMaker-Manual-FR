fenêtre_a_focus

Avec cette fonction, vous pouvez interroger l'état de la fenêtre (ou de l'onglet) et si elle perd le focus, la fonction retournera false; sinon, elle retournera true. Dans la plupart des cas, vous pouvez simplement utiliser la fonction os_is_paused() pour tester cela, mais dans certains cas très spécifiques (par exemple les jeux sur les Chrome Apps), cette fonction ne se déclenchera pas, auquel cas vous devez utiliser cette fonction à la place.

REMARQUE: Cette fonction n'est valable que pour le HTML5, Windows, et MacOS.

Syntaxe :

window_has_focus();

Les retours :

Exemple :

if !window_has_focus()
{
    PauseGame();
}

Le code ci-dessus vérifiera si la fenêtre de jeu est en focus ou non, et si la fonction retourne false une fonction sera appelée.
