fenêtre_vue_souris_get_x

Cette fonction renvoie la coordonnée x de la souris par rapport à toutes les vues actives et retourne la même valeur mouse_x.

REMARQUE: Pour les fonctions normales de la souris, voir la section sur la saisie de la souris.

Syntaxe :

window_views_mouse_get_x();

Les retours :

Exemple :

myx = window_views_mouse_get_x();

Ceci définit myx comme la coordonnée x de la souris dans la fenêtre par rapport à toutes les vues actives.

Retour : La fenêtre de jeu
