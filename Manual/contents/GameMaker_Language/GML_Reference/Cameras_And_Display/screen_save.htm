screen_save

Cette fonction vous permet d'enregistrer une capture d'écran du jeu tel qu'il apparaît actuellement dans la fenêtre de jeu.

Le fichier sera enregistré dans le répertoire de travail du jeu, qui se trouve sous Windows à l'adresse <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ et sous Mac à l'adresse ~/Library/Application Support/[Game Name]/. L'image capturée sera le rendu final de la surface d'application du jeu, ou (si la surface d'application a été désactivée) ce sera une image de l'affichage ou de la fenêtre entière.

Il est important de noter que la fonction est conçue pour être appelée dans l'événement Draw GUI End Event, qui devrait donner des résultats cohérents sur toutes les plateformes. L'utiliser dans un autre événement peut ne pas fonctionner comme prévu et donner des résultats différents selon les cibles et même les appareils.

Si vous avez besoin de plus de contrôle sur ce qui est sauvegardé, vous devez utiliser la fonction screen_save_part().

NOTE Cette fonction ne fonctionne pas sur la cible HTML5.

Syntaxe :

screen_save(fname);

Arguments Type Description
fname Chaîne de caractères The filename for the screenshot to save

Les retours :

N/A

Exemple :

if keyboard_check_pressed(ord("S"))
{
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

Le code ci-dessus prend une capture d'écran du jeu lorsque l'utilisateur appuie sur la touche "S". Cette capture d'écran est enregistrée dans un répertoire spécifique et numérotée à l'aide d'une variable qui est ensuite incrémentée.
