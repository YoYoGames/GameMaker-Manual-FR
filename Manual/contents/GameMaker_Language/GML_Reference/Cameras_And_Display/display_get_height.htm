affichage_get_hauteur

Cette fonction renvoie la hauteur de l'affichage (en pixels). Notez que sur la cible HTML5, cette valeur est la hauteur de la fenêtre du navigateur, plutôt que la taille physique de l'affichage.

Syntaxe :

display_get_height();

Les retours :

Exemple :

myheight = display_get_height();

La variable "myheight" serait ainsi définie sur la hauteur de l'écran.
