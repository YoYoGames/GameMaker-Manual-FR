écran_save_part

Cette fonction vous permet d'enregistrer une capture d'écran d'une partie du jeu telle qu'elle apparaît actuellement dans l'affichage du jeu. Les coordonnées de sélection sont basées sur la taille absolue de l'écran (ou de la fenêtre s'il ne s'agit pas d'un écran complet), donc (0,0) est toujours le coin supérieur gauche de l'écran (ou de la fenêtre). L'image ci-dessous illustre le fonctionnement de cette fonction :

Le fichier sera enregistré dans le répertoire de travail du jeu. Il est important de noter que la fonction est conçue pour être appelée dans l'événement Draw GUI End Event, qui devrait donner des résultats cohérents sur toutes les plateformes. L'utiliser dans un autre événement peut ne pas fonctionner comme prévu et donner des résultats différents selon les cibles et même les appareils.

NOTE Cette fonction ne fonctionne pas sur la cible HTML5.

Syntaxe :

screen_save_part(fname,x,y,w,h);

ArgumentsType Description
fname The filename for the screenshot to save
x The x coordinate of the top left corner of the area of the screen to capture
y The y coordinate of the top left corner of the area of the screen to capture
w The width of the area of the screen to capture
h The height of the area of the screen to capture

Les retours :

Exemple :

if keyboard_check_pressed(ord("S"))
{
    screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100);
}

Le code ci-dessus prend une capture d'écran d'une partie de l'écran du jeu qui fait 100 pixels carrés lorsque l'utilisateur appuie sur la touche "S". Cette capture d'écran est enregistrée dans un dossier spécifique et numérotée à l'aide d'une variable qui est ensuite incrémentée.
