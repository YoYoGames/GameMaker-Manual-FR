display_set_ui_visibility

Cette fonction peut être utilisée pour afficher ou masquer l'interface utilisateur du système sur Android uniquement. La fonction vous demande de fournir un ou plusieurs indicateurs système sous forme de valeur entière. Si vous utilisez plus d'un drapeau, vous devez les fusionner à l'aide de la fonction "ou", comme le montre l'exemple ci-dessous. Vous pouvez trouver une liste de Android system flags ici.

Syntaxe :

display_set_ui_visibility(flags);

Arguments Type drapeaux
flags Real The Android SYSTEM flags to use (as an integer value).

Les retours :

N/A

Exemple :

var flags = 1024|4096;
display_set_ui_visibility(flags);

Le code ci-dessus utilise le système Android flags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" et "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" pour définir la visibilité de l'affichage.
