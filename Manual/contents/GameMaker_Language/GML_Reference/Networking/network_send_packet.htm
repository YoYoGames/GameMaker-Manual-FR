réseau_envoi_paquet

Cette fonction permet d'envoyer un "paquet" de données sur le réseau. La fonction prend l'identifiant de la socket à connecter, puis vous devez fournir l'identifiant du tampon qui contient les données à envoyer (pour plus d'informations sur buffers, voir Référence - Tampons) et enfin la taille (en octets) du paquet de données. Les paquets envoyés avec cette fonction sont formatés de façon à ce que le jeu GameMaker qui reçoit les données puisse "découper" les paquets correctement, et la fonction retournera le nombre d'octets de données envoyés, ou un nombre inférieur à 0 si l'envoi a échoué. Il convient de noter que la taille finale des données envoyées qui est renvoyée par cette fonction comprendra également les informations de l'en-tête GameMaker, qui représentent 12 octets supplémentaires.

Syntaxe :

network_send_packet(socket, bufferid, size);

ArgumentsType Description
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

Les retours :

Real

Exemple :

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

Les informations ci-dessus chargent en mémoire des données précédemment enregistrées sur buffer et renvoient l'identifiant de buffer à stocker dans la variable "buff". Ce buffer complet est ensuite envoyé sous forme de paquet sur le réseau en utilisant le socket identifié par la variable "sock".
