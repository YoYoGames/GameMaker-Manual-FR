network_create_socket_ext

Cette fonction est utilisée pour créer un nouveau socket client pour que votre jeu puisse communiquer sur le réseau. Vous devez définir le type de socket (voir la liste des constantes ci-dessous) et donner un port à utiliser, et la fonction retournera un identifiant unique qui devra être utilisé dans tous les appels de fonction ultérieurs pour ce socket, ou une valeur inférieure à 0 si la connexion échoue.

Syntaxe :

network_create_socket_ext(protocol, port);

Arguments Type Description
protocol Type de douille Constant The network protocol to use
port Real The port to use

Les retours :

Network Socket ID

Exemple :

client = network_create_socket_ext(network_socket_udp, 6510);

Le code ci-dessus va créer un nouveau socket UDP sur le port 6510.

Retour : Mise en réseau
