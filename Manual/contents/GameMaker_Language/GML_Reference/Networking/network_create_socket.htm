réseau_créer_socket

Cette fonction est utilisée pour créer un nouveau socket client pour que votre jeu puisse communiquer sur le réseau. Vous devez définir le type de socket (voir la liste des constantes ci-dessous) et la fonction renverra un identifiant unique pour ce socket, qui devra être utilisé dans tous les autres appels de fonction pour ce socket, ou une valeur inférieure à 0 si la connexion échoue.

ASTUCE Vous pouvez utiliser l'adresse IP "127.0.0.1" pour vous reconnecter au même appareil qui exécute le jeu.

Syntaxe :

network_create_socket(type);

Arguments Type Description
type Type de douille Constant The type of socket connection to create (see the constants listed above).

Les retours :

Network Socket ID

Exemple :

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

Le code ci-dessus vérifiera si le jeu est exécuté dans un navigateur ou non et créera un nouveau socket TCP ou Web avant de tenter de se connecter à travers celui-ci à l'adresse IP donnée sur le port donné.
