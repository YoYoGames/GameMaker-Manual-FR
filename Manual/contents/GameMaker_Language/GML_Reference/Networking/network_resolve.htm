résolution de réseau

Cette fonction renvoie l'adresse IP de l'URL donnée.

Syntaxe :

network_resolve(url);

ArgumentsType Description
url The URL to get the IP of (a string).

Les retours :

Exemple :

game_ip = network_resolve("www.macsweeneygames.com");

Le code ci-dessus renvoie l'adresse IP de l'URL donnée et la stocke dans la variable "game_ip".
