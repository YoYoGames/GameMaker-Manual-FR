réseau_créer_serveur_raw

Cette fonction est utilisée pour créer un nouveau serveur réseau pour votre jeu, en utilisant l'un des protocoles de connexion autorisés (voir les constantes listées ci-dessous). Vous fournissez le type de serveur, puis vous lui donnez un port à utiliser, et enfin le nombre maximum de connexions autorisées à tout moment au serveur (notez que ce nombre est à votre discrétion, mais un trop grand nombre de clients connectés saturera le réseau ou le CPU du périphérique ne sera pas en mesure de gérer le traitement de ce nombre de joueurs, donc à utiliser avec précaution). La fonction renvoie un identifiant unique qui doit être stocké dans une variable et utilisé pour identifier le serveur dans toutes les autres fonctions réseau, ou une valeur inférieure à 0 si la connexion échoue.

Comme cela crée un serveur "brut", il n'acceptera ni n'utilisera aucun des en-têtes de données intégrés à GameMaker pour la communication, et vous devrez donc utiliser les fonctions network_send_raw() et network_send_udp_raw() pour envoyer des données non formatées au serveur créé.

Syntaxe :

network_create_server_raw(type, port, max_client);

ArgumentsType Description
type The type of server to create (see the constants listed below).
port The port that the server will use.
max_client The maximum number of clients that can connect at once.

Les retours :

Network Socket ID

Exemple :

var port = 6510;
server = network_create_server_raw(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server(network_socket_tcp, port, 32);
}

Le code ci-dessus va essayer de créer un serveur en utilisant le port TCP 6510. Si ce port n'est pas disponible, il parcourra les ports en boucle pour en trouver un qui le soit.
