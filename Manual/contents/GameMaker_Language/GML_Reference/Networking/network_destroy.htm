réseau_destruction

Cette fonction vous permet de supprimer une connexion de prise réseau de votre jeu.

NOTE: Cette fonction ne fonctionnera pas si elle est utilisée dans un projet fonctionnant sur la cible HTML5.

Syntaxe :

network_destroy(socket);

ArgumentsType Description
socket The id of the socket to remove.

Les retours :

Exemple :

if !os_is_network_connected()
{
    network_destroy(sock);
}

Le code ci-dessus vérifiera s'il existe une connexion de données et si aucune n'est trouvée, il détruira le socket avec l'id "sock".
