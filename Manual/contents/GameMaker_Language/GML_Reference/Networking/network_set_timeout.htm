réseau_set_timeout

Cette fonction vous permet de définir le délai d'attente pour la lecture et l'écriture de données vers/depuis un serveur via le socket donné. Notez que le délai d'attente ne génère aucun type d'événement, vous devrez donc gérer les délais d'attente vous-même en utilisant des alarmes (par exemple). Notez que cette valeur n'affecte que l'envoi et la réception de données, et que si vous souhaitez modifier la valeur du délai de connexion, vous devez utiliser la fonction network_set_config().

Syntaxe :

network_set_timeout(socket, read_timeout, write_timeout);

ArgumentsType Description
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

Les retours :

Exemple :

network_set_timeout(sock, 3000, 3000);

Le code ci-dessus fixe le délai d'attente pour la lecture et l'écriture de données via le socket indexé dans la variable "sock" à 3 secondes.
