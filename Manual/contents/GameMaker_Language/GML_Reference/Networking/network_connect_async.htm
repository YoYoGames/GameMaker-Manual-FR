réseau_connexion_asynchrone

Avec cette fonction, vous pouvez envoyer une requête pour vous connecter à un serveur. La fonction prend l'identifiant du socket à utiliser pour la connexion (voir network_create_socket()) et vous demande de fournir l'adresse IP à laquelle se connecter (une chaîne) ainsi que le port à utiliser. (une chaîne de caractères) ainsi que le port à utiliser pour la connexion. Si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La connexion utilise un protocole spécial qui garantit que seuls les jeux GameMaker se connectent les uns aux autres. de vous connecter à un serveur qui n'est pas un jeu GameMaker, vous pouvez utiliser la commande network_connect_async_raw(). Notez que cette fonction est asynchrone et génère un événement de mise en réseau asynchrone de type network_type_non_blocking_connect.

Syntaxe :

network_connect_async(socket, url, port);

ArgumentsType Description
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Les retours :

Exemple :

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

Le code ci-dessus va créer un nouveau socket TCP puis tenter d'établir une connexion asynchrone à travers celui-ci à l'adresse IP donnée sur le port 6510.
