montrer_erreur

Cette fonction affichera une chaîne personnalisée comme message d'erreur.

IMPORTANT Le deuxième argument n'existe que pour la compatibilité avec les anciens projets et n'aura aucun effet sur la boîte de dialogue d'erreur affichée.

NOTE Cette fonction est réservée à un usage de débogage.

Syntaxe :

show_error(str, abort);

Arguments Type Description
str Chaîne de caractères The string to show in the pop-up message.
abort Booléen UNUSED Si l'erreur doit interrompre le jeu (true) ou permettre au joueur de l'ignorer (false).

Les retours :

N/A

Exemple :

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

Le code ci-dessus vérifiera si la salle actuelle est rm_Dungeon, et si ce n'est pas le cas, il affichera un message d'erreur.

Retour : Débogage
