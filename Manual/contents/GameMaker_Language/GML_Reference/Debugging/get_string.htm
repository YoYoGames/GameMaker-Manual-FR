get_string

Cette fonction crée une fenêtre contextuelle affichant un message standard, avec un bouton intitulé "Ok", qui invite l'utilisateur à saisir une chaîne de caractères. La fonction renvoie la chaîne de caractères saisie, ou la valeur par défaut si rien n'a été saisi.

NOTE Cette fonction est destinée à un usage de débogage uniquement sur la cible Windows, mais est dépréciée sur toutes les autres cibles.

Syntaxe :

get_string(str, def);

Arguments Type Description
str Chaîne de caractères The string to show in the pop-up message.
def Chaîne de caractères The default string in the text box.

Les retours :

String

Exemple :

global.test_name = get_string("Test highscore name:", "Anonymous");

Le code ci-dessus demandera à l'utilisateur de donner un nom qui sera ensuite stocké dans la variable globale "test_name". Si rien n'est saisi et que l'utilisateur appuie simplement sur "Ok", la valeur par défaut, "Anonyme", sera renvoyée.
