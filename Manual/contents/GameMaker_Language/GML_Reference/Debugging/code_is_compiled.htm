code_est_compilé

Cette fonction renverra true si la plate-forme est compilée en dehors de la machine virtuelle, comme pour les plates-formes YYC et JS.

Syntaxe :

code_is_compiled();

Les retours :

Exemple :

if code_is_compiled()
{
    show_debug_message("Compiler okay!");
}

Le code ci-dessus vérifie si le jeu a été compilé à l'aide du compilateur YoYo et affiche un message de débogage si c'est le cas.
