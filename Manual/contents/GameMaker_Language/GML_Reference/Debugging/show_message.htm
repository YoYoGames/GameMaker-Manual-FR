montrer_message

Cette fonction crée une boîte de message contextuelle qui affiche la chaîne de caractères donnée et un bouton marqué "Ok" pour la fermer.

NOTE Cette fonction est destinée à un usage de débogage uniquement sur la cible Windows, mais est dépréciée sur toutes les autres cibles.

Syntaxe :

show_message(str);

Arguments Type Description
str Chaîne de caractères The string to show in the pop-up message.

Les retours :

N/A

Exemple :

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

Le code ci-dessus va parcourir en boucle les valeurs stockées dans le tableau "inv" et les ajouter à la variable "tot" avant d'afficher un message avec le total.
