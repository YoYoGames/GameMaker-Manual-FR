bool

Cette fonction tentera de convertir une valeur donnée en un type de données booléen, où la valeur sera retournée sous la forme true si elle est supérieure ou égale à 0,5, et false sinon.

Syntaxe :

bool(n);

Arguments Type Description
n Real The value to convert.

Les retours :

Boolean

Exemple :

if !is_bool(val)
{
    val = bool(val);
}

Le code ci-dessus vérifie la variable "val" pour voir si c'est un booléen et si ce n'est pas le cas, elle est convertie en un booléen.
