tableau_get

Avec cette fonction, vous pouvez récupérer la valeur d'un index dans un tableau. La fonction requiert que vous fournissiez une variable qui contient le tableau et l'index pour récupérer la valeur dans ce tableau. Notez que si l'indice du tableau donné est hors limites, le jeu se plantera avec une erreur. Cette fonction peut également être utilisée pour les tableaux multidimensionnels, à condition de spécifier quelle dimension vous souhaitez obtenir lorsque vous fournissez l'indice du tableau, en suivant ce modèle :

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

Syntaxe :

array_get(variable, index);

Arguments Type Description
variable Array The variable that holds the array.
index Real The index of the array element to get the value from.

Les retours :

Variable (Any valid data type that an array can hold)

Exemple :

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

Le code ci-dessus affichera les 10 premiers éléments de la liste donnée dans la console.
