variable_instance_exists

Cette fonction vous permet de vérifier si une variable d'instance existe ou non. Vous fournissez la valeur unique de l'ID d'instance (qui peut être trouvée dans les propriétés d'instance de l'éditeur de salle, ou qui est renvoyée lorsque vous appelez la fonction instance_create_layer()) ainsi que le nom de la variable à vérifier sous forme de chaîne (voir l'exemple de code ci-dessous). La fonction renverra true si une variable avec le nom donné existe pour l'instance et false sinon.

Syntaxe :

variable_instance_exists(instance_id, name);

Arguments Type Description
instance_id ID de l'instance The unique ID value of the instance to check
name Chaîne de caractères The name of the variable to check for

Les retours :

Boolean

Exemple :

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

Le code ci-dessus va vérifier si la variable appelée "shields" existe dans l'instance qui exécute le code et si ce n'est pas le cas, elle est créée et initialisée à 0.
