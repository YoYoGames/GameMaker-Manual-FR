est_chaîne

Cette fonction renvoie si une variable donnée est une chaîne de caractères ou non. Dans certains cas, vous voudrez vérifier si une variable dans GameMaker contient une chaîne de caractères et non un réel et c'est dans ce cas que vous utiliserez cette fonction. Elle ne renvoie pas la chaîne de caractères mais plutôt true ou false, de sorte qu'une valeur de, par exemple, "poisson" pour n renverra true, mais une valeur de 200 pour n renverra false.

Syntaxe :

is_string(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if is_string(val)
{
    name = "Player: " + val;
}

Le code ci-dessus vérifie la variable "val" pour voir si elle contient une chaîne de caractères et si c'est le cas, il l'ajoute à une autre chaîne de caractères qui est ensuite affectée à la variable "name".
