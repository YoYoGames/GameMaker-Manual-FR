variable_instance_get_names

Cette fonction vous permet de récupérer un tableau contenant les noms des variables d'une instance ou les variables globales d'un jeu. Lorsque vous passez une valeur d'ID d'instance, chaque entrée du tableau sera une chaîne de caractères du nom de la variable qui correspond à une variable de portée d'instance qui a été créée dans l'instance. Cependant, si vous passez le mot-clé global, chaque entrée du tableau sera une chaîne de caractères du nom de la variable correspondant à une variable de portée globale.

Syntaxe :

variable_instance_get_names(instance_id/global);

Arguments Type Description
instance_id/global ID de l'instance ou global The unique ID value of the instance to check or the keyword global

Les retours :

Array (each entry is a string)

Exemple :

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++)
{
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
}

Le code ci-dessus récupère un tableau de toutes les variables d'instance pour l'instance qui exécute le bloc de code, puis les affiche avec leurs valeurs dans la sortie de débogage.
