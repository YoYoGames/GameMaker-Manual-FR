tableau_suppression

Avec cette fonction, vous pouvez supprimer une ou plusieurs valeurs d'un tableau à n'importe quelle position. La fonction vous demande de fournir une variable qui contient le tableau, l'index (position) dans le tableau à supprimer, ainsi que le nombre de valeurs à supprimer.

Syntaxe :

array_delete(variable, index, number);

ArgumentsType Description
variable The variable that holds the array.
index The index (position) in the array to delete the value(s) from.
number The number of values to delete.

Les retours :

Exemple :

array_delete(score_array, 0, 10);

Le code ci-dessus supprimera les 10 premières valeurs du tableau donné en commençant par l'indice 0 (les indices 0 à 9 seront donc supprimés du tableau).
