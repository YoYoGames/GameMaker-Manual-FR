variable_instance_set

Avec cette fonction, vous pouvez définir la valeur d'une variable donnée dans une instance. Vous fournissez la valeur unique de l'ID de l'instance (qui peut être trouvée dans les propriétés de l'instance dans l'éditeur de room, ou qui est retournée lorsque vous appelez la fonction instance_create_layer()) ainsi que le nom de la variable dont vous souhaitez définir la valeur sous forme de chaîne de caractères (voir l'exemple de code ci-dessous), et enfin la valeur à définir (peut être n'importe quel type de données valide). Si la variable n'existe pas déjà dans l'instance, elle sera créée et la valeur lui sera attribuée.

Syntaxe :

variable_instance_set(instance_id, name, val);

Arguments Type Description
instance_id ID de l'instance The unique ID value of the instance to use
name Chaîne de caractères The name of the variable to set (as a string)
val Variable The value to set the variable to

Les retours :

N/A

Exemple :

if !variable_instance_exists(id, "shields")
{
    variable_instance_set(id, "shields", 0);
}

Le code ci-dessus va vérifier si une variable d'instance existe dans l'instance appelante et si ce n'est pas le cas, elle est créée et mise à 0.
