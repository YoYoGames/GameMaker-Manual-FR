is_nan

Cette fonction indique si une variable donnée est NaN (pas un nombre) ou non, renvoyant true si elle l'est, et false si elle ne l'est pas.

Syntaxe :

is_nan(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if is_nan(global.value)
{
    show_debug_message("Value is not a number");
}

Le code ci-dessus vérifie la variable globale "value" pour voir si c'est un nombre ou non et affiche un message de débogage si ce n'est pas le cas.
