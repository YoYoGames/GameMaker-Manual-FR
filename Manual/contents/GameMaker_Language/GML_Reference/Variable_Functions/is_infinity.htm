est_infini

Cette fonction indique si une variable donnée est infinity (un nombre infini) ou non, renvoyant true si c'est le cas, et false si ce n'est pas le cas.

Syntaxe :

is_infinity(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

Le code ci-dessus vérifie la variable globale "value" pour voir si elle est infinie ou non et affiche un message de débogage si c'est le cas.
