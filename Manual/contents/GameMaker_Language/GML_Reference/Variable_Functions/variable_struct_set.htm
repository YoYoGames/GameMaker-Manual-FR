ensemble_de_structures_variables

Cette fonction permet de définir la valeur d'une variable donnée dans un fichier struct. Vous devez fournir la référence struct ainsi que le nom de la variable dont vous souhaitez définir la valeur sous forme de chaîne de caractères (voir l'exemple de code ci-dessous), et enfin la valeur à définir (qui peut être n'importe quel type de données valide). Si la variable n'existe pas déjà dans le site struct, elle sera créée, puis la valeur lui sera attribuée.

Syntaxe :

variable_struct_set(struct, name, val);

Arguments Type Description
struct Structure The struct reference to set
name Chaîne de caractères The name of the variable to set (as a string)
val Variable The value to set the variable to

Les retours :

N/A

Exemple :

if !variable_struct_exists(mystruct, "shields")
{
    variable_struct_set(mystruct, "shields", 0);
}

Le code ci-dessus vérifiera si la variable donnée existe dans le site struct et si ce n'est pas le cas, elle sera créée et mise à 0.
