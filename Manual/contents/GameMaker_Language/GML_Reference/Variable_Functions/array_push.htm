array_push

Avec cette fonction, vous pouvez pousser une ou plusieurs valeurs à la fin d'un tableau sans avoir à connaître la longueur du tableau. La fonction requiert que vous fournissiez une variable qui contient le tableau ainsi qu'au moins une valeur à pousser, bien que vous puissiez optionnellement fournir d'autres arguments et ils seront tous poussés sur le tableau dans un ordre consécutif.

Syntaxe :

array_push(variable, value, [value], [value], [etc...]);

Arguments Type Description
variable Array The variable that holds the array.
value Variable The value to push onto the end of the array
[value], [value], [etc...] Variable OPTIONAL Autres valeurs à ajouter au tableau

Les retours :

N/A

Exemple :

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

Le code ci-dessus va pousser quatre valeurs à la fin du tableau donné.
