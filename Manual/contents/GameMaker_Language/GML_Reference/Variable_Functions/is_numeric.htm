est_numérique

Cette fonction renvoie si une variable donnée est une valeur numérique (real, int32, int64 ou boolean) ou non. Dans certains cas, vous voudrez vérifier si une variable dans GameMaker contient une valeur numérique, et c'est dans ce cas que vous utiliserez cette fonction. La fonction retournera true si l'entrée donnée est numérique, et false sinon.

Syntaxe :

is_numeric(n);

Arguments Type Description
n Variable The input to check.

Les retours :

Boolean

Exemple :

if is_numeric(val)
{
    current_val += val;
}

Le code ci-dessus vérifie la variable "val" pour voir si c'est une valeur numérique et si c'est le cas, il l'ajoute à la variable donnée.
