is_matrix

Cette fonction renvoie si une variable donnée fait référence à une matrice ou non. Dans certains cas, vous souhaitez vérifier le type de données d'une variable dans GameMaker et c'est dans ce cas que vous utiliserez cette fonction. Elle renvoie true ou false selon que la valeur est une matrice ou non.

Syntaxe :

is_matrix(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if !is_matrix(val)
{
    show_debug_message("Not a valid matrix!");
}

Le code ci-dessus vérifie la variable "val" pour voir si elle contient une matrice et si ce n'est pas le cas, il affiche un message dans la console de débogage.
