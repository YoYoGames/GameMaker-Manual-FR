méthode_get_index

Avec cette fonction, vous pouvez récupérer l'index du script dans lequel la méthode a été définie. Si la méthode n'a pas été définie dans un script, la fonction renvoie -1, sinon elle renvoie la valeur de l'index du script.

Syntaxe :

method_get_index(method);

Arguments Type Description
method Méthode The method variable to check

Les retours :

Script Asset or Real

Exemple :

var _index = method_get_index(light_setup);
if (_index != undefined)
{
    show_debug_message(script_get_name(_index));
}

Le code ci-dessus vérifiera l'index du script pour la méthode "light_setup" et s'il n'est pas -1, il affichera le nom du script dans la console.
