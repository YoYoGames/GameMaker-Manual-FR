is_array

Cette fonction peut être utilisée pour vérifier si une variable contient un tableau (elle retournera true) ou non (dans ce cas, elle retournera false).

Syntaxe :

is_array(n);

Arguments Type Description
n Variable The variable to check.

Les retours :

Boolean

Exemple :

if is_array(a)
{
    a = -1;
}

Le code ci-dessus vérifie une variable pour voir s'il s'agit d'un tableau, et si la fonction renvoie true, le tableau est supprimé en mettant la variable à -1.
