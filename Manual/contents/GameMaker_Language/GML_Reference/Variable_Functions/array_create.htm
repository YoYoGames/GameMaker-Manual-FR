array_create

Avec cette fonction, vous pouvez créer un tableau 1D d'une taille donnée. Vous indiquez à la fonction la longueur du tableau à créer, et elle renvoie le "handle" du tableau que vous pouvez ensuite affecter à une variable. Les tableaux créés de cette manière auront chaque entrée initialisée à 0, sauf si vous spécifiez une valeur d'initialisation (facultative). Si vous fournissez la valeur supplémentaire pour l'initialisation du tableau, tous les indices dans le nouveau tableau seront fixés à cette valeur au lieu de 0, mais notez que la fonction aura une plus grande surcharge de performance dans ce cas.

Syntaxe :

array_create(size, [value]);

Arguments Type Description
size Real The size of the array to create.
value Variable OPTIONAL La valeur à utiliser pour initialiser tous les indices du tableau.

Les retours :

Array

Exemple :

instance_array = array_create(100, noone);

Le code ci-dessus crée un nouveau tableau de 100 entrées, chacune étant définie par le mot clé noone, puis l'affecte à la variable "instance_array".
