est_ptr

Cette fonction renvoie si une variable donnée est un pointeur ou non. Dans certains cas, vous voulez vérifier le type de données d'une variable dans GameMaker et c'est là que vous utiliserez cette fonction. Elle renvoie true ou false selon que la valeur est un pointeur ou non.

Syntaxe :

is_ptr(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if !is_ptr(val)
{
    show_debug_message("Not a valid texture!");
}

Le code ci-dessus vérifie la variable "val" pour voir si elle contient un pointeur et si ce n'est pas le cas, il affiche un message dans la console de débogage.
