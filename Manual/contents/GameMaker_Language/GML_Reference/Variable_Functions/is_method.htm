est_méthode

Cette fonction peut être utilisée pour vérifier si une variable est une variable de méthode (elle retournera true) ou non (dans ce cas, elle retournera false).

Syntaxe :

is_method(n);

Arguments Type Description
n Variable The variable to check.

Les retours :

Boolean

Exemple :

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

Le code ci-dessus vérifie une variable pour voir s'il s'agit d'une méthode, et si la fonction renvoie true, alors un message de débogage est affiché sur la console.
