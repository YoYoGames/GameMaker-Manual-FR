longueur du tableau

Cette fonction permet d'obtenir la longueur (nombre d'entrées) d'une dimension de tableau. Vous fournissez la valeur de l'index du tableau et la fonction renvoie une valeur entière représentant le nombre d'entrées que le tableau contient. Cette fonction peut également être utilisée pour les tableaux multidimensionnels, à condition de spécifier la dimension dont vous souhaitez obtenir la longueur lorsque vous fournissez l'indice du tableau, en suivant ce modèle :

// Returns the first dimension of the array
val = array_length(my_array);

// Returns the second dimension of the array
val = array_length(my_array[0]);

// Returns the third dimension of the array
val = array_length(my_array[0][0]);

// etc.

Syntaxe :

array_length(array_index);

ArgumentsType Description
array_index The index of the array to check.

Les retours :

Exemple :

for (var i = 0; i < array_length(a); ++i)
{
    a[i] = -1;
}

Le code ci-dessus va parcourir un tableau en boucle et donner la valeur -1 à chaque entrée.
