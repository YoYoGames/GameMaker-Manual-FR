is_vec4

Dans certains cas, vous voulez vérifier le type de données d'une variable dans GameMaker et c'est là que vous utiliserez cette fonction. Elle renvoie true ou false selon que la valeur est un vecteur à 4 valeurs ou non.

Syntaxe :

is_vec4(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if !is_vec4(val)
{
    show_debug_message("Not a 4 value vector!");
}

Le code ci-dessus vérifie la variable "val" pour voir si elle contient un vec4 et si ce n'est pas le cas, il affiche un message dans la console de débogage.
