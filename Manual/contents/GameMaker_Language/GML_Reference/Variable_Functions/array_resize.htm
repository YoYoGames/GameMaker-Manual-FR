tableau_resize

Cette fonction vous permet de redimensionner un tableau existant à une nouvelle dimension. Vous fournissez le tableau à redimensionner ainsi que le nouveau nombre d'indices pour le tableau, et la fonction redimensionnera ce tableau. Notez que cette fonction est conçue pour redimensionner un tableau vers le bas à une longueur plus petite, car vous pouvez redimensionner vers le haut en définissant simplement un nouvel index dans le tableau. Cela dit, si vous l'utilisez pour redimensionner un tableau, tous les nouveaux indices seront fixés à la valeur par défaut de 0.

Cette fonction peut également être utilisée pour des tableaux multidimensionnels, à condition de préciser quelle dimension vous souhaitez redimensionner lorsque vous fournissez l'indice du tableau, en suivant ce modèle :

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

Le code indique que seul le premier emplacement de la deuxième dimension est redimensionné, car les emplacements d'une dimension donnée du tableau ne sont pas interconnectés et peuvent avoir des tailles différentes ; par exemple :

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

Dans le code ci-dessus, la longueur de la deuxième dimension du tableau est de 10 dans le premier emplacement mais de 22 dans le deuxième.

Syntaxe :

array_resize(array_index, new_size);

ArgumentsType Description
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

Les retours :

Exemple :

if array_length(a) > 10
{
    array_resize(a, 10);
}

Le code ci-dessus vérifie la longueur d'un tableau et s'il a plus de 10 indices, il est redimensionné.
