variable_globale_existe

Avec cette fonction, vous pouvez vérifier si une variable de portée globale existe ou non. Vous fournissez le nom de la variable globale à vérifier sous forme de chaîne de caractères (voir l'exemple de code ci-dessous) et la fonction renvoie true si une variable globale portant le nom donné existe ou false sinon.

Syntaxe :

variable_global_exists(name);

Arguments Type Description
name Chaîne de caractères The name of the global variable to check for (as a string)

Les retours :

Boolean

Exemple :

if !variable_global_exists("enemy_num")
{
    global.enemy_num = instance_number(obj_Enemey_Parent);
}

Le code ci-dessus vérifie si la variable globale appelée "enemy_num" existe et si ce n'est pas le cas, elle est créée et une valeur lui est attribuée.
