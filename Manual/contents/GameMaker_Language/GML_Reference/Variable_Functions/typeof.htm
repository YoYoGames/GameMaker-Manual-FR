typeof

Cette fonction renvoie le type de données d'une variable donnée sous forme de chaîne. Les valeurs de retour possibles sont énumérées dans le tableau ci-dessous :

Chaîne de caractères Description
"number" La variable contient un nombre réel (à virgule flottante) - cela peut inclure NaN et infinity
"string" La variable contient une chaîne de caractères
"array" La variable fait référence à un tableau
"bool" La variable contient un booléen (true / false)
"int32" La variable contient un entier de 32 bits
"int64" La variable contient un entier de 64 bits
"ptr" La variable contient un pointeur
"undefined" La variable est indéfinie
"null" La variable contient une valeur nulle (cela ne devrait pas être vu normalement).
"vec3" La variable contient un vecteur de 3 valeurs
"vec4" La variable contient un vecteur de 4 valeurs
"method" La variable contient une référence de fonction
"struct" La variable contient une référence struct
"ref" La variable contient une référence d'instance
"unknown" La valeur est inconnue. Ce message ne devrait jamais être vu et signifie que quelque chose a mal tourné au niveau le plus basique, comme un écrasement de la mémoire.

Veuillez noter qu'il existe des cas où cette fonction peut ne pas retourner la valeur correcte pour une méthode. Considérons les deux définitions de fonction suivantes :

a = function()
{
    // something
}

function b()
{
    // Something
}

Techniquement, elles sont toutes deux considérées comme des méthodes puisqu'elles lient une fonction à une variable de portée d'instance. Cependant, l'appel de typeof() sur la fonction b renverra "nombre" et non "méthode", alors que l'appel sur a renverra "méthode". Cela est dû au fait que les méthodes créées comme celle de b se voient attribuer des indices de script (qui sont des valeurs entières), car c'est ainsi que le compilateur reconnaît les fonctions de script, même si la fonction n'a pas été définie dans un script.

Syntaxe :

typeof(variable);

Arguments Type Description
variable Tout The variable to get the data type of.

Les retours :

String (see table above)

Exemple :

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

Le code ci-dessus récupère le type de données détenu par la variable globale donnée et renvoie la chaîne de caractères à une variable locale qui est ensuite utilisée pour afficher un message sur la console.
