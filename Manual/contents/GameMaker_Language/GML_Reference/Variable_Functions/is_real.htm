est_réel

Cette fonction renvoie si une variable donnée est un nombre réel (simple, double ou entier) ou non. Dans certains cas, vous voulez vérifier si une variable contient un nombre réel, et c'est là que vous utiliserez cette fonction. Elle ne renvoie pas le nombre réel mais plutôt true ou false, de sorte qu'une valeur de, par exemple, "poisson" renverrait false, alors qu'une valeur de 200 renverrait true.

REMARQUE: Cette fonction renverra false pour toute enum car elles sont enregistrées sous int64s. La fonction correcte pour vérifier ce type de données est is_int64().

Syntaxe :

is_real(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if is_real(val)
{
    score += val;
}

Le code ci-dessus vérifie la variable "val" pour voir si c'est un nombre réel et si c'est le cas, il l'ajoute au score.
