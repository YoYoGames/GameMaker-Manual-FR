suppression_de_la_structure_variable

Cette fonction permet de supprimer une variable d'un site struct. Vous devez fournir l'ID du site struct dont vous souhaitez supprimer la variable et le nom de la variable à supprimer.

Syntaxe :

variable_struct_remove(struct, name);

Arguments Type Description
struct Structure The struct reference to remove the variable from
name Chaîne de caractères The name of the variable to remove (as a string)

Les retours :

N/A

Exemple :

if variable_struct_exists(mystruct, "shields")
{
    variable_struct_remove(mystruct, "shields");
}

Le code ci-dessus vérifiera si la variable donnée existe dans le site struct et si c'est le cas, elle sera supprimée.
