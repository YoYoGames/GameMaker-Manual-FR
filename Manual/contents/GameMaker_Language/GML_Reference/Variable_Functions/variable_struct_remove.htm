struct_remove

Cette fonction permet de supprimer une variable d'un struct. Vous devez fournir le struct dont vous souhaitez supprimer la variable et le nom de la variable à supprimer.

 

Syntaxe :

struct_remove(struct, name);

Arguments Type Description
struct Struct The struct reference to remove the variable from
name String The name of the variable to remove (as a string)

 

Les retours :

N/A

 

Exemple :

if struct_exists(mystruct, "shields")
{
    struct_remove(mystruct, "shields");
}

Le code ci-dessus vérifiera si la variable donnée existe dans le struct et si c'est le cas, elle sera supprimée.
