variable_global_set

Cette fonction vous permet de définir la valeur d'une variable globale donnée. Vous fournissez le nom de la variable globale dont vous voulez définir la valeur sous forme de chaîne de caractères (voir l'exemple de code ci-dessous), puis la valeur à définir (peut être n'importe quel type de données valide). Si la variable globale n'existe pas déjà dans le jeu, elle sera créée, puis la valeur lui sera attribuée.

Syntaxe :

variable_global_set(name, val);

Arguments Type Description
name Chaîne de caractères The name of the global variable to set (as a string)
val Variable The value to set the global variable to

Les retours :

N/A

Exemple :

if !variable_global_exists("enemy_num")
{
    variable_global_set("enemy_num", 0);
}

Le code ci-dessus vérifie si une variable globale existe et si elle n'existe pas, elle est créée et mise à 0.
