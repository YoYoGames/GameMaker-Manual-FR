is_struct

Cette fonction vérifie si la valeur fournie est un struct. Elle renvoie true si c'est le cas, sinon elle renvoie false.

Notez que les variables de méthode renvoient également true, et que les instances d'objet renvoient false.

Syntaxe :

is_struct(val);

Arguments Type Description
val Variable The value to check.

Les retours :

Boolean

Exemple :

if is_struct(a)
{
    delete(a);
}

Le code ci-dessus vérifie une variable pour voir s'il s'agit d'un struct, et si la fonction renvoie true, le struct est supprimé.
