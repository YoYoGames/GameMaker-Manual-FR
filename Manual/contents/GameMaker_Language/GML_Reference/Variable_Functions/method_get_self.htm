méthode_get_self

Avec cette fonction, vous pouvez récupérer l'ID de l'instance ou la référence struct qui est le self contexte utilisé lorsque la méthode est appelée. Si la variable n' est pas une méthode, la fonction renverra undefined. Veuillez noter que la fonction peut également renvoyer la constante pointer_null, auquel cas le self actuel est utilisé au moment de l'appel.

Syntaxe :

method_get_self(method);

ArgumentsType Description
method The method variable to check.

Les retours :

Instance ID, Struct, undefined, or pointer_null

Exemple :

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

Le code ci-dessus récupère le contexte personnel pour la variable de méthode donnée et l'affiche dans la console.
