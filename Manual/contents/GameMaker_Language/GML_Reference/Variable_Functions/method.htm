méthode

Cette fonction permet de lier une fonction (ou méthode) existante à l'instance ou à l'adresse struct donnée, en créant une nouvelle variable de méthode qui pourra être utilisée ultérieurement.

Vous fournissez l'ID de l'instance à utiliser (une instance active dans la room, et non un index d'objet) ou une référence struct, ainsi que l'ID de la fonction (ou la référence de la méthode) que vous souhaitez lier. La fonction renverra une nouvelle méthode qui pourra être appelée à partir de la variable à laquelle elle est assignée (voir l'exemple de code ci-dessous).

La méthode retournée sera "liée" à l'instance ou à la structure donnée ( struct), ce qui signifie qu'elle s'exécutera toujours dans la portée de cette instance/structure.

Vous pouvez lier des fonctions intégrées ainsi que des fonctions/méthodes définies par l'utilisateur, et vous pouvez également fournir undefined comme argument d'instance/structure, ce qui signifie que la portée actuelle sera utilisée pour la liaison. self courante sera utilisée pour la liaison.

Syntaxe :

method(struct_ref_or_instance_id, func);

Arguments Type Description
struct_ref_or_instance_id ID destructure ou d'instance The unique reference or ID value of the struct or instance to use (can be self ou undefined)
func Fonction ou méthode descript The ID of the function (or method reference) to use

Les retours :

Method

Exemple :

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
    enemy_func = method(_inst, enemy_ai);
}

Le code ci-dessus vérifie si une instance d'ennemi existe à la position de la souris. Si c'est le cas, la méthode enemy_ai est liée à l'instance de l'ennemi et renvoyée comme nouvelle variable de méthode "enemy_func".
