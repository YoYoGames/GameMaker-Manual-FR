tableau_équivalents

Avec cette fonction, vous pouvez vérifier si deux tableaux sont égaux (équivalents ou identiques). Vous donnez les deux tableaux à vérifier, et la fonction renvoie true s'ils sont égaux (équivalents ou identiques) ou false s'ils ne le sont pas. Notez que ce n' est pas la même chose que de vérifier si deux tableaux sont identiques en utilisant ==, qui ne vérifiera pas si les deux tableaux contiennent des valeurs équivalentes, mais seulement si les tableaux font référence au même tableau initial. Par exemple :

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

Notez que certaines constantes peuvent ne pas être égales à elles-mêmes, ce qui peut faire échouer cette fonction. Voici un exemple :

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

Voir le tableau d'égalité pour plus d'informations.

Syntaxe :

array_equals(var1, var2);

Arguments Type Description
var1 Array The first array.
var2 Array The second array.

Les retours :

Boolean

Exemple :

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

Le code ci-dessus va vérifier deux tableaux pour voir s'ils contiennent des valeurs équivalentes, et si ce n'est pas le cas, le code va copier tout le contenu du tableau "inventory_array" vers le tableau "item_array".
