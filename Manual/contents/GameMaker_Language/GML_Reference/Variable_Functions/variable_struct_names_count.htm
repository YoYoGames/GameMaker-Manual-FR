nombre_de_noms_de_structures_variables

Avec cette fonction, vous pouvez trouver le nombre total de variables définies pour un struct. Vous fournissez l'ID du struct à vérifier, et la fonction retournera une valeur entière pour le nombre de variables rencontrées, ou (si aucun struct de l'ID donné n'existe) -1.

Syntaxe :

variable_struct_names_count(struct_id);

ArgumentsType Description
struct_id The unique ID value of the struct to check.

Les retours :

Exemple :

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

Le code ci-dessus récupère le nombre de variables dans l'adresse struct et affiche un message de débogage dans la sortie console avec cette valeur.
