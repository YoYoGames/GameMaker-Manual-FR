instanceof

Cette fonction peut être utilisée pour obtenir le nom de la fonction constructeur qui a été utilisée pour créer un struct. Le struct lui-même devrait avoir été créé en utilisant l'opérateur new ainsi que le constructeur lui-même. Vous fournissez la variable avec la référence struct à vérifier et la fonction retournera soit une chaîne avec le nom du constructeur, soit undefined.

Notez que si vous passez à la fonction un littéral struct (c'est-à-dire un struct qui a été créé sans utiliser de fonction constructrice), elle renverra simplement la chaîne "struct".

Cette fonction peut aussi être utilisée pour vérifier si une référence struct est une référence faible Dans ce cas, la fonction renvoie la chaîne "weakref" au lieu du nom de la fonction qui a créé la référence struct. Pour plus d'informations, voir la fonction weak_ref_create().

Le nom du constructeur renvoyé comprendra un texte supplémentaire si la fonction constructeur n'a pas été créée dans un script. Si elle a été créée dans un objet, son nom comprendra l'objet et l'événement dans lequel elle a été créée. Par exemple, un constructeur appelé test_constructor créé dans l'événement Create d'Object1 sera renvoyé sous le nom "test_constructor_gml_Object_Object1_Create_0". Pour éviter cela, créez vos constructeurs dans un fichier . script.

 

Syntaxe :

instanceof(struct_id);

Arguments Type Description
struct Struct The struct reference to use.

 

Les retours :

String or undefined

 

Exemple étendu :

Dans cet exemple, nous devons d'abord définir la fonction qui sera utilisée comme constructeur de notre struct. La fonction suivante est définie dans un actif de script :

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Cette fonction peut ensuite être utilisée avec l'opérateur new pour créer un struct et le remplir avec les variables définies aux valeurs des arguments utilisés dans la fonction :

mystruct = new init_struct(10, 100, "Hello World");

Nous pouvons alors obtenir le nom de la fonction constructeur qui a été utilisée comme ceci :

var _name = instanceof(mystruct);
if (is_string(_name))
{
    show_debug_message(_name);
}

Cela imprimera la chaîne "init_struct" dans le journal de sortie, qui est le nom de la fonction constructeur telle que définie dans son script.
