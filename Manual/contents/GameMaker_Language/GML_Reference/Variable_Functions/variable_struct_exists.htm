variable_struct_exists

Avec cette fonction, vous pouvez vérifier si une variable existe ou non au sein du site struct donné. Vous fournissez la référence struct à utiliser ainsi que le nom de la variable à vérifier sous forme de chaîne (voir l'exemple de code ci-dessous). La fonction renverra true si une variable avec le nom donné existe pour le struct et false sinon.

Syntaxe :

variable_struct_exists(struct, name);

Arguments Type Description
struct Structure The struct reference to check
name Chaîne de caractères The name of the struct variable to check for (as a string)

Les retours :

Boolean

Exemple :

if !variable_struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

Le code ci-dessus vérifie si la variable appelée "shields" existe dans le site struct et si ce n'est pas le cas, elle est créée et initialisée à 0.
