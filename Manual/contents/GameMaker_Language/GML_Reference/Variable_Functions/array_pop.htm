tableau_pop

Cette fonction supprime le dernier élément du tableau donné, et renvoie sa valeur. Si le tableau est vide, undefined est retourné.

Syntaxe :

array_pop(array);

ArgumentsType Description
array The array to remove the last value from.

Les retours :

Variable or undefined

Exemple :

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

Le code ci-dessus extrait la dernière valeur du tableau donné, puis l'affiche à l'écran avec du texte. L'entrée récupérée est également retirée du tableau des scores.
