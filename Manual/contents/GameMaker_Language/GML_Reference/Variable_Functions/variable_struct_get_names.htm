struct_get_names

Avec cette fonction, vous pouvez récupérer un tableau rempli des noms de variables d'un struct. Vous passez la référence struct à vérifier, et chaque entrée du tableau sera une chaîne de noms de variables que le struct contient.

 

Syntaxe :

struct_get_names(struct)

Arguments Type Description
struct Struct The struct reference to check.

 

Les retours :

Array (each entry is a string)

 

Exemple :

var str = "";
var array = struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(struct_get(mystruct, array[i]));
    show_debug_message(str);
}

Le code ci-dessus récupère un tableau des noms de variables pour l'adresse struct et les affiche avec leurs valeurs dans la sortie de débogage.
