variable_struct_get

Avec cette fonction, vous pouvez obtenir la valeur d'une variable nommée donnée dans un struct. Vous fournissez la référence struct ainsi que le nom de la variable dont vous voulez obtenir la valeur sous forme de chaîne (voir l'exemple de code ci-dessous). La fonction renvoie la valeur détenue par la variable ou undefined si la variable nommée n'existe pas.

IMPORTANT ! Si la variable que vous obtenez n'existe pas, la fonction renverra le mot clé undefined et vous risquez d'obtenir des erreurs qui empêcheront le jeu de fonctionner, donc en cas de doute, utilisez d'abord la fonction variable_struct_exists() d'abord.

Syntaxe :

variable_struct_get(struct, name);

ArgumentsType Description
struct The struct reference to use
name The name of the variable to get (as a string)

Les retours :

Variable (any data type) or undefined (if the named variable does not exist)

Exemple :

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

Le code ci-dessus vérifie si une variable existe dans le site struct. Si c'est le cas, la valeur qu'elle contient est récupérée et stockée dans une variable locale. Si elle n'existe pas, la variable locale prend la valeur -1.
