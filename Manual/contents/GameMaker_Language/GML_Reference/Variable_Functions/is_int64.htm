est_int64

Cette fonction retourne si une variable donnée est un entier 64bit ou non. Dans certains cas, vous voulez vérifier le type de données d'une variable dans GameMaker et c'est là que vous utiliserez cette fonction. Elle renvoie true ou false selon que la valeur est un int64 ou non.

Syntaxe :

is_int64(n);

Arguments Type Description
n Variable The argument to check.

Les retours :

Boolean

Exemple :

if !is_int64(val)
{
    show_debug_message("Not a 64bit integer!");
}

Le code ci-dessus vérifie la variable "val" pour voir si elle contient un int64 et si ce n'est pas le cas, il affiche un message dans la console de débogage.
