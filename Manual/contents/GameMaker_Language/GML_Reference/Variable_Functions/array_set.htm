tableau_set

Avec cette fonction, vous pouvez définir la valeur d'un index dans un tableau à une valeur. La fonction exige que vous fournissiez une variable qui contient le tableau ainsi que l'indice à définir et la valeur à définir. Cette fonction peut également être utilisée pour des tableaux multidimensionnels, à condition de spécifier quelle dimension vous souhaitez définir lorsque vous fournissez l'indice du tableau, en suivant ce modèle :

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

Syntaxe :

array_set(variable, index, value);

Arguments Type Description
variable Array The variable that holds the array.
index Real The index of the array element to set the value for.
value Variable The value to set.

Les retours :

N/A

Exemple :

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
}

Le code ci-dessus va définir les 10 premiers éléments du tableau donné à une valeur.
