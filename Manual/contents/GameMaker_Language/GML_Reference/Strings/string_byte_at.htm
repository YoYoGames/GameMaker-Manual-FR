string_byte_at

Renvoie la valeur brute de l'octet en tant que valeur réelle à une position donnée dans la chaîne donnée.

Syntaxe :

string_byte_at(str, index);

ArgumentsType Description
str The string to check.
index The position to get the byte from.

Les retours :

Exemple :

newbyte = string_byte_at("Hello World", 5);

Cela donnera à newbyte la valeur brute en octets de la sixième lettre de "Hello World".
