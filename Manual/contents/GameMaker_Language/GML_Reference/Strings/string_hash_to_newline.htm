string_hash_to_newline

Cette fonction renvoie une chaîne de caractères où le symbole "#" a été converti en une nouvelle ligne. Donc une chaîne de caractères formatée comme :

"Hello#World"

Serait imprimé à l'écran comme :

Hello
World

IMPORTANT ! Cette fonction est fournie pour la compatibilité d'importation entre GameMaker et les versions antérieures du logiciel et, en tant que telle, cette fonction ne doit pas être utilisée, sauf si cela est nécessaire pour reproduire une fonctionnalité obsolète. A la place, vous devriez plutôt utiliser les caractères d'échappement pour marquer une nouvelle ligne. Pour plus d'informations sur la compatibilité avec les importations, cliquez ici.

Syntaxe :

string_hash_to_newline(string);

ArgumentsType Description
string The string to convert over multiple lines

Les retours :

Exemple :

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

Le code ci-dessus convertit la chaîne de caractères contenant le symbole de hachage en une chaîne de caractères répartie sur deux lignes, puis l'imprime à l'écran.
