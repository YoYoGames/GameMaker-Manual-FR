chr

Cette fonction renvoie une chaîne contenant le caractère qui se rapporte au code Unicode d'entrée pour l'affichage. Ce caractère dépend de la page de code du jeu de caractères de la police de dessin actuelle et si aucune police n'est définie, la page de code par défaut de la machine sera utilisée. la machine.

Syntaxe :

chr(val);

ArgumentsType Description
val The Unicode code value to get the string from

Les retours :

Exemple :

mystring = chr(53) + chr(48);

Cela donnerait à mystring la valeur "50" (une chaîne de caractères, pas un nombre entier) car chr(53) est "5" et chr(48) est "0".
