chaîne_remplacer_tout

Vous pouvez utiliser cette fonction pour analyser une chaîne de caractères à la recherche de parties spécifiques, qui peuvent ensuite être remplacées par la nouvelle chaîne de caractères que vous avez spécifiée à tous les endroits où elles apparaissent.

Syntaxe :

string_replace_all(str, substr, newstr);

ArgumentsType Description
str The string to be copied.
substr The substring within the string to be replaced in all instances.
newstr The new substring to replace the previous one(s).

Les retours :

Exemple :

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Ainsi, str2 deviendra str1, mais les occurrences de "x" seront remplacées par "l", de sorte que str2 sera "Hello World".
