chaîne_copie

Avec cette fonction, vous pouvez facilement sélectionner un certain nombre de caractères dans une chaîne de caractères pour les copier dans une autre. Le premier caractère d'une chaîne de caractères est toujours indexé à 1 et non à 0, comme on pourrait s'y attendre. Ainsi, pour copier (par exemple) les cinq premiers caractères d'une chaîne de caractères, vous devez utiliser string_copy(str, 1, 5). Un autre exemple est fourni ci-dessous.

Syntaxe :

string_copy(str, index, count);

ArgumentsType Description
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

Les retours :

Exemple :

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

Le code ci-dessus permet au joueur de saisir une chaîne de caractères sur le clavier, qui est ensuite stockée dans la variable name. Cette variable est ensuite vérifiée pour voir si elle dépasse la longueur maximale de 15 caractères et si c'est le cas, la chaîne clavier_string est remplacée par une copie de 15 caractères de la chaîne name. Cela limite effectivement l'entrée du joueur à 15 caractères.
