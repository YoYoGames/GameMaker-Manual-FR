ord

Cette fonction prend une chaîne d'entrée à un seul caractère et renvoie la valeur Unicode (UTF8) de ce caractère. Notez que lorsqu'elle est utilisée avec les fonctions keyboard_check* la chaîne d'entrée ne peut comporter qu'un seul caractère et ne peut être qu'un nombre de 0 à 9 ou un caractère romain majuscule de A à Z.

 

Syntaxe :

ord(string)

Arguments Type Description
string String The string with which to find the Unicode value.

 

Les retours :

Real

 

Exemple :

if keyboard_check(ord("W"))
{
    y -= 4;
}

Cela déplacera l'instance appelante de quatre pixels vers le haut si la touche W est maintenue enfoncée.
