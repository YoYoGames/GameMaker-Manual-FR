nombre de chaînes

Cette fonction renvoie le nombre de fois que la sous-chaîne donnée apparaît dans une chaîne spécifique. De cette façon, vous pouvez vérifier combien de fois une lettre unique ou une phrase est répétée dans une section de texte stocké.

Syntaxe :

string_count(substr, str);

ArgumentsType Description
substr The substring to check the string for.
str The string to check.

Les retours :

Exemple :

str1 = "Hello World";
ocount = string_count( "o", str1 );

La variable ocount sera ainsi définie comme le nombre de "o" dans str1, dans ce cas 2.
