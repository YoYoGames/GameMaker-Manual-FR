chaîne_caractère_at

Vous pouvez utiliser cette fonction pour retourner un caractère spécifique à une position spécifique dans une chaîne de caractères, l'index commençant à 1 pour le premier caractère. Si aucun caractère n'est trouvé ou si la chaîne est plus courte que la valeur d'index donnée, une chaîne vide "" est renvoyée, mais si l'index donné est égal ou inférieur à 0, le premier caractère de la chaîne est renvoyé.

Syntaxe :

string_char_at(str, index);

ArgumentsType Description
str The string to check.
index The position to get the character from.

Les retours :

Exemple :

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Ainsi, str2 sera réglé sur le septième caractère ("H" comptant comme le premier) de la chaîne str1, dans ce cas "W".
