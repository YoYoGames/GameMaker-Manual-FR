string_digits

Vous pouvez utiliser cette fonction pour analyser une chaîne de caractères donnée et en extraire des chiffres. Par exemple, disons que vous avez ce texte - "J'ai 81 ans". Avec cette fonction, vous obtiendrez une chaîne de retour de "81".

Syntaxe :

string_digits(string);

ArgumentsType Description
string The string to get the digits from.

Les retours :

Exemple :

var t_str = string_digits(input_str);
age = real(t_str);

Le code ci-dessus prend la chaîne d'entrée, la débarrasse de tous les caractères autres que les chiffres, puis définit la variable age pour qu'elle contienne la valeur numérique réelle de la chaîne de retour (ainsi, si la chaîne d'entrée était - par exemple - "Je suis 18", la fonction renverra "18").
