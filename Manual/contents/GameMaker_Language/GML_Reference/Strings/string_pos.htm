string_pos

Cette fonction renvoie la position du caractère d'une instance d'une sous-chaîne dans une chaîne, en commençant par le début de la chaîne (pour rechercher à partir de la fin, utilisez la fonction string_last_pos()). La fonction renvoie 0 si elle n'est pas trouvée du tout, ou la position du premier caractère de la sous-chaîne si elle est trouvée. Gardez à l'esprit que les chaînes de caractères sont indexées à partir de 1, donc 1 est la première position dans la chaîne de caractères, et non 0 comme vous pouvez vous y attendre. Cette fonction peut être utilisée pour filtrer les mots qui peuvent être considérés comme offensants ou pour trouver le bon endroit pour insérer un texte dans une autre chaîne.

Syntaxe :

string_pos(substr, str);

ArgumentsType Description
substr The substring to look for in the string.
str The string.

Les retours :

Exemple :

if (string_pos(",", text) != 0)
{
    string_insert(name, text, string_pos(",", text));
}

Le code ci-dessus recherche une virgule dans la chaîne stockée dans la variable "text", et s'il en trouve une, il insère la sous-chaîne "name" à cette position.
