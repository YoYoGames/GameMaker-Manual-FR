string_last_pos

Cette fonction renvoie la position du caractère d'une instance d'une sous-chaîne dans une chaîne, en cherchant de la fin de la chaîne au début (donc l'inverse de string_pos(). La fonction renvoie 0 si la chaîne de recherche n'est pas trouvée, ou la position du premier caractère de la chaîne de recherche si elle est trouvée. Gardez à l'esprit que les chaînes de caractères sont indexées à partir de 1, donc 1 est la première position dans la chaîne de caractères, et non 0 comme vous pouvez vous y attendre.

Syntaxe :

string_last_pos(substr, str);

ArgumentsType Description
substr The substring to look for in the string.
str The string.

Les retours :

Exemple :

if (string_last_pos(",", text) != 0)
{
    string_insert(name, text, string_last_pos(",", text));
}

Le code ci-dessus recherche une virgule dans la chaîne stockée dans la variable "text", en commençant par la fin de la chaîne. S'il en trouve une, il insère la sous-chaîne "nom" à cette position.
