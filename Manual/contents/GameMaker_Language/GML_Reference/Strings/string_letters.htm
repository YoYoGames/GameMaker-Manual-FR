lettres_chiffres

Cette fonction vous permet de supprimer tous les caractères qui ne sont pas classés comme des lettres. C'est pratique pour empêcher les gens de saisir des caractères indésirables dans une entrée de texte (comme saisir "#" pour forcer une nouvelle ligne).

REMARQUE Cette fonction détecte et renvoie uniquement l'alphabet anglais de 26 lettres de A à Z (majuscules ou minuscules).

Syntaxe :

string_letters(string);

ArgumentsType Description
string The string to convert to letters.

Les retours :

Exemple :

username = string_letters(username);

Le code ci-dessus définit la variable "username" pour qu'elle ne contienne que les lettres de la chaîne originale (si la chaîne originale était "MacSw33ny", la chaîne retournée serait "MacSwny").
