string_lettersdigits

Cette fonction renvoie une copie d'une chaîne de caractères donnée dont on a supprimé toutes les lettres et tous les chiffres, ce qui signifie qu'elle peut être utilisée pour supprimer les caractères indésirables (comme "#" ou " ?"), par exemple, d'un nom de connexion ou d'un mot de passe.

REMARQUE Cette fonction ne permet de restituer que les 26 lettres de l'alphabet anglais de A à Z (en majuscules ou en minuscules) et les chiffres de 0 à 9.

Syntaxe :

string_lettersdigits(string);

ArgumentsType Description
string The string to convert to letters and digits.

Les retours :

Exemple :

if (string_length(password) > string_length(string_lettersdigits(password)))
{
    draw_text(32,32, "Invalid Password! Only Letters and numbers please!");
}

Le code ci-dessus vérifie la longueur d'une chaîne de caractères par rapport à la longueur de la même chaîne, mais sans les symboles. Si elles ne sont pas identiques, un message est affiché à l'écran.
