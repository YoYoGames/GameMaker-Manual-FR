format_chaîne

Transforme un nombre réel en une chaîne de caractères utilisant votre propre formatage, où vous pouvez choisir combien de "places" sont enregistrées dans la chaîne et combien de décimales sont également enregistrées. Les deux peuvent être très pratiques, certains jeux préfèrent afficher un score sous la forme d'un nombre défini de chiffres, tandis que le contrôle des décimales peut être utile pour obtenir une grande précision que les deux décimales de la fonction string() Si le nombre de chiffres spécifié est supérieur à la valeur à afficher et/ou si le nombre plus les décimales qui ont été spécifiées est inférieur au nombre total de chiffres, des espaces seront ajoutés avant la valeur pour compenser la différence (voir l'exemple ci-dessous). Des zéros seront ajoutés à droite de la virgule si la valeur indiquée est inférieure au total et au nombre de décimales à inclure. Par défaut, il n'y a pas d'espace supplémentaire à gauche, et seulement deux décimales à droite, par exemple "265,73".

Syntaxe :

string_format(val, tot, dec);

Arguments Type Description
val Real The real number to be turned into a string.
tot Real The total number of places of the main number to be shown. Zeroes or spaces will be inserted to match this.
dec Real The number of decimal places to be included.

Les retours :

String

Exemple :

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

Cela donnera à str1 la valeur " 1234 ", à str2 la valeur " 3.1415926535 " et à str3 la valeur " 3.14159 ".
