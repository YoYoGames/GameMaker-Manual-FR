string_height_ext

Cette fonction renvoie la hauteur (en pixels) de la chaîne de caractères en entrée, en tenant compte de la séparation des lignes et de la largeur du saut de ligne (qui est définie comme le nombre de pixels que la chaîne de caractères peut occuper avant l'insertion d'un saut de ligne). Il est très pratique pour calculer les distances entre les éléments de texte en se basant sur la plus grande des lettres qui composent la chaîne, ainsi que la longueur réelle de la chaîne elle-même lorsqu'elle est répartie sur plusieurs lignes, comme elle serait dessinée avec draw_text_ext() en utilisant la police actuellement définie. La séparation et la largeur peuvent être définies à -1 pour obtenir l'espacement par défaut.

Syntaxe :

string_height_ext(string, sep, w);

ArgumentsType Description
string The string to measure the height of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Les retours :

Exemple :

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100);

Le code ci-dessus obtient la hauteur de la chaîne de caractères donnée, en tenant compte de la largeur de la séparation et du saut de ligne, puis dessine deux lignes de texte en utilisant la hauteur totale de la chaîne de caractères comme séparateur.
