string_byte_length

Cette fonction renvoie le nombre d'octets d'une chaîne de caractères. Comme ils sont conservés en UTF8, ce nombre ne sera pas égal au nombre de caractères de la chaîne.

Syntaxe :

string_byte_length(string);

Arguments Type Description
string Chaîne de caractères The string to measure the number of bytes of.

Les retours :

Real

Exemple :

bytesize = string_byte_length("Hello World");

Ceci définit bytesize comme le nombre d'octets dans "Hello World".

Retour : Cordes
