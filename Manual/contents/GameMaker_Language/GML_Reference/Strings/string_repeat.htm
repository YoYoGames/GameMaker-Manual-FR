chaîne_répétition

Cette fonction renvoie simplement la même chaîne répétée un nombre donné de fois sur elle-même.

Syntaxe :

string_repeat(str, count);

ArgumentsType Description
str The string to repeat.
count The number of times to repeat the string.

Les retours :

Exemple :

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Le code ci-dessus définira str2 comme "Hello WorldHello WorldHello World".
