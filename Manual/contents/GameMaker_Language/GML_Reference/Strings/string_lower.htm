string_lower

Cette fonction permet de forcer une chaîne de caractères à ne contenir que des caractères minuscules.

NOTE Cette fonction ne détecte que l'alphabet anglais de 26 lettres de A à Z.

Syntaxe :

string_lower(string);

ArgumentsType Description
string The string to convert to lowercase.

Les retours :

Exemple :

str1 = "Hello World";
str2 = string_lower(str1);

Le code ci-dessus définira str2 comme "hello world".
