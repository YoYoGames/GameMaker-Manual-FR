string_last_pos_ext

Cette fonction renvoie la position du caractère d'une instance d'une sous-chaîne dans une chaîne de caractères, en effectuant une recherche en arrière dans la chaîne à partir de la position donnée comme position de départ. La fonction renvoie 0 si la chaîne recherchée n'est pas trouvée, ou la position du premier caractère de la chaîne recherchée si elle est trouvée. Gardez à l'esprit que les chaînes de caractères sont indexées à partir de 1, donc 1 est la première position dans la chaîne de caractères, et non 0 comme vous pouvez vous y attendre.

Syntaxe :

string_last_pos_ext(substr, str, start_pos);

ArgumentsType Description
substr The substring to look for in the string.
str The string.
start_pos The starting position to search from.

Les retours :

Exemple :

if (string_last_pos_ext(",", text, 20) != 0)
{
    string_insert(name, text, string_last_pos_ext(",", text, 20));
}

Le code ci-dessus recherche dans la chaîne stockée dans la variable "text" une virgule avant le 20ème caractère, et s'il en trouve une, il insère la sous-chaîne "name" à cette position.
