string_delete

Cette fonction est utilisée pour supprimer une partie de la chaîne de caractères donnée. Vous fournissez la chaîne de caractères d'entrée, la position de départ dans cette chaîne pour supprimer les caractères (l'index commence à 1) et le nombre de caractères à supprimer. La fonction renvoie une nouvelle chaîne sans cette partie.

Syntaxe :

string_delete(str, index, count);

ArgumentsType Description
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

Les retours :

String

Exemple :

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Cela donnera à str2 la valeur "Hello World", car cela supprime le "o" supplémentaire de "Hello" (qui est présent à la position 5).
