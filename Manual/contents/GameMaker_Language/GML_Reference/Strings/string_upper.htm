string_supper

Cette fonction permet de forcer une chaîne de caractères à ne contenir que des majuscules.

NOTE Cette fonction ne détecte que l'alphabet anglais de 26 lettres de A à Z.

Syntaxe :

string_upper(string);

ArgumentsType Description
string The string to convert to uppercase.

Les retours :

Exemple :

str1 = "Hello World";
str2 = string_upper(str1);

Le code ci-dessus donnera à la variable str2 la valeur "HELLO WORLD".
