string_replace

Vous pouvez utiliser cette fonction pour analyser une chaîne de caractères à la recherche d'une partie spécifique, qui peut ensuite être remplacée par la nouvelle chaîne de caractères que vous avez spécifiée.

Syntaxe :

string_replace(str, substr, newstr);

ArgumentsType Description
str The string to be copied.
substr The substring within the string to be replaced.
newstr The new substring to replace the previous one.

Les retours :

Exemple :

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

Ainsi, str2 deviendra str1, mais son instance de "Earth" sera remplacée par "World", de sorte que str2 sera "Hello World".
