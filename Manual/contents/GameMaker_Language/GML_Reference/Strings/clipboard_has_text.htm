presse-papiers_a_texte

Cette fonction renvoie true si le presse-papiers contient du texte ou false dans le cas contraire.

Syntaxe :

clipboard_has_text();

Les retours :

Boolean

Exemple :

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Le code ci-dessus vérifie la présence de texte dans le presse-papiers et s'il en contient, il est lu sous forme de chaîne dans la variable "str". Enfin, le presse-papiers est effacé en lui attribuant la valeur d'une chaîne vide.
