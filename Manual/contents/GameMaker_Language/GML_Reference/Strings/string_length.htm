longueur_chaîne

Cette fonction renvoie le nombre de caractères composant une chaîne de caractères donnée. Elle peut être utile pour déterminer quand limiter la longueur des caractères d'une entrée de texte personnalisée (par exemple, limiter le nom d'un joueur à 10 caractères). N'oubliez pas que ce paramètre est différent de string_width() en ce sens qu'elle mesure le nombre de caractères de la chaîne, et non sa largeur en pixels telle qu'elle est affichée à l'écran.

Syntaxe :

string_length(string);

Arguments Type Description
string Chaîne de caractères The string to measure the number of characters of.

Les retours :

Real

Exemple :

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

Cette opération vérifiera si la chaîne de caractères du nom est supérieure à dix et si c'est le cas, elle copiera et utilisera uniquement les dix premiers caractères.
