steam_activate_overlay

La superposition Steam est un élément de l'interface utilisateur Steam qui peut être activé au-dessus de presque tous les jeux lancés par Steam. Elle permet à l'utilisateur d'accéder à access à sa liste d'amis, au navigateur Web, au chat et à l'achat de DLC dans le jeu. La touche par défaut d'un utilisateur pour access la superposition pendant un jeu est SHIFT+TAB, mais vous pouvez également faire apparaître n'importe quelle page de la superposition en utilisant cette fonction. Elle prend l'une des six constantes énumérées ci-dessous :

Constant Description
ov_friends La page des amis de l'utilisateur actuel
ov_community La page de la communauté pour votre jeu
ov_players La page montrant les autres personnes qui jouent au même jeu ou avec qui vous avez récemment joué.
ov_settings Les paramètres de recouvrement du client Steam
ov_gamegroup Ouvre le navigateur Web de la communauté Steam vers le groupe de jeu officiel pour ce jeu.
ov_achievements La page des réalisations de votre jeu

Syntaxe :

steam_activate_overlay(overlayindex);

Arguments Description
overlayindex L'index de la page de l'interface utilisateur de l'API Steam à afficher (voir les constantes énumérées ci-dessous).

Les retours :

N/A

Exemple :

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay(ov_friends); break;
    case vk_f2: steam_activate_overlay(ov_community); break;
    case vk_f3: steam_activate_overlay(ov_players); break;
    case vk_f4: steam_activate_overlay(ov_settings); break;
    case vk_f5: steam_activate_overlay(ov_gamegroup); break;
    case vk_f6: steam_activate_overlay(ov_achievements); break;
}

Le code ci-dessus interroge la dernière touche du clavier enfoncée et, s'il s'agit d'une des touches de fonction de 1 à 6, il ouvre la page correspondante de la superposition Steam.
