steam_activate_overlay_user

Cette fonction ouvre l'overlay Steam à l'un des dialogues choisis relatifs à l'ID utilisateur donné. Les chaînes de dialogue possibles sont les suivantes :

  • "steamid" - ouvre le navigateur web de la communauté Steam à la page de l'utilisateur ou du groupe.
  • "chat" - ouvre une fenêtre de chat à l'utilisateur spécifié

Notez que les identifiants de Steam peuvent être de grands nombres et que vous devrez peut-être convertir votre valeur d'identifiant en tant que valeur int64() avant de la fournir à la fonction.

Syntaxe :

steam_activate_overlay_user(dialog_name, steamid);

Arguments Description
dialog_name Le dialogue sur lequel ouvrir la superposition (voir ci-dessous).
steamid L'ID de l'utilisateur ou du groupe de Steam à utiliser.

Les retours :

N/A

Exemple :

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
    case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

Le code ci-dessus interroge la dernière touche du clavier enfoncée et s'il s'agit de la touche de fonction 1 ou de la touche de fonction 2, il ouvrira la superposition Steam pour voir le groupe Steam stocké dans la variable globale "GamegroupID", ou il ouvrira la fenêtre de discussion pour discuter avec l'utilisateur stocké dans la variable globale "FriendID".
