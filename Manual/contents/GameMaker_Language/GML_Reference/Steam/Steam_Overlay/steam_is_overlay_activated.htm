steam_is_overlay_activated

Cette fonction peut être utilisée pour savoir si la superposition Steam est active ou non pour l'utilisateur. Si la superposition est active et visible pour l'utilisateur, la fonction renverra true, et si elle ne l'est pas, false. Cette fonction peut être utilisée, par exemple, pour interroger l'API Steam au sujet de la superposition afin de pouvoir mettre votre jeu en pause pendant que la superposition s'affiche.

Syntaxe :

steam_is_overlay_activated();

Les retours :

Boolean

Exemple :

if steam_is_overlay_activated()
{
    global.Pause = true;
}

Le code ci-dessus vérifie si la superposition Steam est active et, si c'est le cas, il définit la variable globale "Pause" sur true.
