steam_activate_overlay_store

Cette fonction vous permet d'ouvrir la superposition Steam sur la page de la boutique d'un jeu afin que les utilisateurs puissent acheter ou télécharger des DLC (par exemple). Vous devez fournir l'App ID unique du jeu ou du DLC que vous obtiendrez du tableau de bord Steam lorsque vous le configurerez.

Syntaxe :

steam_activate_overlay_store(app_id);

Arguments Description
app_id L'identifiant unique de l'application pour votre jeu.

Les retours :

N/A

Exemple :

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_store(global.DLC_id);
}

Le code ci-dessus recherche la touche F1 sur le clavier et si c'est le cas, la superposition Steam sera ouverte sur la page pour le contenu du jeu en utilisant l'identifiant de l'application stocké dans la variable globale.
