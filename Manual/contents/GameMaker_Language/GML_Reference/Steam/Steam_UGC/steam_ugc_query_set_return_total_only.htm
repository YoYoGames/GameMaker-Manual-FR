steam_ugc_query_set_return_total_only

Cette fonction peut être utilisée pour filtrer davantage une requête UGC donnée, notamment pour demander uniquement le nombre de résultats sans aucune autre information (ce qui signifie que la carte DS générée par la fonction d'envoi contiendra la clé "num_results" sans autres données de carte). sans autres données cartographiques). Le handle de la requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, steam_ugc_create_query_user()) et le second argument est soit true, soit faux. La fonction renverra true si le filtre de la requête a été correctement défini, ou false sinon.

Syntaxe :

steam_ugc_query_set_return_total_only(ugc_query_handle , total_only);

Arguments Description
ugc_query_handle La poignée de requête à utiliser.
total_only Si la requête doit retourner uniquement le nombre total de résultats ou non.

Les retours :

Boolean

Exemple :

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_return_total_only(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Le code ci-dessus crée une demande de requête et stocke son identifiant dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent plus précisément la demande de requête avant d'envoyer la requête.
