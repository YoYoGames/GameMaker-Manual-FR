steam_ugc_create_query_all

Cette fonction peut être utilisée pour interroger la base de données UGC en utilisant certains types de requêtes prédéfinis. La fonction requiert les constantes suivantes pour le type de requête à créer, le type d'élément à interroger et le numéro de page à interroger - notez qu'une requête retournera un nombre maximum de 50 éléments.

Type de requête Constant Description
ugc_query_RankedByVote Voir la documentation de Steam.
ugc_recherche_RankedByPublicationDate -
ugc_query_AcceptedForGameRankedByAcceptanceDate -
ugc_query_RankedByTrend -
ugc_query_FavoritedByFriendsRankedByPublicationDate -
ugc_recherche_CreatedByFriendsRankedByPublicationDate -
ugc_query_RankedByNumTimesReported -
ugc_recherche_CreatedByFollowedUsersRankedByPublicationDate -
ugc_query_NotYetRated -
ugc_query_RankedByTotalVotesAsc -
ugc_query_RankedByVotesUp -
ugc_query_RankedByTextSearch (Recherche de texte classé) -

Type de correspondance Constant Description
ugc_match_Items Voir la documentation de Steam.
ugc_match_Items_Mtx -
ugc_match_Items_ReadyToUse -
ugc_match_Collections -
ugc_match_Artwork -
ugc_match_Videos -
ugc_match_Screenshots (captures d'écran) -
ugc_match_AllGuides -
ugc_match_WebGuides -
ugc_match_Guides intégrés -
ugc_match_UsableInGame -
ugc_match_ControllerBindings (liens avec les contrôleurs) -



La fonction renvoie une valeur unique de poignée de requête qui doit être stockée dans une variable pour être utilisée dans les autres fonctions de requête. Notez que cette fonction ne fait que préparer la requête mais ne l'envoie pas réellement - pour cela vous devez appeler la fonction steam_ugc_send_query() - et vous pouvez utiliser d'autres fonctions steam_ugc_query_*() pour affiner la requête de recherche avant qu'elle ne soit réellement envoyée.

Syntaxe :

steam_ugc_create_query_all(query_type, match_type, page);

Arguments Description
query_type Le type de requête à créer (voir les constantes listées ci-dessous).
match_type Le type d'éléments UGC à interroger (voir les constantes énumérées ci-dessous).
page Le numéro de la page à interroger.

Les retours :

Query Handle

Exemple :

query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);

Le code ci-dessus crée une demande de requête générale et stocke son identifiant dans une variable pour une utilisation ultérieure.

Retour : UGC
