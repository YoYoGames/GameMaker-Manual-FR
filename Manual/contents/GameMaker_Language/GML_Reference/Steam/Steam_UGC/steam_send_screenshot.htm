steam_send_screenshot

Cette fonction vous permet de télécharger une capture d'écran sur la page de profil de la communauté Steam de l'utilisateur actuellement connecté. Le nom de fichier que vous fournissez est le nom du fichier local qui a été créé lorsque vous avez réalisé la capture d'écran à l'aide des fonctions GameMaker Studio 2 screen_save() ou screen_save_part(). La largeur et la hauteur définissent la taille de l'image. La fonction renvoie la valeur 0 si elle échoue pour une raison quelconque et une valeur supérieure à 0 si elle réussit.

Syntaxe :

steam_send_screenshot(filename, width, height);

Arguments Description
filename Le nom du fichier image à télécharger (une chaîne).
width La largeur de l'image (un nombre entier).
height La hauteur de l'image (un nombre entier).

Les retours :

Real

Exemple :

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

Le code ci-dessus interroge l'API Steam pour une demande de capture d'écran et, si c'est le cas, un nom unique pour le fichier image est généré, une capture d'écran est effectuée et le fichier est envoyé à la page communautaire Steam pour l'utilisateur.
