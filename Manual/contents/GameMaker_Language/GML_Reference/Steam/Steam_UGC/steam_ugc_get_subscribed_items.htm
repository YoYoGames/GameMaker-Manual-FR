steam_ugc_get_subscribed_items

Cette fonction va remplir une liste DS avec tous les ID de fichiers publiés pour les éléments auxquels l'utilisateur est actuellement abonné. Vous devez d'abord créer la liste et stocker l'index dans une variable, puis la transmettre à la fonction. La fonction renverra true si tout est correct et que l'API Steam est initialisée, ou false en cas d'erreur.

Syntaxe :

steam_ugc_get_subscribed_items(item_list);

Arguments Description
item_list Un index de liste DS (précédemment créé).

Les retours :

Boolean

Exemple :

steam_list = ds_list_create();
steam_ugc_get_subscribed_items(steam_list);

Le code ci-dessus créera une liste DS vide et la remplira ensuite avec les ID de fichiers pour tous les éléments souscrits par l'utilisateur.
