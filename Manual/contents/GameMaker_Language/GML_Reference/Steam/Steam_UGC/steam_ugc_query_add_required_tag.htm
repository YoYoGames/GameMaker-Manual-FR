steam_ugc_query_add_required_tag

Cette fonction peut être utilisée pour filtrer davantage une requête de CGU donnée, en particulier pour rechercher uniquement les éléments de CGU avec la balise donnée. Le handle de la requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, steam_ugc_create_query_user()) et le deuxième argument est une chaîne que vous souhaitez utiliser comme balise à inclure. La fonction renvoie un message vrai si le filtre de la requête a été correctement défini, ou faux dans le cas contraire.

Syntaxe :

steam_ugc_query_add_required_tag(ugc_query_handle , tag_name);

Arguments Description
ugc_query_handle La poignée de requête à utiliser.
tag_name Le nom de la balise à inclure (une chaîne de caractères).

Les retours :

Boolean

Exemple :

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByTrend, ugc_match_Items, 1);
steam_ugc_query_add_required_tag(query_handle, "RPG");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Le code ci-dessus crée une demande de requête et stocke son identifiant dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent plus précisément la demande de requête avant d'envoyer la requête.
