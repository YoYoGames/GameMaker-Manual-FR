steam_ugc_query_set_ranked_by_trend_days

Cette fonction peut être utilisée pour filtrer davantage toute requête UGC effectuée à l'aide de la constante ugc_query_RankedByTrend, notamment pour effectuer une recherche sur un certain nombre de jours. Le handle de la requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, steam_ugc_create_query_user()) et le second argument est le nombre de jours sur lequel vous souhaitez que la requête soit exécutée. La fonction renvoie true si le filtre de la requête a été correctement défini, ou faux dans le cas contraire.

Syntaxe :

steam_ugc_query_set_ranked_by_trend_days(ugc_query_handle , days);

Arguments Description
ugc_query_handle La poignée de requête à utiliser.
days Le nombre de jours à interroger.

Les retours :

Boolean

Exemple :

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByTrend, ugc_match_Items, 1);
steam_ugc_query_set_ranked_by_trend_days(query_handle, 5);
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Le code ci-dessus crée une demande de requête et stocke son identifiant dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent plus précisément la demande de requête avant d'envoyer la requête.
