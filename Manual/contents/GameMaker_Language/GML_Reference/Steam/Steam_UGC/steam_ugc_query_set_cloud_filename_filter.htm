steam_ugc_query_set_cloud_filename_filter

Cette fonction peut être utilisée pour filtrer davantage une requête UGC donnée, en particulier pour vérifier si un nom de fichier d'élément d'atelier doit correspondre ou non. Le handle de la requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, steam_ugc_create_query_user()) et le deuxième argument est soit true (vrai) soit false (faux), selon que vous exigez ou non que les noms de fichiers correspondent. La fonction retournera true si le filtre de la requête a été correctement défini, ou false sinon.

Syntaxe :

steam_ugc_query_set_cloud_filename_filter(ugc_query_handle , match_cloud_filename);

Arguments Description
ugc_query_handle La poignée de requête à utiliser.
match_cloud_filename Définit si le nom du fichier de l'élément UGC doit correspondre ou non.

Les retours :

Boolean

Exemple :

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_cloud_filename_filter(query_handle, true);
steam_ugc_query_add_excluded_tag(query_handle, "nasty chips");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Le code ci-dessus crée une demande de requête et stocke son identifiant dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent plus précisément la demande de requête avant d'envoyer la requête.

Retour : UGC
