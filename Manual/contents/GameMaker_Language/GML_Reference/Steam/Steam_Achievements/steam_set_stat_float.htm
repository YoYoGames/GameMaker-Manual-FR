steam_set_stat_float

Cette fonction vous permet de définir une statistique spécifique à une nouvelle valeur à virgule flottante. La statistique doit avoir été définie au préalable sur la page des comptes du panneau de configuration Steamworks pour votre jeu et la chaîne de caractères transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de contrôle. Cette fonction peut être utilisée, par exemple, pour déterminer la distance parcourue par le joueur ou le pourcentage du jeu terminé.

Syntaxe :

steam_set_stat_float(stat_name, value);

Arguments Description
stat_name Le nom de la statistique à définir (une chaîne de caractères).
value La valeur (un flottant 32bit) à laquelle il faut fixer la stat.

Les retours :

N/A

Exemple :

var dist_pc = (dist / dist_max) * 100;
steam_set_stat_float("Travelled", dist_pc);

Le code ci-dessus calcule un pourcentage sur la base de la variable de distance parcourue "dist" et de la distance maximale que vous pouvez parcourir "dist_max", puis définit la statistique "Travelled" à la nouvelle valeur.
