steam_set_achievement

Cette fonction permet d'indiquer à l'API Steam d'attribuer (" définir ") un succès au joueur. Ces réalisations doivent avoir été définies sur la page des comptes du panneau de configuration Steamworks pour votre jeu et la chaîne transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de configuration. La superposition de jeu Steam affichera un panneau de notification à l'utilisateur l'informant de la réalisation qu'il a reçue, sauf si la réalisation a déjà été attribuée, auquel cas rien ne se passera.

Syntaxe :

steam_set_achievement(ach_name);

Arguments Description
ach_name Le nom de la réalisation à définir (chaîne de caractères).

Les retours :

N/A

Exemple :

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

Le code ci-dessus récompensera le joueur d'un succès si la variable globale "Deaths" est égale à 10 et si le succès n'a pas déjà été attribué.
