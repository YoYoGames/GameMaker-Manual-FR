steam_langue_du_jeu_actuel

Cette fonction est utilisée pour récupérer la langue actuelle utilisée par Steam (sous forme de chaîne de caractères), par exemple "english".

Syntaxe :

steam_current_game_language();

Les retours :

String

Exemple :

language = steam_current_game_language();

Le code ci-dessus récupère la langue utilisée pour le jeu en cours.
