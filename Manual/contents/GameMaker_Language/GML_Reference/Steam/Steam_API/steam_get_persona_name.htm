steam_get_persona_name

Vous pouvez utiliser cette fonction pour retourner le nom d'utilisateur de l'utilisateur actuellement connecté au client Steam. Il s'agit du nom d'écran visible et non de l'identifiant unique de l'utilisateur (qui peut être trouvé en utilisant la fonction steam_get_user_steam_id()).

Syntaxe :

steam_get_persona_name();

Les retours :

String

Exemple :

if steam_initialised()
{
    global.p_name = steam_get_persona_name();
}

Le code ci-dessus définira une variable globale pour le nom d'écran de l'utilisateur actuel si l'API client Steam est correctement initialisée.
