steam_stats_ready

Lorsque vous utilisez l'API Steam, cette fonction peut être appelée pour vérifier que l'API client Steam a correctement initialisé les statistiques de votre jeu.

Syntaxe :

steam_stats_ready();

Les retours :

Boolean

Exemple :

global.steam_api = false;
if steam_initialised()
{
    if steam_stats_ready() && steam_is_overlay_enabled()
    {
        global.steam_api = true;
    }
}

Le code ci-dessus attribuera la valeur true à une variable globale si l'API client Steam est correctement initialisée, ainsi que la fonctionnalité de statistiques et de superposition Steam, ou la valeur false dans le cas contraire.
