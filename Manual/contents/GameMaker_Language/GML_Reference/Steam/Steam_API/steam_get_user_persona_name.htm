steam_get_user_persona_name

Cette fonction peut être utilisée pour récupérer le nom d'utilisateur (nom d'écran) pour un ID utilisateur spécifique. Vous donnez l'ID de l'utilisateur et la fonction renvoie une valeur d'ID asynchrone unique qui peut être utilisée pour vérifier les détails correctement dans l'événement Steam Async. Cet événement aura la async_load DS map qui sera remplie avec les paires clé/valeur suivantes :

  • "id" - L'ID asynchrone renvoyé par la fonction qui a déclenché l'événement.
  • "event_type" - La chaîne "user_persona_name".
  • "steamid"" - La valeur de l'ID Steam utilisée pour obtenir le nom de l'utilisateur.
  • "persona_name" - Le nom d'utilisateur pour l'ID donné de Steam.

Syntaxe :

steam_get_user_persona_name(steamID);

Arguments Description
steamID L'identifiant unique Steam d'un utilisateur.

Les retours :

Real

Exemple :

request = steam_get_user_persona_name(global.UGC_UserID);

Le code ci-dessus demandera le nom d'utilisateur de l'ID utilisateur stocké dans la variable globale "UGC_UserID", en stockant la valeur retournée dans une variable pour l'analyse dans l'événement asynchrone.
