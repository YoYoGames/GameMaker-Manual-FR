steam_available_languages

Cette fonction peut être utilisée pour récupérer une liste de toutes les langues disponibles sur Steam. La valeur renvoyée est simplement une liste de langues séparées par des virgules.

Syntaxe :

steam_available_languages();

Les retours :

String

Exemple :

language = steam_available_languages();

La commande ci-dessus récupère les langues disponibles pour Steam sous forme de chaîne de caractères et les stocke dans une variable.
