steam_get_app_id

Cette fonction est utilisée pour récupérer l'identifiant unique de l'application que Steam attribue à votre jeu, qui est nécessaire pour utiliser certaines des fonctions de contenu généré par l'utilisateur.

Syntaxe :

steam_get_app_id();

Les retours :

Real

Exemple :

global.app_id = steam_get_app_id();

Le code ci-dessus récupère l'ID d'application unique de votre jeu sur Steam et le stocke dans une variable globale.
