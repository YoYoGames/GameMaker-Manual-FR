Général Steam API

Les fonctions suivantes servent toutes à vérifier la disponibilité de certains aspects de l'API client ou serveur de Steam, et renvoient soit vrai soit faux en fonction de facteurs tels que l'occupation du serveur ou la connexion de l'utilisateur, etc... Cela signifie que ces fonctions doivent être utilisées avant tout autre appel de fonction de l'API Steam pour s'assurer que la configuration client/serveur est correcte et communique avec votre jeu :

  • steam_initialised
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
