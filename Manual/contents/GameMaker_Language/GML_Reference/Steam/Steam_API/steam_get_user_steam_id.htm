steam_get_user_steam_id

Vous pouvez utiliser cette fonction pour retourner l'identifiant unique Steam de l'utilisateur actuellement connecté au client Steam. Il s'agit d'une valeur unique de 64 bits qui peut être utilisée pour identifier les utilisateurs individuels et non leur nom d'utilisateur à l'écran (celui-ci peut être trouvé en utilisant la fonction steam_get_persona_name()). Comme il s'agit d'un int64, vous devrez peut-être le convertir en tant que tel lors de son utilisation dans des fonctions ultérieures utilisant la fonction int64() fonction.

Syntaxe :

steam_get_user_steam_id();

Les retours :

Integer (int64)

Exemple :

if steam_initialised()
{
    global.u_id = steam_get_user_steam_id();
}

Si l'API client de Steam est correctement initialisée, le code ci-dessus définira une variable globale avec l'ID unique de l'utilisateur actuel de Steam.
