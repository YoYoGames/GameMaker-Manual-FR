steam_user_owns_dlc

Si votre jeu possède un DLC créé pour lui, vous pouvez utiliser cette fonction pour vérifier si l'utilisateur l'a acheté avant d'accéder aux fichiers qui lui sont associés. La fonction renvoie true (1) si le joueur possède le contenu, false (0) s'il ne le possède pas ou si l'ID DLC donné n'est pas valide, ou -1 s'il n'est pas connecté à Steam.

Notez que même si l'utilisateur possède le DLC, il se peut qu'il ne soit pas installé, vous devez donc utiliser la fonction steam_user_installed_dlc() pour vous assurer qu'il est installé avant de l'utiliser. Notez que les ID de Steam peuvent être de grands nombres et que vous devrez peut-être convertir votre valeur d'ID en tant que int64() avant de la fournir à la fonction.

Syntaxe :

steam_user_owns_dlc(dlc_id);

Arguments Description
dlc_id L'identifiant unique du DLC à vérifier (un entier int64).

Les retours :

Integer

Exemple :

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

Le code ci-dessus vérifie si l'utilisateur a acheté et installé le DLC avec l'identifiant 10354, et si c'est le cas, il définit une variable globale à une valeur différente.
