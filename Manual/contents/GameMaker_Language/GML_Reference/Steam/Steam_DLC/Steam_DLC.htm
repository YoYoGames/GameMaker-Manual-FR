DLC

Steam prend en charge le contenu téléchargeable (DLC) gratuit et payant. Dans le client Steam, un jeu avec contenu téléchargeable apparaît comme une seule application dans la liste des jeux de l'utilisateur, le contenu téléchargeable étant visible dans la boîte de dialogue des propriétés du jeu. Une fois possédé, le contenu téléchargeable est traité comme une partie intégrante du jeu et Steam met automatiquement à jour le contenu lorsqu'un patch est disponible et installe le contenu lorsque l'utilisateur installe le jeu.

Étant donné que tout est géré par les serveurs Steam et que la configuration de tout DLC se fait via le panneau de configuration Steamworks, seules quelques fonctions sont nécessaires dans GameMaker Studio 2 pour vérifier la présence de ce contenu supplémentaire :

  • steam_user_owns_dlc
  • steam_user_installed_dlc
