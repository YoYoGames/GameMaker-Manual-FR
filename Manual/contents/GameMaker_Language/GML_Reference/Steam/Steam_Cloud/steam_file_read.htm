steam_file_read

Cette fonction lira le contenu du fichier donné dans une chaîne de caractères qui pourra ensuite être analysée dans votre jeu.

Syntaxe :

steam_file_read(filename);

Arguments Description
filename Le nom du fichier à lire.

Les retours :

String

Exemple :

if steam_file_exists("Save.txt")
{
    save_str = steam_file_read("Save.txt");
}

Le code ci-dessus vérifie si un fichier existe sur le nuage Steam et si c'est le cas, il l'ouvre et lit son contenu dans la variable "save_str".
