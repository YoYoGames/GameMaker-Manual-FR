steam_is_cloud_enabled_for_account

Avec cette fonction, vous pouvez vérifier que le service Steam Cloud est activé par l'utilisateur dans les paramètres de son client Steam. Elle renvoie true si c'est le cas et false sinon. Notez que cela ne signifie pas automatiquement que vous pouvez stocker des données dans le nuage, car le service doit également être activé pour votre jeu (vous pouvez le vérifier à l'aide de la fonction steam_is_cloud_enabled_for_app()). Si le nuage Steam est activé pour votre jeu, mais que l'utilisateur l'a désactivé localement, vous pouvez toujours utiliser les fonctions du nuage pour stocker et récupérer des données à partir d'une copie locale de tous les fichiers, mais ils ne seront pas téléchargés vers le nuage à la fin du jeu, ni synchronisés au début du jeu.

Syntaxe :

steam_is_cloud_enabled_for_account();

Les retours :

Boolean

Exemple :

if steam_is_cloud_enabled_for_account
{
    steam_file_share("Save.txt");
}

Le code ci-dessus vérifie si l'utilisateur a activé le cloud Steam et s'il renvoie la réponse vraie, il synchronisera alors le fichier donné.
