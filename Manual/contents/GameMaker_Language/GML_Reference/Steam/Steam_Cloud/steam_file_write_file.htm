steam_file_write_file

Cette fonction vous permet de copier le contenu d'un fichier enregistré localement vers un fichier synchronisé avec le cloud Steam. Le fichier local doit exister avant d'utiliser cette fonction, qui renvoie une valeur de 0 si elle échoue pour une raison quelconque et une valeur supérieure à 0 si elle réussit. une valeur supérieure à 0 si elle réussit.

Syntaxe :

steam_file_write_file(steam_filename, local_filename);

Arguments Description
steam_filename Le fichier Steam Cloud à copier.
local_filename Le fichier local à utiliser pour la copie.

Les retours :

real

Exemple :

steam_file_write_file("rm_koala.png", "Koala2.png");

Le code ci-dessus copiera le contenu du fichier "Koala2.png" dans le fichier "rm_koala.png" du nuage Steam.
