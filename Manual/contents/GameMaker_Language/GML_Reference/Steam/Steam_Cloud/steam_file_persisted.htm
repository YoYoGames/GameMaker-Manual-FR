steam_file_persisted

Avec cette fonction, vous pouvez vérifier le fichier donné pour voir s'il a été synchronisé avec le cloud Steam. Une valeur de retour de true signifie qu'il l'est, tandis que false signifie qu'il ne l'est pas.

Syntaxe :

steam_file_persisted(filename);

Arguments Description
filename Le nom du fichier à vérifier.

Les retours :

Boolean

Exemple :

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

Le code ci-dessus vérifiera si un fichier a été stocké dans le nuage Steam, et s'il ne l'a pas été, il le synchronisera.
