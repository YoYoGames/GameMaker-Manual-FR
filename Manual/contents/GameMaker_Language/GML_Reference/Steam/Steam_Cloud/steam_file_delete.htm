steam_file_delete

Cette fonction supprime le fichier donné du nuage Steam. La fonction renvoie une valeur de 0 si elle échoue pour une raison quelconque et une valeur supérieure à 0 si elle réussit.

Syntaxe :

steam_file_delete(filename);

Arguments Description
filename Le nom du fichier à supprimer.

Les retours :

Real

Exemple :

if steam_file_exists("Save.txt")
{
    steam_file_delete("Save.txt");
}

Le code ci-dessus vérifie si un fichier existe, et si c'est le cas, il supprime le fichier du nuage Steam.
