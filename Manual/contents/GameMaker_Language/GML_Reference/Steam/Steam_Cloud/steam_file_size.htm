taille_fichier_vapeur

Cette fonction vous permet de vérifier la taille d'un fichier stocké sur le cloud Steam. Le nombre réel renvoyé est la taille, en octets, du fichier.

Syntaxe :

steam_file_size(filename);

Arguments Description
filename Le nom du fichier dont il faut vérifier la taille.

Les retours :

Real

Exemple :

file_bytes = steam_file_size("Save.txt");

Le code ci-dessus stocke la taille d'un fichier provenant du nuage Steam dans la variable "file_bytes".
