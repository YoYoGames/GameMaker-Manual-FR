physics_fixture_set_chain_shape

Cette fonction définit une forme de fixation "en chaîne". Un dispositif en chaîne est composé d'un certain nombre de points, qui sont ensuite reliés entre eux à l'aide de formes de bord pour les joindre. La fonction elle-même prend l'index (ID) du dispositif et peut indiquer au dispositif de former une boucle ou non. Si vous lui attribuez la valeur true, le dernier point sera relié au premier par une arête, tandis que si vous lui attribuez la valeur false, il ne le sera pas. Notez que cette fonction ne prépare le dispositif à accepter les points nécessaires à la réalisation de la chaîne, et ceux-ci doivent être ajoutés après avoir appelé cette fonction en utilisant physics_fixture_add_point()comme vous le feriez lors de la construction d'un gabarit de polygone.

Lorsque vous créez un montage en chaîne, vous devez lui donner au moins deux points, mais vous n'êtes pas limité dans le nombre de points ultérieurs que vous pouvez ajouter pour constituer le montage fini.

Syntaxe :

physics_fixture_set_chain_shape(fixture, loop)

ArgumentsType Description
fixture The index of the fixture
loop Whether to close the chain (true) ou non (false).

Les retours :

Exemple :

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(fix, xx, yy);
    xx += 50 + random(150);
    yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

Le code ci-dessus créera une ligne de luminaires "de bord" avec une variété de hauteurs sur la longueur de la pièce.
