physics_fixture_set_angular_damping

Si vous pensez à n'importe quel objet en rotation dans le "monde réel", à moins qu'il ne soit équipé d'un moteur ou qu'il se trouve dans l'espace, il ralentit au fil du temps sous l'influence de forces externes (comme la friction avec l'air qui l'entoure). Nous pouvons utiliser la fonction physics_fixture_set_angular_damping() dans GameMaker pour simuler cet effet et réduire la vitesse de rotation des instances dans le monde physique, car, sans cela, toute instance en rotation continuerait à tourner à l'infini. Les paramètres d'amortissement doivent être compris entre 0 et l'infini, 0 signifiant aucun amortissement, et l'infini un amortissement total. Normalement, vous utiliserez une valeur d'amortissement comprise entre 0 et 1, mais vous pouvez utiliser toute valeur non négative si nécessaire.

Syntaxe :

physics_fixture_set_angular_damping(fixture, damping)

ArgumentsType Description
fixture the index of the fixture
damping the angular damping of the fixture, usually between 0 and 1

Les retours :

Exemple :

physics_fixture_set_angular_damping(fix_Ball, 0.1);

Le code ci-dessus fixera l'amortissement angulaire de l'accessoire indexé dans "fix_ball" à 0.1.
