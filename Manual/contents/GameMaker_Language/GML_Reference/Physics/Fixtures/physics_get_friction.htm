physique_get_friction

Lorsque vous liez un dispositif à une instance à l'aide de la fonction physics_fixture_bind() cela renvoie un "id" pour le dispositif lié. Vous pouvez utiliser cet identifiant pour obtenir la valeur de friction du projecteur lié(et non du projecteur "de base") à tout moment en utilisant cette fonction.

Syntaxe :

physics_get_friction(fixture)

Arguments Type Description
fixture ID du dispositif de physique the id of the bound fixture

Les retours :

Real

Exemple :

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Le code ci-dessus obtient la valeur de friction actuelle pour les propriétés physiques liées de l'instance, puis les définit à une valeur différente.
