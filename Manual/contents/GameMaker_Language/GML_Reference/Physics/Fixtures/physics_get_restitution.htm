physique_get_restitution

Lorsque vous liez un dispositif à une instance à l'aide de la fonction physics_fixture_bind() cela renvoie un "id" pour le dispositif lié. Vous pouvez utiliser cet id pour obtenir la restitution (la propriété "bounciness") du projecteur lié, et non du projecteur "de base", à tout moment en utilisant cette fonction.

Syntaxe :

physics_get_restitution(fixture)

ArgumentsType Description
fixture the id of the bound fixture

Les retours :

Exemple :

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Le code ci-dessus obtient la valeur de restitution actuelle pour les propriétés physiques liées de l'instance, puis les définit à une valeur différente.
