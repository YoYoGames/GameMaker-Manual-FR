physics_set_friction

Lorsque vous liez un dispositif à une instance à l'aide de la fonction physics_fixture_bind() cela renvoie un "id" pour le dispositif lié. Vous pouvez utiliser cet identifiant pour définir la valeur de friction du dispositif lié, et non du dispositif "de base", à tout moment en utilisant cette fonction. Notez que la friction est généralement fixée à une valeur comprise entre 0 et 1, mais vous pouvez utiliser toute valeur non négative si nécessaire.

Syntaxe :

physics_set_friction(fixture, friction)

ArgumentsType Description
fixture the id of the bound fixture
friction the new friction value to apply

Les retours :

Exemple :

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Le code ci-dessus récupère la valeur de friction actuelle pour les propriétés physiques liées de l'instance, puis les définit à une valeur différente.
