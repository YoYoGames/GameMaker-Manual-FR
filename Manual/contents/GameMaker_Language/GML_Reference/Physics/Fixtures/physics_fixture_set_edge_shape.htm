physics_fixture_set_edge_shape

Cette fonction définit une forme d'appareil "bord". Une forme de bord est simplement une ligne qui générera une collision lorsque d'autres fixtures la chevaucheront, et peut être très utile pour générer (par exemple) un terrain, ou pour créer des frontières autour d'une pièce. La position de l'arête est définie en utilisant l'espace local, c'est-à-dire que la position x/y de l'instance est considérée comme (0,0), ce qui doit être pris en compte lors de sa création (dans l'exemple de code ci-dessous, l'instance aurait été placée à (0,0) dans la pièce pour éviter les complications).

Syntaxe :

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

ArgumentsType Description
fixture the index of the fixture
local_x1 start x position for the edge
local_y1 start y position for the edge
local_x2 end x position for the edge
local_y2 end y position for the edge

Les retours :

Exemple :

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
    var fix = physics_fixture_create();
    physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
    physics_fixture_bind(fix, id);
    physics_fixture_delete(fix);
    xx += 50;
    y1 = y2;
    y2 = room_height - 50 - irandom(100);
}

Le code ci-dessus créera une ligne de luminaires "de bord" avec une variété de hauteurs sur la longueur de la pièce.
