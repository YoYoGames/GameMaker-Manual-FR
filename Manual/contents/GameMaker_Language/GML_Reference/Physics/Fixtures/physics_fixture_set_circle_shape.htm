physics_fixture_set_circle_shape

Cette fonction définit une forme de cercle pour votre appareil avec un rayon défini par l'argument "rad".

Syntaxe :

physics_fixture_set_circle_shape(fixture, rad)

ArgumentsType Description
fixture the index of the fixture
rad radius of the circle

Les retours :

Exemple :

physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2);

Le code ci-dessus appliquera une forme de cercle à l'appareil indexé dans la variable "fix_Ball" avec un rayon identique à celui de la largeur de l'adresse sprite "spr_Ball" divisée par 2.
