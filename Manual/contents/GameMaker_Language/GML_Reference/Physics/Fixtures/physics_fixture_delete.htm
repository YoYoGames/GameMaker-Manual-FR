physique_fixture_delete

Puisqu'une fois qu'un dispositif est lié à une instance, cette instance conserve les propriétés du dispositif, nous pouvons supprimer le dispositif de la mémoire avec cette fonction, en particulier si nous ne prévoyons plus de l'utiliser. Veuillez noter que si vous ne supprimez pas les fixtures une fois qu'ils ne sont plus nécessaires peut provoquer une fuite de mémoire qui ralentira et finira par faire planter votre jeu. Vous devez donc prendre soin de nettoyer correctement après avoir défini et lié les fixtures.

Syntaxe :

physics_fixture_delete(fixture)

ArgumentsType Description
fixture the fixture that is to be deleted from memory

Les retours :

Exemple :

var fix, inst;
fix = physics_fixture_create();
physics_fixture_set_circle_shape(fix, 16);
physics_fixture_set_density(fix, 1.0);
inst = instance_create_layer(x, y, "Instances", genericBodyObject);
physics_fixture_bind(fix, inst);
physics_fixture_delete(fix);

Le code ci-dessus va créer un accessoire et affecter son index à la variable "fix". Il définit ensuite la forme et la densité du dispositif avant de le lier à l'instance qui a été créée avec l'index stocké dans la variable "inst". Enfin, le dispositif est supprimé pour éviter les fuites de mémoire, car il n'est plus nécessaire.
