physics_fixture_add_point

Cette fonction définit les points d'un polygone qui a été préalablement fixé à l'aide de la fonction physics_fixture_set_polygon_shape() ou bien elle peut être utilisée pour ajouter des points à une chaîne de gabarits d'arêtes. en utilisant la fonction physics_fixture_set_chain_shape(). Pour une forme polygonale, les points sont relatifs à l'origine du support, la position 0,0 de l'axe x/y étant le centre, et les coordonnées suivantes sont calculées comme d'habitude dans GameMaker, le bas/la droite étant +x,+y et le haut/la gauche étant -x,-y. Voici une illustration pour vous aider à visualiser ce processus :

Comme vous pouvez le constater, n'importe quel polygone convexe simple peut être modélisé de cette façon. les polygones concaves ne sont pas pris en charge pour le moment. Enfin, pour fermer le polygone, vous devez lier le dispositif à une instance. Vous devez également noter que vous devez avoir défini au moins trois points pour votre polygone avant de le lier à une instance. Vous devez également noter que vous devez avoir défini au moins trois points pour votre polygone avant de le lier à une instance, sinon vous obtiendrez une erreur et que l'ajout d'un point à un repère qui n'a pas de forme de polygone définie au préalable entraînera également une erreur. Notez également que le nombre maximum de points autorisés pour un repère est de huit.

Pour les formes en chaîne (qui sont simplement une série de formes d'arêtes connectées), cette fonction définira un point comme "connexion" pour enchaîner les arêtes ensemble. Vous devez donc avoir défini au moins deux points avant de lier le dispositif à l'instance sinon vous obtiendrez une erreur. Cependant, contrairement aux polygones, vous n'êtes pas limité à un maximum de huit points et vous pouvez en ajouter autant que vous le souhaitez pour créer la chaîne et les points sont définis dans les coordonnées réelles de la pièce, plutôt que par rapport à l instance.

Syntaxe :

physics_fixture_add_point(fixture, xpos, ypos)

ArgumentsType Description
fixture the index of the fixture
xpos the x position of the point (relative to the fixture origin for polygon shapes)
ypos the y position of the point (relative to the fixture origin for polygon shapes)

Les retours :

Exemple :

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

Le code ci-dessus applique une forme polygonale au dispositif indexé dans la variable "fix_Ship", puis définit trois points pour créer une forme triangulaire.
