physique_fixture_set_friction

Le frottement est la force qui s'oppose au mouvement relatif d'éléments matériels glissant l'un contre l'autre, ce qui, dans le monde physique de GameMaker, se traduit par la perte d'élan causée par la collision de deux instances auxquelles sont liés des accessoires. qui leur sont liés. Ainsi, lorsque deux instances entrent en collision, leur mouvement est affecté par cette valeur, une friction élevée entraînant une perte d'élan plus importante qu'une valeur plus faible. Notez que la friction est généralement définie sur une valeur comprise entre 0 et 1, mais vous pouvez utiliser toute valeur non négative si nécessaire. valeur non négative si nécessaire.

Syntaxe :

physics_fixture_set_friction(fixture, friction)

ArgumentsType Description
fixture the index of the fixture
friction the friction of the fixture

Les retours :

Exemple :

physics_fixture_set_friction(fix_Brick, 0.1);

Le code ci-dessus fixera le frottement de l'accessoire indexé dans "fix_brick" à 0.1.
