physics_remove_fixture

Cette fonction supprime (ou "dé-liaison") un dispositif d'une ou plusieurs instances. Elle requiert l'"id" unique du dispositif lié (tel que retourné par la fonction physics_fixture_bind() et elle supprimera toutes les propriétés physiques actuellement définies pour l'instance, ce qui vous permettra de redéfinir un nouveau dispositif et de le lier à l'instance. De cette façon, vous pouvez modifier les propriétés physiques des instances sans avoir à les détruire et à les recréer.

Syntaxe :

physics_remove_fixture(id, fixture)

Arguments Type Description
id ID de l'instance The ID of the instance to remove the fixture from
fixture ID du dispositif de physique The ID of the fixture that is to be removed from the instance

Les retours :

N/A

Exemple :

physics_remove_fixture(id, my_fix);

Le code ci-dessus supprimera de l'instance le fixture avec le "id" stocké dans la variable "my_fix".
