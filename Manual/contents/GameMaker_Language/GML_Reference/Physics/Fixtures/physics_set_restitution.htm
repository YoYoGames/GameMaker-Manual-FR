physics_set_restitution

Lorsque vous liez un dispositif à une instance à l'aide de la fonction physics_fixture_bind() cela renvoie un "id" pour le dispositif lié. Vous pouvez utiliser cet identifiant pour définir la valeur de restitution du projecteur lié, et non du projecteur "de base", à tout moment en utilisant cette fonction. La restitution est généralement définie comme une valeur comprise entre 0 et 1, mais vous pouvez utiliser des valeurs plus élevées si nécessaire, bien que les résultats puissent être imprévisibles.

Syntaxe :

physics_set_restitution(fixture, restitution)

ArgumentsType Description
fixture the id of the bound fixture
restitution the new restitution value to apply

Les retours :

Exemple :

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Le code ci-dessus obtient la valeur de restitution actuelle pour les propriétés physiques liées de l'instance, puis les définit à une valeur différente.
