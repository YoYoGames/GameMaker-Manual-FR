physics_apply_angular_impulse

Cette fonction donne une impulsion angulaire à une instance activée par la physique. Cette impulsion déterminera la rotation angulaire de la quantité donnée, en ignorant le couple actuel, et déterminera essentiellement la quantité de "rotation" d'un appareil. Si vous souhaitez appliquer une force angulaire à une instance en utilisant le couple, vous devez utiliser la fonction physics_apply_torque().

Syntaxe :

physics_apply_angular_impulse(impulse)

ArgumentsType Description
impulse The impulse (in Newton metres) to be applied

Les retours :

Exemple :

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

Le code ci-dessus donnera une impulsion angulaire à l'appareil activé par la physique si une touche est pressée.
