phy_collision_points

Cette variable en lecture seule renvoie le nombre de points de collision détectés entre les deux objets dans la collision.

REMARQUE: Cette variable n'est disponible que dans l'événement de collision d'une instance activée pour la physique.

Syntaxe :

phy_collision_points;

Les retours :

(integer value, or undefined if the instance is not physics enabled)

Exemple :

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5);
}

Le code ci-dessus crée des particules à tous les points définis d'une collision entre deux instances activées par la physique.
