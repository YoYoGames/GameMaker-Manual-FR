phy_bullet

Cette variable définit si l'instance se déplace extrêmement vite ou non (par exemple une balle). La valeur par défaut est false mais si elle est définie sur true, cela indique à GameMaker que l'instance se déplacera à une telle vitesse qu'elle nécessitera une détection de collision plus coûteuse afin de s'assurer qu'elle ne passe pas à travers d'autres instances sans être détectée.

Syntaxe :

phy_bullet

Les retours :

(or undefined if the instance is not physics enabled)

Exemple :

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

Le code ci-dessus crée une nouvelle instance, puis la définit comme étant une "balle" dans le monde de la physique avant de lui donner une impulsion massive le long de l'axe y.
