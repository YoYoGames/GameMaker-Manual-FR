phy_com_x

Cette variable en lecture seule retournera la position x du centre de masse de l'instance. Cette position est calculée automatiquement en fonction de la densité, de l'inertie et de la masse de l'instance, telles que définies par les fonctions appropriées.

Syntaxe :

phy_com_x

Les retours :

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemple :

xx = phy_com_x;
yy = phy_com_y;

Le code ci-dessus définit deux variables pour la position x et y du centre de masse des instances.
