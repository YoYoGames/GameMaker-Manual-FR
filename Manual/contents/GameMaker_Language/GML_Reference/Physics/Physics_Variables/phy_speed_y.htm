phy_speed_y

Cette variable peut être utilisée pour obtenir ou modifier la composante y du vecteur vitesse linéaire de l'instance et est définie en pixels par pas (pour les pixels par seconde, voir phy_linear_velocity_y). Modifier cette variable pour une instance statique (c'est-à-dire une instance avec une densité de 0) la transformera en une instance cinématique.

Syntaxe :

phy_speed_y

Les retours :

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemple :

if phy_speed_y != 0
{
    phy_speed_y = 0;
}

Le code ci-dessus vérifiera la composante y du vecteur vitesse linéaire et si elle est différente de 0, il la mettra à 0.
