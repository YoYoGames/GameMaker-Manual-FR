phy_dynamique

Une instance dynamique est une instance qui est entièrement simulée dans le monde de la physique et cette variable en lecture seule renverra true si l'instance vérifiée est entièrement simulée ou false si elle ne l'est pas.

Syntaxe :

phy_dynamic;

Les retours :

(or undefined if the instance is not physics enabled)

Exemple :

if other.phy_dynamic
{
    with (other)
    {
        var dir;
        dir = point_direction(x, y, other.x, other.y);
        physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
    }
}

Le code ci-dessus crée une nouvelle instance, puis la définit comme étant une "balle" dans le monde de la physique avant de lui donner une impulsion massive le long de l'axe y.
