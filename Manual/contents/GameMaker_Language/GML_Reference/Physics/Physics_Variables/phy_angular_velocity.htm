phy_angular_velocity

Cette variable peut être utilisée pour définir la vitesse angulaire de l'instance, ou pour obtenir la vitesse angulaire actuelle, en degrés par seconde. La valeur utilisée peut être positive (pour une rotation dans le sens des aiguilles d'une montre) ou négative (pour une rotation dans le sens inverse des aiguilles d'une montre). Si vous définissez cette valeur sur une instance qui était auparavant statique (c'est-à-dire qu'elle a une densité de 0), elle deviendra un objet cinématique et commencera à tourner.

Syntaxe :

phy_angular_velocity

Les retours :

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemple :

if abs(phy_angular_velocity) > 0
{
    phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;
}
else
{
    phy_angular_velocity = 0;
}

Le code ci-dessus vérifiera la vitesse angulaire de l'instance et si elle n'est pas égale à 0, il ajoutera (ou soustraira) une petite quantité à chaque pas jusqu'à ce que la valeur soit égale à 0.
