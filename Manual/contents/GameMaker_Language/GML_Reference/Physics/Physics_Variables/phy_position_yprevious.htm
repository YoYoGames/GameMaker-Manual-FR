phy_position_yprevious

Cette variable peut être utilisée pour obtenir (ou définir) la position y précédente de l'instance dans le monde physique de la salle de jeu. Il s'agit de la position de l'instance dans le monde physique de l'étape précédente à l'étape actuelle.

Syntaxe :

phy_position_yprevious;

Les retours :

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemple :

xx = phy_position_xprevious;
yy = phy_position_yprevious;

Le code ci-dessus stocke la position x et y précédente de l'instance activée par la physique dans deux variables.
