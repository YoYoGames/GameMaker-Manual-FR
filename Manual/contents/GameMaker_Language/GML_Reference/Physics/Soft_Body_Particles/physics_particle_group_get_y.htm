groupe_de_particules_physique_get_y

Cette fonction permet de récupérer la position y dans la pièce d'un groupe de particules. La valeur group est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des valeurs actuellement définies sur flags.

Syntaxe :

physics_particle_group_get_y(group)

ArgumentsType Description
group The particle group to get.

Les retours :

Exemple :

xx = physics_particle_group_get_x(group1);
yy = physics_particle_group_get_y(group1);

Le code ci-dessus récupère les positions x et y du groupe de particules indexé dans la variable "group1" et les stocke dans des variables.
