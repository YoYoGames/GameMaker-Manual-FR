physics_particle_draw_ext

Cette fonction va dessiner un sprite à la position de toutes les particules qui partagent la même valeur de catégorie définie par l'utilisateur, et qui ont flags qui coïncide avec celles définies pour le masque. La valeur du masque est définie de la même manière que vous définissez le flags lors de la création des particules (voir les fonctions physics_particle_create() pour un exemple), c'est-à-dire en utilisant le bitwise ou "|" pour masquer les bits appropriés pour chaque drapeau.

REMARQUE: Cette fonction dessinera toutes les particules de la simulation physique avec flags qui coïncide avec celles du masque, qu'il s'agisse de particules individuelles ou de groupes, si la catégorie de l'utilisateur est la même.

La particule dessinée ignorera les valeurs de couleur et d'alpha définies lors de sa création, et sera dessinée en utilisant les paramètres d'échelle, d'angle, de couleur et d'alpha donnés dans cette fonction. Notez que cette fonction sert uniquement à dessiner un sprite à la position de la particule et n'affecte en aucun cas le comportement de la particule.

Syntaxe :

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

Arguments Type Description
typemask Physique Particule Constante(s)de drapeau de la physique The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Sprite Asset The sprite to use for each particle.
subimg Real The image_index du sprite à dessiner.
xscale Real The scale factor to apply to the sprite along the x axis.
yscale Real The scale factor to apply to the sprite along the y axis.
ang Real The angle (in degrees) to draw the sprite at.
col Couleur The colour to blend with the sprite (default is c_white).
alpha Real The alpha value to use when drawing (0 - 1).

Les retours :

N/A

Exemple :

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

Le code ci-dessus dessinera un sprite pour toutes les particules de la catégorie "2" définie par l'utilisateur qui ont le même flags que le masque donné, mis à l'échelle par 2 le long des deux axes et avec un alpha de 0,5.
