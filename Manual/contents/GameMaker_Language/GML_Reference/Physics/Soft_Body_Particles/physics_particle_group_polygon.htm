physics_particle_group_polygon

Cette fonction va définir la forme du groupe de particules en cours de création. Vous devez d'abord avoir commencé la définition du groupe en utilisant la fonction physics_particle_group_begin()puis vous devez utiliser cette fonction pour indiquer à GameMaker que vous allez ajouter les points approximatifs qui seront utilisés pour définir la forme du polygone - approximatifs car la forme exacte dépendra de la taille des particules de base, définie par la fonction la fonction physics_particle_set_radius() car la simulation physique tente de faire entrer le plus grand nombre possible de particules dans la forme définie.

Vous définissez ensuite les points du polygone en appelant la fonction physics_particle_group_add_point()en définissant la position x/y dans la pièce pour chacun d'entre eux (minimum de 3, maximum de 8, et ils peuvent être définis comme concaves mais les points à l'intérieur de la forme seront ignorés). peuvent être définis comme concaves mais les points à l'intérieur de la forme seront ignorés). Enfin, vous devez appeler physics_particle_group_end() pour créer le groupe de particules dans la pièce.

Syntaxe :

physics_particle_group_polygon()

Les retours :

Exemple :

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

Le code ci-dessus stocke les flags pour le type de particule et les propriétés du groupe de particules dans des variables puis les utilise pour créer un groupe de particules polygonales de trois côtés à la position de la souris.
