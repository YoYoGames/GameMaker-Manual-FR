groupe_de_particules_physique_début

Cette fonction vous permet de créer un groupe de particules dans une pièce. Un groupe de particules est un groupe dans lequel les particules nécessaires pour créer une forme spécifique sont créées en une seule fois, ce qui vous permet de créer des corps mous simulés avec diverses propriétés définies par le site flags. Les "flags" sont la valeur de retour d'une combinaison des constantes de particules (voir la fonction physics_particle_create() pour plus de détails), tandis que les "groupFlags" sont une combinaison des constantes de groupe suivantes (ou 0 pour aucun flags) :

Constante du drapeau du groupe de particules de physique
Constant Description
phy_particle_group_flag_solid Un groupe de particules solides empêche les autres appareils de se loger en son sein. Si un objet y pénètre, le groupe de particules solides repousse l'objet en question vers sa surface, ce qui confère au groupe de particules solides une force de répulsion particulièrement puissante.
phy_particle_group_flag_rigid Les groupes de particules rigides sont ceux dont la forme ne change pas, même lorsqu'ils entrent en collision avec d'autres appareils.

Ces groupes flags utilisent le masquage de bits pour créer une valeur de sortie finale qui est ensuite vérifiée pour définir les différentes propriétés de base du groupe, en dehors de celles définies par la particule flags. Vous pouvez utiliser aucune, l'une ou les deux de ces constantes en utilisant le bitwise ou "|" pour masquer les bits appropriés (comme indiqué dans l'exemple ci-dessous).

Outre le site flags, vous pouvez également définir la position dans la pièce où créer le groupe de particules (la position donnée sera le centre de masse du groupe fini), la vitesse horizontale et verticale initiale, ainsi que la vitesse angulaire du groupe, ce qui vous permet de créer des groupes mobiles et rotatifs.

Vous pouvez également définir la couleur et l'alpha initiaux de toutes les particules du groupe, les valeurs étant utilisées lors du dessin des particules à l'aide de la fonction physics_particle_draw(). Ensuite, vous définissez la force de cohésion des particules du groupe, ce qui est une autre façon de dire à quel point elles "collent" les unes aux autres. La force peut être une valeur comprise entre 0 et 1, où 0 correspond à une cohésion nulle et 1 à une cohésion maximale.

Enfin, vous devez indiquer la catégorie de votre groupe de particules. Cette valeur est un nombre entier arbitraire que vous donnez à chaque groupe de particules dont vous voulez qu'il ait des propriétés similaires, et elle peut être utilisée pour définir flags plus tard pour cette catégorie spécifique ainsi qu'un certain nombre d'autres choses.

REMARQUE: Vous pouvez utiliser n'importe quelle valeur entière pour la catégorie , sauf 0 qui est réservé par la simulation et est utilisé pour sélectionner toutes les catégories dans d'autres fonctions (comme pour le dessin).

Il est important de noter que l'appel de cette fonction ne crée pas le groupe dans la salle de physique. Cette fonction lance simplement le processus et définit les propriétés de base des particules pour le groupe. Pour créer réellement le groupe de particules, vous devez lui donner un formulaire (voir l'exemple ci-dessous), puis appeler la fonction physics_particle_group_end()à ce moment-là, il sera créé.

REMARQUE: Le type de particule flags que vous définissez influencera également la cohésion du groupe de particules. Ainsi, si vous marquez les particules comme étant de type eau (par exemple), lorsque le groupe entre en collision avec un autre objet fixe, il se désagrège, alors que si vous marquez la particule comme étant de type ressort, elle conservera sa cohésion en cas de collision.

Syntaxe :

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Arguments Type Description
flags Physique Particule Flag Constante(s) The flags to set on the particle.
groupflags Physique Groupe de particules Drapeau Constante(s) The group flags to set on the particle group (see table at the top).
x Real The x position to create the particle group at.
y Real The y position to create the particle group at.
ang Real The angle (in degrees) to rotate the group to on create.
xv Real The initial horizontal velocity.
yv Real The initial vertical velocity.
ang_velocity Real The initial angular velocity.
col Couleur The base colour to use for the particle group.
alpha Real The base alpha to use for the particle group.
strength Real Set the cohesion strength between particles in the group.
category Real The user defined category to which the group belongs.

Les retours :

N/A

Exemple :

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Le code ci-dessus stocke les flags pour le type de particule et les propriétés du groupe de particules dans des variables, puis les utilise pour créer un groupe de particules circulaires d'un rayon de 100px à la position de la souris.
