physics_particle_draw

Cette fonction va dessiner un sprite à la position de toutes les particules qui partagent la même valeur de catégorie définie par l'utilisateur, et qui ont flags qui coïncide avec celles définies pour le masque. La valeur du masque est définie de la même manière que vous définissez le flags lors de la création de la particule (voir la fonction physics_particle_create() pour un exemple), c'est-à-dire en utilisant le bitwise ou "|" pour masquer les bits appropriés pour chaque drapeau.

REMARQUE: Cette fonction dessinera toutes les particules de la simulation physique avec flags qui coïncide avec celles du masque, qu'il s'agisse de particules individuelles ou de groupes, si la catégorie de l'utilisateur est la même.

La particule dessinée utilisera les valeurs de couleur et d'alpha définies lors de sa création, et tournera avec le moment angulaire de la particule. Cependant, si vous souhaitez mettre à l'échelle le site sprite ou modifier toute autre propriété dessinée, vous devez utiliser la fonction physics_particle_draw_ext().

Syntaxe :

physics_particle_draw(typemask, category, sprite, subimg)

Arguments Type Description
typemask Physique Particule Constante(s)de drapeau de la physique The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Sprite Asset The sprite to use for each particle.
subimg Real The image_index du sprite à dessiner.

Les retours :

N/A

Exemple :

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

Le code ci-dessus dessinera un sprite pour toutes les particules de la catégorie "2" définie par l'utilisateur qui ont le même flags que le masque donné.
