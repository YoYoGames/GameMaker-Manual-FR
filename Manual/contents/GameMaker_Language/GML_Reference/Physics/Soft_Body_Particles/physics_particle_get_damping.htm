physics_particle_get_damping

Cette fonction permet de connaître l'amortissement linéaire actuel des particules dans la simulation physique (vous pouvez définir cette valeur à l'aide de la fonction physics_particle_set_damping()).

Syntaxe :

physics_particle_get_damping()

Les retours :

Exemple :

if (physics_particle_get_damping() < 1)
{
    physics_particle_set_damping(physics_particle_get_damping() + 0.01);
}

Le code ci-dessus vérifiera la valeur d'amortissement actuelle pour toutes les particules du système et si elle est inférieure à 1, il y ajoutera 0,01.
