nombre_de_groupes_de_particules_physiques

Cette fonction renvoie le nombre de particules qui sont actives dans un seul groupe. L'indice de groupe (ID) est la valeur qui est renvoyée lorsque vous appelez la fonction physics_particle_group_end().

Syntaxe :

physics_particle_group_count(group)

ArgumentsType Description
group The group index (ID) of the particle group to count.

Les retours :

Exemple :

gp_num = physics_particle_group_count(group1);

Le code ci-dessus récupère le nombre de particules utilisées pour constituer le groupe indexé dans la variable "group1" et stocke la valeur dans une variable.
