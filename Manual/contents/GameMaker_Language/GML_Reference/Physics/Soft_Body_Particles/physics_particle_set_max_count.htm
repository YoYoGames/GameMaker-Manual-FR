physics_particle_set_max_count

Cette fonction permet de définir le nombre total de particules autorisées dans une simulation physique. Si vous fixez cette valeur à 0, il n'y a pas de limite aux particules créées, et si vous la fixez à une autre valeur, cela empêchera la création de toute autre particule lorsque le nombre total de particules est égal à cette valeur. Notez que le nombre minimum de particules que vous pouvez créer dans une simulation est de 128, donc définir cette valeur plus bas n'aura aucun effet (à moins qu'elle ne soit définie sur 0).

Syntaxe :

physics_particle_set_max_count(count)

ArgumentsType Description
count The maximum number of particles to permit.

Les retours :

Exemple :

physics_particle_set_max_count(500);

Le code ci-dessus définit le plafond de particules de la simulation physique à 500.
