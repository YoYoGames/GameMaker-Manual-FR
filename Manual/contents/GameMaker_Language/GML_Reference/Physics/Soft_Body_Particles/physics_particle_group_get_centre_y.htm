physics_particle_group_get_centre_y

Cette fonction vous permet de récupérer la composante y du centre de masse d'un groupe entier de particules. La valeur group est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des valeurs actuellement définies sur flags.

Syntaxe :

physics_particle_group_get_centre_y(group)

ArgumentsType Description
group The particle group to get.

Les retours :

Exemple :

xx = physics_particle_group_get_centre_x(group1);
yy = physics_particle_group_get_centre_y(group1);

Le code ci-dessus va obtenir la position x et y du centre de masse du groupe de particules indexé dans la variable "group1" et les stocker dans des variables.
