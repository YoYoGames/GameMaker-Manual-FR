physics_particle_group_add_point

Cette fonction définit un point dans la pièce pour définir la forme d'un polygone qui sera utilisé pour créer un groupe de particules de corps mou. Vous devez avoir préalablement signalé à GameMaker que vous allez définir une forme de polygone à l'aide de la fonction physics_particle_group_polygon(), puis utiliser cette fonction pour définir les différents points du polygone. Vous devez indiquer au moins trois points lors de la définition de la forme du polygone, mais vous pouvez en indiquer jusqu'à huit, et la fonction permettra la définition de polygones concaves. Toutefois, si vous générez un polygone comportant des cavités, les points qu'il contient seront ignorés et une forme convexe sera créée pour le groupe de particules.

Syntaxe :

physics_particle_group_add_point(x, y)

ArgumentsType Description
x The x position in the room for the point.
y The y position in the room for the point.

Les retours :

Exemple :

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

Le code ci-dessus stocke les flags pour le type de particule et les propriétés du groupe de particules dans des variables puis les utilise pour créer un groupe de particules polygonales de trois côtés à la position de la souris.
