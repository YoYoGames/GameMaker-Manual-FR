physics_particle_delete

Cette fonction permet de supprimer (retirer) une particule de la simulation physique dans la pièce actuelle. La fonction prend l'ID unique de la particule à supprimer, tel que renvoyé par la fonction physics_particle_create().

Syntaxe :

physics_particle_delete(ind)

Arguments Type Description
ind Physique Particule ID The index (ID) of the particle to delete.

Les retours :

N/A

Exemple :

physics_particle_delete(part);

Le code ci-dessus supprimera de la simulation la particule dont l'ID est stocké dans la variable "part".
