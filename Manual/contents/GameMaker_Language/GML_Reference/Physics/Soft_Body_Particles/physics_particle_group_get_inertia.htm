groupe_de_particules_physique_get_inertie

Cette fonction permet de récupérer l'inertie d'un groupe entier de particules. La valeur group est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end()La fonction renvoie une valeur qui est la valeur combinée des valeurs actuellement définies sur flags.

Syntaxe :

physics_particle_group_get_inertia(group)

Arguments Type Description
group ID du groupe de particules de physique The particle group to get.

Les retours :

Real

Exemple :

i = physics_particle_group_get_inertia(group1);

Le code ci-dessus va récupérer l'inertie du groupe de particules indexé dans la variable "group1" et la stocker dans une variable.
