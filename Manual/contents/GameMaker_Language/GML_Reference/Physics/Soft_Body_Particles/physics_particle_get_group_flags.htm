physics_particle_get_group_flags

Cette fonction permet de récupérer le groupe flags pour un groupe de particules. La valeur group est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end()La fonction renvoie une valeur qui est la valeur combinée des valeurs actuellement définies pour flags.

Syntaxe :

physics_particle_get_group_flags(group)

Arguments Type Description
group ID du groupe de particules de physique The particle group to get.

Les retours :

Real

Exemple :

var flags = physics_particle_get_group_flags(group, flags);
if flags != (phy_particle_group_flag_solid | phy_particle_group_flag_rigid)
{
    flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
    physics_particle_set_group_flags(group, flags);
}

Le code ci-dessus créera une variable pour stocker la valeur flags et l'utilisera ensuite pour vérifier le flags du groupe indexé dans la variable "groupe". Si elles ne sont pas les mêmes, le groupe est défini avec ces flags.
