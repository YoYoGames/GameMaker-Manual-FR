physique_particule_get_gravité_échelle

Cette fonction vous permet de connaître le facteur d'échelle de gravité actuel des particules dans la simulation physique (vous pouvez définir cette valeur à l'aide de la fonction physics_particle_set_gravity_scale()).

Syntaxe :

physics_particle_get_gravity_scale()

Les retours :

Exemple :

physics_particle_set_gravity_scale(physics_particle_get_gravity_scale() + 0.1);

Le code ci-dessus va définir l'échelle de gravité.

Retour : Particules molles du corps
