physics_particle_group_circle

Cette fonction va définir la forme du groupe de particules en cours de création. Vous devez d'abord avoir commencé la définition du groupe en utilisant la fonction physics_particle_group_begin()et ensuite vous devez utiliser cette fonction pour définir le rayon approximatif du groupe en pixels - approximatif car le rayon exact dépendra de la taille des particules de base, telle que définie par la fonction physics_particle_set_radius() car la simulation physique essaie de faire entrer le plus grand nombre possible de particules dans la forme définie. Enfin, vous devez appeler physics_particle_group_end() pour créer le groupe de particules dans la pièce.

Syntaxe :

physics_particle_group_circle(radius)

ArgumentsType Description
radius The radius (in pixels) of the circle to create.

Les retours :

Exemple :

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Le code ci-dessus stocke les flags pour le type de particule et les propriétés du groupe de particules dans des variables, puis les utilise pour créer un groupe de particules circulaires d'un rayon de 100px à la position de la souris.
