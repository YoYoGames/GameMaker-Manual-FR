physics_particle_group_get_vel_x

Cette fonction permet de récupérer la vitesse horizontale d'un groupe de particules. La valeur group est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des valeurs actuellement définies sur flags.

Syntaxe :

physics_particle_group_get_vel_x(group)

ArgumentsType Description
group The particle group to get.

Les retours :

Exemple :

vx = physics_particle_group_get_vel_x(group1);
vy = physics_particle_group_get_vel_y(group1);

Le code ci-dessus va obtenir la vitesse horizontale et verticale du groupe de particules indexé dans la variable "group1" et les stocker dans des variables.
