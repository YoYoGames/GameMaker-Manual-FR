test_physique_chevauchement

Cette fonction peut être utilisée pour vérifier et voir si un corps physique (c'est-à-dire le dispositif d'une instance) se chevauche, ou se chevauchera, lorsqu'il est tourné et placé à une position donnée dans la pièce. L'argument "angle" est l'angle de rotation que l'instance appelante a (ou aura) à la position à vérifier, et l'argument "obj" peut être soit un identifiant d'instance unique, et un index d'objet, soit la fonction mots-clés tout ou autre.

Syntaxe :

physics_test_overlap(xpos, ypos, angle, obj);

Arguments Type Description
xpos Real The x position in the room to check
ypos Real The y position in the room to check
angle Real The angle to check (of the calling instance)
obj ID de l'actif ou de l'instance del'objet The object to check for

Les retours :

Boolean

Exemple :

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

Le code ci-dessus vérifie qu'il n'y a pas de chevauchement entre les appareils de physique dans une position et, s'il y en a un, il définit une variable et une alarme.
