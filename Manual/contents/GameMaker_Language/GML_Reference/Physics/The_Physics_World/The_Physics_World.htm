Le monde de la physique

Avant que toute réaction physique ne puisse se produire dans GameMaker, vous devez d'abord définir le monde physique dans lequel elle va se produire. Si vous ne le faites pas, GameMaker utilisera par défaut les fonctions de collision de base, et tout autre code ou paramètre relatif à la physique sera ignoré. Vous pouvez définir le monde physique à partir de l'éditeur de pièce, mais vous pouvez également définir les mêmes propriétés en utilisant le code avec les fonctions suivantes :

  • physics_world_create
  • physics_world_gravity
  • physics_world_update_iterations
  • physics_world_update_speed
  • physics_world_draw_debug
  • physics_draw_debug
  • physics_pause_enable
