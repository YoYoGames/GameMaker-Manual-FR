physics_joint_set_value

Certaines propriétés de l'articulation peuvent être modifiées et définies même après la création de l'articulation. Il y a un certain nombre de constantes qui peuvent être utilisées dans cette fonction et elles peuvent être trouvées ici : Constantes des articulations de la physique.

Syntaxe :

physics_joint_set_value(joint, field, value)

Arguments Type Description
joint Physique ID commune The index of the joint that you wish to change
field Physique Constante commune The constant for the joint property that you wish to change
value Real The new value for the joint property

Les retours :

N/A

Exemple :

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

Le code ci-dessus vérifie si le couple moteur maximum des joints est inférieur à 2 et si c'est le cas, il le fixe à 2.
