physics_joint_get_value

En utilisant une série de constantes prédéfinies, vous pouvez demander à GameMaker de vous dire un certain nombre de choses sur l'état d'une articulation donnée. C'est très utile car cela vous donne la possibilité de supprimer des articulations ou de modifier le comportement d'une instance en fonction de vos besoins du moment. Il y a un certain nombre de constantes qui peuvent être utilisées dans cette fonction et elles peuvent être trouvées ici : Physics Joint Constants, mais sachez que des calculs complexes sont effectués lorsque vous les appelez. Il convient donc de les utiliser avec précaution et uniquement lorsque cela est nécessaire.

Si la propriété n'existe pas (si, par exemple, vous vérifiez la translation prismatique d'une articulation de poulie), la valeur de retour sera 0.

Syntaxe :

physics_joint_get_value(joint, value)

Arguments Type Description
joint Physique ID commune The index of the joint that you wish to test
value Physique Constante commune The constant for the joint property that you wish to test

Les retours :

Real

Exemple :

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

Le code ci-dessus vérifie si la variable "shipJoint" contient un indice d'articulation et si c'est le cas, il calcule la force appliquée à cette articulation à l'aide des deux constantes. Enfin, si la force totale est supérieure à 2, l'articulation est supprimée.
