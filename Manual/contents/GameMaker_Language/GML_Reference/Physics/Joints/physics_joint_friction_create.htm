physique_joint_friction_créer

Le joint de friction est un peu différent de tous les autres joints de la simulation physique en ce sens que la connexion créée ne contraindra pas la position ou le mouvement des instances, mais plutôt leur vitesse et leur rotation. Cela fonctionne en prenant les valeurs d'entrée maximales pour la force et le couple et en les appliquant à la seconde fixation pour ramener la vitesse et le moment angulaire aux mêmes valeurs que celles de la première instance. Ainsi, si vous avez une instance stationnaire et une instance mobile et que vous les reliez par un joint de friction, l'instance mobile ralentira progressivement jusqu'à ce qu'elle soit également stationnaire. Si les deux instances étaient en mouvement, la vitesse de déplacement de la seconde instance sera modifiée pour correspondre à celle de la première instance.

Comme pour la plupart des autres joints, vous fournissez les instances à joindre ensemble, ainsi que la position à laquelle le joint doit être créé dans la pièce. Vous fournissez ensuite la force maximale et le couple maximal (friction directionnelle et rotationnelle), et si vous définissez la valeur "col" sur true, les deux instances peuvent interagir et entrer en collision l'une avec l'autre, mais seulement si elles ont des événements de collision, mais si elle est définie sur false, elles n'entreront jamais en collision.

Syntaxe :

physics_joint_friction_create(inst1, inst2, anchor_x, anchor_y, max_force, max_torque, col)

Arguments Type Description
inst1 ID de l'instance The first instance to connect with the joint
inst2 ID de l'instance The second instance to connect with the joint
anchor_x Real The x coordinate for the joint, within the game world
anchor_y Real The y coordinate for the joint, within the game world
max_force Real The maximum frictional force that will be applied
max_torque Real The maximum rotational force that will be applied
col Booléen Whether the two instances can collide (true) or not (false)

Les retours :

Physics Joint ID

Exemple :

var mainFixture, o_id;
mainFixture = physics_fixture_create();
physics_fixture_set_circle_shape(mainFixture, sprite_get_width(sprite_index) / 2);
o_id = instance_create_layer(x+300, y, "Instances", obj_Rudder);
physics_fixture_bind(mainFixture, id);
physics_fixture_bind(mainFixture, o_id);
physics_joint_friction_create(id, o_id, x, y, 10, 0.3, true);
physics_fixture_delete(mainFixture);

Le code ci-dessus crée un dispositif de fixation, puis le lie à deux instances, avant de les connecter à l'aide d'un joint à friction.
