physics_joint_delete

Une fois que deux instances avec des représentations physiques ont été liées par un joint, celui-ci peut être supprimé à tout moment. Normalement, cela se produit automatiquement lorsque l'une des deux instances est détruite, ou lorsque la room se termine, mais il arrive que vous souhaitiez le faire manuellement. Dans ce cas, vous devez utiliser cette fonction.

Il convient également de noter que lorsqu'une instance faisant partie d'un joint d'engrenage est détruite, le joint d'engrenage doit être supprimé à l'aide de cette fonction avant toute instance impliquée dans la formation de l'engrenage (mais les joints restants seront supprimés automatiquement), par exemple dans l'événement destroy de l'instance.

Syntaxe :

physics_joint_delete(joint)

Arguments Type Description
joint Physique ID commune The index of the joint that you wish to delete

Les retours :

N/A

Exemple :

var reactionForceX, reactionForceY, reactionForce;
if (shipJoint)
{
    _reaction_force_x = physics_joint_get_value(ship_joint, phy_joint_reaction_force_x);
    _reaction_force_y = physics_joint_get_value(ship_joint, phy_joint_reaction_force_y);
    _reaction_force = sqrt((_reaction_force_x + _reaction_force_x) + (_reaction_force_y + _reaction_force_y));
    if (_reaction_force > 2)
    {
        physics_joint_delete(ship_joint);
        ship_joint = -1;
    }
}

Le code ci-dessus vérifie si la variable "shipJoint" contient un indice d'articulation et si c'est le cas, il calcule la force appliquée à cette articulation à l'aide des deux constantes. Enfin, si la force totale est supérieure à 2, l'articulation est supprimée.
