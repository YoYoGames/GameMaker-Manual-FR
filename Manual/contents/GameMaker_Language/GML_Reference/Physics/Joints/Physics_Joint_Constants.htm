Constantes communes de la physique

Il existe un grand nombre de constantes incluses dans GameMaker GML qui sont spécifiques aux articulations entre les appareils dans le monde physique. Elles peuvent être utilisées en conjonction avec de nombreuses fonctions de jointure pour définir ou obtenir des informations en temps réel pendant que la simulation physique est en cours. Vous devez cependant savoir que des calculs complexes sont effectués lorsque vous les appelez, et qu'elles doivent donc être utilisées avec précaution et uniquement lorsque c'est nécessaire. Notez également que si vous pouvez obtenir toutes ces valeurs avec la fonction appropriée, vous ne pouvez définir que celles qui sont marquées comme n'étant pas en lecture seule.

En général, ces constantes sont utilisées en conjonction avec les fonctions suivantes :

  • physics_joint_get_value
  • physics_joint_set_value

Les constantes suivantes peuvent être appliquées à tous les types de joints disponibles :

Physique Constante commune
Constant Description Lecture seulement
phy_joint_anchor_1_x La coordonnée x du premier point d'ancrage du joint dans la pièce. Yes
phy_joint_anchor_1_y La coordonnée y du premier point d'ancrage du joint dans la pièce Yes
phy_joint_anchor_2_x La coordonnée x du deuxième point d'ancrage du joint dans la pièce Yes
phy_joint_anchor_2_y Coordonnée y du deuxième point d'ancrage du joint dans la pièce Yes
phy_joint_reaction_force_x C'est la force de réaction appliquée à la deuxième instance dans une articulation à la position d'ancrage x. Yes
phy_joint_reaction_force_y Il s'agit de la force de réaction appliquée à la deuxième instance dans une articulation à la position d'ancrage y. Yes
phy_joint_reaction_torque Il s'agit du couple appliqué à la deuxième instance dans une articulation à la position d'ancrage. Yes


Ces constantes sont destinées aux articulations auxquelles un moteur est attaché (rotative, prismatique, roue) :

Physique Constante commune
Constant Description Lecture seulement
phy_joint_max_motor_force La valeur spécifiée lors de la création de l'articulation pour la force maximale du moteur. No
phy_joint_max_motor_torque La valeur spécifiée lors de la création du joint pour le couple moteur maximum No
phy_joint_motor_force La force actuelle du moteur Yes
phy_joint_motor_speed La vitesse actuelle du moteur No
phy_joint_motor_torque Le couple actuel du moteur Yes


Pour une articulation rotative, vous pouvez utiliser la constante suivante (ainsi que les constantes du moteur si un moteur a été ajouté) :

Physique Constante commune
Constant Description Lecture seulement
phy_joint_angle L'angle que fait une ligne entre les deux points d'ancrage de l'articulation. Il est calculé à l'aide des coordonnées du monde physique
(pas les coordonnées de la pièce GameMaker ) en radians.		 Oui
phy_joint_angle_limits Active ou désactive la limitation d'angle pour l'articulation. Définissez la valeur sur true pour activer ou false pour désactiver. Non
phy_joint_upper_angle_limit La limite supérieure de l'angle de l'articulation en degrés. Non
phy_joint_lower_angle_limit La limite inférieure de l'angle de l'articulation en degrés. Non


Pour un joint prismatique, vous pouvez utiliser la constante suivante :

Physique Constante commune
Constant Description Lecture seulement
phy_joint_translation Obtient la distance entre les coordonnées x/y de l'ancre et les coordonnées x/y locales. Yes
phy_joint_speed La vitesse actuelle du mouvement de l'articulation. Yes


Pour une distance, une soudure et des joints de roue, vous pouvez utiliser les constantes suivantes (ainsi que celles pour les joints de poulie) :

Physique Constante commune
Constant Description Lecture seulement
phy_joint_damping_ratio Le rapport d'amortissement est non dimensionnel et définit le "ressort" de l'articulation. La valeur de cette constante est généralement comprise entre 0 et 1, mais peut être plus grande. À 1, l'amortissement est critique, ce qui signifie que toutes les oscillations doivent disparaître. No
phy_joint_frequency Ceci renverra (ou définira) la fréquence d'oscillation de l'articulation, en hertz, et typiquement la fréquence devrait être inférieure à la moitié de la fréquence du pas de temps, comme défini par la fonction physics_world_update_speed(). No
phy_joint_length_1 Cette fonction renvoie la longueur de l'articulation entre les premières coordonnées locales x/y et les premières coordonnées x/y de l'ancre (uniquement pour les articulations à distance, qui peuvent être lues). Yes
phy_joint_length_2 Ceci renverra la longueur du joint entre les deuxièmes coordonnées locales x/y et les deuxièmes coordonnées x/y de l'ancre (joints de distance uniquement, ne peuvent être écrits). No


Pour un joint de friction, vous pouvez utiliser les constantes suivantes :

Physique Constante commune
Constant Description Lecture seulement
phy_joint_max_torque La valeur maximale du couple pour l'articulation. No
phy_joint_max_force La valeur de la force maximale pour l'articulation. No


Pour un joint de corde, vous pouvez utiliser la constante suivante :

Physique Constante commune
Constant Description Lecture seulement
phy_joint_max_length L'extension maximale de la connexion entre les deux points d'ancrage. No
