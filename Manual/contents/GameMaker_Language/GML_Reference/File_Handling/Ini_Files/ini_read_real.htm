ini_read_real

Vous pouvez utiliser cette fonction pour lire un nombre à partir d'un fichier de données ini. Les fichiers ini sont divisés en sections, puis chaque section est ensuite divisée en paires clé-valeur. Ainsi, un fichier ini typique ressemblerait à quelque chose comme ceci :

Syntaxe :

ini_read_real(section, key, default);

ArgumentsType Description
section The section of the .ini to read from.
key The key within the relevant section of the .ini to read from.
default The value to return if a value is not found in the defined place (or the .ini file does not exist). Must be a real number.

Les retours :

Exemple :

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Cela ouvrira le fichier "savedata.ini" et fixera le score à la valeur indiquée dans "save1" > "score", puis refermera le fichier .ini. S'il n'y a pas de valeur sous "save1" > "score", ou s'il n'y a pas de fichier "savedata.ini", le score sera mis à 0. n'est présent, le score sera fixé à 0.
