ini_open_from_string

Cette fonction va créer un fichier ini temporaire à partir d'une chaîne de caractères et l'ouvrir en lecture/écriture. La chaîne doit être correctement formatée comme un fichier ini (c'est-à-dire avec des sections, des sauts de ligne, des clés et des valeurs), sinon le fichier ini ne sera pas créé correctement. Notez que ce fichier ini est temporaire et sera supprimé de la mémoire au moment où il sera fermé, perdant toute information qui y était stockée, cependant la fonction ini_close() renvoie une chaîne de caractères du fichier ini complet qui peut ensuite être enregistré sur un serveur ou sur un disque.

Syntaxe :

ini_open_from_string(string);

ArgumentsType Description
string The string containing all the ini information.

Les retours :

N/A

Exemple :

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

Le code ci-dessus définit un ini temporaire pour contenir les informations de la chaîne "str", puis lit l'ini avant de le refermer.
