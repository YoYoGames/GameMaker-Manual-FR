ini_read_string

Vous pouvez utiliser cette fonction pour lire une chaîne (texte) à partir d'un fichier de données ini. Les fichiers ini sont divisés en sections, puis chaque section est ensuite divisée en paires clé-valeur. Ainsi, un fichier ini typique ressemblerait à quelque chose comme ceci :

Syntaxe :

ini_read_string(section, key, default);

Arguments Type Description
section Chaîne de caractères The section of the .ini to read from.
key Chaîne de caractères The key within the relevant section of the .ini to read from.
default Chaîne de caractères The string to return if a string is not found in the defined place (or the .ini file does not exist). Must be a string.

Les retours :

String

Exemple :

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

Cela ouvrira le fichier "savedata.ini" et donnera à global.name la valeur de la chaîne sous "save1" > "Name", puis refermera le fichier .ini. S'il n'y a pas de chaîne sous "save1" > "Name", ou s'il n'y a pas de fichier "savedata.ini", global.name sera réglé sur "Player1".
