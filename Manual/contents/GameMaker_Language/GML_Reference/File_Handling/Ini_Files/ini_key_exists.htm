ini_key_exists

Cette fonction vérifie si une clé existe dans l'ini actuellement ouvert et renvoie true si c'est le cas ou false sinon. Ce n'est pas une vérification nécessaire pour éviter les erreurs car, lorsqu'une clé n'existe pas, la lecture d'une clé inexistante renverra simplement une valeur par défaut. Cependant, cela peut être utile pour voir si un fichier ini a sauvegardé des données spécifiques et quelques autres choses.

Syntaxe :

ini_key_exists(section, key);

ArgumentsType Description
section The section in the open .ini file to check a key in.
key The key to check for.

Les retours :

Exemple :

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
    global.name = "Player1";
}
ini_close();

Cela donnera à la variable global.name la valeur "Player1" si une clé telle que "name" n'existe pas.
