ini_write_real

Vous pouvez utiliser cette fonction pour écrire une valeur (numérique) dans un fichier de données ini. Les fichiers ini sont divisés en sections, puis chaque section est ensuite divisée en paires clé-valeur. Ainsi, un fichier ini typique ressemblerait à quelque chose comme ceci :

Syntaxe :

ini_write_real(section, key, value);

ArgumentsType Description
section The section of the .ini to write to.
key The key within the relevant section of the .ini to write to.
value The real value to write to the relevant destination.

Les retours :

Exemple :

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

Cela va définir le score à 1000, puis ouvrir "savedata.ini" et écrire cette valeur dans "save1" > "Score". Ensuite, le score2 sera fixé à la valeur réelle sous "save1" > "score", soit 1000, et le fichier .ini sera refermé. à nouveau. S'il n'avait pas trouvé cette valeur, le score2 aurait été fixé à 0.
