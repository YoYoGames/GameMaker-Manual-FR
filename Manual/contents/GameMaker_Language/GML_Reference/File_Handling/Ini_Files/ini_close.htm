ini_close

Cette fonction doit être appelée au moment où vous avez fini de lire ou d'écrire dans un fichier ini ouvert. Si vous n'utilisez pas cette fonction après avoir utilisé l'une des fonctions d'écriture ini, rien ne sera écrit sur le disque, car les informations du fichier sont conservées en mémoire jusqu'à ce que cette fonction soit appelée, ce qui force l'écriture. Si vous essayez d'ouvrir un ini sans en avoir préalablement fermé un autre (ou le même), vous obtiendrez également une erreur.

La fonction renverra également une chaîne de caractères avec le fichier ini encodé. Cette chaîne peut ensuite être sauvegardée sur un serveur et/ou utilisée à nouveau avec la fonction ini_open_from_string() pour recréer le fichier ini.

Syntaxe :

ini_close();

Les retours :

String

Exemple :

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Cela ouvrira "savedata.ini" et lira une valeur de la section "save1" avec la clé "score", puis fermera le fichier ini. S'il n'y a pas de valeur sous "save1" appelée "score" ou s'il n'y a pas de fichier "savedata.ini" présent, 0 sera retourné et un nouveau fichier ini sera créé.
