répertoire du programme

Cela renverra le répertoire où l'exécutable du jeu est stocké. Cependant, cela n'est pas toujours utile, notamment parce que certains périphériques exécutent l'exe à partir d'un fichier *.zip. Cette option renvoie donc le même résultat, quel que soit l'endroit où le jeu est exécuté.

AVERTISSEMENT ! Cette fonction peut ne pas fonctionner comme vous l'attendez, car GameMaker est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

REMARQUE: Si vous désactivez le site sandbox, vous ne pourrez toujours pas écrire dans ce répertoire afin d'éviter tout problème éventuel de suppression des fichiers de jeu requis.

Syntaxe :

program_directory

Les retours :

Exemple :

dir = program_directory;

Le répertoire où est stocké l'exécutable sera ainsi stocké dans une variable.
