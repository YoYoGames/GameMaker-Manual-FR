répertoire_existe

Cette fonction renvoie true si le répertoire indiqué existe ou false s'il n'existe pas. Le nom spécifié doit inclure le chemin d'accès complet, et non un chemin d'accès relatif. Par défaut, il n'est pas possible d'accéder à access à partir d'un répertoire extérieur au paquet de jeu, car tous les jeux sont protégés par un bac à sable (voir la section sur le système de fichiers pour plus d'informations).

Syntaxe :

directory_exists(dname)

ArgumentsType Description
dname The name of the directory to look for.

Les retours :

Exemple :

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Elle vérifiera si le répertoire spécifié existe et, si c'est le cas, elle s'y rendra et renverra le premier fichier "en lecture seule" doc trouvé.
