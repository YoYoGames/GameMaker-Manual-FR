répertoire_créer

Cette fonction va créer un répertoire avec le nom donné dans la zone de sauvegarde.

REMARQUE: Cette fonction ne fonctionnera pas sur HTML5 car vous ne pouvez pas créer ou supprimer des répertoires dans le stockage local du navigateur.

AVERTISSEMENT ! Cette fonction peut ne pas fonctionner comme vous l'attendez, car GameMaker est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe :

directory_create(dname)

ArgumentsType Description
dname The name of the directory to create.

Les retours :

Exemple :

if !directory_exists("Games")
{
    directory_create("Games");
}

Cette opération vérifie si le répertoire spécifié existe dans le dossier de données local et, si ce n'est pas le cas, elle le crée pour vous.
