réécriture de fichier_bin

Cette fonction prend le handle du nom de fichier tel que retourné par la fonction file_bin_open() puis réécrit le fichier, en le vidant de toutes les données précédentes pour commencer à écrire depuis le début du fichier.

REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Syntaxe :

file_bin_rewrite(binfile);

ArgumentsType Description
binfile The ID of the file to rewrite.

Les retours :

Exemple :

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

Cela ouvrirait un fichier dans le même répertoire que le jeu, et attribuerait son index à la variable "file". Elle réécrit ensuite le fichier (en le vidant de ses données).
