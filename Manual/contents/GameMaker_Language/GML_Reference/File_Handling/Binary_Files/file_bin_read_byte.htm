file_bin_read_byte

Cette fonction renvoie un octet de données à partir de la position actuelle dans le fichier avec l'ID de fichier donné. Vous fournissez la valeur de l'ID du fichier, telle que retournée par la fonction file_bin_open().

REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Syntaxe :

file_bin_read_byte(binfile);

ArgumentsType Description
binfile The ID of the file to get the data from.

Les retours :

Exemple :

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Cela ouvre un fichier dans le même répertoire que le jeu, et récupère un octet de données avant de le refermer.
