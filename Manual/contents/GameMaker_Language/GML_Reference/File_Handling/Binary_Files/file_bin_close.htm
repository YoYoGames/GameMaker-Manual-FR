file_bin_close

Ferme un fichier binaire précédemment ouvert. Vous fournissez la valeur de l'ID du fichier, telle que retournée par la fonction file_bin_open().

REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Syntaxe :

file_bin_close(binfile);

ArgumentsType Description
binfile The ID of the file to close.

Les retours :

Exemple :

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

Cela ouvre un fichier dans le même répertoire que le jeu, et assigne son index à la variable "file". Elle réécrit ensuite le fichier (en le vidant de ses données), puis le referme.
