file_bin_write_byte

Cette fonction écrit un octet de données dans le fichier identifié par l'ID du fichier à la position d'écriture actuelle. Vous fournissez la valeur de l'ID du fichier, telle que retournée par la fonction file_bin_open() et l'octet de données à écrire.

REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Syntaxe :

file_bin_write_byte(binfile, byte);

ArgumentsType Description
binfile The ID of the file to write to.
byte The data to write.

Les retours :

Exemple :

file_bin_write_byte(file, data);

Cette opération permet d'écrire un octet dans le fichier sélectionné.

Retour : Fichiers binaires
