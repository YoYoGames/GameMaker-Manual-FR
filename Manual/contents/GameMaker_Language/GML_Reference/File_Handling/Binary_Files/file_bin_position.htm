file_bin_position

Cette fonction renvoie la position actuelle en octets, où 0 est la première position, du fichier avec l'ID de fichier donné. Vous fournissez la valeur de l'ID du fichier, telle que retournée par la fonction file_bin_open().

REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Syntaxe :

file_bin_position(binfile);

ArgumentsType Description
binfile The ID of the file to get the position in.

Les retours :

Exemple :

pos = file_bin_position(file);

Cela permet de stocker la position actuelle dans la variable "pos".

Retour : Fichiers binaires
