fichier_trouvé_premier

Cette fonction renvoie le nom du premier fichier qui satisfait au masque et aux attributs. Si un tel fichier n'existe pas, une chaîne vide est retournée.

Le masque peut contenir un chemin avec des caractères génériques, par exemple C:\temp\*.doc.

Les attributs donnent les fichiers supplémentaires que vous voulez voir, ainsi les fichiers normaux sont toujours retournés lorsqu'ils satisfont le masque. Vous pouvez additionner les constantes suivantes pour voir le type de fichiers que vous voulez (en utilisant |). Si vous ne souhaitez pas ajouter d'attributs, utilisez 0.

Attribut de fichier constant
Constant Description
fa_readonly Fichiers en lecture seule
fa_hidden Fichiers cachés
fa_sysfile Fichiers système
fa_volumeid Fichiers d'identification des volumes
fa_directory Annuaires
fa_archive Dossiers archivés

Les attributs ne peuvent être utilisés que sur Windows. Pour toutes les autres plateformes, utilisez 0.

AVERTISSEMENT Il se peut que cette fonction ne fonctionne pas comme vous l'attendez, car GameMaker est protégé par un bac à sable ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

NOTE Cette fonction ne fonctionnera pas du tout sur la cible HTML5.

Syntaxe :

file_find_first(mask, attr);

Arguments Type Description
mask Chaîne de caractères The mask to use for searching.
attr Attribut de fichier Constant(s) The specific file attribute to look for.

Les retours :

String

Exemple :

if directory_exists("\User Content")
{
    fileA = file_find_first("/User Content/*.doc", fa_readonly);
    fileB = file_find_next();
    fileC = file_find_next();
    file_find_close();
}

Ce code vérifie si le répertoire spécifié existe et si c'est le cas, il s'y rend et renvoie le premier fichier "en lecture seule" .doc trouvé. Il recherche ensuite deux autres fichiers et ferme l'outil de recherche de fichiers.

Vous pouvez rechercher un nombre quelconque de fichiers en utilisant une while boucle :

var files = [];
var file_name = file_find_first("/User Content/*.doc", fa_readonly);

while (file_name != "")
{
    array_push(files, file_name);

    file_name = file_find_next();
}

file_find_close();

Le code ci-dessus crée un tableau vide pour stocker tous les noms de fichiers qui ont été trouvés, et commence à chercher les fichiers en lecture seule .doc. Si ce nom de fichier n'est pas une chaîne vide, il le place dans le tableau files, puis tente de trouver le fichier suivant. La boucle se répète jusqu'à ce qu'une chaîne vide soit trouvée (ce qui signifie qu'il n'y a plus de fichiers correspondants), ce qui met fin à la boucle et ferme le chercheur de fichiers à la fin.
