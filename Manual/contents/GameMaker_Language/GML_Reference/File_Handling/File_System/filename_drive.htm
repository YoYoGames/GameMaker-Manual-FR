nom de fichier_drive

Cette fonction renvoie les informations sur le lecteur du nom de fichier.

Syntaxe :

filename_drive(fname);

ArgumentsType Description
fname The file to use.

Les retours :

Exemple :

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

Le code ci-dessus obtient les informations sur le lecteur (sous forme de chaîne) du fichier spécifié.
