chemin d'accès au nom de fichier

L'utilisation de cette fonction renvoie la partie chemin du chemin du fichier indiqué, y compris la barre oblique inverse finale.

Syntaxe :

filename_path(fname);

ArgumentsType Description
fname The file name and path to use.

Les retours :

Exemple :

path = filename_path(working_directory + "Test.ini");

Le code ci-dessus obtient le chemin (sous forme de chaîne) du fichier spécifié.
