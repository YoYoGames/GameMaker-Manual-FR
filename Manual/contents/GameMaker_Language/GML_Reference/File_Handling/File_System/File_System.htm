Système de fichiers

Il existe dans GameMaker un certain nombre de fonctions qui vous permettent de manipuler des fichiers de tous types de différentes manières :

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

Les fonctions suivantes peuvent être utilisées pour modifier les noms de fichiers :

NOTE Ces fonctions ne fonctionnent pas sur les fichiers réels ! Elles ne traitent que les chaînes de caractères liées aux différents éléments du chemin d'accès et doivent être utilisées conjointement avec les autres fonctions de fichier.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Enfin, il existe quelques fonctions spéciales pour obtenir des noms de fichiers qui peuvent être utilisés pour charger et sauvegarder des fichiers avec des filtres de type fichier. Ces fonctions ouvrent une fenêtre de l'explorateur, ce qui fait que le jeu perd le focus. Cela signifie que toute entrée dont dépend le jeu peut ne pas être correctement enregistrée, et nous vous recommandons donc (si vous utilisez ces fonctions dans un événement clavier) d'utiliser l'événement de la touche vers le haut et non les événements enfoncé ou bas.

REMARQUE Ces fonctions sont réservées à Windows et macOS.

AVERTISSEMENT Il se peut que ces fonctions ne fonctionnent pas comme vous l'attendez, car GameMaker est protégé par un bac à sable ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
