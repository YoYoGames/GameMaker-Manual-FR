fichier_existant

Cette fonction renvoie true si le fichier spécifié existe et false s'il n'existe pas. Notez que cette fonction ne peut être utilisée que pour vérifier les fichiers locaux, mais pas les fichiers stockés sur un serveur distant.

Syntaxe :

file_exists(fname);

ArgumentsType Description
fname The name of the file to check for.

Les retours :

Exemple :

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

Ceci vérifie l'existence d'un fichier et s'il existe, il est ouvert à la lecture.
