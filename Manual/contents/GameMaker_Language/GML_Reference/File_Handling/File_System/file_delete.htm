fichier_supprimé

Cette fonction supprime le fichier spécifié du système. Il faut noter que cette fonction ne supprimera que les fichiers que GameMaker est capable de créer et d'analyser : fichiers ini, fichiers texte et fichiers binaires, ou les fichiers destinés à stocker les ressources créées par le jeu, comme sprites ou les surfaces. En revanche, elle ne supprimera aucun autre fichier. La fonction renvoie également true si le fichier a été supprimé avec succès, ou false dans toutes les autres circonstances.

AVERTISSEMENT ! Cette fonction peut ne pas fonctionner comme vous l'attendez en raison de la présence d'un bac à sable dans GameMaker! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.

Syntaxe :

file_delete(fname);

ArgumentsType Description
fname The name of the file to delete.

Les retours :

Exemple :

if file_exists("level.txt")
{
    file_delete("level.txt");
}

Cela permet de vérifier la présence d'un fichier et s'il existe, il est supprimé.
