get_open_filename_ext

Cette fonction ouvre un dialogue et demande au joueur un nom de fichier à ouvrir avec le filtre donné. Le filtre a la forme "nom1|mask1|nom2|mask2|... ", où le masque peut contenir les différentes options avec un point-virgule entre elles et vous pouvez également utiliser un "*" pour signifier n'importe quelle chaîne. Par exemple : "bitmaps|*.bmp;*.wmf", demanderait à l'utilisateur de sélectionner le nom d'un fichier bitmap du système, mais seulement ceux qui ont les deux extensions spécifiées (bmp et png). Cette fonction n' ouvre pas le fichier lui-même ! Elle renvoie seulement une chaîne avec le nom complet et le chemin d'accès du fichier. Si l'utilisateur appuie sur "Annuler", une chaîne vide "" est renvoyée. Vous pouvez également ajouter un répertoire à partir duquel lancer la recherche, et une légende à afficher en haut de la fenêtre au lieu de la légende par défaut.

Une chose importante à noter lorsque vous utilisez cette fonction est qu'elle vous accorde certaines autorisations pour ce fichier, pour la durée de votre jeu. Ainsi, une fois que vous avez obtenu le chemin d'accès au fichier de la part de l'utilisateur, vous pouvez l'envoyer à access encore et encore sans avoir à le demander. Cependant, comme cette fonction sert à lire un fichier, les autorisations accordées ne concernent que la lecture et l'enregistrement ne sera pas autorisé par le système d'exploitation. Si vous avez besoin de lire et d'écrire dans le fichier, vous pouvez obtenir la permission en utilisant la fonction get_save_filename().

NOTE: Cette fonction n'est valable que sur les cibles Windows et macOS.

Syntaxe :

get_open_filename_ext(filter, fname, directory, caption);

ArgumentsType Description
filter The file type filter and/or name.
fname The suggested filename to use.
directory The directory to start the search from.
caption The caption for the window.

Les retours :

Exemple :

var file;
file = get_open_filename_ext("text file|*.txt", "", working_directory, "Open a saved level");
if file != ""
{
    file_text_open_read(file);
}

Le code ci-dessus demandera à l'utilisateur de sélectionner un fichier texte à ouvrir sur le site working_directory, puis vérifiera le nom et le chemin du fichier renvoyé et, s'il existe, ouvrira le fichier pour le lire.
