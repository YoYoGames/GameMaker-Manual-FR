fichier_texte_ouvert_à_partir_de_chaîne

Cette fonction créera un fichier texte à partir d'une chaîne de caractères et l'ouvrira pour la lecture, en retournant le "handle" du fichier qui devra être utilisé dans tous les appels ultérieurs de la fonction fichier pour lire depuis ce fichier. Notez que ce fichier est temporaire et en lecture seule, et qu'en tant que tel, il sera supprimé de la mémoire dès qu'il sera fermé.

REMARQUE: Vous ne pouvez avoir qu'un maximum de 32 fichiers ouverts à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela permet d'écrire les informations et de libérer la mémoire associée au fichier.

AVERTISSEMENT ! Cette fonction peut ne pas fonctionner comme vous l'attendez, car GameMaker est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe :

file_text_open_from_string(string);

ArgumentsType Description
string The string to create the file from.

Les retours :

File ID or -1

Exemple :

file = file_text_open_from_string(reset_str);

Le code ci-dessus prend la chaîne stockée dans la variable "reset_str" et l'utilise pour créer un fichier texte en lecture seule. Le "handle" de ce fichier est ensuite stocké dans la variable "file" pour que toutes les autres fonctions de fichier puissent l'utiliser.
