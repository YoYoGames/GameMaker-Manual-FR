fichier_texte_ouvert_lecture

Cette fonction ouvre le fichier texte avec le nom de fichier indiqué en lecture seule, renvoyant l'identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les actions ultérieures à faire avec ce fichier. Si le fichier n'existe pas, la fonction renvoie la valeur -1.

REMARQUE: Vous ne pouvez avoir qu'un maximum de 32 fichiers ouverts à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela permet d'écrire les informations et de libérer la mémoire associée au fichier.

AVERTISSEMENT ! Cette fonction peut ne pas fonctionner comme vous l'attendez, car GameMaker est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe :

file_text_open_read(fname);

ArgumentsType Description
fname The name of the file to read from.

Les retours :

Text File ID or -1

Exemple :

file = file_text_open_read(working_directory + "level.txt");

Cela ouvrirait "level.txt" dans le même répertoire que le jeu et enregistrerait l'identifiant du fichier dans la variable "file".
