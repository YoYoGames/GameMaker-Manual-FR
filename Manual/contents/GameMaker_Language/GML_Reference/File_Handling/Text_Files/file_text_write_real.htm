fichier_texte_écrit_réel

Cette fonction permet d'écrire un nombre dans le fichier texte précédemment ouvert. Notez que comme la valeur à écrire peut être un nombre réel, toutes les décimales seront écrites avec un point "." comme séparateur. Si le fichier contient déjà des informations, celles-ci seront effacées et la chaîne sera écrite au début du fichier, sauf si vous avez ouvert le fichier avec la fonction file_text_open_append(). Vous pouvez également éviter cela en utilisant la fonction file_text_readln() avec la fonction file_text_eof() pour parcourir en boucle le contenu du fichier jusqu'à la fin, puis commencer à écrire.

Il est important de noter que lorsque vous écrivez de très grands nombres dans un fichier texte à l'aide de cette fonction, ils peuvent être traduits dans un format standard simplifié, comme "6.6624e+003", qui ne peut pas être relu correctement dans GameMaker. Pour éviter ce genre de problème, vous devriez plutôt convertir la valeur en une chaîne de caractères et utiliser la fonction file_text_write_string() à la place.

Syntaxe :

file_text_write_real(fileid, val);

ArgumentsType Description
fileid The id of the file to edit.
val The real value to write to the file.

Les retours :

Exemple :

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

Le code ci-dessus ouvre un fichier pour l'écriture et boucle ensuite sur deux tableaux, en écrivant chaque valeur de tableau sur une nouvelle ligne du fichier. Le fichier est ensuite fermé lorsque la boucle est terminée.
