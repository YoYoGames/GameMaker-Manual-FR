file_text_open_write

Cette fonction ouvre le fichier texte avec le nom de fichier indiqué pour l'écriture seulement (si le fichier n'existe pas, il est créé), renvoyant l'identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les actions ultérieures à faire avec ce fichier.

Notez que si le fichier ne peut pas être créé (à cause d'un nom de fichier illégal, par exemple), la fonction retournera -1.

Si le fichier existe déjà, l'utilisation de cette fonction entraînera son écrasement lors de l'enregistrement des données. Si vous souhaitez ajouter des données à un fichier tout en conservant intact son contenu existant, utilisez la fonction file_text_open_append().

REMARQUE: Vous ne pouvez avoir qu'un maximum de 32 fichiers ouverts à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela permet d'écrire les informations et de libérer la mémoire associée au fichier.

AVERTISSEMENT ! Cette fonction peut ne pas fonctionner comme vous l'attendez, car GameMaker est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe :

file_text_open_write(fname);

Arguments Type Description
fname Chaîne de caractères The name of the file to write to.

Les retours :

Text File ID or -1

Exemple :

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

Le code ci-dessus va ouvrir le fichier "level.txt" pour l'écrire, puis écrire la chaîne de caractères stockée dans la variable "level_data" avant de refermer le fichier.
