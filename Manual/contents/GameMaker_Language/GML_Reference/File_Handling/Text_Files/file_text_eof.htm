texte_fichier_eof

Cette fonction renvoie true lorsque la fin d'un fichier texte ouvert donné a été atteinte ou false sinon.

Syntaxe :

file_text_eof(fileid);

ArgumentsType Description
fileid The id of the file to check.

Les retours :

Exemple :

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    str[num++] = file_text_readln(file);
}
file_text_close(file);

Le code ci-dessus ouvre un fichier pour l'écrire, puis passe en boucle les lignes de texte déjà écrites dans le fichier jusqu'à la fin, en stockant chaque ligne dans le tableau "str".
