fichier_texte_eoln

Avec cette fonction, vous pouvez demander à GameMaker de vérifier le fichier actuellement ouvert pour voir si la ligne en cours de lecture est terminée. La fonction renvoie true si la fin de la ligne a été atteinte et false sinon.

Syntaxe :

file_text_eoln(fileid);

ArgumentsType Description
fileid The id of the file to check.

Les retours :

Exemple :

var file = file_text_open_read(working_directory + "Game_Data.txt");
var num = 0; while (!file_text_eoln(file))
{
    score_array[num] = file_text_read_real(file);
    num++;
}
file_text_close(file);

Le code ci-dessus ouvre un fichier en lecture puis lit les valeurs d'une seule ligne dans un tableau.
