load_csv

Cette fonction chargera un fichier au format CSV et le convertira en une grille DS, en renvoyant la valeur ID unique de la grille créée.

Votre fichier CSV doit respecter la spécification CSV. Si votre syntaxe CSV n'est pas correcte, vous obtiendrez des résultats inattendus lors du chargement du fichier.

Syntaxe :

load_csv(filename)

Arguments Type Description
filename Chaîne de caractères The name of the file to open (as a string)

Les retours :

DS Grid ID

Exemple :

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
    for (var j = 0; j < hh; j++;)
    {
        draw_text(xx, yy, string(file_grid[# i, j]));
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

Le code ci-dessus ouvrira le fichier CSV donné et stockera la grille DS retournée dans la variable "file_grid". Cette grille est ensuite analysée en quelques clics à l'adresse for et son contenu est affiché à l'écran.
