md5_file

En cryptographie, MD5 (Message-Digest algorithm 5) est une fonction de hachage cryptographique largement utilisée avec une valeur de hachage de 128 bits et a été employée dans une grande variété d'applications de sécurité. Elle est également couramment utilisée pour vérifier l'intégrité des fichiers et des chaînes de caractères. Cette fonction prend le fichier donné et génère un MD5 unique pour ce fichier, qui peut ensuite être stocké pour une utilisation ultérieure.

REMARQUE: MD5 n'est pas complètement sécurisé et peut être cassé. Voir cette page pour plus d'informations.

Syntaxe :

md5_file(filename)

ArgumentsType Description
filename The file to generate the MD5 hash for.

Les retours :

Exemple :

hash = md5_file(working_directory + "game_data.ini")

Le code ci-dessus génère un hachage MD5 pour le fichier spécifié et stocke la valeur renvoyée dans la variable "hash".
