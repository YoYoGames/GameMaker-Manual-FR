zip_unzip

Cette fonction ouvrira un fichier zip stocké et extraira son contenu dans le répertoire donné. Notez que si vous ne fournissez pas un chemin complet vers le répertoire ZIP, la racine du lecteur actuel sera utilisée, et si vous voulez le placer dans un chemin relatif au répertoire de travail du bundle du jeu, vous devez utiliser la variable working_directory comme partie du chemin (les chemins relatifs utilisant "." ou ".." ne fonctionneront pas comme prévu et doivent être évités). Notez également que le fichier zip doit faire partie du paquet de jeu (c'est-à-dire un fichier inclus) ou avoir été téléchargé dans la zone de stockage à l'aide de http_get_file().

La fonction renvoie une valeur indiquant le nombre de fichiers extraits, ou elle renvoie 0 ou moins si l'extraction a échoué.

Syntaxe :

zip_unzip(zip_file, target_directory)

ArgumentsType Description
zip_file The zip file to open
target_directory The target directory to extract the files to

Les retours :

Exemple :

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
    show_debug_message("Extraction Failed!");
}

Le code ci-dessus ouvrira le fichier zip stocké dans le répertoire "downloads" et extraira son contenu dans le répertoire "extracted" (en créant ce répertoire s'il n'existe pas déjà), puis vérifiera que l'extraction a été correcte, en affichant un message de débogage en cas d'échec.
