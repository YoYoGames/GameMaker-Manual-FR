sha1_file

En cryptographie, SHA-1 est une fonction de hachage cryptographique conçue par la National Security Agency des États-Unis. Elle est employée dans plusieurs applications très répandues : Git, où elle sert à vérifier les modifications de fichiers, et les protocoles TLS et SSL, PGP, SSH, S/MIME et IPsec. Cette fonction prend un fichier d'entrée et renvoie un condensé de message de 160 bits au format ASCII, unique à ce fichier, qui peut être utilisé pour la vérification de l'intégrité à une date ultérieure.

NOTE: SHA-1 n'est pas complètement sécurisé et peut être cassé. Voir cette page pour plus d'informations.

Syntaxe :

sha1_file(filename)

Arguments Type Description
filename Chaîne de caractères The file to generate the sha1 hash for.

Les retours :

String

Exemple :

hash = sha1_file(working_directory + "game_data.ini")

Le code ci-dessus va générer un hachage sha1 pour le fichier spécifié et stocker la valeur retournée dans la variable "hash".
