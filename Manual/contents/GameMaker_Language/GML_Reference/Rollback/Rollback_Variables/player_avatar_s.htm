joueur_avatar_sprite

Cette variable stocke le sprite de l'instance de joueur actuelle.

Cette variable n'existe que dans les instances de joueur créées par rollback_define_player()et est différente pour chaque joueur.

Lesjoueurs invités afficheront un espace réservé sprite.

Syntaxe :

player_avatar_sprite;

Les retours :

Sprite Asset

Exemple :

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

Cela affiche l'avatar de l'instance du joueur qui exécute le code, dans une zone 64x64, 256px au-dessus de l'instance.
