rollback_confirmed_frame

Cette variable globale stocke le dernier numéro de trame où les entrées de tous les joueurs ont été synchronisées et confirmées.

Vous pouvez comparer cela à rollback_current_frame pour vérifier si la trame actuelle a des entrées confirmées pour tous les joueurs.

Vous pouvez également utiliser la fonction rollback_sync_on_frame() pour vérifier si la trame actuelle est synchronisée.

Syntaxe :

rollback_confirmed_frame

Les retours :

Real

Exemple :

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

Ce code vérifie si le cadre actuel a des entrées confirmées de tous les joueurs, en vérifiant si la valeur du cadre actuel est égale à la valeur du cadre confirmé.

Si c'est le cas, il exécute une fonction pour vérifier si la condition de fin de partie a été satisfaite, afin de pouvoir mettre fin à la partie.

Ceci est fait pour éviter un état de fin de jeu basé sur la prédiction, et attend une trame entièrement synchronisée avant de tenter de vérifier les conditions de fin de jeu.
