player_type

Cette variable indique si l'instance du joueur est un utilisateur connecté à GX.games ou un invité. Elle peut retourner l'une des chaînes suivantes :

Chaîne de caractères Description
"Guest" Le joueur n'est pas connecté à GX.games.
"User" Le joueur est connecté à GX.games

Les utilisateurs invités reçoivent des avatars et des noms fictifs.

Cette variable n'existe que dans les instances de joueur créées par rollback_define_player()et est différente pour chaque joueur.

Syntaxe :

player_type

Les retours :

String
