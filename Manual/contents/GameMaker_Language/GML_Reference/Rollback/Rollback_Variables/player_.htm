nom du joueur

Cette variable stocke le nom de l'instance actuelle du joueur.

Cette variable n'existe que dans les instances de joueur créées par rollback_define_player()et est différente pour chaque joueur.

Lesjoueurs invités renverront un nom fictif.

Syntaxe :

player_name;

Les retours :

String

Exemple :

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

Ceci dessine le nom du joueur 100px au-dessus de son instance, avec un alignement centre-milieu.
