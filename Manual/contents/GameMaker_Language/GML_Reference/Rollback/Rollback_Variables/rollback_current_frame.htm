Retour à la case départ

Il s'agit d'une variable globale qui stocke le nombre d'images qui se sont écoulées depuis le début du jeu multi-joueurs.

Cette variable doit être utilisée à la place de current_time et d'autres variables similaires, comme expliqué dans Variables temporelles.

Syntaxe :

rollback_current_frame

Les retours :

Real

Exemple :

image_angle = dsin(rollback_current_frame) * 90;

Le code ci-dessus utilise rollback_current_frame avec dsin() pour calculer une valeur oscillante entre 1 et -1, qui est multipliée par 90 et appliquée à l'angle de l'instance.
