player_avatar_url

Cette variable stocke l'URL de l'avatar du joueur, qui peut être chargé à l'aide de sprite_add().

Vous pouvez utiliser player_avatar_sprite qui a déjà le sprite chargé.

Cette variable n'existe que dans les instances de joueur créées par rollback_define_player()et est différente pour chaque joueur.

Lesjoueurs invités afficheront un espace réservé sprite.

Syntaxe :

player_avatar_url;

Les retours :

String
