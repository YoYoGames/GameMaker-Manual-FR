rollback_game_running

Cette variable globale indique si la partie de rollback est en cours d'exécution. Si elle est vraie, cela signifie que le jeu de retour a commencé et est en cours d'exécution. Dans tous les autres cas, la valeur est fausse.

Syntaxe :

rollback_game_running

Les retours :

Boolean

Exemple :

if (!rollback_game_running) return;

x += move_x;
y += move_y;

Le code ci-dessus serait exécuté dans l'événement Step d'un objet joueur. Lorsque le jeu de retour n'est pas en cours d'exécution, il quitte l'événement, de sorte que le joueur ne peut pas être contrôlé tant que le jeu de retour n'a pas commencé.
