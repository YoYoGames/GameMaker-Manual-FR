Rollback Multiplayer

Rollback multiplayer dans GameMaker vous permet de mettre en place et de créer rapidement un jeu multijoueur en réseau pour la cible Opera GX.

Cette fonctionnalité utilise les serveurs GX pour connecter les joueurs et synchroniser les entrées, de sorte qu'aucune configuration de serveur n'est requise par le développeur pour démarrer une nouvelle partie en ligne.

Alors que la fonctionnalité en ligne ne peut être utilisée qu'avec la cible Opera GX, vous pouvez utiliser "Sync Test" pour tester votre jeu multijoueur localement, avec des entrées aléatoires ou fictives appliquées aux autres joueurs. Sync Test est fonctionnel sur les cibles Windows et macOS en plus d'Opera GX.
