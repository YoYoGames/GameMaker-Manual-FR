rollback_get_player_prefs

Cette fonction renvoie le jeu de préférences pour l'ID de joueur donné.

Si l'ID d'un joueur n'est pas spécifié, la fonction doit être appelée dans une instance de joueur, auquel cas les préférences de ce joueur seront retournées.

Vous pouvez également utiliser la variable d'instance player_prefs.

Cette fonction renvoie les dernières préférences qui ont été synchronisées pour le joueur donné. Ainsi, la définition des préférences ne mettra pas immédiatement à jour les préférences que cette fonction renvoie.

Veuillez lire : Préférences des joueurs

Syntaxe :

rollback_get_player_prefs([player_id]);

Arguments Type Description
player_id Real OPTIONAL L'ID du joueur pour lequel les préférences sont retournées.

Les retours :

Any

Exemple :

var _my_prefs = rollback_get_player_prefs() ;

image_index = _my_prefs.character ;

Le code ci-dessus récupère les préférences définies pour le joueur actuel, et applique une préférence à une variable d'instance.
