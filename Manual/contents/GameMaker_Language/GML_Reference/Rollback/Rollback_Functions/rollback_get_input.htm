rollback_get_input

Cette fonction renvoie un struct contenant les valeurs d'entrée pour un joueur. Elle prend un argument facultatif spécifiant l'ID du joueur pour lequel les entrées seront retournées.

Lorsqu'elle est utilisée dans une instance de joueur qui a été créée par rollback_define_player()vous n'avez pas besoin de spécifier l'ID du lecteur, vous pouvez donc appeler cette fonction sans aucun argument.

Lorsqu'il est utilisé dans d'autres instances, ou dans des instances de joueur créées manuellement, vous devez spécifier un ID de joueur.

Pour plus d'informations sur les entrées qui sont renvoyées et sur la façon dont elles peuvent être modifiées, voir Définition des entrées.

Syntaxe :

rollback_get_input(player_id);

Arguments Type Description
player_id Real OPTIONAL L'ID du lecteur pour lequel les entrées seront retournées.

Les retours :

Struct

Exemple :

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

Le code ci-dessus récupère l'entrée struct dans une instance de joueur qui a été automatiquement créée par le système Rollback. Sur la base des valeurs renvoyées, il déplace l'instance de 2 pixels dans chaque direction.

Pour un exemple détaillé, voir Création d'un jeu multi-joueurs.
