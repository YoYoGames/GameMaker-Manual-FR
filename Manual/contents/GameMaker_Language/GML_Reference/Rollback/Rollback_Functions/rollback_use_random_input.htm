retour en arrière_utilisation d'une entrée aléatoire

Lorsque vous jouez une partie en utilisant Sync Test, tous les joueurs "distants" reçoivent des valeurs aléatoires pour leurs entrées définies, comme moyen de test de base. Vous pouvez désactiver ce comportement en appelant cette fonction avec false comme argument, et l'activer à nouveau en spécifiant true.

Syntaxe :

rollback_use_random_input(enabled);

Arguments Type Description
enabled Booléen Specifies whether random input should be enabled (true) ou désactivé(false). Par défaut, il est activé.

Les retours :

N/A

Exemple :

rollback_use_random_input(false);

Le code ci-dessus désactive l'entrée aléatoire pendant le test de synchronisation.
