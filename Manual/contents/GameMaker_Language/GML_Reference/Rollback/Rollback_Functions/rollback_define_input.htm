entrée_définition_du_retour

Cette fonction est utilisée pour définir des entrées personnalisées pour tous les joueurs. Elle prend une struct qui doit contenir toutes les entrées avec leurs constantes d'entrée assignées.

Pour savoir comment attribuer des entrées personnalisées, voir Commandes personnalisées.

Si elle est utilisée, cette fonction doit être exécutée avant la fonction "joindre/créer".

Syntaxe :

rollback_define_input(input_struct);

Arguments Type Description
input_struct Structure A struct containing input names as variables, and their assigned inputs as values.

Les retours :

N/A

Exemple :

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

Le code ci-dessus définit des entrées personnalisées pour tous les joueurs. Les deux dernières entrées utilisent plusieurs contrôles, en les listant dans un tableau.

Pour un exemple détaillé, voir Définir les entrées.
