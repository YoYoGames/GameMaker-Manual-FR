rollback_chat

Cette fonction est utilisée pour envoyer un message de type chaîne aux autres joueurs.

Vous spécifiez une chaîne de caractères à envoyer, dont la taille maximale est de 128 octets.

Lorsque le deuxième argument n'est pas spécifié, le message est envoyé à tous les joueurs, y compris l'expéditeur.

Vous pouvez spécifier un ID de joueur dans le second argument qui est le joueur qui recevra le message.

Comment l'utiliser ?

Cette fonction est destinée à être utilisée "localement", et donc à résulter d'une entrée locale, et non d'une entrée Rollback.

Voir l'exemple d'utilisation en bas de page. Écoutez les messages reçus dans l'événement Rollback Event.

Syntaxe :

rollback_chat(message, [to]);

Arguments Type Description
message Chaîne de caractères The message to send, as a string.
to Real OPTIONAL L'ID du joueur auquel le message sera envoyé. Si vous ne spécifiez pas cet argument ou si vous spécifiez -1, le message sera envoyé à tous les joueurs.

Les retours :

N/A

Exemple :

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

Si la touche entrée est pressée localement, la keyboard_string est envoyée à tous les joueurs. Cette chaîne de caractères est ensuite effacée afin qu'un nouveau message puisse être tapé.

Cela déclenchera l'événement Rollback Event lorsque le message sera reçu par tous les clients (y compris l'expéditeur).

Événement de retour en arrière

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
