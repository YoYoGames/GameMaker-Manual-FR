rollback_use_manual_start

Par défaut, une partie multijoueur commence automatiquement lorsque tous les joueurs se sont inscrits.

L'appel de cette fonction désactive ce comportement de démarrage automatique, de sorte que le jeu attend que vous le démarriez manuellement.

Si elle est utilisée, cette fonction doit être exécutée avant la fonction "joindre/créer".

Syntaxe :

rollback_use_manual_start();

Les retours :

N/A

Exemple :

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

Le code ci-dessus active le démarrage manuel, puis rejoint ou crée une nouvelle partie multi-joueurs.
