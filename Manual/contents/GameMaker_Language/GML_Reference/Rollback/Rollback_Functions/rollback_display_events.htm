retour_affichage_événements

Par défaut, lorsque vous lancez le jeu, vous voyez des messages de débogage dans le coin supérieur gauche de votre jeu, affichant les événements multijoueurs au fur et à mesure qu'ils se produisent. Vous pouvez désactiver ces messages en appelant cette fonction avec false comme argument, et les réactiver en spécifiant true.

Syntaxe :

rollback_display_events(enabled);

Arguments Type Description
enabled Booléen Specifies whether event display should be enabled (true) ou désactivé(false). Par défaut, il est activé.

Les retours :

N/A

Exemple :

rollback_display_events(false);

Le code ci-dessus a désactivé l'affichage des événements.
