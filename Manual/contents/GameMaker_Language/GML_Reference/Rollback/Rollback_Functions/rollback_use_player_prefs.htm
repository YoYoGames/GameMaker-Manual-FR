rollback_use_player_prefs

Cette fonction active les préférences des joueurs et vous permet de définir les préférences par défaut pour tous les joueurs.

Les préférences par défaut peuvent être de n'importe quel type. Si vous ne spécifiez pas de préférences par défaut, tous les joueurs doivent avoir des préférences définies avant que le jeu ne puisse commencer.

L'appel de cette fonction désactivera le démarrage automatique, vous devrez donc démarrer le jeu manuellement. Voir : Démarrage manuel ou automatique

Si elle est utilisée, cette fonction doit être exécutée avant la fonction "joindre/créer".

Veuillez lire : Préférences des joueurs

Syntaxe :

rollback_use_player_prefs([default]);

Arguments Type Description
default Tout OPTIONAL Les préférences par défaut à définir pour tous les joueurs.

Les retours :

N/A

Exemple :

rollback_use_player_prefs({
caractère : 0,
couleur : c_white
}) ;

Le code ci-dessus active les préférences du joueur et définit les préférences par défaut dans un struct.
