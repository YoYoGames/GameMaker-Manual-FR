rollback_get_info

Cette fonction renvoie un struct contenant des informations sur un joueur. Elle ne doit être utilisée que pour gérer manuellement les instances de joueurs. Si vous utilisez rollback_define_player()alors chaque instance de joueur recevra automatiquement ces informations sous forme de variables.

La fonction prend l'ID du joueur comme argument, et renvoie un struct avec les variables suivantes :

Variable Type Description
player_avatar_url Chaîne de caractères URL pointant vers l'image de l'avatar du joueur
player_avatar_sprite Sprite Asset Sprite contenant l'avatar du joueur
player_name Chaîne de caractères Le nom du joueur
player_type Chaîne de caractères Indique si le joueur est connecté à GX.games ou non.

Sera "Guest" pour les utilisateurs invités, et "User" pour les utilisateurs connectés.
player_user_id Chaîne de caractères L'ID du joueur sur GX.games
player_prefs Tout Les préférences définies pour le lecteur

Syntaxe :

rollback_get_info(player_id);

Arguments Type Description
player_id Real The ID of the player for which information will be returned.

Les retours :

Struct
