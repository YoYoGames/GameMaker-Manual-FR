Fonctions de retour en arrière

Les fonctions suivantes sont utilisées pour démarrer, rejoindre ou quitter une partie multi-joueurs :

  • retour_joint_jeu
  • retour_création_jeu
  • retour en arrière_début_jeu
  • annuler_sortir_du_jeu

Les fonctions suivantes sont utilisées pour définir et obtenir des informations sur votre jeu :

  • Définir
    • rollback_define_player
    • entrée_définition_du_retour
    • rollback_define_mock_input
    • délai de définition de l'image d'entrée du retour en arrière
    • rollback_use_manual_start
    • retour_utilisation_prefs_joueur
    • rollback_set_player_prefs
  • Obtenez
    • rollback_get_input
    • rollback_get_info
    • rollback_get_player_prefs

Les fonctions suivantes sont prévues pour être utilisées pendant un jeu actif :

  • discussion de retour
  • rollback_sync_on_frame

Les fonctions suivantes sont utilisées pour les tests, ou pour activer/désactiver les fonctions liées aux tests :

  • retour en arrière_utilisation d'une entrée aléatoire
  • retour_affichage_événements
