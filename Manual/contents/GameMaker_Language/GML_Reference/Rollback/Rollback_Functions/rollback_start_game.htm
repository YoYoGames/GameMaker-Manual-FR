rollback_start_game

Cette fonction force la partie multijoueur en cours à commencer, avant que tous les joueurs ne se soient joints. Vous devez d'abord créer une partie multijoueur à l'aide de la fonction rollback_create_game().

Le système attend normalement que tous les joueurs se joignent à lui avant de lancer automatiquement la partie. Cette fonction peut être utilisée pour démarrer la partie plus tôt, avant que tous les joueurs ne se soient joints.

Vous pouvez appeler rollback_use_manual_start() pour que Rollback attende cette fonction même après que tous les joueurs se soient inscrits.

L'appel de cette fonction déclenchera l'événement Rollback Start.

Syntaxe :

rollback_start_game();

Les retours :

N/A

Exemple :

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

Si le jeu Rollback n'est pas en cours d'exécution et que vous appuyez sur la touche Entrée, le jeu démarre.

Cela permet au joueur de commencer la partie manuellement avant que tous les joueurs ne se joignent à lui.
