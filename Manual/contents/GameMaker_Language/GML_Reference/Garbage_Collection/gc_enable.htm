gc_enable

Cette fonction permet d'activer ou de désactiver le ramasse-miettes. Appeler la fonction avec true comme argument l'active et utiliser false le désactive (non recommandé). Il est activé par défaut.

Syntaxe :

gc_enable(enable);

ArgumentsType Description
enable enable (true) ou de désactiver (false) le garbage collector.

Les retours :

Exemple :

if (global.debug == true)
{
    gc_enable(false);
}

Le code ci-dessus désactive la collecte des déchets si la variable globale donnée est true.
