gc_is_enabled

Avec cette fonction, vous pouvez vérifier si le ramasseur d'ordures est activé ou non. La fonction renverra true s'il est activé ou false sinon.

Syntaxe :

gc_is_enabled();

Les retours :

Exemple :

if (!gc_is_enabled())
{
    gc_enable(true);
}

Le code ci-dessus vérifie si le garbage collector est activé et s'il ne l'est pas, il l'active.
