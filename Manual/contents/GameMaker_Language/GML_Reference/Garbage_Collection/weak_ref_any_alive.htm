weak_ref_any_alive

Avec cette fonction, vous pouvez vérifier la référence faible de diverses structures pour voir si elles sont toujours "vivantes" ou non. Vous fournissez un tableau de références faibles à vérifier (chaque référence faible doit avoir été créée en utilisant la fonction weak_ref_create()), et la fonction retournera true si n'importe quel des structs sont toujours référencées quelque part ou false si elles ne le sont pas et ont été collectées. Notez que si vous fournissez un tableau dont l'une des valeurs n'est pas une référence faible, la fonction retournera undefined.

Notez que la fonction prend également deux arguments optionnels, où le premier vous permet de fournir un index initial dans le tableau à vérifier, ainsi qu'un argument pour définir le nombre de positions (longueur) à partir de cet index à vérifier. S'ils sont spécifiés, seuls les indices du tableau compris dans ces paramètres seront vérifiés.

Syntaxe :

weak_ref_any_alive(array, [index], [length]);

Arguments Type Description
array Tableau de références faibles de structures Array containing weak references to the structs that you want to check.
[index] Real OPTIONAL L'index dans le tableau à partir duquel la vérification doit commencer
[length] Real OPTIONAL Le nombre de positions, à partir de l'indice donné, pour vérifier la présence de

Les retours :

Boolean (or undefined)

Exemple :

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

Le code ci-dessus vérifie un tableau de références faibles et si l'une d'entre elles est toujours vivante, une instance est détruite.
