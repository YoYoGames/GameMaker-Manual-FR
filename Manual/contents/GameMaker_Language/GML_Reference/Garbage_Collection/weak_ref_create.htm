weak_ref_create

Avec cette fonction, vous pouvez créer une référence faible à un struct qui peut ensuite être utilisée pour vérifier si le struct est toujours "vivant" (référencé) ou non dans le jeu. Vous fournissez la référence au struct que vous voulez suivre, et la fonction renverra un autre struct qui est une référence faible à ce struct. Notez que vous pouvez vérifier si une référence est "forte" ou "faible" en utilisant la fonction instanceof(), car une référence forte renverra soit "struct" ou le nom de la fonction de construction qui a créé le struct, soit "weakref" si c'est une référence faible. Notez également que la référence faible struct aura une variable "ref" à laquelle il sera possible d'accéder pour obtenir la référence forte du struct en question, à moins qu'il n'ait été mis au rebut, auquel cas il retournera undefined.

Syntaxe :

weak_ref_create(struct_to_track);

Arguments Type Description
struct_to_track Structure The struct that you want to create a weak reference for

Les retours :

Struct Weak Reference

Exemple :

inventory_ref = weak_ref_create(inventory);

Le code ci-dessus crée une référence faible à un struct et la stocke dans une variable d'instance pour une utilisation ultérieure.
