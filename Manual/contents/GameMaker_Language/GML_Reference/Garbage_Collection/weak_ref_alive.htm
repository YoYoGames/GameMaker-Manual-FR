weak_ref_alive

Avec cette fonction, vous pouvez vérifier la référence faible d'un struct pour voir s'il est toujours "vivant" ou non. Vous fournissez la référence faible à vérifier (telle que retournée par la fonction weak_ref_create()), et la fonction renvoie true si struct est toujours référencé quelque part ou false si ce n'est pas le cas et qu'il a été collecté. Notez que si vous fournissez une valeur qui n'est pas une référence faible, la fonction retournera undefined.

Syntaxe :

weak_ref_alive(weak_ref);

Arguments Type Description
weak_ref Référence faible de la structure The weak reference to the struct you want to check.

Les retours :

Boolean (or undefined)

Exemple :

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

Le code ci-dessus vérifie une référence faible à un struct et si elle est toujours vivante, la variable qui la référence est mise à -1, déréférençant le struct et lui permettant d'être collecté par les ordures.
