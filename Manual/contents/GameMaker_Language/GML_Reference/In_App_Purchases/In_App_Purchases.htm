Achats dans l'application

Vous pouvez ajouter des achats intra-applicatifs à vos jeux en utilisant les extensions disponibles sur le site Marketplace et en suivant les guides de configuration indiqués ci-dessous (chaque extension est accompagnée de son propre "mini-manuel" qui explique comment l'utiliser et énumère toutes les fonctions qu'elle contient) :

  • Android : Configurer les IAPs
  • iOS : Configuration des PAI
  • macOS : Configuration des PAI

Les versions précédentes de GameMaker comportaient des fonctions d'achat dans l'application intégrées à runner, mais ces fonctions sont désormais obsolètes et ne fonctionneront plus :

  • iap_data
  • iap_activate
  • iap_status
  • iap_enumerate_products
  • iap_restore_all
  • iap_acquire
  • iap_consume
  • iap_product_details
  • iap_purchase_details
