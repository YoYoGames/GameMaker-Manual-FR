path_get_closed

Cette fonction peut être utilisée pour retourner si le chemin est marqué comme fermé (vrai) ou ouvert (faux), c'est-à-dire si le chemin fait une boucle ou s'il a un début et une fin définitifs.

Syntaxe :

path_get_closed(index);

Arguments Type Description
index Path Asset The index of the path to check.

Les retours :

Boolean

Exemple :

state = path_get_closed(pth_Patrol);

Cela donnera à "state" la valeur true ou false selon l'état de fermeture du chemin indexé dans "pth_Patrol".
