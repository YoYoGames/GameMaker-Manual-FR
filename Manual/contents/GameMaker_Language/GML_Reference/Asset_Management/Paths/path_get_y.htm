chemin_get_y

Avec cette fonction, vous pouvez obtenir la coordonnée y d'une position sur un chemin donné. 0 est le début du chemin, 1 est la fin du chemin, et tout ce qui se trouve entre les deux équivaut à la distance normalisée sur la longueur du chemin. Cette valeur ne doit pas nécessairement être un point de définition du chemin, elle peut être n'importe où sur celui-ci.

Syntaxe :

path_get_y(ind, pos);

Arguments Type Description
index Path Asset The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end).

Les retours :

Real

Exemple :

y = path_get_y(pth_WallWalk, 0.5);

Ceci définira le y de l'instance appelante à la coordonnée y du point situé exactement à mi-chemin du chemin donné.
