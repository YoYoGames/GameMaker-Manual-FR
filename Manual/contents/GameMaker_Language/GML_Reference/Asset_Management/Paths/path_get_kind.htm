path_get_kind

Les chemins peuvent être lisses ou droits (un chemin lisse calcule une trajectoire courbe autour des points de définition, tandis qu'un chemin droit va simplement d'un point à un autre). Cette fonction peut être utilisée pour déterminer si un chemin donné est lisse (true) ou non (false).

Syntaxe :

path_get_kind(index);

ArgumentsType Description
index The index of the path to check.

Les retours :

Exemple :

path_kind = path_get_kind(pth_Patrol);

Cela donnera à la variable "path_kind" la valeur true ou false en fonction du type de chemin dont l'indice est donné.
