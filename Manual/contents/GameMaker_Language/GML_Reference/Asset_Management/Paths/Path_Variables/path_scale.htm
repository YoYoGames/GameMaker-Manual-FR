path_scale

Cette valeur peut être utilisée pour obtenir ou définir l'échelle du chemin actuellement assigné à l'instance (comme défini par la fonction path_start()) avec une valeur par défaut de 1. Il s'agit d'une valeur scalaire, donc 1 correspond à une échelle de 1:1, tandis qu'une valeur de 2, par exemple, doublera l'échelle et une valeur de 0.5 la réduira de moitié.

Syntaxe :

path_scale;

Les retours :

Exemple :

path_scale = 1 + random(2);

Le code ci-dessus définira l'échelle du chemin actuel à une taille aléatoire entre 1 (sa taille originale) et 3 (3 fois sa taille originale).
