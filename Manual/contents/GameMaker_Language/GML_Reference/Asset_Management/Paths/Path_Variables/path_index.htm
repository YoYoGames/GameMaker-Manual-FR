chemin_index

Toutes les ressources de GameMaker ont un numéro d'identification unique. La variable path_index est une variable en lecture seule qui contient la valeur d'index d'un actif de chemin donné qui a été attribué à une instance à l'aide de la fonction path_start(). Si aucun chemin n'a été attribué à l'instance, la variable prend la valeur -1.

Syntaxe :

path_index;

Les retours :

Path Asset

Exemple :

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

Le code ci-dessus vérifie si un chemin a été attribué à l'instance, et si ce n'est pas le cas, il démarre un nouveau chemin (en l'attribuant en même temps à path_index ).
