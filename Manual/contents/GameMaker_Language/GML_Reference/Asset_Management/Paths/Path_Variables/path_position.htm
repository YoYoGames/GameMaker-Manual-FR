path_position

Cette fonction peut être utilisée pour obtenir ou définir la position d'une instance le long d'un chemin. La valeur est normalisée de 0 à 1, donc si vous la fixez, par exemple, à 0.5, l'instance sera déplacée exactement au milieu du chemin.

Syntaxe :

path_position;

Les retours :

Exemple :

path_position = random(1);

Le code ci-dessus place l'instance à une position aléatoire n'importe où sur le chemin actuel.
