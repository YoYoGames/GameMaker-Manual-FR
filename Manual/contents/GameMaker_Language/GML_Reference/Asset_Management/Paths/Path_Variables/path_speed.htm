vitesse du chemin

Vous pouvez utiliser cette fonction pour obtenir ou définir la vitesse d'un chemin après qu'il ait été lancé à l'aide de la fonction path_start(). Vous pouvez utiliser des valeurs négatives pour signifier que l'instance doit suivre le chemin en sens inverse. le chemin en sens inverse.

Syntaxe :

path_speed;

Les retours :

Exemple :

path_speed = -1 + random(2);

Le code ci-dessus permettra à l'instance de parcourir le chemin à une vitesse aléatoire comprise entre -1 et 1.
