Chemins d'accès

GameMaker dispose d'une vaste collection de fonctions et de variables intégrées permettant d'utiliser les chemins dans votre jeu. Cette section traite des fonctions de base utilisées pour traiter les chemins qui ont été précédemment créés dans l'éditeur de chemins, mais il existe d'autres fonctions qui vous permettent de créer vos propres chemins en utilisant du code et de les manipuler de différentes manières.

Les fonctions de base suivantes traitent des chemins :

  • path_start
  • path_end
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • path_get_number
  • path_get_point_speed
  • path_get_point_x
  • path_get_point_y
  • path_get_precision
  • path_get_speed
  • path_get_x
  • path_get_y

Pour faciliter l'utilisation de debugging, vous disposez également d'une fonction permettant de le dessiner à l'écran :

  • draw_path

Chaque instance d'un objet possède également une série de variables intégrées conçues pour faciliter l'utilisation des chemins. Ces variables sont énumérées dans la section suivante :

  • Variables de chemin

Enfin, il est possible de manipuler le contenu d'un chemin ainsi que de créer de nouveaux chemins lorsque le jeu est en cours. Les fonctions qui permettent ces choses et plus encore sont disponibles dans la section suivante :

  • Manipulation des chemins
