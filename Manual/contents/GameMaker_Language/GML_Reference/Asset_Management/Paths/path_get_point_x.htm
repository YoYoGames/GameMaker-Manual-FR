chemin_get_point_x

Cette fonction renvoie la position x (en coordonnées de pièce) du point que vous avez saisi pour la trajectoire que vous indexez. Si le point est en dehors de la plage de la trajectoire (ex : une trajectoire a 8 points et vous demandez la position x du point 10) alors une valeur de 0 sera retournée.

Syntaxe :

path_get_point_x(index, n);

Arguments Type Description
index Path Asset The index of the path to check.
n Real The point number to check.

Les retours :

Real

Exemple :

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

Le code ci-dessus vérifiera si une instance se trouve à la fin d'un chemin. Si c'est le cas, il choisira un point aléatoire sur le chemin et déplacera l'instance vers ce point.
