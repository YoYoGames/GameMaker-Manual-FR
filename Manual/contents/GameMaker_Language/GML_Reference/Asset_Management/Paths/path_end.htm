chemin_fin

L'appel de cette fonction mettra fin au chemin actuel que suit l'instance, tel qu'il a été défini lorsque la fonction path_start() a été appelée.

Syntaxe :

path_end();

Les retours :

Exemple :

if place_meeting(x, y, obj_Blocker)
{
    path_end();
}

Le code ci-dessus mettra fin à la trajectoire actuelle si l'instance détecte une collision avec une instance de l'objet donné.
