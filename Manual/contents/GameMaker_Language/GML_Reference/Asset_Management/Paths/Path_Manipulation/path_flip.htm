path_flip

Cette fonction prend tous les points du chemin et les retourne le long de l'axe horizontal. Si ce n'est pas ce dont vous avez besoin, vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe :

path_flip(index);

ArgumentsType Description
index The index of the path to flip.

Les retours :

Exemple :

path_flip(mypath);

Cela ferait basculer le chemin indexé dans la variable "mypath" le long de l'axe horizontal.
