path_assign

Avec cette fonction, vous pouvez copier les données de chemin d'un chemin à un autre. Le chemin vers lequel la copie est effectuée sera d'abord effacé (s'il a des points de chemin) et sera complètement écrasé par le chemin à partir duquel la copie est effectuée. copié. Aucun chemin n'est supprimé dans le processus et le résultat est deux chemins, avec deux index différents, mais la même forme et les mêmes points. En général, vous voudriez utiliser cette fonction sur un chemin créé en utilisant path_add(), car si vous l'utilisez sur un actif de chemin, cela affectera de façon permanente le chemin pour toutes les instances du jeu, du moment où la fonction est utilisée jusqu'à la fin du jeu.

Syntaxe :

path_assign(index, path);

ArgumentsType Description
index The index of the path to be overwritten.
path The index of the path that will overwrite 'index'.

Les retours :

Exemple :

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

Le code ci-dessus créera un nouveau chemin vide indexé dans la variable "mypath" et copiera ensuite les données du chemin à partir de l'une des trois ressources de chemin disponibles.
