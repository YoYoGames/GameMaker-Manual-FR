path_set_kind

Cette fonction peut être utilisée pour définir le type de chemin que vous souhaitez que la ressource spécifiée soit. Il peut s'agir d'un chemin en ligne droite (défini à 0) ou d'un chemin lissé (défini à 1), auquel cas la précision du chemin doit également être prise en compte (la précision peut être définie à l'aide de path_set_precision()). Cette fonction modifie la ressource réelle du chemin, et affectera de façon permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin de la partie. Si ce n'est pas ce dont vous avez besoin, vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'utilisez pas la fonction). sur la ressource dupliquée (n'oubliez pas d'appeler path_delete() sur la ressource lorsqu'elle n'est plus nécessaire).

Syntaxe :

path_set_kind(index, val);

ArgumentsType Description
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

Les retours :

Exemple :

path_set_kind(pth_Patrol, true);

Ceci définira le chemin donné comme étant un chemin "lisse".
