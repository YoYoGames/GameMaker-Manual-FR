path_insert_point

Avec cette fonction, vous pouvez insérer un nouveau point dans une trajectoire (la trajectoire peut avoir été créée dans l'éditeur de trajectoire ou par code en utilisant path_add()). Le point sera ajouté dans le chemin avant le point "n" qui est spécifié dans la fonction.

Syntaxe :

path_insert_point(index, n, x, y, speed);

Arguments Type Description
index Path Asset The index of the path to insert the point into.
n Real The defining point to insert the new point BEFORE.
x Real The x coordinate (relative to the path) of the new point.
y Real The y coordinate (relative to the path) of the new point.
speed Real The speed factor of the point.

Les retours :

N/A

Exemple :

path_insert_point(mypath, 0, 50, 50, 100);

Ceci va insérer un point au tout début du chemin indexé dans la variable "mypath", à (50,50), avec un facteur de vitesse de 100%.
