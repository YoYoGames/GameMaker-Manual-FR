path_clear_points

Avec cette fonction, vous pouvez supprimer toutes les définitions de points d'un chemin, créant ainsi un chemin "vide". Cela ne supprime pas le chemin, pour cela vous devez utiliser la fonction path_delete()Cependant, il convient de noter que cette fonction modifie l'actif du chemin, ce qui affecte de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu, depuis le moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe :

path_clear_points(index);

ArgumentsType Description
index The index of the path to clear.

Les retours :

Exemple :

if path_get_number(mypath) > 0
{
    path_clear_points(mypath);
}

Le code ci-dessus vérifie s'il y a des points sur le chemin indexé dans la variable "mypath" et s'il y en a, il efface le chemin.
