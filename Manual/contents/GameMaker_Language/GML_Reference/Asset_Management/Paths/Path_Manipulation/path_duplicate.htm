chemin_duplicate

Cette fonction prend un chemin et le copie dans un nouveau chemin. Le nouveau chemin est créé dans le processus, et son index est retourné pour être utilisé dans tous les appels ultérieurs pour utiliser ce nouveau chemin.

Syntaxe :

path_duplicate(index);

ArgumentsType Description
index The index of the existing path to duplicate.

Les retours :

Path Asset

Exemple :

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

Le code ci-dessus choisit l'une des quatre ressources de chemin et la duplique, en stockant l'index du nouveau chemin dans la variable "mypath".
