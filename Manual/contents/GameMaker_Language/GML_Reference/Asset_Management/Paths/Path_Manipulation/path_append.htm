path_append

Avec ce code, vous pouvez ajouter un chemin à un autre, en les joignant effectivement, et pour que cela fonctionne, les deux chemins doivent déjà exister. En général, il est préférable d'utiliser cette fonction sur les chemins créés à l'aide de path_add(), car si vous l'utilisez sur un actif de chemin, cela affectera de façon permanente le chemin pour les instances du jeu, du moment où la fonction est utilisée jusqu'à la fin du jeu.

NOTE Ceci ne supprimera pas le chemin ajouté du jeu. Il est toujours là, mais il n'a plus de points et si vous ne souhaitez plus l'utiliser, vous devez le supprimer avec la fonction path_delete().

Syntaxe :

path_append(index, path);

ArgumentsType Description
index The new index of the path.
path The index of the path that will be appended to 'index'.

Les retours :

N/A

Exemple :

path_append(mypath, mppath);
path_delete(mppath);

Le code ci-dessus ajoute les données de chemin indexées dans la variable "mppath" au chemin indexé dans la variable "mypath", puis supprime "mppath" de la mémoire.
