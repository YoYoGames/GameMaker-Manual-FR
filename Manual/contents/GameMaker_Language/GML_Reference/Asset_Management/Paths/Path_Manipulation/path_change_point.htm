point_de_changement_de_chemin

Avec cette fonction, vous pouvez modifier la position et/ou le facteur de vitesse de n'importe quel point précédemment défini pour une trajectoire (la trajectoire peut avoir été créée dans l'éditeur de trajectoire ou par code en utilisant path_add()). Si elle est utilisée sur un chemin à partir du navigateur d'actifs, notez que la fonction modifiera l'actif réel, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe :

path_change_point(index, n, x, y, speed);

Arguments Type Description
index Path Asset The index of the path to change a point of.
n Real The defining point to change the attributes of.
x Real The new x coordinate (relative to the path) of the point.
y Real The new y coordinate (relative to the path) of the point.
speed Real The new speed factor of the point.

Les retours :

N/A

Exemple :

for (var i = 0; i < path_get_number(path0); ++i;)
{
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
}

Le code ci-dessus boucle sur tous les points du chemin indexé comme "path0" et repositionne tous les points à une position aléatoire dans une zone de 128x128 pixels.
