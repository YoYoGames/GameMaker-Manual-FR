path_set_closed

Cette fonction peut être utilisée pour marquer un chemin donné comme étant ouvert (faux) ou fermé (vrai). Un chemin fermé a son point de départ connecté à son point d'arrivée, formant une boucle, et un chemin ouvert a un départ et une arrivée définitifs, non connectés. Si ce n'est pas ce dont vous avez besoin, vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe :

path_set_closed(index, closed);

Arguments Type Description
index Path Asset The index of the path to change.
closed Booléen Whether the path is closed (true) or not (false).

Les retours :

N/A

Exemple :

path_set_closed(pth_Patrol, true);

Ceci définira le chemin donné comme étant un chemin fermé.
