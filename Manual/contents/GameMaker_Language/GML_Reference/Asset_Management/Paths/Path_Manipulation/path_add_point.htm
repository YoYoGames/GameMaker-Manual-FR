chemin_add_point

Cette fonction permet d'ajouter un point à la trajectoire spécifiée et de définir son facteur de vitesse. Ce point est ajouté à la fin du chemin, quelle que soit la position du point, et le facteur est égal au pourcentage de path_speed que l'instance suivante va réellement lorsqu'elle atteint ce point du chemin. Si vous souhaitez placer un point du chemin à une autre position qui n'est pas la fin, vous devez utiliser la fonction path_insert_point().

Si elle est utilisée sur un chemin à partir du navigateur d'actifs, notez que la fonction modifiera l'actif réel, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe :

path_add_point(index, x, y, speed);

Arguments Type Description
index Path Asset The index of the path to add the point to.
x Real The x coordinate of the new point.
y Real The y coordinate of the new point.
speed Real The speed factor of the point (default value is 100).

Les retours :

N/A

Exemple :

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

Le code ci-dessus va créer une trajectoire circulaire autour de la position x/y de l'instance qui exécute le code.
