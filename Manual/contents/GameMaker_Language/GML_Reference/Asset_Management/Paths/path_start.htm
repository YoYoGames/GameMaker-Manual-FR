Début du chemin

Un chemin est créé à partir d'une série de points de définition qui sont reliés entre eux et ensuite utilisés pour planifier les mouvements d'une instance. Ils peuvent être créés avec du code, ou dans l'éditeur de chemin et ils sont assignés à une instance à utiliser dans le jeu. Vous utiliserez alors cette fonction pour indiquer à votre instance quel chemin suivre, à quelle vitesse suivre le chemin (mesurée en pixels par pas), comment se comporter lorsqu'elle atteint la fin du chemin, et si elle doit suivre la position absolue ou relative du chemin. Cette dernière partie signifie qu'elle commence et suit le chemin exactement comme vous l'avez conçu et placé dans l'éditeur de chemin (absolu), ou qu'elle commence et suit le chemin à partir de la position à laquelle l'instance a été créée (relatif).

Comme vous pouvez le voir dans l'image ci-dessus, les deux instances suivent le même chemin commencé avec path_start(), mais l'une des instances suit le chemin absolu tandis que l'autre suit le chemin relatif. Il convient de noter que lorsque la valeur relative est définie sur false (absolu), l'instance "saute" au début du chemin, quel que soit l'endroit de la pièce où elle est placée.

Les constantes utilisées pour définir le comportement de l'instance lorsqu'elle atteint la fin du chemin sont données sur cette page.

Syntaxe :

path_start(path, speed, endaction, absolute);

Arguments Type Description
path Path Asset The path index to start.
speed Real The speed of which to follow the path in pixels per step, negative meaning going backwards.
endaction Path End Action Constant What to do when the end of the path is reached (see the constants listed above).
absolute Booléen Whether the calling instance should follow the absolute path as it is in the editor (true) or a relative path to its current position (false).

Les retours :

N/A

Exemple :

path_start(path, 4, path_action_reverse, 0);

L'instance appelante suivra le chemin indexé dans la variable "path" à une vitesse de 4 pixels par pas. Elle suivra le chemin par rapport à sa propre position, et lorsqu'elle atteindra la fin, elle inversera sa direction et retournera sur ses pas.
