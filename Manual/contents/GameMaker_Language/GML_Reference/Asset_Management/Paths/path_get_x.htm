path_get_x

Avec cette fonction, vous pouvez obtenir la coordonnée x d'une position sur un chemin donné. 0 est le début du chemin, 1 est la fin du chemin, et tout ce qui se trouve entre les deux équivaut à cette distance dans le chemin. Il n'est pas nécessaire que ce soit un point de définition de la trajectoire, cela peut être n'importe où sur celle-ci.

Syntaxe :

path_get_x(ind, pos);

Arguments Type Description
index Path Asset The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end).

Les retours :

Real

Exemple :

x = path_get_x(pth_WallWalk, 0.5);

Ceci définira le x de l'instance appelante à la coordonnée x du point situé exactement à mi-chemin du chemin donné.
