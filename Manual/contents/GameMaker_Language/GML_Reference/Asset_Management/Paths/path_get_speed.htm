chemin_get_speed

Cette fonction renvoie le facteur de vitesse de toute position donnée sur un chemin. La position doit être une valeur entre 0 et 1 (vous pouvez utiliser path_positionpar exemple, si vous avez besoin de la position actuelle de l'instance), 0 étant la position de départ et 1 la position d'arrivée. La valeur renvoyée sera le facteur par lequel le site path_speed est défini. Par exemple, sur un chemin ne comportant que deux points, 0 et 1, où le facteur de vitesse du point 0 est 100 et celui du point 1 est 0, la fonction path_get_speed(path, 0.5) renverra 50.

Syntaxe :

path_get_speed(ind, pos);

Arguments Type Description
index Path Asset The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end), can be decimal.

Les retours :

Real

Exemple :

speed = path_get_speed(pth_Patrol, 0.5) / 20;

Ceci fixera la vitesse de l'instance appelante au facteur de vitesse du point situé exactement à mi-chemin du chemin donné, divisé par 20. Ainsi, si le facteur de vitesse trouvé est le facteur par défaut 100, la vitesse de l'instance appelante sera fixée à 5.
