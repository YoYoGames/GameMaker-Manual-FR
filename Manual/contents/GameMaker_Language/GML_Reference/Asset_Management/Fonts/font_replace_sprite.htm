font_replace_sprite

Avec cette fonction, vous pouvez utiliser une "bande de sprites" pour remplacer une ressource de police sprite précédemment créée, où chaque sous-image serait un symbole ou une lettre individuelle. L'index est le numéro de carte UTF8 du premier caractère à utiliser et les plages à utiliser sont les mêmes que pour une police normale, en commençant par " !", mais n'oubliez pas que vous ne devez inclure que ce dont vous avez besoin et que vous devez utiliser la fonction ord() pour obtenir la valeur UTF8 correcte pour la lettre de départ.

Vous pouvez également définir l'espacement de la police pour qu'elle soit proportionnelle (true) ou non (false). Une police proportionnelle est espacée en fonction de la largeur réelle des lettres (la lettre "i" prend moins de place que la lettre "w", par exemple), tandis qu'une police non proportionnelle sera espacée en fonction de la largeur de la sous-image, comme une police monospace. Enfin, vous pouvez définir l'espace à laisser entre chaque lettre lors de l'écriture. Il peut s'agir d'une valeur entière, 0 correspondant à l'absence d'espace (les lettres se "toucheront" si elles sont proportionnelles).

Si la fonction échoue, elle échouera en silence sans remplacer la police sprite donnée.

Il convient également de noter qu'il ne peut y avoir d'espace dans la carte de caractères, de sorte que si vous utilisez, par exemple, '.' et '0' mais pas '/', vous devez soit inclure '/' de toute façon, soit laisser une sous-image vide entre les deux caractères utilisés. Syntaxe :

font_replace_sprite(ind, spr, first, prop, sep);

Arguments Type Description
ind Font Asset The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr Sprite Asset The sprite to add a font based on.
first Real The index of the first available character.
prop Booléen Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep Real The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

Les retours :

N/A

Exemple :

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Le code ci-dessus remplacera la police indexée dans la variable globale "Font" par une nouvelle police provenant d'un fichier image externe.
