font_get_fontname

Avec cette fonction, vous pouvez obtenir le nom système réel de la ressource de police donnée. Cette fonction renvoie une chaîne et non un index, et le nom renvoyé dépendra de la police utilisée. Par exemple, vous pouvez avoir un actif de police appelé "fnt_Main" dans le navigateur d'actifs, et la police elle-même peut être la police système Arial de Windows. Dans ce cas, la fonction renvoie "Arial" car c'est le nom système de la police. Si vous avez besoin du nom qui apparaît dans l'Asset Browser, utilisez la fonction font_get_name().

Le comportement décrit ci-dessus ne s'applique qu'aux polices ajoutées par l'intermédiaire de IDE. Lorsqu'elle est utilisée sur une police chargée sur runtime (à l'aide de font_add()), la fonction renvoie le chemin d'accès complet à ce fichier de police au lieu du nom de la police.

Syntaxe :

font_get_fontname(ind);

ArgumentsType Description
ind Index of the font to check.

Les retours :

String

Exemple :

fnt_Name = font_get_fontname(font0);

Le code ci-dessus récupère le nom système d'une ressource de police et le stocke sous forme de chaîne dans la variable "fnt_Name".
