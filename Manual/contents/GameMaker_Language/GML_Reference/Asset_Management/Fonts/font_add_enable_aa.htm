font_add_enable_aa

Cette fonction permet d'activer ou de désactiver l'anticrénelage (AA) pour les polices ajoutées à l'aide de la fonction font_add(). Cette fonction doit être appelée avant d' ajouter des polices. En lui attribuant la valeur true, vous activez l'anticrénelage et en lui attribuant la valeur false, vous le désactivez. Par défaut, l'AA est activé. Notez que cela n'aura aucun effet sur les polices qui ont été ajoutées avant l'appel de la fonction, et que la fonction ne doit être appelée qu'une seule fois lorsque la police est ajoutée, et non pas à chaque dessin/pas où la police est utilisée.

Syntaxe :

font_add_enable_aa(enable);

ArgumentsType Description
enable Whether to enable (true) ou de désactiver (false) l'AA pour les polices ajoutées.

Les retours :

Exemple :

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

Le code ci-dessus vérifie l'état de l'anticrénelage pour les polices ajoutées et s'il n'est pas activé, alors nous l'activons.
