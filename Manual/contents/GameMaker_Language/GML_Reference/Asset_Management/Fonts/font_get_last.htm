font_get_last

Lorsque vous définissez une police dans GameMaker, vous pouvez définir une plage de caractères à inclure. En effet, la police elle-même n'est pas incluse dans votre jeu (pour des raisons juridiques), mais une image de la police est incluse dans une page de texture. la police est incluse dans une page de texture et c'est ce que votre jeu utilisera (comme tout autre élément graphique). Cela signifie que vous devrez limiter au maximum le nombre de caractères utilisés et ne spécifier que la gamme de caractères dont votre jeu aura besoin, afin de ne pas nuire à la qualité du jeu. dont votre jeu aura besoin, afin d'optimiser au maximum la mémoire des textures. Cette fonction peut être utilisée pour trouver le dernier caractère (sous forme de valeur UTF8) qui a été utilisé lorsque votre police a été ajoutée à votre jeu.

Syntaxe :

font_get_last(ind);

ArgumentsType Description
ind Index of the font to check.

Les retours :

Exemple :

num = font_get_last(fnt_Main);

Le code ci-dessus stockera la valeur ASCII de la dernière lettre de la gamme de polices pour la police indexée dans "fnt_Main".
