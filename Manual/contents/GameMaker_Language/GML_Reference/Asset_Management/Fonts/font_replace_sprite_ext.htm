font_replace_sprite_ext

Cette fonction remplacera une police sprite que vous avez créée précédemment à l'aide de la fonction font_add_sprite_ext(). Elle utilise une "bande de sprites", où chaque sous-image est un symbole ou une lettre individuelle, et les images sont mises en correspondance avec les lettres en fonction de l'argument "string_map" de la fonction. Cet argument est une chaîne de caractères que vous pouvez utiliser pour indiquer à GameMaker l'ordre dans lequel les sous-images de la police sprite sont placées, ce qui permet de les mapper automatiquement lors de l'écriture du texte. Ainsi, par exemple, si vous avez une chaîne de caractères de "AaBbCcDdEeFfGgHh", votre police sprite doit avoir les sous-images dans cet ordre.

Vous pouvez également définir l'espacement de la police pour qu'elle soit proportionnelle (true) ou non (false). Une police proportionnelle est espacée en fonction de la largeur réelle des lettres (la lettre "i" prend moins de place que la lettre "w", par exemple), tandis qu'une police non proportionnelle sera espacée en fonction de la largeur de la sous-image, comme une police monospace. Enfin, vous pouvez définir l'espace à laisser entre chaque lettre lors de l'écriture, et il peut s'agir de n'importe quelle valeur entière, 0 correspondant à un espace nul (les lettres se "toucheront" si elles sont proportionnelles). La fonction renvoie une valeur d'index qui doit être stockée dans une variable car elle sera nécessaire dans tous les codes ultérieurs faisant référence à cette police.

Si la fonction échoue, elle échouera en silence sans remplacer la police sprite donnée.

NOTE Cette fonction nécessite que vous ayez préalablement ajouté une police sprite à l'aide de la commande font_add_sprite()car elle la remplacera et ne peut pas être utilisée sur les polices qui font partie des ressources du jeu.

Syntaxe :

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Arguments Type Description
font Font Asset The font to replace.
spr Sprite Asset The sprite to add a font based on.
string_map Chaîne de caractères String from which sprite sub-image order is taken.
prop Booléen Set as proportional font or not.
sep Real The space to leave between each letter.

Les retours :

N/A

Exemple :

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

Le code ci-dessus remplacera un actif de police sprite qui est indexé dans la variable "global.Font" par des images sprite indexées dans la variable "spr_CalcFont".
