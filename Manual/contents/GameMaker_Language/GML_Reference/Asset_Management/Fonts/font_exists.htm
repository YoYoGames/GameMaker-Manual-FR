font_exists

Cette fonction indique si une police avec l'index spécifié existe ou non. Vous pouvez vérifier les indices de police tels qu'ils sont définis dans le navigateur d'actifs, ou les polices qui ont été ajoutées à l'aide de fonctions telles que font_add().

Syntaxe :

font_exists(ind);

ArgumentsType Description
ind Index of the font to check.

Les retours :

Exemple :

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

La police de dessin active est alors définie sur fnt_Main si elle existe.
