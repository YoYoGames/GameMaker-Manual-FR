font_get_uvs

Cette fonction renvoie un tableau contenant les coordonnées UV de la texture de la police sur la page de texture, en remplissant le tableau avec les valeurs suivantes :

  • [0] = gauche
  • [1] = haut
  • [2] = droite
  • [3] = fond

Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, notamment dans le dessin général lors de l'utilisation de primitives ainsi que dans les fonctions Shader.

Syntaxe :

font_get_uvs(font);

ArgumentsType Description
font The index of the font to use.

Les retours :

Exemple :

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Le code ci-dessus stocke les coordonnées UV pour l'arrière-plan donné dans un tableau local, puis affecte les valeurs aux variables d'instance.
