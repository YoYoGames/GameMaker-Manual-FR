font_delete

Cette fonction vous permet de supprimer une police du jeu. Cette suppression est permanente et le fait de changer de pièce ou de redémarrer le jeu ne fera pas revenir la police supprimée. Pour cela, le joueur doit quitter le jeu et le redémarrer de cette façon. et redémarrer le jeu de cette façon, donc faites attention lorsque vous utilisez cette fonction. En général, cette fonction n'est nécessaire que pour libérer de la mémoire qui a été utilisée par une police ajoutée au jeu par les fonctions font_add() ou font_add_sprite().

Syntaxe :

font_delete(ind);

ArgumentsType Description
ind The index of the font to delete.

Les retours :

Exemple :

font_delete(global.Font);

Le code ci-dessus va supprimer la police indexée dans la variable globale "Font".
