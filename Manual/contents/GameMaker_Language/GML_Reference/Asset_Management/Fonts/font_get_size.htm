police_get_size

Avec cette fonction, vous pouvez obtenir la taille de n'importe quelle ressource de police, qui est la valeur en points affichée par le dialogue de la ressource de police.

Syntaxe :

font_get_size(ind);

ArgumentsType Description
ind Index of the font to get the size of.

Les retours :

Exemple :

sz = font_get_size(font0);

Ceci va récupérer la taille de la police indexée par la variable "font0" et la stocker dans la variable "sz".

Retour : Polices de caractères
