Polices de caractères

GameMaker possède un certain nombre de fonctions qui peuvent être utilisées pour obtenir des informations sur les polices incluses, ainsi qu'un autre ensemble de fonctions qui peuvent être utilisées pour modifier, ajouter ou supprimer des polices de votre jeu au moment de l'exécution. En particulier, la fonction font_add() peut être utilisée pour ajouter des polices non standard, comme les polices à glyphes asiatiques, à partir d'un fichier *.ttf ou *.otf.

Il convient de noter que si vous ajoutez une police à votre jeu qui ne possède pas les glyphes requis par le texte que vous souhaitez écrire, le caractère Unicode 9647 (▯) est utilisé pour remplacer ces glyphes manquants lors du rendu dans l'événement de dessin. Ainsi, si votre police ne possède pas, par exemple, le symbole º, l'écriture de 90º produira en fait 90▯.

Les fonctions suivantes peuvent être utilisées pour obtenir des informations sur les ressources de polices :

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • font_get_texture
  • font_get_uvs
  • font_get_info

Les fonctions suivantes peuvent être utilisées pour changer et manipuler les polices pendant un jeu :

  • font_exists
  • font_set_cache_size
  • font_add_enable_aa
  • font_add_get_enable_aa
  • font_add
  • font_add_sprite
  • font_add_sprite_ext
  • font_replace
  • font_replace_sprite
  • font_replace_sprite_ext
  • font_delete
  • font_texture_page_size
