séquence_créer

Cette fonction vous permet de créer un nouvel objet séquence auquel vous pouvez ensuite ajouter des pistes. La fonction renvoie une structure que vous pouvez ensuite utiliser à l'adresse access pour configurer la nouvelle séquence que vous avez créée. Le contenu de cette structure struct est détaillé sur cette page. L'objet séquence struct peut ensuite être utilisé pour créer des instances de la séquence sur une couche de salle à l'aide de la fonction layer_sequence_create(). Notez que lorsque vous créez des objets de séquence de cette manière, vous devez les supprimer en appelant la fonction sequence_destroy() lorsqu'ils ne sont plus nécessaires, sinon vous aurez une fuite de mémoire qui peut ralentir et éventuellement faire planter votre jeu.

Syntaxe :

sequence_create();

Les retours :

Sequence Object Struct

Exemple :

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

Le code ci-dessus crée un nouvel objet séquence struct et définit sa longueur de lecture et son mode de boucle.
