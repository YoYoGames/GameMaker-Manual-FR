objet de recouvrement de l'instance de la séquence

Cette fonction vous permet de remplacer toutes les instances d'un objet utilisé dans une séquence par un autre. Vous fournissez l'ID de l'instance de la séquence struct (tel que renvoyé lorsque l'instance de la séquence a été créée dans la pièce ou en utilisant l'une des fonctions de calque de pièce - voir ici), ainsi que l'index de l'objet (tel que défini dans le navigateur d'actifs) pour l'objet que vous souhaitez remplacer. Enfin, vous donnez un index d'objet ou un ID d'instance à utiliser comme objet qui va remplacer la séquence (fournir un ID d'instance utilisera simplement l'objet à partir duquel l'instance a été créée comme remplacement). Notez que cela ne peut être fait que sur les instances de séquence (pas sur les objets de séquence) et doit être fait avant que la séquence ne commence à être jouée, sinon cela ne fonctionnera pas.

Syntaxe :

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Arguments Type Description
sequence_instance_struct Structure d'instance de séquence The sequence instance struct to modify.
object_id Objet Asset The object index of the object within the sequence to override.
instance_or_object_id ID de l'actif ou de l'instance del'objet The object index or instance ID to use to override the sequence objects.

Les retours :

N/A

Exemple :

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

Le code ci-dessus crée une nouvelle instance de séquence sur la couche donnée, puis la modifie de sorte que toutes les instances de l'objet "obj_Trees_Winter" soient remplacées par des instances de l'objet "obj_Trees_Summer".
