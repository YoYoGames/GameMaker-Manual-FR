séquence_get

Cette fonction permet de récupérer la structure d'objet de séquence à partir d'un index d'actif de séquence. Vous fournissez l'index d'actif de la séquence tel que défini dans le navigateur d'actifs, et la fonction renvoie l'objet de séquence struct auquel il est possible d'accéder, ou -1 si la séquence n'existe pas ou si l'index d'actif donné n'est pas une séquence.

Syntaxe :

sequence_get(sequence_index);

Arguments Type Description
sequence_index Séquence Asset The sequence asset as defined in the Asset Browser.

Les retours :

Sequence Object Struct or -1

Exemple :

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

Le code ci-dessus récupère l'objet séquence struct à partir de l'objet séquence "seq_Loop", puis définit le mode de boucle pour la séquence (toutes les instances de séquence créées ultérieurement utiliseront désormais ce mode de boucle).
