séquence_get_objets

Avec cette fonction, vous pouvez récupérer un tableau de tous les indices d'objets qui ont des instances créées dans la séquence donnée. Vous fournissez soit l'objet de la séquence struct (tel que retourné par la fonction sequence_create() ou sequence_get()) ou l'ID de la séquence (tel que renvoyé par la fonction layer_sequence_get_sequence() ou à partir de la propriété de l'instance de la séquence struct sequence ) et la fonction renverra un tableau, dans lequel chaque élément du tableau est un identifiant object_index pour les différents objets utilisés par la séquence pour créer des instances.

Syntaxe :

sequence_get_objects(sequence_struct_or_id);

Arguments Type Description
sequence_struct_or_id Séquence Asset ou Séquence Object Struct The sequence object struct or ID to get the objects from.

Les retours :

Array

Exemple :

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

Le code ci-dessus obtient le site struct pour un objet séquence, puis récupère les objets qu'il utilise et stocke le tableau dans une variable.
