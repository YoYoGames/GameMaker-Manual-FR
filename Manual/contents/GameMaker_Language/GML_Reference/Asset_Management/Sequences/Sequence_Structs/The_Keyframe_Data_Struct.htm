La structure de données des images clés

Lorsque vous définissez une structure d'image clé pour une piste, vous devez également définir les données qui lui seront associées. Ces données sont constituées de différents canaux, chaque canal étant simplement une donnée d'image clé Struct. Un canal peut recevoir un type de données spécifique en fonction du type de piste pour lequel vous définissez les données d'image clé.

Une donnée d'image clé Struct contient les variables suivantes :

Structure des données d'images-clés de la séquence
Variable Type Description
channel Real Il s'agit du canal auquel les données de l'image clé doivent être appliquées. Il s'agit d'une valeur entière positive commençant à 0. Il convient de noter que lorsque vous créez des pistes de paramètres pour des images clés de "position" ou d'"échelle", vous devez utiliser des valeurs de canal très spécifiques. Ces valeurs sont les suivantes : le canal 0 correspond à la position X ou à l'échelle X, le canal 1 correspond à la position Y ou à l'échelle Y.
spriteIndex Sprite Asset L'index de l'actif sprite à utiliser pour la piste. Cette propriété est uniquement disponible pour les pistes de type seqtracktype_graphic et vous pouvez l'obtenir ou la définir.
soundIndex Actif sonore L'index de l'actif audio à utiliser pour la piste. Cette propriété est uniquement disponible pour les pistes de type seqtracktype_audio et vous pouvez l'obtenir ou la définir.
particleSystemIndex Particle System Asset The particle system asset used for the track. This property is only available for tracks of the type seqtracktype_particlesystem and you can get or set it.
playbackMode Séquence Audio Key Constant Le mode de lecture du son. Cette propriété est uniquement disponible pour les pistes de type seqtracktype_audio et vous pouvez l'obtenir ou la définir. La propriété doit être une des constantes données dans le tableau ci-dessous.
curve Structure de la courbe d'animation Cette propriété nécessite une courbe d'animation struct (voir ici pour plus d'informations) et n'est disponible que pour les pistes du type seqtracktype_real. Si aucune courbe struct n'est utilisée, la valeur de cette propriété sera -1.
value Real Cette propriété est simplement une valeur associée au canal de données d'image clé, et n'est disponible que pour les pistes de type seqtracktype_real lorsqu'aucune curve struct n'est fournie. Il peut s'agir, par exemple, de la position X ou Y de la piste si elle est placée à l'intérieur d'une piste à paramètre "position".
colour Array de Véritables Cette propriété renvoie (ou exige, si elle est définie) un tableau pour la valeur de couleur de l'image clé au format [A, R, G, B]. Cette propriété n'est disponible que pour les pistes de type seqtracktype_colour. Notez que les valeurs de chaque composant doivent être exprimées entre 0 et 1, où 0 correspond à la valeur HEX #00 et 1 correspond à la valeur HEX #FF (0 - 255 comme indiqué dans le sélecteur de couleur pour les pistes en couleur dans l'éditeur de séquences).
sequence Structure de l'objet de la séquence Cette propriété renvoie (ou requiert, si elle est définie) une structure d'objet de séquence et n'est disponible que pour les pistes de type seqtracktype_sequence.
objectIndex Objet Asset Cette propriété renvoie (ou requiert, si elle est définie) un indice d'objet et n'est disponible que pour les pistes de type seqtracktype_instance.
events Tableau de Chaîne de caractèress Cette propriété permet d'accéder à access aux événements et aux messages de diffusion associés aux données d'image clé struct. Vous pouvez obtenir ou définir cette propriété. Lorsque vous l'obtenez, un tableau de chaînes est renvoyé, et lorsque vous le définissez, un tableau de chaînes doit être spécifié. Pour plus d'informations sur les événements, veuillez consulter la section Événements et moments de la séquence. Cette propriété n'est disponible que pour les pistes de type seqtracktype_message.
event Méthode Cette propriété renverra (ou peut être réglée sur) les method associée aux données d'image clé struct. Si aucune méthode n'a été spécifiée ou si vous souhaitez supprimer la méthode, la propriété doit être -1. Cette propriété n'est disponible que pour les pistes de type seqtracktype_moment.
Toute variable supplémentaire figurant dans le(s) tableau(x) ci-dessous

La variable playbackMode peut être l'une des constantes suivantes :

Séquence Audio Key Constant
Constant Description Valeur
seqaudiokey_loop Le son est joué en boucle. 0
seqaudiokey_oneshot Le son ne sera joué qu'une fois puis s'arrêtera. 1

Données de la piste de texte

Le site struct contiendra les variables supplémentaires suivantes s'il est affecté à une piste de texte (seqtracktype_text) :

Structure des données d'images-clés de la séquence
Variable Type Description
text Chaîne de caractères Il s'agit de la chaîne de texte qui est dessinée sur la piste.
wrap Booléen Il s'agit d'un booléen qui indique si le texte doit être enveloppé (true) ou non (false).
alignmentH Constante d'alignement du texte de la séquence Il s'agit de l'alignement horizontal du texte. Constante d'alignement du texte de la séquencedonnées ci-dessous.
alignmentV Constante d'alignement du texte de la séquence Il s'agit de l'alignement vertical du texte. Constante d'alignement du texte de la séquencedonnées ci-dessous.
fontIndex Font Asset Il s'agit de la Font Asset utilisée par la piste de texte.

alignmentH peut être l'une des constantes suivantes :

Constante d'alignement des séquences et des textes (alignmentH)
Constant Description
seqtextkey_left Le texte sera aligné à gauche du cadre.
seqtextkey_right Le texte sera aligné à droite du cadre.
seqtextkey_center Le texte sera centré horizontalement dans le cadre.
seqtextkey_justify Le texte sera justifié dans le cadre

alignmentV peut être l'une des constantes suivantes :

Constante d'alignement des séquences et des textes (alignmentV)
Constant Description
seqtextkey_top Le texte sera aligné sur le haut du cadre.
seqtextkey_bottom Le texte sera aligné sur le bas du cadre.
seqtextkey_middle Le texte sera centré verticalement dans le cadre.
