séquence_destruction

Cette fonction permet de détruire un objet séquence qui a été créé dynamiquement. Vous fournissez soit l'objet séquence struct (tel que renvoyé par la fonction sequence_create()) ou l'ID de la séquence (tel que renvoyé par la fonction layer_sequence_get_sequence() ou à partir de l'instance de séquence struct propriété sequence). Cette fonction doit être utilisée chaque fois qu'une séquence créée dynamiquement n'est plus nécessaire afin de libérer la mémoire qui lui est associée.

Syntaxe :

sequence_destroy(sequence_struct_or_id);

Arguments Type Description
sequence_struct_or_id Séquence Asset ou Séquence Object Struct The sequence object struct or ID to destroy

Les retours :

N/A

Exemple :

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Le code ci-dessus vérifie si l'objet séquence donné existe et si c'est le cas, il est détruit.
