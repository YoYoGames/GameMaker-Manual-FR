séquence_existe

Avec cette fonction, vous pouvez vérifier si un objet séquence existe ou non. Vous fournissez soit l'objet de séquence struct (tel que retourné par la fonction sequence_create() ou sequence_get()) ou l'ID de la séquence (tel que renvoyé par la fonction layer_sequence_get_sequence() ou de la propriété sequence de l'instance de séquence struct, ou de l'index du navigateur d'actifs) et la fonction renverra true si l'objet de séquence existe ou false s'il n'existe pas.

Syntaxe :

sequence_exists(sequence_struct_or_id);

Arguments Type Description
sequence_struct_or_id Séquence Asset ou Séquence Object Struct The sequence to check for, can be the asset reference itself or its Sequence object struct.

Les retours :

Boolean

Exemple :

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Le code ci-dessus vérifie si l'objet séquence donné existe et si c'est le cas, il est détruit.
