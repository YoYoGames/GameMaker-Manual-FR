tileset_get_texture

Cette fonction renvoie un pointeur spécial pour la page de texture du jeu de tuiles. Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, notamment dans les fonctions de primitives 2D, ainsi que dans les fonctions de shaders. Vous pouvez obtenir plus d'informations sur la page de texture renvoyée en utilisant les différentes fonctions de texture_ que vous trouverez ici.

Syntaxe :

tileset_get_texture(tileset);

Arguments Type Description
tileset Ensemble de tuiles Asset The index of the tile set to use.

Renvoie à

Texture

Exemple :

var tex;
tex = tileset_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

Le code ci-dessus dessine une bande triangulaire à 4 sommets texturée avec la texture contenue dans la variable "tex".
