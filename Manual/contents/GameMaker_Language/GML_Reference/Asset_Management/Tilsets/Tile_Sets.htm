Sets de carreaux

Les fonctions suivantes sont disponibles lorsque vous travaillez avec des ressources de jeu de tuiles, telles que créées dans le navigateur de ressources. Ces fonctions ne servent qu' à traiter l'actif lui-même, et non à travailler avec les tuiles ou les cartes de tuiles lorsqu'elles sont présentes dans votre jeu (pour cela, consultez les fonctions de calque appropriées dans la section sur les pièces).

Les fonctions disponibles sont les suivantes :

  • tileset_get_name
  • tileset_get_texture
  • tileset_get_uvs
