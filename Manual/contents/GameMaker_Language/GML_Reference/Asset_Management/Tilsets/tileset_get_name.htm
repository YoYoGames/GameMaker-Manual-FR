tuiles_get_name

Cette fonction renvoie le nom sous forme de chaîne de caractères de l'actif du jeu de tuiles spécifié. Ce nom est celui qui a été spécifié pour le jeu de tuiles dans le navigateur d'actifs de la fenêtre principale de GameMaker. Veuillez noter qu'il s'agit uniquement d'une chaîne de caractères et qu'elle ne peut pas être utilisée pour référencer directement le jeu de tuiles - pour cela, vous aurez besoin de l'index du jeu de tuiles. Vous pouvez cependant utiliser cette chaîne pour obtenir l'index du jeu de tuiles en utilisant la chaîne retournée avec la fonction asset_get_index().

Syntaxe :

tileset_get_name(index);

Arguments Type Description
index Ensemble de tuiles Asset The index of the tile set to get the name of.

Renvoie à

String

Exemple :

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

Le code ci-dessus obtiendra le nom de l'indice de l'ensemble de tuiles pour la couche donnée, en stockant la chaîne de retour dans la variable tileset_name.
