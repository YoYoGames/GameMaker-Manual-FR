script_existant

Cette fonction renverra true ou false selon que le script ou la fonction de script avec l'index donné existe ou non. Notez qu'il ne s'agit pas d'une chaîne de caractères, mais plutôt du nom du poste qui contient l'index unique de chaque script (tel qu'il apparaîtrait dans IDE) ou de la fonction de script nommée, telle que définie dans le poste de script (notez que cela ne fonctionne pas pour les variables de méthode). Pour plus d'informations sur les scripts, voir L'éditeur de script.

Syntaxe :

script_exists(scr);

Arguments Type Description
scr Script Asset The script index that you want to check.

Les retours :

Boolean

Exemple :

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

L'exemple ci-dessus ajoute deux fonctions de script et une valeur dans un tableau, puis procède à l'obtention d'un nombre aléatoire et l'utilise pour choisir une fonction de script à exécuter, sauf si le -1 est choisi, auquel cas rien ne se passera.
