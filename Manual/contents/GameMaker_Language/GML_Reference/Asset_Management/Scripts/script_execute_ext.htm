script_execute_ext

Cette fonction fonctionne de manière similaire à la fonction script_execute(), sauf que vous pouvez fournir un tableau contenant les arguments requis pour la fonction/script appelée. Vous pouvez également fournir deux arguments facultatifs à la fonction pour spécifier un décalage dans le tableau pour obtenir les arguments, ainsi que le nombre d'arguments à utiliser à partir du tableau (cela doit être un maximum de array_length - offset).

Syntaxe :

script_execute_ext(scr, array_args, [offset], [num_args]);

Arguments Type Description
scr Script Asset ou Script Function The function/script that you want to call
array_args Array The array containing the arguments for the function/script
[offset] Real OPTIONAL Le décalage dans le tableau des arguments
[num_args] Real OPTIONAL Le nombre d'arguments à utiliser (à partir de l'offset)

Les retours :

Variable (Will depend on the return value from the function/script being called)

Exemple :

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

L'exemple de code ci-dessus utilisera script_execute_ext pour appeler la fonction donnée, en fournissant un tableau de quatre arguments, mais en n'utilisant qu'un seul d'entre eux au hasard.

REMARQUE Cette fonction ne peut pas être utilisée avec les fonctions intégrées.
