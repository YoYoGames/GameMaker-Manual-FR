script_get_name

Cette fonction renvoie le nom sous forme de chaîne du script spécifié. Ce nom est celui qui a été spécifié pour le script dans le navigateur d'actifs de la fenêtre principale de GameMaker. Pour plus d'informations sur les scripts, consultez la rubrique L'éditeur de script.

Syntaxe :

script_get_name(scr);

Arguments Type Description
scr Script Asset The index of the script that you want to get the name of.

Les retours :

String

Exemple :

scr_name = script_get_name(Help_File);

L'exemple de code ci-dessus stockera le nom de l'index de script indiqué dans la variable "scr_name".
