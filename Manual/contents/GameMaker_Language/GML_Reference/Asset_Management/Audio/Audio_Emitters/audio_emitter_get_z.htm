audio_emitter_get_z

Cette fonction renvoie la position z actuelle de l'émetteur audio donné.

Syntaxe :

audio_emitter_get_z(emitter);

Arguments Type Description
emitter ID de l'émetteur audio The index of the emitter to use.

Les retours :

Real

Exemple :

if audio_emitter_get_z(emitter_player) != 0
{
    var ex = audio_emitter_get_x(emitter_player);
    var ey = audio_emitter_get_y(emitter_player);
    audio_emitter_position(emitter_player, ex, ey, 0);
}

Le code ci-dessus vérifie la position z actuelle d'un émetteur donné et si elle n'est pas égale à 0, elle est mise à 0.
