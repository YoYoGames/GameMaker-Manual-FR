audio_emitter_create

Cette fonction crée un nouvel émetteur audio et renvoie son index. Cet indice doit être stocké dans une variable pour toutes les autres fonctions qui se rapportent à cet émetteur, et lorsqu'il n'est plus nécessaire, il doit être supprimé de la mémoire à l'aide de la fonction audio_emitter_free() afin d'éviter les fuites de mémoire qui peuvent éventuellement faire planter votre jeu.

L'émetteur audio est créé avec les paramètres par défaut suivants :

  • Un gain et un pas de 1
  • Un masque d'écouteur défini uniquement sur l'écouteur par défaut (1).
  • Une position de (0, 0, 0)
  • Une vitesse de 0 (vx = 0, vy = 0, vz = 0)

Syntaxe :

audio_emitter_create();

Les retours :

Audio Emitter ID

Exemple :

s_emit = audio_emitter_create();

Le code ci-dessus crée un nouvel émetteur audio et assigne son index à la variable "s_emit".
