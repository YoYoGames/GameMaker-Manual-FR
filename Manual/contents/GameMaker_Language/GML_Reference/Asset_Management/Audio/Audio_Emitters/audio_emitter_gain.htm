audio_emitter_gain

Cette fonction définit le gain (volume) maximal du son. Le volume perçu pour un son peut changer en fonction de la valeur de fall-off et de la position qu'il occupe par rapport à l'auditeur, mais en réglant le gain avec cette fonction, le volume total ne dépassera jamais la valeur de gain spécifiée. L'image ci-dessous illustre comment le gain affecte le volume de l'émetteur lorsque le fall-off est supérieur à 0 :

Cette fonction modifiera le volume du son pendant qu'il est joué ainsi que tous les sons suivants joués par l'émetteur donné. Notez que sur certaines plates-formes, vous pouvez avoir un gain supérieur à 1, bien qu'une valeur de 1 soit considérée comme "plein volume" et que toute valeur supérieure peut introduire un écrêtage ou une distorsion audio.

NOTE le volume final sera également influencé par le gain audio global qui a été défini par la fonction audio_master_gain().

Syntaxe :

audio_emitter_gain(emitter, gain);

Arguments Type Description
emitter ID de l'émetteur audio The index of the emitter to change.
gain Real The maximum gain (default 1).
time Real  The length of the change in gain in milliseconds. Defaults to 0 (instantaneous change) if not passed.

Les retours :

N/A

Exemple :

if up
{
    gain += 0.05;
    if (gain >= 1) up = false;
}
else
{
    gain -= 0.05;
    if (gain <= 0) up = true;
}

audio_emitter_gain(s_emit, gain);

Le code ci-dessus définit la variable "gain" à différentes valeurs et utilise ensuite cette même variable pour définir le gain de l'émetteur indexé dans la variable "s_emit".
