audio_emitter_set_listener_mask

Cette fonction peut être utilisée pour définir le masque binaire d'un émetteur afin que tous les sons joués par l'émetteur ne soient lus que par les auditeurs spécifiés. Vous pouvez créer un masque de bits en utilisant la fonction audio_get_listener_info() puis en utilisant le bitwise or ("|") pour créer un masque personnalisé avec les auditeurs à partir desquels vous souhaitez que le son soit joué, puis en appliquant ce masque personnalisé à l'émetteur. Ce masque remplacera le masque par défaut ou celui que vous avez pu définir à l'aide de la fonction audio_set_listener_mask().

Syntaxe :

audio_emitter_set_listener_mask(emitterID, mask);

Arguments Type Description
emitterID ID de l'émetteur audio The unique ID of the emitter to set the mask of
mask Real The bitmask data to set for the sound

Les retours :

N/A

Exemple :

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

Le code ci-dessus crée un émetteur audio, puis définit le ou les écouteurs à partir desquels il faut jouer en utilisant les données du masque stockées dans une variable globale.
