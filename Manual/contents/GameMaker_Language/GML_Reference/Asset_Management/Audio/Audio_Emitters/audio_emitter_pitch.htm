audio_emitter_pitch

Cette fonction peut être utilisée pour modifier la hauteur de tous les sons émis par l'émetteur donné. Il s'agit d'un multiplicateur de hauteur, en ce sens que la valeur d'entrée multiplie la hauteur actuelle par cette quantité, donc la valeur par défaut de 1 ne change pas la hauteur, tandis qu'une valeur inférieure à 1 diminuera la hauteur et supérieure à 1 l'augmentera. Il est préférable d'utiliser de petits incréments pour cette fonction car toute valeur inférieure à 0 ou supérieure à 5 peut ne pas être audible de toute façon.

Les sons déjà en cours de lecture sur l'émetteur audio ne doivent pas être redémarrés pour que le changement de hauteur soit audible. Le changement de hauteur est appliqué immédiatement.

Syntaxe :

audio_emitter_pitch(emitter, pitch);

Arguments Type Description
emitter ID de l'émetteur audio The index of the emitter to change.
pitch Real The pitch multiplier (default 1).

Les retours :

N/A

Exemple :

switch (gear)
{
    case 1: audio_emitter_pitch(s_emit, 0.8); break;
    case 2: audio_emitter_pitch(s_emit, 0.9); break;
    case 3: audio_emitter_pitch(s_emit, 0.95); break;
    case 4: audio_emitter_pitch(s_emit, 1); break;
    case 5: audio_emitter_pitch(s_emit, 1.2); break;
}

Le code ci-dessus modifiera la hauteur du son joué par l'émetteur indexé dans la variable "s_emit" en fonction de la valeur de la variable "gear".
