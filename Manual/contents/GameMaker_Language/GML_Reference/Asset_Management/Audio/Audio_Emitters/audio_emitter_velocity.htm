vitesse de l'émetteur audio

Cette fonction peut être utilisée pour donner à un émetteur des effets Doppler et simuler un mouvement audio basé sur le vecteur qui est résolu à partir des positions relatives x, y et z données. Si l'émetteur lui-même ne bouge jamais, vous n'avez normalement pas besoin de définir ces valeurs, mais, par exemple, si vous créez un jeu de tir à défilement dans lequel les ennemis viennent de la droite et défilent vers la gauche, vous devez définir une vitesse de (hspeed, 0, 0) dans l'événement create des ennemis (et mettre à jour l'émetteur de chaque instance dans l'événement step en utilisant audio_emitter_position()) pour donner aux sons émis par les instances ennemies le Doppler correct lorsqu'elles passent devant l'instance du joueur (qui à son tour utilisera la fonction audio_listener_position() pour placer l'écouteur à la bonne position).

Syntaxe :

audio_emitter_velocity(emitter, vx, vy, vz);

Arguments Type Description
emitter ID de l'émetteur audio The index of the emitter to change.
vx Real The x vector value (default 0).
vy Real The y vector value (default 0).
vz Real The z vector value (default 0).

Les retours :

N/A

Exemple :

s_emit = audio_emitter_create();
audio_emitter_position(s_emit, room_width, 0, 0);
audio_emitter_velocity(s_emit, -5, 0, 0);

Le code ci-dessus crée un émetteur audio et assigne son index à la variable "s_emit". Cet émetteur est ensuite placé à un endroit de la pièce et on lui donne une vitesse de 5 pixels par pas le long de l'axe x (il fera un Doppler correct par rapport à l'auditeur comme s'il avait une vitesse horizontale de 5 pixels par pas).
