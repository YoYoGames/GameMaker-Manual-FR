audio_émetteur_libre

Avec cette fonction, vous pouvez supprimer l'emitter donné de la mémoire. Cette opération doit toujours être effectuée lorsque l'émetteur n'est plus utilisé dans la pièce ou dans le jeu, c'est-à-dire dans l'événement Destroy de l'instance ou dans l'événement Room End, sinon vous risquez de vous retrouver avec une fuite de mémoire qui ralentira et fera planter votre jeu.

Syntaxe :

audio_emitter_free(emitter);

Arguments Type Description
emitter ID de l'émetteur audio The index of the emitter to free.

Les retours :

N/A

Exemple :

if lives = 0
{
    audio_emitter_free(s_emit);
    room_goto(rm_Menu);
}

Le code ci-dessus vérifie la valeur de la variable globale "lives" et s'il retourne 0, il détruira l'emitter indexé dans la variable "s_emit" et ira ensuite dans la salle indexée dans la variable "rm_Menu".
