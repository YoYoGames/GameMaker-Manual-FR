audio_pause_sound

Cette fonction vous permet de mettre en pause tout son en cours de lecture. Le son peut être une instance unique d'un son (l'index des sons individuels en cours de lecture peut être stocké dans une variable lorsque vous utilisez les fonctions audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore, auquel cas toutes les instances du son en question seront mises en pause.

Syntaxe :

audio_pause_sound(index);

Arguments Type Description
index ID de l'actif sonore ou de l'instance sonore The index of the sound to pause.

Les retours :

N/A

Exemple :

if keyboard_check_pressed(ord("P"))
{
    global.pause = !global.pause;
    if (global.pause)
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_waterfall);
    }
}

Le code ci-dessus vérifie s'il y a eu une pression sur la touche "P" du clavier et, s'il en détecte une, il attribue à la variable globale "Pause" la valeur true ou false, puis met en pause le son indexé dans la variable "snd_Waterfall" ou reprend le son à partir de son état de pause.
