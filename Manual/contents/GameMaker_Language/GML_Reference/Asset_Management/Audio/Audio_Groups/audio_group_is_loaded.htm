audio_group_is_loaded

Cette fonction vérifie un groupe audio spécifique pour voir s'il a été chargé en mémoire, prêt à être utilisé.

Syntaxe :

audio_group_is_loaded(groupID);

Arguments Type Description
groupID ID du groupe audio The index of the audio group to check (as defined in the Audio Groups Window)

Les retours :

Boolean

Exemple :

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

Le code ci-dessus vérifie si un groupe audio a été chargé et si oui, il le décharge.
