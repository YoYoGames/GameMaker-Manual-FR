audio_group_set_gain

Cette fonction permet d'appliquer un fondu à un groupe de sons sur une période donnée, ou de régler instantanément le gain du groupe.

Le temps est mesuré en millisecondes, et la fonction exige que vous entriez un niveau de gain final que le groupe doit avoir atteint à la fin de ce temps. Ce gain peut être compris entre 0 (silence) et 1 (plein volume) et l'échelle est linéaire, de sorte qu'une valeur de 0,5 correspond à la moitié du volume. Pour modifier instantanément le gain, il suffit de définir l'argument temps à 0. Notez que sur certaines plates-formes, vous pouvez avoir un gain supérieur à 1, bien qu'une valeur de 1 soit considérée comme "plein volume" et que toute valeur supérieure peut introduire un écrêtage audio.

Syntaxe :

audio_group_set_gain(groupID, volume, time);

Arguments Type Description
groupID ID du groupe audio The index of the audio group to stop (as defined in the Audio Groups Window)
volume Real The final value for the group volume.
time Real The length of the change in gain in milliseconds.

Les retours :

N/A

Exemple :

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

Le code ci-dessus vérifie la présence de la touche "espace" et réduit tout le contenu audio de "audiogroup1" à 0 pendant 5 secondes.
