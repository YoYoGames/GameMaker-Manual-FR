audio_group_unload

Cette fonction va décharger en mémoire tous les sons qui sont marqués comme appartenant au groupe audio donné. La fonction renvoie true si le déchargement est initié et false si l'ID du groupe n'est pas valide ou s'il a déjà été marqué pour le déchargement. Notez que tout son en cours de lecture lorsque cette fonction est appelée sera arrêté.

Syntaxe :

audio_group_unload(groupID);

Arguments Type Description
groupID ID du groupe audio The index of the audio group to unload (as defined in the Audio Groups Window)

Les retours :

Boolean

Exemple :

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

Le code ci-dessus vérifie si un groupe audio a été chargé et si oui, il le décharge.
