audio_group_load

Cette fonction chargera en mémoire tous les sons qui sont marqués comme appartenant au groupe audio donné. La fonction renvoie true si le chargement est lancé et false si l'ID du groupe n'est pas valide ou s'il a déjà été marqué pour le chargement. La fonction est asynchrone, de sorte que votre jeu continuera à fonctionner pendant que l'audio est chargé en arrière-plan. Cela signifie qu'elle déclenchera également un événement Load/Save asynchrone pour vous informer que le groupe entier a été chargé en mémoire et que les sons peuvent maintenant être utilisés.

Syntaxe :

audio_group_load(groupID);

Arguments Type Description
groupID ID du groupe audio The index of the audio group to load (as defined in the Audio Groups Window)

Les retours :

Boolean

Exemple :

if !audio_group_is_loaded(audiogroup_level1)
{
    audio_group_load(audiogroup_level1);
}

Le code ci-dessus vérifie si un groupe audio a été chargé et si non, il le charge.
