progression du chargement du groupe audio

Cette fonction vérifie la progression du chargement d'un groupe audio et renvoie une valeur (approximative) comprise entre 0 et 100.

Syntaxe :

audio_group_load_progress(groupID);

Arguments Type Description
groupID ID du groupe audio The index of the audio group to check (as defined in the Audio Groups Window)

Les retours :

Real

Exemple :

if load
{
    var pc = audio_group_load_progress(audiogroup_level1);
    draw_text(32, 32, "Loading: " + string(pc));
}

Le code ci-dessus vérifie une variable et si elle renvoie true, un texte sera affiché pour montrer la progression du chargement de l'audio.
