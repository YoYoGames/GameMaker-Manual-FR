nom_du_groupe_audio

Cette fonction renvoie une chaîne contenant le nom du groupe audio donné pour affichage ou vérification. Lorsque vous définissez un groupe audio dans les options du jeu, vous lui donnez un "nom" unique qui est en fait une constante à utiliser comme valeurd'identification pour le groupe. Cette fonction ne fait que prendre l'ID et retourner une chaîne de caractères contenant le nom de l'ID que vous avez donné.

Syntaxe :

audio_group_name(groupID);

Arguments Type Description
groupID ID du groupe audio The index value constant of the audio group to check (as defined in the Audio Groups Window)

Les retours :

String

Exemple :

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

Le code ci-dessus récupère le nom de la constante de groupe audio donnée et l'affiche à l'écran.
