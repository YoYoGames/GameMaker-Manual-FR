audio_sound_is_playable

Cette fonction peut être utilisée pour vérifier si l'indice sonore donné peut être lu actuellement. Cette fonction est nécessaire en raison des différentes façons dont la lecture du son en flux continu ou non continu est gérée sur la plate-forme cible HTML5. Elle renverra true si le son peut être lu et false s'il ne peut pas l'être. Notez que sur toutes les autres plateformes autres que HTML5, la fonction renverra toujours true.

Syntaxe :

audio_sound_is_playable(index);

Arguments Type Description
index Actif sonore The index of the sound to check.

Les retours :

Boolean

Exemple :

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

Le code ci-dessus jouera le son "snd_Shot" s'il est jouable.
