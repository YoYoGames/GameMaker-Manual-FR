audio_stop_all

Cette fonction arrête tous les sons en cours de lecture.

Syntaxe :

audio_stop_all();

Les retours :

Exemple :

if !global.SFX
{
    audio_stop_all();
}

Le code ci-dessus vérifie la variable globale "SFX" et s'il retourne false, il arrêtera tous les sons en cours de lecture.
