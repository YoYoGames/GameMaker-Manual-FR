audio_get_recorder_count

Cette fonction renvoie le nombre de sources d'enregistrement audio (comme les microphones, etc...) actuellement disponibles dans votre jeu. Ainsi, si vous avez une valeur de retour de, par exemple, quatre, alors vous aurez une entrée audio sur les sources 0, 1, 2 et 3, l'une de ces valeurs étant celle que vous utilisez pour commencer l'enregistrement en utilisant la fonction audio_start_recording(). Cette valeur peut changer à tout moment lorsque des personnes branchent/débranchent des microphones ou d'autres sources d'entrée sur l'appareil. Notez que vous pouvez utiliser la fonction audio_get_recorder_info() pour obtenir des informations sur chaque appareil connecté.

IMPORTANT Cette fonction renvoie toujours 1 sur la cible Opera GX.

Syntaxe :

audio_get_recorder_count();

Les retours :

Real

Exemple :

if audio_get_recorder_count() > 0
{
    channel_index = audio_start_recording(0);
}

Le code ci-dessus vérifie les enregistreurs audio disponibles et s'il y en a 1 ou plus, il commence à enregistrer à partir du premier indexé (source 0).
