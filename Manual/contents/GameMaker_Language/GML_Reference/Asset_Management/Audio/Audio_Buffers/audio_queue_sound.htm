audio_queue_sound

Cette fonction ajoutera les données d'un buffer dans la file d'attente audio que vous avez précédemment créée à l'aide de la fonction audio_create_play_queue(). Vous spécifiez l'index de la file d'attente à ajouter, et l'ID de buffer pour obtenir les données, ainsi que la position (offset) dans buffer et le nombre d'octets à ajouter. Une fois que vous avez ajouté de l'audio d'un buffer à une file d'attente, le buffer ne peut pas être supprimé avant d'avoir libéré la file d'attente à l'aide de la fonction . audio_free_play_queue()Les propriétés de buffer doivent correspondre à celles de la file à laquelle vous ajoutez le son.

Syntaxe :

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

Arguments Type Description
queueIndex ID de la file d'attente audio The index of the queue to add to.
bufferId ID de la mémoire tampon The buffer to add to the queue.
bufferOffset Real The offset within the source buffer to start from.
bufferLength Real The length of the buffer (the number of the bytes in the buffer).

Les retours :

N/A

Exemple :

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

Le code ci-dessus sera appelé dans l'événement d'enregistrement audio asynchrone et attribuera certaines données enregistrées à une adresse buffer, qui sera ensuite ajoutée à une file d'attente audio. Celle-ci est ensuite lue.
