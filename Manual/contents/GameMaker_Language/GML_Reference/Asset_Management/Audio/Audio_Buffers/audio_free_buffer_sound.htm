audio_free_buffer_sound

Cette fonction permet de libérer la valeur d'index du pointeur associée à l'ID du son. Les sons libérés ne seront pas disponibles pour la lecture, et si plusieurs instances du son sont en cours de lecture, elles seront toutes arrêtées. Notez qu'avant de pouvoir supprimer le site buffer lui-même, vous devez d'abord libérer tous les ID sonores qui lui sont associés.

Syntaxe :

audio_free_buffer_sound(index);

ArgumentsType Description
index The index of the buffered sound to free.

Les retours :

Exemple :

audio_free_buffer_sound(soundID);

Le code ci-dessus libère le son en mémoire tampon indexé dans la variable "soundID".
