audio_sound_set_listener_mask

Cette fonction peut être utilisée pour définir le masque binaire d'un son afin qu'il ne soit lu que par les auditeurs spécifiés. Vous pouvez créer un masque binaire en utilisant la fonction audio_get_listener_info() puis en utilisant le bitwise or ("|") pour créer un masque personnalisé avec les auditeurs à partir desquels vous souhaitez que le son soit lu, puis en appliquant ce masque personnalisé au son après l'avoir lu. Ce masque remplacera le masque par défaut ou celui que vous avez pu définir à l'aide de la fonction audio_set_listener_mask().

Cette fonction ne peut être utilisée sur un son qu'après qu'il ait été réglé pour être joué et nécessite l'ID du son tel que renvoyé par la fonction utilisée pour jouer le son.

Syntaxe :

audio_sound_set_listener_mask(soundID, mask);

Arguments Type Description
soundID ID de l'instance sonore The unique ID of the sound to set the mask of
mask Real The bitmask data to set for the sound

Les retours :

N/A

Exemple :

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

Le code ci-dessus joue un son puis définit le(s) écouteur(s) à partir duquel/desquels le son sera joué en utilisant les données du masque stockées dans une variable globale.
