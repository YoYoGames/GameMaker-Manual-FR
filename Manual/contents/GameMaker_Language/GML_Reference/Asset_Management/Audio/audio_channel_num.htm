numéro_canal_audio

Cette fonction vous permet de définir le nombre de canaux audio disponibles pour la lecture de sons dans GameMaker. Cela signifie que vous indiquez le nombre de sons simultanés qui peuvent être joués à un moment donné. Si ce nombre est dépassé, les sons de moindre priorité sont arrêtés afin de libérer un canal pour les sons de priorité supérieure. Vous pouvez utiliser cette fonction pour optimiser votre jeu pour les périphériques, car plus le nombre de canaux audio est faible, meilleures sont les performances, mais gardez à l'esprit que cela peut également entraîner une coupure notable de certains sons si beaucoup sont joués en même temps.

Syntaxe :

audio_channel_num(num);

ArgumentsType Description
num Number of available audio channels (default is 128).

Les retours :

Exemple :

switch (os_browser)
{
    case browser_not_a_browser:
        switch (os_type)
        {
            case os_windows:
            case os_macos:
                audio_channel_num(200);
            break;

            default:
                audio_channel_num(64);
            break;
        }
    break;

    default:
        audio_channel_num(16);
    break;
}

Le code ci-dessus vérifie la plate-forme sur laquelle le jeu est exécuté et modifie le nombre de canaux sonores disponibles afin d'augmenter les performances.
