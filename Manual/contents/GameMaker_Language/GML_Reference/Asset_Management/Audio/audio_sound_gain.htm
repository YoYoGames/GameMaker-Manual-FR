audio_sound_gain

Cette fonction permet d'ajouter ou de supprimer un son sur une période donnée, ou de régler le gain du son instantanément.

Le temps est mesuré en millisecondes, et la fonction exige que vous entriez un niveau de gain final que le son doit avoir atteint à la fin de ce temps. Ce gain peut être compris entre 0 (silencieux) et toute valeur supérieure à 0, bien que normalement vous considériez le volume maximum comme 1. Ce gain peut être compris entre 0 (silencieux) et toute valeur supérieure à 0, bien que normalement vous considériez que le volume maximum est de 1. Tout ce qui est supérieur à 1 peut être utilisé mais, selon le son utilisé et la plate-forme sur laquelle il est compilé, vous pouvez obtenir une distorsion ou un écrêtage lorsque le son est lu. Notez que l'échelle de gain est linéaire, et pour changer instantanément le gain, il suffit de mettre l'argument temps à 0.

Cette fonction affectera toutes les instances du son en cours de lecture dans la pièce si l'index est une ressource sonore, et le volume final sera le volume auquel toutes les autres instances du son seront jouées. Cependant, si vous avez utilisé l'index renvoyé par une fonction telle que audio_play_sound() cela n'affectera que cette seule instance du son.

Syntaxe :

audio_sound_gain(index, volume, time);

Arguments Type Description
index ID de l'actif sonore ou de l'instance sonore The index of the sound to set the gain for.
volume Real Value for the music volume.
time Real The length for the change in gain in milliseconds.

Les retours :

N/A

Exemple :

if val
{
    var snd = audio_play_sound(snd_fountain);
    audio_sound_gain(snd, 0, 0);
    audio_sound_gain(snd, 1, 5000);
}

Le code ci-dessus vérifie une variable et s'il renvoie true, il attribue alors l'index d'un son à jouer à la variable locale "snd". Cette variable est ensuite utilisée pour réduire le volume de ce son spécifique à 0 et le faire passer au volume maximum en 5 secondes.
