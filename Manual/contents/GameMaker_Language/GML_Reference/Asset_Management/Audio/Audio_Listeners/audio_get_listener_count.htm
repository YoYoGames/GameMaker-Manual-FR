audio_get_listener_count

Certaines plates-formes cibles autorisent plus d'un écouteur, il est donc important que vous sachiez combien la cible en a avant de changer ou d'utiliser différents écouteurs. Cette fonction renvoie le nombre d'écouteurs disponibles.

Syntaxe :

audio_get_listener_count();

Les retours :

Exemple :

global.listener_num = audio_get_listener_count();

Le code ci-dessus obtient le nombre d'auditeurs disponibles et stocke la valeur de retour dans une variable globale.

Retour : Écouteurs audio
