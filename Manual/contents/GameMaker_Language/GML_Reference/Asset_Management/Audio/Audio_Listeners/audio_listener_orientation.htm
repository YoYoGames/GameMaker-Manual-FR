Orientation de l'auditeur

Cette fonction permet de modifier l'orientation de l'auditeur dans l'espace audio 3D. La direction du regard et la direction vers le haut sont basées sur les vecteurs qui sont résolus à partir des positions relatives x, y et z données, et ont par défaut la valeur (0, 0, 1) pour la direction du regard et (0, 1, 0) pour la direction vers le haut, comme le montre l'illustration ci-dessous :

NOTE si vous avez plusieurs écouteurs, vous devez utiliser la fonction audio_listener_set_orientation().

En changeant l'orientation de l'auditeur avec cette fonction, vous modifiez la façon dont les sons créés par les émetteurs audio autour de la salle de jeu sont perçus par le joueur de votre jeu. Dans l'exemple ci-dessous, les sons créés par l'émetteur lorsque l'auditeur est dans la position par défaut sembleraient provenir du bas et de la droite de l'auditeur, mais avec la nouvelle position et orientation de l'auditeur, ils seront désormais perçus comme provenant du haut et de la droite.

Syntaxe :

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Arguments Type Description
lookat_x Real The x look vector (default 0).
lookat_y Real The y look vector (default 0).
lookat_z Real The z look vector (default 1).
up_x Real The x up vector (default 0).
up_y Real The y up vector (default 1).
up_z Real The z up vector (default 0).

Les retours :

N/A

Exemple :

dx = dcos(direction);
dy = - dsin(direction);
dz = - dsin(zdirection);
audio_listener_position(x, y, z);
audio_listener_orientation(dx, dy, dz, 0, 0, 1);

Le code ci-dessus définit la position de l'écouteur audio 3D en fonction des variables x et y d'une instance et d'une variable z supplémentaire. Il définit ensuite l'orientation de l'écouteur audio à trois variables calculées à partir de la variable direction de l'instance et d'une variable supplémentaire d'orientation z.
