audio_get_listener_info

Cette fonction va créer une carte DS et la remplir d'informations pour l'auditeur donné.

NOTE Vous êtes responsable de la destruction du site DS Map retourné en utilisant la fonction appropriée.

La carte DS contiendra les clés suivantes :

  • "name" - Le nom de l'écouteur, sous forme de chaîne, "default" étant le nom standard de l'écouteur sur la plupart des plates-formes cibles.
  • "mask" - Le masque de bits pour l'écouteur.
  • "index" - La valeur d'index unique de l'écouteur.

La valeur du masque peut être utilisée pour définir un son ou un émetteur afin qu'il soit joué par plusieurs auditeurs à la fois, en utilisant simplement le bitwise or "|" pour générer un masque pour le son (voir l'exemple de code ci-dessous), tandis que l'index est utilisé pour définir les propriétés telles que la position ou la vitesse pour un auditeur donné en utilisant des fonctions telles que audio_listener_set_position().

Syntaxe :

audio_get_listener_info(num);

Arguments Type Description
num ID de l'auditeurréel ou audio The listener number to get the data for.

Les retours :

DS Map ID

Exemple :

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Le code ci-dessus vérifie le nombre d'écouteurs disponibles puis passe en boucle à travers eux pour obtenir leurs bits de masque, qui sont ensuite combinés pour créer un masque à bit unique qui est appliqué à l'écouteur global.
