audio_get_listener_mask

Cette fonction renvoie les données du masque de bits qui définissent le masque par défaut (global) actuel pour les auditeurs audio.

Syntaxe :

audio_get_listener_mask();

Les retours :

Exemple :

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

Le code ci-dessus récupère les données du masque de l'auditeur actuel et les compare aux données stockées dans une variable globale. Si elles ne sont pas identiques, le code vérifie le nombre d'auditeurs disponibles, puis boucle sur ceux-ci et récupère leurs bits de masque, qui sont ensuite combinés pour créer un masque unique qui est appliqué au système audio pour définir les auditeurs globaux.
