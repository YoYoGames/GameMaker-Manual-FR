audio_listener_set_orientation

Avec cette fonction, vous pouvez modifier l'orientation de l'écouteur donné dans l'espace audio 3D. L'indice par défaut de l'écouteur est 0, mais vous pouvez utiliser la fonction audio_get_listener_info() pour obtenir les différents indices disponibles pour la plateforme cible.

Le vecteur d'observation et le vecteur haut sont basés sur les valeurs qui sont résolues à partir des positions relatives x, y et z données, et ont par défaut la valeur (0, 0, 1 ) pour le vecteur d'observation et (0, 1, 0) pour le vecteur haut, comme le montre l'illustration ci-dessous :

En changeant l'orientation de l'auditeur avec cette fonction, vous modifiez la façon dont les sons créés par les émetteurs audio autour de la salle de jeu sont perçus par le joueur de votre jeu. Dans l'exemple ci-dessous, les sons créés par l'émetteur lorsque l'auditeur est dans la position par défaut sembleraient provenir du bas et de la droite de l'auditeur, mais avec la nouvelle position et orientation de l'auditeur, ils seront désormais perçus comme provenant du haut et de la droite.

Syntaxe :

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Arguments Type Description
index Real ou Audio Listener ID The listener to set the orientation of.
lookat_x Real The x look vector (default 0).
lookat_y Real The y look vector (default 0).
lookat_z Real The z look vector (default 1).
up_x Real The x up vector (default 0).
up_y Real The y up vector (default 1).
up_z Real The z up vector (default 0).

Les retours :

N/A

Exemple :

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

Le code ci-dessus récupère la matrice de vue pour la vue de la caméra [0] et l'utilise ensuite pour définir la position et l'orientation de l'écouteur audio pour l'écouteur dont l'ID est stocké dans la variable globale "Player_Listener".
