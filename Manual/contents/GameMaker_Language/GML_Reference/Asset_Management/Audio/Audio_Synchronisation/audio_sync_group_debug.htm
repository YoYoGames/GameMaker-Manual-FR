audio_sync_group_debug

Cette fonction peut être utilisée pour afficher des informations de débogage sur un groupe de synchronisation donné (l'index du groupe est la valeur renvoyée lorsque le groupe a été créé à l'aide de la fonction audio_create_sync_group()). Vous pouvez appeler la fonction avec une valeur de -1 pour désactiver la superposition.

Lorsque vous êtes sur debugging les groupes de synchronisation audio, la superposition s'affiche :

  • Quel groupe de synchronisation est en train de déboguer
  • Combien de chaînes y a-t-il ?
  • Combien de fois le groupe de synchronisation a échoué (cela signifie qu'il a essayé de faire trop de travail).
  • Une liste des canaux
    • Rouge signifie que le canal n'est pas en train de travailler.
    • Jaune signifie que la voie est dans l'état initial (pas encore de décodage ou de lecture)
    • Vert signifie que la voie est en cours de recherche/décodage
    • Violet signifie que la chaîne a épuisé ses données
  • Une barre de profilage qui indique le travail effectué par le thread de décodage.
    • bleu signifie qu'un fichier est lu pour la première fois.
    • rouge : le thread est en train de lire un fichier pour savoir d'où il doit être lu
    • vert : le thread est en train de décoder des données
    • violet : le thread est en sommeil jusqu'à ce qu'il ait besoin de travailler.

Notez que pour le site général debugging de l'audio, vous disposez également de la fonction audio_debug().

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_sync_group_debug(group_index);

Arguments Type Description
group_index ID du groupe de synchronisation audio The group index to debug, or -1 to switch off the debugging.

Les retours :

N/A

Exemple :

if (debug_mode)
{
    audio_sync_group_debug(sg);
}
else
{
    audio_sync_group_debug(-1);
}

Le code ci-dessus activera ou désactivera la superposition de débogage du groupe de synchronisation audio montrant le groupe de synchronisation indexé dans la variable "sg" selon que le jeu est exécuté en mode de débogage ou non.
