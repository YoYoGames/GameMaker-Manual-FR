audio_start_sync_group

Cette fonction vous permet de lancer la lecture d'un groupe de synchronisation précédemment créé. Vous devez fournir l'index du groupe qui est la valeur retournée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group().

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_start_sync_group(group_index);

Arguments Type Description
group_index ID du groupe de synchronisation audio The group index to play.

Les retours :

N/A

Exemple :

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

L'exemple ci-dessus crée un nouveau groupe de synchronisation et assigne l'index du groupe à la variable "sg". Quatre sons sont ensuite ajoutés au groupe, le gain de trois d'entre eux étant fixé à 0. Enfin, le groupe de synchronisation est joué.
