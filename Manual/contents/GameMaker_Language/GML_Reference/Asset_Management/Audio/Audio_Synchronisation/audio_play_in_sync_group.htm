audio_play_in_sync_group

Cette fonction permet d'attribuer un son à un groupe de synchronisation précédemment créé. L'index du groupe est la valeur renvoyée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group()et l'index du son est l'index d'un son compressé (*.ogg ou *.mp3) provenant de l'Asset Browser. Cette fonction ajoute uniquement le son au groupe, mais ne le lit pas. La valeur de retour sera >= 0 si elle réussit et -1 si elle échoue, où une valeur réussie indique l'ordre du son dans le groupe de synchronisation (donc 0 serait le premier son ajouté, 1 serait le deuxième, etc...).

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_play_in_sync_group(group_index, sound_index);

Arguments Type Description
group_index ID du groupe de synchronisation audio The group index to assign the sound to.
sound_index Actif sonore The sound index to be assigned to the group.

Les retours :

Real

Exemple :

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_play_in_sync_group(sg, sound3);
audio_play_in_sync_group(sg, sound4);
audio_start_sync_group(sg);

L'exemple ci-dessus crée un nouveau groupe de synchronisation et assigne l'index du groupe à la variable "sg". Quatre sons sont ensuite ajoutés au groupe, le gain de trois d'entre eux étant fixé à 0. Enfin, le groupe de synchronisation est joué.
