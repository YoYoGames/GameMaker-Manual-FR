audio_create_sync_group

Crée un groupe de synchronisation et renvoie une valeur d'identification unique pour celui-ci, qui doit ensuite être utilisée dans tous les autres appels de fonction audio pour ce groupe. Si vous souhaitez que le groupe tourne en boucle, passez le paramètre true, sinon passez le paramètre false, mais notez que si vous souhaitez qu'il tourne en boucle, toutes les pistes ajoutées ultérieurement doivent avoir la même longueur. Notez que lorsque vous créez un groupe de synchronisation, vous devrez libérer la mémoire et les sons qui lui sont associés lorsqu'ils ne sont pas utilisés à l'aide de la fonction audio_destroy_sync_group() par exemple, dans les événements Room End ou Destroy.

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_create_sync_group(loop);

Arguments Type Description
loop Booléen Whether the tracks in the group should loop (true) ou non (false).

Les retours :

Audio Sync Group ID

Exemple :

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

L'exemple ci-dessus crée un nouveau groupe de synchronisation et assigne l'index du groupe à la variable "sg". Quatre sons sont ensuite ajoutés au groupe, le gain de trois d'entre eux étant fixé à 0. Enfin, le groupe de synchronisation est joué.
