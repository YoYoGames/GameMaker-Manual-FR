audio_sound_get_pitch

Cette fonction peut être utilisée pour obtenir la hauteur de son d'un son donné. Le son peut être référencé à partir d'un index pour un son individuel en cours de lecture qui a été stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ou un son réel provenant du navigateur d'actifs.

Syntaxe :

audio_sound_get_pitch(index);

Arguments Type Description
index ID de l'actif sonore ou de l'instance sonore The index of the sound to get the pitch of.

Les retours :

Real

Exemple :

if audio_sound_get_pitch(snd_Explode) != 1
{
    audio_sound_pitch(snd_Explode, 1);
}

Le code ci-dessus changera la hauteur de l'audio joué à partir du son indexé comme "snd_Explode" si sa hauteur n'est pas égale à 1.
