audio_resume_sound

Avec cette fonction, vous pouvez reprendre n'importe quel son qui est actuellement en pause (après avoir utilisé la fonction audio_pause_sound()). Le son peut être soit une instance unique d'un son (l'index des sons individuels en cours de lecture peut être stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore, auquel cas toutes les instances du son en question seront relancées.

Syntaxe :

audio_resume_sound(index);

Arguments Type Description
index ID de l'actif sonore ou de l'instance sonore The index of the sound to resume.

Les retours :

N/A

Exemple :

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

Le code ci-dessus vérifie s'il y a eu une pression sur la touche "P" du clavier et, s'il en détecte une, il attribue à la variable globale "Pause" la valeur true ou false, puis met en pause le son indexé dans la variable "snd_Waterfall" ou reprend le son à partir de son état de pause.
