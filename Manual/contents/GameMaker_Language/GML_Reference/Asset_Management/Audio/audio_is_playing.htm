audio_s_playing

Cette fonction vérifiera le son donné pour voir s'il est en cours de lecture. Le son peut être une instance unique d'un son (l'index des sons individuels en cours de lecture peut être stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore, auquel cas toutes les instances du son donné seront vérifiées et si l'une d'entre elles est en cours de lecture, la fonction renverra true, sinon elle renverra false. Notez que cette fonction renverra toujours true si le son vérifié a été précédemment mis en pause à l'aide de la fonction audio_pause_sound() fonction.

Syntaxe :

audio_is_playing(index);

Arguments Type Description
index ID de l'actif sonore ou de l'instance sonore The index of the sound to check.

Les retours :

Boolean

Exemple :

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

Le code ci-dessus vérifie si le son indexé dans la variable "snd_Waterfall" est en cours de lecture et s'il renvoie false, le son sera mis en boucle à la position de la pièce, avec une distance de chute de 300 et une faible priorité.
