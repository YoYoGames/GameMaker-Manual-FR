audio_sound_set_track_position

Cette fonction permet de définir la position (en secondes) pour l'ID sonore ou le poste donné.

L'"index" fourni peut être soit une instance unique d'un son (tel que retourné à partir de audio_play_sound()) ou un actif sonore (par exemple, un actif ajouté via le navigateur d'actifs). Le comportement de cette fonction sera différent selon le type d'index que vous avez spécifié :

  • S'il s'agit d'un ID de son unique que vous utilisez, qui est déjà en train de jouer, alors sa position sera immédiatement modifiée au moment donné.
  • S'il s'agit d'un élément sonore, toutes les lectures ultérieures de cet élément sonore commenceront à la nouvelle heure.

Cette fonction modifie uniquement la position de la piste pour le son en cours de lecture, ou pour sa prochaine lecture (si un actif sonore est fourni). Si le son est lu avec le bouclage activé, les lectures suivantes commenceront toujours à partir du début (0,0 seconde), et non à partir de la position de la piste définie à l'aide de cette fonction. Par exemple, si vous démarrez une boucle sonore à 5,7 secondes, le premier son sera lu à partir de ce point, mais après cela, il continuera à répéter la piste entière, du début à la fin.

Syntaxe :

audio_sound_set_track_position(index, time);

Arguments Type Description
index ID de l'actif sonore ou de l'instance sonore The index of the sound to change.
time Real The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

Les retours :

N/A

Exemple :

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

Le code ci-dessus joue un son et utilise ensuite la valeur ID retournée pour définir la position de départ du son à 32 secondes.
