audio_master_gain

Cette fonction vous permet de définir la valeur absolue du volume global de tous les sons et de toute la musique. Elle est basée sur une échelle linéaire allant de 0 (silencieux) à toute valeur supérieure à 0, bien que normalement le volume maximal soit de 1. Tout ce qui est supérieur à 1 peut être utilisé mais, en fonction du son utilisé et de la plate-forme sur laquelle il est compilé, vous pouvez obtenir une distorsion ou un écrêtage lorsque le son est lu. Cette fonction affectera le volume relatif de tous les sons et de la musique joués dans votre jeu.

Syntaxe :

audio_master_gain(gain);

Arguments Type Description
gain Real Value for the global volume (0 to 1).

Les retours :

N/A

Exemple :

if keyboard_check(vk_up)
{
    if vol < 1 vol += 0.05;
    audio_master_gain(vol);
}
if (keyboard_check(vk_down))
{
    if vol > 0 vol -= 0.05;
    audio_master_gain(vol);
}

Le code ci-dessus vérifie les pressions sur les touches flèches haut et bas, qui augmentent ou diminuent la variable "vol". Cette variable est ensuite utilisée pour définir le gain global de tous les sons et musiques du jeu.
