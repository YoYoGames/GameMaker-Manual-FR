audio_pause_all

Cette fonction vous permet de mettre en pause tous les sons en cours de lecture.

Syntaxe :

audio_pause_all();

Les retours :

Exemple :

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

Le code ci-dessus vérifie si la touche "P" du clavier a été enfoncée et, s'il en détecte une, il attribue à la variable globale "Pause" la valeur true ou false, puis met en pause tous les sons ou redémarre tous les sons précédemment mis en pause.
