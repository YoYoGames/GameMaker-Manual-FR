audio_set_master_gain

Cette fonction vous permet de définir la valeur absolue du volume global de tous les sons et de la musique pour un auditeur spécifique. L'indice par défaut de l'auditeur est 0, mais vous pouvez utiliser la fonction audio_get_listener_info() pour obtenir les différents indices disponibles pour la plate-forme cible. La valeur du gain est basée sur une échelle linéaire allant de 0 (silencieux) à 1 (plein volume) et affectera le volume relatif de tous les sons et de la musique joués dans votre jeu par le biais de cet auditeur.

Syntaxe :

audio_set_master_gain(listenerIndex, gain);

Arguments Type Description
listenerIndex ID de l'auditeurréel ou audio The index of the listener to set the gain on.
gain Real Value for the global volume (0 to 1).

Les retours :

N/A

Exemple :

var num = audio_get_listener_count();
for (var i = 0; i < num; i++)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

Le code ci-dessus passe en boucle les écouteurs disponibles et règle ensuite leur gain maître à 0,75.
