audio_sound_pitch

Cette fonction peut être utilisée pour modifier la hauteur d'un son donné. Le son peut être soit un son référencé à partir d'un index pour un son individuel joué qui a été stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ou un son réel provenant de l'explorateur de ressources. S'il s'agit de l'index d'un son en cours de lecture, seule cette instance sera modifiée, mais si vous utilisez un actif sonore du navigateur d'actifs, toutes les instances de cet actif sonore en cours de lecture seront modifiées.

L'argument de la hauteur est un multiplicateur de hauteur, en ce sens que la valeur d'entrée multiplie la hauteur actuelle par cette quantité, donc la valeur par défaut de 1 ne change pas la hauteur, tandis qu'une valeur inférieure à 1 abaisse la hauteur et supérieure à 1 l'élève. Il est préférable d'utiliser de petits incréments pour cette fonction car toute valeur inférieure à 0 ou supérieure à 5 peut ne pas être audible de toute façon. Il convient de noter que le changement de hauteur total autorisé est limité à (1/256) - 256 octaves, de sorte que toute valeur supérieure ou inférieure ne sera pas enregistrée.

La valeur fixée ci-dessus est celle à laquelle GameMaker tente de fixer la gamme, mais cette valeur n'est pas garantie sur toutes les plateformes cibles. iOS Par exemple, la valeur de (1/256) - 8 est fixée par GameMaker, ce qui signifie que vous devrez peut-être faire des essais sur chaque plate-forme cible et disposer de différentes versions d'une ressource sonore, chacune étant décalée, si vous avez besoin de valeurs d'octave supérieures ou inférieures.

Syntaxe :

audio_sound_pitch(index, pitch);

Arguments Type Description
index Actif sonore ou ID de l'instance sonore The index of the sound to change.
pitch Real The pitch multiplier (default 1).

Les retours :

N/A

Exemple :

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
    case 1: audio_sound_pitch(s_engine, 0.8); break;
    case 2: audio_sound_pitch(s_engine, 0.9); break;
    case 3: audio_sound_pitch(s_engine, 0.95); break;
    case 4: audio_sound_pitch(s_engine, 1); break;
    case 5: audio_sound_pitch(s_engine, 1.2); break;
}

Le code ci-dessus changera la hauteur de l'audio joué à partir du son indexé dans la variable "s_engine" en fonction de la valeur de la variable "gear".
