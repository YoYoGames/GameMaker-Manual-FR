audio_get_type

Lorsque vous ajoutez du son à GameMaker, il peut être soit diffusé en continu, soit en mémoire. Si vous avez besoin de savoir si un index de son donné correspond à de l'audio en streaming ou non, vous pouvez utiliser cette fonction qui renverra 1 pour un son diffusé, 0 pour un son en mémoire, et -1 s'il y a une erreur ou si l'index ne pointe pas vers une ressource sonore valide.

Syntaxe :

audio_get_type(index);

ArgumentsType Description
index The index of the sound to check.

Les retours :

Exemple :

type = audio_get_type(snd_Music_1);

Le code ci-dessus vérifie le type d'audio indexé dans la variable "snd_Music_1" et stocke la valeur retournée dans la variable "type".
