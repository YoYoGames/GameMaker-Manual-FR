flux audio_destroy_stream

Si vous avez précédemment créé un flux audio à partir d'un fichier à l'aide de la fonction audio_create_stream() et que vous n'avez plus besoin de ce son, vous pouvez appeler cette fonction pour le supprimer de la mémoire. Tout autre appel à l'index du son après sa destruction entraînera une erreur. Il faut noter que cette fonction libère le flux mais sur la plate-forme cible, cela peut ne pas apparaître dans un gestionnaire de mémoire. Cela s'explique par le fait que GameMaker met en commun les ressources mémoire pour éviter les surcharges d'allocation de mémoire. La mémoire restera donc allouée jusqu'à ce qu'elle soit requise pour autre chose ou réutilisée pour un nouveau flux.

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_destroy_stream(filename);

Arguments Type Description
filename Chaîne de caractères Path to the audio file that was streamed.

Les retours :

N/A

Exemple :

audio_destroy_stream(snd);

Le code ci-dessus supprime de la mémoire le son indexé dans la variable "snd".
