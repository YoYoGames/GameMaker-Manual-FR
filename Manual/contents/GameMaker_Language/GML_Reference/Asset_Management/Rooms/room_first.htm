room_first

Cette variable en lecture seule renvoie l'index de la toute première salle du jeu (définie par l'ordre dans lequel les salles apparaissent dans le gestionnaire de salles et non par l'ordre dans lequel elles ont été créées).

Syntaxe :

room_first

Les retours :

Room Asset

Exemple :

if lives < 1
{
    room_goto(room_first);
}

Le code ci-dessus vérifiera la variable "lives" et si elle est inférieure à 1, il ira dans la première pièce du jeu.
