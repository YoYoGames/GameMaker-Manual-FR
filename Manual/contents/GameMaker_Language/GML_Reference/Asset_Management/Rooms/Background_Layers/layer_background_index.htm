layer_background_index

Cette fonction peut être utilisée pour définir l'index de l'image de l'arrière-plan sprite qui comporte plusieurs sous-images. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez l'indice d'image à utiliser. Si vous définissez une valeur en dehors de la plage des sous-images, l'indice d'image tournera en boucle.

Syntaxe :

layer_background_index(background_element_id, image_index);

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to change
index Real The image index to use for the background

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Le code ci-dessus récupère l'ID du calque nommé "Background_trees" et l'utilise pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour modifier l'index de l'image de l'élément.
