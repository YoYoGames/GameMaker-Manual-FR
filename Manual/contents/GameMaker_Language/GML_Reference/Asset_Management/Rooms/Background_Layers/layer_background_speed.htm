layer_background_speed

Cette fonction peut être utilisée pour définir la vitesse d'une animation sprite qui a été attribuée à un élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis vous définissez la vitesse de l'animation. Si le site IDE a défini la vitesse de l'animation pour le site sprite utilisé de manière à ce qu'elle soit calculée en tant qu'images par image de jeu, la valeur de la vitesse indiquée est un multiplicateur, 1 étant la valeur par défaut, et si vous la réglez sur 0,5, la vitesse de l'animation sera réduite de moitié, tandis que si vous la réglez sur 2, elle sera doublée. Toutefois, si le site IDE a défini la vitesse de l'animation pour le site sprite utilisé afin qu'elle soit calculée en termes d'images par seconde, la valeur de la vitesse indiquée est une valeur d'images par seconde littérale, de sorte qu'une valeur de 5, par exemple, fera en sorte que l'animation passe en boucle de 5 images à chaque seconde de jeu. Si l'élément d'arrière-plan n'a pas de sprite attaché ou si le sprite utilisé n'a pas de sous-images, cette fonction n'aura aucun effet.

Syntaxe :

layer_background_speed(background_element_id, speed);

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to change
speed Real The speed multiplier/value to use for the element

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

Le code ci-dessus récupère l'ID du calque nommé "Background_water" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour modifier la vitesse de l'image de l'élément.
