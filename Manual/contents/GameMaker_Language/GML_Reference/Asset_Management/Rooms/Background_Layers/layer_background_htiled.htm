layer_background_htiled

En utilisant cette fonction, vous pouvez changer si l'élément d'arrière-plan donné sur un calque doit être empilé horizontalement ou non. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis vous définissez la valeur du tuilage. Si vous définissez la valeur true, l'élément sera affiché en mosaïque horizontale dans la pièce, et si vous définissez la valeur false, l'élément ne sera pas en mosaïque. Cette fonction est réservée aux projets 2D et ne fonctionnera pas correctement en cas de projection avec une caméra 3D.

Syntaxe :

layer_background_htiled(background_element_id, tiled);

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to change
tiled Booléen The horizontal tiling toggle, which can be true ou false

Les retours :

N/A

Exemple :

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

Le code ci-dessus vérifiera l'élément d'arrière-plan attribué au calque sur lequel se trouve l'instance qui exécute le code et s'il n'est pas tuilé dans un sens ou dans l'autre, il le mettra en tuile verticalement et horizontalement.
