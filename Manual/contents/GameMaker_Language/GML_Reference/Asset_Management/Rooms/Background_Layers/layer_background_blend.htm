layer_background_blend

Cette fonction contrôle le mélange (ou "teinte") de l'arrière-plan sprite et la valeur par défaut est -1 (qui représente la constante c_white, qui peut également être utilisée). Toute autre valeur (y compris les constantes de couleur internes comme c_red, ou c_aqua) permet de mélanger la couleur spécifiée avec la couleur d'origine sprite. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez la couleur de fusion à utiliser. Vous pouvez voir ci-dessous un exemple de sprite qui a été mélangé avec différentes couleurs :

Veuillez noter que vous devez essayer de limiter les mélanges sur la plate-forme HTML5 (à moins d'utiliser WebGL), car chaque mélange doit être mis en cache séparément. mélangé sprite doit être mis en cache séparément et le fait de disposer de nombreux mélanges sprites peut nuire aux performances (vous pouvez également définir la taille de cache à l'aide de la fonction sprite_set_cache_size()).

Syntaxe :

layer_background_blend(background_element_id, blend);

ArgumentsType Description
background_element_id The unique ID value of the background element to change
blend The colour to blend with the background sprite (default is c_white)

Les retours :

Exemple :

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_blend(back_id, c_aqua);

Le code ci-dessus récupère la valeur ID de l'arrière-plan assigné au calque "Background_sky" et le teinte ensuite en une couleur.
