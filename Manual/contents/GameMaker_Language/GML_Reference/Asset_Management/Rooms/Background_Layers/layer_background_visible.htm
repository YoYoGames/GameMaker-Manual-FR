couche_arrière-plan_visible

Cette fonction permet d'activer la visibilité d'un élément d'arrière-plan. Vous fournissez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez l'argument visible sur true ou false. Lorsque vous définissez true, l'élément sera affiché, et lorsque vous définissez false, l'élément ne sera pas affiché. Notez que cela dépend de la visibilité de la couche, et même si l'élément d'arrière-plan est marqué comme visible, il ne sera pas affiché si la couche sur laquelle il se trouve est marquée comme non visible.

Syntaxe :

layer_background_visible(background_element_id, visible)

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to change
visible Booléen The visibility toggle, which can be true ou false

Les retours :

N/A

Exemple :

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if layer_background_get_visible(back)
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

Le code ci-dessus vérifie si un bouton de la souris a été pressé et, s'il est détecté, il fait basculer la visibilité de l'élément d'arrière-plan attribué au calque sur lequel se trouve l'instance qui exécute le code.
