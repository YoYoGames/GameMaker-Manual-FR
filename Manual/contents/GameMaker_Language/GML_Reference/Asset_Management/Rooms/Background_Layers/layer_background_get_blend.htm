couche_arrière-plan_get_blend

Cette fonction peut être utilisée pour obtenir la couleur de fusion de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renverra une valeur réelle qui représente la couleur attribuée.

Syntaxe :

layer_background_get_blend(background_element_id);

ArgumentsType Description
background_element_id The unique ID value of the background element to get the information from

Les retours :

Colour Constant

Exemple :

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_blend(back_id) == c_white)
{
    layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

Le code ci-dessus récupère l'ID du calque nommé "Background_sky" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier la couleur de mélange de l'élément et s'il est équivalent à la constante c_white, alors le mélange du calque est réglé sur une couleur aléatoire.
