layer_background_get_vtiled

Cette fonction peut être utilisée pour savoir si l'élément d'arrière-plan est carrelé verticalement ou non. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renverra soit true si l'élément est actuellement tuilé, soit false s'il ne l'est pas.

Syntaxe :

layer_background_get_vtiled(background_element_id);

ArgumentsType Description
background_element_id The unique ID value of the background element to get the information from

Les retours :

Exemple :

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_vtiled(back_id)
{
    layer_background_vtiled(back_id, true);
}

Le code ci-dessus récupère l'ID du calque nommé "Background_sky" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier si l'élément est incliné sur l'axe vertical, et s'il ne l'est pas, il est configuré pour l'être.
