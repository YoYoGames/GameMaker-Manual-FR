couche_arrière-plan_get_stretch

Cette fonction peut être utilisée pour obtenir l'état étiré de l'élément d'arrière-plan sprite. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renvoie soit true si l'élément sprite est actuellement étiré pour s'adapter à la pièce, soit false s'il ne l'est pas.

Syntaxe :

layer_background_get_stretch(background_element_id);

ArgumentsType Description
background_element_id The unique ID value of the background element to get the information from

Les retours :

Exemple :

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_stretch(back_id)
{
    layer_background_stretch(back_id, false);
}

Le code ci-dessus obtiendra l'ID du calque nommé "Background_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier si l'élément sprite sera étiré pour s'adapter à la pièce ou non et s'il est étiré, alors cette propriété est définie sur false.
