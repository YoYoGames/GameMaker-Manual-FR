couche_arrière-plan_get_sprite

Cette fonction peut être utilisée pour obtenir la valeur actuelle de l'index sprite de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renvoie une valeur réelle qui représente l'indice sprite affiché. Si l'élément n'a pas de sprite attribué, la fonction renverra -1.

Syntaxe :

layer_background_get_sprite(background_element_id);

ArgumentsType Description
background_element_id The unique ID value of the background element to get the information from

Les retours :

Sprite Asset or -1

Exemple :

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

Le code ci-dessus récupère l'ID du calque nommé "Background_sky", puis l'utilise pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier l'adresse sprite attribuée à l'élément, en lui attribuant la valeur sprite "spr_Clouds" si ce n'est pas déjà le cas.
