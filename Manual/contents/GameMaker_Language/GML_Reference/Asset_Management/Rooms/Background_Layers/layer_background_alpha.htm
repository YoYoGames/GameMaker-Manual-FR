couche_arrière-plan_alpha

Cette fonction contrôle l'alpha (la transparence) de l'arrière-plan sprite. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la commande layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez la valeur alpha à utiliser. La valeur alpha peut être comprise entre 0 (totalement transparent) et 1 (totalement opaque), la valeur alpha par défaut de l'élément background étant de 1. Notez que si le calque auquel l'élément d'arrière-plan a été attribué n'est pas visible - ou si l'élément lui-même a été rendu invisible - vous ne verrez aucune différence avec cette fonction jusqu'à ce que le calque ou l'élément soit à nouveau visible. à nouveau.

Syntaxe :

layer_background_alpha(background_element_id, alpha);

ArgumentsType Description
background_element_id The unique ID value of the background element to change
alpha The alpha for background sprite, from 0 to 1 (default is 1)

Les retours :

Exemple :

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

Le code ci-dessus récupère la valeur ID de l'arrière-plan attribué au calque "Background_sky", puis définit son alpha à une valeur aléatoire comprise entre 0 et 1.
