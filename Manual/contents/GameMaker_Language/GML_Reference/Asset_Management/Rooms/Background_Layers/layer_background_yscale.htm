couche_arrière-plan_échelle

Cette fonction peut être utilisée pour définir l'échelle le long de l'axe des y d'un élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis vous définissez la valeur de l'échelle. La valeur d'échelle que vous donnez est un multiplicateur qui sera utilisé pour changer la façon dont l'élément d'arrière-plan est affiché, où une valeur de 0,5 affichera l'élément à la moitié de l'échelle, et une valeur de 2 l'affichera à l'échelle double. Notez que les valeurs négatives sont valables, et qu'elles "retourneront" l'élément autour de la position (0,0), donc une échelle x de -1 affichera l'image inversée.

Syntaxe :

layer_background_yscale(background_element_id, yscale);

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to change
yscale Real The speed multiplier/value to use for the element

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_yscale(back_id, -1);

Le code ci-dessus obtiendra l'ID du calque nommé "Background_water" et l'utilisera ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour modifier l'échelle des y de l'élément.
