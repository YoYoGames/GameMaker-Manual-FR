couche_arrière-plan_créer

Cette fonction vous permet d'affecter une ressource sprite à un calque qui sera utilisé comme arrière-plan dans votre projet. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères, ce qui a un impact sur les performances) et un index sprite (qui serait le nom de sprite tel qu'il apparaît dans l'Asset Browser), et l'élément sera ajouté à la couche. La fonction renvoie la valeur d'identification unique de l'élément, qui peut ensuite être utilisée dans d'autres fonctions de couche pour les arrière-plans.

Syntaxe :

layer_background_create(layer_id, sprite)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
sprite Sprite Asset The sprite index to be used

Les retours :

Background Element ID

Exemple :

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Le code ci-dessus crée un nouveau calque et lui ajoute un nouvel élément d'arrière-plan, en définissant un sprite comme étant l'image d'arrière-plan utilisée.
