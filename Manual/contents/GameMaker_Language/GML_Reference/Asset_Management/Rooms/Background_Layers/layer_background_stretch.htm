layer_background_stretch

Cette fonction vous permet de faire en sorte qu'un élément d'arrière-plan sprite s'étire pour s'adapter à la pièce ou reste à 1:1 avec la résolution. Vous fournissez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez l'argument d'étirement sur true ou false. Lorsque vous définissez true, l'élément sprite sera étiré pour s'adapter à la pièce (soit à l'échelle supérieure, soit à l'échelle inférieure, en fonction des tailles de sprite et de la pièce), et lorsque vous définissez false, l'élément sprite sera affiché à sa résolution d'origine.

Syntaxe :

layer_background_stretch(background_element_id, stretch)

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to change
stretch Booléen The stretch toggle, which can be true ou false

Les retours :

N/A

Exemple :

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_stretch(back))
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

Le code ci-dessus vérifie si un bouton de la souris a été pressé et, s'il est détecté, il fait basculer l'étirement de l'élément d'arrière-plan sprite attribué au calque sur lequel se trouve l'instance qui exécute le code.
