layer_background_destroy

Cette fonction va détruire l'élément d'arrière-plan donné. Vous fournissez l'ID de l'arrière-plan (que vous obtenez lorsque vous créez l'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez l'ID du calque avec layer_get_background_id()) et cette fonction le supprimera. Notez que cette fonction ne supprime pas le calque, mais seulement l'arrière-plan. Si l'arrière-plan a été ajouté dans l'éditeur de pièce, la prochaine fois que vous quitterez la pièce et y reviendrez, l'arrière-plan sera recréé. Cependant, si la pièce est persistante, l'arrière-plan sera supprimé, à moins que la persistance de la pièce ne soit à nouveau désactivée.

Syntaxe :

layer_background_destroy(background_element_id)

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to be destroyed

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if (layer_background_exists(lay_id, bck_id))
{
    layer_background_destroy(bck_id);
}

Le code ci-dessus vérifie le calque "Background_trees" pour voir si l'élément d'arrière-plan donné existe et si c'est le cas, alors il est détruit (mais pas le calque).
