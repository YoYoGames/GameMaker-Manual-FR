couche_arrière-plan_existe

Vous pouvez utiliser cette fonction pour vérifier et voir si un élément d'arrière-plan existe sur un calque donné. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()) et la fonction renvoie soit true si l'élément existe, soit false s'il n'existe pas.

REMARQUE: Cette fonction fonctionne dans le cadre de la pièce cible actuelle - par défaut, la pièce dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room().

Syntaxe :

layer_background_exists(layer_id, background_element_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
background_element_id ID de l'élément de fond The unique ID value of the background element to check

Les retours :

Boolean

Exemple :

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
    layer_background_destroy(lay_id, global.TreesBackground);
}

Le code ci-dessus vérifie le calque "Background_trees" pour voir si l'élément d'arrière-plan donné existe et si c'est le cas, alors il est détruit (mais pas le calque).
