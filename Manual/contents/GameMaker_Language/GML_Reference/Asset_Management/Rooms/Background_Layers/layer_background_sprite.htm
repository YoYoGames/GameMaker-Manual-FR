layer_background_sprite

Cette fonction vous permet de définir l'indice sprite de l'élément d'arrière-plan. Vous fournissez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis vous donnez un index sprite à utiliser. L'image de l'élément d'arrière-plan sera remplacée par la nouvelle sprite. Si vous donnez une valeur de -1, l'élément n'aura pas de sprite attribué (mais il existera toujours et pourra avoir un sprite attribué plus tard).

Cette fonction a également l'alias layer_background_change().

Syntaxe :

layer_background_sprite(background_element_id, sprite_index)

Arguments Type Description
background_element_id ID de l'élément de fond The unique ID value of the background element to change
sprite_index Sprite Asset The sprite index of the sprite to use for the background element

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
    layer_background_sprite(back_id, spr_Clouds);
}

Le code ci-dessus récupère l'ID du calque nommé "Background_sky", puis l'utilise pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier l'adresse sprite attribuée à l'élément, en lui attribuant la valeur sprite "spr_Clouds" si ce n'est pas déjà le cas.
