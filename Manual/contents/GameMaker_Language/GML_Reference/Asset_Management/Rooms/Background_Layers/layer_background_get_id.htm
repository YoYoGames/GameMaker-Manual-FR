layer_background_get_id

Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de l'élément d'arrière-plan d'une couche. Vous fournissez l'ID du calque (que vous obtenez lorsque vous utilisez le nom du calque avec la fonction layer_get_id()) et la fonction renvoie la valeur d'identification associée à l'élément d'arrière-plan sur la couche. Notez que cette fonction est spécifiquement conçue pour être utilisée avec des arrière-plans qui ont été ajoutés à l'adresse IDE, car si vous ajoutez un arrière-plan à un calque par le biais d'un code utilisant la fonction layer_background_create()En effet, si vous ajoutez un arrière-plan à une couche par l'intermédiaire d'un code en utilisant la fonction

Syntaxe :

layer_background_get_id(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target

Les retours :

Background Element ID

Exemple :

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

Le code ci-dessus récupère l'ID du calque nommé "Background_trees" et l'utilise pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour modifier l'élément sprite.
