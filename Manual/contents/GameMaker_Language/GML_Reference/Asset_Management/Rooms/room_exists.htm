chambre_existe

Avec cette fonction, vous pouvez vérifier si la pièce que vous spécifiez existe ou non. Cette fonction prend l'index de la pièce (un nombre réel) et non le nom de la pièce (une chaîne). Cette fonction est particulièrement utile lorsque vous créez des salles de façon dynamique à l'aide de la fonction room_add()mais vous pouvez également utiliser les variables en lecture seule room_first et room_last ou les fonctions room_next() et room_previous() pour obtenir l'index d'une pièce spécifique, ou fournir une variable qui a stocké l'index de n'importe quelle autre pièce de votre jeu.

Syntaxe :

room_exists(index);

ArgumentsType Description
index The index of the room to check.

Les retours :

Exemple :

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

Le code ci-dessus vérifie si la pièce indexée dans le tableau "global.rm[]" existe et si c'est le cas, il se rend dans cette pièce.
