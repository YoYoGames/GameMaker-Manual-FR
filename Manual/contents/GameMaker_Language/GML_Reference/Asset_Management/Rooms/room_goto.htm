chambre_goto

Cette fonction vous permet d'accéder à n'importe quelle pièce de votre projet de jeu, qu'elle ait été créée à l'aide du code ou dans le navigateur d'actifs. Vous fournissez l'index de la pièce (stocké dans la variable correspondant au nom de la pièce, ou comme variable retournée par la fonction room_add()). Notez que la room ne changera pas avant la fin de l'événement où la fonction a été appelée, donc tout code après que cette fonction ait été appelée sera toujours exécuté s'il se trouve dans le même événement. Cette fonction déclenchera également l'événement Room End.

REMARQUE Les identifiants de salle ne sont pas basés sur leur ordre dans l'Asset Browser ou le Room Manager, et vous devez donc éviter de fournir directement une valeur numérique. Utilisez plutôt la constante de salle de l'actif que vous voulez référencer (qui est simplement son nom) ou récupérez-la par le biais d'une fonction.

Syntaxe :

room_goto(index);

ArgumentsType Description
index The index of the room to go to.

Les retours :

Exemple :

switch (global.level)
{
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
}

Le code ci-dessus vérifiera une variable globale et changera de pièce en fonction de la valeur qu'elle contient.
