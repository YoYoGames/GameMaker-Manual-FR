layer_set_fx

Cette fonction est utilisée pour assigner un FX struct à une couche de pièce. Vous spécifiez l'ID ou le nom de la couche que vous voulez modifier, puis un FX struct (tel que retourné par fx_create() ou layer_get_fx()) qui sera appliqué à la couche spécifiée, rendant l'effet visible dans la pièce.

Si vous spécifiez un calque qui n'est pas un calque de filtre/effet mais qui est d'un autre type (comme un calque d'instance, un calque d'actif, un calque de tuile, etc.), l'effet sera appliqué au contenu de ce calque ainsi qu'à tous les calques inférieurs.

Si un FX struct est déjà attribué à la couche spécifiée, il sera remplacé par le nouveau.

NOTE L'appel de cette fonction au niveau de la racine d'un script n'aura aucun effet, car les scripts sont exécutés avant le chargement de la première pièce. Vous pouvez d'abord définir une pièce cible et ensuite appeler cette fonction.

Syntaxe :

layer_set_fx(layer_name_or_id, filter_or_effect);

Arguments Type Description
layer_name_or_id Chaîne ou ID de la couche The name or ID of the layer to modify
filter_or_effect FX Struct The FX struct to apply to the layer

Les retours :

N/A

Exemple :

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Le code ci-dessus crée un nouveau FX struct à partir de l'effet "_filter_tintfilter", qui est le filtre "Colour Tint" trouvé dans l'éditeur de pièce. Il assigne une valeur à son paramètre "g_TintCol" qui est la couleur de la teinte, et comme il s'agit d'un vec4 interne, il prend un tableau contenant quatre valeurs (correspondant à ses valeurs rouge, verte, bleue et alpha). Le FX struct pour la teinte est ensuite appliqué à un calque FX.
