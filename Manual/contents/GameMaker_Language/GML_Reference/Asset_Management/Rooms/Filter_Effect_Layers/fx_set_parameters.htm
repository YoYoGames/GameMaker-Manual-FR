fx_set_paramètres

Cette fonction est utilisée pour modifier les paramètres d'un filtre/effet. Vous spécifiez un FX struct (tel que renvoyé par fx_create() ou layer_get_fx()) et un struct contenant ses paramètres (tel que renvoyé par fx_get_parameters()). Vos modifications seront visibles dans la couche à laquelle le FX struct est assigné.

Il n'est pas nécessaire que votre paramètre struct contienne tous les paramètres applicables au filtre/effet, et seules les variables présentes dans le struct donné seront mises à jour pour le filtre/effet.

Syntaxe :

fx_set_parameters(filter_or_effect, parameter_struct);

ArgumentsType Description
filter_or_effect The FX struct to modify
parameter_struct A struct containing the parameters for the filter/effect

Les retours :

Exemple :

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Le code ci-dessus récupère le FX struct pour une couche à laquelle on suppose que le filtre "Colour Tint" est appliqué, et récupère son paramètre struct en appelant fx_get_parameters(). Après cela, il crée une valeur oscillante en utilisant sin() et current_time, qui est ensuite utilisée pour les valeurs RGBA de l'effet de teinte afin qu'il continue à s'animer. Le tableau RGBA est affecté à la variable g_TintCol dans le paramètre struct, et struct est ensuite appliqué au FX struct en appelant fx_set_parameters().
