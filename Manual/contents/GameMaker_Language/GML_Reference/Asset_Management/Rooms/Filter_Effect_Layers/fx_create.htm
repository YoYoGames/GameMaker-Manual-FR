fx_create

Cette fonction est utilisée pour créer un nouveau FX Struct pour un type d'effet donné, qui peut ensuite être appliqué à une couche.

Les noms des Filtres/Effets que vous pouvez passer dans cette fonction sont listés sur cette page : Tous les types de filtres/effets.

Utilisation

Vous pouvez lire les paramètres de l'effet créé en appelant fx_get_parameter/s sur l'objet retourné struct, et leur attribuer de nouvelles valeurs en appelant fx_set_parameter/s.

Notez que le site struct renvoyé par cette fonction ne contiendra aucun membre visible et ne pourra être modifié qu'à l'aide d'une des fonctions de filtre/effet.

Défaillances fonctionnelles

La fonction renvoie -1 si la création de struct a échoué pour une raison quelconque ; cela peut arriver si le nom de l'effet spécifié est incorrect ou si l'effet n'a pas été chargé dans le paquetage du jeu.

Pour que cette fonction fonctionne correctement, assurez-vous que le FX dont vous avez besoin est chargé dans votre paquetage de jeu, comme expliqué sur la page Fonctions de filtre et d'effet:

Si ces conditions ne sont pas remplies, c'est-à-dire si vous ne passez pas une chaîne littérale et que le filtre/effet n'est présent dans aucune pièce, la fonction ne pourra pas charger ce filtre/effet et renverra -1.

Syntaxe :

fx_create(filter_or_effect_name);

Arguments Type Description
filter_or_effect_name Chaîne de caractères The name of the filter/effect to create (get them from All Filter/Effect Types).

Les retours :

FX Struct or -1 (on failure)

Exemple :

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Le code ci-dessus crée un nouveau FX struct à partir de l'effet "_filter_tintfilter", qui est le filtre "Colour Tint" trouvé dans l'éditeur de pièce. Il assigne une valeur à son paramètre "g_TintCol" qui est la couleur de la teinte, et comme il s'agit d'un vec4 interne, il prend un tableau contenant quatre valeurs (correspondant à ses valeurs rouge, verte, bleue et alpha). Le FX struct pour la teinte est ensuite appliqué à un calque FX.
