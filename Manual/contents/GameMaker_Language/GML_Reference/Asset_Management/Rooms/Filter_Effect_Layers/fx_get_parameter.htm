fx_get_parameter

Cette fonction est utilisée pour récupérer la valeur d'un paramètre d'un FX Struct. Vous spécifiez le FX struct à lire (tel que retourné par fx_create() ou layer_get_fx()) et le nom du paramètre sous forme de chaîne de caractères, et la fonction retourne sa valeur actuelle. Il peut s'agir d'une valeur réelle, ou d'un tableau si le paramètre est un vecteur (qui est un groupe de valeurs, tel qu'un vec4 qui stocke une couleur RGBA).

Trouvez les noms des paramètres d'un filtre/effet sur cette page : Types et paramètres de FX.

Syntaxe :

fx_get_parameter(filter_or_effect, parameter_name);

Arguments Type Description
filter_or_effect FX Struct The FX struct to get the parameter from
parameter_name Chaîne de caractères The name of the parameter to get the value of (as a string)

Les retours :

Real or Array

Exemple :

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

Le code ci-dessus récupère le FX struct d'une couche à laquelle on suppose que le filtre "Colour Tint" est appliqué, et récupère la valeur de son paramètre "g_TintCol", qui renvoie un tableau contenant les 4 valeurs de sa couleur de teinte (rouge, vert, bleu et alpha). Il imprime ensuite cette valeur dans le journal de sortie.
