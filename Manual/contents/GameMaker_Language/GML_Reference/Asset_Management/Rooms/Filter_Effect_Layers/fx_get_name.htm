fx_get_name

Cette fonction est utilisée pour récupérer le nom d'un filtre/effet à partir de son struct. Vous spécifiez le FX struct à lire (tel que retourné par fx_create() ou layer_get_fx()) et la fonction retourne son nom sous forme de chaîne. Ce nom peut ensuite être passé dans fx_create() pour créer un nouveau FX struct utilisant le même type de filtre/effet.

Syntaxe :

fx_get_name(filter_or_effect);

ArgumentsType Description
filter_or_effect The FX struct to get the name of

Les retours :

Exemple :

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Le code ci-dessus exécute une for boucle sur toutes les couches présentes dans la pièce, et vérifie chaque couche pour un FX struct. Si une couche possède un FX struct, il vérifie le nom de ce filtre/effet en appelant fx_get_name(); s'il est égal à "_filter_tintfilter", ce qui signifie que c'est un filtre "Colour Tint", il change sa couleur de teinte en bleu.
