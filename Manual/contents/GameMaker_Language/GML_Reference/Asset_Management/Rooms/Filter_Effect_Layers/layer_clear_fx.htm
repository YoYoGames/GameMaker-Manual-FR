couche_clair_fx

Cette fonction est utilisée pour effacer le FX struct qui est appliqué à la couche spécifiée. Vous spécifiez soit l'ID, soit le nom de la couche que vous souhaitez modifier et cette fonction supprimera le FX struct qui est appliqué à cette couche, désactivant ainsi complètement ses effets.

Si vous utilisez cette fonction pour désactiver temporairement un effet et que vous devez le rétablir sur la couche, vous devez d'abord enregistrer le FX struct pour la couche en appelant layer_get_fx(), puis appeler cette fonction pour le désactiver sur la couche. Vous pouvez ensuite le réappliquer à la couche en appelant layer_set_fx(), ce qui réactivera l'effet.

Syntaxe :

layer_clear_fx(layer_name_or_id);

Arguments Type Description
layer_name_or_id Chaîne ou ID de la couche The name or ID of the layer to clear the FX for

Les retours :

N/A

Exemple :

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

Le code ci-dessus vérifie si la touche "E" est enfoncée, puis vérifie si un filtre/effet est appliqué au calque "EffectLayer" ; dans ce cas, il stocke son FX struct dans une variable d'instance temporaire et vide le calque de tout filtre/effet. Lorsque vous appuyez sur la touche "F", il vérifie si la variable temporaire contient autre chose que -1, et dans ce cas, il réapplique cet effet à la couche et réinitialise la variable temporaire à -1. Tout ceci vous permet essentiellement d'activer et de désactiver un effet quand vous le souhaitez sans perdre ses paramètres.
