layer_get_fx

Cette fonction est utilisée pour récupérer le FX struct pour une couche. Vous spécifiez soit l'ID, soit le nom de la couche que vous voulez cibler et la fonction retournera un struct contenant des informations sur l'effet appliqué. Ce struct sera similaire au struct que vous obtenez de fx_create(), et des fonctions fx_get_parameter/s et fx_set_parameter/s peuvent être utilisées pour lire et modifier ses paramètres.

Si aucun filtre/effet n'est appliqué au calque spécifié, la fonction renvoie -1.

Syntaxe :

layer_get_fx(layer_name_or_id);

Arguments Type Description
layer_name_or_id Chaîne ou ID de couche The name or ID of the layer to read

Les retours :

FX Struct (or -1 if not found)

Exemple :

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Le code ci-dessus exécute une for à travers toutes les couches présentes dans la pièce, et vérifie chaque couche pour un FX struct. Si un calque possède un FX struct, il vérifie le nom de ce filtre/effet en appelant fx_get_name() s'il est égal à "_filter_tintfilter", ce qui signifie que c'est un filtre "Colour Tint", il change sa couleur de teinte en bleu.
