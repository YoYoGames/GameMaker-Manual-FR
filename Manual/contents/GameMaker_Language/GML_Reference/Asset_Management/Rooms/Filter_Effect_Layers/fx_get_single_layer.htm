fx_get_single_layer

Cette fonction indique si le filtre/effet fourni utilise le mode Single Layer (true) ou non (false). Voir fx_set_single_layer() pour plus d'informations sur le mode Single Layer.

Vous spécifiez un FX struct (tel que renvoyé par fx_create() ou layer_get_fx()) et la fonction renvoie un booléen qui indique si le mode monocouche est activé ou non pour le FX.

Syntaxe :

fx_get_single_layer(filter_or_effect);

ArgumentsType Description
filter_or_effect The FX struct to read

Les retours :

Boolean

Exemple :

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

Le code ci-dessus récupère le FX struct d'un calque appelé "Distort" et vérifie si le mode Single Layer est activé sur celui-ci. Si ce n'est pas le cas, il active le mode Single Layer pour cet effet.
