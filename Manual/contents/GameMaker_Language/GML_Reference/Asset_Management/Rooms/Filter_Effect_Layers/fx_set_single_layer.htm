fx_set_single_layer

Cette fonction est utilisée pour activer ou désactiver le mode "Single Layer" pour un filtre/effet. Par défaut, un filtre/effet est appliqué au calque auquel il est affecté et à tous les calques inférieurs à ce calque, mais vous pouvez activer le mode Single Layer sur un FX struct pour vous assurer qu'il est uniquement appliqué au calque auquel il est affecté.

Vous spécifiez un FX struct (tel que renvoyé par fx_create() ou layer_get_fx()) et une valeur booléenne pour activer (true) ou désactiver (false) le mode monocouche.

Le visuel suivant montre un filtre utilisé avec le mode monocouche désactivé (qui est le comportement par défaut pour toutes les couches FX) et le même filtre avec le mode monocouche activé :

REMARQUE Lorsque vous activez le mode Single Layer pour un FX, veillez à l'appliquer à un calque qui n'est pas un calque FX, car un calque FX en lui-même ne contient rien et donc aucun filtre/effet ne sera visible.

Syntaxe :

fx_set_single_layer(filter_or_effect, enable);

ArgumentsType Description
filter_or_effect The FX struct to modify
enable A boolean value to enable (true) ou désactiver (false) le mode monocouche.

Les retours :

Exemple :

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

Le code ci-dessus crée un nouveau screenshake FX, active le mode Single Layer sur celui-ci et l'applique ensuite à un calque de pièce. Cela signifie que le filtre screenshake ne sera appliqué qu'au calque "ShakeyThings".
