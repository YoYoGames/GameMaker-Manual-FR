fx_get_paramètres

Cette fonction est utilisée pour récupérer une structure contenant tous les paramètres pour le FX donné struct. Vous spécifiez un FX struct (tel que retourné par fx_create() ou layer_get_fx()) et cette fonction retourne un struct contenant les variables de tous les paramètres applicables à ce filtre/effet. Les paramètres qui utilisent des vecteurs (c'est-à-dire des groupes de valeurs) seront présents sous forme de tableaux.

Vous pouvez obtenir les noms de tous les paramètres dans un tableau séparé en appelant fx_get_parameter_names(), ou les trouver sur cette page : Types et paramètres de FX.

La modification des valeurs dans ce site struct n'appliquera pas ces changements au filtre/effet ; vous devrez appeler fx_set_parameters() pour appliquer la version modifiée de struct au filtre/effet pour que les changements prennent effet.

Syntaxe :

fx_get_parameters(filter_or_effect);

Arguments Type Description
filter_or_effect FX Struct The FX struct for which the parameters will be returned

Les retours :

Struct (or -1 on failure)

Exemple :

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Le code ci-dessus récupère le FX struct pour un calque auquel on suppose que le filtre "Colour Tint" est appliqué, et récupère son paramètre struct en appelant fx_get_parameters(). Après cela, il crée une valeur oscillante en utilisant sin() et current_time, qui est ensuite utilisée pour les valeurs RGBA de l'effet de teinte afin qu'il continue à s'animer. Le tableau RGBA est affecté à la variable g_TintCol dans le paramètre struct, et struct est ensuite appliqué au FX struct en appelant fx_set_parameters().
