fx_get_parameter_names

Cette fonction est utilisée pour récupérer les noms de tous les paramètres présents dans un filtre/effet. Vous spécifiez le FX struct à lire (tel que retourné par fx_create() ou layer_get_fx()) et la fonction retourne un tableau contenant les noms des paramètres du filtre sous forme de chaînes de caractères.

Syntaxe :

fx_get_parameter_names(filter_or_effect);

ArgumentsType Description
filter_or_effect The FX struct to get the parameter names of

Les retours :

(of Strings)

Exemple :

var _fx_struct = layer_get_fx("EffectLayer");
var _param_names = fx_get_parameter_names(_fx_struct);

for (var i = 0; i < array_length(_param_names); i ++) {
    show_debug_message("Parameter " + string(i) + ": " + _param_names[i]);
}

Le code ci-dessus récupère le FX struct pour une couche et obtient les noms de ses paramètres dans un tableau. Il exécute ensuite une for sur ce tableau et imprime chaque paramètre dans le journal de sortie.
