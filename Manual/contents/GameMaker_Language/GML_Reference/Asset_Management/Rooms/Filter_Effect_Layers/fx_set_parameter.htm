fx_set_parameter

Cette fonction est utilisée pour changer la valeur d'un paramètre dans le FX struct donné. Vous spécifiez un FX struct (tel que renvoyé par fx_create() ou layer_get_fx()), un nom de paramètre sous forme de chaîne de caractères, et sa nouvelle valeur. Vos modifications seront visibles dans la couche à laquelle le FX struct est affecté.

Vous pouvez passer plusieurs arguments pour le paramètre s'il s'agit d'un vecteur, mais vous pouvez aussi simplement spécifier un tableau comme seul argument qui contiendra vos multiples valeurs.

Trouvez les noms des paramètres d'un filtre/effet sur cette page : Types et paramètres de FX.

Syntaxe :

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

Arguments Type Description
filter_or_effect FX Struct The FX struct to modify
parameter_name Chaîne de caractères The name of the parameter to change the value of (as a string)
val... Real ou Tableau The new value(s) for that parameter as a real or array

Les retours :

N/A

Exemple :

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Le code ci-dessus crée un nouveau FX struct à partir de l'effet "_filter_tintfilter", qui est le filtre "Colour Tint" trouvé dans l'éditeur de pièce. Il assigne une valeur à son paramètre "g_TintCol" qui est la couleur de la teinte, et comme il s'agit d'un vec4 interne, il prend un tableau contenant quatre valeurs (correspondant à ses valeurs rouge, verte, bleue et alpha). Le FX struct pour la teinte est ensuite appliqué à un calque FX.

Notez que vous pouvez également passer des valeurs au vecteur comme des arguments séparés au lieu d'un tableau :

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
