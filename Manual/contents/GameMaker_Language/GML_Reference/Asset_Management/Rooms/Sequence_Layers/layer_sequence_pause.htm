pause_séquence_couche

Cette fonction permet de mettre en pause la lecture d'une séquence donnée. Vous fournissez l'ID de l'élément de séquence tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche et la fonction mettra la séquence en pause jusqu'à ce que vous recommenciez la lecture en utilisant la fonction layer_sequence_play().

Syntaxe :

layer_sequence_pause(sequence_element_id)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target

Les retours :

N/A

Exemple :

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if (global.Pause)
{
    layer_sequence_pause(_seq);
}

Le code ci-dessus crée une nouvelle séquence et stocke son ID dans une variable locale. Il vérifie ensuite si le jeu est en pause, et si c'est le cas, il met en pause la lecture de la séquence.
