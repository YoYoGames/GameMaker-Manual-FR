couche_séquence_play

Cette fonction permet de lancer la lecture de la séquence donnée. Vous fournissez l'ID de l'élément de séquence tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche et la fonction lira la séquence, que vous pourrez ensuite mettre en pause si nécessaire en utilisant la fonction layer_sequence_pause().

IMPORTANT ! Si votre séquence contient des instances, celles-ci ne doivent pas modifier leurs variables image_xscale / image_yscale / image_angle / x / y car elles seront écrasées lorsque la séquence mettra à jour chaque pas après avoir commencé à être jouée.

Syntaxe :

layer_sequence_play(sequence_element_id)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target

Les retours :

N/A

Exemple :

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++;)
    {
        if (layer_get_element_type(a[i]) == layerelementtype_sequence)
        {
            if (global.Pause)
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

Le code ci-dessus vérifie si le jeu a été mis en pause ou non lorsqu'une touche est enfoncée. Si le jeu est en pause, il passe en boucle tous les éléments de la séquence sur le calque actuel (le calque de l'instance appelante) et met leur lecture en pause. Si le jeu n'est pas en pause, la boucle reprend leur lecture.
