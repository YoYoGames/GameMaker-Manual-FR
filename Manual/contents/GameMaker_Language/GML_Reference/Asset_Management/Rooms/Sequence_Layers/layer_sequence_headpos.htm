couche_séquence_headpos

Cette fonction permet de définir la position de la tête de lecture d'un élément de séquence sur une image spécifique. Vous fournissez l'ID de l'élément de séquence tel que renvoyé par layer_sequence_create() ou par l'une des fonctions de l'élément de couche, ainsi que la nouvelle position à définir. Notez que la position est exprimée en images et que si vous définissez une valeur supérieure au nombre total d'images (ou inférieure à 0), la position finale réelle de la tête de lecture dépendra du type de lecture de la séquence qui a été sélectionné, conformément aux règles suivantes :

  • Pas de bouclage: La tête de lecture se fixe à la fin de la boucle (ou au début si la valeur donnée est négative).
  • Bouclage: La tête de lecture reviendra au début de la séquence plus les images supplémentaires. Par exemple, si vous réglez la tête de lecture d'une séquence de 30 images à la position de l'image 45, la position réelle de la tête de lecture sera de 15. Si vous donnez une valeur négative, la tête de lecture de la séquence se déplacera dans le sens inverse.
  • Bouclage ping-pong: La tête de lecture avance jusqu'à la fin puis revient en arrière du nombre d'images supplémentaires qui dépassent la longueur totale de l'image, en inversant le sens de lecture, par exemple : Si la séquence compte 30 images et que vous réglez la tête de lecture sur 40, la position finale de la tête de lecture sera de 20 et le sens de lecture sera inversé (descendant jusqu'à 0). Notez que vous ne devez jamais donner une valeur négative pour la lecture ping-pong d'une séquence, car cela peut avoir des résultats indéfinis.

Le fait de régler la position de la tête de cette manière n'arrêtera pas la lecture et la séquence continuera simplement à partir de la nouvelle position de la tête de lecture, sauf si elle est mise en pause.

Syntaxe :

layer_sequence_headpos(sequence_element_id, position)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target
position Real The position within the sequence (in frames) to set the playhead position to (can be a decimal value)

Les retours :

N/A

Exemple :

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_headpos(a[i], 0)
    }
}

Le code ci-dessus récupère les ID de tous les éléments affectés à la couche de l'instance qui exécute le code. Le code vérifie ensuite si l'un des éléments renvoyés est une ressource de séquence et, si c'est le cas, la position du lecteur est fixée à 0.
