séquence_couche_get_x

Avec cette fonction, vous fournissez l'ID de l'élément de la séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et elle renverra la position X actuelle dans la salle de jeu pour la séquence.

Syntaxe :

layer_sequence_get_x(sequence_element_id)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target

Les retours :

Real

Exemple :

if layer_sequence_get_x(title_sequence) != room_width / 2
{
    layer_sequence_x(title_sequence, room_width / 2);
}

Le code ci-dessus vérifie la position X de l'ID de l'élément de séquence stocké dans la variable "title_sequence", et s'il n'est pas défini à la moitié de la largeur de la pièce, il est défini à cette valeur.
