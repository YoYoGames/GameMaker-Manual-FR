couche_séquence_get_instance

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et elle renvoie la structure d' instance de séquence. Pour en savoir plus sur le format de la structure d'instance de séquence, consultez cette page.

Syntaxe :

layer_sequence_get_instance(sequence_element_id)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target

Les retours :

Sequence Instance Struct

Exemple :

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

Le code ci-dessus crée un nouvel élément de séquence et récupère ensuite l'ID de son instance de séquence. Celui-ci est ensuite utilisé pour modifier la propriété d'échelle de vitesse de lecture de l'instance de séquence.
