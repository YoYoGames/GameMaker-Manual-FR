layer_sequence_speedscale

Cette fonction permet de modifier la vitesse de lecture d'une séquence donnée. Vous fournissez l'ID de l'élément de séquence tel que renvoyé par layer_sequence_create() ou par l'une des fonctions de l'élément de couche, puis une valeur d'échelle de vitesse. Cette valeur est un multiplicateur, où 1 est la vitesse de lecture par défaut et les valeurs inférieures à 1 ralentiront la lecture et les valeurs supérieures à 1 l'accéléreront. Par exemple, une valeur de 0,5 correspond à la moitié de la vitesse de lecture, tandis qu'une valeur de 2 correspond au double de la vitesse de lecture.

Syntaxe :

layer_sequence_speedscale(sequence_element_id, speedscale)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target
speedscale Real The speed scale to use (a multiplier), default is 1

Les retours :

N/A

Exemple :

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

Le code ci-dessus crée une nouvelle séquence sur le calque de l'instance appelante, puis définit sa valeur d'échelle de vitesse à 0,75 (trois quarts de la vitesse de lecture par défaut).
