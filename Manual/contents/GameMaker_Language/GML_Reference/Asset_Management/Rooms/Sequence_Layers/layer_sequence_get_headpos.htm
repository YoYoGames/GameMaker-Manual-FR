couche_séquence_get_headpos

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renvoie la position actuelle de la tête de lecture (l'image actuelle sur laquelle se trouve la tête de lecture).

Syntaxe :

layer_sequence_get_headpos(sequence_element_id)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target

Les retours :

Exemple :

if layer_sequence_get_headpos(title_sequence) != 0
{
    layer_sequence_headpos(title_sequence, 0);
}

Le code ci-dessus vérifie la position actuelle du playhead de l'ID de l'élément de séquence stocké dans la variable "title_sequence", et s'il n'est pas à 0, il prend cette valeur.
