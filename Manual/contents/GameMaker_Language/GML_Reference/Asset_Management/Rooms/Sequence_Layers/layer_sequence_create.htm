séquence_couche_créer

Avec cette fonction, vous pouvez créer une instance d'un actif de séquence sur la couche donnée. Vous fournissez l'ID du calque qui peut être une chaîne de caractères du nom du calque - tel que défini dans l'éditeur de la pièce - ou l'ID unique du calque - tel que retourné par la fonction layer_get_id()ainsi que la position X et Y dans la pièce pour créer la séquence, et enfin l'ID de la séquence à créer. L'ID de la séquence est la constante de nom que vous avez définie dans l'Asset Browser pour la séquence. La fonction renvoie l'ID unique de l'élément de la séquence, qui peut ensuite être utilisé dans toutes les autres fonctions de calque pour les séquences, ou peut être utilisé pour récupérer l'instance de la séquence struct en utilisant la fonction layer_sequence_get_instance().

Il convient de noter que si la séquence contient des pistes objets, leurs instances seront créées dès la création de l'élément de séquence lui-même, indépendamment de l'emplacement de leurs clés d'actifs sur le Dope Sheet. Le contrôleur de séquence se contente de basculer la visibilité de l'instance pour la cacher et l'afficher en fonction de la position et de la durée de la clé d'actif de la piste, et ne la crée pas et la détruit de manière répétée. Par conséquent, les instances exécutent leurs événements Create lorsque l'élément de séquence est créé et non lorsque leurs clés d'immobilisation commencent.

Syntaxe :

layer_sequence_create(layer_id, x, y, sequence_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the instance layer to target
x Real The x position in the room to create the sequence at
y Real The y position in the room to create the sequence at
sequence_id Séquence Asset The sequence asset to use, as defined in the Asset Browser

Les retours :

Sequence Element ID

Exemple :

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

Le code ci-dessus crée une nouvelle séquence sur le calque "Background" puis la met en pause.
