séquence_de_couche_get_séquence

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renverra la structure de l'objet de la séquence. Cette fonction évite d'avoir à obtenir d'abord l'instance de séquence struct et vous permet d'accéder directement aux données de la séquence access. Pour en savoir plus sur le format de l'objet séquence struct, consultez cette page.

Syntaxe :

layer_sequence_get_sequence(sequence_element_id)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target

Les retours :

Sequence Object Struct

Exemple :

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

Le code ci-dessus crée un nouvel élément de séquence et récupère ensuite ses données de séquence struct. Ces données sont ensuite utilisées pour modifier la vitesse et le type de lecture de la séquence.
