séquence_couche_est_finie

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et elle vérifiera si la lecture de la séquence est terminée ou non, renvoyant true si c'est le cas, et false si ce n'est pas le cas. Notez que cette fonction n'est applicable que lorsque la séquence n'est pas configurée en boucle ou en ping-pong dans le mode de lecture.

Syntaxe :

layer_sequence_is_finished(sequence_element_id)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target

Les retours :

Exemple :

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

Le code ci-dessus vérifie si l'ID de l'élément de séquence stocké dans la variable "title_sequence" est terminé, et si c'est le cas, il relance la lecture et déclenche une alarme.
