couche_séquence_est_pause

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle vérifiera si la séquence est actuellement en pause ou non, renvoyant true si elle est en pause, et false si elle ne l'est pas.

Syntaxe :

layer_sequence_is_paused(sequence_element_id)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target

Les retours :

Exemple :

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

Le code ci-dessus vérifie si l'ID de l'élément de séquence stocké dans la variable "title_sequence" est en pause, et si c'est le cas, il lance la lecture.
