couche_séquence_destroy

Cette fonction permet de détruire (retirer de la pièce) un élément de séquence. Vous fournissez l'ID de l'élément de séquence tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche et la séquence sera détruite.

Syntaxe :

layer_sequence_destroy(sequence_element_id)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target

Les retours :

Exemple :

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_destroy(a[i])
    }
}

Le code ci-dessus récupère les ID de tous les éléments affectés à la couche de l'instance qui exécute le code. Le code vérifie ensuite si l'un des éléments retournés est un actif de la séquence et s'il l'est, il est détruit.
