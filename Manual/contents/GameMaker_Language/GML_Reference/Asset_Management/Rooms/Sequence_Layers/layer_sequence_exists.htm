couche_séquence_existe

Avec cette fonction, vous pouvez vérifier si un élément de séquence existe sur la couche donnée. Vous fournissez l'ID de la couche qui peut être une chaîne de caractères du nom de la couche - tel que défini dans l'éditeur de salle - ou l'ID unique de la couche - tel que retourné par la fonction layer_get_id()ainsi que l'ID de l'élément de séquence - tel que renvoyé par la fonction layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il retournera true si l'élément donné existe ou false sinon.

Syntaxe :

layer_sequence_exists(layer_id, sequence_element_id)

Arguments Type Description
layer_ID Chaîne ou ID de couche The unique ID or name of the layer to check
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target

Les retours :

Boolean

Exemple :

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

Le code ci-dessus vérifie si l'élément de séquence donné existe, et s'il n'existe pas, il crée une nouvelle séquence sur la couche donnée puis la met en pause.
