layer_sequence_get_headdir

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renverra la direction actuelle de la tête de lecture, qui sera l'une des constantes énumérées ci-dessous.

Syntaxe :

layer_sequence_get_headdir(sequence_element_id)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target

Les retours :

Constant Description Valeur
seq_dir_right La séquence joue les images dans un ordre incrémental de gauche à droite. 1
seq_dir_left La séquence joue les images dans un ordre décrémental de droite à gauche. -1

Exemple :

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

Le code ci-dessus vérifie le sens de lecture actuel de l'ID de l'élément de séquence stocké dans la variable "title_sequence", et s'il n'est pas défini sur seq_dir_left, il est défini sur cette valeur.
