angle_séquence_couche

Avec cette fonction, vous pouvez définir l'angle de rotation pour l'élément de séquence donné. Vous fournissez l'ID de l'élément de séquence tel que retourné par layer_sequence_create() ou par l'une des fonctions d'élément de couche avec le nouvel angle à définir et la séquence sera tournée autour de son origine vers la nouvelle position. Les angles dans GameMaker sont calculés avec 0º à droite, et vont dans le sens inverse des aiguilles d'une montre - donc 90º est en haut, 180º est à gauche et 270º est en bas - et l'angle par défaut pour une séquence serait 0º.

Notez que vous ne devez pas modifier les variables image_xscale \ image_yscale \ image_angle \ x \ y pour toute instance d'une séquence qui utilise cette fonction, car elles seront écrasées dès que la séquence se mettra à jour après l'appel de la fonction.

Syntaxe :

layer_sequence_angle(sequence_element_id, angle)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target
angle Real The new angle to rotate the sequence element to

Les retours :

N/A

Exemple :

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

Le code ci-dessus vérifie la valeur contenue dans la variable current_angle, et si elle est inférieure à 90, il l'ajoute puis utilise la valeur pour définir l'angle de l'élément de séquence référencé dans la variable my_seq.
