layer_sequence_y

Avec cette fonction, vous pouvez définir la position le long de l'axe Y (vertical) de la pièce pour l'élément de séquence donné. Vous fournissez l'ID de l'élément de séquence tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche avec la position Y à définir et la séquence sera déplacée vers la nouvelle position.

Syntaxe :

layer_sequence_y(sequence_element_id, pos_y)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target
pos_y The Y position to move the sequence element to

Les retours :

Exemple :

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

Le code ci-dessus vérifie si l'élément de séquence référencé dans la variable "my_seq" existe, et si c'est le cas, il définit la position x/y à celle de l'instance qui exécute le code.
