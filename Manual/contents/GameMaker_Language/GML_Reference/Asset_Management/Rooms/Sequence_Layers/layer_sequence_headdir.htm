répertoire de tête de la séquence des couches

Cette fonction permet de définir la direction de la tête de lecture d'une séquence donnée. Vous fournissez l'ID de l'élément de séquence tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche, puis vous donnez la direction de la tête de lecture qui doit être l'une des constantes suivantes :

Constant Description Valeur
seq_dir_right La séquence lira les images dans un ordre incrémental de gauche à droite. 1
seq_dir_left La séquence lira les images dans un ordre décrémental de droite à gauche. -1

Syntaxe :

layer_sequence_headdir(sequence_element_id, direction)

Arguments Type Description
sequence_element_id ID de l'élément de séquence The unique ID value of the sequence element to target
direction Séquence Direction Constante The playhead direction, a constant, listed above

Les retours :

N/A

Exemple :

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

Le code ci-dessus crée une nouvelle séquence et stocke son ID dans une variable locale. Cet ID est ensuite utilisé pour définir la direction de la tête de lecture afin de décrémenter les images (lecture de droite à gauche).
