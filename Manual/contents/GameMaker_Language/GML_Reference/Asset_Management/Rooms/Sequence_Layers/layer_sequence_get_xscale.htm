layer_sequence_get_xscale

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et elle renverra l'échelle actuelle le long de l'axe X de l'élément de séquence dans la salle de jeu.

Syntaxe :

layer_sequence_get_xscale(sequence_element_id)

ArgumentsType Description
sequence_element_id The unique ID value of the sequence element to target

Les retours :

Exemple :

var _xs = layer_sequence_get_xscale(title_sequence)
if (_xs < 1)
{
    _xs += 0.01;
    layer_sequence_xscale(title_sequence, _xs);
}

Le code ci-dessus récupère l'échelle actuelle le long de l'axe X de l'élément de la séquence avec l'ID stocké dans la variable "title_sequence", et si elle est inférieure à 1, alors 0,01 est ajouté à l'échelle X actuelle.
