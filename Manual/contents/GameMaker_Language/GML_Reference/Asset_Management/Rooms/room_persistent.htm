chambre_persistante

Cette variable peut être utilisée pour obtenir et définir l'indicateur de persistance pour la room actuelle. S'il est défini à true, le room est considéré comme persistant, ce qui signifie que chaque fois que vous quittez le room et que vous revenez, l'état des instances de ce room aura été maintenu. Cependant, si elle est marquée comme false, chaque fois que vous revenez dans la room, elle sera réinitialisée à son état initial. Vous devez noter qu'une room persistante utilise beaucoup plus de mémoire qu'une room normale et il n'est pas recommandé d'en avoir trop dans votre jeu.

Syntaxe :

room_persistent;

Les retours :

Exemple :

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

Le code ci-dessus vérifie la variable "lives" et si elle est inférieure à 1, il définit la persistance de la pièce à false et change ensuite de pièce.
