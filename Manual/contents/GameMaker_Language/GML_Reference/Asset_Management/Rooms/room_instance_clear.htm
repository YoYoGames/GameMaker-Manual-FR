room_instance_clear

Cette fonction permet de vider une room de toutes ses instances (aucun événement de destruction ne sera appelé pour les instances qui sont supprimées). Notez que l'appel de cette fonction sur un asset de room créé dans le navigateur d'assets videra définitivement la room de ses instances, et même en appelant game_restart(), la room ne retrouvera pas son état d'origine (seule la fin du jeu et sa réouverture permettront de retrouver la room dans son état d'origine).

Syntaxe :

room_instance_clear(index);

ArgumentsType Description
index The index of the room to clear all instances in.

Les retours :

Exemple :

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_clear(global.rm);

Le code ci-dessus ajoutera une nouvelle room au jeu, puis y copiera le contenu de la room indexée comme "rm_Base". Il effacera ensuite toutes les instances de cette nouvelle room.
