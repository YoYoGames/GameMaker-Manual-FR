chambre_précédente

Avec cette fonction, vous pouvez récupérer l'index de la pièce avant la pièce entrée dans la fonction. Par exemple, vous pouvez utiliser la variable room pour obtenir l'index de la salle actuelle, puis utiliser cette fonction pour trouver la salle qui la précède, telle qu'elle est répertoriée dans le Room Manager. S'il n'y a pas de pièce avant celle que vous avez entrée, -1 est renvoyé. Notez que cette fonction ne reconnaîtra pas ou ne prendra pas en compte les salles qui ont été ajoutées dynamiquement à l'aide de room_add() ou room_duplicate().

Syntaxe :

room_previous(numb);

ArgumentsType Description
numb The index of the room to get the one before from.

Les retours :

Room Asset

Exemple :

if room_previous(room) != -1
{
    room_goto_previous();
}

Le code ci-dessus vérifiera si la pièce précédente existe et si c'est le cas, il s'y rendra.
