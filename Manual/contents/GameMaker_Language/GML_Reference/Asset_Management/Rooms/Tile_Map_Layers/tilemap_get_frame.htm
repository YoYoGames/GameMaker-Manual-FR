tilemap_get_frame

Comme les tuiles peuvent être animées, il peut parfois être utile de savoir quelle image est actuellement dessinée et de réagir en conséquence. Cette fonction vous permet donc de récupérer l'indice de l'image actuelle pour une carte de tuiles donnée. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renvoie l'indice de trame.

Syntaxe :

tilemap_get_frame(tilemap_element_id)

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to get the frame index of

Les retours :

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

Exemple :

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4
{
    global.activate = true;
}
else
{
    global.activate = false;
}

Le code ci-dessus vérifie le cadre d'animation actuel pour la carte de tuiles donnée sur la couche donnée, et définit une variable globale en fonction de la valeur de retour.
