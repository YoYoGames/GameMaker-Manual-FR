tile_get_index

Cette fonction peut être utilisée pour obtenir l'indice de tuile (la position de la tuile dans l'image du jeu de tuiles) à partir d'un jeu de données de tuiles. Vous spécifiez les données de tuiles, qui peuvent être récupérées à l'aide de la fonction tilemap_get()et la fonction renvoie une valeur entière pour l'index.

Lorsque vous utilisez cette fonction, assurez-vous que les données de tuiles données sont valides, sinon la valeur de l'index retourné ne sera pas valide non plus, ce qui peut entraîner un comportement inattendu.

Syntaxe :

tile_get_index(tiledata)

Arguments Type Description
tiledata Données sur les tuiles the tile-data to check

Les retours :

Real

Exemple :

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

Le code ci-dessus récupère l'ID de la carte des tuiles de la couche donnée, puis l'utilise pour obtenir les données des tuiles de la cellule située sous la position de la souris. Ces données sont ensuite utilisées pour définir l'index de la tuile à un nombre aléatoire.
