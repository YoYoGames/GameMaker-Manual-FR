layer_tilemap_exists

Vous pouvez utiliser cette fonction pour vérifier si un élément de carte à carreaux existe sur une couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et l'ID de l'élément de carte de carreaux (que vous obtenez lorsque vous créez un élément de carte de carreaux à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()) et la fonction renverra soit true si l'élément existe, soit false s'il n'existe pas.

REMARQUE: Cette fonction fonctionne dans le cadre de la pièce cible actuelle - par défaut, la pièce dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room().

Syntaxe :

layer_tilemap_exists(layer_id, tilemap_element_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to check

Les retours :

Boolean

Exemple :

var lay_id = layer_get_id("tilemap_trees");
if (layer_tilemap_exists(lay_id, global.Treestilemap))
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

Le code ci-dessus vérifie la couche "tilemap_trees" pour voir si l'élément de carte de tuiles donné existe et si c'est le cas, alors il est détruit (mais pas la couche).
