tilemap_get_tile_width

Cette fonction vous permet de récupérer la largeur (en pixels) de chaque cellule de l'élément de carte de carreaux. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renverra la largeur de la cellule du jeu de tuiles.

Syntaxe :

tilemap_get_tile_width(tilemap_element_id);

ArgumentsType Description
tilemap_element_id The unique ID value of the tile map element to get the tile cell width of

Les retours :

Exemple :

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

Le code ci-dessus utilise l'ID de la carte des tuiles récupérée pour obtenir la largeur et la hauteur des cellules des tuiles et les utiliser pour définir deux variables globales.
