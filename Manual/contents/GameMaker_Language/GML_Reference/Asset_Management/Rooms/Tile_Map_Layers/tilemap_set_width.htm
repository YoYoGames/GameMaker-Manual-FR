tilemap_set_width

Cette fonction peut être utilisée pour redimensionner un élément de carte de tuiles. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la nouvelle largeur de l'élément en cellules de tuiles.

Syntaxe :

tilemap_set_width(tilemap_element_id, width)

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to set the width of
width Real The width value (in "cells")

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_width(map_id) != room_width div 16)
{
    tilemap_set_width(map_id, room_width div 16);
}

Le code ci-dessus vérifie la largeur d'une carte de tuiles spécifique et si elle n'est pas de la bonne taille, la largeur est définie.
