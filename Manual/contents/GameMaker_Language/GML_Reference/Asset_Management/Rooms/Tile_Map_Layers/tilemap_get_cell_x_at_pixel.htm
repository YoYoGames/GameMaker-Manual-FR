tilemap_get_cell_x_at_pixel

Cette fonction vous permet de récupérer la position de l'axe des x d'une cellule individuelle de carte en indiquant la position relative de l'axe des x dans la pièce. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), ainsi que la position x et y dans la pièce, et la fonction renvoie la position x de la cellule dans la carte des tuiles pour ce point. Notez que si la valeur se trouve en dehors de la zone de la carte des tuiles et qu'aucune cellule n'est disponible, la fonction renverra -1.

Syntaxe :

tilemap_get_cell_x_at_pixel(tilemap_element_id, x, y);

ArgumentsType Description
tilemap_element_id The unique ID value of the tile map element to get the cell x position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

Les retours :

(x-axis cell position or -1 if there is an error)

Exemple :

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

Le code ci-dessus utilise l'ID de la carte des tuiles récupérée pour obtenir la position de la cellule à partir d'un ensemble de coordonnées de la pièce, puis stocke les données de toute tuile trouvée dans une variable d'instance.
