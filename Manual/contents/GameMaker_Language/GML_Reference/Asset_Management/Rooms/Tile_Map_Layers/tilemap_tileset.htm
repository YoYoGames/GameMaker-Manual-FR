tilemap_tileset

Cette fonction vous permet de modifier l'actif du jeu de tuiles attribué à un élément de carte de tuiles donné sur une couche. Vous donnez l'ID de l'élément de mosaïque (que vous obtenez lorsque vous créez un élément de mosaïque à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), puis fournissez un index de jeu de tuiles et le jeu de tuiles recevra le nouveau sprite.

Syntaxe :

tilemap_tileset(tilemap_element_id, tileset_index)

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to change
tileset_index Ensemble de tuiles Asset The new tile set index to use

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_tileset(tile_id) != ts_Nighttime)
{
    tilemap_tileset(tile_id, ts_Nighttime);
}

Le code ci-dessus vérifie le jeu de tuiles actuel attribué à la carte des tuiles sur la couche "Tiles_trees" et si ce n'est pas "ts_Nighttime", alors ce jeu de tuiles est attribué à la carte des tuiles.
