tilemap_set_height

Cette fonction peut être utilisée pour redimensionner un élément de carte de tuiles. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la nouvelle hauteur de l'élément en cellules de tuiles.

Syntaxe :

tilemap_set_height(tilemap_element_id, height)

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to set the height of
height Real The height value (in "cells")

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_height(map_id) != room_height div 16)
{
    tilemap_set_height(map_id, room_height div 16);
}

Le code ci-dessus vérifie la largeur d'une carte de tuiles spécifique et si elle n'est pas de la bonne taille, la largeur est définie.
