tilemap_get_mask

Cette fonction peut être utilisée pour récupérer la valeur du masque de bits pour la carte de tuiles donnée. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renvoie la valeur actuelle du masque ou -1 en cas d'erreur ou 0 si aucun masque n'est spécifié. Pour plus d'informations sur les masques de bits des cartes de tuiles, consultez la fonction tilemap_set_mask().

Syntaxe :

tilemap_get_mask(tilemap_element_id);

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to get the mask of

Les retours :

Real (0 for no mask, -1 for an error)

Exemple :

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_mask(_map_id, _new_mask);
}

Le code ci-dessus récupère l'ID de la carte de tuiles de la couche donnée, puis vérifie la valeur du masque qui lui est associée. Si elle n'est pas identique à la valeur définie dans la variable "new_mask", alors elle est définie à cette valeur.
