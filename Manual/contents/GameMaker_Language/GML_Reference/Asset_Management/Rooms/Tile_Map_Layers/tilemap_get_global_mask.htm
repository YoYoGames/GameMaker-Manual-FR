tilemap_get_global_mask

Cette fonction peut être utilisée pour récupérer la valeur du masque de bits pour toutes les cartes de tuiles, renvoyant la valeur actuelle du masque ou -1 en cas d'erreur ou 0 si aucun masque n'est spécifié. Pour plus d'informations sur les masques de bits des cartes de tuiles globales, voir la fonction tilemap_set_global_mask().

Syntaxe :

tilemap_get_global_mask();

Les retours :

(0 for no mask, -1 for an error, mask value otherwise)

Exemple :

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_global_mask(new_mask);
}

Le code ci-dessus récupère la valeur du masque global associé à toutes les cartes de tuiles. S'il n'est pas identique à la valeur définie dans la variable "new_mask", alors il est défini à cette valeur.
