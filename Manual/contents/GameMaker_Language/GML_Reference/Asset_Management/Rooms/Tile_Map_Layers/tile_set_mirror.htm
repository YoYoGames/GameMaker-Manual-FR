tuile_set_mirror

Cette fonction peut être utilisée pour définir un ensemble donné de données de tuiles pour refléter la tuile ou non. Vous donnez les données de tuiles, qui peuvent être récupérées en utilisant la fonction tilemap_get()puis définissez l'argument miroir à true si vous voulez que la tuile soit reflétée, ou à false si vous voulez que la tuile soit dans son état par défaut, non reflété. La fonction renverra un ensemble de données de tuiles modifié qui peut ensuite être appliqué à l'aide de la fonction tilemap_set() fonction.

Syntaxe :

tile_set_mirror(tiledata, mirror)

ArgumentsType Description
tiledata the tile-data to set
mirror Set to true pour le miroir et false pour le laisser tel quel

Les retours :

Tile Data (Real)

Exemple :

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

Le code ci-dessus récupère l'ID de la carte des tuiles à partir de la couche donnée, puis obtient la position x et y de la cellule de la tuile sous la souris. Cette position est ensuite utilisée pour obtenir les données de la tuile, qui sont reflétées et ensuite utilisées pour définir à nouveau la tuile.
