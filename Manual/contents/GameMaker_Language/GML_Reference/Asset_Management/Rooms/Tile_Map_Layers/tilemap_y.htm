tilemap_y

Cette fonction contrôle la position sur l'axe des y de la pièce de l'élément de carte à carreaux de l'actif sur la couche. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), puis définissez la valeur y à utiliser (en fonction des coordonnées de la pièce).

Syntaxe :

tilemap_y(tilemap_element_id, y);

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to change
y Real The y position for the tile map

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

Le code ci-dessus récupère la valeur d'identification de l'actif de la carte de carreaux attribué au calque "Asset_sky", puis définit sa position y à une valeur aléatoire comprise entre 0 et la hauteur de la pièce.
