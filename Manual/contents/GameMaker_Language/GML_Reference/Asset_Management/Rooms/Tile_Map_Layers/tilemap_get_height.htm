tilemap_get_height

Cette fonction vous permet de récupérer la hauteur (en cellules) de l'élément de carte de tuiles. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la commande layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renvoie la hauteur (nombre de cellules de largeur).

Syntaxe :

tilemap_get_height(tilemap_element_id);

ArgumentsType Description
tilemap_element_id The unique ID value of the tile map element to get the height of

Les retours :

Exemple :

var lay_id = layer_get_id("espadrilles");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

Le code ci-dessus utilise l'ID de la carte des tuiles récupérée pour obtenir la largeur et la hauteur de la carte des tuiles, puis utilise ces valeurs pour définir une variable d'instance.
