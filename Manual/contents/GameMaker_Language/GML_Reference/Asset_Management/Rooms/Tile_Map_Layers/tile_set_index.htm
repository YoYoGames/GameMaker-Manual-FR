tuile_set_index

Cette fonction peut être utilisée pour définir l'index des tuiles (la position de la tuile dans l'image de l'ensemble des tuiles) pour un ensemble de données de tuiles. Vous donnez les données de tuiles, qui peuvent être récupérées à l'aide de la fonction tilemap_get()et ensuite la valeur entière qui représente la position de la tuile (son index) dans l'ensemble des tuiles. La fonction renvoie un ensemble de données de tuiles modifié qui peut ensuite être appliqué à l'aide de la fonction tilemap_set() fonction.

Syntaxe :

tile_set_index(tiledata, index)

Arguments Type Description
tiledata Données sur les tuiles the tile-data to set
index Real The integer index value of the tile to use

Les retours :

Tile Data

Exemple :

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

Le code ci-dessus récupère l'ID de la carte des tuiles de la couche donnée, puis l'utilise pour obtenir les données des tuiles de la cellule située sous la position de la souris. Ces données sont ensuite utilisées pour définir l'index de la tuile à un nombre aléatoire.
