layer_tilemap_get_id

Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de l'élément de carte de carreaux sur une couche. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères - cela aura un impact sur les performances) et la fonction renverra la valeur d'identification associée à l'élément de carte de tuiles sur la couche. Notez que cette fonction est spécifiquement conçue pour être utilisée avec des cartes de carreaux qui ont été ajoutées à l'adresse IDE, car si vous ajoutez une carte de carreaux à une couche par le biais d'un code utilisant la fonction layer_tilemap_create()En effet, si vous ajoutez une carte de carreaux à une couche par le biais d'un code utilisant la fonction , celle-ci renverra l'ID unique de l'élément de carte de carreaux ajouté. Si l'ID de la carte de tuiles est incorrect ou si la carte de tuiles n'existe pas, la fonction renvoie -1.

Syntaxe :

layer_tilemap_get_id(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de la couche The unique ID value of the layer to target (or the layer name as a string)

Les retours :

Tile Map Element ID or -1

Exemple :

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

Le code ci-dessus obtiendra l'ID de la couche nommée "tilemap_trees", puis l'utilisera pour obtenir l'ID de l'élément de carte de carreaux sur cette couche. Cet ID est ensuite utilisé pour supprimer la carte de tuiles de la couche.
