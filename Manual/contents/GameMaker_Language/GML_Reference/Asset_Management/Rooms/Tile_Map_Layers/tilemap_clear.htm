tilemap_clear

En utilisant cette fonction, vous pouvez effacer/régler toutes les tuiles d'une carte de tuiles donnée. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles en utilisant la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), puis vous fournissez les données des tuiles avec lesquelles vous souhaitez effacer la couche. Une valeur par défaut de 0 effacera toutes les tuiles de la couche (essentiellement en rendant toutes les tuiles "vides"), tandis que vous pouvez utiliser les fonctions dédiées tile_* pour créer vos propres données de tuiles pour effacer la carte de tuiles avec.

Syntaxe :

tilemap_clear(tilemap_element_id, tiledata)

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to change
tiledata Données sur les tuiles (réelles) The tile data to use to clear the layer

Les retours :

N/A

Exemple :

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

Le code ci-dessus récupère la valeur ID d'une carte de tuiles créée dans l'éditeur de salle, puis la nettoie en utilisant des tuiles "vides".
