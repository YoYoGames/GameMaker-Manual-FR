layer_tilemap_destroy

Cette fonction détruira l'élément de carte de tuiles donné. Vous fournissez l'ID de l'élément (que vous obtenez lorsque vous créez l'élément à l'aide de la commande layer_tilemap_create() ou lorsque vous utilisez l'ID de la couche avec layer_get_tilemap_id()) et cette fonction le supprimera. Notez que cela ne supprime pas le calque, mais seulement la carte des tuiles, et si la carte des tuiles a été ajoutée dans l'éditeur de la pièce, la prochaine fois que vous quitterez la pièce et y reviendrez, la carte des tuiles sera recréée. Cependant, si la pièce est persistante, le plan des tuiles sera supprimé, à moins que la persistance de la pièce ne soit à nouveau désactivée.

Syntaxe :

layer_tilemap_destroy(tilemap_element_id)

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map to be destroyed

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

Le code ci-dessus vérifie la couche "Tiles_trees" pour voir si l'élément de carte de tuiles donné existe et si c'est le cas, alors il est détruit (mais pas la couche).
