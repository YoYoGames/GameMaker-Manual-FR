tilemap_get

Cette fonction permet de récupérer les données des tuiles d'une cellule de l'élément de carte des tuiles. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), ainsi que la position x et y de la cellule, et la fonction renvoie le "blob" de données de tuiles. Ces données sont essentiellement un masque binaire qui contient l'index de la tuile, les booléens d'inversion, de rotation et de miroir, ainsi que toutes les données de masque qui ont été appliquées (cf. tilemap_set_mask() pour plus de détails), et la valeur des données résultantes peut ensuite être utilisée dans les fonctions tile_* pour modifier les propriétés d'une tuile. Notez que nous utilisons les positions x/y des cellules en fonction de la taille de la grille de la carte des tuiles et non comme une position dans la pièce, donc "cell_x" est une valeur de 0 à la largeur de la carte des tuiles, et "cell_y" est une valeur de 0 à la hauteur de la carte des tuiles. Si vous avez besoin d'obtenir les données d'une tuile à une position spécifique dans la pièce, vous devez utiliser la fonction tilemap_get_at_pixel().

IMPORTANT ! Si les tuiles de la carte des tuiles sont restées inchangées (c'est-à-dire qu'elles n'ont pas été tournées ou retournées, etc.), la valeur de retour des données du jeu de tuiles sera exactement égale à l'index de la tuile dans le jeu de tuiles. Vous pouvez donc créer des "cartes de collision" de tuiles en utilisant une tuile à l'indice 1 dans le jeu de tuiles - par exemple - puis utiliser cette fonction pour vérifier 1 ou 0 (une tuile vide) pour calculer les collisions.

Syntaxe :

tilemap_get(tilemap_element_id, cell_x, cell_y);

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to get the tile-data of
x_cell Real The cell along the x-axis to get the tile data from
y_cell Real The cell along the y-axis to get the tile data from

Les retours :

Tile Data (-1 if there is an error)

Exemple :

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Le code ci-dessus récupère l'ID de la carte des tuiles sur la couche "Tuiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la cellule (0, 0). Les données de cette tuile sont ensuite retournées avant d'être utilisées pour définir à nouveau la tuile sur la carte des tuiles.
