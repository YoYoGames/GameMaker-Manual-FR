tilemap_get_tileset

Cette fonction vous permet de récupérer la valeur d'index de l'actif Tile Set attribué à un élément de carte de tuiles donné sur une couche. Vous donnez l'ID de l'élément Tile Map (que vous obtenez lorsque vous créez un élément Tile Map à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renvoie l'index de l'actif du Tile Set.

Syntaxe :

tilemap_get_tileset(tilemap_element_id);

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the Tile Map element to get the tile set from

Les retours :

Tile Set Asset

Exemple :

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(map_id) != ts_Nighttime
{
    tilemap_tileset(map_id, ts_Nighttime);
}

Le code ci-dessus vérifie le jeu de tuiles actuel attribué à la couche "Tiles_trees" et si ce n'est pas "ts_Nighttime", alors ce jeu de tuiles est attribué à la carte des tuiles.
