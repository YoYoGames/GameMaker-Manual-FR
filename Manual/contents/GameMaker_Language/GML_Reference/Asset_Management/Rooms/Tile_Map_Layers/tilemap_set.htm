jeu de tuiles

Cette fonction peut être utilisée pour définir une nouvelle tuile dans n'importe quelle cellule (carré de la grille) de l'élément de carte à tuiles de la couche. Vous donnez l'ID de l'élément de carte de carreaux (que vous obtenez lorsque vous créez un élément de carte de carreaux à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), puis les données de tuile à définir ainsi que la position dans la carte de tuiles. Vous pouvez récupérer les données des tuiles à l'aide de la fonction tilemap_get() puis utiliser les fonctions tile_ pour les modifier avant de définir la cellule à l'aide de cette fonction. La fonction renvoie true si la tuile a été définie avec succès et false s'il y a eu un problème et qu'elle n'a pas été définie.

Syntaxe :

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

Arguments Type Description
tilemap_element_id ID de l'élément Tile Map The unique ID value of the tile map element to change
tiledata Données sur les tuiles (réelles) The tile data to set
xcell Real The cell (grid) position to set along the x-axis
ycell Real The cell (grid) position to set along the y-axis

Les retours :

Boolean

Exemple :

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Le code ci-dessus récupère l'ID de la carte des tuiles sur la couche "Tuiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la cellule (0, 0). Les données de cette tuile sont ensuite retournées avant d'être utilisées pour définir à nouveau la tuile sur la carte des tuiles.
