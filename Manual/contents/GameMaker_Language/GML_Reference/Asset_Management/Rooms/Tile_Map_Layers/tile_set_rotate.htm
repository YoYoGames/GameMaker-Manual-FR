tile_set_rotate

Cette fonction peut être utilisée pour définir un ensemble donné de données de tuiles pour faire pivoter la tuile de 90 degrés ou non. Vous donnez les données de tuiles, qui peuvent être récupérées en utilisant la fonction tilemap_get()puis définissez l'argument rotate à true si vous souhaitez que la tuile subisse une rotation, ou à false si vous souhaitez que la tuile reste dans son état par défaut, sans rotation. La fonction renverra un ensemble de données de tuiles modifié qui pourra ensuite être appliqué à l'aide de la fonction tilemap_set() fonction.

Syntaxe :

tile_set_rotate(tiledata, rotate)

Arguments Type Description
tiledata Données sur les tuiles The tile-data to set
rotate Booléen Set to true pour le faire pivoter et false pour le laisser tel quel

Les retours :

Real

Exemple :

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

Le code ci-dessus récupère l'ID de la carte des tuiles de la couche donnée, puis obtient la position x et y de la cellule de la tuile sous la souris. Cette position est ensuite utilisée pour obtenir les données de la tuile qui sont tournées et ensuite utilisées pour définir à nouveau la tuile.
