layer_tilemap_create

Cette fonction vous permet d'affecter une ressource de jeu de tuiles à une couche qui sera utilisée comme carte de tuiles dans votre projet. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous la forme d'une chaîne de caractères - cela aura un impact sur les performances), puis une position initiale (x, y) pour ajouter la carte de carreaux à la pièce, la ressource de jeu de carreaux à utiliser, puis la largeur et la hauteur de la carte de carreaux en cellules (par exemple, une largeur de 20 et une hauteur de 10 créera une carte de carreaux de 200 cellules, soit 20 cellules de large et 10 cellules de haut), la taille des cellules étant définie par le jeu de carreaux choisi. Il convient de noter que vous ne pouvez pas placer de tuiles à des positions négatives dans la carte des tuiles, de sorte que toutes les tuiles doivent être placées dans les espaces de cellule 0 à largeur - 1, 0 à hauteur - 1.

Syntaxe :

layer_tilemap_create(layer_id, x, y, tileset, width, height)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
x Real The x position of the tile map in the room
y Real The y position of the tile map in the room
tileset Ensemble de tuiles Asset The Tile Set asset to be used
width Real The width tile map (in cells)
height Real The height of the tile map (in cells)

Les retours :

Tile Map Element ID

Exemple :

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

Le code ci-dessus crée une nouvelle couche et y ajoute un nouvel élément de carte de tuiles, en définissant la position de la carte de tuiles à (0,0) ainsi que l'ensemble de tuiles à utiliser et la largeur et la hauteur de la carte de tuiles.
