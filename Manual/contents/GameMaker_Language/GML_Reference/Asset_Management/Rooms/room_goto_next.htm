chambre_goto_suivante

Grâce à cette fonction, vous pouvez faire en sorte que votre jeu passe à la salle suivante, telle qu'elle est répertoriée dans le gestionnaire de salles au moment où le jeu a été compilé. Si cette salle n'existe pas, une erreur sera lancée et le jeu sera forcé de se fermer. Notez que la salle ne changera pas avant la fin de l'événement dans lequel la fonction a été appelée, donc tout code après l'appel de cette fonction sera toujours exécuté s'il se trouve dans le même événement.

Syntaxe :

room_goto_next();

Les retours :

Exemple :

if room_exists(room_next(room))
{
    room_goto_next();
}

Le code ci-dessus vérifiera s'il y a une autre pièce après la pièce actuelle et si c'est le cas, il ira dans cette pièce.
