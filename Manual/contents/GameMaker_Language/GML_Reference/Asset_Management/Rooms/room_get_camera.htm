chambre_get_caméra

Cette fonction permet d'obtenir l'index unique de la caméra affectée à une vue spécifique dans une pièce autre que la pièce actuelle. Vous indiquez la salle à utiliser, le port de vue à utiliser (de 0 à 7) et la fonction renvoie un index de caméra.

Syntaxe :

room_get_camera(rm, vind);

Arguments Type Description
rm Actif de la salle The index of the room to get the view camera of
vind Real The index of the view port to get the camera of

Les retours :

Camera ID

Exemple :

var cam = room_get_camera(rm_Game, 0);
if (cam != global.MainCam)
{
    room_set_camera(rm_game, 0, global.main_cam);
}

Le code ci-dessus attribue une caméra dans une pièce nouvellement créée au port de vue [0].
