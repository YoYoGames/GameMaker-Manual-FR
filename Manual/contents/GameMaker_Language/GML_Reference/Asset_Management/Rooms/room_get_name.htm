chambre_get_name

Cette fonction peut être utilisée pour retourner le nom de la pièce spécifiée sous forme de chaîne de caractères. Veuillez noter qu'il s'agit uniquement d'une chaîne de caractères et qu'elle ne peut pas être utilisée pour référencer la pièce directement - pour cela, vous aurez besoin de l'index de la pièce. Vous pouvez cependant utiliser cette chaîne pour obtenir l'index de la pièce en utilisant la chaîne retournée avec la fonction asset_get_index().

Syntaxe :

room_get_name(index);

ArgumentsType Description
index The index of the room to check the name of.

Les retours :

Exemple :

var roomname = room_get_name(room);

draw_text(32, 32, roomname);

Le code ci-dessus obtiendra le nom de la pièce actuelle et l'affichera à l'écran.
