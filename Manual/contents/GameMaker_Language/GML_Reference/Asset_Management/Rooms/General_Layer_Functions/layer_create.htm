couche_créer

Cette fonction est utilisée pour créer une nouvelle couche dans la pièce actuelle. Vous fournissez la profondeur que la nouvelle couche doit occuper et la fonction renvoie la valeur unique de l'ID de la couche qui peut ensuite être utilisée pour access la couche dans d'autres fonctions. Vous pouvez également fournir un argument facultatif "nom", qui vous permettra de donner à la couche un nom spécifique (sous forme de chaîne), et au lieu d'utiliser l'ID de la couche pour access la couche, vous pouvez utiliser cette chaîne de nom à la place, bien qu'il faille noter que l'utilisation des chaînes de noms aura un impact plus important sur les performances que l'utilisation de la seule valeur ID. Si vous ne spécifiez pas le nom de la couche, la couche créée se verra attribuer un nom au format "_layer_XXX", où " XXX " est une valeur hexagonale utilisée pour donner un nom unique à la couche.

Lorsque vous donnez une valeur de profondeur, celle-ci sera d'autant plus élevée que le calque est "éloigné" de la caméra et d'autant plus faible que le calque est "proche" de la caméra. Ainsi, si vous avez trois calques avec des profondeurs de -100, 0, 100, les calques seront dessinés dans l'ordre 100, 0, -100, de sorte que le calque "supérieur" (c'est-à-dire le plus proche de la vue de la caméra et donc dessiné par-dessus tout le reste) sera le calque avec la profondeur -100.

L'image suivante montre un schéma du fonctionnement de la profondeur pour les couches :

Il convient de noter que si vous pouvez affecter plus d'un calque à une même profondeur, vous ne pouvez pas garantir que les calques de la même profondeur seront dessinés dans un ordre donné et il est donc préférable d'affecter un seul calque à une profondeur donnée. Notez également que les profondeurs de couche minimale et maximale sont de -16000 à 16000, et que tout ce qui se trouve en dehors de ces profondeurs ne sera pas rendu. Si vous avez besoin d'une profondeur en dehors de cette plage, vous devrez utiliser la fonction layer_force_draw_depth().

Syntaxe :

layer_create(depth, [name])

Arguments Type Description
depth Real The depth that the layer has to be created at.
name Chaîne de caractères OPTIONAL Le nom à donner à la couche (une chaîne de caractères).

Les retours :

Layer ID

Exemple :

global.Bullet_Layer = layer_create(-100);

Le code ci-dessus va créer une nouvelle couche avec une valeur de profondeur de -100 et affecter sa valeur d'identification à une variable globale.
