layer_get_id

Cette fonction peut être utilisée pour obtenir la valeur d'identification unique d'une couche donnée. Dans le site IDE, toutes les couches ont un nom et un type, et pour pouvoir les éditer ou les modifier par le code, vous devez donner la valeur de l'ID de la couche. Cette fonction est utilisée pour récupérer cet ID en utilisant le nom (une chaîne de caractères) de la couche (comme écrit dans le IDE). Si vous créez une nouvelle couche par le biais du code en utilisant la fonction layer_create() alors cette fonction renverra la valeur unique de l'ID à la place (les couches créées dynamiquement n'ont pas de nom). Notez que si vous donnez le nom d'une couche qui n'existe pas dans la pièce actuelle, vous obtiendrez une erreur et le projet se plantera.

Syntaxe :

layer_get_id(layer_name)

ArgumentsType Description
layer_name The name of the layer (a string)

Les retours :

Layer ID or -1 (if the layer specified doesn't exist)

Exemple :

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Le code ci-dessus va d'abord obtenir l'index de l'instance la plus proche de la position x/y donnée et le stocker dans une variable locale. Il obtient ensuite l'ID de couche d'instance unique pour la couche nommée "Instances Front", et déplace l'instance trouvée sur cette couche.
