couche_destruction

Cette fonction va détruire la couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom du calque (sous forme de chaîne - cela aura un impact sur les performances) et cela le supprimera de la pièce actuelle. Si le calque a été conçu dans l'éditeur de pièce, la prochaine fois que vous quitterez la pièce et que vous y reviendrez, le calque sera recréé avec son contenu original. Cependant, si la pièce est persistante, le calque sera supprimé à moins que la persistance de la pièce ne soit désactivée. Lorsque vous détruisez un calque de cette manière, tout son contenu sera également supprimé, de sorte que tout ID de référence pour les arrière-plans ou les cartes de tuiles, etc... ne sera plus valide et toute instance assignée au calque sera détruite (en exécutant leur événement Destroy en même temps, s'ils en ont un, ainsi que l'événement Clean Up).

Syntaxe :

layer_destroy(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to destroy (or the layer name as a string)

Les retours :

N/A

Exemple :

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Le code ci-dessus vérifie si des instances de l'objet "obj_Bullet_Parent" existent, et si ce n'est pas le cas, il détruit le calque avec l'ID stocké dans la variable globale.
