couche_instance_get_instance

Cette fonction peut être utilisée pour obtenir l'ID d'instance unique de l'élément d'instance donné. Vous donnez l'ID de l'élément d' instance (voir l'exemple de code ci-dessous pour savoir comment l'obtenir), et la fonction renvoie une valeur réelle qui représente l'ID d'instance unique de l'élément. Si l'élément n'est pas une instance, la fonction renvoie -1.

Syntaxe :

layer_instance_get_instance(element_id)

Arguments Type Description
element_id ID de l'élément d'instance The unique ID value of the instance element to get the ID of

Les retours :

Instance ID

Exemple :

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
     if (layer_get_element_type(_elements[i]) == layerelementtype_instance)
     {
         var _element = _elements[i];
         var _inst = layer_instance_get_instance(_element);
         _inst.x = _inst.x + 10;
     }
}

Le code ci-dessus vérifie tous les éléments d'instance sur un calque, puis récupère leur valeur d'identification unique et l'utilise pour les déplacer de 10px vers la droite.
