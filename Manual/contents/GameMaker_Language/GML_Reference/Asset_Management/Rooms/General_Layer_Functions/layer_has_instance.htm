layer_has_instance

Cette fonction peut être utilisée pour vérifier si une instance donnée est actuellement assignée à la couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères - cela aura un impact sur les performances) et l'ID de l'instance à vérifier. Vous pouvez également fournir une object_index (c'est-à-dire le nom de l'objet dans le navigateur d'actifs) et la fonction vérifiera si des instances de cet objet se trouvent sur la couche donnée. La fonction renverra true si l'instance se trouve sur la couche et false dans le cas contraire.

Syntaxe :

layer_has_instance(layer_id, instance_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
instance_id ID de l'instance The unique instance ID or the object index of the instance to check for

Les retours :

Boolean

Exemple :

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Le code ci-dessus vérifiera si la couche donnée contient des instances de l'objet "obj_Bullet_Parent" et si ce n'est pas le cas, il détruira la couche.
