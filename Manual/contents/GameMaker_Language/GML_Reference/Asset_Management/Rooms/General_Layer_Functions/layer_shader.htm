layer_shader

Grâce à cette fonction, vous pouvez affecter une ressource shader à une couche donnée et la couche sera alors rendue en utilisant cette shader. Vous fournissez soit la valeur d'identification unique de la couche, soit le nom de la couche (sous forme de chaîne de caractères, ce qui a un impact sur les performances), ainsi que l'ID du site shader à utiliser. Le site shader doit avoir été créé précédemment dans l'Asset Browser et l'index shader (le nom de la ressource shader ) est ensuite transmis à cette fonction. Si des instances ont été ajoutées à la couche attribuée, l'adresse shader sera appliquée à tous les événements de dessin que l'instance utilise - par exemple, si l'instance a un événement Draw GUI Begin, l'adresse shader lui sera automatiquement appliquée. L'adresse shader affectera également tous les autres éléments graphiques dessinés sur cette couche, comme les actifs sprite ou les cartes de tuiles.

Notez que la fonction n'est pas censée être appelée dans des événements de dessin ou des événements d'étape, mais qu'elle ne doit être appelée qu'au début de la pièce dans le code de création de pièce ou dans l'événement de création / de début de pièce d'une instance.

Syntaxe :

layer_shader(layer_id, shader)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
shader Shader Asset The shader index to assign to the layer

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

Le code ci-dessus affectera la ressource shader "shd_Sepia" à la couche donnée pour tous les dessins.
