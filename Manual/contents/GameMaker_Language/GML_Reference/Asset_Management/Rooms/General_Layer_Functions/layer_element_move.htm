couche_élément_mouvement

Vous pouvez utiliser cette fonction pour déplacer un élément d'une couche à une autre. Vous donnez l'ID de l'élément, tel que renvoyé par la fonction utilisée pour créer l'élément ou l'éditeur de pièce ou la fonction layer_get_all_elements()et ensuite vous donnez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant la fonction layer_create()) ou le nom du calque (sous forme de chaîne de caractères - cela aura un impact sur les performances).

Syntaxe :

layer_element_move(element_id, layer_id)

Arguments Type Description
element_id ID de l'élément de couche The unique ID value of the element to move
layer_id Chaîne ou ID de couche The unique ID value of the layer to move the element to (or the layer name as a string)

Les retours :

N/A

Exemple :

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_element_move(a[i], asset_layer)
    }
}

Le code ci-dessus récupère les éléments sur la couche à laquelle l'instance qui exécute le code est assignée, puis vérifie s'il s'agit de ressources sprite. Si c'est le cas, elles sont déplacées vers la couche dont l'ID est stocké dans la variable "asset_layer".
