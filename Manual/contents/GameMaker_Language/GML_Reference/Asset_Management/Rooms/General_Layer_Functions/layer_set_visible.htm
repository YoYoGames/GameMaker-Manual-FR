layer_set_visible

Cette fonction permet d'activer ou de désactiver la visibilité d'une couche. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères - cela aura un impact sur les performances) ainsi que la valeur de basculement de la couche, où visible est true et invisible est false.

Lorsqu'un calque est invisible, rien de ce qui est assigné au calque ne sera dessiné, et si des instances sont assignées au calque, elles n'exécuteront même pas leur événement de dessin (de la même manière que si vous définissez la variable d'instance visible sur false). Notez que les instances qui sont déjà marquées comme invisibles ne seront pas marquées comme visibles si le layer sur lequel elles se trouvent est défini comme visible, car il s'agit de deux paramètres indépendants, bien que leur comportement soit le même.

Syntaxe :

layer_set_visible(layer_id, visible)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
visible Booléen Whether the layer should be visible (true) ou non (false)

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

Le code ci-dessus récupère la valeur de l'ID du calque nommé "Instances" dans l'éditeur de pièce, puis utilise l'ID pour vérifier si le calque est visible ou non, en basculant la visibilité du calque en fonction de la valeur retournée.
