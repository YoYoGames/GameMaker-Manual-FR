layer_get_shader

Cette fonction peut être utilisée pour vérifier si un shader est attribué à une couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous la forme d'une chaîne de caractères - cela aura un impact sur les performances), et la fonction renverra soit l'indice shader du shader attribué, soit -1 si aucun shader n'est attribué.

Syntaxe :

layer_get_shader(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)

Les retours :

Shader Asset or -1 (if no shader is assigned)

Exemple :

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

Le code ci-dessus vérifie si la couche de l'instance qui exécute le code a une shader assignée et si elle ne l'a pas, une est assignée.
