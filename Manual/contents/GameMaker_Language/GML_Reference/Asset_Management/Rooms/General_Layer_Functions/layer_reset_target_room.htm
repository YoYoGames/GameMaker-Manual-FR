layer_reset_target_room

Cette fonction est utilisée pour réinitialiser la cible de la couche à la pièce actuelle. Voir la fonction layer_set_target_room() pour plus d'informations.

Syntaxe :

layer_reset_target_room()

Les retours :

Exemple :

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    layer_sprite_create(l, irandom(1000), irandom(1000), spr_Trees);
}
layer_reset_target_room();

Le code ci-dessus définit la pièce cible comme étant la pièce "rm_Game" et obtient ensuite l'ID du calque appelé "SpriteAssets" dans cette pièce. Cet ID de couche est ensuite utilisé pour ajouter 50 actifs sprite aléatoires à la couche, avant que la cible de la couche ne soit réinitialisée à la pièce actuelle.
