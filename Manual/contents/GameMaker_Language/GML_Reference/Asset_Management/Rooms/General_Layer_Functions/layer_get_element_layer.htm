couche_get_élément_couche

Vous pouvez utiliser cette fonction pour obtenir l'ID du calque sur lequel se trouve l'élément donné. Vous fournissez la valeur unique de l'ID de l'élément (par exemple, telle que renvoyée par la fonction qui a créé l'élément ou par l'éditeur de salle) et la fonction renverra l'ID unique de la couche sur laquelle se trouve l'élément. Si l'ID de l'élément donné n'est pas valide, la fonction renvoie -1.

Syntaxe :

layer_get_element_layer(element_id)

Arguments Type Description
element_id ID de l'élément de couche The unique ID value of the element to get the layer of

Les retours :

Layer ID or -1 (if the element is invalid)

Exemple :

element = layer_get_element_layer(asset_1);

Le code ci-dessus obtient l'ID de la couche pour l'élément avec l'ID stocké dans la variable "asset_1" et le stocke dans une variable.
