couche_get_x

Vous pouvez utiliser cette fonction pour récupérer la position x de la couche dans la pièce actuellement délimitée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom du calque (sous la forme d'une chaîne - cela aura un impact sur les performances) et la fonction renvoie un nombre réel pour la position x du calque, par rapport à la position (0,0) de la pièce. La valeur par défaut est 0.

Syntaxe :

layer_get_x(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to get the x position of

Les retours :

Real

Exemple :

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

Le code ci-dessus vérifie la position de la couche donnée et si elle n'est pas définie à (0, 0), alors elle est définie à cette position.
