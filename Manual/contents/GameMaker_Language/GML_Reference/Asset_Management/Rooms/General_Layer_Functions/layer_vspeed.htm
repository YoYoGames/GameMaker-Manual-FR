couche_vitesse

Vous pouvez utiliser cette fonction pour définir la vitesse verticale (en pixels par image de jeu) de la couche dans la salle actuellement délimitée. Vous fournissez l'ID du calque (que vous obtenez lorsque vous créez le calque à l'aide de la fonction layer_create()) ou le nom du calque (sous forme de chaîne de caractères - cela aura un impact sur les performances) et la valeur de vitesse à définir, où une valeur positive correspond à un déplacement vers le bas et une valeur négative vers le haut.

Syntaxe :

layer_vspeed(layer_id, vspd)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to set the vertical speed of
vspd Real The vertical speed (in pixels per game frame) to set

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

Le code ci-dessus vérifie les vitesses horizontale et verticale de la couche donnée et si elles ne sont pas toutes deux à 0, il les met à 0.
