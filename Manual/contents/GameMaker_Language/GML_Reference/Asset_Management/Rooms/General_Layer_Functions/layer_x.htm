couche_x

Vous pouvez utiliser cette fonction pour définir la position x du calque dans la pièce actuellement délimitée. Vous fournissez l'ID du calque (que vous obtenez lorsque vous créez le calque à l'aide de la fonction layer_create()) ou le nom du calque (sous forme de chaîne - cela aura un impact sur les performances) et la fonction déplacera le calque du nombre de pixels donné le long de l'axe horizontal de la pièce.

Syntaxe :

layer_x(layer_id, x)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to set the x position of
x Real The x position in the room to set the layer to

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

Le code ci-dessus vérifie la position de la couche donnée et si elle n'est pas définie à (0, 0), alors elle est définie à cette position.
