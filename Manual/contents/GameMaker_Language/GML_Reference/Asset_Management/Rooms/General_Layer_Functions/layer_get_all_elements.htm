couche_get_tout_élément

Vous pouvez utiliser cette fonction pour obtenir les ID des éléments de la couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères - cela aura un impact sur les performances) et la fonction renverra un tableau d'identifiants, où chaque entrée du tableau est un identifiant unique pour un élément de cette couche. Par exemple, si la couche est une couche d'actifs, le tableau sera rempli avec les valeurs d'identification de chaque actif sprite qui est attribué à la couche. Notez que l'utilisation du code pour travailler avec les couches signifie que vous pouvez affecter différents types d'éléments à la même couche - vous pouvez donc avoir des actifs sprite ainsi que des instances, par exemple - dans ce cas, vous pouvez utiliser la fonction layer_get_element_type() pour obtenir le type d'élément auquel l'ID se rapporte.

Syntaxe :

layer_get_all_elements(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de la couche The unique ID value of the layer to get the elements from (or the layer name as a string)

Les retours :

Array of Layer Element IDs

Exemple :

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if (layer_get_element_type(a[i]) == layerelementtype_sprite)
    {
        layer_sprite_destroy(a[i])
    }
}

Le code ci-dessus récupère les ID de tous les éléments d'instance affectés à la couche de l'instance qui exécute le code. Le code vérifie ensuite si l'un des éléments renvoyés est un actif de sprite et s'il l'est, il est détruit.
