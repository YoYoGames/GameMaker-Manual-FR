couche_destroy_instances

Cette fonction peut être utilisée pour détruire toutes les instances affectées à la couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères - cela aura un impact sur les performances), et toutes les instances qui se trouvent sur la couche seront supprimées du jeu, en déclenchant leurs événements Destroy et Clean Up.

Syntaxe :

layer_destroy_instances(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer (or the layer name as a string)

Les retours :

N/A

Exemple :

if global.game_over
{
    layer_destroy_instances(layer);
}

Le code ci-dessus vérifiera une variable globale et si elle est true alors toutes les instances qui sont sur la couche de l'instance appelante seront détruites (y compris l'instance appelante).
