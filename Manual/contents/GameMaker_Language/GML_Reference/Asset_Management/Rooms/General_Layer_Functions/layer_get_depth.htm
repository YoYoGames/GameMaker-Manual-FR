couche_get_depth

Vous pouvez utiliser cette fonction pour obtenir la valeur de profondeur associée à une couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom du calque (sous forme de chaîne de caractères - cela aura un impact sur les performances) et la fonction renverra la profondeur du calque sous forme de nombre réel. Notez que la profondeur est définie comme étant plus élevée au fur et à mesure que l'on s'éloigne de la caméra et plus faible au fur et à mesure que l'on s'en rapproche. Ainsi, si vous avez trois couches avec des profondeurs de -100, 0, 100, les couches seront dessinées dans l'ordre 100, 0, -100, de sorte que la couche "supérieure" (c'est-à-dire la plus proche de la vue de la caméra et donc dessinée par-dessus tout le reste) sera la couche avec la profondeur -100.

Notez que si vous fournissez une valeur d'ID de couche invalide, vous obtiendrez une erreur.

Syntaxe :

layer_get_depth(layer_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to get the depth of (or the layer name as a string)

Les retours :

Real

Exemple :

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

Le code ci-dessus vérifie la profondeur de l'ID d'une couche stockée dans une variable globale et si elle n'est pas -100, il détruit la couche et la recrée à la profondeur de -100.
