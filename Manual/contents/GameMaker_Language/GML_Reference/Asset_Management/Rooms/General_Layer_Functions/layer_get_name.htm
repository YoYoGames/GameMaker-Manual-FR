couche_get_name

Vous pouvez utiliser cette fonction pour obtenir le nom de la couche donnée. Vous fournissez la valeur unique de l'ID de la couche et si la couche est l'une des couches nommées créées dans l'éditeur de pièce, alors la fonction renverra une chaîne avec le nom de la couche. Si la couche n'est pas l'une de celles de l'éditeur de pièce (c'est-à-dire qu'elle a été créée à l'aide de la fonction layer_create()), une chaîne vide sera retournée.

Syntaxe :

layer_get_name(layer_id)

Arguments Type Description
layer_id ID de la couche The unique ID value of the layer to get the name of

Les retours :

String

Exemple :

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_name(a[i]) != ""
    {
        ds_list_add(layer_list, a[i])
    }
}

Le code ci-dessus récupère les ID de tous les calques de la pièce et les parcourt en boucle pour vérifier si certains sont des calques nommés. S'ils le sont, leur ID est ajouté à une liste.
