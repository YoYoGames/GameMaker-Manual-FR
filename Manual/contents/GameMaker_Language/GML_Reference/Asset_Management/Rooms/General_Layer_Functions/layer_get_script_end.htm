couche_get_script_end

Cette fonction renvoie l'index de la fonction de script de la fonction assignée à exécuter à la fin du rendu du calque donné, ou elle renvoie -1 si aucune fonction n'est assignée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom du calque (sous forme de chaîne de caractères - cela aura un impact sur les performances). Vous pouvez attribuer des fonctions de script à une couche avec layer_script_begin() et layer_script_end().

Syntaxe :

layer_get_script_end(layer_id);

Arguments Type Description
layer_id Chaîne ou ID de la couche The unique ID value of the layer to target (or the layer name as a string)

Les retours :

Script Function or -1 (if no function is assigned)

Exemple :

if layer_get_script_end(layer) == -1
{
    layer_script_end(layer, scr_ResetShaderValues);
}

Le code ci-dessus vérifie si le calque de l'instance qui exécute le code a une fonction de script assignée et si ce n'est pas le cas, une fonction est assignée.
