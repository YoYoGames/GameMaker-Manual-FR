layer_depth

Cette fonction peut être utilisée pour modifier la profondeur de la couche donnée, en changeant l'ordre dans lequel son contenu sera rendu à l'écran. Vous fournissez l'ID du calque (que vous obtenez lorsque vous créez le calque à l'aide de la fonction layer_create()) ou le nom du calque (sous la forme d'une chaîne de caractères - cela aura un impact sur les performances), puis indiquez la nouvelle profondeur à laquelle le calque doit être défini (une valeur entière comprise entre -16000 et 16000). La profondeur du calque est définie comme étant plus élevée lorsqu'elle est "éloignée" de la caméra et plus faible lorsqu'elle est "proche" de la caméra. Ainsi, si vous avez trois calques avec des profondeurs de -100, 0, 100, les calques seront dessinés dans l'ordre 100, 0, -100, de sorte que le calque "supérieur" (c'est-à-dire le plus proche de la vue de la caméra et donc dessiné au-dessus de tout le reste) sera le calque avec la profondeur -100.

Notez que vous pouvez vérifier la profondeur d'une couche à tout moment en utilisant la fonction layer_get_depth(). Notez également que les profondeurs minimale et maximale des couches sont de -16000 à 16000, et que tout ce qui se trouve en dehors de ces profondeurs ne sera pas rendu. Si vous avez besoin d'une profondeur en dehors de cette plage, vous devrez utiliser la fonction layer_force_draw_depth().

Gardez à l'esprit que modifier la profondeur d'un calque peut changer les filtres et les effets qui lui sont appliqués, car si vous modifiez la profondeur pour qu'elle soit inférieure à celle d'un calque FX, l'effet ne sera plus appliqué au calque.

Syntaxe :

layer_depth(layer_id, depth)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to set the depth of (or the layer name as a string)
depth Real The new depth for the layer

Les retours :

N/A

Exemple :

if layer_get_depth(layer) != -100
{
    layer_depth(layer, -100);
}

Le code ci-dessus récupère la profondeur de la couche sur laquelle se trouve l'instance qui exécute le code, et si elle n'est pas -100, alors la profondeur est fixée à -100.
