chambre_dernière

Cette variable en lecture seule renvoie l'index de la toute dernière salle du jeu (celle-ci est définie par l'ordre dans lequel les salles apparaissent dans le gestionnaire de salles et non par l'ordre dans lequel elles ont été créées). Notez que cette variable ne reconnaîtra pas ou ne prendra pas en compte les salles qui ont été ajoutées dynamiquement à l'aide de room_add() ou room_duplicate().

Syntaxe :

room_last;

Les retours :

Room Asset

Exemple :

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

Le code ci-dessus vérifie si une touche a été pressée et si c'est le cas, il se rend dans la dernière pièce du jeu.
