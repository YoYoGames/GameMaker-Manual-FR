chambre_set_hauteur

Cette fonction vous permet de modifier (ou de définir) la hauteur de n'importe quelle pièce de votre jeu , à l'exception de la pièce actuelle.

Syntaxe :

room_set_height(index, h);

Arguments Type Description
index Actif de la salle The index of the room to set the height of.
h Real The new height of the room in pixels.

Les retours :

N/A

Exemple :

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Cette opération créera une nouvelle salle et stockera son index dans la variable "global.myroom". Elle définit ensuite sa largeur à 640 pixels, sa hauteur à 480 pixels, sa légende à "Game Room" et sa persistance à "false".
