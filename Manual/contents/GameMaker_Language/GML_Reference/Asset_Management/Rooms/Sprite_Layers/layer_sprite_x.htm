couche_sprite_x

Cette fonction contrôle la position sur l'axe des x de la pièce de l'élément d'actif sprite sur la couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur x à utiliser (en fonction des coordonnées de la pièce).

Syntaxe :

layer_sprite_x(sprite_element_id, x);

ArgumentsType Description
sprite_element_id The unique ID value of the sprite element to change
x The x position for the sprite

Les retours :

Exemple :

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

Le code ci-dessus récupère la valeur d'identification de l'actif sprite "Clouds" affecté au calque "Asset_sky", puis définit sa position x à une valeur aléatoire comprise entre 0 et la largeur de la pièce.
