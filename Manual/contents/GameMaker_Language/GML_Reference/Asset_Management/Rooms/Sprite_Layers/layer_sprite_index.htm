layer_sprite_index

Cette fonction peut être utilisée pour définir l'index d'image d'un élément sprite qui possède plusieurs sous-images sur une couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez l'indice d'image à utiliser. Si vous définissez une valeur en dehors de la plage des sous-images, l'indice d'image tournera en boucle.

Syntaxe :

layer_sprite_index(sprite_element_id, image_index);

Arguments Type Description
sprite_element_id Sprite Asset The unique ID value of the sprite element to set
index Real The image index to use for the sprite

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

Le code ci-dessus récupère l'ID du calque nommé "sprite_trees", puis l'utilise pour obtenir l'ID de l'élément sprite donné sur ce calque. Cet ID est ensuite utilisé pour modifier l'index de l'image de l'élément.
