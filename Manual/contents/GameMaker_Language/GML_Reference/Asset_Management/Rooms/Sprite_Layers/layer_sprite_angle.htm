angle_sprite_couche

Cette fonction vous permet de modifier l'angle de l'élément sprite donné sur une couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur de l'angle, de 0 à 359 dans le sens inverse des aiguilles d'une montre, où 0 est à droite, 90 est en haut, 180 est à gauche et 270 est en bas. Si vous définissez une valeur supérieure à 360, celle-ci sera mise en boucle afin de la ramener dans la plage 0 - 359.

Syntaxe :

layer_sprite_angle(sprite_element_id, angle);

Arguments Type Description
sprite_element_id ID de l'élément Sprite The unique ID value of the sprite element to change
angle Real The angle of the sprite (default is 0)

Les retours :

N/A

Exemple :

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
    layer_sprite_angle(asset_sprite, 0);
}

Le code ci-dessus vérifiera l'élément sprite attribué à la couche sur laquelle se trouve l'instance qui exécute le code et si son angle est différent de 0, il le mettra à 0.
