couche_sprite_existe

Vous pouvez utiliser cette fonction pour vérifier et voir si un élément sprite existe sur une couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères - cela aura un impact sur les performances) et l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()) et la fonction renverra soit true si l'élément existe, soit false s'il n'existe pas.

REMARQUE: Cette fonction fonctionne dans le cadre de la pièce cible actuelle - par défaut, la pièce dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room().

Syntaxe :

layer_sprite_exists(layer_id, sprite_element_id)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id ID de l'élément Sprite The unique ID value of the sprite element to check

Les retours :

Boolean

Exemple :

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

Le code ci-dessus vérifie la couche "Asset_trees" pour voir si l'élément donné sprite existe et si c'est le cas, alors il est détruit (mais pas la couche).
