couche_sprite_alpha

Cette fonction contrôle l'alpha (la transparence) de l'élément sprite sur la couche d'actif. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur alpha à utiliser. La valeur alpha peut être comprise entre 0 (totalement transparent) et 1 (totalement opaque), la valeur alpha par défaut de l'élément sprite étant 1. Notez que si le calque auquel l'élément sprite a été attribué n'est pas visible - ou si l'élément lui-même a été rendu invisible - vous ne verrez aucune différence avec cette fonction tant que le calque ou l'élément n'aura pas été rendu visible à nouveau.

Syntaxe :

layer_sprite_alpha(sprite_element_id, alpha);

ArgumentsType Description
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

Les retours :

Exemple :

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

Le code ci-dessus récupère la valeur d'identification de l'actif sprite nommé "Clouds" affecté au calque "Asset_sky", puis définit son alpha sur une valeur aléatoire comprise entre 0 et 1.
