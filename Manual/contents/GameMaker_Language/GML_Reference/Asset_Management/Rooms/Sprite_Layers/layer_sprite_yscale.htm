couche_sprite_échelle

Cette fonction vous permet de déterminer si l'élément sprite donné d'une couche doit être mis à l'échelle le long de l'axe des y ou non. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur de l'échelle. Une échelle de 1 indique qu'il n'y a pas de mise à l'échelle (1:1), des valeurs plus petites réduisent l'échelle (0,5, par exemple, correspond à la moitié de la hauteur du sprite utilisé), des valeurs plus grandes l'augmentent et des valeurs négatives reflètent le sprite et le mettent à l'échelle, sauf si la valeur utilisée est exactement -1 (dans ce cas, le sprite utilisé est simplement reflété de haut en bas autour de sa position (0, 0) sans mise à l'échelle).

Syntaxe :

layer_sprite_yscale(sprite_element_id, yscale);

ArgumentsType Description
sprite_element_id The unique ID value of the sprite element to change
yscale The yscale value (default is 1)

Les retours :

Exemple :

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

Le code ci-dessus vérifie l'élément sprite attribué au calque sur lequel se trouve l'instance qui exécute le code et, s'il est mis à l'échelle dans l'une ou l'autre direction, il définit l'échelle de l'axe des x et l'échelle de l'axe des y à 1.
