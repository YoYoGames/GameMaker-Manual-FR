vitesse_sprite_couche

Cette fonction peut être utilisée pour régler la vitesse d'une image animée qui a été assignée à un élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis vous définissez la vitesse de l'animation. Si le site IDE a défini la vitesse d'animation du site sprite utilisé comme étant calculée en Frames Per Game Frame, la valeur de la vitesse indiquée est un multiplicateur, 1 étant la valeur par défaut, et si vous la réglez sur 0,5, la vitesse d'animation sera réduite de moitié, tandis que si vous la réglez sur 1,5, elle sera doublée. Toutefois, si le site IDE a défini la vitesse de l'animation pour le site sprite utilisé comme étant calculée en images par seconde, la valeur de la vitesse est une valeur d'images par seconde littérale, de sorte qu'une valeur de 5, par exemple, fera tourner l'animation en boucle de 5 images par seconde pendant le jeu. Si le site sprite utilisé ne comporte pas de sous-images, cette fonction n'aura aucun effet.

Syntaxe :

layer_sprite_speed(sprite_element_id, speed);

Arguments Type Description
sprite_element_id Sprite Asset The unique ID value of the sprite element to change
speed Real The speed multiplier/value to use for the element

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

Le code ci-dessus obtiendra l'ID du calque nommé "Assets_water", puis l'utilisera pour obtenir l'ID de l'élément sprite donné sur ce calque. Cet ID est ensuite utilisé pour modifier la vitesse de l'image de l'élément.
