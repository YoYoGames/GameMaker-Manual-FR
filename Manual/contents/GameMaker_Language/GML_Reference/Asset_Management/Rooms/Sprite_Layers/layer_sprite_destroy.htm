layer_sprite_destroy

Cette fonction va détruire l'élément sprite donné. Vous fournissez l'ID de sprite (que vous obtenez lorsque vous créez sprite à l'aide de layer_sprite_create() ou lorsque vous utilisez l'ID de la couche avec layer_get_sprite_id()) et cette fonction le supprimera. Notez que cela ne supprime pas la couche, mais seulement le sprite. Si le sprite a été ajouté dans l'éditeur de pièce, le sprite sera recréé la prochaine fois que vous quitterez la pièce et y reviendrez. Cependant, si la pièce est persistante, le sprite sera supprimé, à moins que la persistance de la pièce ne soit à nouveau désactivée.

Syntaxe :

layer_sprite_destroy(sprite_element_id)

ArgumentsType Description
sprite_element_id The unique ID value of the sprite to be destroyed

Les retours :

Exemple :

var lay_id = layer_get_id("Asset_Trees");
var spr_id = layer_sprite_get_id(lay_id, "graphic_254367CB");

if (layer_sprite_exists(lay_id, spr_id))
{
    layer_sprite_destroy(lay_id);
}

Le code ci-dessus vérifie la couche "Asset_Trees" pour voir si l'élément donné sprite existe et si c'est le cas, alors il est détruit (mais pas la couche).
