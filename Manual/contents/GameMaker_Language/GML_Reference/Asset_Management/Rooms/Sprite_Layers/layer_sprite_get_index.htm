layer_sprite_get_index

Cette fonction peut être utilisée pour obtenir la valeur actuelle de l'indice d'image de l'élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renvoie une valeur réelle qui représente l'indice d'image affiché pour l'élément sprite. La fonction renvoie -1 si l'élément sprite n'existe pas ou si aucun élément sprite valide ne lui est attribué.

Syntaxe :

layer_sprite_get_index(sprite_element_id);

ArgumentsType Description
sprite_element_id The unique ID value of the sprite element to get the information from

Les retours :

Real or -1

Exemple :

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_index(spr_id) < 4)
{
    layer_sprite_index(spr_id, 4);
}

Le code ci-dessus récupère l'ID du calque nommé "sprite_sky", puis l'utilise pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier si l'index de l'image de l'élément est inférieur à 4, et si c'est le cas, il est fixé à 4.
