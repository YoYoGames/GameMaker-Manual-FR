layer_sprite_get_id

Cette fonction peut être utilisée pour récupérer la valeur d'identification unique d'un élément sprite sur une couche. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create() ou lorsque vous utilisez le nom de la couche avec layer_get_id()) et le nom de l'élément sprite tel que défini dans l'éditeur de salle. La fonction renvoie la valeur de l'ID associée à cet élément sprite sur la couche.

Notez que cette fonction est spécifiquement conçue pour être utilisée avec sprites qui a été ajouté dans le IDE sur une couche d'actif, et si vous avez ajouté un sprite à une couche par le biais du code utilisant la fonction layer_sprite_create()Si vous avez ajouté un élément sprite à une couche par le biais d'un code utilisant la fonction , alors l'ID renvoyé par cette fonction doit être utilisé pour toute référence future (car cet élément n'aura pas de nom à transmettre à cette fonction).

Si la couche spécifiée n'existe pas, ou si l'élément sprite donné est introuvable, la fonction renvoie -1.

Syntaxe :

layer_sprite_get_id(layer_id, sprite_element_name)

Arguments Type Description
layer_id Chaîne ou ID de couche The unique ID value of the layer to target
sprite_element_name Chaîne de caractères The unique name of the sprite element on the layer as defined in the Room Editor

Les retours :

Sprite Element ID

Exemple :

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

Le code ci-dessus obtiendra l'ID de la couche nommée "Assets_trees", puis l'utilisera pour récupérer l'ID de l'élément sprite "graphic_254367CB" sur cette couche. L'ID de l'élément sprite récupéré est ensuite utilisé pour modifier l'élément sprite.
