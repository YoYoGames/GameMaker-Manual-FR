layer_sprite_get_sprite

Cette fonction peut être utilisée pour obtenir l'indice actuel sprite de l'élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renvoie une valeur réelle qui représente l'indice sprite affiché. Si l'élément n'a pas de sprite attribué, la fonction renverra -1.

Syntaxe :

layer_sprite_get_sprite(sprite_element_id);

ArgumentsType Description
sprite_element_id The unique ID value of the sprite element to get the information from

Les retours :

Sprite Asset or -1

Exemple :

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
    layer_sprite_change(spr_id, spr_Clouds);
}

Le code ci-dessus récupère l'ID du calque nommé "sprite_sky", puis l'utilise pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier le sprite attribué à l'élément, en le définissant sur le sprite "spr_Clouds" s'il ne l'est pas déjà.
