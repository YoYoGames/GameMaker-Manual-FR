changement de couche_sprite

Cette fonction vous permet de modifier la ressource sprite attribuée à un élément sprite donné sur une couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis fournissez une ressource d'élément d'index sprite qui sera modifiée. Notez que si l'élément sprite a des sous-images, il s'animera également (ceci peut être contrôlé en utilisant les fonctions layer_sprite_index() et layer_sprite_speed() ). Notez que vous pouvez attribuer une valeur de -1 comme nouvel indice sprite et aucun sprite ne sera affiché, bien que l'élément existe toujours et puisse être modifié à nouveau ultérieurement.

Syntaxe :

layer_sprite_change(sprite_element_id, sprite_index)

Arguments Type Description
sprite_element_id ID de l'élément Sprite The unique ID value of the sprite element to change
sprite_index Sprite Asset The new sprite index to use

Les retours :

N/A

Exemple :

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
    layer_sprite_change(s, spr_nighttime);
}

Le code ci-dessus vérifie l'index sprite de l'élément avec l'ID stocké dans la variable globale global.Asset_sprite et s'il n'est pas "spr_Nighttime", alors cette sprite est attribuée à l'élément.
