couche_sprite_y

Cette fonction contrôle la position sur l'axe des y de la pièce de l'élément d'actif sprite sur la couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur y à utiliser (en fonction des coordonnées de la pièce).

Syntaxe :

layer_sprite_y(sprite_element_id, y);

ArgumentsType Description
sprite_element_id The unique ID value of the sprite element to change
y The y position for the sprite

Les retours :

Exemple :

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

Le code ci-dessus récupère la valeur d'identification de l'actif sprite "Clouds" affecté au calque "Asset_sky", puis définit sa position y à une valeur aléatoire comprise entre 50 et la hauteur de la pièce moins 200.
