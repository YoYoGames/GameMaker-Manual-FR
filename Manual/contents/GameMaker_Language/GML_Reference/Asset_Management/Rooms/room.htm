chambre

Cette variable contient l'indice de la salle en cours d'exécution dans votre jeu. Il ne s'agit pas d'une variable en lecture seule, cependant, la modifier ne changera pas la valeur de l'indice de la salle actuelle, mais plutôt la salle pour qu'elle corresponde à l'indice que vous avez défini pour la variable. Il convient d'être prudent car si l'indice que vous modifiez n'est pas valide, le jeu affichera une erreur et se fermera. En général, il est préférable d'utiliser room_goto() pour changer de pièce.

Remarque: Les identifiants de salle ne sont pas basés sur leur ordre dans le navigateur d'actifs ou dans le gestionnaire de salles, et vous devez donc éviter de fournir directement une valeur numérique. Utilisez plutôt la constante de salle de l'actif que vous souhaitez référencer (qui est simplement son nom) ou récupérez-la par le biais d'une fonction.

Syntaxe :

room;

Les retours :

Room Asset

Exemple :

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

Le code ci-dessus vérifiera si la pièce actuelle est le premier niveau, et dans ce cas, jouera la musique de ce niveau.
