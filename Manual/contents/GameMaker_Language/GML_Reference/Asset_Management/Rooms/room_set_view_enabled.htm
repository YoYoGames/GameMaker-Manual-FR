room_set_view_enabled

Cette fonction vous permet d'activer (true) ou de désactiver (false) la vue de n'importe quelle pièce de votre jeu, à l'exception de la pièce actuelle.

Syntaxe :

room_set_view_enabled(index, val);

Arguments Type Description
index Actif de la salle The index of the room to set.
val Booléen Whether to enable (true) ou désactiver (false) les vues dans la pièce donnée.

Les retours :

N/A

Exemple :

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

Cela permettra d'afficher des vues dans la salle indexée dans "global.myroom".
