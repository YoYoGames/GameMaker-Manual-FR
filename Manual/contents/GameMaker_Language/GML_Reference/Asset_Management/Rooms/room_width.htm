room_width

Cette variable contient la largeur de la pièce actuelle en pixels. Vous pouvez modifier cette variable pour changer la largeur de la pièce à tout moment.

Syntaxe :

room_width

Les retours :

Exemple :

if bbox_right > room_width
{
    x += room_width - bbox_right;
}

Le code ci-dessus vérifie si la boîte de délimitation sprite de l'instance actuelle est supérieure à la largeur de la pièce, et si c'est le cas, il déplace l'instance vers le haut.
