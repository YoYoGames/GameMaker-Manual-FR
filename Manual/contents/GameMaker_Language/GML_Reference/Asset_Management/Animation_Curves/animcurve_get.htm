animcurve_get

Cette fonction renvoie une structure contenant toutes les données de la courbe d'animation donnée. Vous fournissez l'ID de l'actif de la courbe d'animation (tel que défini dans l'Asset Browser), et la fonction renvoie un struct avec les variables suivantes :

Structure de la courbe d'animation
Nom de la variable Type de données Description
name Chaîne de caractères Il s'agit du nom de la courbe d'animation.
channels Array de Structure de canal de courbe d'animations Il s'agit d'un tableau, où chaque élément du tableau est un canal struct.


La variable channels dans le site struct est un tableau où chaque élément du tableau est une variable Structure du canal de la courbe d'animation avec des données relatives à un canal dans la courbe. Le canal struct est expliqué sur cette page.

Comme pour les canaux, les points d'un seul canal sont stockés comme structs dans un tableau, où chaque élément du tableau est un point unique struct. Le point struct est expliqué sur cette page.

Notez que si la fonction échoue (par exemple, si la ressource Courbe d'animation donnée n'existe pas), elle renvoie -1.

Syntaxe :

animcurve_get(curve_id);

Arguments Type Description
curve_id Courbe d'animation Asset The asset browser ID (index) of the animation curve to get.

Les retours :

Animation Curve Struct or -1

Exemple :

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if (_channel.type != animcurvetype_linear)
{
    _channel.type = animcurvetype_linear;
}

Le code ci-dessus récupère la courbe d'animation struct pour l'actif courbe "ac_ButtonTween", puis vérifie le type de courbe pour le premier canal, et s'il n'est pas linéaire, il le définit comme linéaire.
