animcurve_existe

Cette fonction retournera vrai si la courbe d'animation Courbe d'animation Asset ou Structure de la courbe d'animation donné existe et est une courbe d'animation valide, ou false s'il ne l'est pas.

Syntaxe :

animcurve_exists(curve_struct_or_id);

Arguments Type Description
curve_struct_or_id Structure de la courbe d'animation ou Courbe d'animation Asset The Animation Curve that will be checked.

Les retours :

Boolean

Exemple :

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

Le code ci-dessus vérifie si la courbe d'animation stockée dans la variable personnalisée spring_curve existe. Si c'est le cas, il exécute une méthode personnalisée pour utiliser cette courbe d'animation.
