Courbes d'animation

Les courbes d'animation peuvent être configurées dans le navigateur d'actifs en tant qu'actifs à l'aide de l'éditeur de courbes d'animation et elles peuvent ensuite être utilisées dans l'éditeur de séquences pour contrôler les pistes d'une séquence. Cependant, elles peuvent également être utilisées dans votre code pour modifier des valeurs au fil du temps dans vos jeux, que ce soit pour des effets d'interpolation sur des boutons, des mouvements, des changements de couleur, etc. Vous pouvez également créer vos propres courbes d'animation à l'adresse runtime en utilisant les fonctions énumérées ici, ce qui en fait un outil très flexible pour vos jeux.

Les fonctions disponibles pour les courbes d'animation sont les suivantes :

  • animcurve_get
  • animcurve_get_channel_index
  • animcurve_get_channel
  • animcurve_channel_evaluate
  • animcurve_create
  • animcurve_existe
  • animcurve_channel_new
  • animcurve_point_new
  • animcurve_destroy
