animcurve_get_channel

Cette fonction renvoie la structure contenant les données de canal pour le canal spécifié dans un actif de courbe d'animation. Vous fournissez l'ID de la courbe d'animation ou struct ainsi que le nom ou l'index du canal, et la fonction renvoie un struct avec le format suivant :

Nom de la variable Type de données Description
name chaîne de caractères Il s'agit du nom du canal.
type Type d'interpolation de la courbe d'animation Constant Il s'agira de l'une des constantes suivantes : animcurvetype_linear pour une interpolation linéaire entre les points, animcurvetype_catmullrom pour une interpolation "lisse" entre les points à l'aide de l'interpolation catmull-rom, ou animcurvetype_bezier pour une interpolation de Bézier.
iterations entier Si le canal utilise l'interpolation catmull-rom ("lisse"), cette valeur indique combien de points ont été générés pour chaque segment de la courbe (notez que ces points supplémentaires sont internes à la fonction et uniquement utilisés pour les calculs de runtime ). Si le canal utilise l'interpolation linéaire, cette valeur existera toujours mais peut être ignorée car elle n'a aucun rapport avec la façon dont la courbe est interpolée.
points pointeur de tableau Il s'agit d'un tableau, où chaque élément du tableau est un point struct.


L'ID de la courbe d'animation est le nom de la courbe d'animation telle qu'elle a été définie dans l'Asset Browser, ou le pointeur struct tel que renvoyé par la fonction animcurve_create(). Le nom du canal est une chaîne qui fait référence au canal tel qu'il a été défini dans l'actif de la courbe d'animation, ou vous pouvez fournir une valeur d'index, comprise entre 0 et n, où n est le dernier canal de l'actif de la courbe (par exemple, si une courbe d'animation comporte 4 canaux, ils seront indexés de 0 à 3). Notez que le passage d'une valeur d'index nécessitera moins de traitement que le passage d'un nom de canal. Si la fonction échoue (c'est-à-dire si aucun canal n'existe avec le nom ou l'index donné), vous obtiendrez une erreur.

Les points d'un canal unique sont stockés sous la forme structs dans un tableau, où chaque élément du tableau est un point unique struct. Le point struct a les variables suivantes :

Nom de la variable Type de données Description
posx réel La position dans le temps (normalisée de 0 à 1) du point.
value réel La valeur du point.

Syntaxe :

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

Arguments Type Description
curve_struct_or_id Structure de la courbe d'animation The ID or struct pointer of the animation curve to target
channel_name_or_index String ou Réel The channel name (a string) or the channel index (an integer).

Les retours :

Animation Curve Channel Struct

Exemple :

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

Le code ci-dessus récupère le canal struct pour le canal 0 dans la courbe de l'actif "ac_ButtonTween", puis boucle sur les points de la courbe du canal et ajoute un à leur valeur.

Retour : Courbes d'animation
