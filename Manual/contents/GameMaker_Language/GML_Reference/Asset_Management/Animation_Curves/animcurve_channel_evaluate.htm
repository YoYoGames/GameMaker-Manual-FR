animcurve_channel_evaluate

Cette fonction peut être utilisée pour obtenir la valeur à un moment précis dans le temps d'une structure de canal. Vous fournissez le pointeur struct pour le canal (tel que renvoyé par la fonction animcurve_get_channel()ou tel que renvoyé dans la courbe d'animation struct par la fonction animcurve_get()) et la position "x" (temps) le long de la courbe du canal à évaluer. Cette position doit être comprise entre 0 et 1, et la fonction renvoie la valeur de la courbe à cette position, ou elle renvoie 0 si le canal struct fourni n'est pas valide.

Syntaxe :

animcurve_channel_evaluate(channel_struct, posx);

Arguments Type Description
channel_struct Structure du canal de la courbe d'animation The struct pointer for the channel to evaluate.
posx Real The position in time to check (from 0 to 1).

Les retours :

Real

Exemple :

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

Le code ci-dessus obtient le canal struct pour l'actif de courbe d'animation "ac_AlphaCurve". Il utilise ensuite la valeur d'évaluation renvoyée pour définir l'alpha de l'image pour l'instance.
