animcurve_point_new

Cette fonction peut être utilisée pour créer une nouvelle structure de points à ajouter à un canal de courbe d'animation. Lorsqu'une structure de points struct est créée, elle est vide et vous devez définir les variables suivantes pour générer les données des points :

Animation Curve Point Struct
Nom de la variable Type de données Description
posx réel La position dans le temps (normalisée de 0 à 1) du point.
value réel La valeur du point.

Chaque point struct que vous créez doit être ajouté à un tableau et c'est ce tableau qui est ajouté à la variable "channels" de la courbe d'animation struct (comme dans l'exemple ci-dessous).

Syntaxe :

animcurve_point_new();

Les retours :

Animation Curve Point Struct

Exemple :

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Le code ci-dessus crée une nouvelle courbe d'animation struct et la stocke dans la variable "my_curve". La variable struct est ensuite remplie avec un nom et un tableau de canaux. Le tableau de canaux contient un seul canal avec trois points.
