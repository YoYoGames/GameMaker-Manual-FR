animcurve_destroy

Cette fonction peut être utilisée pour détruire toutes les courbes d'animation qui ont été précédemment créées avec la fonction animcurve_create(). L'appel de cette fonction supprime la courbe d'animation de la mémoire et nettoie également tous les canaux ou points qu'elle contient (il n'est jamais nécessaire de les nettoyer manuellement). Notez que vous ne pouvez pas détruire les courbes d'animation créées dans le navigateur d'actifs, mais uniquement celles créées sur runtime.

Syntaxe :

animcurve_destroy(curve_struct);

Arguments Type Description
curve_struct Structure de la courbe d'animation The pointer to the curve struct to destroy.

Les retours :

N/A

Exemple :

animcurve_destroy(my_curve);

Le code ci-dessus va détruire la courbe d'animation (précédemment créée) struct indexée dans la variable "my_curve".
