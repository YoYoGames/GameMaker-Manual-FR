balises_claires_de_l'actif

Avec cette fonction, vous pouvez effacer toutes les balises présentes sur un poste à partir du navigateur de postes. Vous fournissez soit le nom du poste (sous forme de chaîne), soit son index. Si vous fournissez une valeur d'index de poste, vous devrez fournir l'argument facultatif de type de poste (une constante), car des postes de types différents peuvent avoir le même index, même s'ils ne peuvent pas avoir le même nom. Les types d'actifs disponibles sont répertoriés dans le tableau ci-dessous :

Constant Description
asset_object Le nom donné fait référence à un objet.
asset_sprite Le nom donné fait référence à un sprite.
asset_sound Le prénom fait référence à un son.
asset_room Le prénom fait référence à un room.
asset_tiles Le nom donné fait référence à un ensemble de carreaux.
asset_path Le nom donné fait référence à un chemin.
asset_script Le prénom fait référence à un script.
asset_font Le nom donné fait référence à une police de caractères.
asset_timeline Le prénom fait référence à une ligne de temps.
asset_shader Le nom donné fait référence à un shader.
asset_animationcurve Le nom donné fait référence à une courbe d'animation.
asset_sequence Le nom donné fait référence à une Séquence.

Si la fonction réussit à effacer les étiquettes, elle renvoie true, sinon elle renvoie false.

Syntaxe :

asset_clear_tags(name,[asset_type]);

Arguments Type Description
name_or_index Chaîne ou actif The name of the asset (a string) or its index value (an integer).
[asset_type] Type d'actif Constant OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument.

Les retours :

Boolean

Exemple :

asset_clear_tags(obj_Enemy_Parent, asset_object);

Le code ci-dessus effacera toutes les balises de l'objet donné.
