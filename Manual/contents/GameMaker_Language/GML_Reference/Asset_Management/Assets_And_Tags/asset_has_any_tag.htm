asset_has_any_tag

Avec cette fonction, vous pouvez vérifier si une ou plusieurs chaînes de balises sont affectées à n'importe quel poste du navigateur de postes. Vous fournissez soit le nom du bien (sous forme de chaîne), soit son index, ainsi qu'une seule chaîne d'étiquettes ou un tableau où chaque élément est une seule chaîne d'étiquettes. Si vous fournissez une valeur d'index d'asset, vous devrez fournir l'argument facultatif de type d'asset (une constante), car des assets de types différents peuvent avoir le même index, même s'ils ne peuvent pas avoir le même nom. Les types de biens disponibles sont répertoriés dans le tableau ci-dessous :

Constant Description
asset_object Le nom donné fait référence à un objet.
asset_sprite Le nom donné fait référence à un sprite.
asset_sound Le prénom fait référence à un son.
asset_room Le prénom fait référence à un room.
asset_tiles Le nom donné fait référence à un ensemble de carreaux.
asset_path Le nom donné fait référence à un chemin.
asset_script Le prénom fait référence à un script.
asset_font Le nom donné fait référence à une police de caractères.
asset_timeline Le prénom fait référence à une ligne de temps.
asset_shader Le nom donné fait référence à un shader.
asset_animationcurve Le nom donné fait référence à une courbe d'animation.
asset_sequence Le nom donné fait référence à une Séquence.

Si la fonction réussit et qu'une ou plusieurs des balises sont présentes pour le bien, elle renvoie true; sinon, elle renvoie false. Si vous avez besoin de vérifier une correspondance précise avec une balise ou un ensemble de balises, utilisez la fonction asset_has_tags().

Syntaxe :

asset_has_any_tag(name_or_index, tags, [asset_type]);

Arguments Type Description
name_or_index Chaîne ou Actif The name of the asset (a string) or its index value (an integer).
tags Chaîne de caractères ou Tableau de chaînes de caractères A single asset tag string or an array with various asset tags.
[asset_type] Type d'actif Constant OPTIONAL! The type of asset to check the tags for, only used when supplying an index for the first argument.

Les retours :

Boolean

Exemple :

var _a = array_create(3);
_a[0] = "boss";
_a[1] = "main_boss";
_a[2] = "final_boss";

if asset_has_any_tag(object_index, _a, asset_object)
{
    instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

Le code ci-dessus crée un tableau de balises et vérifie ensuite si l'une d'entre elles est appliquée à l'objet donné. Si c'est le cas, il crée une instance d'un autre objet.
