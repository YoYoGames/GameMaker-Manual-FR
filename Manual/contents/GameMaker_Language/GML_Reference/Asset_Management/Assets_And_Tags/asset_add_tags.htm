asset_add_tags

Cette fonction permet d'ajouter une ou plusieurs chaînes de balises à n'importe quel poste du navigateur de postes. Vous fournissez soit le nom du bien (sous forme de chaîne), soit son index, ainsi qu'une seule chaîne de balises ou un tableau où chaque élément est une seule chaîne de balises. Si vous fournissez une valeur d'index d'asset, vous devrez fournir l'argument facultatif de type d'asset (une constante), car des assets de types différents peuvent avoir le même index, même s'ils ne peuvent pas avoir le même nom. Les types de biens disponibles sont répertoriés dans le tableau ci-dessous :

Constant Description
asset_object Le nom donné fait référence à un objet.
asset_sprite Le nom donné fait référence à un sprite.
asset_sound Le prénom fait référence à un son.
asset_room Le prénom fait référence à un room.
asset_tiles Le nom donné fait référence à un ensemble de carreaux.
asset_path Le nom donné fait référence à un chemin.
asset_script Le prénom fait référence à un script.
asset_font Le nom donné fait référence à une police de caractères.
asset_timeline Le prénom fait référence à une ligne de temps.
asset_shader Le nom donné fait référence à un shader.
asset_animationcurve Le nom donné fait référence à une courbe d'animation.
asset_sequence Le nom donné fait référence à une Séquence.

Si la fonction réussit à ajouter la ou les étiquettes, elle renvoie true; sinon, elle renvoie false.

Syntaxe :

asset_add_tags(name_or_index, tags, [asset_type]);

Arguments Type Description
name_or_index Chaîne ou actif The name of the asset (a string) or its index value (an integer).
tags Chaîne ou tableau de chaînes de caractères A single asset tag string or an array with various asset tags.
[asset_type] Type d'actif Constant OPTIONAL Le type de poste sur lequel les balises doivent être placées, utilisé uniquement lorsqu'un index est fourni pour le premier argument.

Les retours :

Boolean

Exemple :

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_add_tags(obj_Enemy_Boss_Parent, _a, asset_object);

Le code ci-dessus va créer un tableau de balises et les ajouter à l'objet donné.
