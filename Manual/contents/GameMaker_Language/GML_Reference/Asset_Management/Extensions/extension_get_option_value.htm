extension_get_option_value

Cette fonction renvoie la valeur de l'option donnée dans l'extension portant le nom donné.

La fonction renvoie undefined si le nom d'extension fourni n'est pas valide.

Syntaxe :

extension_get_option_value(extension_name, option_name);

Arguments Type Description
extension_name Chaîne de caractères The name of your extension asset as a string
option_name Chaîne de caractères The name of the option to read

Les retours :

Any

Exemple :

var _enabled = extension_get_option_value("MyExtension", "enabled");

Ce code récupère la valeur de l'option enabled dans l'extension MyExtension.
