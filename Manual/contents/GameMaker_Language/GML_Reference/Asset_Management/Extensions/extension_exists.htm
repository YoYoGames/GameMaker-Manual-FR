extension_exists

Cette fonction indique si une extension portant le nom donné existe(true) ou non(false).

Syntaxe :

extension_exists(extension_name);

Arguments Type Description
extension_name Chaîne de caractères The name of your extension asset as a string

Les retours :

Boolean

Exemple :

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

Ce code vérifie si l'extension MyExtension existe, et si c'est le cas, il récupère ses options sous la forme d'un struct, et lit une valeur à partir de celui-ci.
