extension_get_options

Cette fonction renvoie une struct contenant toutes les options, et leurs valeurs, pour l'extension portant le nom donné.

La fonction renvoie undefined si le nom d'extension fourni n'est pas valide.

Syntaxe :

extension_get_options(extension_name);

Arguments Type Description
extension_name Chaîne de caractères The name of your extension asset as a string

Les retours :

Struct

Exemple :

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

Ce code récupère les options struct pour une extension et y lit une valeur.
