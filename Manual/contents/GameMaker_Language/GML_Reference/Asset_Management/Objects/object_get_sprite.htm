object_get_sprite

Cette fonction vous dira si l'objet que vous vérifiez a un sprite ou non, et si c'est le cas, elle retournera l'index de ce sprite, ou -1 si ce n'est pas le cas. Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez avoir un objet sans sprite alors qu'une instance de ce même objet peut en avoir un et vice-versa, ou ils peuvent même avoir des sprites différents. Vous pouvez définir une instance individuelle sprite à l'aide de la variable sprite_index variable instance.

Syntaxe :

object_get_sprite(obj);

ArgumentsType Description
obj The index of the object to check

Les retours :

Sprite Asset

Exemple :

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

L'exemple ci-dessus va vérifier le sprite_index de l'instance par rapport au sprite du object_index de l'instance. S'ils ne sont pas identiques, il attribuera à l'instance le même sprite que celui de l'index de l'objet.
