Objet de l'événement

Cette variable en lecture seule renvoie l'index de l'objet de l'instance qui exécute l'événement à vérifier.

Syntaxe :

event_object;

Les retours :

Object Asset

Exemple :

global.obj = event_object;

Le code ci-dessus stocke l'index de l'objet de l'instance qui réalise l'événement dans une variable globale.
