action_événement DEPRECATED

Cette variable en lecture seule renvoie l'indice de l'action en cours d'exécution, en commençant par 0 sur les versions précédentes de GameMaker. Cependant, cette variable est désormais obsolète dans GameMaker. Elle n'a été conservée qu'à des fins d'archivage et renvoie toujours 0, car toutes les actions sont concaténées pour améliorer la vitesse d'exécution.

Syntaxe :

event_action;

Les retours :

Real

Exemple :

num = event_action;

Le code ci-dessus stocke le numéro de l'action en cours dans la variable "num".
