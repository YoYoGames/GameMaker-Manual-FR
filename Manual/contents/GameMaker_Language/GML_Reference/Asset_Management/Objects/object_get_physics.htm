objet_get_physique

Cette fonction vous dira si l'objet que vous vérifiez a été marqué comme "physics enabled" - auquel cas elle renverra true, - ou non - auquel cas elle renverra false.

Syntaxe :

object_get_physics(obj);

ArgumentsType Description
obj The index of the object to check.

Les retours :

Exemple :

if object_get_physics(object_index)
{
    phy_active = true;
}

Le code ci-dessus vérifie si l'objet à partir duquel il est créé est compatible avec la physique et, si c'est le cas, il active la simulation physique pour l'instance.
