objet_get_solid

Cette fonction vous indiquera si l'objet que vous vérifiez a été marqué comme "solide" ou non. Un objet solide génère un événement de collision spécial lorsque vous utilisez le système de collision traditionnel (c'est-à-dire que le monde physique est désactivé). Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Ainsi, vous pouvez avoir un objet solide et une instance normale du même objet et vice-versa. Vous pouvez définir l'indicateur de solidité d'une instance individuelle en utilisant la variable solid d'une instance à l'aide d'une variable d'instance.

Syntaxe :

object_get_solid(obj);

ArgumentsType Description
obj The index of the object to check.

Les retours :

Exemple :

if (!solid) && (object_get_solid(object_index))
{
    solid = true;
}

Le code ci-dessus vérifiera l'instance en cours d'exécution pour voir si elle est solide ou non, ainsi que l'index de l'objet de l'instance pour voir s'il est marqué comme solide ou non. Si l'instance n' est pas solide mais que l'index de l'objet est marqué comme solide, il mettra "solid" à true pour cette instance.
