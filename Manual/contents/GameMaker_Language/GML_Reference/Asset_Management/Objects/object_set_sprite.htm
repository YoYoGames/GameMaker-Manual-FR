objet_set_sprite

Cette fonction vous permet de définir l'indice sprite d'un objet spécifique. Cela signifie que toutes les instances de cet objet qui sont créées après que l'indice de sprite a été modifié seront créées avec ce nouvel indice de sprite, tandis que les instances qui sont déjà dans la pièce peuvent ne pas être affectées.

Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez définir l'indice sprite des instances individuelles en utilisant la variable sprite_index et ainsi avoir dix instances ayant toutes un sprite différent de l'objet à partir duquel elles sont créées, et même si vous changez l'indice sprite de l'objet en utilisant cette fonction, toutes les instances qui sont actuellement dans la room resteront telles quelles, et seules les instances créées après l'appel de la fonction commenceront avec le nouveau sprite.

Syntaxe :

object_set_sprite( index, spr );

Arguments Type Description
index Objet Asset The index of the object to change.
spr Sprite Asset The sprite to assign to the object.

Les retours :

N/A

Exemple :

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

Le code ci-dessus définit l'index de l'objet "obj_Player" sprite à différentes valeurs en fonction de la pièce dans laquelle se trouve l'instance qui exécute le code. Il crée ensuite une instance de "obj_Player".
