object_get_visible

Cette fonction vous dira si l'objet que vous vérifiez a été marqué comme "visible" (exécute son événement de dessin) ou non (n'exécute pas son événement de dessin). Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Ainsi, vous pouvez avoir un objet visible et une instance invisible du même objet et vice-versa. Vous pouvez définir la visibilité d'une instance individuelle en utilisant la variable visible instance.

Syntaxe :

object_get_visible(obj);

ArgumentsType Description
obj The index of the object to check.

Les retours :

Exemple :

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

Le code ci-dessus vérifiera l'instance en cours d'exécution pour voir si elle est visible ou non, ainsi que l'index de l'objet de l'instance pour voir si elle est marquée comme visible ou non. Si l'instance n' est pas visible mais que l'index de l'objet est marqué comme visible, le code mettra "visible" à true pour cette instance.
