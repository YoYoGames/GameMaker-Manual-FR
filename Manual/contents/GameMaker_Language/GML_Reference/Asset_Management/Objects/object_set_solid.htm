objet_set_solid

Cette fonction vous permet de marquer un objet spécifique comme étant solide ou non (pour plus d'informations sur le drapeau "solid", voir la section sur l'éditeur d'objets). Cela signifie que toutes les instances de cet objet qui sont créées après que solid ait été modifié seront créées avec ce nouvel état, alors que les instances qui sont déjà dans la pièce peuvent ne pas être affectées.

Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez définir l'indicateur de solidité d'instances individuelles à l'aide de la variable solid et ainsi avoir dix instances toutes marquées comme solides même si l'objet en question a l'indicateur défini comme faux dans la variable IDE, et même si vous changez l'indicateur de solidité de l'objet en vrai à l'aide de cette fonction, toutes les instances qui se trouvent actuellement dans la pièce resteront telles quelles, et seules les instances créées après l'appel de la fonction commenceront avec l'indicateur de solidité en vrai.

Syntaxe :

object_set_solid( index, solid );

Arguments Type Description
index Objet Asset The index of the object to change.
solid Booléen The new solidity of the object (true=solide, false=non solide).

Les retours :

N/A

Exemple :

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

Le code ci-dessus vérifie l'indicateur de solidité de l'objet "obj_Block" et s'il est faux, il est changé en indicateur de solidité.
