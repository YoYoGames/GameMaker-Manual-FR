object_get_name

Cette fonction renvoie le nom de l'objet spécifié sous la forme d'une chaîne de caractères. Ce nom est celui qui a été spécifié pour l'objet dans l'Asset Browser de la fenêtre principale de GameMaker. Veuillez noter qu'il s'agit uniquement d'une chaîne de caractères et ne peut pas être utilisé pour référencer l'objet directement - pour cela, vous aurez besoin de l'index de l'objet. Vous pouvez toutefois utiliser cette chaîne pour obtenir l'index de l'objet en utilisant la chaîne retournée avec la fonction asset_get_index().

Syntaxe :

object_get_name(obj);

ArgumentsType Description
obj The index of the object to check.

Les retours :

Exemple :

str = object_get_name(object_index);

Le code ci-dessus obtiendra le nom de l'index de l'objet pour l'instance qui exécute le code et stockera la valeur de retour dans la variable "str".
