objet_existant

Cette fonction indique si la valeur spécifiée est un objet (true) ou non (false). Notez que cette fonction vérifie si un objet est présent dans le navigateur d'objets, et non dans la salle de jeu. Pour cela, vous devez utiliser la fonction instance_exists().

Syntaxe :

object_exists(obj);

ArgumentsType Description
obj The index of the object to check.

Les retours :

Exemple :

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

Le code ci-dessus lit les données d'un buffer qui a été reçu via le réseau. Comme il contient un identifiant d'objet (utilisé pour la création d'instances), nous devons d'abord nous assurer que la valeur n'est pas invalide et que l'objet existe réellement. Pour cela, nous appelons object_exists() dans une condition, et si c'est vrai, nous créons une instance de cet objet à la position reçue.
