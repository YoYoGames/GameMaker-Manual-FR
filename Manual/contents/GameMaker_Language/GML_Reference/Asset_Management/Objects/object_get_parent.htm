objet_get_parent

Cette fonction vous donnera l'indice d'objet de tout parent qui a été assigné à l'objet spécifié, ou retournera -100 pour montrer que l'objet n'a pas de parent assigné, ou -1 si l'objet vérifié n'existe pas. Pour plus d'informations sur les parents, voir la section sur l'éditeur d'objets.

Syntaxe :

object_get_parent(obj);

ArgumentsType Description
obj The index of the object to check

Les retours :

Object Asset

Exemple :

par = object_get_parent(object_index);

Le code ci-dessus récupère le parent de l'index de l'objet pour l'instance qui exécute le code et stocke la valeur de retour dans la variable "par".
