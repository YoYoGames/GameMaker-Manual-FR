alarm_set

Cette fonction peut être utilisée pour définir une alarme. Vous fournissez le numéro d'alarme de 0 à 11, puis la valeur à laquelle l'alarme doit être réglée. La valeur doit être un nombre entier, et vous pouvez la définir à -1 pour arrêter l'alarme (les valeurs non entières seront arrondies au nombre entier le plus proche). Il s'agit d'une méthode alternative au réglage direct du tableau d'alarme.

Syntaxe :

alarm_set(index, value);

Arguments Type Description
index Real The alarm index to set, from 0 to 11.
value Real The value (an integer) to set the alarm to.

Les retours :

N/A

Exemple :

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

Le code ci-dessus vérifie toutes les alarmes de l'instance appelante et si elles sont supérieures à 0, il leur attribue la valeur -1, les empêchant ainsi de poursuivre leur décompte.
