instance_activate_layer

Cette fonction vous permet d'activer une couche qui a été désactivée précédemment. Vous devez fournir l'ID de la couche, qui peut être soit le nom de la couche tel qu'il a été écrit dans l'éditeur de code (sous forme de chaîne), soit la valeur réelle de l'ID de la couche telle qu'elle est renvoyée par la fonction layer_create() et toutes les instances désactivées sur cette couche seront à nouveau activées. Notez que si vous avez désactivé une couche qui a des instances persistantes, vous devrez réactiver la couche à nouveau avec cette fonction avant de changer de pièce, sinon toutes les instances persistantes ne seront pas reportées et seront rejetées. Notez également que l'activation n'est pas instantanée, et qu'une instance qui a été activée de cette manière ne sera pas considérée comme active avant la fin de l'événement dans lequel la fonction a été appelée.

Syntaxe :

instance_activate_layer(layer_id);

Arguments Type Description
layer_id ID ou chaîne de caractères dela couche The layer name string (or ID value) to be used

Les retours :

N/A

Exemple :

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

Le code ci-dessus active toutes les instances situées dans la pièce, puis désactive celles qui se trouvent en dehors des limites de la vue actuelle de la caméra, à l'exception des instances du calque "Player_Layer" qui sont réactivées à la fin.

Retour : Désactiver les instances
