instance_désactivation_couche

Avec cette fonction, vous pouvez désactiver toutes les instances assignées à une couche spécifique. Vous devez fournir l'ID de la couche, qui peut être soit le nom de la couche tel qu'il est écrit dans l'éditeur de code (sous forme de chaîne), soit la valeur réelle de l'ID de la couche telle qu'elle est retournée par la fonction layer_create()Notez que vous ne pouvez désactiver que les couches d'instance avec cette fonction. Notez que si vous avez désactivé une couche dont les instances d'objets sont marquées comme persistantes, vous devrez réactiver la couche à l'aide de la fonction instance_activate_layer() avant de changer de pièce, sinon toutes les instances persistantes de la couche ne seront pas reportées et seront rejetées. Notez également que la désactivation n'est pas instantanée, et qu'une instance qui a été désactivée de cette manière ne sera pas considérée comme inactive avant la fin de l'événement dans lequel la fonction a été appelée.

NOTE: Si vous désactivez une instance au démarrage de la room (c'est-à-dire à partir du code de création de la room, ou à partir d'un événement de création d'une instance dans la room), toutes les instances qui sont placées dans la room à partir de l'éditeur de room exécuteront toujours leur événement de création avant d'être désactivées.

AVERTISSEMENT: La désactivation des instances dont la physique est activée n' empêchera PAS leurs appareils d'interagir dans la simulation physique. Pour cela, vous devez définir leur variable phy_active à true ou false lorsque vous activez/désactivez les instances.

Syntaxe :

instance_deactivate_layer(obj);

Arguments Type Description
layer ID ou chaîne de caractères dela couche The layer name string (or ID value) to be used

Les retours :

N/A

Exemple :

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Le code ci-dessus désactive toutes les instances affectées à la couche "Enemy_Layer", puis active une région dans la pièce.

Retour : Désactiver les instances
