instance_deactivate_region

Cette fonction vous permet de définir une région dans la salle pour désactiver les instances qui ont été précédemment activées. Cette région peut être marquée comme "intérieure" ou "extérieure", comme le montre l'image suivante :

Vous pouvez voir dans l'image ci-dessus que l'instance "apple" est toujours inactive car, même si le site sprite lui-même ne chevauche pas la région, la bounding box le fait. Ainsi, les instances sont considérées comme étant dans la région spécifiée lorsque leur boîte englobante la recouvre, et l'état du masque de collision (précis ou non) n' est pas pris en considération. Notez que la désactivation n'est pas instantanée, et une instance qui a été désactivée de cette manière ne sera pas considérée comme inactive avant la fin de l'événement dans lequel la fonction a été appelée.

NOTE: Si vous désactivez une instance au démarrage de la room (c'est-à-dire à partir du code de création de la room, ou à partir d'un événement de création d'une instance dans la room), toutes les instances qui sont placées dans la room à partir de l'éditeur de room exécuteront toujours leur événement de création avant d'être désactivées.

AVERTISSEMENT: La désactivation des instances dont la physique est activée n' empêchera PAS leurs appareils d'interagir dans la simulation physique. Pour cela, vous devez définir leur variable phy_active à true ou false lorsque vous activez/désactivez les instances.

Syntaxe :

instance_deactivate_region(left, top, width, height, inside, notme);

Arguments Type Description
left Real The x coordinate of the left of the rectangular region to deactivate.
top Real The y coordinate of the top of the rectangular region to deactivate.
width Real The width of the region to deactivate.
height Real The height of the region to deactivate.
inside Booléen Whether to deactivate instances on the inside of the region (true) or the outside (false).
notme Booléen Whether to exclude the calling instance from deactivation (true) or not (false).
collision_space Collision Space  The collision space where instances will be deactivated

Les retours :

N/A

Exemple :

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

Le code ci-dessus active toutes les instances, puis désactive une région dans la pièce.
