instance_activate_all

Avec cette fonction, vous pouvez demander à GameMaker d'activer toutes les instances qui ont été précédemment désactivées dans une pièce. Notez que l'activation n'est pas instantanée, et qu'une instance qui a été activée de cette manière ne sera pas considérée comme active avant la fin de l'événement dans lequel la fonction a été appelée. active avant la fin de l'événement dans lequel la fonction a été appelée.

Syntaxe :

instance_activate_all();

Les retours :

Exemple :

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

Le code ci-dessus active toutes les instances situées dans la pièce, puis désactive celles qui se trouvent à 64px en dehors des limites de la vue actuelle de la caméra.

Retour : Désactiver les instances
