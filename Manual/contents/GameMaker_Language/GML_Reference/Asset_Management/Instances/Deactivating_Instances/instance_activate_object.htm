instance_activate_object

Cette fonction vous permet d'activer une seule instance ou toutes les instances d'un objet spécifique parmi toutes celles qui ont été désactivées précédemment. Notez que si vous avez désactivé une instance ou un objet qui a été marqué comme Persistant, vous devrez le réactiver à nouveau avec cette fonction avant de changer de salle. vous devrez le réactiver à nouveau avec cette fonction avant de changer de salle, sinon il ne sera pas reporté et sera rejeté. Notez également que l'activation n'est pas instantanée, et qu'une instance qui a été activée de cette manière ne sera pas considérée comme active. ne sera pas considérée comme active avant la fin de l'événement dans lequel la fonction a été appelée.

Syntaxe :

instance_activate_object(obj);

ArgumentsType Description
obj The object or instance to activate (the keyword all can also be used).

Les retours :

Exemple :

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);
instance_activate_object(obj_Lights);

Le code ci-dessus active toutes les instances situées dans la pièce, puis désactive celles qui se trouvent en dehors des limites de la vue actuelle de la caméra, à l'exception de l'objet "obj_Lights" qui est réactivé à la fin.
