instance_position

Cette fonction vous permet de vérifier qu'une position n'entre pas en collision avec une autre instance ou avec toutes les instances d'un objet. Lorsque vous utilisez cette fonction, vous vérifiez un point unique dans la room pour une instance ou un objet. Cette vérification sera faite par rapport à la boîte englobante de l'instance ou par rapport au masque de l'instance si cette instance a des collisions précises vérifiées et retournera l'instance unique. id. Si vous n'avez pas besoin de l'identifiant de l'instance en collision, vous devriez envisager d'utiliser plutôt position_meeting() à la place. Cette fonction accepte également les mots-clés spéciaux all et other et retournera le mot-clé noone si aucune collision ne se produit ou la valeur d'ID unique de l'instance trouvée si une collision se produit.

Syntaxe :

instance_position( x, y, obj );

Arguments Type Description
x Real The x position to check for instances.
y Real The y position to check for instances.
obj ID de l'actif ou de l'instance del'objet The object to check for instances of.

Les retours :

Instance ID or noone

Exemple :

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst)
    {
        image_index = 1;
    }
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

Le code ci-dessus vérifiera s'il y a une collision avec une instance de "obj_Pause_Button" à la position de la souris, et s'il y en a une, il utilisera l'id retourné pour définir son image_index à une nouvelle valeur avant de créer une nouvelle instance de l'objet "obj_Menu".
