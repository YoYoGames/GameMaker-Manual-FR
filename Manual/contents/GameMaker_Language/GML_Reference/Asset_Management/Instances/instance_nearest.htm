instance_nearest

Cette fonction va vérifier toutes les instances de l'objet donné pour voir laquelle est la plus proche du point x/y donné. Toutes les vérifications se feront de la position x/y donnée à l'origine (la position x/y) des instances de l'objet spécifié. Si aucune instance de l'objet n'existe, la fonction renvoie le mot-clé noonemais s'il existe des instances, elle renverra le mot clé id de l'instance trouvée. Veuillez noter que si l'instance qui exécute le code a le même indice d'objet que l'objet vérifié, elle sera incluse dans la vérification (cela inclut la vérification des objets parents si l'instance appelante est également un enfant du parent).

Syntaxe :

instance_nearest(x, y, obj);

ArgumentsType Description
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

Les retours :

Instance ID or noone

Exemple :

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if (inst != id)
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

Le code ci-dessus déplace l'instance actuelle de 10000 pixels puis vérifie sa position précédente pour trouver l'instance la plus proche du même type d'objet. Si cette instance est elle-même, elle ne fera rien d'autre que de revenir à sa position initiale, mais si l'instance trouvée est différente, elle tracera une ligne entre les deux.
