identifiant de l'instance

Ce tableau en lecture seule contient tous les id s de chaque instance active dans la salle. Cela signifie que si vous avez utilisé l'une des fonctions Instance Deactivate, les instances qui ont été désactivées ne seront pas incluses dans ce tableau (si vous avez utilisé une valeur de ce tableau précédemment, il retournera maintenant le mot-clé noone).

Syntaxe :

instance_id[num];

Les retours :

Instance ID

Exemple :

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

Le code ci-dessus passera en boucle toutes les instances de la pièce et ajoutera 0.1 à leur vitesse.
