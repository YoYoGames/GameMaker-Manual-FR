hspeed

hspeed est l'une des propriétés "intégrées" que toutes les instances possèdent et définit la vitesse de déplacement horizontal (le long de l'axe des x) de l'instance en pixels par pas. Ainsi, un hspeed de 3 signifie un mouvement de 3 pixels vers la droite (+x) à chaque step. (+x) à chaque pas, et une hspeed de -3 signifie un mouvement de 3 pixels vers la gauche (-x) à chaque pas.

Notez que si vous définissez les variables speed et/ou direction, la valeur hspeed sera automatiquement mise à jour pour refléter ces changements. refléter ces changements, et de même, la modification de la valeur hspeed affectera également les valeurs speed et direction. Notez également que gravity, gravity_direction et friction peuvent tous modifier la valeur de cette variable lorsqu'ils sont utilisés dans vos jeux.

Syntaxe :

hspeed;

Les retours :

(single precision floating point value)

Exemple :

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

Le code ci-dessus changera la vitesse horizontale en fonction des touches pressées.
