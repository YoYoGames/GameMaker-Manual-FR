gravity

gravity est l'une des variables "intégrées" que possèdent toutes les instances et, lorsqu'elle est définie, elle appliquera une force constante dans l'instance. gravity_direction de l'instance, influençant à la fois l'instance speed et direction. Notez que la gravité est une force cumulative et accélérera l'objet si vous choisissez de ne pas plafonner la vitesse finale, et il est habituel de régler cette variable sur de petites valeurs décimales comme 0.01. Si vous fixez la valeur de gravity à 0, alors aucune gravité ne sera appliquée à l'instance (c'est la valeur par défaut).

Syntaxe :

gravity

Les retours :

(single precision floating point value)

Exemple :

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

Le code ci-dessus n'appliquera la gravité que si l'instance ne trouve aucune instance de "obj_Ground" sous elle.
