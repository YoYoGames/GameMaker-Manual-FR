visible

Une instance peut être marquée comme visible ou non en donnant à cette variable la valeur true (visible) ou false (invisible). Cela fonctionne en disant à GameMaker d'ignorer l'événement de dessin pour cette instance, il faut donc faire attention en utilisant cette variable car cela signifie qu'aucun code placé dans l'événement de dessin ne sera exécuté tant que l'instance n'est pas visible. Notez également que si l'objet layer à laquelle l'instance est assignée est marquée comme invisible, le réglage de cette variable à true n'aura aucun effet jusqu'à ce que la couche elle-même soit également marquée comme visible.

Syntaxe :

visible;

Les retours :

Exemple :

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

Le code ci-dessus vérifiera le drapeau visible de l'"autre" instance lors d'un événement de collision et, s'il est défini sur true, ramènera cette instance à sa position précédente.
