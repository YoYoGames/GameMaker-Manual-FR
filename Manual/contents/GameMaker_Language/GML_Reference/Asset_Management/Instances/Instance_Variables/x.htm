x

La valeur x d'une instance est la position horizontale dans la pièce actuelle, mesurée en pixels. Cette valeur peut être soit 0, positive ou négative, où 0 est le côté gauche de la room et où un déplacement vers la droite augmente x, un déplacement vers la gauche diminue x (une valeur négative pour x signifie que l'instance est sortie du côté gauche de la room). négative pour x signifie que l'instance est sortie du côté gauche de la room). Vous pouvez même donner à x une valeur réelle comme 12.345, mais comme vous ne pouvez pas obtenir 0.345 d'un pixel, l'instance peut sembler ne pas avoir bougé, bien que ces valeurs soient maintenues et utilisées en interne par GameMaker et par le système de gestion de l'image. et utilisées en interne par GameMaker et sont parfaitement valides.

En définissant les valeurs x (et y), vous pouvez faire sauter l'objet dans la pièce jusqu'à la position de votre choix, ou vous pouvez ajouter et soustraire de plus petites quantités pour donner l'illusion d'un mouvement sans réellement utiliser les fonctions intégrées speed et direction intégrées. Vous devez en tenir compte lorsque vous créez vos jeux, car souvent, un type de mouvement se prête mieux à un style de jeu particulier qu'à un autre.

Syntaxe :

x

Les retours :

(single precision floating point value)

Exemple :

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

Le code ci-dessus utilisera les touches fléchées pour ajouter ou soustraire à la position x et ainsi créer l'illusion d'un mouvement.
