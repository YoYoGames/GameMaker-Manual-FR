xprevious

Cette variable intégrée renvoie la position x précédente de l'instance. Cette variable sera définie juste avant le début de l'événement begin step mais elle peut également être définie par le code à tout moment, ce qui signifie que vous pouvez lui donner votre propre valeur personnalisée si nécessaire.

Syntaxe :

xprevious

Les retours :

(single precision floating point value)

Exemple :

if x != xprevious || y != yprevious
{
    moved = true;
}

Le code ci-dessus vérifie s'il existe une différence entre les valeurs xprevious et yprevious et les valeurs x et y actuelles. Si c'est le cas, il définit une variable à true.
