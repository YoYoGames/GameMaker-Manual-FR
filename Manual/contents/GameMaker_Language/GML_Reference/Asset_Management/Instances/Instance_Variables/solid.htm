solide

Une instance peut être marquée comme solide par le biais des propriétés de l'objet dans l'éditeur d'objets, ou en modifiant la valeur de cette variable intégrée. Si solid a pour valeur true, lorsqu'une collision est détectée, l'instance qui entre en collision est automatiquement ramenée à la position qu'elle occupait lors de l'étape précédant la collision (et tout code ou action est alors exécuté dans l'événement de collision). S'il est défini sur false, tous les positionnements doivent être traités par l'événement de collision.

Syntaxe :

solid;

Les retours :

Exemple :

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

Le code ci-dessus vérifiera l'indicateur de solidité de l'"autre" instance dans une collision et s'il est défini sur true, il ramènera cette instance à sa position précédente.
