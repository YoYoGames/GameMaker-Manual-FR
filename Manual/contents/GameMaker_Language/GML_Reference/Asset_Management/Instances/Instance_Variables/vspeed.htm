vspeed

vspeed est l'une des propriétés "intégrées" que toutes les instances possèdent et définit la vitesse de déplacement vertical (le long de l'axe des y) de l'instance en pixels par pas. Ainsi, un vspeed de 3 signifie un mouvement de 3 pixels vers le bas (+y) à chaque step. (+y) à chaque pas, et une vspeed de -3 signifie un mouvement de 3 pixels vers le haut (-y) à chaque pas.

Notez que si vous définissez les variables speed et/ou direction, la valeur vspeed sera automatiquement mise à jour pour refléter ces changements. refléter ces changements, et de même, la modification de la valeur vspeed affectera également les valeurs speed et direction. Notez également que gravity, gravity_direction et friction peuvent tous modifier la valeur de cette variable lorsqu'ils sont utilisés dans vos jeux.

Syntaxe :

vspeed;

Les retours :

(single precision floating point value)

Exemple :

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

Le code ci-dessus changera la vitesse verticale en fonction des touches pressées.
