liste_de_lieux_instances

Avec cette fonction, vous pouvez vérifier la présence d'une collision entre une position et toutes les instances d'un objet en utilisant le masque de collision de l'instance qui exécute le code de vérification. Lorsque vous utilisez cette fonction, vous demandez effectivement à GameMaker de déplacer l'instance vers la nouvelle position, de vérifier s'il y a une collision, puis de revenir en arrière et de vous dire si une collision a été trouvée ou non. Cela fonctionne pour les collisions précises, mais seulement si l'instance et les instances de l'objet à vérifier ont des masques de collision précis sélectionnés, sinon seules les collisions de la boîte englobante sont appliquées.

Vous fournissez également une liste de DS, de sorte que les id de toutes les instances qui entrent en collision avec l'instance appelante seront ajoutées à la fin de la liste donnée. Vous pouvez effacer la liste avant d'appeler cette fonction afin qu'elle ne contienne que les résultats de cet appel de fonction. Vous avez également la possibilité d'ordonner la liste en fonction de la distance entre l'origine de l'instance effectuant la vérification et l'origine des instances trouvées en collision.

Notez que la fonction accepte également le mot clé spécial alldans ce cas, toutes les instances trouvées en collision seront listées. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée.

Syntaxe :

instance_place_list(x, y, obj, list, ordered);

Arguments Type Description
x Real The x position to check for instances.
y Real The y position to check for instances.
obj ID de l'actif ou de l'instance del'objet The object to check for instances of.
list ID de la liste DS The DS list to use to store the IDs of colliding instances.
ordered Booléen Whether the list should be ordered by distance (true) ou non (false).

Les retours :

Real (The number of instances found to be in collision)

Exemple :

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);

if (_num > 0)
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}

ds_list_destroy(_list);

Le code ci-dessus vérifiera la présence d'une collision avec toutes les instances trouvées à la position de l'instance appelante. Celles-ci seront ajoutées à une liste DS, qui sera ensuite parcourue en boucle pour détruire chacune des instances en collision.
