numéro d'instance

Cette fonction vous permet de savoir combien d'instances actives de l'objet spécifié existent dans la salle. Lors de la vérification à l'aide de cette fonction, si l'objet est un parent, tous les objets enfants seront également inclus dans la valeur de retour. Notez également que les instances qui ont été désactivées avec les fonctions de désactivation d'instance ne seront pas incluses dans cette vérification.

Syntaxe :

instance_number(obj);

ArgumentsType Description
obj The object to total the number of instances of.

Les retours :

Exemple :

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

Le code ci-dessus vérifie le nombre d'instances créées à partir du même objet que l'instance actuelle et, s'il y en a moins de 50, il en crée une autre à un endroit aléatoire de la pièce.
