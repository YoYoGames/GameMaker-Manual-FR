liste_slot_squelette

Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des slots créés dans le cadre de l'animation du squelette sprite. Les noms seront des chaînes de caractères et pourront ensuite être utilisés dans les autres fonctions de slot de l'animation du squelette pour ces types de sprite.

Syntaxe :

skeleton_slot_list(sprite, list);

Arguments Type Description
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the list from.
list ID de la liste DS The ID of the DS list to populate with the bone names.

Les retours :

N/A

Exemple :

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

Le code ci-dessus crée une liste de DS puis la remplit avec les noms de slot pour une utilisation ultérieure.
