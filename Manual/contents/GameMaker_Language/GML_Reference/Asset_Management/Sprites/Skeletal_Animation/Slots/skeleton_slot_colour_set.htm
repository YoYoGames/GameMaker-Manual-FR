jeu de couleurs pour les fentes du squelette

Avec cette fonction, vous pouvez définir la couleur et l'alpha d'un slot d'attachement de sorte que tous les sprites qui lui sont attachés seront dessinés avec ces valeurs de mélange. Gardez à l'esprit que l'instance sprite peut avoir une couleur de fusion et un paramètre alpha (image_blend et image_alpha), tout comme l'attachement (voir la fonction skeleton_attachment_create_colour()), et donc la couleur et l'alpha finaux que l'attachement sprite assigné à l'emplacement aura seront une combinaison de toutes ces valeurs.

Syntaxe :

skeleton_slot_colour_set(slot, colour, alpha);

Arguments Type Description
slot Chaîne de caractères The slot name to set, a string
colour Couleur The colour to set, either an integer, a constant, or a hex value.
alpha Real The alpha to set from 0 to 1.

Les retours :

N/A

Exemple :

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Le code ci-dessus crée une liste de DS et la remplit avec les noms des emplacements trouvés à la position de la souris dans la pièce. Il boucle ensuite sur la liste des emplacements et réinitialise la couleur des emplacements trouvés en blanc avec un alpha de 1 si la couleur n'est pas déjà blanche.
