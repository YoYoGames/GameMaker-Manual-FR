skeleton_find_slot

Avec cette fonction, vous pouvez trouver quels créneaux se trouvent à une position pièce-espace spécifiée dans le Spine sprite associé à l'instance actuelle. Vous créez une liste de DS et fournissez son ID ainsi qu'une position x/y à vérifier et la liste sera remplie avec une chaîne de noms pour chacun des emplacements d'attachement disponibles pour le sprite (y compris toute modification d'attachement). Notez que la liste est toujours triée par ordre décroissant en partant de l'emplacement le plus haut.

Syntaxe :

skeleton_find_slot(x, y, list);

Arguments Type Description
x Real The x position in the room to check.
y Real The y position in the room to check.
list ID de la liste DS The ID of the DS list to populate with the DS maps.

Les retours :

N/A

Exemple :

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

Le code ci-dessus crée une liste de DS et la remplit ensuite avec les noms des emplacements trouvés à la position de la souris dans la pièce.
