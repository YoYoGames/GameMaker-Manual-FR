squelette_skin_set

Une animation squelettique sprite peut être dessinée avec une variété de textures (appelées "skins"), ce qui vous permet d'utiliser une seule animation pour de multiples usages différents. Par exemple, vous pouvez avoir une animation de marche générique et celle-ci peut être utilisée par vos personnages NPC, vos ennemis, etc... dans un jeu simplement en changeant le skin avec lequel elle est dessinée. Ces peaux doivent avoir été préalablement créées et nommées dans votre programme d'animation, et ce nom est transmis à la fonction (sous forme de chaîne) pour que le site sprite utilise cette peau lorsqu'il est dessiné.

Syntaxe :

skeleton_skin_set(skinname);

ArgumentsType Description
skinname The name (a string) of the skin to use.

Les retours :

Exemple :

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Le code ci-dessus vérifie le skin actuellement attribué à l'animation et s'il s'agit de "skin_Enemy1", il choisit et définit un nouveau skin parmi trois options.
