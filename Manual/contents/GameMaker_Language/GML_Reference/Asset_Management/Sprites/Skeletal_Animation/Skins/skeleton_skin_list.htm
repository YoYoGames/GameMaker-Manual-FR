skeleton_skin_list

Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des skins inclus dans l'animation du squelette sprite. Les noms seront des chaînes de caractères et pourront ensuite être utilisés dans les autres fonctions de skin de l'animation du squelette pour ces types de sprite.

Syntaxe :

skeleton_skin_list(sprite, list);

Arguments Type Description
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the list from.
list ID de la liste DS The ID of the DS list to populate with the animation names.

Les retours :

N/A

Exemple :

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Le code ci-dessus crée une liste DS puis la remplit avec les noms de skin. Une peau aléatoire est ensuite choisie et appliquée à l'instance avant que la liste ne soit détruite.
