Bones

Comme son nom l'indique, l'animation squelette sprites se compose d'une série d'"os", sur lesquels une peau est ajoutée pour créer l'animation finale du personnage. Normalement, vous souhaitez configurer ces os dans le logiciel d'animation (Spine), puis exporter le fichier sprite avec un ensemble d'animations déjà prédéfinies afin de pouvoir passer de l'une à l'autre selon vos besoins. Cependant, il arrive que vous souhaitiez intercepter les données des os et les manipuler manuellement ou obtenir des informations sur la position d'os spécifiques, et dans ce cas, vous pouvez utiliser les fonctions suivantes :

  • skeleton_bone_data_get
  • skeleton_bone_data_set
  • skeleton_bone_state_get
  • skeleton_bone_state_set
  • skeleton_bone_list
