squelette_attachement_créer

Lorsque vous créez votre animation squelettique sprite, vous pouvez attribuer des emplacements d'attachement et des attachements à y placer. Il s'agit simplement d'images qui ne font pas partie de l'animation mais qui, lorsqu'elles sont attachées, se déplacent avec l'os auquel elles sont attachées. Normalement, vous attribuez ces pièces jointes dans votre programme d'animation (Spine), mais cette fonction vous permet de créer les vôtres au moment de l'exécution à l'aide d'un actif sprite de votre jeu. La fonction requiert que vous donniez un nom à la pièce jointe (sous forme de chaîne), puis que vous définissiez le sprite_index et l'image_index à utiliser, ainsi que l'origine x et y (qui peut être différente de celle définie par sprite dans les propriétés sprite ). Vous pouvez également définir les transformations que vous souhaitez appliquer à l'image lorsqu'elle est jointe. Si la pièce jointe a été créée avec succès, la fonction renvoie 1. Dans le cas contraire (vous avez fourni un index sprite invalide ou la base sprite n'est pas un Spine sprite), elle renvoie -1.

Syntaxe :

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

Arguments Type Description
name Chaîne de caractères The name (as a string) of the attachment to create.
sprite Sprite Asset The sprite_index to get the attachment image from.
ind Real The image_index to get the attachment image from.
xorigin Real The x origin for the image being used.
yorigin Real The y origin for the image being used.
xscale Real The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the image. 0=normal, 90=turned 90 degrees counter-clockwise etc.

Les retours :

Real (1 if successful, -1 if not)

Exemple :

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

Le code ci-dessus vérifie le nom de l'accessoire actuellement attribué au slot nommé "slot_leftHand" et si une chaîne vide est renvoyée, un nouveau sprite est attaché.
