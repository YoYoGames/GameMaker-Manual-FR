attachement_squelette_get

Une animation squelettique sprite peut avoir d'autres sprites ajoutés en tant qu'attachements, ces sprites étant ajoutés à un slot nommé (le nom est donné lorsque vous créez le slot d'attachement dans votre programme d'animation) et ils seront dessinés avec l'animation du sprite actuel. Avec cette fonction, vous pouvez obtenir le nom (sous forme de chaîne) de l'accessoire pour le slot donné du sprite actuellement attribué. Notez que les pièces jointes sprites sont référencées par leur chaîne de nom telle qu'elle a été assignée dans Spine, ou lorsque vous avez appelé la fonction skeleton_attachment_create().

Syntaxe :

skeleton_attachment_get(slot);

Arguments Type Description
slot Chaîne de caractères The slot name (a string) to get the attachment of.

Les retours :

String

Exemple :

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Le code ci-dessus vérifie le nom de l'accessoire actuellement attribué au slot nommé "slot_leftHand" et si une chaîne vide est renvoyée, un nouveau sprite est attaché.
