dessiner_squelette

Cette fonction ne doit être utilisée qu' avec sprites qui a été créé à l'aide d'un programme d'animation de squelettes comme Spine. Bien que vous puissiez dessiner ces sprites à l'aide des fonctions de dessin normales (telles que draw_self()), il peut arriver que vous souhaitiez dessiner une seule image ou montrer un changement de texture de peau sans modifier les propriétés de sprite. Dans ce cas, vous pouvez dessiner sprite à l'aide de cette fonction, où vous indiquez l'index de sprite, puis le nom de l'animation dont vous souhaitez obtenir l'image (une chaîne de caractères, telle que définie dans le programme utilisé pour créer sprite). Ensuite, vous donnez le nom du skin à utiliser (encore une fois, tel qu'il a été défini lors de la création de sprite ), et une valeur d'image. La valeur de l'image sert à indiquer à GameMaker quelle partie de l'animation doit être affichée et est une valeur comprise entre 0 et la valeur image_number. Les autres arguments sont les mêmes que ceux utilisés pour le dessin normal de sprite et modifieront l'échelle, l'angle, la couleur de fusion et l'alpha de l'image d'animation finale dessinée.

ATTENTION ! En raison de la façon dont les animations squelettiques sont interpolées entre les différentes images "clés", il y a une surcharge supplémentaire associée à cet appel et il est donc recommandé d'éviter de dessiner des animations basées sur des squelettes sprites en utilisant cette méthode, sauf en cas de nécessité absolue.

Syntaxe :

draw_skeleton(sprite, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Arguments Type Description
sprite Sprite Asset The index of the sprite to draw.
animname Chaîne de caractères The name of the animation to get the frame from (a string).
skinname Chaîne de caractères The name of the skin to use (a string).
frame Real The animation frame to draw (from 0 to image_number - 1).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Couleur The colour with which to blend the sprite.
alpha Real The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Les retours :

N/A

Exemple :

draw_skeleton(spr_Jump, "jump", "skin1", 0, x, y, image_xscale, image_yscale, image_angle, c_white, 0.5);

Le code ci-dessus dessinera la première image de l'animation squelettique donnée sprite, en utilisant l'ensemble d'animation "jump", le skin défini comme "skin1" et à la même échelle et position que les propriétés de l'instance définissent, mais avec un alpha de 0,5.
