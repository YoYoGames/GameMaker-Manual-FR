draw_skeleton_time

Cette fonction va dessiner l'animation donnée en utilisant le skin donné à un moment précis de l'animation. La valeur du temps doit être comprise entre 0 (le début) et la durée finale de l'animation, que vous pouvez trouver en utilisant la fonction skeleton_animation_get_duration(). Vous pouvez définir la valeur du temps à des valeurs supérieures à la durée totale de l'animation et l'animation reviendra en boucle au début, mais vous courez le risque de perdre la précision de la virgule flottante à mesure que le temps accumulé augmente.

Syntaxe :

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour)

Arguments Type Description
sprite Sprite Asset The index of the sprite to draw.
animname Chaîne de caractères The name of the animation to get the frame from (a string).
skinname Chaîne de caractères The name of the skin to use for drawing (a string).
time Real The time-frame to draw the animation at (from 0 to the end duration, in seconds).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Couleur The colour with which to blend the sprite.

Les retours :

N/A

Exemple :

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Le code ci-dessus dessine l'animation squelettique donnée sprite en utilisant le delta-time pour définir l'image à dessiner.
