draw_enable_skeleton_blendmodes

Cette fonction est utilisée pour activer(true) ou désactiver(false) les modes de fusion par fente pour le squelette sprites.

Syntaxe :

draw_enable_skeleton_blendmodes(enable)

Arguments Type Description
enable Booléen Whether to enable or disable blend modes

Les retours :

N/A

Exemple :

draw_enable_skeleton_blendmodes(true);

Le code ci-dessus permet d'activer les modes de fusion par fente pour le squelette sprites.

Retour : Dessin et divers
