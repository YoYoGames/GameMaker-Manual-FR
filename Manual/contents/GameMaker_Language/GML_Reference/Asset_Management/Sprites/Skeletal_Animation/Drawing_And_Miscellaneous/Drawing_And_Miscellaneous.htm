Dessin et divers

Les fonctions suivantes sont des outils supplémentaires pour obtenir différents ensembles de données sur un sprite d'animation de squelette :

  • skeleton_get_minmax
  • skeleton_get_num_bounds
  • skeleton_get_bounds

Vous pouvez également utiliser diverses fonctions de dessin conçues spécifiquement pour ces types de sprite (outre les fonctions ordinaires draw_sprite() ordinaires). Ces fonctions vous donnent un plus grand contrôle sur ce qui est dessiné et vous permettent de modifier des caractéristiques spécifiques de l'animation, telles que le timing et les peaux :

  • skeleton_collision_draw_set
  • draw_skeleton
  • draw_skeleton_instance
  • draw_skeleton_collision
  • draw_skeleton_time
  • draw_enable_skeleton_blendmodes
  • draw_get_enable_skeleton_blendmodes
  • draw_enable_skeleton_blend_override
  • draw_get_enable_skeleton_blend_override
