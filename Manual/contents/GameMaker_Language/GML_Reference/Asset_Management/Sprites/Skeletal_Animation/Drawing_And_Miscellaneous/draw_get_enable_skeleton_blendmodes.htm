draw_get_enable_skeleton_blendmodes

Cette fonction indique si les modes de fusion par tranche pour le squelette sprites sont activés(true) ou désactivés(false).

Syntaxe :

draw_get_enable_skeleton_blendmodes()

Les retours :

Boolean

Exemple :

if (!draw_get_enable_skeleton_blendmodes())
{
    draw_enable_skeleton_blendmodes(true);
}

Le code ci-dessus active les modes de fusion par fente pour le squelette sprites, s'ils sont désactivés.
