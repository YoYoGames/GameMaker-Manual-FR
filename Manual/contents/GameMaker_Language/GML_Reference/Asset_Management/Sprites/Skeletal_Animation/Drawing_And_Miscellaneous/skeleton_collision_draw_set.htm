squelette_collision_draw_set

Cette fonction vous permet d'activer (true) ou de désactiver (false) l'affichage des données de collision pour l'animation squelettique sprite utilisée par l'instance. Si cette fonction est activée, la boîte englobante et le masque de collision précis seront dessinés comme des contours autour de sprite.

Cette fonction ne fonctionne que lorsque le dessin du squelette sprite est géré par l'objet. Pour dessiner la boîte de délimitation des collisions lors du dessin manuel d'un squelette sprite, utilisez draw_skeleton_collision().

Syntaxe :

skeleton_collision_draw_set(flag);

Arguments Type Description
flag Booléen Set to true pour activer le dessin, et false pour le désactiver.

Les retours :

N/A

Exemple :

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

Le code ci-dessus vérifie si le jeu est exécuté en mode débogage et si c'est le cas, il bascule les données de collision du squelette pour l'instance à afficher.
