animation_squelette_get_duration

Cette fonction renvoie le temps nécessaire à l'exécution de l'animation donnée avant de revenir au début. La valeur de retour est exprimée en secondes.

Syntaxe :

skeleton_animation_get_duration(animname);

ArgumentsType Description
animname The name (a string) of the animation set to use.

Les retours :

Exemple1 :

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

Le code ci-dessus définira le image_index à la valeur correcte pour l'animation squelettique actuellement assignée sprite.

Exemple2 :

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Le code ci-dessus dessine l'animation squelettique donnée sprite en utilisant le delta-time pour définir l'image à dessiner.
