animation_squelette_est_looping

Avec cette fonction, vous pouvez vérifier si l'animation en cours sur la piste donnée est configurée en boucle.

Syntaxe :

skeleton_animation_is_looping(track);

Arguments Type Description
piste Real L'index de la piste d'animation

Les retours :

Boolean

Exemple :

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

Le code ci-dessus vérifie si l'animation de la piste 0 est configurée en boucle. Si c'est le cas, il affiche un message de débogage.
