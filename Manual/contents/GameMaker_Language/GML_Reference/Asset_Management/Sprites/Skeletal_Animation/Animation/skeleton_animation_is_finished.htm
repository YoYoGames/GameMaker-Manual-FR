animation_squelette_est_finie

Avec cette fonction, vous pouvez vérifier si l'animation squelettique non-bouclée en cours a fini de jouer sur la piste donnée.

Syntaxe :

skeleton_animation_is_finished(track);

Arguments Type de données Description
piste Real L'index de la piste d'animation

Les retours :

Boolean

Exemple :

var _finished = skeleton_animation_is_finished(1);
if (_finished)
{
    show_debug_message("The animation has finished playing");
}

Le code ci-dessus vérifie d'abord si l'animation de la piste 1 a fini de jouer et enregistre le résultat dans une variable locale _finished. Il affiche ensuite un message de débogage au cas où la lecture de l'animation serait terminée.
