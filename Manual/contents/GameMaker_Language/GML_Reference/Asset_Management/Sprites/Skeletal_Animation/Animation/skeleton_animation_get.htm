animation_squelette_get

Avec cette fonction, vous pouvez obtenir le jeu d'animation actuel utilisé par votre animation squelettique sprite. La valeur de retour est une chaîne de caractères, qui sera le nom du jeu tel que vous l'avez défini dans votre programme d'animation squelettique.

Syntaxe :

skeleton_animation_get();

Les retours :

Exemple :

if keyboard_check_pressed(vk_space)
{
    if (skeleton_animation_get() != "jump")
    {
        skeleton_animation_set("jump");
    }
}

Le code ci-dessus changera le jeu d'animation utilisé en jeu de "saut" lorsque la touche espace sera pressée, mais seulement si le jeu actuel utilisé n'est pas déjà "saut".
