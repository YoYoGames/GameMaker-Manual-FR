squelette_animation_set_position

Cette fonction règle l'animation assignée à la piste donnée sur la position que vous fournissez. La position doit être une valeur réelle comprise entre 0 et 1. La position doit être une valeur réelle comprise entre 0 et 1. Si vous fournissez une valeur en dehors de cette plage, la position de la piste sera fixée à la partie fractionnaire de cette valeur. Supposons que vous utilisiez une valeur de 0,4, une valeur de 1,4, une valeur de 2,4, ... Toutes ces valeurs régleront la position sur 0,4.

Syntaxe :

skeleton_animation_set_position(track, position);

Arguments Type Description
track Real The animation track to set the frame of.
position Real The new position for the animation frame.

Les retours :

N/A

Exemple :

skeleton_animation_set_position(1, 0.25);

Le code ci-dessus définira la position de la piste 1 de l'animation du squelette à 0,25 ou, autrement dit, à 25% du début de l'animation.
