animation_squelette_get_ext

Une seule animation squelettique sprite peut avoir plusieurs jeux d'animation, et ces jeux peuvent être assignés à différentes pistes afin que vous puissiez "mélanger et assortir" les jeux d'animation. Cette fonction renvoie le nom du jeu d'animation actuellement utilisé par le numéro de piste donné (tel que défini par la fonction skeleton_animation_set_ext).

Syntaxe :

skeleton_animation_get_ext(track);

Arguments Type Description
track Real The track number to get the animation set name of.

Les retours :

String

Exemple :

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

Le code ci-dessus changera le jeu d'animation utilisé par la piste 1 en "bodyfight" s'il ne l'est pas déjà.
