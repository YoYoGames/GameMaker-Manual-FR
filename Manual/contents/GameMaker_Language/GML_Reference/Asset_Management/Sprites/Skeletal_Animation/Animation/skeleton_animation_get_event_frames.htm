animation_squelette_get_event_frames

Cette fonction peut être utilisée pour récupérer toutes les images de l'événement donné, dans l'animation donnée. Vous fournissez le nom du squelette de l'animation (sous forme de chaîne, tel que défini dans le programme utilisé pour créer l'animation, ou tel que renvoyé par la fonction skeleton_animation_get()) et un nom d'événement. La fonction renvoie un tableau contenant les images de cet événement.

Si le nom de l'événement spécifié n'existe pas, la fonction renvoie un tableau dont le premier (et seul) élément est -1.

Syntaxe :

skeleton_animation_get_event_frames(anim_name, event_name);

Arguments Type Description
anim_name Chaîne de caractères The animation name to get the frames of.
event_name Chaîne de caractères The event name to get the frames for.

Les retours :

Array of Reals

Exemple :

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

Le code ci-dessus récupère les images pour l'événement "Footstep". S'il renvoie un tableau avec les images valides, il le parcourt en boucle et vérifie si l'image actuelle est égale à l'une des images du tableau. Dans ce cas, il joue l'effet sonore "Footstep" et interrompt la boucle.
