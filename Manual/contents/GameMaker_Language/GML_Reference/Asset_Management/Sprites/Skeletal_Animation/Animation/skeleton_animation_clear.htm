animation_squelette_claire

Cette fonction vide la piste d'animation spécifiée de toutes les animations, prêtes à être réaffectées.

Syntaxe :

skeleton_animation_clear(track);

Arguments Type Description
track Real The animation track to clear.
reset Boolean  Whether to reset the skeleton to its setup pose. Default is false.
duration Real  How long animating the skeleton to setup pose should take, in seconds.

Les retours :

N/A

Exemple :

if mouse_check_button(mb_right)
{
    skeleton_animation_clear(1);
}

Le code ci-dessus effacera la piste 1 de l'animation si le bouton droit de la souris est enfoncé.
