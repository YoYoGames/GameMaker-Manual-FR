liste d'animation du squelette

Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des animations incluses dans l'animation du squelette sprite. Les noms seront des chaînes de caractères et pourront ensuite être utilisés dans les autres fonctions d'animation pour ces types de sprite.

Syntaxe :

skeleton_animation_list(sprite, list);

Arguments Type Description
sprite Sprite Asset The sprite index of the Spine skeletal animation to get the list from.
list ID de la liste DS The ID of the DS list to populate with the animation names.

Les retours :

N/A

Exemple :

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++;)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

Le code ci-dessus crée une liste DS de tous les noms d'animation pour le site sprite utilisé par l'instance. Il les parcourt ensuite en boucle et définit la valeur de mixage pour toutes les animations avec la première animation à 0.5.
