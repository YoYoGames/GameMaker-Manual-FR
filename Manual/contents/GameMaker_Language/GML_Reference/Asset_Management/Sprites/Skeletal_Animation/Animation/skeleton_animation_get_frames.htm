skeleton_animation_get_frames

Cette fonction peut être utilisée pour récupérer le nombre d'images d'une animation squelette donnée. Vous fournissez le nom de l'animation squelette (sous forme de chaîne, telle que définie dans le programme utilisé pour créer l'animation, ou telle que renvoyée par la fonction skeleton_animation_get()) et la fonction renvoie le nombre d'images qu'elle possède sous forme de valeur entière. La fonction renvoie 0 si l'animation spécifiée n'existe pas.

Syntaxe :

skeleton_animation_get_frames(anim_name);

Arguments Type Description
anim_name Chaîne de caractères The animation name to get the frames of.

Les retours :

Real

Exemple :

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

Le code ci-dessus obtiendra le nombre d'images dans l'animation, puis fixera sprite à la dernière image et arrêtera l'animation.
