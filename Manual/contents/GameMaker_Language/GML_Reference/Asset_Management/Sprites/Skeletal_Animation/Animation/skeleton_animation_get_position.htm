animation_squelette_get_position

Cette fonction renvoie la position de l'animation sur la piste d'animation spécifiée sous la forme d'une valeur allant de 0 à 1. De cette façon, vous pouvez obtenir la position de la piste sous la forme d'un pourcentage où la valeur 0 correspond à 0% et la valeur 1 à 100%.

Elle retournera -1 si aucune animation n'est assignée à la piste spécifique donnée ou si l'instance n'a pas de sprite défini.

Syntaxe :

skeleton_animation_get_position(track);

Arguments Type Description
track Real The animation track to get the position of.

Les retours :

Real

Exemple :

var _position = skeleton_animation_get_position(0);
var _position_percent = _position * 100;
show_debug_message("The skeleton animation position on track 0 is at " + string(_position_percent) + "%.");

Le code ci-dessus vérifie la position actuelle de l'animation du squelette affectée à la piste 0 et la stocke dans une variable temporaire _position. Il convertit ensuite cette valeur en pourcentage en la multipliant par 100 et enregistre le résultat dans une autre variable temporaire _position_percent.
Enfin, il affiche un message de débogage qui indique la valeur en pourcentage.
