Animation

Lorsque vous travaillez avec une animation squelettique sprites créée à l'aide de Spine, vous pouvez avoir plusieurs animations par site, et vous devez donc pouvoir passer de l'une à l'autre aux moments appropriés de votre jeu. Pour vous aider à le faire, GameMaker propose les fonctions suivantes :

  • skeleton_animation_get
  • skeleton_animation_set
  • skeleton_animation_get_ext
  • skeleton_animation_set_ext
  • skeleton_animation_get_duration
  • skeleton_animation_mix
  • skeleton_animation_list
  • skeleton_animation_clear
  • skeleton_animation_get_frames
  • skeleton_animation_get_frame
  • skeleton_animation_set_frame
  • animation_squelette_get_event_frames
  • animation_squelette_est_looping
  • animation_squelette_est_finie
