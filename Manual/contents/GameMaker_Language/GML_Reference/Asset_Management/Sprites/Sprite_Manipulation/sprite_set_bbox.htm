sprite_set_bbox

Cette fonction peut être utilisée pour définir les valeurs du rectangle de délimitation d'un sprite. Vous fournissez l'index sprite à utiliser, puis les valeurs gauche, haut, droite et bas pour les positions du rectangle de délimitation. Les positions sont des valeurs absolues, où la position (0, 0) correspond au coin supérieur gauche de sprite, quel que soit le décalage de sprite, les pixels "vides" que sprite peut avoir, ou l'endroit où il est dessiné dans la pièce.

REMARQUE Cette fonction affecte le poste sprite de sorte que toutes les autres instances de ce poste sprite auront la même boîte de délimitation.

Syntaxe :

sprite_set_bbox(ind, left, top, right, bottom);

ArgumentsType Description
ind The index of the sprite to set the bounding box on.
left The left side of the bounding box
top The top of the bounding box.
right The right side of the bounding box
bottom The bottom of the bounding box.

Renvoie à

Exemple :

if sprite_get_bbox_mode(sprite_index) == 2
{
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
}

Le code ci-dessus vérifie le mode de boîte englobante de sprite affecté à sprite_index et, s'il est défini sur manual, sa boîte englobante est modifiée.
