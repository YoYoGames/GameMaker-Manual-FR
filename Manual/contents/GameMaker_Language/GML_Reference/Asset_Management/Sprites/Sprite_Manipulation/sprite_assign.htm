sprite_assign

Cette fonction prend deux index sprite précédemment créés (ou inclus), et copie l'image de l'un à l'autre. De cette façon, vous pouvez copier (ou "cloner") un sprite dans un autre index. Notez que vous ne pouvez pas copier vers une ressource de jeu. Vous devez avoir créé le sprite à copier au préalable à l'aide de la commande sprite_add() ou sprite_duplicate() pour le copier.

Syntaxe :

sprite_assign(index, sprite);

ArgumentsType Description
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

Renvoie à

Exemple :

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

Le code ci-dessus vérifie si la variable globale "Player_Sprite" contient un sprite et si c'est le cas, il utilise sprite_assign pour le remplacer par un autre qui a été chargé depuis un fichier externe. Si ce n'est pas le cas, un sprite est chargé et son index est stocké dans cette variable.
