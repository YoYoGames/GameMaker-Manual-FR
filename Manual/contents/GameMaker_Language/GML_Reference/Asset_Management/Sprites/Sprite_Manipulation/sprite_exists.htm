sprite_existe

Cette fonction retourne si un sprite avec l'index spécifié existe ou non dans le projet en cours d'exécution.

Syntaxe :

sprite_exists(index);

ArgumentsType Description
index The index of the sprite to check.

Renvoie à

Exemple :

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

Le code ci-dessus vérifie un tableau pour voir s'il contient un indice sprite valide et si c'est le cas, il attribue ce sprite à l'instance, mais si ce n'est pas le cas, il attribue un sprite à partir des ressources incluses.
