sprite_set_offset

Cette fonction peut être utilisée pour définir l'origine x et y d'un site sprite et prend des valeurs relatives basées sur la position (0,0), qui est le coin supérieur gauche du site sprite. L'image suivante illustre cela :

NOTE Cette fonction affecte l'actif du sprite de sorte que toutes les autres instances de ce sprite auront le même décalage.

Syntaxe :

sprite_set_offset(ind, xoff, yoff);

ArgumentsType Description
ind The index of the sprite to change the offset of.
xoff The x position of the origin.
yoff The y position of the origin.

Renvoie à

Exemple :

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

Le code ci-dessus affecte le sprite indexé dans "spr_Base" au sprite indexé dans "spr_Custom" et utilise ensuite les valeurs de décalage x et y de "spr_Base" pour définir l'origine du nouveau sprite.
