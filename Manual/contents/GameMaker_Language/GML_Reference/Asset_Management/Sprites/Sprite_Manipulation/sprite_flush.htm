sprite_flush

Avec cette fonction, vous pouvez supprimer de la mémoire de texture la page de texture donnée pour le site sprite donné. Vous fournissez l'index sprite (tel que défini lors de la création de la ressource sprite ) et la page de texture à laquelle il est assigné sera retirée de la mémoire de texture. Notez que si la page de texture est utilisée ailleurs dans la room (par une autre instance sprite ou un arrière-plan, etc...), vous risquez d'avoir un léger impact sur les performances car la page est immédiatement rechargée, il faut donc faire attention en utilisant cette fonction. Notez que la fonction retournera -1 si le flush n'est pas supporté par la ressource choisie, ou bien 0 si tout a fonctionné correctement.

Syntaxe :

sprite_flush(ind)

ArgumentsType Description
ind The index (resource name) of the sprite asset to flush

Les retours :

(-1 or 0)

Exemple :

sprite_flush(spr_Player_Aura);

Le code ci-dessus vide de mémoire le site sprite "spr_Player_Aura".
