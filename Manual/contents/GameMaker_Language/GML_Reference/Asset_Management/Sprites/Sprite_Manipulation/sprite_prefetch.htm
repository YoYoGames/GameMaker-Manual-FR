sprite_prefetch

Cette fonction peut être utilisée pour préextraire (placer dans la mémoire de texture) une page de texture avec l'adresse sprite donnée. Vous fournissez l'index sprite tel que défini lors de la création de l'actif sprite et la page de texture sur laquelle il se trouve sera chargée en mémoire. Notez que la fonction renverra -1 si la préextraction n'est pas prise en charge par la ressource choisie ou si la plate-forme cible est HTML5, ou elle renverra 0 si tout a fonctionné correctement.

REMARQUE: Il y a un impact sur les performances lorsque la texture est téléchargée dans la mémoire de texture sur la plupart des périphériques, il est donc recommandé d'appeler sprite_prefetch sur tout graphique requis au début d'un niveau pour éviter tout blocage.

Syntaxe :

sprite_prefetch(ind)

ArgumentsType Description
ind The sprite index to fetch

Les retours :

(-1 or 0)

Exemple :

sprite_prefetch(spr_Player_Aura);

Le code ci-dessus placera le site sprite référencé dans la mémoire de texture, prêt à être utilisé.
