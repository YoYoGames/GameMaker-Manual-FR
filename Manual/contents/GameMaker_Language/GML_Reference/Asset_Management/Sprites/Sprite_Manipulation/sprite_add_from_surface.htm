sprite_add_from_surface

Cette fonction fonctionne exactement de la même manière que sprite_create_from_surface() sauf qu'au lieu de créer un nouveau sprite à partir de la zone de la surface indexée que vous sélectionnez, elle ajoute la zone définie de la surface en tant que nouvelle sous-image à un sprite précédemment créé (cela signifie que vous ne pouvez pas l'ajouter directement à un sprite à partir de l'arbre des ressources, mais seulement à un précédemment créé à partir d'une surface, ou un qui a été dupliqué à partir d'une ressource sprite à l'aide de sprite_duplicate()).

Il convient de noter que la mémoire utilisée lors de l'ajout d'un sprite de cette manière sera plus importante que vous ne le pensez. En effet, GameMaker stockera le site sprite en tant que page de texture et il sera également stocké dans la mémoire du GPU. La mémoire totale sera donc plus importante que celle d'un fichier image du même site sprite.

Syntaxe :

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

Arguments Type Description
index Sprite Asset The index of the sprite to add the new image to.
surface ID de la surface The index of the surface from which we get the image.
x Real The x position to copy from.
y Real The y position to copy from.
w Real The width of the area to be copied (from the x position).
h Real The height of the area to be copied (from the y position).
removeback Booléen Indicates whether to make all pixels with the background colour (left-bottom pixel) transparent.
smooth Booléen Indicates whether to smooth the edges.

Les retours :

N/A

Exemple :

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i += 1)
{
    sprite_add_from_surface(spr_custom, surf, i, 0, 32, 32, false, false);
}

Le code ci-dessus crée un sprite à partir de la surface indexée dans la variable "surf", en affectant son index à la variable "spr_Custom", puis utilise une boucle for pour se déplacer sur la surface et capturer diverses sections qui sont ajoutées dans le sprite comme sous-images.
