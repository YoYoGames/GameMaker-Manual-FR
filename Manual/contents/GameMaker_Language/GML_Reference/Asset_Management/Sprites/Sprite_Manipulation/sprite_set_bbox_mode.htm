sprite_set_bbox_mode

Cette fonction peut être utilisée pour définir le mode de boîte englobante d'une sprite. Vous fournissez l'index de sprite et le mode à utiliser, qui doit être l'une des constantes suivantes :

Mode de la boîte englobante Constante
Constant Description
bboxmode_automatic Automatique - La boîte de délimitation sera calculée automatiquement, sur la base du paramètre de tolérance du sprite.
bboxmode_fullimage Image complète - La boîte de délimitation sera définie de manière à utiliser la largeur et la hauteur complètes du site sprite, sans tenir compte de la tolérance et des pixels "vides".
bboxmode_manual Manuel - La boîte englobante est définie manuellement sur des valeurs définies par l'utilisateur, ce qui nécessite l'utilisation de la fonction sprite_set_bbox()

REMARQUE Cette fonction affecte l'actif du sprite de sorte que toutes les autres instances de ce sprite auront le même mode de boîte englobante.

Syntaxe :

sprite_set_bbox_mode(ind, mode);

Arguments Type Description
ind Sprite Asset The index of the sprite to change the mode of.
mode Mode de la boîte englobante Constante The mode to set (a constant).

Renvoie à

N/A

Exemple :

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Le code ci-dessus vérifie le mode bbox pour le sprite actuel et s'il n'est pas automatic, il est défini à cette valeur.
