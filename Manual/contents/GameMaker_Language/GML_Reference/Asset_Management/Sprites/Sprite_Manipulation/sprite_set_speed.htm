sprite_set_speed

Cette fonction peut être utilisée pour définir la vitesse et le type d'animation de base pour la ressource sprite. Ces paramètres sont normalement définis dans l'éditeur de sprites, mais il peut arriver que vous deviez les définir manuellement pour une ressource sprite donnée. Vous devez fournir l'index sprite à définir, la vitesse d'animation et le type de vitesse d'animation à appliquer. Le type de vitesse d'animation peut être l'une des constantes suivantes :

Constant Description
spritespeed_framespersecond Le site sprite a été défini avec une animation en images par seconde.
spritespeed_framespergameframe Le site sprite a été défini avec dans l'animation en frames par frame de jeu.


La sélection du type de vitesse à utiliser pour calculer les animations est très importante car il existe une énorme différence entre les deux options. Les deux exemples suivants illustrent cette différence :

  • Si vous avez une sprite qui dessine 1 image par seconde et que vous réglez la vitesse de l'image sur 0,5, elle dessinera à 0,5 image par seconde. Si la fréquence d'images de votre jeu est de 60 images par seconde, le site sprite affichera une image toutes les 120 images du jeu.
  • Si vous avez un sprite qui dessine 1 image par image de jeu et que vous réglez la vitesse de l'image sur 0,5, il dessinera 0,5 image par image de jeu. Si la fréquence d'images de votre jeu est de 60 images par seconde, le site sprite affichera 30 images par 60 images de jeu.

Vous pouvez trouver la valeur de la vitesse utilisée pour l'animation sprite en utilisant la fonction sprite_get_speed()et vous pouvez obtenir le type de vitesse de l'animation à l'aide de la fonction sprite_get_speed_type().

IMPORTANT ! Cette fonction affectera la ressource sprite. Ainsi, toute modification apportée par cette fonction affectera toutes les instances qui utilisent le site sprite et toutes les instances futures également.

Syntaxe :

sprite_set_speed(index, speed, type)

Arguments Type Description
index Sprite Asset The index of the sprite to set the animation speed of
speed Real The speed value to set
type Constante de vitesse du sprite The speed type to use

Les retours :

Constant (see above)

Exemple :

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Le code ci-dessus définit la vitesse d'animation de base pour le site sprite attribué à l'instance à 10 images par seconde.
