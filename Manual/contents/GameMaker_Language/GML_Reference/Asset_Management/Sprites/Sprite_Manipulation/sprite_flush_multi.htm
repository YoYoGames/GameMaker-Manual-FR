sprite_flush_multi

Avec cette fonction, vous pouvez supprimer de la mémoire de texture un nombre quelconque de pages de texture pour l'adresse sprites donnée. Vous fournissez les indices sprite sous forme de tableau et les pages de texture auxquelles ils sont assignés seront retirées de la mémoire de texture. Notez que si l'une des pages de texture est utilisée ailleurs dans la room (par une autre instance sprite ou un arrière-plan, etc...), vous risquez d'avoir un léger impact sur les performances car la page est immédiatement rechargée en mémoire, il faut donc faire attention en utilisant cette fonction. Notez que la fonction renverra -1 si le flush n'est pas supporté pour les ressources choisies, ou elle renverra 0 si tout a fonctionné correctement.

Syntaxe :

sprite_flush_multi(array);

ArgumentsType Description
array Array with the sprite indices to flush

Les retours :

(-1 or 0)

Exemple :

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

Le code ci-dessus crée un tableau dont chaque élément contient un index sprite. Ce tableau est ensuite utilisé pour effacer ces textures sprite de la mémoire.
