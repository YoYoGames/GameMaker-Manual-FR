sprite_set_nineslice

Cette fonction est utilisée pour appliquer une structure en neuf tranches à une adresse sprite. Vous fournissez l'index sprite et la structure en neuf tranches struct à appliquer, qui peut être récupérée à l'aide de la commande sprite_nineslice_create() ou sprite_get_nineslice().

NOTE Cette fonction affecte l'actif du sprite. Ainsi, toute modification apportée à l'aide de cette fonction affectera toutes les instances qui utilisent ce site sprite et toutes les instances futures également.

Syntaxe :

sprite_set_nineslice(ind, nineslice);

Arguments Type Description
ind Sprite Asset The index of the sprite to modify
nineslice Structures à neuf tranches The Nine Slice struct to apply

Les retours :

N/A

Exemple :

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Le code ci-dessus crée une nouvelle tranche struct, l'active et définit les décalages pour les guides. L'adresse struct est ensuite appliquée à une adresse sprite, ce qui modifie ses propriétés de neuf tranches.
