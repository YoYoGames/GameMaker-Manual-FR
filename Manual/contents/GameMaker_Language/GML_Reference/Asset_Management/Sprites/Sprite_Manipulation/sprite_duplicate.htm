sprite_duplicate

Cette fonction renvoie l'index d'un nouveau site sprite qui est un double (copie) de celui qui est entré comme argument "index". Si la fonction échoue, -1 est renvoyé. Cette fonction doit être utilisée pour copier n'importe quel sprites à partir des actifs d'origine avant que des transformations puissent être effectuées sur eux. Par exemple, si vous souhaitez modifier le rectangle de délimitation d'un sprite, ou définir son alpha à partir d'un autre sprite, vous devez d'abord le dupliquer, puis effectuer l'opération sur le sprite dupliqué et l'utiliser. Un sprite dupliqué sera placé sur sa propre page de texture unique lors de sa création, ce qui signifie que la duplication de plusieurs sprites créera de multiples pages de texture et aura un impact sur les performances, donc n'utilisez cette fonction que si nécessaire.

REMARQUE: Lorsque vous dupliquez un sprite dans GameMaker, vous devez vous souvenir de le supprimer à nouveau (avec la touche sprite_delete()) lorsqu'il n'est plus nécessaire, sinon il y a un risque de fuite de mémoire qui ralentira et finira par faire planter votre jeu.

Syntaxe :

sprite_duplicate(index);

ArgumentsType Description
index The index of the sprite to duplicate.

Renvoie à

Sprite Asset or -1

Exemple :

new_spr = sprite_duplicate(sprite_index)

Le code ci-dessus duplique l'adresse sprite actuellement utilisée comme indice sprite de l'instance et stocke l'indice de cette nouvelle sprite dans la variable "new_spr".
