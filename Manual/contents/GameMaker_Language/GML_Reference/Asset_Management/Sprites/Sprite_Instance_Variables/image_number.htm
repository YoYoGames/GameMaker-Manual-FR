numéro de l'image

Cette variable en lecture seule peut être utilisée pour obtenir le nombre de sous-images dans une sprite qui a été assignée à une instance (si vous avez besoin du nombre de sous-images pour une sprite autre que celle assignée à l'instance, vous devez utiliser la variable sprite_get_number()). Veuillez noter que lorsqu'il y a (par exemple) 1 sous-image, cette variable renvoie "1" mais l'image_index de cette sous-image est 0.

Syntaxe :

image_number;

Les retours :

(single precision floating point value)

Exemple :

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

Le code ci-dessus vérifie si le site sprite est animé, et si c'est le cas, il vérifie ensuite si l'image_index actuelle est supérieure au nombre de sous-images et si c'est le cas, il détruit l'instance.
