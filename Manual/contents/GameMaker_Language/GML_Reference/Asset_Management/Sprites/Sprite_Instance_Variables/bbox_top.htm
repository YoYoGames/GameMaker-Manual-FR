bbox_top

Cette variable en lecture seule renvoie la position dans la room (le long de l'axe des y) du haut du rectangle de délimitation de l'instance, où le rectangle de délimitation est défini par la largeur et la hauteur maximales du masque de l'instance (tel que défini par sprite_index ou par mask_index). Même si un sprite a un masque de collision précis, la boîte englobante existe et est utilisée pour certaines choses, et vous pouvez donc utiliser cette variable pour la trouver. Veuillez noter que lorsque l'instance n'a pas de sprite assigné, la valeur retournée sera la même que la position Y de l'instance.

Syntaxe :

bbox_top;

Les retours :

(integer)

Exemple :

if bbox_top < 0
{
    y = sprite_yoffset;
}

Le code ci-dessus vérifie si la boîte englobante de l'instance est en dehors de la pièce et si c'est le cas, il place l'instance à une nouvelle position.
