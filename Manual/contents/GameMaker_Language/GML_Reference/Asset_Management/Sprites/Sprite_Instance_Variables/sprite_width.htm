largeur de sprite

Cette variable en lecture seule renvoie la largeur du site sprite qui a été attribué à l'instance. Cette largeur est renvoyée en pixels et dépendra de la valeur de la variable image_xscale. Si vous avez besoin de la largeur non mise à l'échelle, vous devez utiliser la variable sprite_get_width().

Syntaxe :

sprite_width;

Les retours :

Real

Exemple :

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

Le code ci-dessus vérifie la largeur de la ressource sprite attribuée à l'instance qui exécute le code par rapport à la largeur de la ressource sprite et si elle n'est pas la même, il réinitialise la ressource image_xscale à 1,
