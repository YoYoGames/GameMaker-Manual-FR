angle_image

Cette valeur définit l'angle (rotation) de sprite et est mesurée en degrés, la droite étant à 0º, le haut à 90º, la gauche à 180º et le bas à 270º. Mettez cette variable à 0 pour réinitialiser le dessin de sprite tel qu'il a été défini dans l'éditeur de sprite. Veuillez noter que pour que les changements de cette variable soient visibles, l'instance ne doit pas avoir d'événement de dessin (et donc GameMaker dessinera par défaut le sprite) ou être dessinée à l'aide d'une des fonctions de dessin étendues telles que draw_self() ou draw_sprite_ext().

Syntaxe :

image_angle;

Les retours :

Real

Exemple :

image_angle = point_direction(x, y, mouse_x, mouse_y);

Le code ci-dessus fera pivoter le sprite de l'instance pour qu'il pointe toujours vers la position de la souris.
