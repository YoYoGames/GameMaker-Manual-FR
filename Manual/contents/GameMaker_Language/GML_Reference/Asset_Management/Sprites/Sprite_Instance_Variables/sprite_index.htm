sprite_index

Cette variable renvoie l'index de l'actuel sprite pour l'instance, ou -1 si l'instance n'a pas de sprite associé à elle. Vous pouvez la modifier pour donner à l'instance une autre sprite en lui donnant le nom d'une sprite de l'arbre des ressources ou en utilisant une variable dans laquelle est indexée une sprite chargée en externe. Changer le sprite ne change pas l'index du cadre actuellement visible, donc si vous changez le sprite du cadre numéro 3, le nouveau sprite sera dessiné avec ce cadre visible (en supposant qu'il ait le même nombre de cadres).

Syntaxe :

sprite_index

Les retours :

Sprite Asset

Exemple :

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

Le code ci-dessus passera en boucle toutes les instances de "obj_Check" pour vérifier s'il existe une ligne de collision entre elles et l'instance qui exécute le code. Le site sprite de ces instances sera modifié en fonction de la valeur de retour (vrai ou faux) pour la ligne de collision.
