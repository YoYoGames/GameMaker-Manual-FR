vitesse de l'image

Cette variable détermine la vitesse à laquelle GameMaker fera défiler les sous-images de l'instance courante sprite. La valeur de la vitesse donnée est un multiplicateur, 1 étant la valeur par défaut. Si vous la réglez à 0,5, la vitesse de l'animation sera réduite de moitié - telle que définie dans l'éditeur de sprites ou l'éditeur d'images - tandis que si vous la réglez à 2, elle sera doublée. Si le site sprite utilisé ne comporte pas de sous-images, cette variable n'aura aucun effet.

Syntaxe :

image_speed;

Les retours :

(single precision floating point value)

Exemple :

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

Le code ci-dessus crée une instance de l'objet "obj_Explosion" et définit son image_speed à 0,5 (ce qui réduit de moitié la vitesse de déplacement).
