image_alpha

Cette variable est utilisée pour obtenir ou définir la valeur alpha de sprite. L'alpha est toujours calculé comme une valeur entre 0 et 1, où 0 est complètement transparent et 1 est complètement opaque. Veuillez noter que pour que les changements de cette variable soient visibles, l'instance ne doit pas avoir d'événement de dessin (et GameMaker dessinera donc par défaut le sprite) ou être dessinée à l'aide de l'une des fonctions de dessin étendues telles que draw_self() ou draw_sprite_ext().

Syntaxe :

image_alpha;

Les retours :

Real

Exemple :

image_alpha = clamp(image_alpha - 0.01, 0, 1);

Le code ci-dessus va réduire lentement l'image_alpha jusqu'à ce qu'elle atteigne 0.
