image_xscale

Cette valeur définit la mise à l'échelle horizontale appliquée à l'adresse sprite qui a été attribuée à l'instance actuelle. Une échelle de 1 indique qu'il n'y a pas de mise à l'échelle (1:1), des valeurs plus petites réduisent l'échelle (0,5, par exemple, réduit de moitié la largeur de sprite), des valeurs plus grandes l'augmentent et des valeurs négatives retournent sprite et le mettent à l'échelle, sauf si la valeur utilisée est exactement -1 (auquel cas sprite est simplement retourné autour de son origine sans mise à l'échelle).

Syntaxe :

image_xscale;

Les retours :

(single precision floating point value)

Exemple :

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Le code ci-dessus met à l'échelle le site sprite et une fois qu'il a atteint 5 fois sa taille d'origine, une nouvelle instance d'un autre objet est créée et l'instance est détruite.
