sprite_xoffset

Cette variable en lecture seule renvoie le xoffset local (la composante x de l'origine telle que définie dans l'éditeur sprite ) du site sprite qui a été attribué à l'instance. Ce xoffset est renvoyé en pixels et dépendra de la taille de l'instance. image_xscalede sorte que (par exemple) si un carré de 32x32 sprite est défini comme ayant l'origine x à 16, lorsque ce sprite est mis à l'échelle par 2, cette variable renverra 32. Si vous avez besoin du décalage en x non mis à l'échelle, vous devez utiliser la variable sprite_get_xoffset().

Syntaxe :

sprite_xoffset

Les retours :

Real

Exemple :

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

Le code ci-dessus vérifie si le bord gauche du sprite de l'instance est à l'extérieur de la pièce et si c'est le cas, il déplace l'instance de façon à ce que l'ensemble du sprite soit visible.
