masque_index

Lorsque vous définissez un objet dans GameMaker, vous pouvez lui attribuer un masque qui sera utilisé pour les collisions plutôt que celui qui correspond à l'adresse définie sprite. Cette variable peut être utilisée pour trouver le sprite_index de ce masque (ou elle retournera -1 si aucun sprite n'a été assigné) ou pour définir le masque d'une instance avec le sprite choisi. Le réglage de l'index du masque signifie que vous pouvez avoir, par exemple, un sprite pour l'instance avec une forme irrégulière, mais lui donner un masque de collision circulaire qui est obtenu à partir d'un sprite différent.

Syntaxe :

mask_index;

Les retours :

Sprite Asset

Exemple :

mask_index = spr_Round;

Le code ci-dessus définit le masque de l'instance à celui de sprite "spr_Round".
