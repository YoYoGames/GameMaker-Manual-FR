image_yscale

Cette valeur définit la mise à l'échelle verticale (le long de l'axe des y) appliquée au site sprite qui a été attribué à l'instance actuelle. Une échelle de 1 indique qu'il n'y a pas d'échelle (1:1), des valeurs plus petites réduisent l'échelle (0,5, par exemple, réduit de moitié la hauteur de sprite), des valeurs plus grandes l'augmentent et des valeurs négatives reflètent le miroir de sprite et le mettent à l'échelle, sauf si la valeur utilisée est exactement -1 (dans ce cas, sprite est simplement reflété le long de l'axe des y sans aucune échelle).

Syntaxe :

image_yscale;

Les retours :

(single precision floating point value)

Exemple :

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Le code ci-dessus met à l'échelle le site sprite et une fois qu'il a atteint 5 fois sa taille d'origine, une nouvelle instance d'un autre objet est créée et l'instance est détruite.
