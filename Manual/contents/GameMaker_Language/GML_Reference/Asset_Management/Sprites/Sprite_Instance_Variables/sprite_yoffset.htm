sprite_yoffset

Cette variable en lecture seule renvoie le décalage y local (la composante y de l'origine telle que définie dans l'éditeur sprite ) du site sprite qui a été attribué à l'instance. Ce décalage en pixels dépend de la taille de l'instance. image_yscalede sorte que (par exemple) si un carré de 32x32 sprite est défini comme ayant l'origine y à 16, lorsque ce sprite est mis à l'échelle par 2, cette variable renverra 32. Si vous avez besoin de l'origine y non mise à l'échelle, vous devez utiliser la variable sprite_get_yoffset().

Syntaxe :

sprite_yoffset

Les retours :

Real

Exemple :

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

Le code ci-dessus vérifie si le bord supérieur du sprite de l'instance est à l'extérieur de la pièce et si c'est le cas, il déplace l'instance de façon à ce que l'ensemble du sprite soit visible.
