La structure à neuf tranches

Une structure Nine Slice peut être créée en utilisant sprite_nineslice_create()ou être récupérée à partir d'un site sprite en utilisant sprite_get_nineslice(). Cette structure struct contient des propriétés qui peuvent être modifiées et appliquées à une structure sprite à l'aide de . sprite_set_nineslice().

Si elle est extraite d'un site sprite, la modification des valeurs de ce site struct aura une incidence sur le site original sprite.

Les propriétés suivantes seront disponibles dans cette structure :

Structures à neuf tranches
Variable Type Description
enabled Booléen Ce booléen sera vrai si le découpage en neuf est activé pour le site sprite, et faux s'il est désactivé. Vous pouvez modifier ce paramètre pour activer ou désactiver le découpage en neuf pour le site sprite.
left Real Il s'agit du décalage du guide gauche sur l'axe x, par rapport au bord gauche du site sprite.
right Real Il s'agit du décalage du guide droit sur l'axe x, par rapport au bord droit du site sprite.
top Real Il s'agit du décalage du guide supérieur sur l'axe y, par rapport au bord supérieur du site sprite.
bottom Real Il s'agit du décalage du guide inférieur sur l'axe y, par rapport au bord inférieur du site sprite.
tilemode Matrice de Mode de tuile Constantes Cela renvoie un tableau contenant les modes de tuiles pour les tranches de bord et de centre.

Modes d'utilisation des tuiles

Le mode de tuile de chaque tranche peut être lu ou modifié en utilisant ces constantes comme index du tableau :

Bordure à neuf tranches constante
Constant Description
nineslice_left La tranche du bord gauche
nineslice_top La tranche du bord supérieur
nineslice_right La tranche du bord droit
nineslice_bottom La tranche du bord inférieur
nineslice_centre La tranche centrale

Le mode de tuile d'une tranche sera (ou peut être réglé sur) l'une des constantes suivantes :

Tile Mode Constant
Constant Description
nineslice_stretch La tranche sera étirée
nineslice_repeat La tranche sera répétée
nineslice_mirror La tranche sera répétée en miroir
nineslice_blank La tranche ne sera pas étirée ou répétée, ce qui entraîne une zone vide après celle-ci.
nineslice_hide La tranche n'apparaîtra pas du tout

Comme tilemode est un tableau, vous devez utiliser la syntaxe des tableaux pour modifier le mode de tuiles d'une tranche. L'exemple de code ci-dessous définit le Tile Mode de la tranche supérieure sur Repeat:

Exemple :

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
