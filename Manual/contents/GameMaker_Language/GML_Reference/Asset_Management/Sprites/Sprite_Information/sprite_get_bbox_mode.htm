sprite_get_bbox_mode

Cette fonction renvoie le "mode" actuel pour les calculs de la boîte englobante. Vous fournissez l'indice sprite du site sprite à vérifier, et la fonction renvoie l'une des constantes indiquées ci-dessous.

Syntaxe :

sprite_get_bbox_mode(ind);

ArgumentsType Description
ind The index of the sprite to check.

Renvoie à

Bounding Box Mode Constant

Constant Description
bboxmode_automatic Automatique - La boîte de délimitation sera calculée automatiquement, sur la base du paramètre de tolérance du sprite.
bboxmode_fullimage Image complète - La boîte de délimitation sera définie de manière à utiliser la largeur et la hauteur complètes du site sprite, sans tenir compte de la tolérance et des pixels "vides".
bboxmode_manual Manuel - La boîte de délimitation a été réglée manuellement sur des valeurs définies par l'utilisateur (soit dans l'éditeur sprite, soit en utilisant la fonction sprite_set_bbox())

Exemple :

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

La commande ci-dessus vérifie le mode bbox pour le site sprite actuel et si ce n'est pas automatic, elle le définit à cette valeur.
