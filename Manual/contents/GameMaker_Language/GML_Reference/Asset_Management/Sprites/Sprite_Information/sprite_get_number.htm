sprite_get_number

Un sprite doit avoir au moins une sous-image et cette fonction renvoie le nombre de sous-images qu'il possède. N'oubliez pas qu'une sous-image de sprite est numérotée à partir de 0, donc si cette fonction renvoie 3 (la sprite vérifiée a 3 sous-images), elles seront numérotées 0, 1 et 2.

Syntaxe :

sprite_get_number(index);

ArgumentsType Description
index The index of the sprite.

Renvoie à

Exemple :

image_index = floor(random(sprite_get_number(sprite_index)));

Le code ci-dessus définira l'index de l'image à un cadre aléatoire basé sur le nombre de sous-images que le site sprite possède.
