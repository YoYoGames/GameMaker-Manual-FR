sprite_get_bbox_left

Cette fonction renvoie la position relative de la gauche de la boîte englobante sprite. Cette valeur est donnée sous la forme d'une valeur relative basée sur le coin supérieur gauche de la ressource de base sprite qui est (0,0). C'est la même valeur que celle que l'on peut trouver dans l'éditeur sprite pour les propriétés du masque de collision. L'image ci-dessous montre comment elle est calculée :

Syntaxe :

sprite_get_bbox_left(ind);

ArgumentsType Description
ind The index of the sprite to check.

Renvoie à

Exemple :

var ww, hh;
ww = sprite_get_bbox_left(sprite_index) - sprite_get_bbox_right(sprite_index);
hh = sprite_get_bbox_bottom(sprite_index) - sprite_get_bbox_top(sprite_index);

Le code ci-dessus calcule la largeur et la hauteur du masque de collision en fonction des positions latérales relatives de la boîte englobante.
