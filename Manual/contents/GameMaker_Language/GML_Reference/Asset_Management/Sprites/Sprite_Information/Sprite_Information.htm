Informations sur les sprites

Les fonctions suivantes peuvent être utilisées pour obtenir des informations sur un actif sprite. Veuillez noter que ces fonctions obtiennent des informations à partir de la base sprite (c'est-à-dire l'actif sprite, tel que créé dans l'Asset Browser) et non à partir de sprite tel qu'il est utilisé dans une instance. Par exemple, vous pouvez utiliser la sprite variable d'instance sprite_width pour obtenir la largeur de sprite qu'une instance utilise et cette largeur prendra en compte toute mise à l'échelle qui aurait pu être appliquée. Cependant, si vous avez besoin de connaître la largeur d'un sprite tel qu'il serait normalement, sans aucune transformation appliquée, vous devrez utiliser la fonction sprite_get_width() ci-dessous.

Pour plus d'informations sur les variables d'instance de sprite, voir la section sur les variables d'instance.
