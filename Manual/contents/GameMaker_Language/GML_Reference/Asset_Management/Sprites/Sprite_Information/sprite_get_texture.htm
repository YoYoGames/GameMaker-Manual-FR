sprite_get_texture

Cette fonction renvoie un pointeur spécial pour la page de texture sprite. Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, en particulier dans les fonctions de primitives 2D, ainsi que dans les fonctions de Shader. Vous pouvez obtenir plus d'informations sur la page de texture renvoyée en utilisant les différentes fonctions de texture_ que vous trouverez ici.

Syntaxe :

sprite_get_texture(spr, subimg);

Arguments Type Description
spr Sprite Asset The index of the sprite to use.
subimg Real The sub-image of the sprite to use.

Renvoie à

Texture

Exemple :

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

Le code ci-dessus dessine une bande triangulaire à 4 sommets texturée avec la texture contenue dans la variable "tex".
