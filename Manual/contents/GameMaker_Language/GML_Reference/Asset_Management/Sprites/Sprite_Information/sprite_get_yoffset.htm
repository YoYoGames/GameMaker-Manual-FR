sprite_get_yoffset

Lorsque vous définissez un sprite dans l'éditeur sprite, vous devez définir l'origine de ce sprite. Il s'agit du point auquel le site sprite sera "attaché" ou "dessiné" lorsqu'il sera utilisé avec une instance. Cette fonction renvoie le décalage relatif de l'axe des y de l'origine en se basant sur le coin supérieur gauche qui est la position 0,0, avec un +y correspondant à une descente et un -y à une montée. L'image suivante illustre cela :

Syntaxe :

sprite_get_yoffset(index);

ArgumentsType Description
index The index of the sprite to find the yoffset of.

Renvoie à

Exemple :

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = _yoffset;
}

Le code ci-dessus garantit qu'une instance est maintenue dans la pièce en fonction de sprite (de sorte que sprite soit toujours visible).
