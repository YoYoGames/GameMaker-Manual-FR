sprite_get_info

Cette fonction est utilisée pour récupérer des informations pour l'adresse sprite donnée. Vous fournissez un index sprite (qui peut être un actif ajouté via le navigateur d'actifs ou un sprite ajouté à runtime) et la fonction renvoie une structure avec les variables suivantes :

Structure d'information sur les sprites
Variable Type Description
largeur réel Largeur du sprite (en pixels)
hauteur réel Hauteur du sprite (en pixels)
xoffset réel Décalage ou origine en X du sprite (en pixels)
yoffset réel Décalage ou origine en Y du sprite (en pixels)
transparent boolean true si le site sprite est marqué comme transparent, sinon false
(Ceci ne peut être spécifié qu'à travers sprite_add() ou des fonctions similaires, et sera false pour sprites créé dans le IDE)
lisse boolean true si le sprite est marqué comme lisse, sinon false
(Ceci ne peut être spécifié que par sprite_add() ou des fonctions similaires, et sera false pour sprites créé dans le IDE)
type réel Le type de sprite :
0 - Bitmap (Regular sprites)
1 - SWF
2 - Spine
bbox_left réel Position du bord gauche de la boîte englobante (en pixels)
bbox_top réel Position du bord supérieur de la boîte englobante (en pixels)
bbox_right réel Position du bord droit de la boîte englobante (en pixels)
bbox_bottom réel Position du bord inférieur de la boîte englobante (en pixels)
nom chaîne de caractères Le nom du sprite
num_sous-images réel Le nombre de sous-images (ou de cadres) dans le sprite.
masque d'utilisation boolean true si ce sprite utilise un masque de collision, sinon false
num_masks réel Le nombre de masques dans ce sprite (sera supérieur à 1 si le sprite utilise des masques par trame)
nineslice struct La structure à neuf tranches pour ce site sprite, ou undefined s'il n'a pas de données à neuf tranches.
messages tableau Tableau des messages de diffusion pour ce site sprite, où chaque message de diffusion est un struct contenant des informations sur le message (pour plus d'informations, voir "Données générales Sprite ").
info_cadre tableau Tableau de trames pour ce site sprite, où chaque trame est un struct contenant des informations sur son timing (plus d'informations sous "Données générales Sprite ")

Cette variable supplémentaire n'est disponible que pour les sprites Bitmap (ordinaires) :

Structure d'information sur les sprites
Variable Type Description
cadres tableau Tableau de trames pour ce sprite, où chaque trame est un struct contenant des informations sur sa texture (pour plus d'informations, voir "Données Bitmap Sprite ")

Ces variables supplémentaires ne sont disponibles que pour les sprites Spine:

Structure d'information sur les sprites
Variable Type Description
num_atlas réel Le nombre de textures atlas utilisées
texture_atlas tableau Tableau des ID de texture utilisés pour l'atlas
prémultiplié boolean true si ce sprite est marqué comme prémultiplié, sinon false
noms_d'animation tableau Tableau contenant les noms de chaque animation dans ce sprite
noms de peau tableau Tableau contenant les noms de chaque peau dans ce sprite
ossements tableau Tableau contenant structs pour chaque os de ce sprite (pour plus d'informations, voir "Spine Sprite Data")
Machines à sous tableau Tableau contenant structs pour chaque emplacement dans ce sprite (pour plus d'informations, voir "Données sur la colonne vertébrale Sprite ")

La fonction renverra undefined si la variable sprite n'existe pas. Notez également que les informations retournées dans ce struct doivent être considérées comme étant en lecture seule, car la modification de l'une de ces variables n'affectera pas le sprite.

Les sections ci-dessous contiennent des informations sur les tableaux et le site structs inclus dans le site struct retourné en fonction du type sprite:

Données générales Sprite

Cette section contient des informations sur les variables incluses dans le site struct pour tous les types de sprites.

La variable messages est un tableau qui contient des informations sur les messages de diffusion existant dans le site sprite donné. Chaque message de diffusion dans ce tableau est un struct contenant les variables suivantes :

Nom de la variable Type de données Description
cadre réel Le timing de ce message de diffusion à partir du début de l'animation (en frames)
message chaîne de caractères La chaîne du message de diffusion

La variable frame_info est un tableau qui contient des informations sur la synchronisation des images du sprite. Chaque image de ce tableau est un struct contenant les variables suivantes :

Nom de la variable Type de données Description
cadre réel Le timing pour le début de cette trame (en trames)
durée réel La durée de ce cadre (en cadres)
image_index réel L'index de l'image de ce cadre

Bitmap Sprite Données

Cette section contient des informations sur les variables incluses dans le site struct pour Bitmap sprites (sprites qui ne sont pas Spine ou SWF type sprites).

La variable frames est un tableau qui contient des informations sur les textures des images du sprite. Chaque image de ce tableau est un struct contenant les variables suivantes :

Nom de la variable Type de données Description
x réel La position X de ce cadre sur sa page de texture (en pixels)
y réel La position Y de ce cadre sur sa page de texture (en pixels)
w réel La largeur logique de la trame (en pixels) utilisée en interne.
h réel La hauteur logique du cadre (en pixels) utilisée en interne.
texture réel L'ID de la page de texture pour ce cadre
largeur originale réel La largeur originale du cadre (en pixels)
hauteur_originale réel La hauteur originale du cadre (en pixels)
largeur de la récolte réel La largeur réelle de l'image sur la page de texture après recadrage et mise à l'échelle (puisque GameMaker recadre automatiquement l'espace vide autour d'une image et la met à l'échelle si elle ne tient pas).
hauteur de la récolte réel La hauteur réelle du cadre sur la page de la texture après recadrage et mise à l'échelle.
x_offset réel Le décalage X entre le bord gauche de l'image originale et le bord gauche de la section recadrée.
y_offset réel Le décalage Y entre le bord supérieur de l'image originale et le bord supérieur de la section recadrée.

Colonne vertébrale Sprite Données

Cette section contient des informations sur les variables incluses dans le site struct pour Spine sprites.

La variable bones est un tableau qui contient des informations sur tous les os du site Spine sprite. Chaque os de ce tableau est un struct contenant les variables suivantes :

Nom de la variable Type de données Description
parent chaîne de caractères Nom de l'os parent, ou undefined si l'os n'a pas de parent.
nom chaîne de caractères Le nom de cet os
indice réel L'indice de cet os
longueur réel La longueur de cet os
x réel La position X de cet os
y réel La position Y de cet os
rotation réel La rotation de cet os
scale_x réel (Interne à Spine) Valeur de l'échelle sur X
échelle_y réel (Interne à Spine) Valeur de l'échelle sur Y
cisaillement_x réel (interne à Spine) Valeur de cisaillement sur X
cisaillement_y réel (interne à Spine) Valeur de cisaillement sur Y
mode de transformation réel (Interne à Spine) Le mode de transformation

NOTE: Veuillez consulter la documentation de Spine pour plus d'informations concernant les variables internes de Spine.

La variable slots est un tableau qui contient des informations sur tous les créneaux du site Spine sprite. Chaque emplacement de ce tableau est un struct contenant les variables suivantes :

Nom de la variable Type de données Description
nom chaîne de caractères Le nom de l'emplacement
indice réel L'indice de l'emplacement
os chaîne de caractères Le nom de l'os de l'emplacement, ou "(none)" s'il n'y a pas d'os pour cet emplacement.
pièce jointe chaîne de caractères Nom de la pièce jointe
rouge réel Composante rouge de la couleur du slot (0-1)
vert réel Composante verte de la couleur du slot (0-1)
bleu réel Composante bleue de la couleur du slot (0-1)
alpha réel Composante alpha de la couleur de l'emplacement (0-1)
blendMode réel (Interne à Spine) Mode de mélange pour le slot
dark_red* réel Composante rouge de la couleur foncée du slot (0-1)
dark_green* réel Composante verte de la couleur foncée du slot (0-1)
dark_blue* réel Composante bleue de la couleur foncée du slot (0-1)
dark_alpha* chaîne de caractères Composante alpha de la couleur foncée du slot (0-1)

* NOTE: Les variables de couleur foncée ne sont disponibles que si le slot a la fonction Tint Black activée.

Syntaxe :

sprite_get_info(index);

Arguments Type Description
index Sprite Asset The index of the sprite to get the information for.

Renvoie à

Sprite Info Struct (or undefined)

Exemple :

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
    var _messages = _info.messages;
    var _messageCount = array_length(_messages);
    
    for (var i = 0; i < _messageCount; i ++)
    {
        var _message = _messages[i];
        show_debug_message("Message at frame " + string(_message.frame) + ": " + _message.message);
    }
}

Le code ci-dessus récupère l'info struct pour le sprite de l'instance, puis vérifie si ce n'est pas undefined et si son type est 0 (ce qui signifie que c'est un bitmap sprite). Dans ce cas, il récupère le tableau des messages de diffusion pour cette sprite et exécute ensuite une boucle for pour imprimer chaque message de diffusion (avec sa trame) dans le journal de sortie.
