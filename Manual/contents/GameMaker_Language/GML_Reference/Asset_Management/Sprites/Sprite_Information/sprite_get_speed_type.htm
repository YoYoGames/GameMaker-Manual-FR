sprite_get_speed_type

Cette fonction peut être utilisée pour récupérer le type d'animation sprite tel que défini pour la ressource sprite dans l'éditeur de sprites. La valeur renvoyée sera l'une des constantes suivantes :

Constante de vitesse du sprite
Constant Description
spritespeed_framespersecond Le site sprite a été défini avec une animation en images par seconde.
spritespeed_framespergameframe Le site sprite a été défini avec dans l'animation en frames par frame de jeu.

Les deux exemples suivants illustrent la différence que cela peut faire lors du calcul des vitesses d'animation du sprite :

  • Si vous avez une sprite qui dessine 1 image par seconde et que vous réglez la vitesse de l'image sur 0,5, elle dessinera à 0,5 image par seconde. Si la fréquence d'images de votre jeu est de 60 images par seconde, le site sprite affichera une image toutes les 120 images du jeu.
  • Si vous avez un sprite qui dessine 1 image par image de jeu et que vous réglez la vitesse de l'image sur 0,5, il dessinera 0,5 image par image de jeu. Si la fréquence d'images de votre jeu est de 60 images par seconde, le site sprite affichera 30 images par 60 images de jeu.

Vous pouvez trouver la valeur de la vitesse utilisée pour l'animation sprite à l'aide de la fonction sprite_get_speed()et vous pouvez définir la vitesse et le type d'animation à l'aide de la fonction sprite_set_speed().

Syntaxe :

sprite_get_speed_type(index)

Arguments Type Description
index Sprite Asset The index of the sprite to get the animation type of

Les retours :

Sprite Speed Constant (see above)

Exemple :

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

Le code ci-dessus récupère la vitesse de sprite et le type d'animation de sprite et les stocke dans des variables d'instance pour une utilisation ultérieure.
