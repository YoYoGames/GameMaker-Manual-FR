sprite_get_nineslice

Cette fonction renvoie la structure Nine Slice d'un sprite, contenant toutes les propriétés Nine Slice définies dans l'éditeur de sprites ou à l'aide de sprite_set_nineslice(). Le contenu de cette struct est détaillé sur cette page.

Si le site sprite fourni n'a pas de Nine Slice struct qui lui est attribué, un nouveau struct avec des propriétés Nine Slice par défaut sera créé pour le sprite et renvoyé.

La modification des valeurs de ce site struct aura pour effet de modifier les propriétés des neuf tranches du site original sprite, ce qui aura une incidence sur tous les futurs appels de tirage effectués avec ce site sprite.

Syntaxe :

sprite_get_nineslice(ind);

ArgumentsType Description
ind The index of the sprite from which the Nine Slice struct will be retrieved

Les retours :

Nine Slice Struct (or -1 if the sprite doesn't exist)

Exemple :

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

Le code ci-dessus récupère le site struct à partir d'un site sprite, active le découpage en neuf tranches pour ce site et définit les décalages des guides.
