sprite_get_uvs

Cette fonction renvoie un tableau contenant les coordonnées UV et d'autres données pour la texture de la sous-image sprite sur la page de texture. La fonction renvoie un tableau avec les 8 éléments suivants :

  • [0] = gauche
  • [1] = haut
  • [2] = droite
  • [3] = fond
  • [4] = nombre de pixels que le compilateur d'actifs a supprimés du côté gauche du site sprites.
  • [5] = nombre de pixels que le compilateur d'actifs a supprimés de la partie supérieure de sprites.
  • [6] = pourcentage normalisé de données de pixels de la largeur originale de sprites qui ont été sauvegardées dans la page de texture.
  • [7] = pourcentage normalisé des données de pixel de la hauteur originale de sprites qui ont été sauvegardées sur la page de texture.

Ce tableau peut ensuite être utilisé dans d'autres fonctions de dessin, notamment lors du travail en 3D ou à l'aide des fonctions primitives 2D, ainsi que lors du travail avec les fonctions Shader.

NOTE: Cette fonction ne fonctionnera pas avec le vecteur sprites ou l'animation de squelette sprites.

Syntaxe :

sprite_get_uvs(sprite, subimage);

ArgumentsType Description
sprite The index of the sprite to use.
subimage The sub-image of the sprite to use.

Renvoie à

(1D, 8 elements)

Exemple :

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Le code ci-dessus stocke les coordonnées UV pour le site sprite donné dans un tableau local, puis affecte les valeurs aux variables d'instance.
