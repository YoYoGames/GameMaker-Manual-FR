sprite_get_height

Avec cette fonction, vous pouvez trouver la hauteur de la ressource de base sprite, sans transformations, en pixels.

Syntaxe :

sprite_get_height(index);

ArgumentsType Description
index The index of the sprite to find the height of.

Renvoie à

Exemple :

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

Le code ci-dessus vérifie la hauteur de sprite telle qu'elle est dans l'instance actuelle et s'il y a une différence entre cette hauteur et celle de la base originale sprite, il réinitialise l'échelle de l'axe des ordonnées.
