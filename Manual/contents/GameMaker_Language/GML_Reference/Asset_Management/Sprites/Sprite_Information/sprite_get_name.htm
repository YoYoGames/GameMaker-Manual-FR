sprite_get_name

Cette fonction renvoie le nom sous forme de chaîne de caractères du site sprite spécifié. Ce nom est celui qui a été spécifié pour le sprite dans l'arbre des ressources de la fenêtre principale de GameMaker. Veuillez noter qu'il s'agit uniquement d'une chaîne de caractères et qu'elle ne peut pas être utilisée pour référencer directement le site sprite - pour cela, vous aurez besoin de l'index du sprite. Vous pouvez toutefois utiliser cette chaîne pour obtenir l'index du sprite en utilisant la chaîne retournée avec la fonction asset_get_index().

Syntaxe :

sprite_get_name(index);

Arguments Type Description
index Actif The index of the sprite to get the name of.

Renvoie à

String

Exemple :

str = sprite_get_name(sprite_index);

Le code ci-dessus récupère le nom de l'index sprite pour l'instance qui exécute le code et stocke la valeur de retour dans la variable "str".
