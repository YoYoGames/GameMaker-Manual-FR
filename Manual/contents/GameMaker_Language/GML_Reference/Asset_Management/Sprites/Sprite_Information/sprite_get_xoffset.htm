sprite_get_xoffset

Lorsque vous définissez un sprite dans l'éditeur sprite, vous devez définir l'origine de ce sprite. Il s'agit du point auquel le site sprite sera "attaché" ou "dessiné" lorsqu'il sera utilisé avec une instance. Cette fonction renvoie le décalage relatif de l'axe des x de l'origine en se basant sur le coin supérieur gauche qui est la position 0,0, avec un +x étant à droite et un -x étant à gauche de cela. L'image suivante illustre cela :

Syntaxe :

sprite_get_xoffset(index);

ArgumentsType Description
index The index of the sprite to find the xoffset of.

Renvoie à

Exemple :

if x - sprite_get_xoffset(sprite_index) < 0
{
    x = sprite_get_xoffset(sprite_index);
}

Le code ci-dessus garantit qu'une instance est maintenue dans la pièce sur la base de sprite (de sorte que sprite soit toujours visible).
