shader_get_uniform

Étant donné que vous ne pouvez pas modifier la valeur d'une constante shader au sein même de shader, vous devez la définir avant d'appeler shader à l'aide de l'une des fonctions de définition uniforme disponibles. Toutefois, pour pouvoir le faire, vous devez d'abord appeler cette fonction pour obtenir le "handle" de la constante shader que vous souhaitez modifier.

NOTE: Bien qu'un shader soit composé de deux programmes discrets (sommet et fragment), cette fonction ne fera pas la différence entre les deux et renverra le handle de la constante shader de l'un ou l'autre.

Syntaxe :

shader_get_uniform(shader, uniform);

Arguments Type Description
shader Shader Asset The index of the shader to use.
uniform Chaîne de caractères The shader constant to get the handle of (a string).

Les retours :

Shader Uniform Handle

Exemple :

shader_params = shader_get_uniform(shd_glass, "u_vRefractColour");

Le code ci-dessus permet d'obtenir le gestionnaire de la constante "u_vRefractColour" sur shader.
