shader_set_uniform_matrix

Cette fonction permet de définir la (ou les) valeur(s) d'une constante shader à la matrice de transformation actuelle (définie à l'aide des fonctions matricielles). Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform().

NOTE: Tous les uniformes doivent être définis après l' appel de la fonction shader_set()et avant d'appeler shader_reset().

Syntaxe :

shader_set_uniform_matrix(handle);

Arguments Type Description
handle Manche uniforme de shader The handle of the shader constant to set.

Les retours :

N/A

Exemple :

shader_set(shader_Glass);
shader_matrix = shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);
draw_self();
shader_reset();

Le code ci-dessus obtiendra le gestionnaire de la constante "u_vMatrix" de shader, puis définira cette constante comme la matrice de transformation actuelle.
