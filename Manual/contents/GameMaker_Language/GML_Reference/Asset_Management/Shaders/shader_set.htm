shader_set

Avec cette fonction, vous pouvez définir la cible de dessin à l'adresse shader donnée et tous les dessins ultérieurs seront effectués à l'aide de celle-ci. Vous pouvez mettre fin à l'utilisation de shader avec la fonction shader_reset().

Syntaxe :

shader_set(shader);

ArgumentsType Description
shader The index of the shader to use.

Les retours :

Exemple :

shader_set(shader_Glass);
draw_self();
shader_reset();

Le code ci-dessus définit un shader à utiliser pour le dessin, puis dessine le sprite actuel utilisé pour l'instance qui l'utilise.
