shader_set_uniform_i

Cette fonction vous permet de définir la (ou les) valeur(s) d'une constante shader. Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform()et vous devrez savoir de quel type de constante il s'agit pour lui transmettre le nombre correct de valeurs entières, c'est-à-dire que si vous avez une vec2, vous devrez transmettre deux valeurs à la fonction.

NOTE: Tous les uniformes doivent être définis après l' appel de la fonction shader_set()et avant d'appeler shader_reset().

Syntaxe :

shader_set_uniform_i(handle, value1 [, value2, value3, value4]);

Arguments Type Description
handle Manche uniforme de shader The handle of the shader constant to set.
value1 ... value4 Real The integer value (or values) to set the shader constant to.

Les retours :

N/A

Exemple :

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

Le code ci-dessus récupère l'identifiant de la constante "u_vParams" de shader ( vec3), puis définit cette constante sur les valeurs entières données.
