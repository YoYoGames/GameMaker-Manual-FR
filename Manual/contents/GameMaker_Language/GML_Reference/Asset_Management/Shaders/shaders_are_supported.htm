shaders_sont_supportés

Cette fonction vérifie si la plate-forme cible choisie prend en charge shaders et renvoie true si c'est le cas, et false dans le cas contraire. Il est important de noter que sur Android, si le projet n'a pas de ressources shader définies, la fonction renverra toujours false, que le périphérique prenne en charge shaders ou non.

Syntaxe :

shaders_are_supported();

Les retours :

Exemple :

global.GFX = false;
if (shaders_are_supported())
{
    if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
    {
        global.GFX = true;
    }
}

Le code ci-dessus définira une variable globale à false, puis, si la plate-forme prend en charge shaders et si les deux shaders vérifiés ont été compilés correctement, elle sera définie à true.
