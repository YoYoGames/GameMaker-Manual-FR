shader_set_uniform_i_array

Avec cette fonction, vous pouvez définir une constante shader pour contenir un tableau de valeurs. Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform()et vous devrez avoir préalablement initialisé le tableau.

NOTE: Tous les uniformes doivent être définis après l' appel de la fonction shader_set()et avant d'appeler shader_reset().

Syntaxe :

shader_set_uniform_i_array(handle, array);

Arguments Type Description
handle Manche uniforme de shader The handle of the shader constant to set.
array Array A previously initialised array of integer values.

Les retours :

N/A

Exemple :

shader_set(shader_Glass);
col_array[0] = 255;
col_array[2] = 255;
col_array[3] = 64;
col_array[4] = 128;
shader_params = shader_get_uniform(shader_tint, "cColourArray");
shader_set_uniform_i_array(shader_params, col_array);
draw_self();
shader_reset();

Le code ci-dessus récupère l'identifiant de la constante "cColourArray" de shader puis définit cette constante sur le tableau donné.
