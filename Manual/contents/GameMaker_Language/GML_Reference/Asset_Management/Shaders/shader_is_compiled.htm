shader_est_compilé

Cette fonction vérifiera un shader au moment de l'exécution pour s'assurer qu'il a été compilé avec succès. Si c'est le cas, la fonction renvoie true, sinon elle renvoie false. Cette fonction doit être utilisée au début du jeu pour s'assurer que la plate-forme qui exécute votre jeu a compilé avec succès tout shaders utilisé (en particulier sur Windows où certains ordinateurs peuvent utiliser DX9 avec Shader niveau 2.0 et non une version ultérieure utilisant shader niveau 3.0).

Si votre shader n'a PAS été compilé et que vous appelez shader_set() le jeu se plantera, il vaut donc la peine d'avoir une sorte de vérification chaque fois que vous utilisez autre chose que le simple GLSL ES shaders.

Syntaxe :

shader_is_compiled(shader);

ArgumentsType Description
shader The shader to check.

Les retours :

Boolean

Exemple :

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

Le code ci-dessus attribue à une variable globale la valeur false, puis, si les deux sites shaders vérifiés ont été compilés correctement, la variable sera attribuée à true.
