shader_set_uniform_f_array

Avec cette fonction, vous pouvez définir une constante shader pour contenir un tableau de valeurs. Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform()et vous devrez avoir préalablement initialisé le tableau.

NOTE: Tous les uniformes doivent être définis après l' appel de la fonction shader_set()et avant d'appeler shader_reset().

Syntaxe :

shader_set_uniform_f_array(handle, array);

Arguments Type Description
handle Manche uniforme de shader The handle of the shader constant to set.
array Array A previously initialised array of floating point values.

Les retours :

N/A

Exemple :

shader_set(shader_Glass);
tex_array[0] = 0.5;
tex_array[1] = 0.1;
tex_array[2] = 0.25;
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f_array(shader_params, tex_array);
draw_self();
shader_reset();

Le code ci-dessus récupère le handle de la constante "u_vParams" de shader puis définit cette constante dans le tableau donné.
