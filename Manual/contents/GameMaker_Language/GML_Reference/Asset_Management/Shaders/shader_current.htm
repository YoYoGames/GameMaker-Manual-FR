shader_current

Cette fonction renvoie la valeur de l'ID de l'index de shader actuellement utilisé pour le rendu, ou elle renvoie -1 si aucun shader n'est utilisé.

Syntaxe :

shader_current();

Les retours :

Shader Asset or -1 (if no shader is assigned)

Exemple :

if (shader_current() == -1)
{
    shader_set(sh_warp)
}

Le code ci-dessus vérifie l'état actuel de shader et s'il renvoie -1 (aucun shader n'est utilisé), alors un shader est défini.
