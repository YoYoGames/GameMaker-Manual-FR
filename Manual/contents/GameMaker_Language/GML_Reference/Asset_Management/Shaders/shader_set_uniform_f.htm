shader_set_uniform_f

Cette fonction vous permet de définir la (ou les) valeur(s) d'une constante shader. Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform()et vous devrez savoir de quel type de constante il s'agit pour lui transmettre le nombre correct de valeurs à virgule flottante, c'est-à-dire que si vous avez une vec2, vous devrez transmettre deux valeurs à la fonction.

Une exception à la règle ci-dessus concerne les valeurs de couleur 32 bits. Vous pouvez transmettre à shader jusqu'à huit valeurs de couleur 32bit et elles seront automatiquement converties pour vous en valeurs vec4 de 0 - 1. Cependant, pour cela, vous devez utiliser le nom de la constante shader in_Colour , pour une seule couleur, ou in_Colour0, in_Colour1, etc... jusqu'à in_Colour7 pour des valeurs d'entrée multiples. Les exemples suivants montrent comment cela se présente dans le site shader lui-même :

    //Single colour
    attribute vec2 in_Position;
    attribute vec4 in_Colour;
    attribute vec2 in_TextureCoord;

    //Multiple colours
    attribute vec2 in_Position;
    attribute vec4 in_Colour0;
    attribute vec4 in_Colour1;
    attribute vec2 in_TextureCoord;

Notez également que si vous voulez faire des normales compressées etc... en utilisant un format de couleur de 4 octets, vous devrez également utiliser le même format "in_Colour" comme indiqué ci-dessus.

NOTE: Tous les uniformes doivent être définis après l' appel de la fonction shader_set()et avant d'appeler shader_reset().

Syntaxe :

shader_set_uniform_f(handle, value1 [, value2, value3, value4]);

Arguments Type Description
handle Manche uniforme de shader The handle of the shader constant to set.
value1 ... value4 Real The floating point value (or values) to set the shader constant to.

Les retours :

N/A

Exemple :

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25);
draw_self();
shader_reset();

Le code ci-dessus récupère le gestionnaire de la constante "u_vParams" de shader ( vec3), puis définit cette constante en fonction des valeurs données.
