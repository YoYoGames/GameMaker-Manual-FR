shader_set_uniform_matrix_array

Avec cette fonction, vous pouvez définir une constante shader pour contenir un tableau de valeurs matricielles. Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform()et vous devrez avoir préalablement initialisé le tableau comme un tableau de valeurs à virgule flottante, où chaque groupe successif de 16 flottants est une matrice 4x4.

NOTE: Tous les uniformes doivent être définis après l' appel de la fonction shader_set()et avant d'appeler shader_reset().

Syntaxe :

shader_set_uniform_matrix_array(handle, array);

Arguments Type Description
handle Manche uniforme de shader The handle of the shader constant to set.
array Array A previously initialised array of floating point values.

Les retours :

N/A

Exemple :

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

Le code ci-dessus récupère l'identifiant de la constante "cMatrixArray" sur shader, puis définit cette constante sur le tableau donné.
