shader_get_sampler_index

Comme vous ne pouvez pas modifier la valeur d'un échantillonneur shader au sein même de shader, vous devez le définir avant d'appeler shader à l'aide de l'une des fonctions de définition d'uniformes disponibles. Cependant, pour pouvoir le faire, vous devez d'abord appeler cette fonction pour obtenir le "handle" de l'échantillonneur shader que vous souhaitez définir.

NOTE Bien qu'un shader soit composé de deux programmes discrets (vertex et fragment), cette fonction ne fera pas la différence entre les deux et renverra le handle de l'échantillon shader de l'un ou l'autre.

Syntaxe :

shader_get_sampler_index(shader, uniform);

Arguments Type Description
shader Shader Asset The index of the shader to use.
uniform Chaîne de caractères The shader sampler to get the handle of (a string).

Les retours :

Shader Sampler Handle

Exemple :

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Le code ci-dessus récupère l'identifiant de l'échantillonneur dans le fichier shader indexé comme "GlassShader", puis définit cette constante shader sur la texture sprite donnée.
