shader_reset

Cette fonction réinitialise le shader utilisé pour le dessin et doit être appelée lorsque vous ne souhaitez plus utiliser le shader actuel (défini à l'aide de la fonction shader_set()).

Syntaxe :

shader_reset();

Les retours :

Exemple :

shader_set(shader_Glass);
draw_self();
shader_reset();

Le code ci-dessus va définir un shader à utiliser pour le dessin, puis dessiner le sprite actuel utilisé pour l'instance qui l'utilise. Enfin, il réinitialisera le shader pour revenir au shader par défaut de GameMaker.
