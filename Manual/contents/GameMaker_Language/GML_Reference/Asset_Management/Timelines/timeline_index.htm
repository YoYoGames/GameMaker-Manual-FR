timeline_index

Cette variable contient l'index de la ligne de temps actuellement associée à l'instance. Vous pouvez la définir sur une ligne de temps particulière pour l'utiliser, ou la définir sur -1 pour arrêter d'utiliser une ligne de temps pour l'instance (si aucune ligne de temps n'est définie pour l'instance, -1 est également retourné). Notez que ceci ne démarre pas la ligne de temps - pour cela, utilisez la variable timeline_running.

Syntaxe :

timeline_index

Les retours :

Timeline Asset

Exemple :

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

Le code ci-dessus attribuera à l'instance qui exécute le code une ligne de temps indexée dans la variable "global.tl" si cette ligne de temps existe.

Retour : Lignes du temps
