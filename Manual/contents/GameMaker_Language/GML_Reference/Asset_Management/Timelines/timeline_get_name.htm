timeline_get_name

Cette fonction peut être utilisée pour obtenir le nom d'une ligne de temps sous forme de chaîne de caractères. Si la ligne de temps a été créée dynamiquement à l'aide de la fonction timeline_add() le nom renvoyé aura le format format "__newtimelineN" où "N" est le numéro de la ligne de temps (à partir de 0). Veuillez noter qu'il s'agit uniquement d'une chaîne de caractères et qu'elle ne peut pas être utilisée pour faire référence à la ligne de temps directement - pour cela, vous aurez besoin de l'index de la ligne de temps. pour cela vous auriez besoin de l'index de la ligne de temps. Vous pouvez, cependant, utiliser cette chaîne pour obtenir l'index de la ligne de temps en utilisant la chaîne retournée avec la fonction asset_get_index().

Syntaxe :

timeline_get_name(ind);

ArgumentsType Description
ind The index of the time line to check the name of.

Les retours :

Exemple :

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

Le code ci-dessus obtient le nom de la ligne de temps actuellement assignée et l'affiche dans la console.
