suppression de la chronologie

Avec cette fonction, vous pouvez supprimer n'importe quelle ligne de temps de votre jeu. Si cette ligne de temps a été créée dans le navigateur d'actifs de GameMaker, veuillez noter qu'elle est complètement supprimée du jeu et que si vous essayez de rappeler la ligne de temps indexée après avoir utilisé cette fonction, une erreur se produira. Si la ligne de temps a été créée dynamiquement à l'aide de la fonction timeline_add() cette fonction doit être utilisée dès que la ligne de temps n'est plus nécessaire afin d'éviter toute fuite de mémoire qui pourrait ralentir (et éventuellement faire planter) votre jeu.

Syntaxe :

timeline_delete(ind);

ArgumentsType Description
ind The index of the time line to delete.

Les retours :

Exemple :

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

Le code ci-dessus vérifie si une ligne de temps existe et est indexée dans la variable globale "tl" et si c'est le cas, il supprime cette ligne de temps.
