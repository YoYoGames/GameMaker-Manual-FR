timeline_moment_clear

Cette fonction permet d'effacer tous les codes et toutes les actions à un moment précis d'une ligne de temps préalablement définie.

Syntaxe :

timeline_moment_clear(ind, step);

Arguments Type Description
ind Actif chronologique The index of the timeline to clear.
step Real The moment to clear.

Les retours :

N/A

Exemple :

timeline_moment_clear(global.tl, room_speed * 30);

Le code ci-dessus effacera le moment spécifié de la ligne de temps indexée par la variable "global.tl".
