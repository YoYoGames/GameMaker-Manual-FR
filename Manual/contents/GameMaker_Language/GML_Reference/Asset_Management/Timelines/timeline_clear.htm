timeline_clear

Cette fonction permet d'effacer les "moments" d'une ligne temporelle spécifique, en supprimant tous les codes et actions de cette ligne temporelle et en la laissant vide.

Syntaxe :

timeline_clear(ind);

Arguments Type Description
ind Actif chronologique The index of the timeline to clear.

Les retours :

N/A

Exemple :

if timeline_position > 200
{
    timeline_clear(global.tl);
    timeline_index = -1;
}

Le code ci-dessus effacera la ligne de temps spécifiée indexée par la variable "global.tl" de tous les moments où un moment spécifique a été dépassé.
