timeline_exists

Avec cette fonction, vous pouvez vérifier si une ligne de temps existe (retour true) ou non (retour false). Ceci est particulièrement utile lors de la création dynamique de lignes de temps à l'aide de la fonction timeline_add() mais vous devez noter que si vous recherchez une variable non initialisée (qui serait autrement affectée à l'index d'une ligne de temps), une erreur sera générée.

Syntaxe :

timeline_exists(ind);

ArgumentsType Description
ind The index of the time line to check for.

Les retours :

Exemple :

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

Le code ci-dessus vérifie si une ligne de temps existe et est indexée dans la variable globale "tl" et si c'est le cas, il supprime cette ligne de temps.
