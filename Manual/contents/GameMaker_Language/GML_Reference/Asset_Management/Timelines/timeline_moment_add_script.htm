timeline_moment_add_script

Avec cette fonction, vous pouvez ajouter dynamiquement une fonction de script aux lignes de temps à n'importe quel "moment" de cette ligne de temps, où un "moment" est l'équivalent d'un tick (ou pas) de jeu. De cette façon, vous pouvez créer une nouvelle ligne de temps en utilisant la fonction timeline_add() et ajouter des comportements différents à n'importe quel moment, ou simplement modifier une ressource ligne de temps précédemment créée avec de nouveaux comportements. Notez que la fonction ne peut pas exiger d'arguments supplémentaires lors de son utilisation, et si vous l'utilisez pour modifier une ressource ligne de temps présente dans le navigateur d'actifs, alors toutes les instances qui utilisent cette ligne de temps seront affectées, et le changement durera jusqu'à la fin du jeu (appeler game_restart() ne réinitialisera pas non plus le changement).

Syntaxe :

timeline_moment_add_script(ind, step, script);

Arguments Type Description
ind Actif chronologique The index of the timeline to add a moment to.
step Real The moment (step) to add to.
script Fonction de script The index of the script function to add into the moment.

Les retours :

N/A

Exemple :

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

Le code ci-dessus va créer une nouvelle ligne de temps et stocker son index dans la variable "global.tl". Il ajoutera ensuite trois scripts à la ligne de temps, choisis au hasard, à intervalles d'une minute.
