taille de la ligne de temps

Avec cette fonction, vous pouvez obtenir le nombre total de moments actifs pour une ligne de temps (un moment "actif" est un moment auquel du code ou GML Visual a été ajouté). Cela peut être pratique lors de la création de lignes de temps dynamiques en utilisant les fonctions timeline_moment_add_script() et timeline_moment_clear() car vous pouvez vérifier si une ligne de temps donnée a le nombre correct de moments actifs ou aucun.

Syntaxe :

timeline_size(ind);

ArgumentsType Description
ind The index of the timeline get the information from.

Les retours :

Exemple :

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

Le code ci-dessus vérifie la taille de la timeline donnée, et s'il retourne 0 (c'est-à-dire que la timeline n'a pas de moments actifs), il ajoutera une fonction de script à utiliser à un moment aléatoire dans la timeline indexée dans la variable globale "tl".
