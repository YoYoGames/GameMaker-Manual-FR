chronologie_fonctionnement

Cette variable contient l'état actuel de la ligne temporelle assignée et renvoie true si elle est en cours et false si elle ne l'est pas. Vous pouvez également définir cette variable sur true ou false pour démarrer et arrêter la ligne temporelle à tout moment. Il convient de noter qu'une ligne temporelle arrêtée n' est pas réinitialisée, et donc la redémarrer à un moment ultérieur la fera repartir du moment exact où elle a été arrêtée.

Syntaxe :

timeline_running;

Les retours :

Exemple :

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

Le code ci-dessus vérifiera si l'instance exécute une ligne de temps, et si ce n'est pas le cas, il réinitialisera la ligne de temps assignée pour qu'elle démarre au premier moment, puis la lancera.
