timeline_add

Avec cette fonction, vous pouvez ajouter une nouvelle ligne de temps (vide) dans votre jeu. La fonction renvoie l'index de cette nouvelle ligne de temps qui doit être stockée dans une variable pour être utilisée dans tous les autres appels de fonction qui impliquent cette nouvelle ligne de temps. Vous devez également vous assurer d'utiliser la fonction timeline_delete() lorsque vous ne souhaitez plus utiliser la ligne de temps (comme à la fin de la pièce) afin d'éviter toute fuite de mémoire qui pourrait ralentir ou faire planter votre jeu. Pour ajouter des moments à une ligne de temps créée de cette façon, voir la fonction timeline_moment_add_script().

Syntaxe :

timeline_add();

Les retours :

Timeline Asset

Exemple :

global.tl = timeline_add();

Le code ci-dessus crée une nouvelle ligne de temps et stocke son index dans la variable "global.tl".

Retour : Lignes du temps
