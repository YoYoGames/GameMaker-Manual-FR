tampon_get_adresse

Avec cette fonction, vous pouvez obtenir un pointeur vers l'adresse brute et alignée buffer. access Ceci est principalement destiné à être utilisé avec les extensions car vous pouvez leur transmettre cette valeur, leur permettant ainsi d'accéder aux données de buffer. Notez qu'il y a quelques choses que vous ne pouvez pas faire avec l'adresse buffer lorsque vous y accédez depuis une extension :

  • Vous ne pouvez pas libérer la mémoire.
  • Vous ne pouvez pas redimensionner la mémoire.
  • Vous ne pouvez pas écrire au-delà de l'adresse buffer plus la taille buffer (vous pouvez utiliser la fonction buffer_get_size() pour cela) ou vous obtiendrez une erreur hors limites.

Syntaxe :

buffer_get_address(buffer);

ArgumentsType Description
buffer The index of the buffer to use.

Les retours :

Exemple :

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

Le code ci-dessus obtiendra l'adresse mémoire de buffer indexée dans la variable "buff_model" et obtiendra ensuite l'adresse mémoire de la fin de buffer en utilisant la fonction buffer_get_size(), en stockant les deux valeurs dans des variables locales pour une utilisation ultérieure. pour une utilisation ultérieure.
