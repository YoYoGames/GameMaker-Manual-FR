recherche de tampon

Cette fonction peut être utilisée pour se déplacer dans un fichier buffer, en trouvant le début, la fin ou une position relative à celle qui a été utilisée en dernier lors de la lecture ou de l'écriture des données. La valeur "offset" est le décalage (en octets) à ajouter à la position de recherche donnée, par exemple, si la base est relative et l'offset est 4, alors la position buffer se déplacera de 4 octets à partir de sa position actuelle. Veuillez noter ce qui suit :

  • Vous pouvez utiliser des valeurs négatives pour le décalage à rechercher dans le buffer ainsi que des valeurs positives.
  • Si le site buffer est de type "enveloppant" et que vous dépassez la fin du site buffer, la position de recherche sera également enveloppée.
  • Si le site buffer n'est pas de type "wrap", la recherche se fera au début ou à la fin du site buffer, même si le décalage fait sortir la recherche des limites du site buffer.

Les constantes suivantes sont acceptées comme argument "de base" pour chercher à :

Constante de recherche de la mémoire tampon
Constant Description
buffer_seek_start Le début du tampon
buffer_seek_relative Une position relative à la position actuelle de lecture/écriture.
buffer_seek_end La fin du tampon

Syntaxe :

buffer_seek(buffer, base, offset);

Arguments Type Description
buffer ID de la mémoire tampon The index of the buffer to use.
base Real The base position to seek.
offset Real The data offset value.

Les retours :

N/A

Exemple :

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

Le code ci-dessus trouve le début du site buffer avec l'identifiant stocké dans la variable "buff" et y écrit une série de valeurs entières signées de 16 bits.
