tampon_get_taille

Cette fonction permet d'obtenir la taille de l'adresse buffer en octets.

Syntaxe :

buffer_get_size(buffer);

Arguments Type Description
buffer ID de la mémoire tampon The index of the buffer to get the size of.

Les retours :

Real

Exemple :

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

Le code ci-dessus va créer un nouveau buffer et stocker son index dans la variable locale "_temp", la taille de ce nouveau buffer étant la même que celle du buffer indexé dans la variable "player_data".
