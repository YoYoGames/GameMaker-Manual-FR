tampon_suppression

Cette fonction permet de supprimer de la mémoire un buffer précédemment créé à l'aide de la fonction buffer_create() de la mémoire, en libérant les ressources utilisées pour le créer et en supprimant toutes les données qu'il peut contenir actuellement.

REMARQUE: il est important de toujours retirer de la mémoire les ressources créées dynamiquement lorsque vous n'en avez plus besoin afin d'éviter les fuites de mémoire.

IMPORTANT ! Lorsque vous créez un buffer, la valeur d'index utilisée pour l'identifier est une valeur entière commençant à 0. Ces index sont réutilisés par GameMaker, de sorte qu'une valeur d'index détruite de buffer peut être utilisée par une nouvelle valeur créée par la suite, Nous vous recommandons de donner la valeur -1 à toute variable contenant un indice de tampon après avoir détruit le site buffer.

Syntaxe :

buffer_delete(buffer)

ArgumentsType Description
buffer The index of the buffer to delete.

Les retours :

Exemple :

buffer_delete(player_buffer);
player_buffer = -1;

Le code ci-dessus supprime le site buffer précédemment créé avec l'identifiant stocké dans la variable "player_buffer" et donne à cette variable la valeur -1.
