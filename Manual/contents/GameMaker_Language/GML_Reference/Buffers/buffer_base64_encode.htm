tampon_base64_encode

Cette fonction convertit les données de l'adresse buffer en une chaîne encodée au format base64. Il s'agit d'un schéma d'encodage couramment utilisé pour tout média devant être stocké ou transféré sur Internet sous forme de texte, et qui rend la sortie illisible à l'œil humain. rend la sortie illisible à l'œil humain. Pour l'utiliser, vous devez spécifier un buffer déjà créé, la valeur de décalage (qui est le point dans le buffer auquel vous souhaitez commencer l'encodage) ainsi que la taille, en octets, de la mémoire buffer à encoder.

Syntaxe :

buffer_base64_encode(buffer, offset, size);

ArgumentsType Description
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Les retours :

Exemple :

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

Le code ci-dessus va créer le codage des données complètes stockées dans le buffer indexé par la variable "buff", et stocker la chaîne retournée dans la variable locale "b_str".
