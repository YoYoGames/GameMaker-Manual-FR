tampon_get_alignement

Avec cette fonction, vous pouvez obtenir l'alignement d' un octet pour l'ID buffer donné.

Syntaxe :

buffer_get_alignment(buffer);

ArgumentsType Description
buffer The index of the buffer to check.

Les retours :

Exemple :

alignment = buffer_get_alignment(buff);

Le code ci-dessus récupère l'alignement de buffer à partir de la valeur indexée dans la variable "buff" et le stocke dans une variable.
