buffer_read

Cette fonction peut être utilisée pour lire les données d'un site buffer créé au préalable. La valeur de retour dépendra du type de données que vous lisez, qui est lui-même défini par les constantes suivantes :

Constant Description
buffer_u8 Un nombre entier non signé de 8 bits. Il s'agit d'une valeur positive comprise entre 0 et 255.
buffer_s8 Un nombre entier signé de 8 bits. Il peut s'agir d'une valeur positive ou négative comprise entre -128 et 127 (0 est considéré comme positif).
buffer_u16 Un nombre entier non signé de 16 bits. Il s'agit d'une valeur positive comprise entre 0 et 65 535.
buffer_s16 Un nombre entier signé de 16 bits. Il peut s'agir d'une valeur positive ou négative comprise entre -32 768 et 32 767 (0 est considéré comme positif).
buffer_u32 Un nombre entier non signé de 32 bits. Il s'agit d'une valeur positive comprise entre 0 et 4 294 967 295.
buffer_s32 Un nombre entier signé de 32 bits. Il peut s'agir d'une valeur positive ou négative comprise entre -2 147 483 648 et 2 147 483 647 (0 étant considéré comme positif).
buffer_u64 Un nombre entier non signé de 64 bits.
buffer_f16 Un flottant de 16 bits. Il peut s'agir d'une valeur positive ou négative comprise dans la plage de +/- 65504. (Non pris en charge actuellement !)
buffer_f32 Un flottant de 32 bits. Il peut s'agir d'une valeur positive ou négative comprise entre +/-16777216.
buffer_f64 Un flotteur 64 bits.
buffer_bool Une valeur booléenne. Ne peut être que 1 ou 0 (true ou false)
buffer_string Une corde de n'importe quelle taille.
buffer_text Une chaîne de caractères de n'importe quelle taille, sans le caractère final nul.

Si la fonction réussit, elle renvoie une valeur du type donné, mais si elle échoue, elle provoque une erreur runner.

REMARQUE L'utilisation d'un type de données incorrect pour les données lues entraînera le renvoi de valeurs erronées.

Syntaxe :

buffer_read(buffer, type)

Arguments Type Description
buffer ID de la mémoire tampon The index of the buffer to read from.
type Type de données de la mémoire tampon Constante The type of data that is to be read from the buffer (see the list of constants above).

Les retours :

Real, Boolean or String

Exemple :

var cmd = buffer_read(buff, buffer_s16);

Le code ci-dessus lit à partir du site buffer avec l'identifiant stocké dans la variable "buff" une valeur signée de 16 bits dans la variable locale "cmd".
