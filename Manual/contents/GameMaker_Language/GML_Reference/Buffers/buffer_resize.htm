redimensionnement du tampon

Cette fonction vous permet de redimensionner un site buffer pour qu'il ait la taille (en octets) que vous spécifiez.

Syntaxe :

buffer_resize(buffer, newsize);

ArgumentsType Description
buffer The index of the buffer to change the size of.
newsize The new size of the buffer (in bytes).

Les retours :

Exemple :

if (buffer_get_size(buff) < 16384)
{
    buffer_resize(buff, 16384);
}

Le code ci-dessus vérifiera la taille de l'adresse buffer indexée dans la variable "buff" et si elle est inférieure à la valeur donnée, l'adresse buffer sera redimensionnée.
