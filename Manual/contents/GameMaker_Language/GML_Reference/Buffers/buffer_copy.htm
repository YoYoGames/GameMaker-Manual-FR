buffer_copy

Cette fonction peut être utilisée pour copier un segment (ou la totalité) des données stockées dans un buffer vers un autre. Lorsque vous utilisez deux buffers et que vous copiez de l'un à l'autre, les deux buffers doivent avoir été préalablement créés à l'aide de la fonction buffer_create() (par exemple), et vous pouvez spécifier un décalage de données (en octets) pour le point de départ des données à copier à partir de la source buffer par rapport au début de buffer, ainsi qu'un autre décalage de données pour définir la position à laquelle copier les données dans la destination . dans la destination buffer.

REMARQUE: Vous ne pouvez pas copier sur le même site buffer.

Syntaxe :

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

ArgumentsType Description
src_buffer The index of the buffer to copy from.
src_offset The data offset to start copying from (in bytes).
size The size of the data to copy (in bytes).
dest_buffer The index of the buffer to copy to.
dest_offset The offset position to copy the data to (in bytes).

Les retours :

Exemple :

buffer_copy(buff1, 0, 2048, buff2, 2048);

Le code ci-dessus copiera les données stockées dans le buffer indexé dans la variable "buff1", puis les collera dans le buffer indexé dans la variable "buff2", mais décalé de 2048 octets.
