buffer_copy_from_vertex_buffer

Cette fonction peut être utilisée pour copier une partie (ou l'ensemble) des données de sommet stockées dans un sommet buffer dans un régulier buffer précédemment créé. Lors de la copie d'un sommet buffer dans un régulier buffer avec cette fonction, les deux buffers doivent avoir été créés au préalable (à l'aide des fonctions vertex_create_buffer() et buffer_create() par exemple). Vous pouvez spécifier la plage de données de sommet que vous souhaitez copier dans buffer, où le sommet de départ peut être n'importe où entre 0 et le nombre de sommets -1, et vous pouvez donner le nombre de sommets à copier à partir de ce point. Vous pouvez utiliser la fonction vertex_get_number() sur le sommet buffer pour obtenir le nombre total de sommets stockés. Enfin, vous indiquez l'index buffer dans lequel copier les données de sommet, ainsi qu'un décalage de données pour définir la position dans laquelle copier les données de sommet dans la destination buffer.

Syntaxe :

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Arguments Type Description
vertex_buffer Vertex Buffer ID The index of the vertex buffer to copy from.
start_vertex Real The starting vertex.
num_vertices Real The total number of vertices to use.
dest_buffer ID de la mémoire tampon The index of the buffer to copy to.
dest_offset Real The offset position to copy the data to (in bytes).

Les retours :

N/A

Exemple :

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

Le code ci-dessus copiera les données de vertex stockées dans le vertex buffer indexé dans la variable "model_buffer", puis les collera dans le buffer indexé dans la variable "player_buffer".
