buffer_set_surface

Avec cette fonction, vous pouvez écrire des informations d'un buffer vers une surface donnée. Le site buffer à partir duquel la lecture est effectuée et la surface sur laquelle l'écriture est effectuée doivent tous deux avoir été créés au préalable, et vous pouvez fournir un décalage dans le site buffer pour commencer la lecture. Notez que la lecture commencera toujours au début du site buffer plus la valeur du décalage et non à la position de recherche actuelle plus la valeur du décalage.

REMARQUE: cette fonction a été modifiée à partir de la mise à jour 2.3.1 de GameMaker. Les versions précédentes utilisaient la constante buffer_surface_copy, qui est désormais obsolète.

Syntaxe :

buffer_set_surface(buffer, surface, offset);

ArgumentsType Description
buffer The index of the buffer to use.
surface The index of the surface to use.
offset The data offset value.

Les retours :

Exemple :

buffer_set_surface(buff, application_surface, 0);

Le code ci-dessus copiera toutes les données stockées sur le site buffer indexées dans la variable "buff" sur la surface de l'application sans décalage.
