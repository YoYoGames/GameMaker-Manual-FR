remplissage de la mémoire tampon

Cette fonction peut être utilisée pour remplir un buffer précédemment créé avec un type de données et une valeur donnés. Les données avec lesquelles vous remplissez le buffer doivent être en accord avec l'argument "type" de cette fonction, ce qui signifie que vous ne pouvez pas essayer de remplir avec une chaîne de caractères et utiliser le type entier 16 bits non signé, par exemple. chaîne de caractères et utiliser le type "unsigned 16bit integer", par exemple. Les constantes de type sont les mêmes que celles utilisées par les fonctions buffer_read() et buffer_write() . La "taille" est la taille de l'adresse buffer (en octets) que vous souhaitez remplir, tandis que le décalage est la valeur du décalage (également en octets) à partir du début de l'adresse buffer pour commencer le remplissage.

Syntaxe :

buffer_fill(buffer, offset, type, value, size);

ArgumentsType Description
buffer The index of the buffer to fill.
offset The data offset value (in bytes).
type The type of data that is to be written to the buffer (see the list of constants here).
value The data to write.
size The size of the buffer (in bytes) that you wish to fill.

Les retours :

Exemple :

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

Le code ci-dessus trouve le début du site buffer avec l'identifiant stocké dans la variable "buff" et y écrit une série de valeurs entières signées de 16 bits.
