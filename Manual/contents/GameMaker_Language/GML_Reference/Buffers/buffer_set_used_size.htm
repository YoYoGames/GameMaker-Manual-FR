buffer_set_used_size

Cette fonction est principalement utilisée dans les extensions et vous permet de définir la taille "utilisée" de l'adresse buffer, qui correspond au nombre d'octets qui y ont été écrits.

Lorsque vous écrivez des données dans un buffer à partir d'une extension, GameMaker ne sait pas quelle part du buffer a été remplie par le code de l'extension et n'est pas en mesure de lire ces données. Cette fonction peut être appelée par l'extension pour indiquer au moteur combien d'octets de données ont été écrits sur le site buffer, afin que le moteur puisse lire ces données.

Syntaxe :

buffer_set_used_size(buffer, size);

ArgumentsType Description
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

Les retours :

Exemple :

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

Les fonctions ci-dessus seraient appelées dans une extension pour écrire certaines données sur un site GameMaker buffer (par le biais de son adresse mémoire), puis pour définir le nombre d'octets qui y ont été écrits afin que le moteur puisse lire ces données.
