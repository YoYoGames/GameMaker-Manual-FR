Taille de la mémoire tampon

Cette fonction renvoie la taille (en octets) de la constante de type de données de la mémoire tampon (listée ici).

Syntaxe :

buffer_sizeof(type);

Arguments Type Description
type Type de données de la mémoire tampon Constante The type of data that is to be checked (see the list of constants here).

Les retours :

Real

Exemple :

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

Le code ci-dessus calcule d'abord la taille du site buffer à créer en multipliant le type de données 8 bits non signées par 12 (puisque nous utiliserons le site buffer pour contenir 12 données), puis il utilise cette valeur pour définir un site buffer fixe à la taille correcte.
