tampon_save_ext

Cette fonction permet de sauvegarder dans un fichier une partie du contenu d'un buffer, prête à être relue en mémoire à l'aide de la fonction buffer_load() de la fonction. Le "offset" définit la position de départ dans la mémoire tampon pour la sauvegarde (en octets), et la "taille" est la taille de la zone buffer à sauvegarder à partir de ce décalage (également en octets).

Syntaxe :

buffer_save_ext(buffer, filename, offset, size);

ArgumentsType Description
buffer The index of the buffer to save.
filename The name of the file to save as.
offset The offset within the buffer to save from (in bytes).
size The size of the buffer area to save (in bytes).

Les retours :

Exemple :

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Enregistre dans un fichier une partie du contenu actuel du site buffer avec l'identifiant stocké dans la variable "buff".
