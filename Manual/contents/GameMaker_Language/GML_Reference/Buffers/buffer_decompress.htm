tampon_décompression

Avec cette fonction, vous pouvez décompresser un buffer précédemment compressé en utilisant la compression zlib. Vous fournissez l'ID buffer à décompresser, et la fonction renverra un nouvel ID buffer qui contient les données non compressées. Si la décompression a échoué (par exemple, vous fournissez un buffer qui n'a pas été compressé), la fonction retournera une valeur inférieure à 0.

Syntaxe :

buffer_decompress(buffer);

ArgumentsType Description
buffer The index of the buffer to decompress.

Les retours :

Buffer ID

Exemple :

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

Le code ci-dessus chargera d'abord un fichier buffer, puis le décompressera et enfin lira les données de la chaîne de caractères du fichier buffer décompressé dans une variable globale.
