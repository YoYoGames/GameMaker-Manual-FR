option_groupe_asynchrone_tampon

Cette fonction permet de définir certaines options spécifiques à la plate-forme pour le groupe buffer en cours de sauvegarde/chargement. Les options disponibles sont les suivantes :

  • "Subtitle" ou "slottitle" - La valeur de cette option est une chaîne de caractères qui sera affichée à l'utilisateur lorsqu'il gérera ses données de sauvegarde dans l'OS. Ceci n'est important que lors de la sauvegarde des données, pas lors du rechargement.
  • "showdialog" - La valeur de cette option est un booléen true ou false. Si elle est réglée sur true, GameMaker affichera l'interface utilisateur du système lors de la sauvegarde ou du chargement, sinon il effectuera un quicksave/quickload sans interface utilisateur. Normalement, vous n'avez besoin de cette option que si vous prenez en charge plusieurs emplacements de sauvegarde pour permettre à l'utilisateur de choisir un emplacement, mais si vous ne prenez en charge qu'un seul emplacement par utilisateur, définissez cette option sur false.
  • "savepadindex" - Pour cette option, vous devez spécifier l'index du pad du joueur qui sauvegarde ou charge et le système écrira et lira des données dans le dossier de sauvegarde de ce joueur.
  • "saveslotsize" - Cette option nécessite que vous spécifiiez la taille réelle en octets que vous souhaitez sauvegarder (afin que vous puissiez faire un buffer_seek() et buffer_tell() pour l'obtenir, par exemple). Notez qu'il n'est pas obligatoire de fournir cette valeur car toutes les sauvegardes sont pré-assignées à un espace minimum, qui varie généralement avec la plate-forme cible.
  • "ps_create_backup" - (PlayStation uniquement) - Définir cette option à true permet de s'assurer que les données de sauvegarde sont sauvegardées lorsque le répertoire de sauvegarde est démonté (c'est-à-dire lorsque la fonction buffer_async_group_end() est appelée). Consultez la documentation de la console pour plus d'informations.

IMPORTANT ! Cette fonction n'est actuellement valable que pour les modules cibles PSVita, PS4 et XBox One. Sur toutes les autres cibles, elle ne fera rien.

Syntaxe :

buffer_async_group_option(option, value);

Arguments Type Description
option Chaîne de caractères The option to set.
value Real ou Chaîne de caractères The value to set (can be a real or string, depending on the option).

Les retours :

N/A

Exemple :

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

Le code ci-dessus démarre un groupe buffer puis définit les options du groupe avant de définir 4 fichiers à sauvegarder de manière asynchrone. La définition du groupe est ensuite terminée (à ce moment-là, la sauvegarde commence).
