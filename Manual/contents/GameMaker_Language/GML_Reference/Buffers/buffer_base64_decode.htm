buffer_base64_decode

Avec cette fonction, vous pouvez décoder une chaîne de caractères encodée en base64 (créée en utilisant la fonction buffer_base64_encode() ) dans un buffer. Cette fonction créera le buffer (comme un tampon "grow" aligné sur 1 octet) et renverra l'index unique pour le buffer qui devra être utilisé dans tous les autres appels de fonction.

REMARQUE: Il est important de supprimer de la mémoire toute ressource créée dynamiquement comme celle-ci lorsque vous n'en avez plus besoin afin d'éviter les fuites de mémoire. Ainsi, lorsque vous en avez terminé avec le site buffer que vous avez créé, vous devez le libérer à nouveau à l'aide de la fonction buffer_delete().

Syntaxe :

buffer_base64_decode(string);

ArgumentsType Description
string The base64 encoded string to decode

Les retours :

Buffer ID

Exemple :

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

Le code ci-dessus ouvrira un fichier ini, puis lira une chaîne de caractères dans la fonction de décodage. Cette fonction renvoie un indice buffer, qui est stocké dans la variable "buff", contenant les données précédemment encodées et sauvegardées. Le fichier ini est ensuite fermé.
