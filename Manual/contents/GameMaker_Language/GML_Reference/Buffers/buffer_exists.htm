tampon_existant

Cette fonction peut être utilisée pour vérifier une variable afin de voir si elle contient une valeur d'identification valide buffer ou non. Si c'est le cas, la fonction renvoie true, sinon elle renvoie false.

Syntaxe :

buffer_exists(buffer)

ArgumentsType Description
buffer The index of the buffer to check.

Les retours :

Exemple :

if buffer_exists(buff)
{
    buffer_delete(buff);
}

Le code ci-dessus vérifie si la variable "buff" contient un identifiant buffer et si c'est le cas, le buffer est supprimé.
