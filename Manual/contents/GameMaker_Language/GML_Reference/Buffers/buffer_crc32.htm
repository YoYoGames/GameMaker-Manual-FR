buffer_crc32

Cette fonction prend les données d'entrée d'un buffer et renvoie un hachage de somme de contrôle crc32. Vous spécifiez l'ID buffer du buffer à utiliser, puis une valeur de décalage (en octets) pour savoir où commencer, et enfin une taille (toujours en octets) pour la région à hacher, et la fonction renvoie une valeur entière de 32 bits pour cette région.

Syntaxe :

buffer_crc32(buffer, offset, size);

ArgumentsType Description
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Les retours :

Exemple :

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

Le code ci-dessus créera une valeur de hachage de somme de contrôle crc32 pour les données complètes stockées sur le site buffer indexé par la variable "buff", et stockera la valeur de hachage entière retournée dans la variable "check_val".
