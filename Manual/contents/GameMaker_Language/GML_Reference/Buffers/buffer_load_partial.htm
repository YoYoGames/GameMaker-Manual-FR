Chargement partiel de la mémoire tampon

Cette fonction chargera certaines des données buffer qui ont été précédemment sauvegardées à l'aide des fonctions buffer_save() dans un buffer existant. Vous indiquez l'identifiant du fichier buffer précédemment créé dans lequel vous souhaitez charger les données, puis le fichier buffer sauvegardé à charger, et enfin le décalage par rapport au début du fichier buffer (en octets) à partir duquel vous souhaitez charger les données. Les arguments suivants servent à définir la longueur des données de buffer (en octets) à partir du point de décalage initial que vous souhaitez charger et le point de décalage vers lequel charger les données dans buffer (à nouveau, en octets).

Veuillez lire la page buffer_load() pour les notes spécifiques à la plate-forme.

Syntaxe :

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

ArgumentsType Description
buffer The index of the buffer to load into (destination).
filename The name of the file to load from (source).
offset The offset within the destination buffer to load to (in bytes).
src_len The length of the part of the source buffer to load (in bytes).
dest_offset The offset where to start putting the partial data in the destination buffer (in bytes).

Les retours :

N/A

Exemple :

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

Le code ci-dessus va créer un nouveau site "grow" buffer et y charger une partie des données sauvegardées dans le fichier "save.dat".
