buffer_poke

Avec la fonction buffer_write() vous pouvez écrire des données sur le site buffer à la position de recherche actuelle, chaque donnée faisant avancer cette position du nombre d'octets écrits ou lus. Cependant, il peut être nécessaire de modifier un élément de données donné sans vouloir changer la position de recherche actuelle, et c'est dans ce cas que vous utiliserez cette fonction. Il suffit de fournir à la fonction un index buffer, puis la position de décalage par rapport au début de buffer (en octets) à l'intérieur de buffer pour écrire, ainsi que le type de données et la valeur à écrire.

Syntaxe :

buffer_poke(buffer, offset, type, value);

Arguments Type Description
buffer ID de la mémoire tampon The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to write the given data to.
type Type de données de la mémoire tampon Constante The type of data that is to be written to the buffer (see the list of constants here).
value Variable The data to add to the buffer, in accordance with the type specified.

Les retours :

N/A

Exemple :

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

Le code ci-dessus ajoutera la valeur de la composante bleue de la couleur utilisée pour le mélange d'images dans le site buffer indexé dans la variable "buff", à la troisième position dans le site buffer et en tant que valeur 8bit non signée.
