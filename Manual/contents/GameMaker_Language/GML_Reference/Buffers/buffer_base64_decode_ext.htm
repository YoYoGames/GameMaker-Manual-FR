tampon_base64_decode_ext

Avec cette fonction, vous pouvez décoder une chaîne de caractères codée en base64 (créée à l'aide de la fonction buffer_base64_encode() ) en buffer. Contrairement à la fonction buffer_base64_decode(), cette fonction ne créera pas un buffer pour vous, mais vous devriez déjà avoir créé le buffer (voir buffer_create()), dont vous utiliserez l'identifiant avec cette fonction. Le "offset" est la position dans le buffer pour décoder la chaîne donnée (en octets).

Syntaxe :

buffer_base64_decode_ext(buffer, string, offset);

ArgumentsType Description
buffer The index of the buffer to decode the string into.
string The base64 encoded string to decode.
offset The data offset value.

Les retours :

Exemple :

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

Le code ci-dessus va créer un buffer et stocker son identifiant unique dans la variable "buff", puis ouvrir un fichier ini et lire une chaîne de caractères dans la variable locale "str". Cette chaîne est ensuite décodée dans le nouveau buffer avant de refermer le fichier ini. avant de refermer le fichier ini.
