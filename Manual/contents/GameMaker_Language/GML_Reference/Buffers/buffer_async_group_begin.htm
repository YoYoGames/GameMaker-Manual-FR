tampon_async_groupe_début

Cette fonction est appelée lorsque vous voulez commencer l'enregistrement de plusieurs buffers dans plusieurs fichiers (ou le chargement de plusieurs buffers). Le nom de groupe est une chaîne de caractères et sera utilisé comme nom de répertoire pour l'endroit où les fichiers sont sauvegardés, et devrait être utilisé comme partie du chemin d'accès au fichier lorsque vous rechargerez les fichiers dans le IDE plus tard (en utilisant l'une des fonctions buffer_load() ). Cette fonction ne doit être utilisée qu' avec les fonctions buffer_save_async() et buffer_load_async() et vous devez également terminer le groupe asynchrone en appelant la fonction buffer_async_group_end() sinon les fichiers ne seront pas sauvegardés/chargés.

Notez que pour les plateformes de console (comme la PS4 par exemple), le nom du groupe sera utilisé comme description de l'emplacement de sauvegarde, et l'utilisation de cette fonction peut vous éviter d'avoir l'interface utilisateur affichée pour chaque fichier en cours de sauvegarde. Notez également que lorsque vous utilisez UWP, vous pouvez faire circuler vos sauvegardes entre les fenêtres natives Xbox et Microsoft Store en créant un groupe identique à un groupe ERA Xbox, par exemple :

if os_type == os_uwp
{
    buffer_async_group_begin("root/yourgroupname");
}
else
{
    buffer_async_group_begin("yourgroupname");
}

Syntaxe :

buffer_async_group_begin(groupname);

Arguments Type Description
groupname Chaîne de caractères The name of the group (as a string).

Les retours :

N/A

Exemple :

buffer_async_group_begin("SaveGame");
save1 = buffer_save_async(buff1, "Player_Save1.sav", 0, 16384);
save2 = buffer_save_async(buff2, "Player_Save2.sav", 0, 16384);
save3 = buffer_save_async(buff3, "Player_Save3.sav", 0, 16384);
save4 = buffer_save_async(buff4, "Player_Save4.sav", 0, 16384);
buffer_async_group_end();

Le code ci-dessus démarre un groupe buffer puis le configure pour sauvegarder 4 fichiers de manière asynchrone. La définition du groupe est ensuite terminée (à ce moment-là, la sauvegarde commence).
