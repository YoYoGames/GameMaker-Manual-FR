buffer_load_ext

Cette fonction chargera les données buffer qui ont été précédemment sauvegardées à l'aide des fonctions buffer_save() dans un buffer existant. Vous devez indiquer l'ID du fichier buffer précédemment créé dans lequel vous souhaitez charger les données, puis le fichier buffer sauvegardé dans lequel vous souhaitez charger les données, et enfin le décalage par rapport au début du fichier buffer (en octets) dans lequel vous souhaitez charger les données.

Veuillez lire la page buffer_load() pour les notes spécifiques à la plate-forme.

Syntaxe :

buffer_load_ext(buffer, filename, offset);

ArgumentsType Description
buffer The index of the buffer to load into.
filename The name of the file to load from.
offset The offset within the buffer to load to (in bytes).

Les retours :

Exemple :

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

Le code ci-dessus récupère d'abord la position de la fin du buffer indexée dans la variable "player_buffer" et charge ensuite les données de la donnée dans cette position (notez que cet exemple ne fonctionnera qu'avec les types "grow" ou "wrap" buffer ).
