tampon_compressé

Avec cette fonction, vous pouvez compresser une partie (ou la totalité) d'un buffer en utilisant la compression zlib. Vous fournissez l'ID de buffer à compresser (tel que retourné par buffer_create()), le décalage dans le buffer à utiliser en octets, et la taille des données du buffer à compresser (également en octets). La fonction renverra une nouvelle valeur d'ID buffer pour le buffer compressé, ou une valeur inférieure à 0 si elle a échoué pour une raison quelconque. Cette fonction ne modifiera pas l'adresse originale buffer.

Syntaxe :

buffer_compress(buffer, offset, size);

ArgumentsType Description
buffer The index of the buffer to compress.
offset The offset within the buffer to compress (in bytes).
size The size of the buffer area to compress (in bytes).

Les retours :

Buffer ID

Exemple :

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

Le code ci-dessus va créer un buffer puis le remplir avec les données d'une chaîne. Ce buffer est ensuite compressé et sauvegardé, et la source et le buffers compressé sont supprimés.
