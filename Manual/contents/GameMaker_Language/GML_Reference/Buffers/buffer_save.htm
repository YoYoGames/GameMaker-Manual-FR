Sauvegarde de la mémoire tampon

Cette fonction permet de sauvegarder le contenu d'un buffer dans un fichier, prêt à être relu en mémoire à l'aide de la fonction buffer_load() fonction.

REMARQUE: Sous HTML5, le contenu du site buffer sera enregistré sous forme de chaînes codées en base64 lorsque cette fonction est utilisée.

Syntaxe :

buffer_save(buffer, filename);

ArgumentsType Description
buffer The index of the buffer to save.
filename The name of the file to save as.

Les retours :

Exemple :

buffer_save(buff, "Player_Save.sav");

Enregistre dans un fichier le contenu actuel du site buffer avec l'identifiant stocké dans la variable "buff".
