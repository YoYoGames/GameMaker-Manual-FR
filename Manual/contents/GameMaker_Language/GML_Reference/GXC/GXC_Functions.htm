Opéra GX et GXC

Cette section décrit les fonctions fournies pour la plate-forme cible Opera GX. Les fonctions répertoriées sur cette page vous permettent d'enregistrer votre écran et de le télécharger sur le GXC avec votre score au défi, mais notez que vous devez utiliser la bibliothèque API GXC pour y parvenir. Veuillez consulter la section Opera GX sur le site YoYo Games Helpdesk pour obtenir des instructions sur la mise en œuvre de la bibliothèque API GXC et le travail avec les défis.

Les fonctions suivantes sont prévues pour le téléchargement d'enregistrements vidéo avec défis :

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_enregistrement
  • gxc_resume_movie_recording
