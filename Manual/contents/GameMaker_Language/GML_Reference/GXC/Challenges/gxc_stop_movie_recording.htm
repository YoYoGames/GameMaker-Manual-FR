gxc_stop_movie_recording

Cette fonction est utilisée pour arrêter un enregistrement GXC qui a été précédemment lancé avec gxc_start_movie_recording(). Elle soumet également l'enregistrement à l'URL spécifié dans les arguments, cependant le comportement est différent selon le type de chaîne spécifié :

  • Si vous spécifiez un nom de fichier, l'enregistrement sera sauvegardé sous ce nom dans votre dossier Téléchargements.
  • Si vous spécifiez une URL, l'enregistrement sera téléchargé vers cette URL.

La méthode URL doit être utilisée lorsque l'enregistrement doit être téléchargé vers le GXC, et le fichier doit être sauvegardé localement uniquement pour vérifier si l'enregistrement a réussi.

Syntaxe :

gxc_stop_movie_recording(filename);

Arguments Type Description
filename Chaîne de caractères The file name or URL where the recording is to be saved or uploaded

Les retours :

N/A

Exemple :

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

Le code ci-dessus utilise la bibliothèque API GXC pour charger un score dans le défi actuellement actif. Lorsqu'une réponse est reçue dans sa méthode de rappel, il récupère l'URL de chargement de la vidéo et y charge l'enregistrement.
