gxc_resume_movie_recording

Cette fonction est utilisée pour reprendre un enregistrement GXC qui a été précédemment mis en pause avec gxc_pause_movie_recording().

Syntaxe :

gxc_resume_movie_recording();

Les retours :

Exemple :

if (keyboard_check_pressed(vk_enter))
{
    gxc_resume_movie_recording();
}

Cette fonction permet de reprendre l'enregistrement de la vidéo GXC actuellement en pause si vous appuyez sur la touche Entrée.
