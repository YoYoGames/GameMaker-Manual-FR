gxc_start_movie_recording

Cette fonction est utilisée pour démarrer l'enregistrement de votre défi sur GXC. Vous spécifiez la largeur, la hauteur et la fréquence d'images de l'enregistrement, ainsi qu'un booléen pour savoir si l'audio doit être enregistré ou non.

Syntaxe :

gxc_start_movie_recording(width, height, fps, record_audio);

Arguments Type Description
width Real The width of the recording
height Real The height of the recording
fps Real The frame rate (frames-per-second) of the recording
record_audio Booléen Whether audio should be recorded (true) ou non (false)

Les retours :

N/A

Exemple :

gxc_start_movie_recording(320, 240, 4, false);

Cela lancera un enregistrement vidéo à une résolution de 320x240 à 4 images par seconde, sans qu'aucun son ne soit enregistré.
