scheduler_resolution_set

Cette fonction est utilisée pour changer la résolution de l'ordonnanceur de threads de Windows. La valeur de la résolution fournie dans l'argument doit être en millisecondes.

Par défaut, GameMaker utilise une résolution de 1 milliseconde, qui est la plus petite résolution autorisée et donne les meilleures performances. Modifier la résolution à une valeur plus élevée peut affecter négativement les performances, mais cela peut également réduire la consommation d'énergie.

Vous pouvez fournir -1 comme argument pour rétablir la résolution à sa valeur par défaut (telle que définie par Windows). Notez que la modification de la résolution de l'ordonnanceur aura un impact sur tous les processus exécutés en même temps que votre jeu.

REMARQUE: Cette fonction est réservée à Windows .

Syntaxe :

scheduler_resolution_set(milliseconds);

ArgumentsType Description
milliseconds The new resolution value (in milliseconds) or -1 for default

Les retours :

Exemple :

scheduler_resolution_set(2);

Cet exemple définit la résolution du planificateur de threads Windows à 2 millisecondes.
