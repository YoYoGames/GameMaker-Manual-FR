appel_externe

Si vous avez créé un appel de fonction externe vers un dll ou un dylib à l'aide de external_define()vous pouvez utiliser cette fonction pour l'appeler. Vous fournissez le nom de la fonction précédemment définie ainsi que chacun des arguments qu'elle requiert (chaque argument doit être du bon type, réel ou chaîne) et la fonction renvoie le résultat de l'appel externe.

Syntaxe :

external_call(id, args[0...15]);

Arguments Type Description
id Fonction externe The name of the function that you want to call
args[0...10] Réel ou Chaîne The different arguments that you want to pass through to the external function

Les retours :

Variable (the type of value returned will depend on the defined function)

Exemple :

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

L'exemple de code ci-dessus appelle une fonction externe préalablement définie et stocke la valeur renvoyée dans une variable locale.
