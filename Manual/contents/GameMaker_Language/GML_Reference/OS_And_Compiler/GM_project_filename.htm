GM_project_filename

Cette constante contient le chemin d'accès complet et le nom du fichier du projet YYP.

Syntaxe :

GM_project_filename;

Les retours :

String

Exemple :

draw_text(32, 32, GM_project_filename);

Le code ci-dessus dessine la valeur de la constante du nom de fichier du projet GM.
