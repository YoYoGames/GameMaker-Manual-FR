GM_build_type

Cette chaîne de caractères constante contient le type de construction : "exe" pour un exécutable créé ou "run" pour un test lancé depuis le site IDE.

Syntaxe :

GM_build_type;

Les retours :

String

Exemple :

draw_text(32, 32, GM_build_type);

Le code ci-dessus affichera le type de construction de l'exécutable en cours d'exécution.
