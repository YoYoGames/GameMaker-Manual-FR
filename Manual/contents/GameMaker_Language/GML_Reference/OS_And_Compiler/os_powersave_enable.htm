os_powersave_enable

Cette fonction vous permet d'activer ou de désactiver les fonctions d'économie d'énergie de l'appareil. Ceci est important car certains jeux (par exemple ceux qui utilisent les fonctions d'inclinaison) peuvent ne pas générer d'événements que le système d'exploitation peut interpréter comme une entrée utilisateur et ainsi éteindre l'écran ou quitter le jeu. En réglant cette fonction sur false, vous pouvez désactiver les fonctions d'économie d'énergie et vous assurer que l'écran (et le jeu) fonctionne toujours.

NOTE: Ceci est limité aux cibles iOS et Android.

Syntaxe :

os_powersave_enable(flag)

ArgumentsType Description
flag Set to true ou false pour activer ou désactiver les fonctions d'économie d'énergie.

Les retours :

Exemple :

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

Le code ci-dessus vérifie le type de système d'exploitation et s'il s'agit de Android ou de iOS, les fonctions d'économie d'énergie sont désactivées.
