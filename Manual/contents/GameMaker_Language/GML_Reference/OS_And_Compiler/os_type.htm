os_type

Cette variable en lecture seule contient une des diverses constantes dont dispose GameMaker pour vous indiquer pour quel système d'exploitation le jeu a été créé. Notez que ce n' est pas nécessairement le même système d'exploitation que celui de l'appareil qui l'exécute, car, par exemple, votre jeu peut fonctionner sur un système d'exploitation Amazon Fire, mais il a été créé pour la plate-forme Android (dans ce cas, os_type sera os_android).

Les constantes suivantes peuvent être retournées :

OS Type Constant
Constant Description
os_windows Système d'exploitation Windows
os_uwp Plate-forme universelle Windows 10 Windows
os_operagx Opéra GX
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos Apple tvOS
os_android Android
os_ps4 Sony PlayStation 4
os_ps5 Sony PlayStation 5
os_xboxone Microsoft Xbox One
os_xboxseriesxs Microsoft Xbox Série X/S
os_switch Nintendo Switch
os_unknown OS inconnu

Syntaxe :

os_type;

Les retours :

OS Type Constant

Exemple :

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

Le code ci-dessus vérifie le système d'exploitation qui exécute le jeu et définit une variable globale en conséquence.
