os_get_region

Cette fonction renvoie une chaîne contenant le code régional à deux ou trois lettres du système d'exploitation qui exécute le jeu, conformément à la norme ISO3166-1. Si cette information n'est pas disponible, elle contiendra simplement une chaîne vide "".

REMARQUE: Ce n'est pas le code régional de l'emplacement qui est renvoyé, mais le code de langue régionale du système d'exploitation.

Syntaxe :

os_get_region()

Les retours :

Exemple :

switch (os_get_language())
{
    case "zh":
        var _region = os_get_region();
        if (_region == "HK" || _region == "MO" || _region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

Le code ci-dessus vérifie la langue du système d'exploitation et s'il s'agit du chinois, il vérifie ensuite la région du système d'exploitation, ouvrant un fichier *.ini différent en fonction des valeurs renvoyées.
