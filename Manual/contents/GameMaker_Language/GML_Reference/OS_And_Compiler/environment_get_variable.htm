environnement_get_variable

Cette fonction renvoie la valeur (une chaîne) de la variable d'environnement portant le nom donné (également une chaîne). Vous pouvez obtenir les variables d'environnement disponibles sur macOS et Ubuntu (Linux) en tapant "env" dans l'application terminal. informations sur les variables d'environnement de Windows, si vous utilisez l'invite de commande, tapez "echo %PATH%", et en utilisant PowerShell, c'est "ls env". Notez que sur macOS et Ubuntu (Linux), la variable d'environnement "HOME" renvoie la variable d'environnement " ". renvoie le chemin d'accès "~/", qui correspond à "/Users/<username>" sur macOS et à "/home/<username>" sur Ubuntu (Linux).

REMARQUE: Cette fonction est réservée aux sites Windows, macOS et Ubuntu (Linux).

Syntaxe :

environment_get_variable(name);

ArgumentsType Description
name The name (a string) of the environment variable to check.

Les retours :

Exemple :

e_str = environment_get_variable("APPDATA");

Le code ci-dessus renvoie le chemin complet de la variable d'environnement "%appdata%", qui est normalement "C:\Users\{username}\AppData\Roaming".
