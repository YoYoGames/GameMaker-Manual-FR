os_is_network_connected (réseau connecté)

Avec cette fonction, vous pouvez vérifier si votre appareil dispose actuellement d'une connexion Internet. Elle renverra true si c'est le cas, ou false dans le cas contraire, et selon le système d'exploitation, elle peut tenter d'établir une connexion avant de renvoyer une valeur.

La fonction a un argument optionnel pour activer/désactiver la tentative d'établir une connexion lorsqu'elle est appelée, et cette option est actuellement seulement pour la cible Nintendo Switch. Lorsqu'elle est définie à true, la fonction tentera d'établir une connexion, et lorsqu'elle est définie à false, elle sera simplement limitée à la vérification de la connexion. Notez que la tentative d'établir une connexion peut provoquer un blocage du système d'exploitation Switch pendant quelques secondes.

NOTE Cette fonction vérifie l'API interne de l'appareil qui contrôle les connexions et peut donc retourner vrai s'il existe une connexion Bluetooth, une connexion Wi-Fi ou même simplement une connexion réseau normale qui permet l'accès à Internet access.

Syntaxe :

os_is_network_connected([attempt_connection]);

Arguments Type Description
attempt_connection Booléen OPTIONAL Pour la cible Nintendo Switch uniquement. Défini à true pour tenter une connexion au niveau du système d'exploitation lorsqu'il est appelé.

Les retours :

Boolean

Exemple :

if os_is_network_connected()
{
    global.connected = true;
}

Le code ci-dessus vérifie si l'appareil a une connexion à l'Internet et si c'est le cas, il définit une variable globale.
