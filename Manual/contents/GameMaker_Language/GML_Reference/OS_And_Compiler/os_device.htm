os_device

Cette variable en lecture seule contient l'une des diverses valeurs constantes énumérées ci-dessous pour vous indiquer sur quel périphérique vous exécutez actuellement le jeu. Notez que cette variable est dépréciée au profit de la fonction os_get_info() qui renvoie des informations plus précises sur le périphérique sur lequel le jeu est exécuté.

Syntaxe :

os_device

Les retours :

Device Type Constant

Type de dispositif Constant
Constant Description
device_ios_ipad iPad
device_ios_ipad_retina iPad plus récent avec écran Retina de 2048 x 1536
device_ios_iphone6 iPhone6 avec un écran de 1334 x 750
device_ios_iphone6plus Un iPhone 6 plus grand avec un écran 1920 x 1080
device_ios_iphone5 iPhone5 avec une taille d'écran de 640 x 1136)
device_ios_iphone Ancien iPhone/iPod Touch (écran 480 x 320) ou téléphone Android
device_ios_iphone_retina iPhone/iPod Touch plus récent avec écran Retina de 960 x 640
device_emulator Le dispositif est en fait un émulateur (Windows Phone ou Android).
device_tablet Tablette Android
device_ios_unknown Inconnu ou non iOS

Exemple :

if (os_browser = browser_not_a_browser)
{
    switch (os_device)
    {
        case device_ios_ipad: global.Config = 2; break;
        case device_ios_iphone: global.Config = 3; break;
        case device_ios_iphone_retina: global.Config = 4; break;
        case device_ios_unknown: global.Config = 5; break;
    }
}
else
{
    global.Config = 1;
}

Le code ci-dessus vérifie si le jeu est exécuté dans un navigateur ou non et définit une variable globale à une valeur dépendant du résultat de la vérification.
