os_browser

Cette variable en lecture seule contient l'une des diverses constantes (énumérées ci-dessous) dont dispose GameMaker pour vous indiquer dans quel navigateur vous exécutez actuellement le jeu (le cas échéant).

Syntaxe :

os_browser

Les retours :

Browser Type Constant

Type de navigateur Constant
Constant Description
browser_not_a_browser Le jeu ne se déroule pas dans un navigateur
browser_unknown Navigateur inconnu
browser_ie Internet Explorer
browser_ie_mobile Internet Explorer sur un appareil mobile
browser_firefox Mozilla Firefox
browser_chrome Google Chrome
browser_safari Safari
browser_safari_mobile Safari sur un appareil mobile
browser_opera Opéra
browser_tizen Navigateur pour appareils mobiles Tizen
browser_windows_store Application Windows

Exemple :

if (os_browser == browser_not_a_browser)
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

Le code ci-dessus vérifie si le jeu est exécuté dans un navigateur ou non et définit une variable globale à une valeur dépendant du résultat de la vérification.
