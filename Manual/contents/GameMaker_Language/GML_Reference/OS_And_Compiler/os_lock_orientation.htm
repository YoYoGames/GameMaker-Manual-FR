os_lock_orientation

Cette fonction vous permet de "verrouiller" votre appareil sur l'orientation actuelle jusqu'à ce que vous le "libériez" pour autoriser à nouveau toutes les orientations activées par les options de jeu pour cette plate-forme cible. Notez que vous voudrez probablement confirmer l'orientation souhaitée avant de verrouiller.

Syntaxe :

os_lock_orientation(flag)

ArgumentsType Description
flag Set to true ou false pour activer ou désactiver le verrouillage de l'orientation.

Les retours :

Exemple :

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

Le code ci-dessus vérifie le type de système d'exploitation et s'il s'agit de Android ou de iOS, le verrouillage de l'orientation est signalé comme étant true.
