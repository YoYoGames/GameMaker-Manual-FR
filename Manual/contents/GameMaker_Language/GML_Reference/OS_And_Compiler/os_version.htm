os_version

Cette variable vous indiquera le numéro de version du système d'exploitation qui exécute votre jeu. Par exemple, si vous l'exécutez sur Windows 10, os_version sera égal à 655360.

REMARQUE: Si vous souhaitez obtenir de plus amples informations sur le système d'exploitation Windows, vous pouvez utiliser l'outil . environment_get_variable().

Syntaxe :

os_version

Les retours :

Exemple :

if (os_type == os_android) && (os_version > 10)
{
    global.GFX = 1;
}

Le code ci-dessus vérifie le type d'OS et le numéro de version et s'ils sont tous deux corrects, alors la variable globale est fixée à 1.
