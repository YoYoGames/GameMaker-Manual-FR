draw_get_font

Cette fonction permet d'obtenir la police actuellement attribuée pour le dessin du texte. La fonction renvoie -1 si aucune police n'est définie, ou la valeur ID (un nombre entier positif) de la police attribuée.

Syntaxe :

draw_get_font();

Les retours :

Font Asset or -1

Exemple :

var _cur_font = draw_get_font();
var _y_offset = 0;

switch (_cur_font)
{
    case ft_small:
        _y_offset = 10;
    break;

    case ft_medium:
        _y_offset = 22;
    break;

    case ft_big:
        _y_offset = 34;
    break;

    default:
        _y_offset = 8;
}

draw_text(room_width / 2, 200 + _y_offset, "MENU");

Le code ci-dessus récupère la police actuellement appliquée et exécute une instruction de commutation sur celle-ci, en appliquant une valeur de décalage Y différente en fonction de la police. Il utilise ensuite cette valeur de décalage pour dessiner du texte.
