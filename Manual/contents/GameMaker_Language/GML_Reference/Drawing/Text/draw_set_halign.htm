draw_set_halign

Cette fonction est utilisée pour aligner le texte le long de l'axe horizontal. La modification de l'alignement horizontal changera la position et la direction dans lesquelles tout autre texte sera dessiné, la valeur par défaut étant fa_left. Les constantes suivantes sont acceptées :

Constant Alignement
fa_left
fa_center
fa_right

Syntaxe :

draw_set_halign(halign);

Arguments Type Description
halign Alignement horizontal Constant Horizontal alignment constant (from the table above).

Les retours :

N/A

Exemple :

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

Le code ci-dessus dessine deux chaînes sur la même ligne, le score étant aligné à gauche et la santé à droite.
