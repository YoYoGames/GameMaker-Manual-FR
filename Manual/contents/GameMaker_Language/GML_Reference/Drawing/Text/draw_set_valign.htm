draw_set_valign

Cette fonction est utilisée pour aligner le texte le long de l'axe vertical. La modification de l'alignement vertical changera la position et la direction dans lesquelles tout autre texte sera dessiné, la valeur par défaut étant fa_top. Les constantes suivantes sont acceptées :

Constant Alignement
fa_top
fa_middle
fa_bottom

Syntaxe :

draw_set_valign(valign);

Arguments Type Description
valign Alignement vertical Constant Vertical alignment constant (from the table above).

Les retours :

N/A

Exemple :

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

Le code ci-dessus dessinera le score centré sur le centre du texte.

Retour : Texte
