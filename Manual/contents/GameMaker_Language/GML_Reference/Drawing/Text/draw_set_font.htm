draw_set_font

Cette fonction permet de définir la police à utiliser pour tous les autres dessins de texte. Cette police doit avoir été ajoutée aux actifs de police du jeu ou avoir été créée à l'aide de la fonction font_add() ou font_add_sprite().

Syntaxe :

draw_set_font(font);

ArgumentsType Description
font The name of the font to use.

Les retours :

Exemple :

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

Le code ci-dessus dessinera le texte donné en utilisant la police indexée dans la variable "fnt_Game" et colorée en bleu.
