draw_text_ext_colour

Cette fonction est une combinaison de la fonction de base draw_text() de base avec les fonctions draw_text_ext() et draw_text_colour() vous permettant de définir des couleurs dégradées pour le texte ainsi que l'espacement des lignes et la largeur maximale par ligne.

NOTE Le mélange en dégradé n'est pas disponible pour la cible HTML5, sauf si WebGL est activé, bien que vous puissiez toujours définir les couleurs de mélange et la police sera mélangée avec la première couleur donnée. Cependant, tous les mélanges de cette manière créent une police dupliquée qui est ensuite stockée sur le site cache et utilisée lorsque cela est nécessaire, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu. Vous pouvez régler la taille de la police cache pour essayer de limiter ce phénomène si nécessaire en utilisant la fonction font_set_cache_size().

Syntaxe :

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Arguments Type Description
x Real The x coordinate of the drawn string.
y Real The y coordinate of the drawn string.
string Chaîne de caractères The string to draw.
sep Real The distance in pixels between lines of text.
w Real The maximum width in pixels of the string before a line break.
c1 Couleur The colour for the top left of the drawn text.
c2 Couleur The colour for the top right of the drawn text.
c3 Couleur The colour for the bottom right of the drawn text.
c4 Couleur The colour for the bottom left of the drawn text.
alpha Real The alpha for the text.

Les retours :

N/A

Exemple :

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

Le code ci-dessus dessine le texte que l'utilisateur tape au clavier, en le divisant sur de nouvelles lignes chaque fois que la longueur de la chaîne de cette ligne dépasse 300 pixels. Le code maintient également une séparation de 5 pixels entre les lignes si cela se produit. Chaque ligne de texte sera colorée à l'aide d'un dégradé de bleu, avec du bleu clair en haut et du bleu foncé en bas.
