dessiner_texte_couleur

Cette fonction permet de dessiner du texte de manière similaire à la fonction draw_text() mais vous pouvez maintenant choisir les couleurs à utiliser pour colorer le texte ainsi que la valeur alpha, et ces nouvelles valeurs seront utilisées à la place de la couleur de dessin de base et de l'alpha. de la couleur et de l'alpha du dessin de base.

REMARQUE: le mélange par gradient n'est pas disponible pour la cible HTML5, sauf si WebGL est activé, mais vous pouvez toujours définir les couleurs de mélange et la police sera mélangée avec la première couleur donnée. Cependant, tous les mélanges de cette manière créent une police dupliquée qui est ensuite stockée dans la base de données des polices. qui est ensuite stockée dans le site cache et utilisée au besoin, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu. Vous pouvez définir la taille de la police cache pour tenter de limiter ce phénomène, si cela s'avère nécessaire en utilisant la fonction font_set_cache_size().

Syntaxe :

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);

ArgumentsType Description
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Les retours :

Exemple :

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

Le code ci-dessus permet de dessiner deux sections de texte sur la même ligne, le premier texte étant dessiné en blanc (puisqu'il s'agit de la couleur de base du dessin) et le second texte étant dessiné avec un dégradé de vert citron à vert normal.
