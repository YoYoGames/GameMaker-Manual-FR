dessiner_texte

Avec cette fonction, vous pouvez dessiner n'importe quelle chaîne à n'importe quelle position dans la pièce (pour dessiner des nombres réels, vous devez utiliser la fonction string() pour les convertir en texte). Pour combiner des chaînes de caractères, vous pouvez utiliser + (voir l'exemple ci-dessous) et vous pouvez également utiliser \n à l'intérieur d'une chaîne de caractères pour ajouter un saut de ligne afin qu'elle soit dessinée sur plusieurs lignes (pour savoir comment formater correctement une chaîne de caractères et quels caractères d'échappement vous pouvez utiliser, veuillez consulter ici). La couleur du texte et l'alpha sont régis par les valeurs de base actuelles de l'alpha et de la couleur, définies par les commandes draw_set_alpha() et draw_set_colour().

REMARQUE: La position réelle du texte sera influencée par les valeurs d'alignement définies par les options draw_set_halign() et draw_set_valign().

Syntaxe :

draw_text(x, y, string);

ArgumentsType Description
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

Les retours :

Exemple :

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

Le code ci-dessus dessine une chaîne à la position x/y de l'instance, qui utilise la chaîne stockée dans la variable globale "Name" et la répartit sur deux lignes.
