draw_text_ext_transformed

Cette fonction est une combinaison de la fonction de base draw_text() de base avec les fonctions draw_text_ext() et draw_text_transformed() vous permettant de mettre à l'échelle et de faire pivoter du texte tout en maintenant un espacement de ligne spécifique et une largeur maximale par ligne. Notez que l'argument "width" est basé sur une échelle de 1, donc si l'échelle est différente, cette valeur doit être modifiée proportionnellement. proportionnellement. Par exemple, si la largeur de base d'un saut de ligne est de 300 et que vous définissez l'échelle à 2, le texte apparaîtra de manière erronée, dépassant la largeur donnée. Vous auriez dû fixer la largeur à 150 pour compenser la mise à l'échelle.

Syntaxe :

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle);

ArgumentsType Description
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.

Les retours :

Exemple :

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle);

Le code ci-dessus dessinera le texte donné au milieu de la pièce, avec une longueur de chaîne maximale de 300 pixels, un espacement entre chaque ligne de 10 pixels, une rotation et une mise à l'échelle de deux fois sa taille d'origine.
