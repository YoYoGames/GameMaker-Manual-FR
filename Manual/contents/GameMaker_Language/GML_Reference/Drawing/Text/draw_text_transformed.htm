draw_text_transformed

Cette fonction permet de dessiner du texte de manière similaire à la fonction draw_text() mais vous pouvez maintenant choisir de mettre le texte à l'échelle le long de l'axe horizontal ou vertical (ce qui a pour effet de l'étirer ou de le rétrécir). GameMaker le dessine selon un angle (où 0 est la valeur normale et chaque degré au-dessus de 0 fait tourner le texte dans le sens inverse des aiguilles d'une montre).

Syntaxe :

draw_text_transformed(x, y, string, xscale, yscale, angle);

ArgumentsType Description
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale (default 1).
yscale The vertical scale(default 1).
angle The angle of the text.

Les retours :

Exemple :

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

Le code ci-dessus dessinera le texte donné au milieu de la pièce, en le faisant tourner et en le mettant à l'échelle au double de sa taille originale.
