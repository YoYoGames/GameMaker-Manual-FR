draw_text_ext_transformed_colour

Cette fonction est une combinaison de la fonction de base draw_text() de base avec la fonction draw_text_ext(), draw_text_transformed() et draw_text_colour() et de la fonction de base. En tant que telle, elle est la plus générale de toutes les fonctions de dessin de texte et offre une grande flexibilité lors du dessin. Notez que l'argument " width " (largeur) est basé sur une échelle de 1, donc si l'échelle est différente, cette valeur doit être modifiée proportionnellement. Par exemple, si la largeur de base d'un saut de ligne est de 300 et que vous définissez l'échelle à 2, le texte apparaîtra de façon erronée, dépassant la largeur donnée de largeur donnée. Vous auriez dû fixer la largeur à 150 pour compenser la mise à l'échelle.

REMARQUE: le mélange par gradient n'est pas disponible pour la cible HTML5, sauf si WebGL est activé, mais vous pouvez toujours définir les couleurs de mélange et la police sera mélangée avec la première couleur donnée. Cependant, tous les mélanges de cette manière créent une police dupliquée qui est ensuite stockée dans la base de données des polices. qui est ensuite stockée dans le site cache et utilisée au besoin, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu. Vous pouvez définir la taille de la police cache pour tenter de limiter ce phénomène, si cela s'avère nécessaire en utilisant la fonction font_set_cache_size().

Syntaxe :

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentsType Description
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Les retours :

Exemple :

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Le texte donné sera dessiné avec un dégradé de bleu à bleu foncé vers le bas, à la position (200,200) dans la pièce, tourné à l'envers et à la moitié de la taille normale de la police. Il y aura 20 pixels entre chaque ligne si plus d'une ligne est dessinée. ligne et chaque ligne aura une largeur maximale de 200 pixels.
