dessiner_texte_transformé_couleur

Cette fonction est une combinaison de la fonction de base draw_text() de base avec les fonctions draw_text_transformed() et draw_text_colour() qui vous permettent de mettre à l'échelle et de faire pivoter du texte, de le colorer avec un remplissage en dégradé et de modifier sa valeur alpha, en ignorant les paramètres alpha et couleur de base pour le dessin.

NOTE Le mélange en dégradé n'est pas disponible pour la cible HTML5 à moins que WebGL ne soit activé, bien que vous puissiez toujours définir les couleurs de mélange et la police sera mélangée avec la première couleur donnée. Cependant, tous les mélanges de cette manière créent une police dupliquée qui est ensuite stockée dans la base de données . qui est ensuite stockée dans le site cache et utilisée au besoin, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu. Vous pouvez définir la taille de la police cache pour tenter de limiter ce phénomène, si cela s'avère nécessaire en utilisant la fonction font_set_cache_size().

Syntaxe :

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentsType Description
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Les retours :

Exemple :

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

Le code ci-dessus dessinera le texte donné au milieu de la pièce, en l'arrondissant et en le mettant à l'échelle au double de sa taille originale, avec un gradient de couleur allant du jaune au rouge ainsi qu'un alpha de 0,5.
