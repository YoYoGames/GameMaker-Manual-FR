dessiner_texte

Cette fonction dessine le texte de la même manière que la fonction draw_text() sauf que vous pouvez maintenant définir l'espace entre chaque ligne de texte - si le texte occupe plus d'une ligne - et limiter la largeur (en pixels) de la chaîne de caractères par ligne, de sorte que si une ligne dépasse cette valeur, GameMaker divisera automatiquement le texte vers la ligne suivante à l'espace blanc disponible le plus proche (si le texte n'a pas d'espace blanc, il dépassera cette valeur de largeur maximale). maximale). Notez que tout espace blanc placé au début de la chaîne sera supprimé avant d'être analysé pour le dessin. Notez également qu'une valeur de -1 pour l'argument de séparation des lignes donnera par défaut une séparation basée sur la hauteur du caractère "M" dans le texte. hauteur du caractère "M" dans la police choisie.

Syntaxe :

draw_text_ext(x, y, string, sep, w);

ArgumentsType Description
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

Les retours :

Exemple :

draw_text_ext(100, 50, keyboard_string, 3, 300);

Le code ci-dessus dessine le texte que l'utilisateur tape sur le clavier, en le divisant sur de nouvelles lignes chaque fois que la longueur de la chaîne de cette ligne dépasse 300 pixels. Le code maintient également une séparation de 3 pixels entre les lignes si cela se produit.
