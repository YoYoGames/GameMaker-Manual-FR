vertex_normal

Cette fonction ajoutera des données de normales de surface au sommet actuellement défini pour le site personnalisé primitive. Vous devez fournir le site buffer pour écrire les données ainsi que les composantes x, y et z de la normale.

Syntaxe :

vertex_normal(buffer, nx, ny, nz);

Arguments Type Description
buffer Vertex Buffer ID The buffer to write the information to.
nx Real The x component of the surface normal.
ny Real The y component of the surface normal.
nz Real The z component of the surface normal.

Les retours :

N/A

Exemple :

vertex_normal(buff, 0, 1, 1);

Le code ci-dessus définit la normale à la surface du sommet en cours de définition.
