vertex_end

Avec cette fonction, vous mettez fin à la construction du site personnalisé primitive. Une fois que vous avez appelé cette commande, le primitive peut être utilisé dans la fonction vertex_submit() pour être utilisé dans un shader ou vous pouvez geler le buffer (rendant le vertex buffer utilisé en lecture seule et beaucoup plus rapide) en utilisant la fonction vertex_freeze().

Syntaxe :

vertex_end(buffer);

Arguments Type Description
buffer Vertex Buffer ID The buffer to end writing to.

Les retours :

N/A

Exemple :

vertex_end(buff);

Le code ci-dessus mettra fin à la définition d'un primitive personnalisé.
