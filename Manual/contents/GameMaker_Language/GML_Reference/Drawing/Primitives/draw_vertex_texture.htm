dessiner_texture_vertex

Cette fonction définit la position d'un vertex texturé pour un primitive. L'aspect final du primitive dépendra du type de primitive choisi pour le dessin (Voir draw_primitive_begin() pour plus d'informations), de l'ordre dans lequel vous y ajoutez les vertex et de la position du point de départ que vous avez défini pour la texture. Pour terminer et dessiner le primitive, vous devez appeler draw_primitive_end().

Vous pouvez également choisir de ne dessiner qu'une partie de la texture sur la surface du site primitive en définissant la position xtex/ytex appropriée pour l'échantillonnage (ces coordonnées sont également appelées points UV ). Ce point est défini par une valeur comprise entre 0 et 1, le coin supérieur gauche de l'image à utiliser étant le point (0,0) et le coin inférieur droit étant le point (1,1). De cette façon, vous pouvez texturer un site primitive avec seulement une partie de l'image source.

Lorsque vous dessinez une texture primitive, la texture sera mélangée à la couleur définie par draw_set_colour. Cela signifie que pour dessiner la texture sans mélange, il faut définir la valeur c_white, comme le montre l'exemple ci-dessous.

NOTE Pour qu'une texture soit répétée, sa taille doit être une puissance de deux, c'est-à-dire : 32x32, 128x128, etc...

Syntaxe :

draw_vertex_texture(x, y, xtex, ytex)

Arguments Type Description
x Real The x coordinate of the vertex.
y Real The y coordinate of the vertex.
xtex Real The width of the texture sample to use.
ytex Real The height of the texture sample to use.

Les retours :

N/A

Exemple :

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Le code ci-dessus va dessiner une bande de triangle à 4 sommets (formant un rectangle) texturée avec la texture contenue dans la variable "tex", et la texture entière sera utilisée pour couvrir le primitive complété .
