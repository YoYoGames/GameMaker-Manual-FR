vertex_position

Cette fonction ajoute des données de position 2D au sommet en cours de définition pour le site personnalisé primitive. Vous devez fournir l'adresse buffer pour écrire les données ainsi que la position x et y pour le dessin.

Syntaxe :

vertex_position(buffer, x, y);

Arguments Type Description
buffer Vertex Buffer ID The buffer to write the information to.
x Real The x position.
y Real The y position.

Les retours :

N/A

Exemple :

vertex_position(buff, x - 100, y - 125);

Le code ci-dessus définira la position du sommet en cours de définition.
