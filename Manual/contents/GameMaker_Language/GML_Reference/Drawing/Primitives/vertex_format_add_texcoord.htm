format_de_verre_add_texcoord

Dites à GameMaker d'accepter les données de position de la texture (u et v) comme faisant partie du nouveau format de vertex en cours de création.

Syntaxe :

vertex_format_add_texcoord();

Les retours :

Exemple :

vertex_format_begin();
vertex_format_add_texcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

Le code ci-dessus créera un nouveau format de vertex avec seulement les valeurs de texture et de position, puis stockera l'identifiant du format dans la variable "my_format".
