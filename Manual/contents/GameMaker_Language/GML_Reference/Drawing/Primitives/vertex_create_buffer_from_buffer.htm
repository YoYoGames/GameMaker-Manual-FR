vertex_create_buffer_from_buffer

Comme pour la fonction vertex_create_buffer()cette fonction va créer un nouveau sommet buffer, mais les données de sommet qu'elle stocke sont copiées à partir de buffer qui est spécifié comme source. Le buffer créé est une croissance spéciale buffer qui est préformatée avec le format de sommet pour construire primitives pour une utilisation avec (par exemple) shaders. Cette fonction nécessite que vous fournissiez le pointeur d'un fichier régulier précédemment créé buffer, et un format de sommet qui doit être appliqué aux données copiées.

Syntaxe :

vertex_create_buffer_from_buffer(buffer, format);

Arguments Type Description
buffer ID de la mémoire tampon The buffer to create the vertex buffer from.
format Type primitif Constante The primitive vertex format to use.

Les retours :

Vertex Buffer ID

Exemple :

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

Le code ci-dessus crée un nouveau format de vertex, puis crée un nouveau vertex buffer à partir d'un buffer régulier précédemment créé, en lui appliquant le format de vertex personnalisé.
