vertex_argb

Cette fonction va définir les valeurs ARGB pour le sommet en cours de définition pour le site personnalisé primitive. Vous devez fournir le site buffer pour écrire les données ainsi que les valeurs rouge, verte, bleue et alpha que vous souhaitez utiliser sous la forme d'un seul entier non signé de 32 bits - échantillon alpha dans les 8 bits les plus élevés, suivi de l'échantillon rouge, de l'échantillon vert et enfin de l'échantillon bleu dans les 8 bits les plus bas. Vous pouvez utiliser la notation hexagonale ($AARRGGBB), un nombre réel ou utiliser l'une des fonctions de make_colour_*() pour définir la valeur de la couleur.

Syntaxe :

vertex_argb(buffer, argb);

Arguments Type Description
buffer Vertex Buffer ID The buffer to write the information to.
argb Couleur The ARGB value to set.

Les retours :

N/A

Exemple :

vertex_argb(buff, $FFFFFFFF);

Le code ci-dessus définira les valeurs ARGB du sommet défini comme étant blanc.
