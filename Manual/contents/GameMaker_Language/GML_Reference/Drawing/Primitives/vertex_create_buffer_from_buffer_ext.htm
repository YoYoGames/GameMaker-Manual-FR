Création de tampon de sommet à partir d'un tampon externe

Comme pour la fonction vertex_create_buffer()cette fonction va créer un nouveau sommet buffer, mais les données de sommet qu'elle stocke sont copiées à partir de buffer qui est spécifié comme source. buffer est préformaté avec le format de sommet pour construire primitives pour une utilisation avec (par exemple) shaders, et vous pouvez également fournir un décalage dans la source buffer à copier et le nombre de sommets que le buffer final devrait avoir. Notez que si le nombre de sommets ne correspond pas à ceux qui sont copiés, vous pouvez obtenir des données de sommets corrompues.

Syntaxe :

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

Arguments Type Description
buffer ID de la mémoire tampon The buffer to create the vertex buffer from.
format Type primitif Constante The primitive vertex format to use.
src_offset Real The offset within the the source buffer to copy from.
vert_num Real The number of vertices the buffer should have.

Les retours :

Vertex Buffer ID

Exemple :

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

Le code ci-dessus crée un nouveau format de sommet, puis crée un nouveau sommet buffer à partir d'un sommet régulier précédemment créé buffer, en lui appliquant le format de sommet personnalisé avec un décalage de 0. La fonction indique au nouveau sommet buffer qu'il doit s'attendre à 512 sommets.
