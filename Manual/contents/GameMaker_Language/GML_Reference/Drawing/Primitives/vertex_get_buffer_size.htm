taille_du_buffer_des_verrex

Avec cette fonction, vous pouvez obtenir la taille du vertex buffer en octets.

Syntaxe :

vertex_get_buffer_size(buffer);

Arguments Type Description
buffer Vertex Buffer ID The buffer to get the size of.

Les retours :

Real

Exemple :

bufferBytes = vertex_get_buffer_size(buff);

Le code ci-dessus obtiendra le nombre d'octets utilisés par le vertex buffer donné et stockera la valeur dans la variable "bufferBytes".
