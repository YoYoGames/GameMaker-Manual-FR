format_de_verre_début

Avant de pouvoir définir votre nouveau format de sommets, vous devez indiquer à GameMaker que vous le faites à l'aide de cette fonction. Vous devez l'appeler en premier, puis définir les valeurs du format à l'aide des fonctions appropriées, et enfin appeler vertex_format_end() pour terminer la définition et renvoyer le "handle" du nouveau format.

Syntaxe :

vertex_format_begin();

Les retours :

Exemple :

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Le code ci-dessus créera un nouveau format de vertex avec seulement des valeurs de couleur et de position, puis stockera l'identifiant du format dans la variable "my_format".
