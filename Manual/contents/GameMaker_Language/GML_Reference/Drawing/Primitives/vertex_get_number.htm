vertex_get_number

Cette fonction vous permet de connaître le nombre de sommets individuels définis dans un sommet donné buffer.

Syntaxe :

vertex_get_number(buffer);

Arguments Type Description
buffer Vertex Buffer ID The vertex buffer to check.

Les retours :

Real

Exemple :

v_num = vertex_get_number(v_buffer);

Le code ci-dessus stockera le nombre de sommets stockés dans le sommet donné buffer dans une variable.
