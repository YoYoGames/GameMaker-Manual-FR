vertex_texcoord

Cette fonction définira les coordonnées de la texture à utiliser pour le sommet actuellement défini pour le site personnalisé primitive. Vous devez fournir l'adresse buffer dans laquelle écrire les données ainsi que la position UV à utiliser dans la texture.

Syntaxe :

vertex_texcoord(buffer, u, v);

Arguments Type Description
buffer Vertex Buffer ID The buffer to write the information to.
u Real The first texture coordinate to use (0 - 1).
v Real The second texture coordinate to use (0 - 1).

Les retours :

N/A

Exemple :

vertex_texcoord(buff, 0, 0);

Le code ci-dessus définira les valeurs UV du sommet en cours de définition.
