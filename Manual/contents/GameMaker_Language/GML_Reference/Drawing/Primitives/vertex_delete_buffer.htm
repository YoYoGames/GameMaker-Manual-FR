vertex_delete_buffer

Cette fonction peut être utilisée pour supprimer de la mémoire du système un sommet précédemment créé buffer (voir vertex_create_buffer()) de la mémoire du système. Cela supprimera également toutes les données de vertex pour tout primitives personnalisé qu'il contenait.

Syntaxe :

vertex_delete_buffer(buffer);

Arguments Type Description
buffer Vertex Buffer ID The vertex buffer to be removed.

Les retours :

N/A

Exemple :

vertex_delete_buffer(v_buff);

Le code ci-dessus supprimera de la mémoire le site buffer indexé dans la variable "v_buff".
