Début du sommet

Avec cette fonction, vous commencez la définition d'un primitive personnalisé. Vous assignez un buffer pour y écrire le primitive, et le format de sommet à utiliser (préalablement défini à l'aide des fonctions de format de sommet). Vous définissez ensuite les points nécessaires pour chaque sommet du primitive avant d'appeler la fonction vertex_end() pour finaliser la création de primitive.

Syntaxe :

vertex_begin(buffer, format);

Arguments Type Description
buffer Vertex Buffer ID The buffer to be written to.
format ID du format des sommets The vertex format to use.

Les retours :

N/A

Exemple :

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

Le code ci-dessus va définir un nouveau format de sommet, créer un nouveau buffer et lancer le processus de définition d'un nouveau primitive.
