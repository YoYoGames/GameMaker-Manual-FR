dessiner_vertex

Cette fonction définit la position d'un sommet pour un primitive. L'aspect final du primitive dépendra du type de primitive choisi pour le dessin et de l'ordre dans lequel vous y ajoutez les sommets. Voir draw_primitive_begin() pour plus d'informations. Pour terminer et dessiner le primitive, vous devez appeler draw_primitive_end().

Syntaxe :

draw_vertex(x, y)

ArgumentsType Description
x The x coordinate of the vertex.
y The y coordinate of the vertex.

Les retours :

Exemple :

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

Le code ci-dessus dessinera un simple triangle primitive.
