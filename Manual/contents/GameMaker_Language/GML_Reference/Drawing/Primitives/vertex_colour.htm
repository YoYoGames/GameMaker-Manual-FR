vertex_colour

Si votre format de sommet défini prend une valeur de couleur, vous pouvez utiliser cette fonction pour ajouter ces données au sommet défini pour le site primitive actuel. La fonction a besoin d'une adresse buffer pour stocker les données et prendra soit une constante de couleur, soit une valeur hexadécimale (en utilisant le format standard GameMaker de BGR, par exemple : $FF0000 pour le bleu) ainsi qu'une valeur alpha de 0 (transparent) à 1 (totalement opaque).

Syntaxe :

vertex_colour(buffer, colour, alpha);

Arguments Type Description
buffer Vertex Buffer ID The buffer to write the information to.
colour Couleur The colour for this vertex (can be a constant or a hex value).
alpha Real The alpha value for the vertex (from 0 to 1).

Les retours :

N/A

Exemple :

vertex_colour(b, c_white, 1);

Le code ci-dessus définira le sommet défini comme étant blanc avec une valeur alpha de 1.
