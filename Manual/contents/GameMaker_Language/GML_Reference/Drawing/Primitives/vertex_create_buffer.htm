vertex_create_buffer

Avec cette fonction, vous pouvez créer un nouveau sommet buffer. Il s'agit d'une croissance spéciale buffer créée par GameMaker qui est préformatée pour être utilisée lors de la construction de primitives (pour être utilisée avec shaders, par exemple). La fonction renverra un index pour le buffer qui devra être utilisé dans tous les autres appels à cette fonction.

Lorsque vous utilisez un vertex buffer créé avec cette fonction, il vous suffit d'appeler vertex_begin() pour commencer à lui assigner des données de sommet afin de commencer à définir votre primitive personnalisé, qui sera ensuite conservé dans le buffer, prêt à être soumis au shader. Le buffer peut être réutilisé si nécessaire (sauf si vous avez utilisé la fonction vertex_freeze() ), chaque appel de vertex_begin() efface les données précédentes de buffer, prêtes à accepter les nouvelles données.

Syntaxe :

vertex_create_buffer();

Les retours :

Vertex Buffer ID

Exemple :

v_buff = vertex_create_buffer();

Le code ci-dessus créera un nouveau vertex buffer et stockera son handle dans la variable "v_buff".
