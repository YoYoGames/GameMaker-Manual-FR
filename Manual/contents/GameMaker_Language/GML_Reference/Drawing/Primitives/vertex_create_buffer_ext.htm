création_de_tampon_de_vertueux

Comme pour la fonction vertex_create_buffer()cette fonction va créer un nouveau sommet buffer. Il s'agit d'une croissance spéciale buffer créée par GameMaker qui est préformatée pour être utilisée lors de la construction de primitives pour une utilisation avec shaders. Vous pouvez spécifier une taille initiale de départ pour buffer (en octets) et elle renverra un index pour buffer qui devra ensuite être utilisé dans tous les appels ultérieurs à buffer.

Syntaxe :

vertex_create_buffer_ext(size);

ArgumentsType Description
size The initial size of the buffer (in bytes).

Les retours :

Vertex Buffer ID

Exemple :

v_buff = vertex_create_buffer_ext(1024 * 1024);

Le code ci-dessus crée un nouveau vertex buffer, d'une taille initiale de 1 Mo, et stocke son identifiant dans la variable "v_buff".
