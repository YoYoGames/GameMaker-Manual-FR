vertex_format_delete

Cette fonction doit être appelée chaque fois que vous avez fini d'utiliser les formats vectoriels créés. Vous fournissez la valeur de l'ID du format (telle que retournée par la fonction vector_format_end()), et cette fonction libérera la mémoire associée à ce format. Notez que si vous essayez d'utiliser à nouveau ce format après avoir appelé cette fonction, vous obtiendrez une erreur.

Syntaxe :

vertex_format_delete(formatID);

Arguments Type Description
formatID ID du format des sommets The vertex format to delete

Les retours :

N/A

Exemple :

vertex_format_delete(myFormat);

Le code ci-dessus supprimera de la mémoire le format de vertex créé dans la variable "myFormat".
