format_vertex_add_position_3d

Dites à GameMaker d'accepter les données de position 3D (x, y et z) dans le cadre du nouveau format de sommet en cours de création.

Syntaxe :

vertex_format_add_position_3d();

Les retours :

Exemple :

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position_3d();
my_format = vertex_format_end();

Le code ci-dessus créera un nouveau format de vertex avec seulement des valeurs de couleur et de position, puis stockera l'identifiant du format dans la variable "my_format".
