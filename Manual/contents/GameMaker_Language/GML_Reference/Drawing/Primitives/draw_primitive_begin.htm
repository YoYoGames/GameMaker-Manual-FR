draw_primitive_begin

Cette fonction doit être appelée avant que vous puissiez définir n'importe quel primitives. Il existe 6 types de primitives que vous pouvez définir comme les constantes suivantes :

Type primitif Constante
Constant Description
pr_pointlist Une liste de points
pr_linelist Une liste de lignes
pr_linestrip Une bande de lignes
pr_trianglelist Une liste de triangles
pr_trianglestrip Une bande triangulaire
pr_trianglefan Un éventail de triangle

L'image suivante illustre l'aspect général de ces éléments et l'ordre dans lequel vous devez définir les sommets :

Veuillez noter que sur certaines plateformes (Windows, UWP, XBox), le type pr_trianglefan n'est pas supporté en natif et GameMaker effectue donc une conversion lors de la compilation du jeu pour les faire fonctionner. Cela signifie que sur ces plateformes, le type pr_trianglefan sera beaucoup plus lent à utiliser que les autres. Notez également que pour utiliser cette fonction sur HTML5, vous devez activer WebGL dans les options du jeu.

Syntaxe :

draw_primitive_begin(kind)

Arguments Type Description
kind Type primitif Constante The kind of primitive you are going to draw.

Les retours :

N/A

Exemple :

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

Le code ci-dessus dessine les trois quarts d'un cercle à partir de primitives.
