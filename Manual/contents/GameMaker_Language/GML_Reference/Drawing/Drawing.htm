Dessin

Cette section contient toutes les fonctions liées au dessin dans la salle de jeu, ainsi qu'au contrôle de la façon dont les choses seront dessinées (mélange, alpha, suppression, etc...). Il existe un grand nombre de fonctions différentes pour le dessin, et elles sont réparties dans les catégories suivantes pour faciliter la recherche de ce dont vous avez besoin :

  • Couleur et alpha
  • Contrôle du GPU
  • Mipmapping
  • Formes de base
  • Sprites et tuiles
  • Texte
  • Primitives et formats de sommets
  • Surfaces
  • Éclairage
  • Particules
  • Textures
  • Shaders
  • Lecture vidéo

Deux fonctions spéciales sont également disponibles, l'une permettant d'activer ou de désactiver l'événement de dessin pour votre projet, l'autre permettant de vider l'ensemble du pipeline de dessin :

  • draw_enable_drawevent
  • draw_flush
