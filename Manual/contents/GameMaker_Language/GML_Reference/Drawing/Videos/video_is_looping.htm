video_s_looping

Cette fonction indique si la vidéo actuellement chargée est configurée pour tourner en boucle(true) ou non(false). Vous pouvez définir une vidéo en boucle en appelant video_enable_loop().

Syntaxe :

video_is_looping();

Les retours :

Boolean

Exemple :

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

Le code ci-dessus vérifie si la vidéo actuellement chargée tourne en boucle et, dans ce cas, désactive les contrôles du lecteur en définissant une variable d'instance personnalisée sur false.
