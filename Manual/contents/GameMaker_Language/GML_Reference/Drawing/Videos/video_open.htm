vidéo_ouvert

Cette fonction ouvre un fichier vidéo en utilisant le chemin d'accès spécifié. Le chemin d'accès peut faire référence à un fichier dans les fichiers inclus de votre projet, à un fichier présent localement sur l'appareil de l'utilisateur ou à une URL pour charger un fichier depuis l'Internet (ce qui peut ne pas fonctionner sur toutes les plateformes).

Comme vous ne pouvez charger qu'une seule vidéo à la fois, cette fonction charge la vidéo spécifiée dans le seul "emplacement vidéo" disponible, qui est ensuite manipulé via les autres fonctions vidéo.

Il existe des rappels asynchrones associés à la lecture vidéo. Consultez la section Rappels asynchrones pour plus d'informations.

Syntaxe :

video_open(path);

Arguments Type Description
path Chaîne de caractères Path to the video file

Les retours :

N/A

Exemple :

my_video = video_open("splash.mp4");

Le code ci-dessus charge le fichier splash.mp4 à partir des fichiers inclus dans le jeu.
