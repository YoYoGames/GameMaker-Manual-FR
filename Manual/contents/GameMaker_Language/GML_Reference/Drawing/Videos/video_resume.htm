CV vidéo

Cette fonction reprend le fichier vidéo qui est actuellement chargé. Il doit avoir été mis en pause avec video_pause() avant d'appeler cette fonction.

Syntaxe :

video_resume();

Les retours :

N/A

Exemple :

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

Le code ci-dessus récupère le statut de la vidéo et vérifie ensuite si le lecteur a appuyé sur Espace. Dans ce cas, il met la vidéo en pause si elle est en cours de lecture, et la reprend si elle est en pause.
