recherche_vidéo

Cette fonction vous permet de rechercher la position donnée dans la vidéo actuellement chargée. Vous spécifiez la valeur temporelle à rechercher, qui est exprimée en millisecondes.

Cette fonction fonctionne mieux lorsque la vidéo est en pause et que la distance entre la position actuelle et la nouvelle position de recherche n'est pas trop faible.

Syntaxe :

video_seek_to(time);

Arguments Type Description
time Real The time to seek to (in milliseconds).

Les retours :

N/A

Exemple :

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

Le code ci-dessus déplace la vidéo en cours de lecture de 2 secondes en avant, lorsque la touche de déplacement vers la droite est enfoncée, et la nouvelle position est inférieure à la durée de la vidéo. Cependant, si elle n'est pas dans la durée, la vidéo est fermée.

Dans cet exemple, video_seek_to() est utilisé pour rechercher la vidéo à la nouvelle position.
