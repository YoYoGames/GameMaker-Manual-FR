video_set_volume

Cette fonction modifie le volume audio de la vidéo qui est actuellement chargée. Elle prend une valeur entre 0 et 1, où 0 est silencieux et 1 est le volume maximum.

Assurez-vous que cette fonction est appelée après un appel à video_open() mais avant un appel à video_close(), sinon elle ne fera rien (car aucune vidéo ne sera chargée).

Sur certaines plates-formes, cette fonction ne fonctionnera qu'après le chargement complet de la vidéo. Utilisez donc des rappels asynchrones pour savoir quand la vidéo est prête, puis modifiez le volume.

Syntaxe :

video_set_volume(value);

Arguments Type Description
value Real Volume value between 0 and 1

Les retours :

N/A

Exemple :

my_video = video_open("splash.mp4");
video_set_volume(0.5);

Le code ci-dessus charge le fichier splash.mp4 depuis les fichiers inclus dans le jeu, et modifie son volume à 0,5 (50% de son volume maximum).
