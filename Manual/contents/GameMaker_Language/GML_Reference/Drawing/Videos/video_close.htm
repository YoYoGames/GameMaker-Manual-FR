vidéo_close

Cette fonction ferme le fichier vidéo qui est actuellement chargé. Assurez-vous que cette fonction n'est appelée qu'après un appel à video_open(), sinon elle ne fera rien.

Syntaxe :

video_close();

Les retours :

N/A

Exemple :

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

Le code ci-dessus ferme la vidéo lorsque la touche Echap est pressée.
