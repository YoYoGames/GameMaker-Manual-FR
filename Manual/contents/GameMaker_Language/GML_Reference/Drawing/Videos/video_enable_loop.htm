video_enable_loop

Cette fonction active ou désactive le bouclage pour la vidéo actuellement chargée. Définissez l'argument enable à true pour activer le bouclage, et false pour le désactiver.

Par défaut, le bouclage est désactivé.

Syntaxe :

video_enable_loop(enable);

Arguments Type Description
enable Booléen Whether to enable (true) ou désactiver(faux) le bouclage

Les retours :

N/A

Exemple :

video_open("Loading.mp4");
video_enable_loop(true);

Le code ci-dessus charge une vidéo et active le bouclage sur celle-ci.
