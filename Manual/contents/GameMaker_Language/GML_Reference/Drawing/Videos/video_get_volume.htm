video_get_volume

Cette fonction renvoie le volume de la vidéo actuellement chargée, qui est une valeur comprise entre 0 et 1. Vous pouvez modifier le volume de la vidéo chargée en utilisant video_set_volume().

Syntaxe :

video_get_volume();

Les retours :

Real

Exemple :

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

Le code ci-dessus serait exécuté dans un événement Step. Il obtient le volume de la vidéo en cours de lecture, et s'il est inférieur à 1, il l'augmente de 0,02, en augmentant progressivement le volume jusqu'à ce qu'il atteigne le niveau maximum.
