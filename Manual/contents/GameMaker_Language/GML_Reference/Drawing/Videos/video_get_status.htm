vidéo_get_status

Cette fonction renvoie l'état de la vidéo actuellement chargée. L'état peut être l'une des constantes suivantes :

Constante d'état vidéo
Constant Description
video_status_closed Aucune vidéo n'est actuellement chargée, ou la vidéo a été fermée avec video_close()
video_status_preparing La vidéo est en cours de préparation et n'a pas encore commencé à être lue.
video_status_playing La vidéo est en cours de lecture
video_status_paused La vidéo est mise en pause (voir video_pause())

Syntaxe :

video_get_status();

Les retours :

Video Status Constant

Exemple :

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

Le code ci-dessus récupère le statut de la vidéo et vérifie ensuite si le lecteur a appuyé sur Espace. Dans ce cas, il met la vidéo en pause si elle est en cours de lecture, et la reprend si elle est en pause.
