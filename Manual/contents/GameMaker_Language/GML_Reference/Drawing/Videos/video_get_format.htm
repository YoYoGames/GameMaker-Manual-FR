video_get_format

Cette fonction renvoie le format de couleur de la vidéo actuellement chargée. Il peut s'agir de l'une des constantes suivantes :

Format vidéo constant
Constant Description
video_format_rgba La surface vidéo utilise le modèle de couleur RGBA
video_format_yuv La surface vidéo utilise le modèle de couleur YUV

Windows, macOS, Opera GX, Android, iOS, et HTML5 utilisent le modèle RGBA pour les vidéos. Toutes les autres plateformes utilisent le modèle YUV.

Comme les méthodes de dessin pour les vidéos RGBA et YUV sont différentes, la valeur de retour de cette fonction peut être utilisée pour exécuter un code différent en fonction du format de la vidéo en cours de lecture. Voir Draw Video pour un exemple.

Syntaxe :

video_get_format();

Les retours :

Video Format Constant

Retour : Lecture de la vidéo
