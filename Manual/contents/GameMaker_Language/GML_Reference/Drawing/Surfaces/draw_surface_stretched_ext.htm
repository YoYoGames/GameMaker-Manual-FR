draw_surface_stretched_ext

Cette fonction fait exactement la même chose que la fonction draw_surface_stretched() avec la possibilité supplémentaire de définir le mélange de couleurs et la valeur alpha pour la surface lorsqu'elle est dessinée (similaire à la fonction draw_surface_ext()).

Syntaxe :

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

Arguments Type Description
id ID de la surface The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
w Real The width at which to draw the surface.
h Real The height at which to draw the surface.
colour Couleur The colour with which to colour the surface.
alpha Real The alpha with which to blend the surface.

Les retours :

N/A

Exemple :

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

Cela va dessiner la surface donnée avec son coin gauche à la position x/y de l'instance et elle sera étirée pour occuper une zone de 200x200 pixels sans mélange, mais avec une transparence partielle.
