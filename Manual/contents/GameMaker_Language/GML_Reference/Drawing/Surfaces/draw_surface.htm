dessiner_surface

Cette fonction permet de dessiner une surface à une position donnée dans la pièce, le coin supérieur gauche de la surface étant dessiné à la position x/y spécifiée.

Syntaxe :

draw_surface(id, x, y);

Arguments Type Description
id ID de la surface The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.

Les retours :

N/A

Exemple :

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

Le code ci-dessus dessine la surface indexée dans "surf" à la même position que la vue de la caméra [0].
