surface_reset_target

Avec cette fonction, vous réinitialisez tout dessin ultérieur de la surface cible vers l'écran. Veuillez noter que pour commencer à dessiner sur une surface, vous devez d'abord appeler la fonction surface_set_target() puis celle-ci après avoir terminé, pour chaque surface cible que vous avez définie, sinon rien ne sera dessiné à l'écran car tous les dessins ultérieurs (même dans d'autres instances) seront effectués sur la surface. Vous devez également savoir que rien ne sera vu si la surface elle-même n'est pas dessinée à l'écran dans l'événement draw d'une instance.

REMARQUE: Si vous n'avez pas précédemment défini une cible de rendu avec la fonction surface_set_target()l'utilisation de cette fonction mettra fin de manière silencieuse (c'est-à-dire sans aucun message d'erreur) à toute autre exécution de code pour l'événement.

Syntaxe :

surface_reset_target();

Les retours :

Exemple :

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

Le code ci-dessus vérifie quelle vue est actuellement dessinée, et si c'est la vue [0], il définit la cible de dessin sur une surface et dessine toutes les instances de l'objet "obj_Effect" avant de réinitialiser à nouveau la cible de dessin. Si la vue n'est pas view[0], la surface est dessinée à l'écran.
