surface_get_target_ext

Cette fonction récupère l'ID de la surface assignée à l'une des 4 cibles de rendu disponibles pour les surfaces. Vous fournissez l'index de la cible de rendu à vérifier, et la fonction retournera -1 si aucune surface n'est assignée, ou une valeur entière >= 0, représentant l'ID de la surface assignée (comme retourné par la fonction surface_create()).

Syntaxe :

surface_get_target_ext(index);

ArgumentsType Description
index The render target index to check (from 0 to 3).

Les retours :

Surface ID or -1 (if no target surface is set)

Exemple :

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

Le code ci-dessus vérifie d'abord si la cible de rendu 0 de shader a été définie comme une surface, et si ce n'est pas le cas, une surface est attribuée.
