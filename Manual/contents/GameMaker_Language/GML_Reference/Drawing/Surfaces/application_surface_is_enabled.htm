application_surface_is_enabled

Cette fonction renvoie true si la surface de l'application est utilisée pour le dessin, ou false si l'écran buffer est utilisé.

Syntaxe :

application_surface_is_enabled();

Les retours :

Exemple :

if keyboard_check_pressed(vk_space)
{
    if application_surface_is_enabled()
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

Le code ci-dessus vérifie l'appui sur une touche et active ou désactive la surface de l'application en fonction de son état (comme dans un menu d'options).
