surface_set_target_ext

Cette fonction est à utiliser avec les fonctions de shaders et définit le MRT (0 - 3) pour le niveau natif shaders (DX9, DX11, OpenGL).

REMARQUE: Lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment, car elles sont stockées dans la mémoire des textures. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists() avant d'y faire référence directement.

Syntaxe :

surface_set_target_ext(index, surface_id);

ArgumentsType Description
index The render target index to use (from 0 to 3).
surface_id The ID of the surface to use.

Les retours :

Exemple :

surface_set_target_ext(0, surf);

Le code ci-dessus va définir la cible de rendu 0 à l'ID de la surface indexée dans la variable "surf".
