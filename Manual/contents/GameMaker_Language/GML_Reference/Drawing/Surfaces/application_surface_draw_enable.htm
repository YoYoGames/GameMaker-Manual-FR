application_surface_draw_enable

Vous pouvez utiliser cette fonction pour activer ou désactiver le dessin automatique de la surface de l'application. Par défaut, cette fonction est activée, mais dans de nombreux cas, vous souhaiterez prendre en charge le moment et la manière dont la surface est dessinée (lorsque vous utilisez shaders par exemple), auquel cas Dans ce cas, vous utiliserez cette fonction pour la définir sur false afin de pouvoir la dessiner vous-même quand et comment vous le souhaitez. Il est important de noter que lorsque vous désactivez le dessin automatique et dessinez vous-même la surface de l'application, vous pouvez rencontrer certains problèmes avec la composante alpha de sprites et la surface elle-même. Cela est dû au fait que GameMaker dessinera la surface de l'application sans mélange alpha lorsque le dessin automatique est activé. Si vous désactivez le dessin automatique, vous devez gérer cela vous-même, en utilisant le code suivant (par exemple) :

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

Syntaxe :

application_surface_draw_enable(flag);

ArgumentsType Description
flag The flag will be either true (activé, la valeur par défaut) ou false (désactivé).

Les retours :

Exemple :

application_surface_draw_enable(false);

Le code ci-dessus désactivera le dessin automatique de la surface de l'application.
