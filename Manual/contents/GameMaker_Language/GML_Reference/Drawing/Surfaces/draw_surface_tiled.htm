draw_surface_tiled

Cette fonction va prendre une surface et la carreler de manière répétée dans toute la pièce, à partir des coordonnées que vous donnez dans la fonction.

Syntaxe :

draw_surface_tiled(id, x, y);

Arguments Type Description
id ID de la surface The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.

Les retours :

N/A

Exemple :

draw_surface_tiled(surf, x, y);

Cela va dessiner la surface indexée dans "surf" à la position x et y de l'instance, et la carreler dans toutes les directions de la pièce.
