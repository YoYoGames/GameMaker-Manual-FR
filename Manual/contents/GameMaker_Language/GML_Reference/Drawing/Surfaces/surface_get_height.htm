surface_get_height

Cette fonction renvoie simplement la hauteur, en pixels, de la surface indexée. Il convient de noter que si vous appelez cette fonction pour vérifier le site application_surface immédiatement après avoir modifié sa taille à l'aide de la fonction surface_resize() elle ne renverra pas la nouvelle valeur car la modification nécessite une étape ou deux pour être entièrement traitée. Après avoir attendu une étape, il devrait retourner la nouvelle taille correctement.

REMARQUE: Lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment, car elles sont stockées dans la mémoire des textures. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists() avant d'y faire référence directement.

Syntaxe :

surface_get_height(surface_id);

ArgumentsType Description
surface_id The ID of the surface to get the height of.

Les retours :

Exemple :

sh = surface_get_height(surf);

Le code ci-dessus va stocker la hauteur de la surface indexée dans la variable "surf" dans la variable "sh".
