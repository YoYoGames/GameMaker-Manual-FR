surface_copy_part

Cette fonction prend simplement l'image d'une surface et la copie sur une autre à la position locale spécifiée dans cette surface (où la position (0,0) est le coin supérieur gauche de la surface de destination). Vous pouvez spécifier une position locale x et y à partir de laquelle copier, ainsi que la largeur et la hauteur de la section. Veuillez noter que ces coordonnées sont basées sur la taille de la surface et non sur la position à laquelle la surface est dessinée dans la pièce. Si la surface de destination contient déjà des informations, celles-ci seront écrasées par la copie, et la fonction ne modifie en rien la surface source.

REMARQUE: Lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment, car elles sont stockées dans la mémoire des textures. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists() avant d'y faire référence directement.

Syntaxe :

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

Arguments Type Description
destination ID de la surface The ID of the surface to copy the other surface to.
x Real The x position to copy to.
y Real The y position to copy to.
source ID de la surface The ID of the surface to be copied.
xs Real The x position in the source surface to copy from.
ys Real The y position in the source surface to copy from.
ws Real The width of the area in the source surface to copy from.
hs Real The height of the area in the source surface to copy from.

Les retours :

N/A

Exemple :

if view_current == 0
{
    var _cam1_x = camera_get_view_x(view_camera[1]);
    var _cam1_y = camera_get_view_y(view_camera[1]);

    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, _cam1_x - mouse_x, _cam1_y - mouse_y);
}
else
{
    draw_surface(surf, 0, 0);
}

Le code ci-dessus va vérifier la vue courante en cours de dessin et si c'est la vue [0], il copie la surface indexée dans la variable "temp_surf" sur la surface indexée dans la variable "surf". La surface copiée correspond à un rectangle formé par la position relative de la souris dans la surface telle qu'elle serait dessinée dans la vue [1]. Si la vue actuelle n'est pas la vue [0], la surface "surf" est dessinée à l'écran.
