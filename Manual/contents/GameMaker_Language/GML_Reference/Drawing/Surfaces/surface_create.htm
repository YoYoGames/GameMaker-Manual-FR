surface_create

Cette fonction est utilisée pour créer une surface et renvoie l'index de la surface qui doit être stocké dans une variable pour les appels de fonction futurs. Lorsque la surface est créée pour la première fois, elle peut contenir du "bruit" car il s'agit simplement d'une zone de mémoire qui est mise de côté dans ce but (et cette mémoire peut encore contenir des informations), vous pouvez donc vouloir effacer la surface avant de l'utiliser avec une fonction telle que draw_clear_alpha().

Il est fortement recommandé de créer toutes les surfaces avec une taille correspondant à une puissance de 2, c'est-à-dire une taille de 16, 128, 512 ou 1024 pixels, par exemple. Ce n'est pas exactement nécessaire sur certaines plates-formes (comme Windows et MacOS) mais cela augmentera certainement la compatibilité sur ces cibles, tandis que pour HTML5 et les appareils, c'est essentiel et très important que vous vous en souveniez ou vous pourriez rencontrer des problèmes plus tard.

Syntaxe :

surface_create(w, h);

ArgumentsType Description
w The width of the surface to be created.
h The height of the surface to be created.

Les retours :

Surface ID

Exemple :

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
    surface_set_target(surf);
    draw_clear_alpha(c_black, 0);
    surface_reset_target();
    view_surface_id[0] = surf;
}

Le code ci-dessus vérifie si une surface existe et si ce n'est pas le cas, il crée une surface de 1024 pixels de large et 1024 pixels de haut, en assignant l'index à la variable "surf". La cible de dessin est ensuite fixée à la nouvelle surface, qui est effacée et rendue transparente avant que la cible de dessin ne soit réinitialisée à l'affichage. Enfin, la surface est affectée à une vue.
