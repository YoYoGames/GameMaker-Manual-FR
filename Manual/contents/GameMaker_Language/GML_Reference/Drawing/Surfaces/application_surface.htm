application_surface

Cette variable globale intégrée peut être utilisée pour access la surface de l'application, à utiliser dans toutes les fonctions de surface. Cette surface est disponible en permanence et constitue l'endroit où le jeu est dessiné par GameMaker.

Les événements réguliers de dessin(Draw Begin, Draw et Draw End) dessinent sur cette surface par défaut. Les autres événements Draw (comme Pre/Post Draw et Draw GUI) ne dessinent pas sur la surface de l'application. Voir Événements de dessin pour plus d'informations sur le fonctionnement de ces événements.

Syntaxe :

application_surface;

Les retours :

Surface ID

Exemple :

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

Le code ci-dessus va redimensionner la surface de l'application pour avoir un rapport 1:1 avec la couche GUI.
