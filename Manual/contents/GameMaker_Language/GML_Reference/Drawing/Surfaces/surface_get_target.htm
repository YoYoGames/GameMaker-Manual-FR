surface_get_target

Cette fonction vous permet d'obtenir l'ID de la surface actuelle sur laquelle vous souhaitez dessiner. Si vous avez créé cette surface en utilisant surface_create()alors ce sera le nombre entier positif qui est l'ID unique de cette surface.

Si aucune surface personnalisée n'est définie, mais que la surface d'application par défaut est toujours activée, la fonction renvoie généralement 0. Cependant, si vous avez manipulé l'application_surface précédemment, vous pourriez souhaiter vérifier que l'ID obtenu ne correspond pas à la valeur actuelle de la variable application_surface, au cas où celle-ci ne serait plus 0.

Si aucune surface personnalisée n'est définie et que la surface d'application par défaut a également été désactivée, la fonction renvoie -1.

Syntaxe :

surface_get_target();

Les retours :

Surface ID or -1 (if no target surface is set)

Exemple :

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

Le code ci-dessus va vérifier si la cible de rendu actuelle est une surface ou non, et si c'est le cas, il réinitialise la cible puis dessine une surface.
