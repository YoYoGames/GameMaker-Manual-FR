surface_create_ext

Cette fonction vous permet d'attacher une surface à un élément de canevas qui existe déjà dans votre page Web, ce qui signifie que vous pouvez effectivement diviser des parties de votre jeu à dessiner à différents endroits de la page. Pour cela, vous devez avoir défini correctement l'élément canvas dans la page *.html de votre jeu en utilisant les tailles et les noms corrects qui correspondent aux surfaces que vous souhaitez créer. Ainsi, vous aurez votre canevas "principal", puis vos éléments de canevas de surface secondaire, qui seront assignés à l'aide de cette fonction aux surfaces correctes. L'image suivante est un exemple de la configuration d'une page comportant trois éléments de canevas :

La mise en page peut être délicate, surtout si vous souhaitez que tous les éléments s'alignent correctement, mais une fois la tâche difficile de la création de la mise en page terminée, vous pouvez ajouter ce fichier html comme fichier de page par défaut pour le jeu en utilisant l'onglet HTML5 des options de jeu. La prochaine chose à faire est de configurer votre salle et vos vues, car chaque surface devra être associée à une vue spécifique pour "capturer" les images du jeu (voir la variable de vue view_surface_id[0...7]). L'image ci-dessous montre la configuration de la salle de jeu pour l'exemple de toile ci-dessus :

Enfin, vous utiliserez cette fonction pour créer les surfaces, le nom étant le même que celui utilisé pour les éléments de la toile et la taille correspondant à la taille de cette même toile. La fonction renvoie l'indice de la surface, qui doit être stocké dans une variable pour les futurs appels de fonction. Lorsque la surface est créée pour la première fois, elle peut contenir du "bruit" car il s'agit simplement d'une zone de mémoire mise de côté à cet effet (et cette mémoire peut encore contenir des informations), vous pouvez donc effacer la surface avant de l'utiliser avec une fonction comme draw_clear_alpha().

REMARQUE: Cette fonction n'est disponible qu'avec le module HTML5.

Syntaxe :

surface_create_ext(name, w, h);

ArgumentsType Description
name The name of the canvas element to link the surface to.
w The width of the surface to be created.
h The height of the surface to be created.

Les retours :

Surface ID

Exemple :

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

Le code ci-dessus crée deux surfaces de tailles différentes, en les affectant chacune à un élément de canevas différent, puis ces surfaces sont affectées à deux vues afin de capturer la bonne partie de la pièce.
