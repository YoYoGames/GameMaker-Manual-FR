texture_get_uvs

Cette fonction renvoie un tableau 1D avec 4 éléments représentant les coordonnées UV pour le pointeur de texture donné, en remplissant le tableau avec les valeurs suivantes :

  • [0] = gauche
  • [1] = haut
  • [2] = droite
  • [3] = fond
  • [4] = nombre de pixels que le compilateur d'actifs a supprimé du côté gauche de sprites (uniquement pour les actifssprite ).
  • [5] = nombre de pixels que le compilateur d'actifs a supprimé de la partie supérieure de sprites (uniquement pour les actifssprite ).
  • [6] = pourcentage normalisé des données de pixel de la largeur originale de sprites qui ont été sauvegardées dans la page de texture (sprite assets seulement)
  • [7] = pourcentage normalisé des données de pixel de la hauteur originale de sprites qui ont été sauvegardées dans la page de texture (sprite seulement).

Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, notamment dans les fonctions générales 3D et certaines des fonctions 2D primitive, ainsi que dans les fonctions Shader. Si vous avez besoin de l'UVS pour un sprite, vous pouvez utiliser le sprite_get_uvs() et pour une police, font_get_uvs().

Syntaxe :

texture_get_uvs(texid)

Arguments Type Description
texid Texture The texture pointer to get the UVS for

Les retours :

Array (4 - 8 elements)

Exemple :

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

Le code ci-dessus récupère d'abord la texture de la surface stockée dans surf_back, puis les coordonnées UV de cette texture. Il récupère ensuite les coordonnées UV gauche, haut, droite et bas du tableau retourné et les stocke dans des variables locales.
