textures_debug_messages

Cette fonction peut être utilisée pour activer ou désactiver les messages de débogage de la texture. Lorsqu'elle est activée (définie à true), des informations supplémentaires sur l'utilisation des pages de texture seront envoyées à la fenêtre de sortie. Réglez-la sur false pour désactiver à nouveau cette sortie.

Syntaxe :

texture_debug_messages(enable);

ArgumentsType Description
enable Enable or disable the texture debug messages

Les retours :

Exemple :

if global.debug
{
    texture_debug_messages(true);
}

Le code ci-dessus vérifie la valeur d'une variable globale et si elle est définie sur true, les messages de débogage de la texture sont activés.
