texture_set_stage

Cette fonction définit une texture à utiliser pour le "slot" de scène donné. Le nombre de "slots" disponibles dépend de la plate-forme sur laquelle vous compilez, avec un maximum de 8 pour Windows, Mac et Linux, mais sur les appareils bas de gamme Android (par exemple), ce nombre peut être aussi bas que 2. Il est également intéressant de noter que le premier slot (1) est toujours utilisé automatiquement par GameMaker.

NOTE: Cette fonction ne fera rien en dehors du contexte d'un shader en cours d'exécution ! Voir Shaders pour plus d'informations.

Syntaxe :

texture_set_stage(stage, tex);

Arguments Type Description
stage Real The texture "slot" to use.
tex Pointeur de texture The texture to use.

Les retours :

N/A

Exemple :

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Le code ci-dessus récupère l'identifiant de l'échantillonneur dans le fichier shader indexé comme "GlassShader", puis définit cette constante shader sur la texture sprite donnée.
