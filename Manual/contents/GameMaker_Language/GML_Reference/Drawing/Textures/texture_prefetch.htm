texture_prefetch

Cette fonction peut être utilisée pour "prélever" une page de texture ou un groupe de pages de texture, c'est-à-dire les charger dans la VRAM lorsque cela est nécessaire. Vous fournissez l'ID unique de la page de texture (tel que renvoyé par les fonctions texturegroup_* ) pour prélever une seule page, ou vous pouvez fournir un nom de groupe de texture (tel que défini dans l'éditeur de groupe de texture) pour prélever toutes les pages de texture dans le groupe.

Syntaxe :

texture_prefetch(tex_id);

Arguments Type Description
tex_id Pointeur ou chaîne de caractèresde latexture The texture page pointer or a texture group name (a string)

Les retours :

N/A

Exemple :

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    texture_prefetch(_tex_array[i]);
}

Le code ci-dessus va prélever toutes les pages de texture sous le groupe de texture "MainMenu".
