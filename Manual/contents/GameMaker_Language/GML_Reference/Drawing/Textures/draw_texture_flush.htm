draw_texture_flush

Avec cette fonction, vous pouvez supprimer toutes les textures de la mémoire vidéo, et elles seront ensuite rechargées à la première utilisation. C'est la seule façon efficace de gérer la mémoire vidéo lorsque vous avez plusieurs pages de textures pour un jeu. entre les niveaux de votre jeu et organisez les graphiques à l'aide de la fonction de groupe de textures pour vous assurer que le nombre minimum de textures est utilisé.

Il convient de noter que par défaut, sur toutes les cibles, les pages de texture ne seront chargées qu'au fur et à mesure qu'elles sont nécessaires et vous pouvez utiliser les différentes fonctions de prefetch et flush pour sprites (trouvées ici) pour gérer les choses aussi bien que cette fonction.

Syntaxe :

draw_texture_flush();

Les retours :

Exemple :

draw_texture_flush();

Le code ci-dessus vide la mémoire vidéo des données de texture, et serait probablement placé dans l'événement create de la première instance d'un objet placé dans la pièce.
