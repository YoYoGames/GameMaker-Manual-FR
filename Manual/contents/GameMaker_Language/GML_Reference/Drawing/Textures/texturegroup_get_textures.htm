texturegroup_get_textures

Cette fonction peut être utilisée pour récupérer les ID des pages de texture des pages individuelles qui composent un groupe de texture. Vous fournissez la chaîne d'identification du groupe de texture (telle que définie dans l'éditeur de groupe de texture), et la fonction renverra un tableau 1D, où chaque entrée du tableau est un ID de page de texture unique. Si la fonction échoue - c'est-à-dire si un groupe invalide est fourni, ou si aucune texture n'est assignée au groupe - alors le tableau sera vide (longueur 0).

Syntaxe :

texturegroup_get_textures(tex_id);

Arguments Type Description
tex_id Chaîne de caractères The name of the texture group to check (a string)

Les retours :

Array of Texture Pointers

Exemple :

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

Le code ci-dessus récupère les ID des pages de texture pour le groupe de texture "MainMenu", puis vérifie s'ils sont déballés, et s'ils le sont, ils sont placés dans la VRAM.
