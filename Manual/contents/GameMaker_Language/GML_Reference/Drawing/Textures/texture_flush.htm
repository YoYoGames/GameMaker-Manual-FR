texture_flush

Cette fonction peut être utilisée pour "vider" une page de texture ou un groupe de pages de texture, c'est-à-dire les supprimer de la VRAM lorsqu'elles ne sont plus nécessaires. Vous fournissez l'ID unique de la page de texture (tel que retourné par les fonctions texturegroup_* ) pour vider une seule page, ou vous pouvez fournir un nom de groupe de texture (tel que défini dans l'éditeur de groupe de texture) pour vider toutes les pages de texture dans le groupe.

Syntaxe :

texture_flush(tex_id);

Arguments Type Description
tex_id Pointeur ou chaîne de caractèresde latexture The texture page pointer or a texture group name (a string)

Les retours :

N/A

Exemple :

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_flush(_tex_array[i]);
}

Le code ci-dessus fera disparaître toutes les pages de texture du groupe de texture "MainMenu".
