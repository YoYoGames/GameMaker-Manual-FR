largeur_de_la_texture

Renvoie la largeur de la texture avec l'identifiant donné, qui est toujours une valeur comprise entre 0 et 1. Cette valeur peut ensuite être utilisée lors du mappage des textures aux modèles ou à primitives.

Syntaxe :

texture_get_width(tex);

Arguments Type Description
tex Pointeur de texture The texture page asset pointer to use

Les retours :

Real

Exemple :

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

Le code ci-dessus permet d'obtenir la largeur de la texture provenant d'une ressource sprite.
