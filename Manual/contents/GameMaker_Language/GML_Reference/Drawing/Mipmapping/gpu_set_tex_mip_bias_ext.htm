gpu_set_tex_mip_bias_ext

Cette fonction permet de définir la valeur du biais de mipmap pour un échantillonneur shader donné. Vous fournissez la valeur de l'index de l'échantillonneur shader (tel que retourné par la fonction shader_get_sampler_index()), puis la valeur du biais, où 0 signifie qu'il n'y a pas de biais, 1 correspond à la première mipmap, 2 correspond à la deuxième mipmap etc.... Ceci contrôle la vitesse à laquelle la mip map est échangée et rendra généralement les textures de shader plus floues plus la valeur est élevée et plus la "distance" vue est grande. Notez que cette fonction peut aussi prendre des valeurs négatives, dans ce cas les textures de shader seront plus nettes sur une plus grande distance, plus la valeur est faible.

Syntaxe :

gpu_set_tex_mip_bias_ext(sampler_index, bias);

ArgumentsType Description
sampler_index The index of the shader sampler to get
bias The mipmap bias value to use (default: 0)

Les retours :

Exemple :

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Le code ci-dessus définit le biais du filtre mip à 0 pour l'échantillonneur de texture shader donné s'il n'a pas déjà été défini à 0.
