gpu_set_tex_max_mip_ext

Cette fonction vous permet d'obtenir le niveau maximum de mipmap actuellement défini qui doit être utilisé pour un échantillonneur shader donné. Vous fournissez la valeur de l'index de l'échantillonneur shader (tel que renvoyé par la fonction shader_get_sampler_index()) et la fonction renverra le niveau maximum actuel de mipmap, où une valeur de 0 correspond à la résolution la plus élevée, 1 correspond à l'utilisation de la première mipmap, 2 à la seconde etc... Notez que cela peut être très utile pour éviter les artefacts de saignement lors du rendu des textures, par exemple, en définissant la bordure de la page de texture à 8px et en définissant ensuite le niveau maximum de mipmap à 3, vous serez sûr de ne pas avoir de problèmes de saignement à des distances de rendu plus importantes.

Syntaxe :

gpu_set_tex_max_mip_ext(sampler_index, maxmip);

ArgumentsType Description
sampler_index The index of the shader sampler to get
maxmip The maximum mipmap level value to use (default: 16)

Les retours :

Exemple :

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_max_mip_ext(_sampleIndex) != 4)
{
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Le code ci-dessus définit le niveau maximum de mipmap à 4 pour l'échantillonneur de texture shader donné s'il n'a pas déjà été défini à 4.
