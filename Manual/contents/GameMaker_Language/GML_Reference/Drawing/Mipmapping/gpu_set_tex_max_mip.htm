gpu_set_tex_max_mip

Avec cette fonction, vous pouvez définir le niveau maximum de mipmap qui doit être utilisé, où une valeur de 0 est la résolution la plus élevée, 1 est d'utiliser le premier mipmap, 2 est le second etc.... Notez que cela peut être très utile pour éviter les artefacts de saignement lors du rendu des textures, par exemple, en définissant la bordure de la page de texture à 8px et en définissant ensuite le niveau maximum de mipmap à 3, vous vous assurerez de ne pas avoir de problèmes de saignement à de plus grandes distances de rendu.

Syntaxe :

gpu_set_tex_max_mip(maxmip);

ArgumentsType Description
maxmip The maximum mipmap level (default: 16)

Les retours :

Exemple :

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

Le code ci-dessus vérifie le niveau maximum actuel de mipmap et s'il n'est pas égal à 4, il est fixé à 4.
