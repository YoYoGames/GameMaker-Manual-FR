gpu_set_tex_min_mip

Avec cette fonction, vous pouvez définir le niveau minimum de mipmap qui est actuellement utilisé, où une valeur de 0 est la plus haute résolution, 1 est d'utiliser le premier mipmap, 2 est le deuxième etc...

Syntaxe :

gpu_set_tex_min_mip(minmip);

ArgumentsType Description
minmip The minimum mipmap level to use

Les retours :

Exemple :

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

Le code ci-dessus vérifiera le niveau minimum actuel de mipmap et s'il n'est pas égal à 0, il sera mis à 0.
