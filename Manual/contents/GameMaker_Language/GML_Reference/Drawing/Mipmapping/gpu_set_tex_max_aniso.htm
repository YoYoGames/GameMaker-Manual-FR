gpu_set_tex_max_aniso

Cette fonction permet de définir le niveau maximal actuel d'anisotropie lors de l'utilisation du mode filtre tf_anisotropic (voir gpu_get_tex_mip_filter() pour plus d'informations). La valeur d'entrée doit être comprise entre 1 et 16.

Syntaxe :

gpu_set_tex_max_aniso(maxaniso);

ArgumentsType Description
maxaniso The maximum level for anisotropic filtering (default: 16)

Les retours :

Exemple :

if gpu_get_tex_max_aniso() != 8
{
    gpu_set_tex_max_aniso(8);
}

Le code ci-dessus vérifie le niveau maximum actuel du filtrage anisotrope et s'il n'est pas égal à 8, il est fixé à 8.
