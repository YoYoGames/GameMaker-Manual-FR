Mipmapping

Cette section contient toutes les fonctions liées à l'utilisation des fonctions de mipmapping. Avant d'utiliser ces fonctions, vous devez avoir activé le mipmapping pour les pages de texture requises dans l'éditeur de groupe de texture et/ou activé le mipmapping pour les sprites qui ont été configurés pour n'utiliser qu'une seule page de texture dans les options générales du jeu. L'image ci-dessous montre la différence que le mipmapping peut faire lors du rendu de votre projet :

Une fois que vous avez activé le mipmapping pour le projet ou une page de texture, vous pouvez alors utiliser les fonctions suivantes pour obtenir et définir des valeurs qui modifieront l'aspect de votre projet lors de son exécution.

Les fonctions listées ci-dessous peuvent être utilisées pour obtenir les différentes valeurs associées au mipmapping dans GameMaker:

  • gpu_get_tex_mip_filter
  • gpu_get_tex_mip_filter_ext
  • gpu_get_tex_mip_bias
  • gpu_get_tex_mip_bias_ext
  • gpu_get_tex_min_mip
  • gpu_get_tex_min_mip_ext
  • gpu_get_tex_max_mip
  • gpu_get_tex_max_mip_ext
  • gpu_get_tex_max_aniso
  • gpu_get_tex_max_aniso_ext
  • gpu_get_tex_mip_enable
  • gpu_get_tex_mip_enable_ext

Les fonctions énumérées ci-dessous peuvent être utilisées pour définir les différentes valeurs associées au mipmapping dans GameMaker:

  • gpu_set_tex_mip_filter
  • gpu_set_tex_mip_filter_ext
  • gpu_set_tex_mip_bias
  • gpu_set_tex_mip_bias_ext
  • gpu_set_tex_min_mip
  • gpu_set_tex_min_mip_ext
  • gpu_set_tex_max_mip
  • gpu_set_tex_max_mip_ext
  • gpu_set_tex_max_aniso
  • gpu_set_tex_max_aniso_ext
  • gpu_set_tex_mip_enable
  • gpu_set_tex_mip_enable_ext
