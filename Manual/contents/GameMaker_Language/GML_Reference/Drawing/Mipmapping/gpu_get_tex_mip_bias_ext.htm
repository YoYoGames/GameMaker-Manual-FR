gpu_get_tex_mip_bias_ext

Cette fonction vous permet de récupérer la valeur du biais mipmap pour un échantillonneur shader donné. Vous fournissez la valeur de l'index de l'échantillonneur shader (tel que renvoyé par la fonction shader_get_sampler_index()) et la fonction renverra une valeur de 0,0 pour aucun biais, ou une valeur supérieure où 1 équivaut au premier mipmap, 2 au second mipmap, etc... Cela contrôle la vitesse à laquelle la mip map est échangée et rendra généralement les textures de shader plus floues plus la valeur est élevée et plus la "distance" vue est grande. Notez que ce paramètre peut aussi renvoyer des valeurs négatives, dans ce cas les textures de shader seront plus nettes sur une plus grande distance, plus la valeur est faible.

Syntaxe :

gpu_get_tex_mip_bias_ext(sampler_index);

ArgumentsType Description
sampler_index The index of the shader sampler to get

Les retours :

(default: 0)

Exemple :

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Le code ci-dessus définit le biais du filtre mip à 0 pour l'échantillonneur de texture shader donné s'il n'a pas déjà été défini à 0.
