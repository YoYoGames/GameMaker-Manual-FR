draw_set_circle_precision

Pour optimiser les performances lors de la création de cercles, GameMaker dessine une forme polygonale avec suffisamment de côtés pour qu'elle paraisse circulaire. Cependant, en fonction de la taille des cercles que vous souhaitez dessiner, il se peut que la modification de cette valeur vous permette d'augmenter les performances de votre jeu ou d'améliorer l'apparence des cercles. cette valeur permet d'améliorer les performances de votre jeu ou de donner un meilleur aspect aux cercles. La valeur de précision que vous pouvez entrer dans cette fonction est le nombre de côtés du polygone circulaire, et ce nombre peut être compris entre 4 et 64, mais doit être un nombre divisible par 4, avec une valeur par défaut de 24.

REMARQUE: si vous souhaitez dessiner une forme à l'aide de shader, vous devez savoir que la plupart des sites shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc la fonction shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_set_circle_precision(precision);

ArgumentsType Description
precision The precision of all circles drawn after this function is called.

Les retours :

Exemple :

draw_set_circle_precision(12);

Le code ci-dessus dessinera tous les cercles en utilisant un polygone à 12 côtés.
