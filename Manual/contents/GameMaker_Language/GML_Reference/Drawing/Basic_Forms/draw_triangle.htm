dessiner_triangle

Avec cette fonction, vous pouvez dessiner soit le contour d'un triangle, soit un triangle rempli.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_triangle(x1, y1, x2, y2, x3, y3, outline);

Arguments Type Description
x1 Real The x coordinate of the triangle's first corner.
y1 Real The y coordinate of the triangle's first corner.
x2 Real The x coordinate of the triangle's second corner.
y2 Real The y coordinate of the triangle's second corner.
x3 Real The x coordinate of the triangle's third corner.
y3 Real The y coordinate of the triangle's third corner.
outline Booléen Whether the triangle is drawn filled (false) or as a one pixel wide outline (true).

Les retours :

N/A

Exemple :

draw_set_colour(c_aqua);
draw_triangle(50, 50, 200, 50, 50, 200, 0);

Cela dessinera un triangle rectangle isocèle rempli de couleur aigue-marine, dont le premier angle est situé à (50,50), le deuxième à (200,50) et le troisième à (50,200).
