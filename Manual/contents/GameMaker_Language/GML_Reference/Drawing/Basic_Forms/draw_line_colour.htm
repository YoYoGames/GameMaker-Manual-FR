dessiner_ligne_couleur

Cette fonction vous permet de dessiner une ligne d'un pixel de large dont la couleur est mélangée entre la couleur 1 au premier point et la couleur 2 au deuxième point. Les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour(). Veuillez noter que la ligne dessinée peut nécessiter des valeurs différentes (+/-1 sur les x, y) pour être dessinée avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

REMARQUE: si vous souhaitez dessiner une forme à l'aide de shader, vous devez savoir que la plupart des sites shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc la fonction shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_line_colour(x1, y1, x2, y2, col1, col2);

ArgumentsType Description
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
col1 The colour of the start of the line.
col2 The colour of the end of the line.

Les retours :

Exemple :

draw_line_colour(50, 50, 300, 50, c_red, c_blue);

Cela dessinerait une ligne horizontale de (50,50) à (300,50) avec un gradient lisse de rouge à bleu.
