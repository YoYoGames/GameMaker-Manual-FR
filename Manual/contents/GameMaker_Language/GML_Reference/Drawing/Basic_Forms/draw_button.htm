bouton de dessin

Cette fonction dessine un "bouton" rectangulaire très simple en utilisant la couleur de dessin et l'alpha actuellement sélectionnés, l'argument "up" définissant l'effet de bord biseauté, comme le montre l'image ci-dessous :

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, vous devez savoir que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_button(x1, y1, x2, y2, up);

Arguments Type Description
x1 Real The x coordinate of the left of the button
y1 Real The y coordinate of the top of the button
x2 Real The x coordinate of the right of the button
y2 Real The y coordinate of the bottom of the button
up Booléen Whether the button is up (true) ou vers le bas (false)

Les retours :

N/A

Exemple :

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

Ceci dessinera un bouton qui apparaîtra enfoncé si le bouton gauche de la souris est maintenu enfoncé.
