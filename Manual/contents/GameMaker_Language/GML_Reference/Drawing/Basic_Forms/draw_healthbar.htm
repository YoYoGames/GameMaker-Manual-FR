dessiner_barre de santé

Cette fonction vous permet de dessiner une barre de couleur pour afficher une valeur constante. Bien que la fonction utilise le mot "healthbar", vous pouvez l'utiliser pour afficher tout ce que vous souhaitez tant que la quantité à afficher est un pourcentage compris entre 0 et 100 (un pourcentage de n'importe quelle valeur peut être calculé avec la formule (CURRENT_Value / MAXIMUM_value) * 100), donc, par exemple, vous pouvez l'utiliser pour afficher la puissance, la santé, le mana, le temps ou tout autre élément qui bénéficierait d'un affichage en barre.

Vous pouvez définir différents éléments qui modifient l'aspect visuel de la barre et ceux-ci sont illustrés dans l'image ci-dessous (notez que les couleurs utilisées ici sont c_green pour la valeur maximale et c_red pour la valeur minimale, mais vous pouvez utiliser les couleurs que vous souhaitez) :

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, vous devez savoir que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Arguments Type Description
x1 Real The x coordinate of the left of the healthbar
y1 Real The y coordinate of the top of the healthbar
x2 Real The x coordinate of the right of the healthbar
y2 Real The y coordinate of the bottom of the healthbar
amount Real The variable which defines total health (between 0 and 100)
backcol Couleur The colour of the background for the bar
mincol Couleur The colour of the bar when at no health (0)
maxcol Couleur The colour of the bar when at full health (100)
direction Real Where the bar is "anchored" (0 = left, 1 = right, 2 = top, 3 = bottom)
showback Booléen Whether the bar's background is drawn (true) ou non (false). Si elle est fausse, l'argument backcol n'est pas pris en compte.
showborder Booléen Whether the bar's elements have a 1px wide black border (true) ou non (false).

Les retours :

N/A

Exemple :

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

Le code ci-dessus utilise la valeur en pourcentage que vous obtenez des variables "hp" et "hp_max" pour dessiner une barre de santé rouge/verte standard.
