draw_roundrect_colour_ext

Avec cette fonction, vous pouvez dessiner soit le contour d'un rectangle arrondi, soit un rectangle arrondi rempli où la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) est le coin inférieur droit. Si le rectangle est rempli, les arguments de couleur seront utilisés pour générer un gradient de couleur du centre vers les bords, où la couleur 1 est la couleur du centre et la couleur 2 la couleur des bords. Vous devez également fournir des valeurs de rayon pour les axes x et y (en pixels) et les coins seront arrondis de ces valeurs. Vous pouvez définir la précision du dessin des coins à l'aide de la fonction draw_set_circle_precision(). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les axes x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, vous devez savoir que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline);

Arguments Type Description
x1 Real The x coordinate of the left of the rounded rectangle.
y1 Real The y coordinate of the top of the rounded rectangle.
x2 Real The x coordinate of the right of the rounded rectangle.
y2 Real The y coordinate of the bottom of the rounded rectangle.
xrad Real The radius of the curve along the x axis from the rectangle corners.
yrad Real The radius of the curve along the y axis from the rectangle corners.
col1 Couleur The center colour.
col2 Couleur The outside edge colour.
outline Booléen Whether the rectangle is an outline (true) or filled in (false).

Les retours :

N/A

Exemple :

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255);
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0);

Cela dessine un carré rempli avec des coins arrondis, dont le rayon est défini par la distance entre l'instance qui effectue le dessin et la souris. Cette valeur est également utilisée pour calculer la couleur de fusion centrale.
