draw_point_colour

Avec cette fonction, vous pouvez dessiner un pixel unique n'importe où sur l'écran avec une couleur que vous définissez. Les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour().

REMARQUE: si vous souhaitez dessiner une forme à l'aide de shader, vous devez savoir que la plupart des sites shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc la fonction shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_point_colour(x, y, col1);

ArgumentsType Description
x The x coordinate of the point.
y The y coordinate of the point.
col1 The colour of the point.

Les retours :

Exemple :

draw_point_colour(50, 50, c_red);

Cela dessinerait un pixel rouge à (50,50).
