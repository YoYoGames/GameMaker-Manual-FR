dessiner_arrondi_couleur

Avec cette fonction, vous pouvez dessiner soit le contour d'un rectangle arrondi, soit un rectangle arrondi rempli dont la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) le coin inférieur droit. S'il est rempli, vous pouvez définir les couleurs individuelles pour le centre et les bords, et si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de gradient de l'une à l'autre (les paramètres de couleur remplacent la couleur de base définie avec la fonction draw_set_colour()Vous pouvez définir le degré de précision du dessin des coins avec la fonction draw_set_circle_precision()Vous pouvez définir le degré de précision du dessin des coins avec la fonction "-", mais les coins sont toujours dessinés avec un rayon fixe. Si vous souhaitez modifier le rayon des coins, utilisez la fonction draw_roundrect_colour_ext(). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, vous devez savoir que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_roundrect_colour(x1, y1, x2, y2, col1, col2, outline);

Arguments Type Description
x1 Real The x coordinate of the left of the rounded rectangle.
y1 Real The y coordinate of the top of the rounded rectangle.
x2 Real The x coordinate of the right of the rounded rectangle.
y2 Real The y coordinate of the bottom of the rounded rectangle.
col1 Couleur The center colour.
col2 Couleur The outside edge colour.
outline Booléen Whether the rectangle is an outline (true) or filled in (false).

Les retours :

N/A

Exemple :

draw_roundrect_colour(50, 50, 200, 200, c_black, c_white, false);

Cela dessine un carré rempli avec des coins arrondis et un gradient lisse noir/blanc du centre vers les bords.
