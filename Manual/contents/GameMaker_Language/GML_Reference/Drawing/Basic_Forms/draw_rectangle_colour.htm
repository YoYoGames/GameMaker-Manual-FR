draw_rectangle_colour

Avec cette fonction, vous pouvez dessiner le contour d'un rectangle ou un rectangle rempli en définissant une zone où la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) est le coin inférieur droit. S'il est rempli, vous pouvez définir les couleurs individuelles pour chaque point d'angle et si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de gradient de l'un à l'autre (les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour()). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, vous devez savoir que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_rectangle_colour(x1, y1, x2, y2, col1, col2, col3, col4, outline);

Arguments Type Description
x1 Real The x coordinate of the left of the rectangle.
y1 Real The y coordinate of the top of the rectangle.
x2 Real The x coordinate of the right of the rectangle.
y2 Real The y coordinate of the bottom of the rectangle.
col1 Couleur The colour of the top left corner.
col2 Couleur The colour of the top right corner.
col3 Couleur The colour of the bottom right corner.
col4 Couleur The colour of the bottom left corner.
outline Booléen Whether the rectangle is an outline (true) or filled in (false).

Les retours :

N/A

Exemple :

draw_rectangle_colour(50, 50, 200, 200, c_black, c_white, c_black, c_white, false);

Cela dessinerait un carré rempli avec un gradient noir/blanc lisse entre les coins.
