largeur de ligne

Cette fonction permet de tracer une ligne d'une largeur donnée entre deux points quelconques de la salle de jeu. Veuillez noter que la ligne dessinée peut nécessiter des valeurs différentes (+/-1 sur les x, y) pour être dessinée avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge. les diverses plates-formes prises en charge.

REMARQUE: si vous souhaitez dessiner une forme à l'aide de shader, vous devez savoir que la plupart des sites shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc la fonction shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_line_width(x1, y1, x2, y2, w);

ArgumentsType Description
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
w The width of the line in pixels.

Les retours :

Exemple :

draw_set_colour(c_red);
draw_line_width(100, 100, 200, 200, 6);

Cela va dessiner une ligne diagonale rouge, de 6 pixels de large, du point (100,100) au point (200,200).
