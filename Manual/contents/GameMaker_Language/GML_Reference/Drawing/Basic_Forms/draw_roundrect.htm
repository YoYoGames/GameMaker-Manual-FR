draw_roundrect

Avec cette fonction, vous pouvez dessiner soit le contour d'un rectangle arrondi, soit un rectangle arrondi rempli dont la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) le coin inférieur droit. Vous pouvez définir la précision du dessin des coins à l'aide de la fonction draw_set_circle_precision()mais cette fonction utilise un rayon fixe pour les coins (si vous souhaitez modifier le rayon des coins, utilisez la fonction draw_roundrect_ext()). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, vous devez savoir que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le site shader doit être personnalisé en conséquence.

Syntaxe :

draw_roundrect(x1, y1, x2, y2, outline);

Arguments Type Description
x1 Real The x coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle).
y1 Real The y coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle).
x2 Real The x coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle).
y2 Real The y coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle).
outline Booléen Whether the rounded rectangle is drawn filled (false) or as a one pixel wide outline (true).

Les retours :

N/A

Exemple :

draw_set_colour(c_lime);
draw_roundrect(50, 100, 250, 200, 0);

Ceci dessinera un rectangle arrondi rempli, avec son coin supérieur gauche théorique (avant d'être courbé) à (50,100) et son coin inférieur droit théorique à (250,200).
