partie_particules_créer_couleur

Cette fonction est idéale pour les effets qui ne nécessitent aucune des fonctionnalités offertes par les émetteurs de particules (par exemple, pour créer de la fumée à partir d'un missile, ou un simple effet d'explosion) car elle vous permet de créer rapidement et facilement des particules à n'importe quelle position dans la salle de jeu. Vous pouvez également colorer les particules "à la volée" lorsqu'elles sont créées avec cette fonction, et cette couleur remplacera la couleur prédéfinie de la particule, mais elle ne se fondra pas dans la durée de vie de la particule. Notez que vous devez avoir créé le système de particules et le type de particule au préalable pour que cette fonction puisse être utilisée.

Syntaxe :

part_particles_create_colour(ind, x, y, parttype, colour, number);

Arguments Type Description
ind ID du système de particules The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Type de particule ID The index (type) of the particles to be created.
colour Couleur The colour blending for the particles.
number Real The number of particles to create.

Les retours :

N/A

Exemple :

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

Le code ci-dessus générera une seule particule à chaque pas où l'instance avec le code a une vitesse supérieure à un. Ces particules seront d'une couleur aléatoire.
