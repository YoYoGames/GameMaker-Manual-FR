partie_particules_créer

Cette fonction est idéale pour les effets qui ne nécessitent aucune des fonctionnalités offertes par les émetteurs de particules (par exemple, pour créer de la fumée à partir d'un missile, ou un simple effet d'explosion) car elle vous permet de créer rapidement et facilement des particules à n'importe quelle position dans la salle de jeu. Notez que vous devez avoir créé le système de particules et le type de particule précédemment pour que cette fonction puisse être utilisée.

Syntaxe :

part_particles_create(ind, x, y, parttype, number);

Arguments Type Description
ind ID du système de particules The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Type de particule ID The index (type) of the particles to be created.
number Real The number of particles to create.

Les retours :

N/A

Exemple :

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

Le code ci-dessus vérifie si le bouton de la souris est pressé et s'il renvoie true, il génère 5 particules à la position de la souris.
