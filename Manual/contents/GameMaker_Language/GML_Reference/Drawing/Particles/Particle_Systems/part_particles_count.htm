partie_particules_count

Avec cette fonction, vous pouvez vérifier si un système de particules a actuellement des particules créées dans la pièce, et elle renverra également le nombre de celles-ci.

Syntaxe :

part_particles_count(ind);

Arguments Type Description
ind ID du système de particules The index of the particle system.

Les retours :

Real

Exemple :

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

Le code ci-dessus vérifiera le nombre de particules dans le système de particules local indexé dans la variable "Sname" et s'il n'y en a pas, il détruira le système et ensuite l'instance.
