ordre de tirage des pièces

Cette fonction vous permet de définir la manière dont les particules sont dessinées lorsqu'elles sont créées à l'écran. Le système par défaut utilise un look ancien > nouveau (la fonction est réglée sur true), où les anciennes particules sont dessinées à une profondeur plus élevée que les nouvelles et apparaissent donc "sous" les nouvelles particules, mais en réglant cette fonction sur false vous pouvez inverser cet ordre et avoir les nouvelles particules dessinées plus haut et apparaissent donc "sous" les anciennes. Les images ci-dessous illustrent cela, l'image de gauche étant la valeur par défaut de true et l'image de droite étant false:

NOTE: Lorsque les particules sont dessinées avec un mode de fusion additif, l'effet de cette fonction n'est pas toujours évident.

Syntaxe :

part_system_draw_order(ind, oldtonew);

Arguments Type Description
ind ID du système de particules The index of the particle system to change.
oldtonew Booléen Whether old particles should be drawn behind newer ones (true) or vice versa (false).

Les retours :

N/A

Exemple :

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Cela va créer un nouveau système de particules avec l'index "mysystem", puis il définit les particules pour dessiner les nouvelles particules au dessus des anciennes.
