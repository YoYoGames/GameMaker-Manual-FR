partie_system_get_layer

Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de la couche pour le système de particules donné. Vous fournissez la valeur d'identification du système de particules (telle que renvoyée par la fonction part_system_create_layer()) et la fonction renvoie la valeur d'identification de la couche.

Syntaxe :

part_system_get_layer(ind);

Arguments Type Description
ind ID du système de particules The particle system ID value to get the layer ID of

Les retours :

Layer ID or 0 (if the layer is an internally managed one)

Exemple :

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

Le code ci-dessus vérifiera un système de particules pour voir sur quelle couche il se trouve et s'il n'est pas sur la couche donnée, il sera changé.
