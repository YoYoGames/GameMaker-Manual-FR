partie_système_destruction

Avec cette fonction, vous pouvez détruire un système de particules donné et le supprimer de la mémoire. Cette fonction doit toujours être appelée lorsque le système n'est plus nécessaire, comme à la fin d'une room, ou dans l'événement de destruction d'une instance, sinon vous pourriez vous retrouver avec des particules apparaissant dans des rooms ultérieures sans aucun moyen de les supprimer, ainsi qu'avec une fuite de mémoire qui finira par faire planter votre jeu.

REMARQUE: Cette fonction détruit et supprime également tous les émetteurs qui ont pu être créés et associés au système en cours de destruction.

Syntaxe :

part_system_destroy(ind);

Arguments Type Description
ind ID du système de particules The index of the particle system to destroy.

Les retours :

N/A

Exemple :

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

Le code ci-dessus vérifie si la valeur d'une variable globale est inférieure ou égale à zéro, et si c'est le cas, il détruit le système de particules référencé dans la variable donnée, puis passe à la pièce suivante.
