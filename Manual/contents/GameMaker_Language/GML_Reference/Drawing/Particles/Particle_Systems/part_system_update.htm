part_system_update

Cette fonction fera avancer le système de particules donné d'un pas dans le temps de jeu et peut être utilisée pour simuler le comportement du système de particules lorsque la mise à jour automatique est désactivée, ou elle peut être utilisée pour faire avancer un système de particules à un point spécifique en une seule étape. Par exemple, si vous disposez d'un système de particules pour la neige et que vous voulez donner l'impression que la neige tombe dès que le joueur entre dans la pièce, vous pouvez faire avancer vos particules de neige de quelques secondes dans l'événement de création d'une instance afin que la première fois qu'elles soient dessinées, il y ait des particules partout dans l'écran (l'exemple de code ci-dessous le montre). Vous pouvez également utiliser cette fonction lorsque la mise à jour automatique du système est désactivée par la fonction part_system_automatic_update() pour faire avancer le système.

Syntaxe :

part_system_update(ind);

Arguments Type Description
ind ID du système de particules The index of the particle system to update.

Les retours :

N/A

Exemple :

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

Le code ci-dessus fera avancer de 3 secondes le système de particules indexé dans la variable globale "RainSys".
