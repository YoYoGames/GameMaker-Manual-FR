part_system_exists

Avec cette fonction, vous pouvez vérifier si le système de particules indexé existe ou non dans le jeu. Notez que si la variable vérifiée est une variable non initialisée (à laquelle un système de particules aurait normalement été assigné), une erreur sera générée.

Syntaxe :

part_system_exists(ind);

Arguments Type Description
ind ID du système de particules The index of the particle system to check for.

Les retours :

Boolean

Exemple :

if !part_system_exists(global.Sname)
{
    global.Sname = part_system_create();
}

Le code ci-dessus vérifie si le système de particules référencé dans la variable globale existe et s'il n'existe pas, il est créé.
