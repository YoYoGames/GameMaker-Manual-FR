partie_système_mise à jour

Avec cette fonction, vous pouvez forcer un système de particules à dessiner toutes les particules actuelles à l'écran. Si part_system_automatic_draw() est désactivée, cette fonction affichera les particules lorsqu'elle est utilisée dans l'événement de dessin d'une instance, ou elle peut être utilisée lorsque la cible de dessin est définie sur une surface (en utilisant la fonction surface_set_target()) pour dessiner les particules du système sur cette surface.

Syntaxe :

part_system_drawit(ind);

Arguments Type Description
ind ID du système de particules The index of the particle system to draw the particles of.

Les retours :

N/A

Exemple :

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

Le code ci-dessus vérifie si la surface indexée dans la variable "surf" existe, et si c'est le cas, il fixe la cible de dessin à la surface, dessine le système de particules avec l'ID stocké dans la variable globale, puis réinitialise la cible de dessin pour que tout le dessin normal apparaisse à nouveau à l'écran.
