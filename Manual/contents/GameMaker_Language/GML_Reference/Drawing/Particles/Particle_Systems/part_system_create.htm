partie_system_create

Cette fonction est utilisée pour créer un nouveau système de particules et renvoie un numéro d'index unique qui doit être stocké et utilisé dans toutes les autres fonctions relatives à ce système. Le système se verra attribuer une couche gérée et aura une profondeur de 0. Le système se verra attribuer une couche gérée et aura une profondeur de 0. Les couches gérées ne sont pas accessibles à l'utilisateur et sont utilisées uniquement pour la gestion interne lorsque la profondeur est utilisée à la place des couches. Normalement, vous devriez utiliser la fonction part_system_create_layer() au lieu de celle-ci.

Syntaxe :

part_system_create();

Les retours :

Particle System ID

Exemple :

global.Sname = part_system_create();

Cela créera un nouveau système de particules et stockera l'index dans la variable globale "Sname".
