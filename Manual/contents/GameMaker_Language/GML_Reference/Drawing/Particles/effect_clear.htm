effect_clear

Cela effacera toutes les particules actuellement visibles dans la pièce qui ont été créées en utilisant l'une des méthodes suivantes effect_create_above() ou effect_create_below().

Syntaxe :

effect_clear();

Les retours :

Exemple :

effect_clear();

En appelant cette fonction, la pièce sera débarrassée de toutes les particules créées avec les fonctions effect_create_*.
