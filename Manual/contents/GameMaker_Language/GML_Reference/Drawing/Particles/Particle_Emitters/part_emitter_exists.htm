partie_émetteur_existe

Avec cette fonction, vous pouvez vérifier si l'émetteur de particules indexé existe ou non dans le système donné. Notez que si la variable vérifiée est une variable non initialisée (à laquelle un émetteur de particules aurait normalement été assigné), une erreur sera générée.

Syntaxe :

part_emitter_exists(ps, ind);

Arguments Type Description
ps ID du système de particules The particle system to check for an emitter.
ind ID de l'émetteur de particules The index of the emitter to search for.

Les retours :

Boolean

Exemple :

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

Le code ci-dessus va rechercher l'émetteur indexé dans la variable "permit" et s'il existe, il va éclater 30 particules à partir de celui-ci.
