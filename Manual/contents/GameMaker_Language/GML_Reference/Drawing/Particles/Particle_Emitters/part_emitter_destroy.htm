partie_émetteur_destruction

Cette fonction supprimera l'emitter spécifié du système donné et l'effacera de la mémoire (cela empêchera également toute particule d'être produite par l'emitter donné, mais ne les supprimera PAS de la pièce). Cette fonction devrait toujours être appelée lorsque l'émetteur donné n'est plus nécessaire pour le système afin d'éviter les fuites de mémoire et les erreurs.

Syntaxe :

part_emitter_destroy( ps, ind );

Arguments Type Description
ps ID du système de particules The particle system to destroy the emitter from.
ind ID de l'émetteur de particules The index of the emitter to destroy.

Les retours :

N/A

Exemple :

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit1);
}

Le code ci-dessus va vérifier si l'émetteur de particules indexé dans la variable "p_emit" existe dans le système de particules donné et si c'est le cas, il est détruit.
