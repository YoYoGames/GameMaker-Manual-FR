partie_emitter_burst

Cette fonction vous permet de configurer un émetteur pour qu'il émette un type spécifique de particules. Elle est généralement utilisée dans les événements d'alarme et de destruction, car il s'agit d'un code unique qui crée le nombre de particules spécifié en une seule fois, selon la distribution, la forme et la position définies par la fonction. part_emitter_region(). Si vous avez besoin que les particules apparaissent à chaque étape, vous devez utiliser la fonction part_emitter_stream() plutôt que d'appeler cette fonction à chaque étape.

Syntaxe :

part_emitter_burst(ps, ind, parttype, number);

Arguments Type Description
ps ID du système de particules The particle system that the emitter is in.
ind ID de l'émetteur de particules The index of the emitter to burst from.
parttype Type de particule ID The index (type) of the particles to be created.
number Real The number of particles to create.

Les retours :

N/A

Exemple :

part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30));

Le code ci-dessus fera éclater un nombre aléatoire de particules entre 30 et 60 à partir de l'émetteur choisi.
