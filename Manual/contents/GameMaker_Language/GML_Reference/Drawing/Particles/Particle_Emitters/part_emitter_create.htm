création d'un émetteur partiel

Cette fonction est utilisée pour créer un nouvel émetteur de particules dans le système de particules donné. La fonction renvoie la valeur de l'index du nouvel émetteur qui doit être stockée dans une variable et utilisée dans toutes les autres fonctions qui font référence à l'émetteur. L'émetteur lui-même doit être détruit lorsqu'il n'est plus utilisé afin d'éviter les fuites de mémoire (ceci peut être réalisé en utilisant les fonctions spécifiques de destruction de l'émetteur ou en détruisant l'ensemble du système de particules auquel l'émetteur appartient).

NOTE Les ID des émetteurs renvoyés par cette fonction sont réutilisés, par exemple, si vous créez d'abord trois émetteurs de particules, les ID 0, 1 et 2 leur seront attribués. Si vous détruisez ensuite le deuxième émetteur (dont l'ID est 1) en utilisant la fonction part_emitter_destroy et que vous créez ensuite un autre émetteur, cet émetteur réutilisera l'ID de 1. Puisque l'ID de l'émetteur 2 est utilisé, l'émetteur suivant aura un ID de 3, et ainsi de suite.

Syntaxe :

part_emitter_create(ps);

Arguments Type Description
ps ID du système de particules The particle system to create the emitter in.

Les retours :

Particle Emitter ID

Exemple :

p_emit1 = part_emitter_create(Sname);

Cela créera un nouvel émetteur de particules et stockera son indice dans la variable "p_emit".
