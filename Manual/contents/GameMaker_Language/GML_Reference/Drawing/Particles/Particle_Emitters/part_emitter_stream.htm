part_emitter_stream

Cette fonction vous permet de configurer un émetteur pour qu'il diffuse un type spécifique de particules. Elle est généralement utilisée dans l'événement de création car il s'agit d'un code unique qui configure l'émetteur pour qu'il génère le nombre de particules spécifié à chaque étape du jeu. Les particules sont émises selon la distribution, la forme et la position définies par la fonction part_emitter_region(). Vous pouvez définir le nombre de particules émises à zéro et appeler à nouveau cette fonction pour "désactiver" l'émission de particules. La fonction acceptera également des nombres négatifs pour la quantité, auquel cas l'émetteur produira des particules en fonction du hasard. Par exemple, si le nombre de particules est fixé à -5, il y a une chance sur cinq pour qu'une particule soit produite à chaque étape. Si vous avez besoin que les particules apparaissent en une seule fois plutôt qu'à chaque étape, vous devez utiliser la fonction part_emitter_burst().

Syntaxe :

part_emitter_stream(ps, ind, parttype, number);

Arguments Type Description
ps ID du système de particules The particle system that the emitter is in.
ind ID de l'émetteur de particules The index of the emitter to burst from.
parttype Type de particule ID The index (type) of the particles to be created.
number Real The number of particles to create per step.

Les retours :

N/A

Exemple :

part_emitter_stream(global. Sname, p_emit1, p1, 1);

Le code ci-dessus va diffuser une particule à chaque étape du jeu jusqu'à ce que l'émetteur soit détruit ou que le flux soit mis à 0.
