Émetteurs de particules

Les émetteurs de particules sont utilisés par GameMaker pour émettre des particules sur une zone de l'écran qui peuvent avoir différentes formes et distributions. Ils peuvent également créer un flux continu de particules ou en faire jaillir un certain nombre d'un seul coup, selon la façon dont les fonctions sont utilisées.

Étant donné qu'un émetteur de particules est une ressource créée dynamiquement, vous devez le créer et stocker l'index renvoyé dans une variable afin de référencer l'émetteur dans tous les appels de fonction ultérieurs. Il est très important que vous détruisiez également l'émetteur lorsqu'il n'est plus nécessaire, sinon vous aurez une fuite de mémoire qui ralentira et finira par faire planter votre jeu. Il est également intéressant de noter que les émetteurs de particules sont conservés pour toujours après leur création, même si l'indice n'est plus stocké. Ainsi, même si vous changez de pièce ou redémarrez le jeu, les systèmes et les particules resteront visibles dans toutes les autres pièces. Vous devez donc vous assurer de les détruire dès que vous n'en avez plus besoin.

Les fonctions suivantes sont disponibles pour paramétrer les émetteurs et leur permettre de créer des particules. Notez que chacune d'entre elles reçoit en premier argument l'index du système de particules auquel elle appartient.

  • part_emitter_exists
  • part_emitter_create
  • part_emitter_clear
  • part_emitter_region
  • part_emitter_burst
  • part_emitter_stream
  • part_emitter_destroy
  • part_emitter_destroy_all
