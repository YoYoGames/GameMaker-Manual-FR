région_émetteur_partiel

Cette fonction est utilisée pour définir la position d'un émetteur de particules dans la pièce actuelle. Vous spécifiez le rectangle de délimitation de la zone dans la fonction, puis vous utilisez l'une des séries de constantes pour définir la forme finale de l'émetteur dans ce rectangle de délimitation ainsi que la distribution des particules dans la forme. Les constantes disponibles pour les formes de distribution sont énumérées ci-dessous :

Constante de forme de l'émetteur de particules
Constant Description
ps_shape_rectangle Une forme rectangulaire qui remplit la zone donnée.
ps_shape_ellipse Une ellipse, dont la largeur et la hauteur sont définies par la zone.
ps_shape_diamond Une forme de diamant dont les pointes sont situées à la moitié de la largeur et de la hauteur.
ps_shape_line Une seule ligne, dont le point de départ est la gauche et le haut et le point d'arrivée est la droite et le bas.

Outre les formes de distribution, vous pouvez également définir la courbe de distribution des particules qui doivent être émises. Les constantes disponibles pour les courbes de distribution sont énumérées ci-dessous :

Constante de distribution de l'émetteur de particules
Constant Description
ps_distr_linear Une distribution linéaire où toutes les particules ont une chance égale d'apparaître n'importe où dans la zone.
ps_distr_gaussian Une distribution gaussienne où plus de particules sont générées au centre plutôt que sur les bords.
ps_distr_invgaussian Une distribution gaussienne inverse où plus de particules sont générées sur les bords qu'au centre.

NOTE: Si vous avez besoin que l'emitter se déplace avec une instance, vous devrez utiliser cette fonction dans l'événement step et mettre à jour la position de l'emitter de cette façon. Il convient également de noter que pour les émissions ponctuelles où vous n'avez pas besoin de créer des particules sur une zone ou avec une distribution spécifique, il est souvent plus facile d'utiliser la fonction part_particles_create().

Syntaxe :

part_emitter_region(ps, ind, xmin, xmax, ymin, ymax, shape, distribution);

Arguments Type Description
ps ID du système de particules The particle system that the emitter is in.
ind ID de l'émetteur de particules The index of the emitter to set.
xmin Real The x coordinate of the left side of the region.
xmax Real The x coordinate of the right side.
ymin Real The y coordinate of the top of the region.
ymax Real The y coordinate of the bottom of the region.
shape Constante de forme de l'émetteur de particules The shape of the region.
distribution Constante de distribution de l'émetteur de particules The distribution style of the particles.

Les retours :

N/A

Exemple :

part_emitter_region(global.Sname , emit_1, x - 50, x + 50, y - 50, y + 50, ps_shape_ellipse, ps_distr_linear);

Le code ci-dessus va définir un émetteur pour une forme elliptique de 100px de haut et 100px de large autour de la position actuelle (x,y) de l'instance qui exécute le code.
