part_emitter_clear

Avec cette fonction, vous pouvez remettre l'emitter donné du système de particules spécifié dans son état par défaut. Cela arrêtera également toutes les particules qui sont diffusées par l'emitter à ce moment-là, et si vous souhaitez utiliser l'emitter à nouveau, vous devrez définir la position de la région et le type de particule en utilisant la fonction part_emitter_region() fonction.

Syntaxe :

part_emitter_clear(ps, ind);

Arguments Type Description
ps ID du système de particules The particle system that the emitter is in.
ind ID de l'émetteur de particules The index of the emitter to clear.

Les retours :

N/A

Exemple :

part_emitter_clear(Sname, p_emit1 );

Le code ci-dessus va effacer l'émetteur de particules indexé dans la variable "p_emit1".
