partie_type_couleur_rgb

Cette fonction vous permet de définir le mélange de couleurs rouge, vert et bleu pour toutes les particules créées du type donné. Vous fournissez une valeur minimale et une valeur maximale pour chacun des trois composants et les particules créées auront une couleur aléatoire basée sur la plage de paramètres donnée. Toutes les valeurs doivent être comprises entre 0 et 255.

Syntaxe :

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
rmin Real The minimum the final colour's red component can be.
rmax Real The maximum the final colour's red component can be.
gmin Real The minimum the final colour's green component can be.
gmax Real The maximum the final colour's green component can be.
bmin Real The minimum the final colour's blue component can be.
bmax Real The maximum the final colour's blue component can be.

Les retours :

N/A

Exemple :

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

Le code ci-dessus définit chaque particule émise du type de particule indexé dans la variable globale "Blood_Part" pour qu'elle soit uniquement de différentes nuances de rouge.
