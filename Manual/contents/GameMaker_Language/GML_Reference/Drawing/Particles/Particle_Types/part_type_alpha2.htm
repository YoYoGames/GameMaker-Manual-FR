partie_type_alpha2

Cette fonction peut être utilisée pour définir un gradient de valeur alpha (transparence) pour chaque particule créée du type donné. Le premier alpha est celui avec lequel toutes les particules commenceront, et le second alpha est celui avec lequel la particule finira, et un changement de gradient lisse se produira sur l'alpha au cours de la vie des particules de l'un à l'autre. Cela peut aller de 0 (transparent) à 1 (opaque).

Syntaxe :

part_type_alpha2(ind, alpha1, alpha2);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
alpha1 Real The start alpha of the particle.
alpha2 Real The end alpha of the particle.

Les retours :

N/A

Exemple :

part_type_alpha2(global.Snow_Part, 0, 1);

Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable globale "Snow_Part" pour avoir une valeur alpha de 0 (transparent) et ensuite s'estomper pour avoir un alpha de 1 (opaque) à la fin de leur vie.
