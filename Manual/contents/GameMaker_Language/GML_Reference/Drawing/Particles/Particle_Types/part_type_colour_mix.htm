type_de_partie_mélange_de_couleurs

Avec cette fonction, vous pouvez définir le type de particule donné comme étant un mélange aléatoire de deux couleurs.

Syntaxe :

part_type_colour_mix(ind, colour1, colour2);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
colour1 Couleur The first colour the blend will take from.
colour2 Couleur The second colour the blend will take from.

Les retours :

N/A

Exemple :

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

Le code ci-dessus définira la couleur de chaque particule émise du type de particule indexé dans la variable globale "P_Damage" pour être un mélange aléatoire entre les couleurs rouge et jaune.
