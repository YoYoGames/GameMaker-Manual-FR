partie_type_existe

Avec cette fonction, vous pouvez vérifier si le type de particule indexé existe ou non dans le jeu. Notez que si la variable vérifiée est une variable non initialisée (à laquelle un type de particule aurait normalement été assigné), une erreur sera générée.

Syntaxe :

part_type_exists(ind);

Arguments Type Description
ind Type de particule ID The index of the particle type to check for.

Les retours :

Boolean

Exemple :

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

Le code ci-dessus vérifie si la variable globale "p1" stocke l'index d'un type de particule et si ce n'est pas le cas, il en crée un.
