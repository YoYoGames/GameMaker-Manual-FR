partie_type_couleur2

Cette fonction peut être utilisée pour définir un gradient de deux couleurs pour chaque particule créée du type donné. La première couleur est celle avec laquelle toutes les particules commenceront, et la deuxième couleur est celle avec laquelle la particule finira, et un changement de gradient doux se produira à la couleur au cours de la vie des particules d'une couleur à l'autre.

Syntaxe :

part_type_colour2(ind, colour1, colour2);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
colour1 Couleur The colour the particle will start at.
colour2 Couleur The colour the particle will end at.

Les retours :

N/A

Exemple :

part_type_colour2( part_Health, c_red, c_white);

Le code ci-dessus fera en sorte que toutes les particules créées du type indexé dans la variable "part_Health" aient un mélange de couleurs du rouge au blanc pendant leur durée de vie.
