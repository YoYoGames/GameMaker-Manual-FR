type_de_pièce_couleur1

Cette fonction est utilisée pour définir un type de particule comme étant d'une seule couleur pour la durée totale de la vie de chaque particule individuelle.

Syntaxe :

part_type_colour1(ind, colour1);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
colour1 Couleur The single colour to make the particle type.

Les retours :

N/A

Exemple :

part_type_colour1(global.Snow_Part, c_white);

Le code ci-dessus fera en sorte que toutes les particules créées du type de particule indexé dans la variable globale "Snow_Part" soient uniquement blanches.
