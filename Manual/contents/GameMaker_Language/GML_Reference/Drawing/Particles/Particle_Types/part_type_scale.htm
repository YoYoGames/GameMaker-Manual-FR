partie_type_échelle

Cette fonction définit l'échelle horizontale et verticale de la particule avant tout autre changement de taille. Cette fonction fonctionne sur le ratio de la taille de sprite, donc une valeur de 1 (la valeur par défaut) est un ratio 1:1, ce qui signifie que 0,5 serait la moitié et 2 le double.

Syntaxe :

part_type_scale(ind, xscale, yscale);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
xscale Real The horizontal scaling of the base particle.
yscale Real The vertical scaling of the base particle.

Les retours :

N/A

Exemple :

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

Le code ci-dessus définit diverses valeurs de particules, y compris l'échelle qui définit l'échelle x à huit fois la taille et l'échelle y à la moitié de la taille.
