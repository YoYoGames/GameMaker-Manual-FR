partie_type_vie

Il s'agit de la fonction qui régit la durée pendant laquelle chaque particule individuelle du type indiqué reste à l'écran. Comme pour les autres fonctions de particules, vous fournissez une valeur minimale et maximale (en pas) et la durée de vie de chaque particule sera un nombre aléatoire de pas dans la plage spécifiée. Pour que toutes les particules aient la même durée de vie, définissez les deux valeurs pour qu'elles soient identiques.

Syntaxe :

part_type_life(ind, life_min, life_max);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
life_min Real The minimum lifespan of the particles.
life_max Real The maximum lifespan of the particles.

Les retours :

N/A

Exemple :

part_type_life(part_Flare, room_speed, room_speed * 2);

Le code ci-dessus fera en sorte que toutes les particules créées du type de particule indexé dans la variable "part_Flare" aient une durée de vie comprise entre 1 et 2 secondes, en fonction de la vitesse actuelle de la pièce.
