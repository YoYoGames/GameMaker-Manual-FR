part_type_clear

Avec cette fonction vous pouvez "réinitialiser" une particule, en ramenant toutes les valeurs de chacune des fonctions relatives à la particule (vie, couleur, alpha, orientation etc...) à leurs valeurs par défaut. Notez que cette fonction ne supprime pas de l'écran les particules actuellement visibles dans la pièce, pour cela vous devez utiliser la fonction part_particles_clear().

Syntaxe :

part_type_clear(ind);

Arguments Type Description
ind Type de particule ID The index of the particle type to clear.

Les retours :

N/A

Exemple :

part_type_clear(global.explode_part);

Le code ci-dessus remet le type de particule indexé dans la variable globale "explode_part" à ses valeurs par défaut.
