part_type_colour3

Cette fonction peut être utilisée pour définir un gradient de trois couleurs pour chaque particule créée du type donné. La première couleur est celle avec laquelle toutes les particules commenceront, la deuxième couleur est celle qui sera mélangée à la moitié de sa durée de vie et la troisième couleur est celle avec laquelle la particule finira. Un gradient de couleur se produira tout au long de la vie de la particule, d'une couleur à l'autre.

Syntaxe :

part_type_colour3( ind, colour1, colour2, colour3 );

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
colour1 Couleur The colour the particle will start at.
colour2 Couleur The colour the particle will be halfway through its lifespan.
colour3 Couleur The colour the particle will end at.

Les retours :

N/A

Exemple :

part_type_colour3( part_Health, c_red, c_white, c_maroon);

Le code ci-dessus fera en sorte que toutes les particules créées du type indexé dans la variable "part_Health" aient un mélange de couleurs allant du rouge au blanc et au marron au cours de leur vie.
