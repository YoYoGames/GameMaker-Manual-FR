partie_type_sprite

Cette fonction peut être utilisée pour définir un type de particule afin d'utiliser une sprite personnalisée à partir des actifs du jeu. Vous pouvez sélectionner l'adresse sprite et indiquer à GameMaker d'utiliser l'adresse sprite et de l'animer ou non. Si vous choisissez l'animation, vous pouvez également choisir de modifier les sous-images à un intervalle tel que l'animation commencera et se terminera au début et à la fin de la durée de vie de chaque particule créée (ainsi, si vous avez une durée de vie définie sur un minimum de 15 pas et un maximum de 30 pas, quelle que soit la valeur de la durée de vie, l'animation sera adaptée, avec une exécution plus rapide pour les durées de vie plus courtes et plus lente pour les plus longues). Vous pouvez également choisir de sélectionner une sous-image aléatoire de sorte que si le site sprite n'est pas animé, une autre image sera choisie au hasard pour créer le site sprite, ou si la particule est animée, l'animation commencera à un point aléatoire.

Syntaxe :

part_type_sprite(ind, sprite, animate, stretch, random);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
sprite Sprite Asset The index of the sprite to set the particle type to.
animate Booléen Whether to follow the sprite's animation normally (true) or not (false).
stretch Booléen Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random Booléen Whether to choose a random sub-image (true) or not (false).

Les retours :

N/A

Exemple :

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Le code ci-dessus définit le type de particule indexé dans la variable globale "p1" à la forme du sprite indexé dans "spr_Coins", puis définit diverses autres propriétés de la particule.
