part_type_direction

Cette fonction est utilisée pour déterminer la direction de la particule lors de sa création et peut également être utilisée pour que la particule augmente ou diminue sa direction au cours de sa vie. Les valeurs de direction minimale et maximale sont par défaut à 0, ce qui signifie que la particule se déplacera vers la droite sans variation, mais ces valeurs peuvent être modifiées pour limiter la direction du mouvement en suivant les directions standard de GameMaker, à savoir 0 pour la droite, 90 pour le haut, 180 pour la gauche et 270 pour le bas. Si vous les réglez sur la même valeur, les particules ne se déplaceront que dans une seule direction.

Vous pouvez également définir une valeur d'incrémentation qui ajoutera (s'il s'agit d'un nombre positif) ou soustraira (s'il s'agit d'un nombre négatif) une quantité de degrés à la direction pendant sa durée de vie. Cette valeur peut être au minimum de (+/-) 0,01.

Enfin, vous pouvez définir le facteur "wiggle". Il s'agit d'une valeur qui sera ajoutée ou soustraite aléatoirement à la direction à chaque étape de la vie des particules. Évidemment, les valeurs les plus grandes sont plus prononcées que les plus petites, et cette valeur peut même être négative, la plage maximale étant comprise entre -20 et 20.

Syntaxe :

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
dir_min Real The minimum direction the particle can start at.
dir_max Real The maximum direction the particle can start at.
dir_incr Real How much the particle direction should increase or decrease per step.
dir_wiggle Real How much should randomly be added or subtracted from the particle's direction per step.

Les retours :

N/A

Exemple :

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

Le code ci-dessus définira diverses valeurs de particules, y compris la direction qui sera une valeur aléatoire entre 0 et 359 degrés. Elle augmentera également d'un degré à chaque étape et une quantité aléatoire sera ajoutée à chaque étape, entre 0 et 20.
