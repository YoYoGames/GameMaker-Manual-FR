type_de_partie_créer

Avec cette fonction, vous pouvez créer un nouveau type de particule et la valeur de retour doit être stockée dans une variable pour être utilisée dans toutes les fonctions de particule suivantes.

Syntaxe :

part_type_create();

Les retours :

Particle Type ID

Exemple :

mypart = part_type_create();

Cela créera un nouveau type de particule, en stockant son index dans la variable "mypart".
