partie_type_gravité

Cette fonction va définir la gravité qui doit affecter chaque particule du type donné qui est créée. La valeur de la force de gravité est ajoutée à la vitesse de la particule à chaque étape et est généralement une petite valeur comme 0,5, tandis que la direction est la direction de la "traction" de la gravité et suit les directions standard de GameMaker de 0° pour la droite, 90° pour le haut, 180° pour la gauche et 270° pour le bas.

Syntaxe :

part_type_gravity(ind, grav_amount, grav_direction);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
grav_amount Real Strength of the gravity.
grav_direction Real The direction of the gravity.

Les retours :

N/A

Exemple :

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Le code ci-dessus définira diverses valeurs de particules, y compris la gravité qui ajoutera 0,2 à la vitesse à chaque pas avec une direction de 270º, ce qui tirera la particule "vers le bas" vers le bas de l'écran.
