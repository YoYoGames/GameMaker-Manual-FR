partie_type_couleur_hsv

Cette fonction vous permet de définir une gamme de teintes, de saturations et de valeurs pour toutes les particules du type donné. Vous fournissez une valeur minimale et une valeur maximale pour chacun des trois composants et les particules créées auront une couleur aléatoire basée sur la gamme de paramètres donnée. De cette façon, vous pouvez créer des particules de la même teinte mais de saturations différentes, ou de teintes différentes mais de la même valeur (luminosité) etc... Toutes les valeurs doivent être comprises entre 0 et 255.

Syntaxe :

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
hmin Real The minimum the final colour's hue component can be.
hmax Real The maximum the final colour's hue component can be.
smin Real The minimum the final colour's saturation component can be.
smax Real The maximum the final colour's saturation component can be.
vmin Real The minimum the final colour's value component can be.
vmax Real The maximum the final colour's value component can be.

Les retours :

N/A

Exemple :

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

Le code ci-dessus fait en sorte que chaque particule émise du type de particule indexé dans la variable globale "Stars" ait des couleurs et des saturations différentes, mais la même valeur (luminosité).
