type_partie_mélange

Avec cette fonction, vous pouvez faire en sorte que vos particules soient dessinées avec un mode de fusion additif (true) ou non (false). Le mode de fusion additif est un mode de fusion spécial qui ajoute les valeurs de luminosité de chaque particule à mesure qu'elles se chevauchent, de sorte que les couleurs claires deviennent progressivement plus claires (jusqu'à paraître blanches) à mesure qu'elles se chevauchent, et que les couleurs sombres deviennent de plus en plus transparentes, le noir étant presque invisible. Cette fonction peut donner de beaux effets de particules mais n'est pas toujours nécessaire. Par exemple, pour une traînée de fumée, le mélange additif sera désactivé pour que l'effet reste gris, mais pour un effet de flamme, il sera probablement activé pour rendre les particules plus translucides et plus brillantes.

Syntaxe :

part_type_blend(ind, additive);

Arguments Type Description
ind Type de particule ID The index of the particle type to change.
additive Booléen Whether the particles should be blended additively (true) or normally (false).

Les retours :

N/A

Exemple :

part_type_blend(part_Fire, true);

Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable "part_Fire" pour avoir un mode de fusion additif.
