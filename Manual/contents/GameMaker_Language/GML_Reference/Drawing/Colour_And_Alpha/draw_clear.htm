dessiner_clair

Cette fonction peut être utilisée pour effacer tout l'écran (sans mélange alpha) de la couleur donnée. Elle ne peut être utilisée que dans l'événement draw d'une instance (elle ne s'affichera pas si elle est utilisée dans un autre événement). Elle peut également être utile pour effacer les surfaces lorsqu'elles sont nouvellement créées.

Syntaxe :

draw_clear(col);

ArgumentsType Description
col The colour with which the screen will be cleared

Les retours :

Exemple :

draw_clear(c_blue);

Cela effacera l'écran avec la couleur bleue.

Dos : Couleur et alpha
