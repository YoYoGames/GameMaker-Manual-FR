draw_getpixel_ext

Avec cette fonction, vous pouvez obtenir la valeur AGBR 32 bits complète de tout pixel dessiné sur la cible de rendu actuelle. Cela signifie que les résultats dépendront de l'événement dans lequel la fonction est appelée, ainsi que de la surface cible utilisée.

IMPORTANT Cette fonction entraîne une baisse considérable des performances et ne doit donc être utilisée qu'en cas de nécessité absolue.

Syntaxe :

draw_getpixel_ext(x, y);

ArgumentsType Description
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

Les retours :

Real

Exemple :

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

Le code ci-dessus récupère la valeur de la couleur 32 bits à la position de la souris, puis la divise en ses valeurs composantes et les stocke dans des variables.
