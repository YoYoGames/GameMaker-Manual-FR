draw_clear_alpha

Cette fonction peut être utilisée pour effacer la totalité de l'écran avec une couleur donnée et la composante alpha de la destination est définie à la valeur que vous avez définie. peuvent être configurées pour utiliser l'alpha de destination que vous avez défini. Cette fonction est uniquement destinée à être utilisée dans l'événement de dessin d'une instance (elle ne s'affichera pas si elle est utilisée dans un autre événement), et elle peut également être très utile pour effacer les surfaces lorsqu'elles sont nouvellement créées.

Il est intéressant de noter que lorsque vous utilisez le module HTML5, vous pouvez utiliser cette fonction pour rendre l'arrière-plan du canevas transparent, ce qui permettra alors à toutes les images utilisées dans le HTML hôte d'être vues avec le jeu s'exécutant sur elles. Pour obtenir cet effet vous devez d'abord vous rendre dans l'éditeur de salle et supprimer toute couche d'arrière-plan de l'éditeur de salle, puis décocher l'option Clear View Background qui se trouve dans les propriétés de la vue. Enfin, vous devrez créer un nouvel objet, le placer dans la première pièce du jeu, et lui attribuer une profondeur très élevée (par exemple, 10000). Ensuite, dans l'événement Pre Draw, utilisez cette fonction pour définir l'alpha du canevas sur la couleur et la transparence que vous souhaitez lui donner. que vous souhaitez lui donner.

Syntaxe :

draw_clear_alpha(col, alpha);

ArgumentsType Description
col The colour with which the screen will be cleared
alpha The transparency of the colour with which the screen will be cleared

Les retours :

Exemple :

draw_clear_alpha(c_red, 0.5);

L'écran sera alors transparent à 50% (0,5) avec la couleur rouge.
