make_colour_hsv

GameMaker fournit cette fonction (ainsi que d'autres) pour permettre à l'utilisateur de créer ses propres couleurs. Cette fonction particulière prend trois éléments, la teinte, la saturation et la valeur (également appelée "luminosité") pour créer la couleur souhaitée. Ces valeurs sont comprises entre 0 et 255, ce qui permet de créer 16 777 216 (256*256*256) couleurs ! Vous pouvez voir ci-dessous une image de l'aspect de ces composants lorsqu'ils sont séparés :

L'image de gauche est une décomposition des différents composants de la fonction, et celle de droite est une illustration de la manière dont la modification de ces composants affecte la couleur finale. Comme vous pouvez le voir, même sans modifier la teinte (elle reste à 0, le rouge), nous pouvons changer radicalement la couleur avec les deux autres paramètres, la saturation faisant passer la couleur du gris (0) à un ton pur (255), et la valeur la faisant passer du plus sombre (0) au plus clair (255).

Syntaxe :

make_colour_hsv(hue, sat, val);

ArgumentsType Description
hue The hue of the colour
sat How saturated the colour is
val How dark the colour is

Les retours :

Colour Constant

Exemple :

col = make_colour_hsv(100, 145, 255);

Le code ci-dessus utilise la fonction pour créer une couleur et stocker sa valeur dans la variable "col" pour une utilisation ultérieure.

Dos : Couleur et alpha
