draw_set_alpha

Cette fonction permet de définir l'alpha de base du dessin pour le jeu. Cette valeur peut être définie de 0 à 1, 0 étant totalement transparent et 1 étant totalement opaque (valeur par défaut), et affectera tous les autres dessins, y compris les arrière-plans, sprites, les polices, primitives et la 3D.

Notez que si vous dessinez l'un des actifs et que vous fournissez un argument alpha dans le cadre de la fonction (par exemple, lorsque vous utilisez la commande draw_sprite_ext()), alors la valeur alpha définie ici sera ainsi, le dessin d'un sprite à l'aide de draw_sprite_ext() avec un argument alpha de 1 rendra le sprite totalement opaque, quel que soit l'alpha défini avec draw_set_alpha() (cela fonctionne de la même manière avec l'option draw_self() et en définissant le paramètre image_alpha de l'instance). Notez également que lorsqu'une instance dessine par défaut (c'est-à-dire qu'elle possède un sprite mais pas d'événement Draw), la variable image_alpha sera fixée à l'alpha défini par draw_set_alpha(), de sorte qu'une valeur de 1 pour image_alpha et un alpha de 0,5 pour le dessin dessineront sprite à l'alpha 0,5, tandis qu'une valeur de 0,25 pour image_alpha et un alpha de 0,5 pour le dessin dessineront à l'alpha 0,5. de 0,5 dessineront sprite avec un alpha de 0,25.

Syntaxe :

draw_set_alpha(alpha);

ArgumentsType Description
alpha The alpha to set (between 0 and 1)

Les retours :

Exemple :

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Le code ci-dessus dessine un texte à la position spécifiée avec un effet d'ombre créé en modifiant l'alpha et la couleur du texte.
