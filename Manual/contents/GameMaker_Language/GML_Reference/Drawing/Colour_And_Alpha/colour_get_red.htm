couleur_get_rouge

Cette fonction renvoie la quantité de rouge utilisée pour obtenir la couleur donnée, la valeur étant comprise entre 0 et 255, 0 correspondant à l'absence de rouge et 255 à un rouge total. L'image suivante illustre cela :

Syntaxe :

colour_get_red(col);

ArgumentsType Description
col The colour to check

Les retours :

Exemple :

r_comp = colour_get_red(c_teal);

Le code ci-dessus récupère la composante rouge de la constante de couleur "c_teal" et enregistre la valeur dans la variable "r_comp".
