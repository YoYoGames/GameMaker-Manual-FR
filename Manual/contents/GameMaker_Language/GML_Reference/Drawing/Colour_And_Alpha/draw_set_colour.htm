draw_set_colour

Cette fonction vous permet de définir la couleur de base du dessin pour le jeu.

Cela affectera le dessin des polices, des formulaires, primitives et 3D, mais pas sprites (dessiné manuellement ou par une instance). Si les graphiques concernés sont dessinés avec leurs propres valeurs de couleur, cette valeur sera ignorée.

Syntaxe :

draw_set_colour(col);

Arguments Type Description
col Couleur The colour to set for drawing.

Les retours :

N/A

Exemple :

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Le code ci-dessus dessine un texte à la position spécifiée avec un effet d'ombre créé en modifiant l'alpha et la couleur du texte.
