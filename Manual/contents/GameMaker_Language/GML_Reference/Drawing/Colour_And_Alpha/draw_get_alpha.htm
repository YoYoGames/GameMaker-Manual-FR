draw_get_alpha

Cette fonction renvoie la valeur actuelle de l'alpha du dessin, qui se situe entre 0 (totalement transparent) et 1 (totalement opaque). L'alpha du dessin affecte la transparence de toutes les fonctions de dessin, et peut être défini avec la fonction draw_set_alpha().

Syntaxe :

draw_get_alpha()

Les retours :

Exemple :

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

Le code ci-dessus stocke l'alpha actuel du dessin dans une variable locale, et modifie l'alpha du dessin en fonction d'une variable d'instance. Après avoir dessiné du texte, il réinitialise l'alpha à la valeur stockée dans la variable locale.
