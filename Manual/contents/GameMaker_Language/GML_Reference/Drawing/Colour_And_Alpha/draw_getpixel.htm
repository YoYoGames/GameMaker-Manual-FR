draw_getpixel

Avec cette fonction, vous pouvez obtenir la valeur de la couleur de n'importe quel pixel qui est dessiné sur la cible de rendu actuelle. Cela signifie que les résultats dépendront de l'événement dans lequel la fonction est appelée, ainsi que de la surface cible utilisée. Notez que cette fonction ne renverra aucune valeur alpha, pour cela vous devez utiliser draw_getpixel_ext().

IMPORTANT Cette fonction entraîne une baisse considérable des performances et ne doit donc être utilisée qu'en cas de nécessité absolue.

Syntaxe :

draw_getpixel(x, y);

ArgumentsType Description
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

Les retours :

Colour Constant

Exemple :

draw_set_colour(draw_getpixel(500,200));

Le code ci-dessus définit la couleur de dessin de tous les textes, formulaires et primitives à la couleur du pixel situé à (500,200) sur l'écran du jeu.
