colour_get_saturation

Cette fonction renvoie la saturation de la couleur donnée. Il s'agit de la quantité de tonalité de la couleur qui est mélangée à la couleur finale et qui fait partie de la méthode de teinte, saturation et valeur (luminosité) pour définir une couleur. L'image suivante illustre comment cette valeur correspond à l'échelle de couleurs HSV :

Syntaxe :

colour_get_saturation(col);

ArgumentsType Description
col The colour to check

Les retours :

Exemple :

col = make_colour_hsv(random(255), colour_get_sat(c_teal), 255);

Le code ci-dessus récupère la saturation utilisée pour créer la constante de couleur "c_teal" et utilise ensuite cette valeur pour définir une couleur aléatoire ayant la même saturation, en stockant la couleur résultante dans la variable "col".
