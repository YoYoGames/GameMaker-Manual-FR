couleur_get_green

Cette fonction renvoie la quantité de vert utilisée pour obtenir la couleur donnée, la valeur étant comprise entre 0 et 255, où 0 correspond à aucun vert et 255 à tout le vert. L'image suivante illustre cela :

Syntaxe :

colour_get_green(col);

ArgumentsType Description
col The colour to check

Les retours :

Exemple :

g_comp = colour_get_green(c_teal);

Le code ci-dessus récupère la composante verte de la constante de couleur "c_teal" et enregistre la valeur dans la variable "g_comp".
