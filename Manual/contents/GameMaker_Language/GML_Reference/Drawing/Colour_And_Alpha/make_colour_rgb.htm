make_colour_rgb

GameMaker fournit cette fonction (ainsi que d'autres) pour permettre à l'utilisateur de créer ses propres couleurs. Cette fonction particulière prend trois composants, le rouge, le vert et le bleu de la couleur que vous souhaitez créer. Ces valeurs sont comprises entre 0 et 255, ce qui vous permet de créer 16 777 216 (256*256*256) couleurs ! Vous pouvez voir ci-dessous une image de l'aspect de ces composants lorsqu'ils sont séparés :

L'image de gauche est une décomposition des différents composants de la fonction, puis à droite, une illustration de la manière dont la modification de ces composants affecte la couleur finale.

Syntaxe :

make_colour_rgb(red, green, blue);

ArgumentsType Description
red The red component of the colour
green The green component of the colour
blue The blue component of the colour

Les retours :

Colour Constant

Exemple :

col = make_colour_rgb(100, 145, 255);

Le code ci-dessus utilise la fonction pour créer une couleur et stocker sa valeur dans la variable "col" pour une utilisation ultérieure.
