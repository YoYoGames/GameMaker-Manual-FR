draw_get_colour

Cette fonction renvoie la couleur de dessin actuelle qui est utilisée pour dessiner les formulaires, le texte, primitives et les modèles 3D non texturés. Elle peut être définie avec la fonction draw_set_colour() fonction.

Syntaxe :

draw_get_colour()

Les retours :

Colour Constant

Exemple :

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

Le code ci-dessus stocke la couleur de dessin actuelle dans une variable locale, et change la couleur de dessin en fonction d'une variable d'instance. Après avoir dessiné du texte, il réinitialise la couleur à la valeur stockée dans la variable locale.
