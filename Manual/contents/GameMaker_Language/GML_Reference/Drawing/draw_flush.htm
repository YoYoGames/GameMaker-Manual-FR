draw_flush

Avec cette fonction, vous pouvez vider l'ensemble du pipeline de dessin. Il s'agit d'une fonction de débogage uniquement et, d'une manière générale, elle ne doit pas être utilisée, sauf sur indication d'un membre de l'équipe d'assistance YoYo Games. car son utilisation sans discernement entraînera de graves problèmes de performances dans votre jeu.

Syntaxe :

draw_flush();

Les retours :

Exemple :

draw_flush();

Le code ci-dessus vide le pipeline de dessin.
