draw_light_define_ambient

Cette fonction est utilisée pour contrôler la lumière ambiante d'une scène, c'est-à-dire la lumière que vous avez dans une scène même sans avoir défini de sources lumineuses ponctuelles ou directionnelles. Il s'agit en fait de la couleur et de la luminosité (ou de l'obscurité) générales d'une scène. La couleur par défaut est c_black.

Syntaxe :

draw_light_define_ambient(col);

ArgumentsType Description
col The colour to use (either a constant, a real or a hex value).

Les retours :

Exemple :

draw_light_define_ambient(c_white);

Le code ci-dessus définira l'éclairage ambiant comme étant blanc.
