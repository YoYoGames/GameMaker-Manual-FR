draw_light_get

Cette fonction obtiendra les paramètres de lumière spécifiés sous forme de tableau avec les 6 éléments suivants -

  • [0] = activé / désactivé (true / false)
  • [1] = Type de lumière constante (voir tableau ci-dessous)
  • [2] = position x
  • [3] = position y
  • [4] = position z
  • [5] = rayon de la lumière (uniquement pour les lumières ponctuelles, 0 pour les lumières directionnelles)
  • [6] = couleur claire (un vrai)

Le type de lumière (élément [1]) peut être l'une des deux constantes suivantes :

Type de lumière constante
Constant Description
lighttype_dir La lumière est une lumière directionnelle
lighttype_point La lumière est une lumière ponctuelle

Syntaxe :

draw_light_get(ind);

Arguments Type Description
ind Real The index number of the light (from 0 to 7)

Les retours :

Array

Exemple :

light_a = draw_light_get(1);
if (light_a[5] < 200)
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

Le code ci-dessus obtiendra les valeurs utilisées pour définir la lumière indexée comme "1", puis vérifiera le rayon et s'il est inférieur à 200, il sera augmenté et le rayon de la lumière sera fixé à la nouvelle valeur.
