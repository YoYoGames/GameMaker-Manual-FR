draw_get_lighting

Cette fonction indique si l'éclairage est activé (true) ou non (false) pour l'ensemble de la scène.

Syntaxe :

draw_get_lighting()

Les retours :

Exemple :

if !draw_get_lighting()
{
    draw_set_lighting(true);
}

Le code ci-dessus vérifie si l'éclairage est activé pour la scène, et si non, il l'active.
