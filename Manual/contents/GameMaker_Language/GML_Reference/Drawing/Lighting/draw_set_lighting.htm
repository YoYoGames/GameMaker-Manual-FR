draw_set_lighting

Cette fonction est utilisée pour activer tous les effets de lumière. Par défaut, elle est désactivée (false).

Syntaxe :

draw_set_lighting(enable);

Arguments Type Description
enable Booléen Enable or disable all lighting (true ou faux)

Les retours :

N/A

Exemple :

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Le code ci-dessus permet d'activer l'éclairage de l'ensemble de la scène, puis de définir une lumière blanche directionnelle dans l'espace de la pièce, et enfin d'allumer cette lumière.
