draw_set_swf_aa_level

Cette fonction peut être utilisée pour définir le niveau d'anticrénelage (AA) pour les vecteurs du format SWF sprites. Il peut s'agir d'une valeur réelle comprise entre 0 et 1 qui "lisse" les bords de ces sprites. Notez que pour voir cet effet, vous devez d'abord avoir activé l'AA à l'aide de la fonction draw_enable_swf_aa().

Syntaxe :

draw_set_swf_aa_level(AA);

ArgumentsType Description
AA The anti-aliasing value to use from 0 to 1.

Les retours :

Exemple :

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Le code ci-dessus vérifie la valeur AA pour le format SWF sprites, et si elle est égale à 0, il active AA et fixe la valeur à 0,5.
