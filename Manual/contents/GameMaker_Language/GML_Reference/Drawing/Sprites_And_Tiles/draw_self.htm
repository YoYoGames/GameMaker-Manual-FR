draw_self

Cette fonction dessine le sprite assigné à l'instance exactement comme il serait dessiné si l'événement de dessin ne contenait aucun code ou action, et reflétera les changements apportés aux variables du sprite dans d'autres événements.

NOTE: Si l'instance n'a pas de sprite assigné à elle, cette fonction va lancer une erreur !

Syntaxe :

draw_self();

Les retours :

Exemple :

draw_self();

L'instance se dessine alors avec les propriétés définies par les variables intégrées de sprite.
