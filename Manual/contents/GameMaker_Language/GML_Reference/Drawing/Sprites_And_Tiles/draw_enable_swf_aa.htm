draw_enable_swf_aa

Cette fonction vous permet d'activer ou de désactiver l'anticrénelage (AA) pour le format vectoriel SWF sprites. L'AA lisse simplement les bords des images vectorielles pour leur donner un aspect plus agréable. La quantité d'AA utilisée dépendra de la valeur définie à l'aide de la fonction draw_set_swf_aa_level(). Par défaut, cette fonction est désactivée.

Syntaxe :

draw_enable_swf_aa(enable);

ArgumentsType Description
enable Enable (true) ou désactiver (false) AA pour tous les SWF sprites.

Les retours :

Exemple :

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Le code ci-dessus vérifie la valeur AA pour le format SWF sprites, et si elle est égale à 0, il active AA et fixe la valeur à 0,5.
