draw_sprite_stretched_ext

Cette fonction fait exactement la même chose que la fonction draw_sprite_stretched() avec la possibilité supplémentaire de définir le mélange de couleurs et la valeur alpha pour le site sprite lorsqu'il est dessiné (similaire à la fonction draw_sprite_ext()).

REMARQUE Lorsque vous dessinez avec cette fonction, le décalage x et le décalage y (ou origines) de sprite sont ignorés et sprite est dessiné avec le coin supérieur gauche à la position x/y spécifiée dans la pièce.

REMARQUE Le mélange des couleurs n'est recommandé pour la cible HTML5 que lorsque WebGL est activé, bien que vous puissiez toujours définir la couleur de mélange si elle n'est pas activée et le mélange de sprite se fera normalement. Cependant, tout mélange de cette manière crée un double de sprite qui est ensuite stocké dans cache et utilisé au besoin. C'est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu, à moins que vous n'activiez WebGL. Si vous ne souhaitez pas utiliser WebGL, vous pouvez définir la taille de la police cache pour essayer de limiter ce phénomène si nécessaire en utilisant la fonction sprite_set_cache_sizee().

Syntaxe :

draw_sprite_stretched_ext(sprite, subimg, x, y, w, h, colour, alpha);

ArgumentsType Description
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.
w The width of the area the stretched sprite will occupy.
h The height of the area the stretched sprite will occupy.
colour The colour with which to blend the sprite. c_white is to display it normally.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Les retours :

N/A

Exemple :

draw_sprite_stretched_ext(sprite_index, image_index, x, y, sprite_width, sprite_height / 2, c_white, 0.5);

Ceci dessinera les instances assignées à sprite et sa sous-image avec le coin gauche à la position x/y de l'instance. Sa largeur est la même que celle de sprite, et sa hauteur est celle de sprite divisée par deux. Elle sera également mélangée normalement mais aura une valeur alpha partiellement transparente.
