draw_sprite_pos

Avec cette fonction, vous pouvez dessiner un sprite déformé sur la zone définie par les quatre coordonnées d'angle. Les deux premiers arguments sont le sprite à dessiner et la sous-image du sprite (la même que dans la fonction draw_sprite()) mais les arguments suivants sont ceux qui définissent la position de chacun des quatre coins de la boîte englobante du site sprite donné. Elles doivent être données dans le sens des aiguilles d'une montre, de sorte que la première coordonnée est le coin supérieur gauche, puis le coin supérieur droit, puis le coin inférieur droit et enfin le coin inférieur gauche. droite et enfin le coin inférieur gauche. Vous pouvez également définir une valeur pour l'alpha du site sprite afin de le dessiner avec transparence. L'image ci-dessous illustre le fonctionnement de cette fonction :

AVERTISSEMENT ! L'image ci-dessus n'est qu'une illustration. Si vous utilisez cette fonction sur un sprite, vous obtiendrez des résultats différents et vous risquez de subir un "cisaillement" de la texture en raison de la façon dont un sprite est construit à partir d'un quadruple primitives.

NOTE: Cette fonction n'est utile que pour les bitmaps sprites et ne fonctionne pas avec les SWF ou JSON (Spine) sprites.

REMARQUE: lorsque vous utilisez cette fonction, le paramètre Recadrage automatique doit être désactivé pour les pages de texture.

Syntaxe :

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

ArgumentsType Description
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x1 The first x coordinate.
y1 The first y coordinate.
x2 The second x coordinate.
y2 The second y coordinate.
x3 The third x coordinate.
y3 The third y coordinate.
x4 The fourth x coordinate.
y4 The fourth y coordinate.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Les retours :

Exemple :

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

Le code ci-dessus dessinera le site sprite associé à l'instance qui exécute le code, déformé autour de la position x/y de l'instance et avec un alpha totalement opaque.
