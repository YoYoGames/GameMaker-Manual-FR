gpu_get_state

Cette fonction récupère l'état actuel du GPU et le renvoie sous la forme d'une carte DS. Celle-ci peut ensuite être manipulée ou même sauvegardée, et vous pouvez renvoyer cette carte au GPU en utilisant la fonction gpu_set_state().

Syntaxe :

gpu_get_state();

Les retours :

DS Map ID

Exemple :

gpu_map = gpu_get_state();

Le code ci-dessus stocke l'état actuel du GPU dans une variable.
