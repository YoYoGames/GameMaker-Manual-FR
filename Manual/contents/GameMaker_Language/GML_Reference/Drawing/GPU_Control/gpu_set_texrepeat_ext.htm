gpu_set_texrepeat_ext

Cette fonction peut être utilisée pour définir si un seul "slot" d'échantillonneur répète la texture donnée lors de l'utilisation de Shaders dans GameMaker. Si vous lui attribuez la valeur true, la texture sera répétée si les coordonnées uv se situent dans la plage 0-1, tandis que si vous lui attribuez la valeur false, la texture ne sera pas répétée. Le cas le plus probable d'utilisation de ces fonctions est la répétition d'une texture en 3D, mais pour que cela fonctionne et ne tire pas les images du reste de la page de texture, le site sprite utilisé devra être marqué comme étant sur une "page de texture séparée" dans l'éditeur Sprite.

NOTE Ce paramètre sera remplacé par la valeur définie lors de l'appel de la fonction gpu_set_texrepeat().

Syntaxe :

gpu_set_texrepeat_ext(sampler_id, enable);

ArgumentsType Description
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Les retours :

Exemple :

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texfilter_ext(s_tex))
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Le code ci-dessus vérifie si le filtrage de texture est activé ou non pour un ID d'échantillonneur spécifique (stocké dans une variable locale) et l'active en conséquence.
