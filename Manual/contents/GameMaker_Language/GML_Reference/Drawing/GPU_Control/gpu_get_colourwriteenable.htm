gpu_get_colourwriteenable

Cette fonction peut être utilisée pour récupérer les valeurs actuelles d'activation d'écriture des couleurs. La fonction renvoie un tableau 1D à 4 éléments contenant les éléments suivants qui seront soit true (activé) soit false (désactivé). Par défaut, l'écriture de toutes les couleurs est réglée sur true:

  • [0] = Canal rouge activé/désactivé
  • [1] = Canal vert activé/désactivé
  • [2] = Canal bleu activé/désactivé
  • [3] = Canal alpha activé/désactivé

Syntaxe :

gpu_get_colorwriteenable();

Les retours :

(4 elements only; see above for details)

Exemple :

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Le code ci-dessus récupère les valeurs d'écriture de la couleur actuelle et définit ensuite la composante alpha à false.
