gpu_set_state

Cette fonction définit l'état actuel du GPU en utilisant la carte DS fournie. La carte fournie peut être créée à l'aide de la fonction gpu_get_state().

Syntaxe :

gpu_set_state(ds_map);

ArgumentsType Description
ds_map The GPU state to set as a DS Map.

Les retours :

Exemple :

gpu_set_state(gpu_map);

Le code ci-dessus définit l'état du GPU en utilisant la carte fournie dans la variable "gpu_map".
