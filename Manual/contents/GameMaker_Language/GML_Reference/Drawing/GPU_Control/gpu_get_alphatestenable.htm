gpu_get_alphatestenable

Cette fonction vous permet de vérifier si le test alpha est activé (renvoie true) ou non (renvoie false). Pour plus d'informations sur le test alpha, voir la fonction gpu_set_alphatestref().

Syntaxe :

gpu_get_alphatestenable();

Les retours :

Exemple :

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

Le code ci-dessus vérifie si le test d'alpha est activé et, si ce n'est pas le cas, il active le test d'alpha et fixe le seuil de test à 128 (seuls les pixels dont l'alpha est supérieur à 0,5 seront dessinés).
