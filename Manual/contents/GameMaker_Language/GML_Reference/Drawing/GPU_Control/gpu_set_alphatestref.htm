gpu_set_alphatestref

Cette fonction permet de définir la valeur de référence pour le test alpha (lorsqu'il a été activé à l'aide de la touche gpu_set_alphatestenable()). Il s'agit du seuil de "coupure" à partir duquel les pixels avec alpha ne seront pas dessinés.

En fait, si cette valeur est égale à 0 et que l'interpolation linéaire est activée (soit dans les options du jeu, soit à l'aide de la commande gpu_set_texfilter()), vous pouvez toujours obtenir des erreurs visuelles sur les bords des objets qui se chevauchent. En effet, vous pouvez avoir des zones presque entièrement transparentes qui bloquent les objets dessinés derrière elles, mais si vous définissez cette valeur de référence à, par exemple, 254 (c'est-à-dire une valeur inférieure au maximum), cela garantit que seuls les pixels 100 % opaques seront dessinés. Vous pouvez également désactiver l'interpolation des textures, ce qui vous permet normalement d'utiliser les valeurs par défaut sans problème et sans avoir besoin d'utiliser cette fonction, mais cela dépend beaucoup de l'œuvre source.

REMARQUE: Il s'agit d'une fonction de dessin avancée, donc si vous n'êtes pas sûr d'en avoir besoin ou si vous ne savez pas exactement ce qu'elle fait, il est préférable de ne pas l'utiliser car elle peut sérieusement corrompre la façon dont votre jeu est dessiné.

Syntaxe :

gpu_set_alphatestref(val);

ArgumentsType Description
val Sets the reference value for alpha testing from 0 to 255 (default value is 0)

Les retours :

Exemple :

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

Le code ci-dessus vérifie si le test d'alpha est activé et, si ce n'est pas le cas, il active le test d'alpha et fixe le seuil de test à 128 (seuls les pixels dont l'alpha est supérieur à 0,5 seront dessinés).
