Contrôle du GPU

Lorsque vous dessinez des objets dans une pièce, vous aurez souvent besoin d'un contrôle plus granulaire sur la façon dont le dessin sera réalisé. Vous pouvez souhaiter qu'il n'y ait pas de mélange alpha, ou vous pouvez avoir besoin d'un détourage, ou vous pouvez avoir besoin d'activer ou de désactiver l'interpolation de texture. Tout cela est possible - et bien plus encore ! - en utilisant les fonctions du contrôleur GPU listées ci-dessous. Notez que la plupart de ces fonctions ont une portée globale et qu'une fois définies ou modifiées, elles affecteront la façon dont tout sera dessiné à partir de ce moment-là.

Les fonctions suivantes peuvent être utilisées pour obtenir les valeurs de différentes propriétés du GPU :

  • gpu_get_blendenable
  • gpu_get_ztestenable
  • gpu_get_zfunc
  • gpu_get_zwriteenable
  • gpu_get_fog
  • gpu_get_cullmode
  • gpu_get_blendmode
  • gpu_get_blendmode_ext
  • gpu_get_blendmode_ext_sepalpha
  • gpu_get_blendmode_src
  • gpu_get_blendmode_dest
  • gpu_get_blendmode_srcalpha
  • gpu_get_blendmode_destalpha
  • gpu_get_colourwriteenable
  • gpu_get_alphatestenable
  • gpu_get_alphatestref
  • gpu_get_texfilter
  • gpu_get_texfilter_ext
  • gpu_get_texrepeat
  • gpu_get_texrepeat_ext

Les fonctions suivantes peuvent être utilisées pour définir les valeurs de différentes propriétés du GPU :

  • gpu_set_blendenable
  • gpu_set_ztestenable
  • gpu_set_zfunc
  • gpu_set_zwriteenable
  • gpu_set_fog
  • gpu_set_cullmode
  • gpu_set_blendmode
  • gpu_set_blendmode_ext
  • gpu_set_blendmode_ext_sepalpha
  • gpu_set_colourwriteenable
  • gpu_set_alphatestenable
  • gpu_set_alphatestref
  • gpu_set_texfilter
  • gpu_set_texfilter_ext
  • gpu_set_texrepeat
  • gpu_set_texrepeat_ext

Veuillez noter qu'avant d'utiliser les fonctions du mode de fusion, il est important de comprendre comment elles fonctionnent afin d'obtenir des résultats corrects lors de leur application. Pour cela, nous avons créé le guide suivant :

  • Guide d'utilisation des Blendmodes

Enfin, il existe les fonctions suivantes qui peuvent être utilisées pour manipuler la pile GPU :

  • gpu_push_state
  • gpu_pop_state
  • gpu_get_state
  • gpu_set_state
