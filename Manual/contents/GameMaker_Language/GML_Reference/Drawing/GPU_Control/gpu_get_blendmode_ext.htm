gpu_get_blendmode_ext

Cette fonction peut être utilisée pour récupérer le mode de fusion étendu actuel utilisé pour le dessin. La fonction renvoie un tableau 1D à 2 éléments contenant les éléments suivants :

  • [0] = Source Constante du facteur de mode de mélange (la valeur par défaut est bm_src_alpha)
  • [1] = Destination Facteur de mode de mélange constant (la valeur par défaut est bm_inv_src_alpha)

Les valeurs du tableau seront l'une des constantes suivantes ("s" désigne une valeur provenant de la source tandis qu'un "d" désigne une valeur provenant de la destination) :

Facteur de mode de mélange Constant
Constant Facteur de mélange (rouge, vert, bleu, alpha)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-R, 1-G, 1-B, 1-A)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Ad, Ad, Ad, Ad, Ad)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) où f = min(As, 1-Ad)

Notez que vous pouvez modifier ces valeurs et passer le tableau complet à la fonction gpu_set_blendmode_ext() comme un seul argument.

Syntaxe :

gpu_get_blendmode_ext();

Les retours :

Array (2 elements only; see above for constants)

Exemple :

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

Le code ci-dessus récupère le mode de fusion étendu actuel, modifie la source, puis définit à nouveau le mode de fusion étendu.
