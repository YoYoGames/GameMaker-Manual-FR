gpu_get_zfunc

Cette fonction peut être utilisée pour récupérer le mode de comparaison z. La valeur renvoyée sera l'une des constantes suivantes (la valeur par défaut est cmpfunc_lessequal) :

Fonction de comparaison Constante
Constant Description
cmpfunc_never Jamais
cmpfunc_less Moins
cmpfunc_equal Equal
cmpfunc_lessequal Inférieur ou égal
cmpfunc_greater Greater
cmpfunc_notequal Non égal
cmpfunc_greaterequal Supérieur ou égal
cmpfunc_always Toujours

Syntaxe :

gpu_get_zfunc();

Les retours :

Comparison Function Constant (see table above)

Exemple :

if gpu_get_zfunc() != cmpfunc_greater
{
    gpu_set_zfunc(cmpfunc_greater);
}

Le code ci-dessus vérifie si la méthode de test z est définie sur cmpfunc_greater et si ce n'est pas le cas, elle est définie sur cette constante.
