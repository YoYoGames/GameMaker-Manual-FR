gpu_set_texfilter

Cette fonction peut être utilisée pour définir le filtrage des textures (interpolation linéaire) de toutes les images dessinées sur l'écran du jeu. Lorsque cette fonction est activée (true), toutes les textures seront lissées lorsqu'elles seront dessinées (y compris sprites car elles sont également considérées comme des textures), ce qui signifie qu'en cas de mise à l'échelle ou de déplacement, s'il n'y a pas un rapport de pixel de 1:1, il y aura une "tache" sur les différents pixels, ce qui peut faire paraître les images floues selon le style artistique utilisé. Si cette option est désactivée (false), les images seront dessinées sur la base du pixel le plus proche lorsqu'elles seront mises à l'échelle ou déplacées, ce qui peut donner des images "en bloc". La valeur par défaut est false, et elle peut également être modifiée dans les options globales du jeu pour les différentes plates-formes cibles.

REMARQUE: Ce paramètre remplacera toute interpolation d'étage de texture définie pour shaders à l'aide de la fonction gpu_set_texfilter_ext().

REMARQUE: sur la cible HTML5, cette fonction ne fonctionnera que si WebGL est activé.

Syntaxe :

gpu_set_texfilter(enable);

ArgumentsType Description
enable Enable or disable texture filtering (true / false)

Les retours :

Exemple :

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

Le code ci-dessus vérifie si le filtrage des textures est activé ou non et l'active en conséquence.
