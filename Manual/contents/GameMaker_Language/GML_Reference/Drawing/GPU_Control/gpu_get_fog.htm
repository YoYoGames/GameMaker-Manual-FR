gpu_get_fog

Cette fonction peut être utilisée pour récupérer les paramètres du brouillard. La fonction renvoie un tableau 1D à 4 éléments avec les informations suivantes :

  • [0] = activation de la bascule (un booléen, soit true ou false), par défaut false
  • [1] = Couleur constante, par défaut c_black
  • [2] = distance de départ(réel), par défaut 0
  • [3] = distance finale(réel), par défaut 1

Notez que vous pouvez modifier ces valeurs et passer le tableau complet à la fonction gpu_set_fog() comme un seul argument.

Syntaxe :

gpu_get_fog();

Les retours :

Array (4 elements only; see above for details)

Exemple :

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Le code ci-dessus récupère les paramètres actuels du brouillard et définit ensuite l'élément de couleur du tableau à c_red avant de définir à nouveau le brouillard en utilisant le tableau modifié.
