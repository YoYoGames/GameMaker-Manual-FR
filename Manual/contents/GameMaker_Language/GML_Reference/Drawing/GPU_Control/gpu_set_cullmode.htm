gpu_set_cullmode

Cette fonction peut être utilisée pour définir le mode d'élimination de la face arrière. Un polygone (primitive) a une face avant et une face arrière. On dit que le côté avant est le côté où les sommets sont définis dans le sens inverse des aiguilles d'une montre. Normalement, les deux côtés sont dessinés, mais si vous créez une forme fermée (comme une pyramide), c'est un gaspillage de puissance de traitement car le côté arrière du triangle ne sera jamais visible. C'est dans ces cas-là que vous pouvez activer l'élimination des faces arrières, ce qui permet d'économiser environ la moitié du temps de dessin, même si cela signifie que vous devez définir vos polygones de la bonne manière pour éviter les problèmes.

La valeur du mode d'élimination utilisée doit être l'une des constantes suivantes (la valeur par défaut est cull_noculling) :

Constant Description
cull_noculling Aucun abattage ne sera effectué
cull_clockwise Tous les triangles dans le sens des aiguilles d'une montre seront éliminés
cull_counterclockwise Tous les triangles dans le sens inverse des aiguilles d'une montre seront éliminés.

Syntaxe :

gpu_set_cullmode(cullmode);

Arguments Type Description
cullmode Mode d'abattage Constant The cull mode to use (see list above)

Les retours :

N/A

Exemple :

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Le code ci-dessus récupère le mode d'élimination actuel et s'il n'est pas cull_clockwise, il est réglé sur cette constante.
