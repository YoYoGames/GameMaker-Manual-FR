gpu_set_zfunc

Cette fonction peut être utilisée pour définir le mode de comparaison des tests du tampon z (voir gpu_set_ztestenable() pour plus d'informations). Les valeurs disponibles sont l'une des constantes suivantes (la valeur par défaut est cmpfunc_lessequal) :

Constant Description
cmpfunc_never Jamais
cmpfunc_less Moins
cmpfunc_equal Equal
cmpfunc_lessequal Inférieur ou égal
cmpfunc_greater Greater
cmpfunc_notequal Non égal
cmpfunc_greaterequal Supérieur ou égal
cmpfunc_always Toujours

Syntaxe :

gpu_set_zfunc(cmp_func);

Arguments Type Description
cmp_func Fonction de comparaison Constante The comparison mode to use (see list above)

Les retours :

N/A

Exemple :

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Le code ci-dessus active le test du tampon Z et définit son mode de comparaison avant de dessiner un arrière-plan sprite, puis le désactive à nouveau pour poursuivre le dessin.
