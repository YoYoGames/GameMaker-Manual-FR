gpu_set_colourwriteenable

Cette fonction vous permet d'activer ou de désactiver les canaux de couleur et le canal alpha pour tous les dessins ultérieurs. Par exemple, vous pouvez utiliser cette fonction pour créer des surfaces à canal alpha uniquement (désactivez l'écriture RVB avant d'utiliser la surface), ou pour créer des effets spéciaux lors du dessin à l'écran. La valeur par défaut de chacun des composants est true, et peut être fournie sous la forme de quatre arguments uniques ou d'un tableau 1D à 4 éléments contenant les éléments suivants, qui seront soit true (activé), soit false (désactivé) :

  • [0] = Canal rouge activé/désactivé
  • [1] = Canal vert activé/désactivé
  • [2] = Canal bleu activé/désactivé
  • [3] = Canal alpha activé/désactivé

Syntaxe :

gpu_set_colorwriteenable(red, green, blue, alpha);

Arguments Type Description
red Booléen Enable/disable the red channel
green Booléen Enable/disable the green channel
blue Booléen Enable/disable the blue channel
alpha Booléen Enable/disable the alpha channel

Les retours :

N/A

Exemple :

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Le code ci-dessus récupère les valeurs d'écriture de la couleur actuelle et définit ensuite la composante alpha à false.
