gpu_get_cullmode

Cette fonction peut être utilisée pour récupérer le mode d'élimination de la face arrière. La valeur renvoyée sera l'une des constantes suivantes (la valeur par défaut est cull_noculling) :

Mode d'abattage Constant
Constant Description
cull_noculling Aucun abattage ne sera effectué
cull_clockwise Tous les triangles dans le sens des aiguilles d'une montre seront éliminés
cull_counterclockwise Tous les triangles dans le sens inverse des aiguilles d'une montre seront éliminés.

Syntaxe :

gpu_get_cullmode();

Les retours :

Culling Mode Constant (see above for constants)

Exemple :

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Le code ci-dessus récupère le mode d'élimination actuel et s'il n'est pas cull_clockwise, il est réglé sur cette constante.
