gpu_set_fog

Cette fonction peut être utilisée pour activer ou désactiver le dessin du brouillard. Le brouillard peut être utilisé dans les jeux 3D pour faire en sorte que les instances situées au loin soient estompées ou même invisibles, ce qui permet de créer une atmosphère et de masquer le fait que vous ne dessinez pas les instances situées au loin.

Vous définissez s'il est activé (true) ou désactivé (false), la couleur que le brouillard doit utiliser pour le mélange, ainsi que les distances de début et de fin du dessin. Les valeurs de distance sont essentiellement des valeurs de profondeur en pixels, et sont relatives à la position de la caméra.

Par exemple, une position de 0 est la plus proche de la caméra, et à chaque augmentation, elle s'éloigne. Par défaut, la caméra d'une vue est placée à une profondeur de -16000, donc si vous voulez que le brouillard commence à la profondeur 0 et se termine à la profondeur 1000, vous devez définir la distance de début à 16000 et la distance de fin à 17000.

La fonction peut prendre quatre arguments individuels (donnés ci-dessus), ou un seul tableau avec la structure suivante (l'exemple de code ci-dessous montre cette méthode) :

  • [0] = activation de la bascule (un booléen, soit true ou false), par défaut. false
  • [1] = couleur (réelle), par défaut c_black
  • [2] = distance de départ (réelle), par défaut 0
  • [3] = distance finale (réelle), par défaut 1

Syntaxe :

gpu_set_fog(enable, colour, start, end);

Arguments Type Description
enable Booléen Enable or disable fog
colour Couleur The fog colour
start Real The distance to start applying fog from (relative to the camera)
end Real The distance to end the fog (relative to the camera)

Les retours :

N/A

Exemple :

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Le code ci-dessus récupère les paramètres actuels du brouillard et définit ensuite l'élément de couleur du tableau à c_red avant de définir à nouveau le brouillard en utilisant le tableau modifié.
