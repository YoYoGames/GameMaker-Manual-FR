gpu_set_blendenable

Cette fonction peut être utilisée pour activer ou désactiver le mélange alpha. En gros, si cette fonction est réglée sur false, toutes les images dessinées seront opaques à 100 %, ce qui signifie que toutes les zones transparentes ou semi-transparentes du site sprite ou de l'arrière-plan seront visibles sous forme de couleur solide. seront visibles sous la forme d'une couleur unie. Il est conseillé de désactiver le mélange alpha dans la mesure du possible (en particulier lorsque vous développez pour des appareils mobiles), car cela augmente considérablement la vitesse d'affichage.

Syntaxe :

gpu_set_blendenable(enable);

ArgumentsType Description
enable Enable or disable alpha blending value (true ou false).

Les retours :

Exemple :

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

Le code ci-dessus désactive le mélange alpha pour dessiner un arrière-plan sprite, puis le réactive pour continuer à dessiner.
