ds_queue_read

Cette fonction vous permet de recréer une file d'attente DS sauvegardée (une file d'attente qui a été précédemment écrite sous forme de chaîne de caractères à l'aide de la commande ds_queue_write()). Vous devez d'abord créer une nouvelle file d'attente DS pour y lire la chaîne. Si la file d'attente DS existe déjà et que des informations y sont stockées, celles-ci seront effacées avant la lecture. Cette fonction est d'une importance vitale lors de la création de mécanismes de sauvegarde/chargement pour votre jeu.

Notez que si la chaîne DS spécifiée a été écrite par GameMaker : Studio 1.2.x runtime (ou plus ancien), vous devez spécifier l'argument facultatif "legacy", en lui attribuant la valeur true car le format de la chaîne a changé après cela.

Syntaxe :

ds_queue_read(id, str [, legacy]);

Arguments Type Description
id ID de la file d'attente DS The id of the data structure to read into.
str Chaîne de caractères The string to read from.
legacy Booléen OPTIONAL Peut être soit true ou false, soit être complètement omis.

Les retours :

N/A

Exemple :

queue = ds_queue_create();
ini_open("save.ini");
var str = ini_read_string("Queues", "0", "");
if (str != "")
{
    ds_queue_read(queue, str);
}
ini_close();

Le code ci-dessus crée une file d'attente et stocke l'index dans la variable "queue". Il ouvre ensuite un fichier ini et y lit une chaîne de caractères, en vérifiant d'abord que la chaîne n'est pas renvoyée comme vide. Cette chaîne est ensuite lue dans la file d'attente DS nouvellement créée.
