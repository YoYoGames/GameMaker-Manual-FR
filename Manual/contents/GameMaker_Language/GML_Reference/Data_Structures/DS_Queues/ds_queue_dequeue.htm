ds_queue_dequeue

Cette fonction déchargera la valeur de tête de la file d'attente DS, la supprimant de la file d'attente et renvoyant la valeur à stocker dans une variable. Si la file d'attente est vide, la fonction renvoie la constante undefined, sinon elle renvoie la valeur contenue dans la file d'attente.

Syntaxe :

ds_queue_dequeue(id);

ArgumentsType Description
id The id of the queue to dequeue from.

Les retours :

Variable (Data type value stored in the queue) or undefined

Exemple :

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

Le code ci-dessus vérifie la file d'attente DS indexée dans la variable "move_queue" pour voir si elle est vide, et si elle ne l'est pas, il met en file d'attente les deux valeurs de la tête de la file et les utilise pour définir une direction pour le mouvement.
