ds_queue_head

Cette fonction ne lira que la première valeur de la file (celle qui est "en tête"). Elle ne déchargera pas la valeur de la file, ce qui signifie qu'elle pourra toujours être lue dans le futur par cette fonction ou par la fonction ds_queue_dequeue(). Si la file d'attente est vide, la fonction renvoie la constante undefined, sinon elle renvoie la valeur réelle ou la chaîne de caractères contenue dans la file d'attente.

Syntaxe :

ds_queue_head(id);

ArgumentsType Description
id The id of the data structure to read from.

Les retours :

(Data type value stored in the queue)

Exemple :

num = ds_queue_head(control_queue);

Le code ci-dessus va lire la valeur de la file d'attente indexée dans la variable "control_queue" et stocker la valeur de retour dans la variable "num".
