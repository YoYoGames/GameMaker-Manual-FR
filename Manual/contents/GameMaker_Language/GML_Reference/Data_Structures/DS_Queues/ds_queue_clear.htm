ds_queue_clear

Avec cette fonction, vous pouvez effacer toutes les données de la structure de données de la file d'attente donnée. Cela ne détruit PAS la structure de données (pour cela, vous devez utiliser la fonction ds_queue_destroy()), elle efface seulement toutes les données de celle-ci et renvoie une file vide.

Syntaxe :

ds_queue_clear(id);

ArgumentsType Description
id The id of the data structure to clear.

Les retours :

Exemple :

if (count = 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la file d'attente DS indexée dans la variable "command_queue", déclenche une alarme et remet la variable à 0.

Retour : Files d'attente DS
