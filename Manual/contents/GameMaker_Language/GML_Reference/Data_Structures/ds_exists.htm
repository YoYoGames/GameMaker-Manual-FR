ds_exist

Avec cette fonction, vous pouvez vérifier si une structure de données du type donné existe. Vous fournissez la valeur "index" (telle qu'elle est contenue dans une variable) et le "type" de DS, qui peut être l'une des constantes énumérées ci-dessous, et la fonction renverra true si la structure de données existe et false sinon.

Type DS Constant
Constant Description
ds_type_map Une structure de données cartographiques
ds_type_list Une structure de données de type liste
ds_type_stack Une structure de données de pile
ds_type_grid Une structure de données en grille
ds_type_queue Une structure de données de file d'attente
ds_type_priority Une structure de données de priorité

NOTE Vous ne pouvez pas utiliser cette fonction pour vérifier le type d'une structure de données, car le même numéro d'index peut être utilisé par plusieurs structures de données de types différents.

Syntaxe :

ds_exists(ind, type);

Arguments Type Description
ind Tout ID de DS The variable index to check for the data structure
type Type DS Constant The type of data structure to check for (see the list of constants above)

Les retours :

Boolean

Exemple :

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

Le code ci-dessus vérifie la variable (précédemment initialisée) "ai_grid" pour voir si elle indexe une structure de données de type grille DS, et si ce n'est pas le cas, il en crée une et stocke son index dans la variable.
