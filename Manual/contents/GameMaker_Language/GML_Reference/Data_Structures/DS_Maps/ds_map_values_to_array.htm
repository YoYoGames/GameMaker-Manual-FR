ds_map_values_to_array

Cette fonction permet de récupérer toutes les valeurs contenues dans une carte DS pour chaque clé de la carte. Vous fournissez l'ID de la carte DS dont vous voulez récupérer les clés (tel que retourné par ds_map_create()) et la fonction renvoie un tableau dont chaque entrée correspond à une valeur de la carte DS (par exemple, si la carte comporte 10 clés, un tableau de 10 entrées sera renvoyé, chaque entrée correspondant à une valeur pour une clé). La fonction dispose d'un second argument facultatif où vous pouvez fournir un tableau que vous avez créé, auquel cas les données de la valeur de la clé de la carte seront ajoutées à toute donnée existante dans le tableau. Notez que la fonction modifiera directement le tableau fourni, mais renverra également une référence à celui-ci (ou une référence à un nouveau tableau si aucun n'est fourni).

Syntaxe :

ds_map_values_to_array(id, [array])

Arguments Type Description
id ID de la carte DS The id of the map to use.
[array] Array OPTIONAL Le tableau auquel ajouter les valeurs de la carte DS.

Les retours :

Array

Exemple :

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

L'exemple ci-dessus récupère toutes les valeurs d'une carte DS pour chaque clé de la carte et les stocke dans une variable locale sous forme de tableau. Elle boucle ensuite dans le tableau et ajoute chacune des valeurs à une variable locale qui est ensuite utilisée pour afficher le total de toutes les valeurs.
