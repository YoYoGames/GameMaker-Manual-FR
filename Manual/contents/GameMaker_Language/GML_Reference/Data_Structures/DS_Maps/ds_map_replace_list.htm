ds_map_replace_list

Avec cette fonction, vous pouvez remplacer une liste de DS qui a été stockée dans la "clé" donnée par une autre liste qui a été créée précédemment. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderez ensuite à l'aide de la fonction json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité.

Syntaxe :

ds_map_replace_list(id, key, value)

Arguments Type Description
id ID de la carte DS The id of the map to use.
key Chaîne de caractères The key to replace.
value ID de la liste DS The id of the ds_list to use to replace the one previously stored in the given key.

Les retours :

N/A

Exemple :

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

Le code ci-dessus va créer une DS List et la remplir avec les valeurs de diverses variables globales avant de remplacer une liste précédemment stockée dans la DS Map "j_map".
