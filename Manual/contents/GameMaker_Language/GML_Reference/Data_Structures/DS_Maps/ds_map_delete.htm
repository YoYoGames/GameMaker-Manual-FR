ds_map_delete

Cette fonction vous permet de supprimer une clé donnée (et sa valeur correspondante) de la carte DS donnée, précédemment créée.

Syntaxe :

ds_map_delete(id, key);

Arguments Type Description
id ID de la carte DS The id of the map to change.
key Chaîne de caractères The key to delete (along with its associated value).

Les retours :

N/A

Exemple :

ds_map_delete(inventory, "shield");

Le code ci-dessus supprimera la clé "shield" (et la valeur à laquelle elle est associée) de la carte DS (inventaire).
