ds_map_secure_load_buffer

Cette fonction chargera une carte DS sauvegardée sécurisée à partir d'un buffer. Vous devez avoir préalablement chargé le buffer en mémoire (à l'aide de la commande buffer_load() bufferNotez que le site buffer doit avoir été créé à l'aide de la fonction ds_map_secure_save_buffer() pour que cela fonctionne correctement, et notez également que si la carte DS chargée contient un tableau, celui-ci sera converti en une liste DS lors du chargement.

IMPORTANT ! L'une des caractéristiques d'un fichier sauvegardé sécurisé est qu'il est verrouillé sur l'appareil sur lequel il a été créé. Vous ne pouvez donc pas charger un fichier sauvegardé sur un appareil dans un projet exécuté sur un autre appareil.

REMARQUE: Cette fonction n'est pas prise en charge par HTML5 et UWP.

Syntaxe :

ds_map_secure_load_buffer(buffer);

Arguments Type Description
buffer ID de la mémoire tampon The buffer ID of the buffer to load the map data from.

Les retours :

DS Map ID

Exemple :

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

Le code ci-dessus chargera une carte DS sauvegardée en toute sécurité à partir d'un site buffer et stockera sa valeur d'index dans une variable pour une utilisation ultérieure.
