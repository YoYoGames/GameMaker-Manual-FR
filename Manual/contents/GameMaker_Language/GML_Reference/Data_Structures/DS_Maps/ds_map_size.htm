ds_map_size

Cette fonction vous permet de savoir combien de paires clé/valeur la carte DS (précédemment créée) contient.

Syntaxe :

ds_map_size(id);

ArgumentsType Description
id The id of the data structure to check.

Les retours :

Exemple :

if (ds_map_size(inventory) > 49)
{
    full = true;
}

Le code ci-dessus vérifie la taille de la carte DS (c'est-à-dire le nombre de paires clé/valeur) et si elle est supérieure à 49, il définit la variable "full" à true.
