ds_map_find_last

Cette fonction renvoie la dernière clé stockée dans la carte DS. Il ne s'agit pas de la dernière clé dans l'ordre où vous les avez ajoutées ! Les cartes DS ne sont pas stockées sous une forme linéaire, pour cela utilisez les listes DS, donc tout ce que fait cette fonction est de trouver la dernière clé telle que stockée par l'ordinateur. Cela peut être utile si vous devez itérer dans la carte DS à la recherche de quelque chose, mais il faut l'éviter si possible car cela peut être lent.

Notez que cette fonction renvoie undefined si la carte DS donnée est vide.

Syntaxe :

ds_map_find_last(id);

ArgumentsType Description
id The id of the map to use.

Les retours :

Variable or undefined

Exemple :

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

Le code ci-dessus crée quelques variables temporaires, puis obtient la taille de la carte DS et trouve la dernière clé stockée par l'ordinateur dans la carte. Il utilise ensuite une boucle for pour parcourir la carte DS à la recherche de la valeur clé "gold". Si elle la trouve, elle interrompt la boucle.
