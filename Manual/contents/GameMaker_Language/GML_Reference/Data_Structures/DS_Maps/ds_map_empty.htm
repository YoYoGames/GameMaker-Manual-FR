ds_map_empty

Cette fonction renvoie simplement false si la carte DS spécifiée (précédemment créée) contient des paires clé/valeur, ou true dans le cas contraire.

Syntaxe :

ds_map_empty(id);

ArgumentsType Description
id The id of the data structure to check.

Les retours :

Exemple :

if ds_map_empty(inventory)
{
    weight = 0;
}

Le code ci-dessus vérifie si la carte DS indexée dans la variable "inventory" contient des paires clé/valeur et si ce n'est pas le cas, il définit la variable "weight" à 0.
