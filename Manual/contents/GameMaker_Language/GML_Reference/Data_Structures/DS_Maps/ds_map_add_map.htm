ds_map_add_map

Cette fonction vous permet d'affecter une carte DS (précédemment créée) à une clé dans la carte DS donnée. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderez ensuite à l'aide de la fonction json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité. Si une carte DS a une autre carte ajoutée de cette manière, alors la destruction de la carte parente détruira également les cartes contenues et libérera leur mémoire. ds_map_clear() sur la carte parent détruira également et nettoiera les cartes marquées.

Syntaxe :

ds_map_add_map(id, key, value)

Arguments Type Description
id ID de la carte DS The id of the map to use.
key Chaîne de caractères The key for the added map.
value ID de la carte DS The id of the map to add.

Les retours :

N/A

Exemple :

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Le code ci-dessus crée deux maps DS, puis en remplit une avec une liste et un tableau avant de l'ajouter à la seconde, qui est ensuite encodée en une chaîne JSON. La carte est ensuite détruite pour la supprimer de la mémoire, ainsi que toutes les autres cartes ou listes qu'elle contient.
