ds_map_keys_to_array

Cette fonction vous permet de récupérer toutes les clés contenues dans une carte DS. Vous fournissez l'ID de la carte DS à partir de laquelle vous voulez récupérer les clés (comme retourné par ds_map_create()) et la fonction renvoie un tableau dont chaque entrée est une clé de la carte DS. La fonction a un second argument facultatif où vous pouvez fournir un tableau que vous avez créé, auquel cas les données de la clé de la carte seront ajoutées à toutes les données existantes dans le tableau. Notez que la fonction modifiera le tableau fourni directement, mais renverra également une référence à celui-ci (ou une référence à un nouveau tableau si aucun n'est fourni).

Syntaxe :

ds_map_keys_to_array(id, [array])

Arguments Type Description
id ID de la carte DS The id of the map to use.
[array] Array OPTIONAL Le tableau auquel ajouter les clés de la carte DS.

Les retours :

Array

Exemple :

map_keys = ds_map_keys_to_array(inventory);

Le code ci-dessus récupère les clés d'une carte DS et les stocke ensuite dans un tableau pour une utilisation ultérieure.
