ds_map_replace_map

Avec cette fonction, vous pouvez remplacer un DS Map qui a été stocké dans la "clé" donnée par une autre carte qui a été créée précédemment. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderez ensuite à l'aide de la fonction json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité.

Syntaxe :

ds_map_replace_map(id, key, value)

Arguments Type Description
id ID de la carte DS The id of the ds_map to use.
key Chaîne de caractères The key to replace.
value ID de la carte DS The id of the ds_map to use to replace the one previously stored in the given key.

Les retours :

N/A

Exemple :

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

Le code ci-dessus va créer un DS Map et le remplir d'un tableau et d'un DS List avant de remplacer une carte précédemment stockée dans le DS Map "j_map".
