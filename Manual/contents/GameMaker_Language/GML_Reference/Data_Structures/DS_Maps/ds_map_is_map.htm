ds_map_is_map

Avec cette fonction, vous pouvez vérifier si une carte DS est stockée dans la clé de carte donnée. Si la clé donnée contient un ID de carte DS, la fonction renvoie true, sinon elle renvoie false.

Notez que cela ne détectera que les cartes qui ont été ajoutées à l'aide de la fonction ds_map_add_map() .

Syntaxe :

ds_map_is_map(id, key)

ArgumentsType Description
id The id of the map to use.
key The key to replace.

Les retours :

Exemple :

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_map(inventory, key))
    {
        ds_map_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

Le code ci-dessus boucle sur une carte DS et vérifie si l'une de ses clés correspond à une autre carte DS. Si c'est le cas, la carte DS stockée est détruite, et à la fin de la boucle, la carte DS principale est également détruite.
