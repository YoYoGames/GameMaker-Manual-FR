ds_map_exists

Cette fonction renvoie true si la clé spécifiée existe dans la carte DS (précédemment créée), et false dans le cas contraire.

Syntaxe :

ds_map_exists(id, key);

Arguments Type Description
id ID de la carte DS The id of the data structure to check
key Chaîne de caractères The key to check for

Les retours :

Boolean

Exemple :

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

Le code ci-dessus vérifiera la carte DS indexée dans la variable "inventaire" pour la clé "potions" et si elle n'existe pas, il l'ajoutera à la carte.
