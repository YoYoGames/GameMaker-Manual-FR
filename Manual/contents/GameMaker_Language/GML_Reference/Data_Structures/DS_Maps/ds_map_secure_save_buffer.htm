ds_map_secure_save_buffer

Cette fonction permet de sauvegarder une carte DS précédemment créée sur un site buffer. Vous devez fournir la valeur d'ID de la carte DS (telle que retournée par la fonction ds_map_create()) et l'ID du site buffer dans lequel vous souhaitez écrire (tel que renvoyé par la fonction buffer_create()). Notez que si la carte DS à sauvegarder contient un tableau, celui-ci sera converti en une liste DS lors de la sauvegarde.

IMPORTANT ! Le fichier de carte DS sauvegardé en toute sécurité ne peut être chargé que sur l'appareil qui l'a créé. Si vous essayez de charger un fichier qui a été sauvegardé sur un autre appareil, il ne fonctionnera pas.

REMARQUE: Cette fonction n'est pas prise en charge par HTML5 et UWP.

Syntaxe :

ds_map_secure_save_buffer(filename);

Arguments Type Description
id ID de la carte DS The DS map ID value.
buffer ID de la mémoire tampon The buffer to save to.

Les retours :

N/A

Exemple :

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

Le code ci-dessus va créer une buffer et une carte DS, puis remplir la carte avec certaines valeurs et les écrire dans la buffer avant de supprimer la carte.
