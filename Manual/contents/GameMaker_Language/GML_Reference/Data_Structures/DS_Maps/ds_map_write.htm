ds_map_write

Cette fonction transformera les données de la carte DS de la carte spécifiée en une chaîne de caractères qui peut ensuite être écrite dans un fichier *.ini ou *.txt pour un stockage facile. Cette chaîne peut ensuite être relue dans un nouveau ds_map en utilisant la commande ds_map_read().

REMARQUE: La chaîne renvoyée n'est pas une chaîne lisible par l'homme, mais plutôt un vidage du contenu de la structure de données.

Syntaxe :

ds_map_write(id);

ArgumentsType Description
id The id of the data structure to use

Les retours :

Exemple :

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Le code ci-dessus ouvre un fichier ini prêt à être écrit. Il utilise ensuite ds_map_write() pour générer une chaîne de caractères qui est stockée dans la variable temporaire "t_string". Enfin, il écrit cette chaîne dans le fichier ini avant de le fermer.
