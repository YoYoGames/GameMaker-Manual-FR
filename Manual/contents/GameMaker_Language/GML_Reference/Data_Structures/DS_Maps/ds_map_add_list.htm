ds_map_add_list

Cette fonction permet d'attribuer une liste de DS (préalablement créée) à une clé dans la carte de DS donnée. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderez ensuite à l'aide de la fonction json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité. Si une liste a été ajoutée à une carte DS de cette manière, la destruction de la carte parente détruira également les listes contenues et libérera leur mémoire. ds_map_clear() sur la carte parent détruira et nettoiera également toutes les listes marquées.

Syntaxe :

ds_map_add_list(id, key, value)

Arguments Type Description
id ID de la carte DS The id of the map to use.
key Chaîne de caractères The key for the added list.
value ID de la liste DS The id of the list to add.

Les retours :

N/A

Exemple :

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Le code ci-dessus va créer une liste et la remplir avec les différentes valeurs des variables globales. Cette liste est ensuite "imbriquée" dans une carte DS, et la carte est ensuite encodée en une chaîne JSON, avant que la carte ne soit détruite, la supprimant de la mémoire, ainsi que toutes les listes qu'elle contient.
