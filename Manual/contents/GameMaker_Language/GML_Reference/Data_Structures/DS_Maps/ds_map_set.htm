ds_map_set

Cette fonction vous permet de définir la valeur d'une clé dans une carte DS donnée. Vous fournissez la valeur de l'ID de la carte DS (telle que retournée par la fonction ds_map_create()), puis vous indiquez la clé que vous souhaitez définir et la valeur à laquelle elle doit être définie. Les clés peuvent être des entiers ou des chaînes de caractères, et si la clé donnée n'existe pas, elle sera créée pour vous et définie par la valeur. Cette fonction est identique à l'utilisation de l'accesseur DS map pour définir/créer une paire clé/valeur. La fonction ne renvoie rien, donc si vous devez vérifier si la valeur de la clé a été remplacée ou si une nouvelle clé a été créée, vous devez utiliser la fonction ds_map_replace().

Syntaxe :

ds_map_set(id, key, value)

ArgumentsType Description
id The id of the map to use.
key The key to set.
value The value to set the key to.

Les retours :

Exemple :

if is_undefined(ds_map_find_value(map, "score"))
{
    ds_map_set(map, "score", 0);
}

Le code ci-dessus vérifiera si la clé donnée existe et si elle n'existe pas, elle sera créée et définie.
