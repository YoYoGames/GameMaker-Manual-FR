ds_map_read

Cette fonction va prendre une chaîne de caractères qui a été précédemment créée par la fonction ds_map_write() et la lira dans une carte DS précédemment créée. Si la carte dans laquelle la chaîne est lue contient des paires clé/valeur, celles-ci seront effacées avant que la carte sauvegardée ne soit reconstruite.

Notez que si la chaîne DS spécifiée a été écrite par GameMaker : Studio 1.2.x runtime (ou plus ancien), vous devez spécifier l'argument facultatif "legacy", en lui attribuant la valeur true car le format de la chaîne a changé après cela.

Syntaxe :

ds_map_read(id, str, [legacy]);

Arguments Type Description
id ID de la carte DS The id of the data structure to read the string into
str Chaîne de caractères The string to read
legacy Booléen OPTIONAL Peut être soit true ou false, soit être complètement omis.

Les retours :

N/A

Exemple :

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

Le code ci-dessus crée une nouvelle carte DS et stocke son index id dans la variable "inventory". Il ouvre ensuite un fichier ini et lit une chaîne de ce fichier dans la variable temporaire "t_string". Enfin, il vérifie que la chaîne de caractères est valide (pas la valeur par défaut de "") et si c'est le cas, il lit la chaîne dans la carte DS nouvellement créée avant de refermer le fichier ini.
