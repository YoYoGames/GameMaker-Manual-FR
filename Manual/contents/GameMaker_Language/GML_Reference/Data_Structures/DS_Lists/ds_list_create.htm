ds_list_create

Cette fonction crée une nouvelle structure de données de liste et renvoie la valeur de l'index. Cette valeur doit être stockée dans une variable et utilisée dans tous les autres appels de fonction relatifs à la liste. Notez que si vous essayez d'entrer une valeur dans la liste à l'adresse access avant qu' elle n' ait été remplie - c'est-à-dire si la liste est "vide" - la valeur de retour peut être 0 car, en interne, les quelques premières entrées de la liste sont définies lors de sa création afin de minimiser les problèmes de performance lors de l'ajout initial d'éléments à la liste (bien que la fonction ds_list_size() retournera toujours 0 sur une liste nouvellement créée). Si vous souhaitez vous assurer que la liste est "réellement" vide lors de sa création, vous devez appeler la fonction ds_list_clear() après la création de la liste, ce qui signifie que toute valeur renvoyée pour les emplacements de liste non remplis sera undefined.

IMPORTANT ! Lorsque vous créez une structure de données, la valeur de l'index pour l'identifier est une valeur entière commençant à 0. Cela signifie que des structures de données de différents types peuvent avoir la même valeur d'index, donc en cas de doute vous devriez utiliser la fonction ds_exists() avant d'y accéder. Notez également que les index sont réutilisés, de sorte qu'une valeur d'index de structure de données détruite peut être utilisée par une structure nouvellement créée par la suite.

Syntaxe :

ds_list_create();

Les retours :

DS List ID

Exemple :

list = ds_list_create();

Cela créera une nouvelle liste et assignera son id d'index à la variable d'instance "list".
