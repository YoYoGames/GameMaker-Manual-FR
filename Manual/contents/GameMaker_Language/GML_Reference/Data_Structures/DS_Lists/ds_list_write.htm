ds_list_write

Cette fonction renvoie une chaîne de caractères qui peut ensuite être stockée ou transférée dans une autre structure de données à l'aide de la fonction ds_list_read() fonction.

REMARQUE: La chaîne renvoyée n'est pas une chaîne lisible par l'homme, mais plutôt un vidage du contenu de la structure de données.

Syntaxe :

ds_list_write(id);

ArgumentsType Description
id The id of the data structure to write.

Les retours :

Exemple :

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

Le code ci-dessus ouvre un fichier ini et écrit ensuite une chaîne de caractères contenant les informations stockées dans la liste DS indexée dans la variable "list". La liste est ensuite effacée et le fichier ini fermé.
