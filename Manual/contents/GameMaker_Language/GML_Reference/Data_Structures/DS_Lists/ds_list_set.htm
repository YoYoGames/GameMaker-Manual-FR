ds_list_set

Cette fonction peut être utilisée pour définir une entrée de liste précédemment ajoutée. Vous donnez l'ID de la liste (tel que retourné lorsque vous avez créé la liste) et la position dans la liste à définir ainsi que la valeur à définir. Notez que si l'entrée à définir est en dehors des limites de la liste (par exemple, vous définissez l'entrée 20 de la liste mais la liste actuelle ne contient que 10 entrées), la liste sera remplie jusqu'à la position donnée et chaque entrée sera définie à 0. Cette fonction est identique à l'utilisation de l'accesseur DS list.

Syntaxe :

ds_list_set(id, pos, val);

Arguments Type Description
id ID de la liste DS The id of the list to add to.
pos Variable The position within the list to set.
[val2, ... max_val] Variable The value to set in the list.

Les retours :

N/A

Exemple :

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

Le code ci-dessus va ajouter la valeur stockée dans la variable "score" dans la liste indexée dans la variable "sc_list".
