ds_list_sort

Cette fonction vous permet de trier toutes les valeurs d'une liste, dans l'ordre croissant ou décroissant. Si la liste contient des chaînes de caractères, celles-ci seront triées par ordre alphabétique, sur la base de l'alphabet anglais de 26 lettres.

Syntaxe :

ds_list_sort(id, ascend);

ArgumentsType Description
id The id of the list to sort.
ascend Whether the values should be ascending (true) ou par ordre décroissant (false).

Les retours :

Exemple :

if newgame
{
    ds_list_sort(name_list, true);
}

Le code ci-dessus triera la liste indexée dans la variable "name_list" si la variable "newgame" est signalée comme true.
