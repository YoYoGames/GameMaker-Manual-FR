ds_list_clear

Cette fonction vous permet d'effacer toutes les données de la structure de données d'une liste donnée. Cela ne détruit PAS la structure de données (pour cela, vous devez utiliser la fonction ds_list_destroy()), elle efface seulement toutes les données et rend la liste vide (de taille nulle). Notez que l'effacement d'une liste déréférencera toutes les structures de données qui y sont stockées, ce qui entraînera une fuite de mémoire. Vous devrez donc parcourir la liste et détruire manuellement tous les éléments de structure de données avant de l'effacer pour éviter cela. Le seul cas où cela n'est pas nécessaire est celui où vous avez marqué des éléments de la liste comme étant une autre liste DS ou une carte DS, auquel cas ces éléments seront détruits (et non effacés !) et leur mémoire nettoyée automatiquement lorsque le parent est effacé.

Syntaxe :

ds_list_clear(id);

ArgumentsType Description
id The id of the data structure to clear.

Les retours :

Exemple :

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la liste DS indexée dans la variable "command_list", déclenche une alarme et remet la variable à 0.
