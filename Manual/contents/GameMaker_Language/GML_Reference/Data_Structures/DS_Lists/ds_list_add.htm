ds_list_add

Cette fonction peut être utilisée pour ajouter une nouvelle valeur (réel ou chaîne) à la liste, qui sera ajoutée à la fin. La fonction peut prendre d'autres arguments facultatifs (autant que vous le souhaitez), ce qui vous permet d'ajouter plusieurs valeurs consécutives à la liste en un seul appel.

Syntaxe :

ds_list_add(id, val1 [, val2, ... max_val]);

Arguments Type Description
id ID de la liste DS The id of the list to add to.
val Variable The value to add to the list.
[val2, ... max_val] Variable Optional values to be added to the list.

Les retours :

N/A

Exemple :

ds_list_add(sc_list, score);

Le code ci-dessus va ajouter la valeur stockée dans la variable "score" dans la liste indexée dans la variable "sc_list".
