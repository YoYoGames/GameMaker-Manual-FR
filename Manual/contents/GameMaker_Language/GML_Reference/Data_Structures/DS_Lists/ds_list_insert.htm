ds_list_insert

Cette fonction ajoutera la valeur donnée dans la liste à la position donnée. Si la liste contient d'autres valeurs après la position donnée, leur position sera déplacée d'une unité pour faire de la place, ce qui augmentera la liste d'une unité.

Syntaxe :

ds_list_insert(id, pos, val);

Arguments Type Description
id ID de la liste DS The id of the list to add to.
pos Real The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Variable The value to add to the list.

Les retours :

N/A

Exemple :

ds_list_insert(list, 9, score);

Le code ci-dessus ajoutera la valeur stockée dans la variable "score" à la 10ème position de la liste indexée par la variable "list" (les listes sont comptées à partir de 0, donc une liste de dix valeurs est numérotée de 0 à 9).
