ds_list_find_index

Avec cette fonction, vous pouvez rechercher une valeur dans la liste donnée et la position de cette valeur dans la liste sera retournée. Notez que s'il y a plus d'une entrée dans la liste avec la même valeur, la position de n'importe laquelle d'entre elles peut être retournée, et que si la valeur n'existe pas, alors -1 sera retourné.

Syntaxe :

ds_list_find_index(id, val);

Arguments Type Description
id ID de la liste DS The id of the list to use.
val Variable The value to find.

Les retours :

Real

Exemple :

pos = ds_list_find_index(list, "Player1");

Le code ci-dessus vérifie la liste indexée dans la variable "list" pour la valeur "Player1" et stocke la position retournée dans la variable "pos".
