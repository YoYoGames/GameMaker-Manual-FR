ds_list_shuffle

Cette fonction vous permet de mélanger une liste, ce qui réorganisera toutes les valeurs des composants dans des positions aléatoires par rapport à celles dans lesquelles ils ont été ajoutés à l'origine à la liste.

REMARQUE: cette fonction mélangera les éléments de la liste aux mêmes positions chaque fois que le jeu est relancé, car GameMaker génère à chaque fois la même graine aléatoire initiale pour faciliter le code de debugging. Pour éviter ce comportement, utilisez randomise() au début de votre jeu. Cela n'est vrai que lorsque vous testez le jeu et que vous l'envoyez à l'adresse debugging, car l'exécutable final ne présentera pas ce comportement et sera aléatoire à chaque partie.

Syntaxe :

ds_list_shuffle(id);

ArgumentsType Description
id The id of the list to shuffle.

Les retours :

Exemple :

if restart
{
    ds_list_shuffle(card_list);
}

Le code ci-dessus mélangera la liste indexée dans la variable "card_list" si la variable "restart" est signalée comme true.
