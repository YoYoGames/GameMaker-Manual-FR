ds_list_is_list

Cette fonction permet de vérifier si une autre liste de DS est stockée à la position donnée dans une liste de DS. Si la position donnée contient un ID de liste DS, la fonction renvoie true; sinon, elle renvoie false.

Notez que cela ne détectera que les listes qui ont été marquées manuellement à l'aide de la fonction ds_list_mark_as_list() .

Syntaxe :

ds_list_is_list(id, pos);

ArgumentsType Description
id The id of the list to check.
pos The position within the list to check.

Les retours :

Exemple :

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_list(ships, i))
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Le code ci-dessus boucle sur une liste DS et vérifie si l'une des entrées contient d'autres ID de liste. Si c'est le cas, ces listes sont détruites, puis la liste principale est détruite à la fin de la boucle.
