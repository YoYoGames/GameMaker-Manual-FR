ds_priority_add

Cette fonction permet d'ajouter une valeur (soit un nombre réel, soit une chaîne de caractères) à une file d'attente prioritaire, tout en lui attribuant une valeur de priorité.

Syntaxe :

ds_priority_add(id, val, priority);

Arguments Type Description
id ID de priorité DS The id of the priority queue to add to.
val Variable The value to add to the priority queue.
priority Real The priority of the value to add.

Les retours :

N/A

Exemple :

ds_priority_add(ai_priority, AI_Search, 5);

Le code ci-dessus ajoute une fonction de script à la file d'attente prioritaire indexée dans la variable "ai_priority" et lui attribue une priorité de 5.
