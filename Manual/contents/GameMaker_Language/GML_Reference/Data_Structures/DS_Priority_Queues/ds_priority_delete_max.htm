ds_priority_delete_max

Cette fonction renvoie la valeur qui a la plus haute priorité dans la file d'attente, puis supprime la valeur (et la priorité) de la structure de données. Si plusieurs valeurs ont la même priorité, n'importe laquelle d'entre elles peut être renvoyée dans n'importe quel ordre, mais toutes les autres valeurs ayant la même priorité seront toujours dans la file d'attente.

Syntaxe :

ds_priority_delete_max(id);

ArgumentsType Description
id The id of the priority queue to use.

Les retours :

(Data type stored in the priority)

Exemple :

if ai_move
{
    script_execute(ds_priority_delete_max(ai_priority));
}

Le code ci-dessus vérifie une variable d'instance et s'il renvoie true, il exécutera un script indexé dans la file d'attente des priorités avec la valeur de priorité la plus élevée, puis supprimera ce script de la file d'attente.
