ds_priority_copy

Cette fonction peut être utilisée pour copier le contenu d'une file prioritaire dans une autre. Notez que cela ne supprime PAS le contenu de la file prioritaire d'origine, ni ne détruit la file prioritaire d'origine. Lors de l'utilisation de cette fonction, la file d'attente prioritaire vers laquelle la copie est effectuée doit avoir été créée précédemment et si elle contenait des éléments avant la copie, ceux-ci seront effacés en premier (ce qui signifie que ces informations seront perdues).

Syntaxe :

ds_priority_copy(id, source);

ArgumentsType Description
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

Les retours :

Exemple :

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
p_queue = ds_priority_create();
ds_priority_copy(p_queue, other.p_queue);
}

La fonction ci-dessus crée une nouvelle instance, puis dans cette instance, elle crée une nouvelle file d'attente prioritaire DS et copie le contenu de la file d'attente prioritaire de l'instance qui exécute le bloc de code, dans la file d'attente prioritaire de l'instance nouvellement créée.
