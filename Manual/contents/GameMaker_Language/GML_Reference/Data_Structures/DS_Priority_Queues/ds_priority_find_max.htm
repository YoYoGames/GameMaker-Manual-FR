ds_priority_find_max

Cette fonction permet de trouver la valeur stockée dans la file d'attente des priorités avec la priorité la plus élevée. Si plusieurs valeurs ont la même priorité, elles peuvent être renvoyées dans n'importe quel ordre. Cependant, contrairement à ds_priority_delete_max()cette fonction ne supprime pas la valeur de la file d'attente.

Syntaxe :

ds_priority_find_max(id);

ArgumentsType Description
id The id of the priority queue to use.

Les retours :

Real or String

Exemple :

if ai_move
{
    script_execute(ds_priority_find_max(ai_priority));
}

Le code ci-dessus vérifie une variable d'instance et s'il renvoie true, il exécutera une fonction de script indexée dans la file d'attente des priorités avec la valeur de priorité la plus élevée.
