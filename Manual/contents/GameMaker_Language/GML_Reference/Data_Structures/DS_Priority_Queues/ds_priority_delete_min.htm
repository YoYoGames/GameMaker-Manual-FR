ds_priority_delete_min

Cette fonction renvoie la valeur qui a la priorité la plus basse dans la file d'attente, puis supprime la valeur (et la priorité) de la structure de données. Si plusieurs valeurs ont la même priorité, n'importe laquelle d'entre elles peut être renvoyée dans n'importe quel ordre, mais toutes les autres valeurs ayant la même priorité seront toujours dans la file d'attente.

Syntaxe :

ds_priority_delete_min(id);

ArgumentsType Description
id The id of the priority queue to use.

Les retours :

(Data type stored in the priority)

Exemple :

if ai_move
{
    script_execute(ds_priority_delete_min(ai_priority));
}

Le code ci-dessus vérifie une variable d'instance et s'il renvoie true, il exécute une fonction de script indexée dans la file d'attente des priorités avec la valeur de priorité la plus basse, puis supprime ce script de la file d'attente.
