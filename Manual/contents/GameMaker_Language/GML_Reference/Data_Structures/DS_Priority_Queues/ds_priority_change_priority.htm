ds_priority_change_priority

Cette fonction prend une valeur donnée et change sa priorité dans la file d'attente des priorités référencée. La valeur donnée doit déjà exister dans la file d'attente des priorités.

Syntaxe :

ds_priority_change_priority(id, val, priority);

Arguments Type Description
id ID de priorité DS The id of the priority queue to change.
val Variable The value to change the priority of.
priority Real The new priority of the value.

Les retours :

N/A

Exemple :

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

Le code ci-dessus vérifie une variable globale et si elle est inférieure à une certaine valeur, il modifie la priorité de l'index de la fonction de script dans la file d'attente des priorités.
