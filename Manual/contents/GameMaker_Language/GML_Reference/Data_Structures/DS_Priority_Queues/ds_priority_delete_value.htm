ds_priority_delete_value

Cette fonction supprime simplement la valeur donnée, ainsi que sa priorité, de la file d'attente indexée des priorités.

Syntaxe :

ds_priority_delete_value(id,val);

ArgumentsType Description
id The id of the priority queue to use.
val The value to delete from the priority queue.

Les retours :

Exemple :

if (ai_move == false)
{
    ds_priority_delete_value(ai_priority, AI_Move);
}

Le code ci-dessus vérifie une variable d'instance et s'il renvoie false, il retire la fonction de script indexée de la file d'attente des priorités.
