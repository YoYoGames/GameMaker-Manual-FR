ds_grid_write

Cette fonction peut être utilisée pour convertir la ds_grid donnée en une chaîne de caractères, qui peut ensuite être stockée dans un fichier externe (par exemple). Vous pouvez relire la chaîne retournée par cette fonction dans une ds_grid en utilisant la fonction ds_grid_read().

NOTE La chaîne retournée n'est pas une chaîne lisible par l'homme, mais plutôt un vidage du contenu de la structure de données.

Syntaxe :

ds_grid_write(index);

ArgumentsType Description
index The index of the grid to write.

Les retours :

Exemple :

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

Le code ci-dessus ouvrira un fichier ini (en le créant s'il n'existe pas déjà) et écrira la ds_grid donnée sous forme de chaîne dans ce fichier.
