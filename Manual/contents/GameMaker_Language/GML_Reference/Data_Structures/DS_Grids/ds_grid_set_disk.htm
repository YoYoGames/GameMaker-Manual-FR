ds_grid_set_disk

Avec cette fonction, vous pouvez définir une région circulaire d'une grille à une certaine valeur. Vous devez fournir une cellule de grille de départ (en tant que coordonnée d'axe x et y) ainsi que le rayon du disque à définir et la valeur que vous souhaitez donner au disque, comme le montre l'illustration suivante comme le montre l'illustration ci-dessous :

Syntaxe :

ds_grid_set_disk(index, xm, ym, r, val);

ArgumentsType Description
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to set with the cells within the disk.

Les retours :

Exemple :

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

Le code ci-dessus va définir une région circulaire avec un rayon de 5 cellules dans la grille DS indexée dans la variable "grid" à une valeur de -4.
