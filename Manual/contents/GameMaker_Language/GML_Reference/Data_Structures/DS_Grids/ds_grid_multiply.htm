ds_grid_multiply

Cette fonction multiplie la valeur de la cellule de grille donnée par le montant spécifié.

NOTE Cette fonction ne fonctionne qu'avec des nombres réels, pas avec des chaînes de caractères.

Syntaxe :

ds_grid_multiply(index, x, y, val);

Arguments Type Description
index ID de la grille DS The index of the grid.
x Real The x position of the cell in the grid.
y Real The y position of the cell in the grid.
val Real The value to multiply with the cell.

Les retours :

N/A

Exemple :

ds_grid_multiply(mygrid, 5, 5, 2)

Le code ci-dessus va multiplier par 2 la valeur stockée dans la cellule de la grille DS.

Retour : Grilles DS
