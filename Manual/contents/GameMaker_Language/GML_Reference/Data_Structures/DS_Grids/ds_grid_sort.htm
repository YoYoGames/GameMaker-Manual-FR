ds_grid_sort

Cette fonction peut être utilisée pour trier une grille DS sur la base des valeurs d'une colonne donnée (de la même manière que vous trieriez des fichiers par date, taille etc... dans l'explorateur de fichiers du système d'exploitation). L'image suivante montre un exemple :

Syntaxe :

ds_grid_sort(index, column, ascending);

ArgumentsType Description
index The index of the grid to sort.
column The column to use for sorting the rows
ascending Whether to sort lowest to highest (true), ou du plus haut au plus bas (false).

Les retours :

Exemple :

ds_grid_sort(grid, 3, false)

Cela prendrait toutes les valeurs de la grille DS indexées dans la variable "grid" et les trierait en fonction des valeurs trouvées dans la 3ème colonne de la grille (comme indiqué dans l'image ci-dessus).
