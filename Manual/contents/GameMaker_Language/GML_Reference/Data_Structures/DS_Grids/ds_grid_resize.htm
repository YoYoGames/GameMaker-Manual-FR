ds_grid_resize

Avec cette fonction, vous pouvez redimensionner la grille DS donnée pour qu'elle ait une largeur et/ou une hauteur différente. Si la taille de la grille est supérieure à la grille actuelle, les nouvelles cellules auront une valeur de base de 0, et si la taille est inférieure, les valeurs contenues dans les cellules qui ne sont plus dans la nouvelle taille seront perdues. qui ne sont plus dans la nouvelle taille seront perdues. Toutes les autres cellules ne seront pas modifiées.

Syntaxe :

ds_grid_resize(index, w, h);

ArgumentsType Description
index This index of the grid to resize.
w The new width of the grid.
h The new height of the grid.

Les retours :

Exemple :

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Le code ci-dessus va redimensionner la grille DS indexée dans la variable globale "Grid" puis la vider afin que chaque cellule prenne la valeur -1.
