ds_grid_shuffle

Cette fonction peut être utilisée pour rendre aléatoire la position de toutes les valeurs dans toutes les cellules d'une grille.

NOTE Cette fonction mélangera les cellules de la grille aux mêmes positions chaque fois que le jeu est relancé, car GameMaker génère à chaque fois la même graine aléatoire initiale afin de rendre le code debugging beaucoup plus facile. Pour éviter ce ce comportement, utilisez randomise() au début de votre jeu. Ceci n'est vrai que lorsque vous testez le jeu et que vous l'envoyez à l'adresse debugging, car l'exécutable final ne présentera pas ce comportement. final ne présentera pas ce comportement et sera aléatoire à chaque jeu.

Syntaxe :

ds_grid_shuffle(index);

ArgumentsType Description
index The index of the grid to shuffle.

Les retours :

Exemple :

ds_grid_shuffle(grid)

Cela prendrait toutes les valeurs de la grille DS indexées dans la variable "grid" et les mélangerait pour obtenir un ordre différent.
