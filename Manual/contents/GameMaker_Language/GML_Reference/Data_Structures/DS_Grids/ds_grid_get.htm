ds_grid_get

Cette fonction peut être utilisée pour obtenir la valeur de n'importe quelle cellule de la grille DS donnée. Si vous passez des coordonnées de grille invalides à la fonction, la valeur renvoyée sera undefined et une erreur sera affichée dans la fenêtre de sortie.

Syntaxe :

ds_grid_get(index, x, y);

ArgumentsType Description
index The index of the grid.
x The x position of the cell you want to find the value of.
y The y position of the cell you want to find the value of.

Les retours :

Variable

Exemple :

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

Le code ci-dessus sélectionne une cellule aléatoire dans la grille DS indexée dans la variable "grid" et stocke sa valeur dans la variable "val".
