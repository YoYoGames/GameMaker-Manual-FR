ds_grille_valeur_disque_y

Cette fonction permet d'obtenir la coordonnée y (dans la région en forme de disque de la grille) de la valeur recherchée. Vous donnez l'index de la grille DS (tel que renvoyé par ds_grid_create()) ainsi que les positions x/y de la cellule centrale du disque. Ensuite, vous indiquez le rayon (sous forme de valeur entière) autour de la cellule centrale à rechercher, avant de fournir la valeur à rechercher. Si la valeur recherchée n'existe pas, la fonction renvoie -1.

Syntaxe :

ds_grid_value_disk_y(index, xm, ym, r, val);

ArgumentsType Description
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to find.

Les retours :

Exemple :

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
    xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
    ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

Le code ci-dessus vérifie une grille DS pour une valeur spécifique dans une région du disque. Si elle est trouvée, il stocke alors la position x et y de la valeur dans deux variables pour une utilisation ultérieure.
