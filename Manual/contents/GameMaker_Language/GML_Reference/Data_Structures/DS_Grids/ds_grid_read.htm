ds_grid_read

Cette fonction peut être utilisée pour convertir une chaîne de caractères créée précédemment par la fonction ds_grid_write() en une grille DS. La grille DS doit avoir été créée précédemment (voir l'exemple ci-dessous).

Notez que si la chaîne DS spécifiée a été écrite par GameMaker : Studio 1.2.x runtime (ou plus ancien), vous devez spécifier l'argument facultatif "legacy", en lui attribuant la valeur true car le format de la chaîne a changé après cela.

Syntaxe :

ds_grid_read(index, string [, legacy]);

Arguments Type Description
index ID de la grille DS The index of the grid to read.
string Chaîne de caractères The string to read into the DS grid.
legacy Booléen OPTIONAL Peut être soit true ou false, soit être complètement omis.

Les retours :

N/A

Exemple :

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

Le code ci-dessus crée une grille DS basée sur la taille de la pièce (chaque carré de pixels 32x32 représente une cellule de la grille) et lit ensuite un ensemble de données de grille précédemment sauvegardées dans un fichier ini dans la nouvelle grille DS.
