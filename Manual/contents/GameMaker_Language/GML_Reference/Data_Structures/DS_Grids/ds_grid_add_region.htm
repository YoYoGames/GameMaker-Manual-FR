ds_grid_add_region

Cette fonction peut être utilisée pour ajouter une valeur donnée (réelle ou chaîne de caractères) à toutes les valeurs des cellules situées dans la zone définie d'une grille. La valeur à ajouter doit être du même type que celle contenue dans les cellules de la grille, c'est-à-dire que vous ne pouvez pas ajouter une chaîne de caractères à un réel ou vice-versa. c'est-à-dire que vous ne pouvez pas ajouter une chaîne de caractères à un réel ou vice-versa, et pour les chaînes de caractères, cela correspond à une concaténation.

Syntaxe :

ds_grid_add_region(index, x1, y1, x2, y2, val);

ArgumentsType Description
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to add to the region cells.

Les retours :

Exemple :

ds_grid_add_region(grid, 2, 4, 5, 5, ".")

Cela ajouterait "." à toutes les chaînes contenues dans les cellules de la région définie de la grille DS référencée par la variable "grid".
