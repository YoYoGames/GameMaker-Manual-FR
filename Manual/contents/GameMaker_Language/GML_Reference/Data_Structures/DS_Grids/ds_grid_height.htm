ds_grid_height

Cette fonction renvoie la hauteur de la grille donnée. Cette valeur est le nombre de cellules de la grille le long de l'axe des y et est toujours un nombre entier, comme le montre l'image ci-dessous :

Syntaxe :

ds_grid_height(index);

ArgumentsType Description
index This index of the grid to find the height of.

Les retours :

Exemple :

for (var i = 0; i < ds_grid_width(grid); ++i)
{
    for (var j = 0; j < ds_grid_height(grid); ++j)
    {
        if (ds_grid_get(grid, i, j) == 1)
        {
            instance_create_Layer(i * 32, j * 32, "Walls", obj_Wall);
        }
    }
}

Le code ci-dessus va parcourir la grille DS indexée dans la variable "grid" et si la valeur trouvée dans une cellule spécifique est égale à 1, il va alors créer une instance de "obj_Wall" à la position appropriée dans la pièce.
