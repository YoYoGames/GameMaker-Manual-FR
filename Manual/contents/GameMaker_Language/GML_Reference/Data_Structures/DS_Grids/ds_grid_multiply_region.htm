ds_grid_multiply_region

Cette fonction permet de spécifier une région de la grille dans laquelle la valeur de chaque cellule sera multipliée par une quantité donnée.

NOTE Cette fonction ne fonctionne qu'avec des nombres réels, pas avec des chaînes de caractères.

Syntaxe :

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

Arguments Type Description
index ID de la grille DS The index of the grid.
x1 Real The x position of the left of the region in the grid.
y1 Real The y position of the top of the region in the grid.
x2 Real The x position of the right of the region in the grid.
y2 Real The y position of the bottom of the region in the grid.
val Real The value to multiply with the region cells.

Les retours :

N/A

Exemple :

ds_grid_multiply_region(mygrid, 5, 5, 10, 10, 2);

Le code ci-dessus prend toutes les valeurs trouvées dans la zone de grille rectangulaire définie et les multiplie par 2.
