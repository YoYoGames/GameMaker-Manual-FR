ds_grid_clear

Cette fonction peut être utilisée pour effacer une grille DS donnée à une valeur spécifique. Toutes les cellules de la grille contiendront alors cette valeur, qui peut être un nombre réel ou une chaîne de caractères. L'image ci-dessous illustre le fonctionnement de cette fonction :

Syntaxe :

ds_grid_clear(index, val);

ArgumentsType Description
index This index of the grid to clear.
val The new value for all grid cells.

Les retours :

Exemple :

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Le code ci-dessus va redimensionner la grille DS indexée dans la variable globale "Grid" puis la vider afin que chaque cellule prenne la valeur -1.
