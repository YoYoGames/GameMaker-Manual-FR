ds_stack_top

Cette fonction ne lira que la première valeur de la pile (celle qui est "en haut"). Elle ne lèvera pas la valeur, ce qui signifie qu'elle pourra toujours être lue dans le futur par cette fonction ou par la fonction ds_stack_pop(). Si la pile est vide, la fonction renvoie la constante undefined, sinon elle renvoie la valeur réelle ou la chaîne de caractères contenue dans la pile.

Syntaxe :

ds_stack_top(id);

ArgumentsType Description
id The id of the data structure to read from.

Les retours :

Variable (Data type value that is stored in the stack) or undefined

Exemple :

num = ds_stack_top(control_stack);

Le code ci-dessus va lire la valeur de la pile indexée dans la variable "control_stack" et stocker la valeur de retour dans la variable "num".
