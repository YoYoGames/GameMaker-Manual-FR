ds_stack_size

Cette fonction renvoie la "taille" de la pile, c'est-à-dire le nombre d'éléments qui y ont été poussés.

Syntaxe :

ds_stack_size(id);

ArgumentsType Description
id The id of the data structure to check.

Les retours :

Exemple :

if !ds_stack_empty(control_stack)
{
    num = ds_stack_size(control_stack);
}

Le code ci-dessus vérifie une pile DS pour voir si elle est vide ou non, et si elle ne l'est pas, il obtient le nombre d'éléments qu'elle contient et stocke la valeur dans une variable.
