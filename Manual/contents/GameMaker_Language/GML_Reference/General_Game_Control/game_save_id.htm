game_save_id

Cette variable en lecture seule renvoie l'ID du chemin complet du répertoire utilisé par votre jeu pour sauvegarder les fichiers. Ce répertoire peut être visible ou non par les autres applications, selon la plate-forme.

Sur la cible HTML5, il retournera une chaîne vide.

NOTE Cette variable existe uniquement à titre d'information et ne doit pas être utilisée pour obtenir un chemin d'accès à un fichier ou quelque chose de similaire, car elle ne fonctionnera pas correctement sur toutes les plateformes.

Syntaxe :

game_save_id

Les retours :

String

Exemple :

save_dir = game_save_id;

Cela permettra de stocker le répertoire de sauvegarde des fichiers dans une variable pour une utilisation ultérieure.
