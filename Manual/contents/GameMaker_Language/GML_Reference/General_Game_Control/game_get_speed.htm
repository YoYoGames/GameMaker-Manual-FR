jeu_get_speed

Cette fonction peut être utilisée pour obtenir la vitesse du jeu, soit en nombre d'images de jeu à exécuter par seconde, soit en nombre de microsecondes par image de jeu. Notez qu'il ne s'agit pas de la valeur FPS de la vitesse d'exécution réelle (pour cela, utilisez la variable fps_real ) mais plutôt le nombre d'images de jeu (FPS) que le jeu tentera de maintenir chaque seconde, ou la longueur de chaque image de jeu en microsecondes que le jeu tentera de maintenir (MPF). Lorsque vous utilisez cette fonction, vous devez indiquer l'une des constantes suivantes, qui déterminera le type de la valeur de retour :

Vitesse constante du jeu
Constant Description
gamespeed_fps Obtient la vitesse du jeu en images par seconde.
gamespeed_microseconds Obtient la vitesse du jeu en microsecondes par image.

Ainsi, par exemple, si la vitesse du jeu est fixée à 30 dans les options du jeu et que vous utilisez le type FPS, la fonction renverra 30, mais si vous utilisez le type MPF, la fonction renverra 33333.

Syntaxe :

game_get_speed(type);

Arguments Type Description
type Vitesse constante du jeu The type of method used to get the game speed (see the constants above).

Les retours :

Real

Exemple :

if game_get_speed(gamespeed_fps) != 60
{
    game_set_speed(60, gamespeed_fps);
}

Le code ci-dessus vérifie si le jeu fonctionne avec une vitesse de 60 FPS et si non, il est réglé sur 60 FPS.
