jeu_set_speed

Cette fonction peut être utilisée pour définir la vitesse du jeu. Vous pouvez la définir de deux façons - en images de jeu par seconde (FPS) ou en microsecondes par image de jeu (MPF) - en utilisant l'une des deux constantes suivantes :

Constant Description
gamespeed_fps Définit la vitesse du jeu en images par seconde.
gamespeed_microseconds Définit la vitesse du jeu en utilisant des microsecondes par image.

Ainsi, par exemple, une vitesse de jeu de 30 images par seconde correspondrait à 33333 microsecondes par image de jeu, ce qui serait ensuite exprimé par cette fonction comme suit :

game_set_speed(30, gamespeed_fps);

ou :

game_set_speed(33333, gamespeed_microseconds);

Syntaxe :

game_set_speed(speed, type);

Arguments Type Description
speed Real The new game speed (as either FPS or MPF).
type Vitesse constante du jeu The type of method used to set the game speed (see the constants above).

Les retours :

N/A

Exemple :

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Le code ci-dessus vérifie si le jeu s'exécute dans un navigateur et définit la vitesse du jeu en conséquence sous forme de valeur FPS.
