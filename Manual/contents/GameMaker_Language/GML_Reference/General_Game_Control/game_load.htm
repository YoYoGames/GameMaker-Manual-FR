game_load

Avec cette fonction, vous pouvez charger un jeu qui a été précédemment sauvegardé à l'aide de la fonction game_save(). Notez que cela rétablira la version du jeu qui a été utilisée pour créer la sauvegarde, donc toute mise à jour effectuée après celle-ci ne sera pas visible. Pour plus d'informations, lisez la page sur game_save().

Syntaxe :

game_load(filename);

Arguments Type Description
filename Chaîne de caractères The name of the file to load from.

Les retours :

N/A

Exemple :

if keyboard_check_pressed(ord("L"))
{
    if (global.Save) game_load("Save.dat");
}

Le code ci-dessus chargera une partie précédemment sauvegardée si une variable globale est true lorsque le joueur appuie sur la touche "L".
