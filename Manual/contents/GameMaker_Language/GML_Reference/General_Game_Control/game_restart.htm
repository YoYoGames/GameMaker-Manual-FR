redémarrage du jeu

Cette fonction vous permet de redémarrer le jeu. Cela revient essentiellement à lancer le jeu pour la première fois et l'événement de début de partie sera donc déclenché, ainsi que l ' événement de fin de partie.

Il convient de noter que certaines choses ne seront pas remises à zéro lorsque cette fonction est appelée :

  • Les variables globales ne seront pas réinitialisées à moins d'être explicitement codées comme telles - par exemple, la variable globale intégrée score ne repartira pas à zéro après un redémarrage du jeu si elle a déjà été modifiée dans le jeu.
  • L'état du GPU ne sera pas modifié (ainsi, si vous avez défini la couleur ou l'alpha du dessin, par exemple, il restera à la valeur modifiée).
  • La vitesse du jeu restera celle que vous avez définie dans votre code de jeu (si vous l'avez modifiée, ce changement sera perpétué).
  • Toute ressource du navigateur de ressources qui a été modifiée au moment de l'exécution dans le jeu - par exemple si vous changez l'origine d'une ressource sprite ou déplacez la position d'une ressource de chemin - ne sera pas réinitialisée.
  • Les ressources dynamiques telles que buffers, les surfaces, les structures de données ou les ressources importées sprites ne seront pas non plus nettoyées ou supprimées (bien que vous puissiez perdre les références à ces ressources, il faut donc veiller, lorsque vous utilisez cette fonction, à utiliser des références globales pour la ressource dynamique ou à les nettoyer avant l'appel de la fonction).

Syntaxe :

game_restart();

Les retours :

N/A

Exemple :

if keyboard_check_pressed(ord("R")) game_restart();

Cela permettrait de redémarrer le jeu lorsque le joueur appuie sur la touche "R".
