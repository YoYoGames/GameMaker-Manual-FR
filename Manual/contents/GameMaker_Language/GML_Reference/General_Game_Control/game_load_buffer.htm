tampon de chargement de jeu

Avec cette fonction, vous pouvez charger un état de jeu qui a été sauvegardé précédemment. Le jeu est chargé à partir d'un "grow" buffer créé précédemment (voir Buffers) et le buffer doit avoir un état de jeu sauvegardé à l'aide de la fonction game_save_buffer() fonction.

Syntaxe :

game_load_buffer(buffer);

ArgumentsType Description
buffer The buffer id to load from.

Les retours :

Exemple :

if keyboard_check_pressed(ord("L"))
{
    if global.Checkpoint game_load_buffer(save_buff);
}

Le code ci-dessus chargera un état de jeu précédemment sauvegardé depuis le site buffer indexé dans la variable "save_buff", uniquement si la variable globale est true, lorsque le joueur appuie sur la touche "L".
