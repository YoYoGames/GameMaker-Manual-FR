highscore_add

Cette fonction permet d'ajouter un nom et un score à la liste globale interne des meilleurs scores. Il n'est pas nécessaire de vérifier la valeur pour voir si elle est suffisamment élevée pour être inscrite dans la liste des scores, car GameMaker ne stockera que les valeurs supérieures à la dixième position stockée.

Syntaxe :

highscore_add(str, numb);

Arguments Type Description
str Chaîne de caractères The string (name) to attribute the score to.
numb Real The number (score) to add.

Les retours :

N/A

Exemple :

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

Le code ci-dessus vérifie les vies et si elles sont à 0, il ajoute le score actuel et la chaîne de caractères contenue dans la variable globale "Name" dans la liste des meilleurs scores, avant de mettre le score à 0 et d'envoyer le joueur dans la pièce indexée dans "rm_MainMenu".
