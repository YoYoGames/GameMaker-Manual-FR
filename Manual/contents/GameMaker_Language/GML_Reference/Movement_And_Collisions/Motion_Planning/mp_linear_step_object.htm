mp_linear_step_object

Avec cette fonction, vous demandez à une instance de faire un "pas" vers un point spécifique, spécifié par les valeurs xgoal et ygoal. La taille du pas (c'est-à-dire le nombre de pixels que l'instance doit déplacer à chaque pas) est indiquée par stepsize, et si l'instance est déjà à la position, elle s'arrêtera et ne se déplacera pas plus loin, contrairement à d'autres fonctions plus simples comme move_towards_point(). Le stepsize est également la distance "en avant" à laquelle l'objet vérifiera à chaque pas s'il y a collision, et vous pouvez choisir si l'instance s'arrête en cas de collision avec un objet ou une instance de votre choix.

Syntaxe :

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

Arguments Type Description
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj ID de l'actif ou de l'instance del'objet The object that is to block the path. Can be an object index, an instance id or the special keyword, all

Les retours :

Boolean

Exemple :

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_explosion);
    instance_destroy();
}

Le code ci-dessus va faire en sorte que l'objet se déplace vers la souris à une vitesse de 5 pixels par pas, en vérifiant uniquement les collisions avec l'objet "obj_Wall". Une fois qu'il aura atteint la position de la souris, il créera un objet "obj_Explosion" et se détruira.
