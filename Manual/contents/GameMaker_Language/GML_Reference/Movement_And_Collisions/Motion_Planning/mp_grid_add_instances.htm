mp_grid_add_instances

Cette fonction utilise le masque de collision de l'instance (déterminé par les propriétés de sprite ou l'indice de masque de l'instance appelante) pour marquer les cellules comme interdites ou non dans une grille MP, où les cellules interdites ne peuvent être traversées par aucune des fonctions de pathfinding. Vous pouvez spécifier dans la fonction s'il faut considérer les collisions précises ou non et les cellules interdites marquées changeront en fonction de ce paramètre. Cette image illustre ce comportement :

Les deux instances ci-dessus ont été ajoutées à la grille du MP à l'aide de la fonction mp_grid_add_instances(), precise étant réglé sur true. Comme vous pouvez le constater, l'instance verte a uniquement marqué les cases de la grille qu'elle "touche" comme étant interdites, car son masque sprite est également défini sur precise. Cependant, la seconde instance a également marqué d'autres cellules qui ne semblent pas se toucher. Cela est dû au fait que le masque sprite n' est pas défini comme précis, ce qui signifie que même si vous utilisez le paramètre précis dans la fonction, seule la boîte englobante de sprite sera prise en compte. C'est également ce qui se produirait si vous donniez la valeur false à l'argument precise de la fonction - même les instances avec un masque sprite précis seront ajoutées dans la grille sur la base de leurs boîtes englobantes.

Syntaxe :

mp_grid_add_instances(id, obj, prec);

Arguments Type Description
id MP Grid ID Index of the mp_grid that is to be used
obj ID de l'actif ou de l'instance del'objet Object index, or instance id, of the instances to be added into the mp_grid
prec Booléen Whether the check is based on pixel-perfect collisions (true = slow) or its bounding box in general (false = fast).

Les retours :

N/A

Exemple :

mp_grid_add_instances(grid, obj_Wall, true);

Le code ci-dessus ajoutera toutes les instances de "obj_Wall" dans la grille mp_grid indexée dans la variable "grid" en utilisant le masque de collision précis plutôt que la boîte englobante.
