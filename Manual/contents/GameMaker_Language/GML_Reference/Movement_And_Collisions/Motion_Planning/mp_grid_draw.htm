mp_grid_draw

Cette fonction va dessiner la grille MP spécifiée (telle que définie par mp_grid_create()), en marquant les cellules libres en vert et les cellules interdites en rouge. Cette fonction est essentielle en tant qu'outil de débogage, mais il convient de noter qu'elle est très lente et ne fonctionne que lorsqu'elle est utilisée dans l'événement draw de l'instance, et que vous pouvez définir l'alpha de draw pour modifier l'opacité de la grille, ce qui vous permet de la dessiner en tant que superposition et de voir ce qui se trouve réellement dans la pièce en même temps.

Syntaxe :

mp_grid_draw(id);

Arguments Type Description
id MP Grid ID Index of the mp_grid that is to be drawn

Les retours :

Boolean

Exemple :

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

Le code ci-dessus dessinera la mp_grid indexée dans la variable "grid" comme une superposition semi-transparente (mais seulement si l'instance qui exécute le code a une profondeur inférieure à toutes les autres).
