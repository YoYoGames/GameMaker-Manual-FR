mp_potential_step

Cette fonction permet à l'instance de faire un pas vers une position particulière définie par xgoal/ygoal, tout en essayant d'éviter les obstacles. Lorsque l'instance rencontre une instance solide (ou toute instance pour laquelle checkall est vrai), elle change la direction de son mouvement pour essayer d'éviter cette instance et de la contourner. Il n'est pas garanti que cette approche fonctionne, mais dans la plupart des cas faciles, elle déplacera effectivement l'instance vers le but. La fonction retourne si le but a été atteint ou non.

Syntaxe :

mp_potential_step(xgoal, ygoal, stepsize, checkall)

ArgumentsType Description
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

Les retours :

Exemple :

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step(inst.x, inst.y, 5, false);
}

Le code ci-dessus vérifie d'abord s'il y a des instances de "obj_Enemy" dans la pièce. Si c'est le cas, il trouve l'instance la plus proche et stocke son identifiant dans une variable. Cette variable est ensuite utilisée pour indiquer à mp_potential_step de déplacer l'instance avec le code vers la position x/y de l'objet qui a été trouvé à une vitesse de 5 pixels par pas tout en évitant uniquement les instances marquées comme solides.
