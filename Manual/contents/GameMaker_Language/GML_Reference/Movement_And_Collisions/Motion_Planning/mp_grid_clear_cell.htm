mp_grid_clear_cell

Cette fonction peut être utilisée pour effacer une "cellule" spécifique d'une grille MP. Les cellules ne sont pas calculées comme des coordonnées de la pièce, mais plutôt comme des coordonnées de la grille, où (0,0) est le coin supérieur gauche de la grille. Cela signifie que pour effacer une cellule à une position spécifique dans la pièce, nous devons changer les coordonnées x et y en coordonnées de cellule en les divisant par la résolution de la grille MP. L'exemple de code ci-dessous montre comment cela fonctionne.

Syntaxe :

mp_grid_clear_cell(id, h, v);

Arguments Type Description
id MP Grid ID Index of the mp_grid that is to be used
h Real Horizontal position of the cell to clear
v Real vertical position of the cell to clear

Les retours :

Boolean

Exemple :

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

Le code ci-dessus va faire en sorte que tous les "obj_Box" se détruisent et qu'ils marquent comme libres les cellules qu'ils occupaient dans la grille mp_grid indexée dans la variable "grid". Dans cet exemple, nous trouvons la cellule appropriée en prenant les coordonnées x/y de l'objet et en les divisant par la résolution de la grille (en utilisant floor pour que les valeurs restent des entiers).
