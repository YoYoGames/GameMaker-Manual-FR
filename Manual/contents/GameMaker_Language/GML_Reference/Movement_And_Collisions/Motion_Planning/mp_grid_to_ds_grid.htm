mp_grid_to_ds_grid

Avec cette fonction, vous pouvez copier la grille MP donnée dans une grille DS. La grille DS doit avoir la même taille que la grille MP, bien qu'elle ne doive pas nécessairement l'être (des données seront perdues si elle est plus petite, et si elle est plus grande, toutes les cellules supplémentaires de la grille seront à 0). Une cellule de la grille DS contiendra la valeur -1 si elle a été marquée comme occupée dans la grille MP, ou 0 si elle ne l'était pas.

Syntaxe :

mp_grid_to_ds_grid(source, destination);

Arguments Type Description
source MP Grid ID Index of the mp_grid that is to be used
destination ID de la grille DS Index of the ds_grid that is to be used to copy the grid data to.

Les retours :

Boolean

Exemple :

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

Le code ci-dessus va créer une nouvelle grille DS et ensuite copier les données de la grille MP contenues dans la variable "mp_grid" dans la nouvelle grille DS.
