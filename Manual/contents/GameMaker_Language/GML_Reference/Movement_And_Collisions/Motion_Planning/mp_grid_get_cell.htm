mp_grid_get_cell

Avec cette fonction, vous pouvez vérifier n'importe quelle cellule de la grille mp_grid pour voir si elle a été marquée comme occupée ou non, en donnant l'index de la grille mp_grid (précédemment créée) et les coordonnées x et y de la cellule à vérifier.

Si elle a été occupée ou si la position vérifiée est en dehors des limites de la grille, la fonction renvoie -1, sinon elle renvoie 0.

Syntaxe :

mp_grid_get_cell(id, x , y);

Arguments Type Description
id MP Grid ID Index of the mp_grid that is to be used
x1 Real The x coordinate of the grid to check.
y1 Real The y coordinate of the grid to check.

Les retours :

Real

Exemple :

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

Le code ci-dessus va vérifier la cellule mp_grid qui correspond à la position de la souris et si elle est occupée, il met la variable image_blend en rouge, et si elle n'est pas occupée, il la met en vert.
