mp_linear_step

Avec cette fonction, vous demandez à une instance de faire un "pas" vers un point spécifique, spécifié par les valeurs xgoal et ygoal. La taille du pas (c'est-à-dire le nombre de pixels que l'instance doit déplacer à chaque pas) est indiquée par stepsize, et si l'instance est déjà à la position, elle s'arrêtera et ne se déplacera pas plus loin, contrairement à d'autres fonctions plus simples comme move_towards_point(). Le stepsize est également la distance "en avant" à laquelle l'objet vérifiera à chaque pas s'il y a une collision, et vous pouvez choisir si l'instance s'arrête en cas de collision avec n'importe quelle instance ou seulement avec celles qui sont marquées comme solides. La fonction retournera si elle a atteint la position cible (true) ou si elle a échoué (false).

NOTE Cette fonction n'essaie pas de faire des détours si elle rencontre un obstacle, elle échoue simplement et arrête de se déplacer.

Syntaxe :

mp_linear_step(xgoal, ygoal, stepsize, checkall);

ArgumentsType Description
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

Les retours :

Exemple :

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Le code ci-dessus fera en sorte que l'objet se déplace vers la souris à une vitesse de 5 pixels par pas. Une fois qu'il aura atteint la position de la souris, il créera un objet "obj_Explosion" et se détruira.
