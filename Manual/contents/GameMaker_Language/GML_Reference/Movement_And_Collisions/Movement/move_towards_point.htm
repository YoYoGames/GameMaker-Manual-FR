déplacement vers un point

Cette fonction indique simplement à l'instance qui exécute le code de se déplacer vers un point donné à une vitesse donnée. Gardez à l'esprit que c'est à vous de lui dire ce qu'elle doit faire une fois qu'elle y est arrivée, car elle ne s'arrêtera pas d'elle-même. Cela signifie que vous pouvez obtenir une instance "vibrante" lorsqu'elle se déplace (par exemple) de 3 pixels vers l'avant, dépasse la position cible x/y de 2, puis recule à nouveau de 3 pixels, dépasse le point cible de 1 et ainsi de suite... Il y a plusieurs façons d'éviter cela, par exemple en utilisant distance_to_point() pour voir à quelle distance de la position spécifiée l'instance se trouve avant de se déplacer.

NOTE: cette fonction définit la vitesse de l'instance, donc même si vous arrêtez d'utiliser ce code, l'instance continuera à se déplacer dans la direction précédente, il est donc nécessaire de mettre l'instance speed à 0 si vous souhaitez qu'elle s'arrête. Voir l'exemple de code ci-dessous.

Syntaxe :

move_towards_point( x, y, sp );

Arguments Type Description
x Real The x position of the point to move towards.
y Real The y position of the point to move towards.
sp Real The speed to move at in pixels per step.

Les retours :

N/A

Exemple :

if point_distance(x, y, target.x, target.y) > 5
{
    move_towards_point(target.x, target.y, 5);
}
else speed = 0;

Le code ci-dessus va déplacer l'instance vers la position de l'instance indexée dans la variable "target" à une vitesse de 5 pixels par pas. Une fois cette position atteinte, la vitesse est remise à 0.
