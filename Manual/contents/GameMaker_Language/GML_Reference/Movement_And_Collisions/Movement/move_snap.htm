move_snap

Cette fonction est utilisée pour "accrocher" l'instance à une grille d'une taille donnée. Elle sera accrochée à la position correspondante la plus proche sur la grille "invisible" que les valeurs hsnap et vsnap définissent.

Syntaxe :

move_snap(hsnap, vsnap);

ArgumentsType Description
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Les retours :

Exemple :

with (obj_Pieces)
{
    if (!place_snapped(32, 32))
    {
        move_snap(32, 32);
    }
}

Le code ci-dessus vérifie toutes les instances de "obj_Pieces" pour voir si elles sont fixées à une grille de 32x32 pixels, et si elles ne le sont pas, il les fixe à la position la plus proche dans cette grille.
