déplacer_contact_solide

Cette fonction déplacera l'instance qui exécute le code d'un nombre déterminé de pixels dans la direction spécifiée jusqu'à ce qu'elle rencontre une instance dont l'indicateur est solide marqué comme vrai. Vous pouvez utiliser -1 ou 0 pour la distance maximale qui est de 1000 pixels par défaut, c'est-à-dire que GameMaker déplacera l'instance continuellement de 1000 pixels jusqu'à ce qu'elle ne soit plus en collision.

Syntaxe :

move_contact_solid( dir, maxdist );

ArgumentsType Description
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Les retours :

Exemple :

if !place_meeting(x, y + 1, obj_Floor)
{
    move_contact_solid(270, -1);
}

Le code ci-dessus vérifiera sous l'instance s'il y a une collision avec "obj_Floor", et s'il n'y en a pas, il la déplacera vers le bas jusqu'à ce qu'il y ait une collision, ou que 1000 pixels aient été couverts.
