place_snapped

Avec cette fonction, vous pouvez vérifier et voir si l'origine d'une instance (sa position x et y) est alignée sur une grille avec les valeurs hsnap et vsnap que vous avez spécifiées.

Syntaxe :

place_snapped(hsnap, vsnap);

ArgumentsType Description
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

Les retours :

Exemple :

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

Le code ci-dessus vérifie toutes les instances de "obj_Pieces" pour voir si elles sont fixées à une grille de 32x32 pixels, et si elles ne le sont pas, il les fixe à la position la plus proche dans cette grille.
