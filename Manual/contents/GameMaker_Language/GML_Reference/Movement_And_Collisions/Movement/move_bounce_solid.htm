move_bounce_solid

Avec cette fonction, vous pouvez commander à une instance de ne rebondir que sur les instances marquées comme solides dans la room. Vous pouvez également lui demander d'utiliser un contrôle de collision précis lorsqu'il est activé, mais sachez que cela nécessite que toutes les instances aient des masques précis activés. et ralentira considérablement votre jeu lorsque de nombreuses instances sont impliquées, en raison de la quantité de traitement à effectuer. Ceci devrait normalement aller dans l'événement step d'une instance, mais peut aussi être utilisé sélectivement dans un événement de collision.

Syntaxe :

move_bounce_solid(adv);

ArgumentsType Description
adv Whether to enable advanced bouncing (true) or not (false).

Les retours :

Exemple :

move_bounce_solid(false);

Cela permettra un rebondissement non précis sur les instances marquées comme "solides".
