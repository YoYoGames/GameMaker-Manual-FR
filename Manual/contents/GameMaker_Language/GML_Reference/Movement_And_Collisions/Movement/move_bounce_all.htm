move_bounce_all

Avec cette fonction, vous pouvez commander à une instance de rebondir sur toutes les instances de la pièce, à la seule exception de celles qui n'ont pas de sprite ou d'index de masque assigné. Vous pouvez également lui demander d'utiliser un contrôle de collision précis lorsqu'il est activé, mais sachez que cela nécessite que toutes les instances aient des masques précis activés et que cela ralentira considérablement votre jeu lorsque de nombreuses instances sont impliquées, en raison de la quantité de traitement à effectuer. Cette fonction doit normalement être utilisée dans l'événement step d'une instance, mais elle peut également être utilisée de manière sélective dans un événement de collision, comme le montre l'exemple de code ci-dessous.

Syntaxe :

move_bounce_all( adv );

ArgumentsType Description
adv Whether to enable advanced bouncing (true) or not (false).

Les retours :

Exemple :

if other.visible
{
    move_bounce_all(false);
}

Le code ci-dessus serait placé dans un événement de collision avec un autre objet. Il vérifie d'abord si l'objet est visible et, s'il l'est, il exécute l'action move_bounce_all(). Notez que dans ce cas, le rebond est sélectif et ne sera calculé que pour cette collision, plutôt que pour chaque instance, à chaque étape.
