motion_set

Cette fonction définit une nouvelle direction de mouvement et une nouvelle vitesse pour l'instance qui exécute le code. Notez que cela ne s'ajoute pas à la vitesse et à la direction actuelles de l'instance (pour cela, utilisez motion_add()), mais la force à utiliser les nouveaux paramètres.

Syntaxe :

motion_set(dir, speed);

ArgumentsType Description
dir The new direction.
speed The new speed.`

Les retours :

Exemple :

if irandom(9) = 1
{
    motion_set(random(360), 1 + random(3));
}

Le code ci-dessus fera en sorte que l'instance change de vitesse et de direction à des intervalles aléatoires.
