motion_add

Cette fonction modifiera les valeurs actuelles de direction et speed de l'instance qui exécute le code, en ajoutant les nouvelles valeurs aux valeurs actuelles. Si vous souhaitez simplement modifier ces valeurs, vous devriez utiliser la fonction motion_set().

Syntaxe :

motion_add(dir, speed);

ArgumentsType Description
dir The added direction.
speed The added speed.

Les retours :

N/A

Exemple :

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

le code ci-dessus serait appelé dans l'événement de collision avec un autre objet. Il ajoute à la direction du mouvement et à la vitesse de l'instance un vecteur basé sur la position et la vitesse de l'autre instance impliquée dans la collision. Il limite ensuite la vitesse si elle dépasse 8 (pixels par pas).
