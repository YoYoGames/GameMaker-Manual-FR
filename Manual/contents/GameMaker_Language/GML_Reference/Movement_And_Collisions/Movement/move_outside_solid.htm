déplacer_extérieur_solide

Avec cette fonction, vous pouvez dire à une instance de se déplacer hors d'une collision avec n'importe quelle instance marquée comme étant solide dans n'importe quelle direction et avec n'importe quel nombre de pixels à chaque pas, avec une valeur de -1 ou 0 pour la distance maximale qui est par défaut de 1000px, c'est-à-dire que GameMaker déplacera l'instance continuellement de 1000 pixels jusqu'à ce qu'elle soit hors collision.

Syntaxe :

move_outside_solid(dir, maxdist);

ArgumentsType Description
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Les retours :

Exemple :

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

Le code ci-dessus se place dans un événement de collision et vérifie si l'"autre" instance de la collision est marquée comme "solide". Si c'est le cas, il déplace l'instance hors de la collision.
