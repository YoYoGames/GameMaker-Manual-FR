déplacer_aléatoire

Avec cette fonction, vous pouvez placer l'instance à une position quelconque dans la pièce, mais alignée sur une grille "invisible". Ainsi, une valeur de 32 pour le hsnap et le vsnap placera l'instance à une position aléatoire qui est alignée sur une grille de 32x32 carrés. Vous pouvez mettre ces valeurs à 1 pour obtenir une position n'importe où dans la pièce.

REMARQUE: cette fonction se positionnera aux mêmes endroits à chaque fois que le jeu sera relancé, car GameMaker génère à chaque fois la même graine aléatoire initiale afin de rendre le code de debugging beaucoup plus facile. Pour éviter ce comportement, utilisez randomise() au début de votre jeu. Cela n'est vrai que lorsque vous testez le jeu et que vous l'envoyez à l'adresse debugging, car l'exécutable final ne présentera pas ce comportement et sera aléatoire à chaque partie.

Syntaxe :

move_random(hsnap, vsnap);

ArgumentsType Description
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Les retours :

Exemple :

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

Ceci déplacera l'instance à une position aléatoire n'importe où dans la pièce lorsque la touche espace est pressée.
