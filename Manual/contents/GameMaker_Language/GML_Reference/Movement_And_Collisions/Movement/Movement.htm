Mouvement

Dans tout jeu, le mouvement et la position sont d'une importance capitale et GameMaker dispose donc d'une sélection complète de fonctions pour faire face à toutes les situations. Les deux principales façons de déplacer une instance sont soit de définir la position réelle, soit de définir un vecteur vitesse/direction, et cela peut être fait soit en utilisant les variables d'instance intégrées, soit en utilisant des fonctions de mouvement spécifiques. Ces deux options sont expliquées dans les sections ci-dessous :

Les variables suivantes sont toutes incluses "intégrées" dans chaque instance et peuvent être lues et modifiées pour changer différents comportements lorsque l'instance se déplace (elles affecteront les vecteurs de vitesse et de direction) :

  • direction
  • friction
  • gravity
  • gravity_direction
  • hspeed
  • vspeed
  • speed

En plus des variables mentionnées ci-dessus, vous pouvez également définir la position de l'instance directement en utilisant ce qui suit :

  • x
  • y
  • xprevious
  • yprevious
  • xstart
  • ystart

Les fonctions suivantes peuvent toutes être utilisées pour déplacer une instance d'une manière ou d'une autre, certaines affectant les vecteurs de vitesse/direction et d'autres affectant directement la position x/y réelle dans la pièce :

  • motion_add
  • motion_set
  • move_towards_point
  • move_bounce_all
  • move_bounce_solid
  • move_contact_all
  • move_contact_solid
  • move_outside_all
  • move_outside_solid
  • move_random
  • move_snap
  • move_wrap
  • place_snapped
