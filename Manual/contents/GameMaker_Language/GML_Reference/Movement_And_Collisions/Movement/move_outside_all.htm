move_outside_all

Avec cette fonction, vous pouvez dire à une instance de se déplacer hors d'une collision dans n'importe quelle direction et de n'importe quel nombre de pixels à chaque pas, avec une valeur de -1 ou 0 pour le maxdist qui est par défaut de 1000px, c'est-à-dire que GameMaker déplacera l'instance continuellement vers le haut de 1000 pixels jusqu'à ce qu'elle soit hors collision.

Syntaxe :

move_outside_all(dir, maxdist);

ArgumentsType Description
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Les retours :

Exemple :

if place_meeting(x, y, all)
{
    move_outside_all(90, 1);
}

Le code ci-dessus déplacera l'instance d'un pixel vers le haut à chaque pas de collision avec une autre instance.
