ligne de collision

Collision_line vérifie le long d'une ligne allant du point x1,y1 au point x2,y2 s'il y a une collision avec une instance de l'objet spécifié par l'argument "obj". cette vérification peut être précise ou non, mais pour que les collisions précises soient activées, l'objet ou l'instance que vous vérifiez doit également avoir des collisions précises activées pour leur sprite. Si ce n'est pas le cas, la vérification par défaut est basée sur les boîtes englobantes. L'image suivante illustre ce fonctionnement :

N'oubliez pas que pour que les collisions précises soient prises en compte, l'objet sprite et la fonction de collision doivent tous deux avoir la précision activée. Il faut également noter que la valeur de retour de la fonction peut être l'id de n'importe laquelle des instances considérées comme étant en collision le long de la ligne, donc si trois instances chevauchent la ligne définie, n'importe lequel de leurs ids pourrait être la valeur de retour de la fonction.

Notez qu'au lieu d'un indice d'objet, vous pouvez fournir une instance id pour rechercher une instance spécifique, ou les mots-clés d'instance all, ou other (en fonction de l'événement et de la portée actuelle du code).

Syntaxe :

collision_line(x1, y1, x2, y2, obj, prec, notme);

Arguments Type Description
x1 Real The x coordinate of the start of the line.
y1 Real The y coordinate of the start of the line.
x2 Real The x coordinate of the end of the line.
y2 Real The y coordinate of the end of the line.
obj ID de l'actif ou de l'instance del'objet The object to check for instance collisions.
prec Booléen Whether the check is based on precise collisions (truece qui est plus lent) ou sa boîte de délimitation en général (false, plus rapide).
notme Booléen Whether the calling instance, if relevant, should be excluded (true) ou non (false).

Les retours :

Instance ID or noone

Exemple :

with (obj_Enemy)
{
    if collision_line(100, 400, 100, 600, id, false, false) instance_destroy();
}

Ce code demande à toutes les instances de "obj_Enemy" de vérifier le long d'une ligne de 100x, 400y à 100x, 600y s'il y a une collision avec elles-mêmes, et si c'est le cas, elles sont détruites.
