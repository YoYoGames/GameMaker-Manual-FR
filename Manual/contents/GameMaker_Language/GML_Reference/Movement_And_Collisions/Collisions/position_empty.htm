position_vide

Cette fonction vérifie si une position donnée entre en collision avec une instance ayant un masque de collision valide à la position donnée.

Syntaxe :

position_empty(x, y);

ArgumentsType Description
x The x position to check.
y The y position to check.

Les retours :

Exemple :

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if position_empty(xx, yy)
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

Cela va vérifier une position aléatoire quelque part dans la pièce pour une collision et s'il n'y en a pas, cela va créer une instance de obj_Bomb.
