position_destroy

Cette fonction détruit simplement toutes les instances qui se trouvent en collision avec la position spécifiée. Les collisions sont basées sur le masque des instances, et si une partie du masque chevauche le point cible, la fonction détruira cette instance. Les instances détruites de cette manière déclencheront leurs événements Destroy et Clean Up.

Syntaxe :

position_destroy(x, y);

ArgumentsType Description
x The x coordinate of where to destroy colliding instances.
y The y coordinate of where to destroy colliding instances.

Les retours :

Exemple :

if mouse_check_button_pressed(mb_left)
{
    position_destroy(mouse_x, mouse_y);
}

Cela détruira toutes les instances qui entrent en collision avec la position du curseur de la souris lorsque le bouton gauche de la souris est enfoncé.
