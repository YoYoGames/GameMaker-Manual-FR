position_change

Cette fonction vérifiera la présence d'une collision entre une position et toute instance au point donné, et si c'est le cas, elle transformera toutes les instances en collision en instances de l'objet choisi. Vous pouvez définir l'argument "perf" à true, ce qui forcera GameMaker à exécuter les événements Destroy et Clean Up de l'instance trouvée ainsi que l'événement Create de la nouvelle instance, ou false à n'exécuter aucun de ces événements. Veuillez noter que si vous choisissez de ne pas exécuter les événements Destroy, Clean Up et Create, toute instance créée qui utilise une variable normalement définie dans l'événement create fera planter le jeu car cette variable n'existera plus.

Syntaxe :

position_change(x, y, obj, perf);

Arguments Type Description
x Real The x coordinate of where to change colliding instances.
y Real The y coordinate of where to change colliding instances.
obj Objet Asset The new object the calling object will change into.
perf Booléen Whether to perform that new object's Create event (true) or not (false).

Les retours :

N/A

Exemple :

position_change(200, 200, obj_Bird, true);

Cela va transformer toutes les instances entrant en collision au point (200,200) en une instance de "obj_Bird", en exécutant l'événement Create de "obj_Bird" pour chacune d'entre elles.
