place_free

Vous pouvez utiliser cette fonction pour vérifier si l'instance appelante entre en collision avec une instance marquée comme solide dans votre jeu. Il convient de noter que pour que cela fonctionne, l'instance qui exécute le code doit disposer d'un masque de collision valide (soit pour le site sprite lui-même, soit par le biais de la fonction mask_index) et elle n'enregistrera les collisions qu'avec les instances marquées solid qui ont également un masque valide.

La fonction elle-même fonctionne essentiellement en prenant l'instance et en testant les collisions avec les seules instances marquées solidement lorsqu'elles sont placées à la position spécifiée par les arguments x/y. La vérification des collisions peut être soit précise, soit basée sur la boîte englobante de l'instance, selon le type de masque de collision sélectionné.

Notez que les coordonnées x/y données seront ramenées à l'entier le plus proche avant que la vérification ne soit effectuée.

Syntaxe :

place_free(x, y);

ArgumentsType Description
x The x position to check.
y The y position to check.

Les retours :

Exemple :

if place_free(mouse_x, mouse_y)
{
    x = mouse_x;
    y = mouse_y;
}

Le code ci-dessus vérifiera la présence d'une collision avec toute instance marquée par un drapeau solide si l'instance appelante était placée à la même position que la souris. Si aucune collision n'est détectée, les coordonnées x/y de l'instance sont définies comme celles de la souris.
