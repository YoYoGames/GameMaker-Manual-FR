point_in_circle

En utilisant cette fonction, vous définissez une zone circulaire et GameMaker déterminera si le point donné se trouve dans ses limites ou non. Si le point se trouve à l'intérieur du cercle défini, la fonction renvoie true; sinon, elle renvoie false.

Syntaxe :

point_in_circle(px, py, x1, y1, rad);

ArgumentsType Description
px The x coordinate of the point to check.
py The y coordinate of the point to check.
x1 The x coordinate of the circle centre.
y1 The y coordinate of the circle centre.
rad The radius of the circle.

Les retours :

Exemple :

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
    over = true;
}
else
{
    over = false;
}

Le code ci-dessus utilise la fonction point_in_circle pour vérifier si la position de la souris se trouve dans la zone circulaire définie, en donnant à la variable "over" la valeur true si c'est le cas, ou false sinon.
