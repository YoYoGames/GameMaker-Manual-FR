clickable_add_ext

Avec cette fonction, vous pouvez créer une icône cliquable personnalisée qui agit comme un "bouton" en dehors du canevas de jeu, mais dans la fenêtre du navigateur elle-même. La fonction renvoie une valeur d'identification qui doit être stockée dans une variable à utiliser dans toutes les autres fonctions de bouton qui font référence au nouveau bouton cliquable. Cette fonction est très utile pour empêcher le navigateur de créer une fenêtre pop-up lorsqu'on clique sur le bouton, car tout est fait par GameMaker lui-même.

Cette fonction fonctionne exactement de la même manière que la fonction clickable_add() où vous devez définir la position par rapport au (0,0) de la fenêtre du navigateur, donner l'entrée de la page de texture pour le sprite que vous souhaitez que l'icône du bouton ait (voir : sprite_get_tpe()), spécifier soit une URL à ouvrir, soit un script (du navigateur d'actifs de GameMaker) à ouvrir ou à exécuter, puis définir la cible et les paramètres à utiliser. Cependant, cette fonction dispose également de deux options supplémentaires qui vous permettent de définir l'alpha de l'icône de 0 (totalement transparent) à 1 (totalement opaque) et l'échelle de l'icône, qui est une valeur multiplicatrice appliquée à l'adresse sprite utilisée. Ainsi, une valeur de 1 correspond à l'échelle par défaut, tandis que (par exemple) 0,5 correspond à la moitié de l'échelle et 2 au double.

Il convient également de noter que si l'argument URL que vous transmettez est le nom d'une fonction de script (c'est-à-dire qu'il ne commence pas par "http://") ET que le script s'appelle gmcallback_*, vous pouvez obtenir des entrées directes dans un événement html5. Les noms des fonctions de script de GameMaker ne sont pas obscurcis s'ils commencent par gmcallback_.

Syntaxe :

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Arguments Type Description
x Real The x position within the window.
y Real The y position within the window.
tpe Entrée de la page des textures The texture page entry for the sprite to be used.
url Chaîne de caractères The URL (website address) to link to, or the script to be run.
target Chaîne de caractères This is the target area to open the URL in (see description).
params Chaîne de caractères The various parameters for the icon (see description).
alpha Real The image alpha of the icon (default 1).
scale Real The scale of the icon (default 1).

Les retours :

Clickable ID

Exemple :

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

Le code ci-dessus crée une icône DOM cliquable à la position (32, 32) de la page sur laquelle le canevas de jeu est exécuté. L'icône utilise l'adresse sprite référencée dans la page de texture sous le nom de "spr_MS_Home". Lorsqu'on clique sur l'icône, une nouvelle fenêtre s'ouvre pour l'URL spécifiée et avec les propriétés définies pour cette fenêtre. Le bouton est également mis à l'échelle pour atteindre deux fois la taille originale de sprite et un alpha d'image de 1.
