webgl_enabled

Cette variable en lecture seule indique si WebGL est activé (true) ou non (false) pour votre jeu. Elle ne fonctionnera que pour les jeux qui s'exécutent via un navigateur (c'est-à-dire HTML5). Pour toutes les autres plateformes, elle renverra la valeur true.

Syntaxe :

webgl_enabled

Les retours :

Exemple :

if webgl_enabled
{
    global.quality = 1;
}
else global.quality = 0;

Le code ci-dessus vérifie l'indicateur WebGL et définit ensuite la variable globale "qualité" en conséquence.
