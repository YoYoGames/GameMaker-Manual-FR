analytics_event_ext

Cette fonction enverra le texte spécifié au fournisseur de données analytiques que vous avez configuré dans les options de jeu HTML5. Cette fonction peut être utilisée pour créer une analyse personnalisée afin de suivre quelque chose en dehors de la portée du fournisseur utilisé. Cette fonction peut être utilisée pour créer une analyse personnalisée afin de suivre quelque chose en dehors de la portée du fournisseur utilisé. Elle acceptera également des paires paramètre/valeur personnalisées, où le paramètre est une chaîne et la valeur un nombre réel. Pour Google Analytics, vous ne pouvez ajouter qu'une seule paire supplémentaire, tandis que Flurry accepte jusqu'à 7.

Syntaxe :

analytics_event_ext(string, string_param, value);

ArgumentsType Description
string A string to send to the provider.
string_param[0 -9] The parameter to send (a string).
value[0 - 9] The value of the parameter (a real number).

Les retours :

Exemple :

var num, time;
ini_open("Options.ini");
num = ini_read_real("Data", "Plays", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Plays", num, "Time", time);
ini_write_real("Data", "Plays", num);
ini_close();

Le code ci-dessus récupérera les informations de lecture d'un fichier ini ainsi que l'heure actuelle et enverra ensuite ces détails au fournisseur d'analyses.
