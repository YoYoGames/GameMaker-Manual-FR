clickable_set_style

Cette fonction vous permet de définir les propriétés de style CSS pour le bouton donné par l'intermédiaire des paires clé/valeur du fichier DS Map. Vous devez avoir préalablement créé l'élément bouton (à l'aide de la fonction clickable_add()) et le site DS Map et fournir les indices stockés pour chacun d'eux comme arguments.

Syntaxe :

clickable_set_style(index, map)

Arguments Type Description
index ID cliquable The index of the clickable icon to style.
map ID de la carte DS The index of the DS Map pour définir le style.

Les retours :

N/A

Exemple :

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

Le code ci-dessus changera le style du bouton cliquable avec l'indice stocké dans la variable "bouton".
