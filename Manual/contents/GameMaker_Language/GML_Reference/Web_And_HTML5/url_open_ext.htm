url_open_ext

Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert. Le paramètre "target" que vous spécifiez est le même que la valeur "name" standard de JavaScript lorsque vous utilisez la méthode open(), et vous devez savoir que tous les paramètres, sauf "_self", peuvent entraîner le blocage du navigateur, ou demander à l'utilisateur s'il souhaite l'autoriser.

Les cibles valides sont :

Cible Description
_blank Ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet (cela ne fonctionnera pas si les fenêtres pop-up sont bloquées par l'utilisateur, auquel cas vous pouvez utiliser la fonction clickable_* à la place).
_self Ouvre le document lié dans le même cadre que celui dans lequel il a été cliqué (c'est la valeur par défaut).
_parent Ouvre le document lié dans le cadre parent.
_haut Ouvre le document lié dans le corps complet de la fenêtre.

Syntaxe :

url_open_ext(url, target);

Arguments Type Description
url Chaîne de caractères The URL (website address) to link to.
target Chaîne de caractères This is the target area to open the URL in.

Les retours :

N/A

Exemple :

url_open_ext("http://gamemaker.io", "_blank");

La page d'accueil de GameMaker s'ouvre alors dans une nouvelle fenêtre.
