Texte de changement cliquable

Avec cette fonction, vous pouvez modifier l'adresse sprite et la position d'une icône cliquable précédemment créée avec clickable_add(). Gardez à l'esprit que la position est basée sur la position (0,0) de la fenêtre, et non du canevas, et que le site sprite doit être référencé directement à partir de la page de la texture (voir : sprite_get_tpe()). Cette fonction vous permet également de modifier l'échelle de sprite utilisée (sous forme de multiplicateur, de sorte que 1 est la valeur par défaut, 0,5 correspond à la moitié et 2 au double) et la valeur alpha de 0 (totalement transparent) à 1 (totalement opaque) pour l'icône finale à l'écran.

Syntaxe :

clickable_change_ext(index, tpe, x, y, alpha, scale)

Arguments Type Description
index ID cliquable The index of the clickable icon to change.
tpe Entrée de la page des textures The texture page entry for the sprite to be used
x Real The new x position within the window.
y Real The new y position within the window.
scale Real The scale of the icon (default 1).
alpha Real The image alpha of the icon (default 1).

Les retours :

N/A

Exemple :

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

Le code ci-dessus changera l'index de l'image et la position de l'icône cliquable indexée dans la variable "global.Help" en fonction de la pièce actuelle, en modifiant également l'échelle et l'alpha de l'icône.
