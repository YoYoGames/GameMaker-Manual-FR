url_open

Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert.

Syntaxe :

url_open(url);

Arguments Type Description
url Chaîne de caractères The URL (website address) to link to.

Les retours :

N/A

Exemple :

url_open("http://gamemaker.io");

Cela ouvrirait la page d'accueil de GameMaker dans la fenêtre actuelle.
