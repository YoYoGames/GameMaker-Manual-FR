ajouts cliquables

Cette fonction vous permet de créer une icône cliquable personnalisée qui agit comme un "bouton" en dehors du canevas de jeu, mais dans la fenêtre du navigateur elle-même. La fonction renvoie une valeur d'identification qui doit être stockée dans une variable à utiliser dans toutes les autres fonctions de bouton qui font référence au nouveau bouton cliquable. Cette fonction est très utile pour empêcher le navigateur de créer une fenêtre pop-up lorsqu'on clique sur le bouton, car tout est fait par GameMaker lui-même.

Vous devez d'abord définir la position, qui est relative au (0,0) de la fenêtre du navigateur (coin supérieur gauche), puis donner l'entrée de la page de texture pour le sprite que vous souhaitez que l'icône du bouton ait (voir : sprite_get_tpe()). Vous pouvez ensuite spécifier soit une URL à ouvrir, soit un script (du navigateur d'actifs de GameMaker) à ouvrir ou à exécuter. Le paramètre "target" que vous spécifiez est identique à la valeur "name" standard de JavaScript lorsque vous utilisez la méthode open() (sachez que tout autre paramètre que "_self" peut entraîner un blocage du navigateur ou demander à l'utilisateur s'il souhaite l'autoriser) et l'argument "params" est identique au paramètre "specs" de JavaScript pour contrôler les propriétés que la nouvelle fenêtre/onglet doit afficher (tous les navigateurs ne prennent pas forcément en charge toutes les fonctionnalités). Si vous utilisez l'icône pour exécuter un script interne, ces deux derniers arguments peuvent être laissés sous forme de chaînes vides "".

Les cibles valides sont :

Cible Description
_blank Ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet.
_self Ouvre le document lié dans le même cadre que celui dans lequel il a été cliqué (c'est la valeur par défaut).
_parent Ouvre le document lié dans le cadre parent.
_haut Ouvre le document lié dans le corps complet de la fenêtre.

Les paramètres valides sont :

Paramètres Description
height=[px] La hauteur de la fenêtre, la valeur minimale étant de 100.
width=[px] La largeur de la fenêtre, la valeur minimale étant de 100.
left=[px] La position gauche de la fenêtre.
top=[px] La position supérieure de la fenêtre (IE uniquement).
location=[booléen] Affichage ou non du champ d'adresse (1 par défaut).
menubar=[booléen]". Affichage ou non de la barre de menu (1 par défaut).
resizable=[boolean]". Si la fenêtre est redimensionnable ou non (la valeur par défaut est 1).
scrollbars=[boolean]". Affichage ou non des barres de défilement (la valeur par défaut est 1).
status=[booléen] Ajout ou non d'une barre d'état (la valeur par défaut est 1).
barre de titre = [booléen]". Affichage ou non de la barre de titre. Cette option est ignorée, sauf si l'application appelante est une application HTML ou une boîte de dialogue sécurisée (la valeur par défaut est 1) ;
toolbar=[boolean]". Affichage ou non de la barre d'outils du navigateur (par défaut, oui).


Il convient également de noter que si l'argument URL que vous transmettez est le nom d'une fonction de script (c'est-à-dire qu'il ne commence pas par "http://") ET que le script s'appelle gmcallback_*, vous pouvez obtenir des entrées directes dans un événement html5. Les noms des fonctions de script de GameMaker ne sont pas obscurcis s'ils commencent par gmcallback_.

Syntaxe :

clickable_add(x, y, tpe, url, target, params);

Arguments Type Description
x Real The x position within the window.
y Real The y position within the window.
tpe Entrée de la page des textures The texture page entry for the sprite to be used.
url Chaîne de caractères The URL (website address) to link to, or the script to be run.
target Chaîne de caractères This is the target area to open the URL in (see description).
params Chaîne de caractères The various parameters for the icon (see description).

Les retours :

Clickable ID

Exemple :

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Le code ci-dessus crée une icône DOM cliquable à la position (32, 32) de la page sur laquelle le canevas de jeu est exécuté. L'icône utilise l'adresse sprite référencée dans la page de la texture sous le nom de "spr_MS_Home". Lorsqu'on clique sur l'icône, une nouvelle fenêtre s'ouvre pour l'URL spécifiée et avec les propriétés définies pour cette fenêtre.
