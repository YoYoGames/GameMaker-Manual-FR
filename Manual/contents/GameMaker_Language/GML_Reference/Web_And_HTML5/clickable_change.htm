clickable_change

Grâce à cette fonction, vous pouvez modifier l'adresse sprite et la position d'une icône cliquable précédemment créée à l'aide de la fonction clickable_add(). Veuillez noter que la position est basée sur la position (0,0) de la fenêtre, et non du canevas, et que le site sprite doit être référencé directement à partir de la page de la texture (voir : sprite_get_tpe()).

Syntaxe :

clickable_change(index, tpe, x, y)

Arguments Type Description
index ID cliquable The index of the clickable icon to change.
tpe Entrée de la page des textures The texture page entry for the sprite to be used.
x Real The new x position within the window.
y Real The new y position within the window.

Les retours :

N/A

Exemple :

switch (room)
{
    case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
    case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
}

Le code ci-dessus modifiera l'index de l'image et la position de l'icône cliquable indexée dans la variable "global.Help" en fonction de la pièce actuelle.
