événement_analytique

Cette fonction enverra le texte spécifié au fournisseur d'analyses que vous avez configuré via les options de jeu HTML5. Cette fonction peut être utilisée pour créer une analyse personnalisée afin de suivre quelque chose en dehors de la portée du fournisseur utilisé.

Syntaxe :

analytics_event(string);

ArgumentsType Description
string A string to send to the provider

Les retours :

Exemple :

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

Le code ci-dessus vérifiera une variable pour voir si elle est true et si c'est le cas, un événement analytique spécial sera déclenché avec la chaîne spécifiée.
