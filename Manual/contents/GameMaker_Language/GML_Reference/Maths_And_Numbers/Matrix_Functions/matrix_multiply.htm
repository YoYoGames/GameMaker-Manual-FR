matrix_multiply

Avec cette fonction, vous pouvez multiplier deux matrices ensemble pour créer une nouvelle matrice de transformation. La fonction renvoie l'indice de la nouvelle matrice, qui doit être stocké dans une variable pour une utilisation ultérieure.

NOTE: Vous ne pouvez pas utiliser une constante de matrice comme argument avec cette fonction, donc si vous souhaitez multiplier la matrice de vue (par exemple) avec une matrice personnalisée que vous avez construite, vous devez d'abord appeler matrix_get() et affecter les valeurs de la matrice de vue à une variable de tableau, puis la multiplier par votre matrice personnalisée, et enfin définir la matrice choisie (vue, projection ou monde).

Syntaxe :

matrix_multiply(matrix1, matrix2);

ArgumentsType Description
matrix1 The first matrix index to use.
matrix2 The second matrix index to use.

Les retours :

Matrix Array

Exemple :

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

Le code ci-dessus récupère la matrice de vue actuelle, la multiplie par une matrice personnalisée, puis utilise l'indice de matrice résultant pour définir à nouveau la matrice de vue.
