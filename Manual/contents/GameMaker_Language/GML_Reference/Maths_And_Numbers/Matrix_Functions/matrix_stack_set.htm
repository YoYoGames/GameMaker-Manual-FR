ensemble_de_pile_matrice

Cette fonction écrase le sommet actuel de la pile de matrices avec la matrice spécifiée.

Syntaxe :

matrix_stack_set(matrix);

ArgumentsType Description
matrix The matrix index to use.

Les retours :

Exemple :

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

Le code ci-dessus construira une nouvelle matrice et stockera l'indice de la matrice résultante dans la variable "m" avant de remplacer le haut de la pile de la matrice par celle-ci.
