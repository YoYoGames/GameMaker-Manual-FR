matrix_build

Cette fonction peut être utilisée pour créer votre propre matrice personnalisée et renvoie un tableau matriciel, qui doit être stocké dans une variable pour référence et utilisation futures. Elle accepte des valeurs de translation, de rotation et d'échelle tridimensionnelles(x, y, z) et les utilise pour construire un tableau matriciel.

Le tableau matriciel contient 16 valeurs au total, dont les 4 premiers éléments sont ligne/colonne 1les 4 éléments suivants sont ligne/colonne 2 et ainsi de suite, dans le cadre d'une matrice 4x4. Le fait que le tableau soit ordonné par rangées ou par colonnes dépend de la plate-forme cible, car le moteur de rendu graphique utilisé par une plate-forme peut utiliser des matrices à majorité de rangées ou à majorité de colonnes.

REMARQUE Lorsque vous construisez une nouvelle matrice de cette manière, l'ordre des opérations est YXZ.

Syntaxe :

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);

ArgumentsType Description
x The x component of the translation vector.
y The y component of the translation vector.
z The z component of the translation vector.
xrotation The angle to rotate around the x-axis (in degrees °).
yrotation The angle to rotate around the y-axis (in degrees °).
zrotation The angle to rotate around the z-axis (in degrees °).
xscale The x scale amount.
yscale The y scale amount.
zscale The z scale amount.

Les retours :

Matrix Array

Exemple :

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

Le code ci-dessus construira une nouvelle matrice transformée et stockera l'indice de la matrice résultante dans la variable "t_matrix".
