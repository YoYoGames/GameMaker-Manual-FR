matrix_stack_push

Cette fonction est utilisée pour pousser une nouvelle matrice au sommet de la pile de matrices actuelle. Elle va d'abord multiplier la matrice donnée avec la matrice actuellement en haut de la pile, puis pousser la matrice résultante vers la pile.

Cette fonction est utile pour appliquer des transformations matricielles multiples à vos visuels sans avoir à multiplier manuellement diverses matrices ensemble.

Par exemple, vous pouvez avoir une matrice de base poussée sur la pile qui dessine des graphiques dans une certaine zone rectangulaire (disons, une fenêtre). Après avoir dessiné quelques graphiques à l'aide de cette matrice, vous pouvez pousser une autre matrice sur la pile pour dessiner des graphiques dans une sous-zone (comme un bouton dans la fenêtre). Après avoir dessiné quelque chose dans cette sous-zone, vous pouvez appeler matrix_stack_pop() pour retirer sa matrice de la pile et continuer à dessiner des images dans la zone principale de la fenêtre.

Syntaxe :

matrix_stack_push(matrix);

ArgumentsType Description
matrix The matrix to push to the stack

Les retours :

Exemple :

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

Le code ci-dessus va construire deux nouvelles matrices et les pousser sur la pile de matrices. La matrice du monde est ensuite placée au sommet de la pile, un sprite est dessiné et les transformations sont ensuite retirées de la pile.
