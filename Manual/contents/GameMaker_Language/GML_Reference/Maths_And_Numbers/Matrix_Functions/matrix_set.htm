matrice_set

Cette fonction prend un tableau de 16 valeurs, correspondant à un type de matrice 4x4 donné, où les éléments [0 - 3] seraient la rangée 1, les éléments [4 -7] seraient la rangée 2, etc... (voir l'image sur la page principale). Vous pouvez créer une telle matrice en utilisant les fonctions matrix_build() ou matrix_get() ou simplement en construisant vous-même le tableau et en le passant à la fonction. Les types de matrice disponibles sont la vue, la projection et le monde, pour lesquels vous utiliserez l'une des constantes suivantes :

Constant Description
matrix_view La matrice de vue actuelle
matrix_projection La matrice de projection actuelle
matrix_world La matrice mondiale actuelle

Syntaxe :

matrix_set(type, matrix);

Arguments Type Description
type Type de matrice Constant The type of matrix to get the values of (see the constants listed above)
matrix Matrice The matrix data as an array

Les retours :

N/A

Exemple :

matrix_set(matrix_world, m_array);

Le code ci-dessus va définir les valeurs de la matrice du monde actuel à celles stockées dans la matrice du tableau "m_array".
