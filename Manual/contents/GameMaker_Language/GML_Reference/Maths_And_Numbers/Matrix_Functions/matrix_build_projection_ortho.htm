matrice_construction_projection_ortho

Cette fonction construit une matrice de projection orthographique basée sur les paramètres spécifiés ci-dessous (il s'agit de la méthode de projection par défaut utilisée lorsque vous créez une pièce dans GameMaker sans rien changer). Il est parfois nécessaire de passer d'une projection en perspective à une projection orthographique, ce à quoi sert cette fonction. Elle est généralement utilisée pour dessiner une superposition, par exemple pour montrer le score ou d'autres aspects, car elle donne une vue "plate" des éléments dessinés (c'est-à-dire sans perspective) dans un jeu en 3D. Voir l'image ci-dessous pour avoir une idée de la différence entre les vues orthographiques et les vues en perspective.

REMARQUE: vous devrez peut-être aussi désactiver temporairement la suppression des surfaces cachées si vous souhaitez que les informations soient dessinées quelle que soit la valeur de la profondeur actuelle.

Syntaxe :

matrix_build_projection_ortho(width, height, znear, zfar);

ArgumentsType Description
w The width of the projection.
h The height of the projection.
znear The near clipping plane.
zfar The far clipping plane.

Les retours :

Matrix Array

Exemple :

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Le code ci-dessus crée une nouvelle matrice d'observation et une matrice orthographique, stocke leurs ID dans des variables locales, puis les utilise pour définir les matrices de vue et de projection pour la caméra affectée au port de vue [0].
