irandom

Cette fonction très utile ne renvoie que des entiers (nombres entiers). Ainsi, par exemple, pour obtenir un nombre aléatoire compris entre 0 et 9, vous pouvez utiliser irandom(9) et elle renverra un nombre compris entre 0 et 9 inclus. Notez que les flottants peuvent également être utilisés, mais la valeur supérieure après le point sera exclue. Ainsi, irandom(9.7) renverra uniquement une valeur comprise entre 0 et 9. La fonction a une limite supérieure de $7fffffffffffffffLL, il faut donc faire attention si vous utilisez de très grands nombres.

NOTE Cette fonction renverra la même valeur à chaque fois que le jeu sera relancé, car GameMaker génère à chaque fois la même graine aléatoire initiale afin de rendre le code debugging beaucoup plus facile. Pour éviter ce comportement, utilisez randomise() au début de votre jeu.

Syntaxe :

irandom(n);

ArgumentsType Description
n The upper range from which the random number will be selected.

Les retours :

Exemple :

if (irandom(9) == 1)
{
    score += 100;
}

Cela donnera une chance sur dix (puisque le 0 est inclus) d'ajouter 100 au score.
