au hasard

Cette fonction est bonne pour les probabilités où il n'est pas nécessaire de renvoyer un entier (nombre entier). Par exemple, random(100) renverra une valeur comprise entre 0 et 99, mais cette valeur peut être 22,56473 ! Vous pouvez également utiliser des nombres réels et non des nombres entiers dans cette fonction, comme par exemple random(0.5), qui renvoie une valeur comprise entre 0 et 0,4999999.

REMARQUE: cette fonction renverra la même valeur à chaque fois que le jeu sera relancé, car GameMaker génère à chaque fois la même graine aléatoire initiale afin de rendre le code debugging beaucoup plus facile. Pour éviter ce comportement, utilisez randomise() au début de votre jeu.

Syntaxe :

random(n);

ArgumentsType Description
n The upper range from which the random number will be selected.

Les retours :

Exemple :

if (random(10) >= 9)
{
    score += 100;
}

Cela donnera environ une chance sur dix d'ajouter 100 au score.
