exp

Il s'agit de la fonction power(e,n), où e est approximativement 2,718281828 (également connu sous le nom de nombre d'Euler), et n est le nombre de fois qu'il doit être multiplié par lui-même. En gros, cela veut dire "quelle est la croissance que j'obtiens après n unités de temps (et une croissance continue de 100%) ? (et une croissance continue de 100%)". Ainsi, exp(3) donnerait 20,08, ce qui signifie qu'après 3 unités de temps, nous avons 20,08 fois le montant avec lequel nous avons commencé.

Syntaxe :

exp(n);

ArgumentsType Description
n The power of e.

Les retours :

Exemple :

val = exp(2);

Cela fixera la valeur à e^2, autour de 7,4.
