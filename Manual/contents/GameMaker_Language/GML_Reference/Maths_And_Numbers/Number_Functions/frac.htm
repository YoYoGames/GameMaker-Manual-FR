frac

Cette fonction renvoie la partie fractionnaire de n, c'est-à-dire la partie derrière le point décimal. Elle ne renvoie que les décimales derrière le point d'une valeur, ainsi frac(3.125) renverra 0,125, frac(6.921) renverra 0,921, etc...

Syntaxe :

frac(n);

ArgumentsType Description
n The number to change.

Les retours :

Exemple :

val = frac(3.4);

La valeur de val sera ainsi fixée à 0,4.
