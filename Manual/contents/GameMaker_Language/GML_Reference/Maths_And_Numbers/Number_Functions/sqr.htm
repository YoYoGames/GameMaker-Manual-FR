sqr

Multiplie un nombre par lui-même et renvoie donc le carré de ce nombre. EG : sqr(5) renverrait 25 puisque 5*5=25.

Syntaxe :

sqr(val);

ArgumentsType Description
val The number to square.

Les retours :

Exemple :

score += sqr(dmg);

Le code ci-dessus ajoutera le carré de la variable "dmg" au score.
