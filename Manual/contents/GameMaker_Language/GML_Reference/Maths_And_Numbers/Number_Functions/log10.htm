log10

Cette fonction est similaire à la fonction log2(n) sauf que log10(n) donne le nombre de 10 qu'il faut multiplier pour obtenir n. Donc log10(100) donne 2, car la multiplication de 10 par lui-même donne 100. 100, log10(1000) renvoie 3 car 10x10x10=1000 etc... Le nombre n'a pas besoin d'être divisible par dix, cependant, mais cela retournera quand même une valeur appropriée, par exemple si nous avons log10(101) nous obtenons 2.00432137... ce qui est combien de fois vous pouvez diviser 101 par 10.

Syntaxe :

log10(n);

ArgumentsType Description
n The input value.

Les retours :

Exemple :

logval = log10(num);

Le code ci-dessus obtient le log10 de la valeur stockée dans "num".
