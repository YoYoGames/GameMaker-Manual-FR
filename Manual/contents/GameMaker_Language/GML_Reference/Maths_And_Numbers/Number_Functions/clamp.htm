pince

Cette fonction vous permet de maintenir une valeur d'entrée entre une plage spécifiée.

Syntaxe :

clamp(val, min, max)

ArgumentsType Description
val The value to clamp.
min The minimum value to clamp between.
max The maximum value to clamp between.

Les retours :

Exemple :

speed = clamp(speed, 1, 10);

Le code ci-dessus bloquera la vitesse de sorte qu'elle ne descende jamais en dessous de 1 ou ne dépasse pas 10.
