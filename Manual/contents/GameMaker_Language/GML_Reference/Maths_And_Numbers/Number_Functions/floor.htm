sol

Renvoie le plancher de n, c'est-à-dire n arrondi à un nombre entier inférieur. Cette fonction est similaire à la fonction round() mais elle n'arrondit que vers le bas, quelle que soit la valeur décimale. Ainsi, floor(5.99999) renverra 5, tout comme floor(5.2), floor(5.6457) etc...

Syntaxe :

floor(n);

ArgumentsType Description
n The number to floor.

Les retours :

Exemple :

val = floor( 3.9 );

La valeur de Val sera ainsi fixée à 3.
