signe

Cette fonction indique si un nombre est positif, négatif ou ni l'un ni l'autre et renvoie respectivement 1, -1, 0. Par exemple - sign(458) renverra 1, sign(-5) renverra -1 et sign(0) renverra 0.

Syntaxe :

sign(n);

ArgumentsType Description
n The number to get the sign of.

Les retours :

Exemple :

y += sign(y - mouse_y);

Le code ci-dessus ajoutera 1, -1 ou 0 à y en fonction du résultat de y - mouse_y.
