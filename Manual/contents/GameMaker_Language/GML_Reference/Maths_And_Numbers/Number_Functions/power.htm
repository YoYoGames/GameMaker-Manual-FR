puissance

Cette fonction renvoie la valeur d'un nombre multiplié par lui-même un nombre "n" de fois. Par exemple, power(5,3) multipliera 5 par lui-même 3 fois et renverra 125, ce qui revient à dire 5*5*5=125. Veuillez noter que la valeur "x (le nombre à modifier) ne peut pas être une valeur négative.

Syntaxe :

power(x, n);

ArgumentsType Description
x The number to change.
n How many times to multiply x by itself.

Les retours :

Exemple :

score += power(dmg, 3);

Cela ajoutera la valeur tenue dans la variable "dmg" à la puissance 3 à la variable score.
