gamme_aléatoire

Cette fonction renvoie un nombre aléatoire compris dans l'intervalle spécifié, et cette valeur de retour ne doit pas nécessairement être un nombre entier. Par exemple, random_range(20,50) renverra un nombre aléatoire compris entre 20 et 50, mais la valeur peut être un nombre réel comme 38,65265. Les nombres réels peuvent également être utilisés comme arguments d'entrée.

REMARQUE: cette fonction renverra la même valeur à chaque fois que le jeu sera relancé, car GameMaker génère à chaque fois la même graine aléatoire initiale afin de rendre le code debugging beaucoup plus facile. Pour éviter ce comportement, utilisez randomise() au début de votre jeu.

Syntaxe :

random_range(n1, n2);

ArgumentsType Description
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

Les retours :

Exemple :

score += random_range(500, 600);

Cela ajoutera entre 500 et 600 points au score total.
