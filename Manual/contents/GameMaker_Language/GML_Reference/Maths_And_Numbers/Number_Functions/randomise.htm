randomiser

Cette fonction définit la graine à une valeur aléatoire. Si vous avez besoin de conserver une valeur constante sur un certain nombre d'exécutions d'un jeu, vous devriez utiliser la fonction random_set_seed(). Veuillez noter que lorsque vous utilisez les fonctions de nombres aléatoires dans GameMaker, la graine initiale est toujours la même, ce qui facilite grandement le dépistage des erreurs et le site debugging. Si vous souhaitez tester avec un véritable aléa, vous devez appeler cette fonction au début de votre jeu. La fonction renvoie la nouvelle valeur de la graine aléatoire (un nombre entier 32 bits non signé).

Syntaxe :

randomise();
// or
randomize();

Les retours :

(unsigned 32 bit value)

Exemple :

randomise();

Le code ci-dessus va randomiser la graine.
