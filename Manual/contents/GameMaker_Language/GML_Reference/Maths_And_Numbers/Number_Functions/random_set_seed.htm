random_set_seed

Pour générer un nombre aléatoire, GameMaker commence par un nombre de départ aléatoire. Avec cette fonction, vous pouvez définir cette graine à une valeur connue et ainsi "forcer" le résultat de tous les événements aléatoires ultérieurs à être le même à chaque exécution du programme. Par exemple, cette fonction peut être utilisée conjointement avec la fonction random_get_seed() pour créer du contenu généré de manière procédurale et sauvegarder les résultats sans avoir besoin d'énormes sauvegardes (vous ne sauvegardez que la graine, pas besoin d'autre chose). Si vous avez besoin de résultats vraiment aléatoires pour tout, vous devriez utiliser la fonction randomise() fonction.

REMARQUE Bien que cette graine donne des résultats cohérents sur chaque plateforme cible, les résultats peuvent varier d'une plateforme à l'autre en raison du fonctionnement différent de chaque cible.

Syntaxe :

random_set_seed(val);

ArgumentsType Description
val The seed to set.

Les retours :

N/A

Exemple :

if debug
{
    random_set_seed(1);
}

Le code ci-dessus définit la graine aléatoire à 1 seulement si la variable "debug" est vraie.
