log2

Cette fonction demande essentiellement "combien de 2 devez-vous multiplier pour obtenir n ?". Un exemple courant d'utilisation en programmation serait de calculer le nombre de bits nécessaires pour représenter un nombre. Pour ce faire, nous calculons le log2(x) du nombre, l'arrondissons au chiffre inférieur et ajoutons 1. Par exemple, log2(100) renvoie 6,643856, ce qui, arrondi au chiffre inférieur, donne 6, auquel nous ajoutons 1 et qui nous donne 7. Nous avons donc besoin de 7 bits pour représenter 100. Il peut également être utilisé pour décrire une croissance ou une décroissance exponentielle. exponentielle, et est souvent appelé logarithme binaire.

Syntaxe :

log2(n);

ArgumentsType Description
n The input value.

Les retours :

Exemple :

colourbits = floor(log2(colour)) - 1;

Le code ci-dessus stocke le nombre de bits nécessaires pour représenter la valeur stockée dans la variable "couleur".
