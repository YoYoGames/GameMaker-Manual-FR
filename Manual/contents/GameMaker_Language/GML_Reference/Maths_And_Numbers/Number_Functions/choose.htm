choisissez

Parfois, vous voulez spécifier autre chose que des nombres pour une sélection aléatoire, ou les nombres que vous voulez ne sont pas dans un ordre réel ou dans une plage définie. Dans ces cas, vous utiliserez choose() pour générer un résultat aléatoire. Par exemple, si vous voulez créer un objet avec un sprite aléatoire au début, vous pouvez utiliser cette fonction pour définir l'index sprite sur l'un des sprites donnés. Notez que vous pouvez avoir autant d'arguments que vous le souhaitez (notez que plus d'arguments signifie que la fonction sera plus lente à analyser).

REMARQUE Cette fonction renverra la même valeur à chaque fois que le jeu sera relancé, car GameMaker génère à chaque fois la même graine aléatoire initiale afin de rendre le code debugging beaucoup plus facile. Pour éviter ce comportement, utilisez randomise() au début de votre jeu.

Syntaxe :

choose(val0, val1, val2... max_val);

Arguments Type Description
val0... max_val Variable Any type of value(s).

Les retours :

Variable (One of the given arguments)

Exemple :

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

Le code ci-dessus utilise le choix pour définir un certain nombre de propriétés pour l'instance.
