plafond

Cette fonction prend un nombre réel quelconque et l'arrondit à l'entier le plus proche. Il convient d'être prudent avec cette fonction car une erreur fréquente consiste à l'utiliser pour arrondir une valeur aléatoire et à s'attendre à ce qu'elle soit toujours supérieure à 1, par exemple :

int = ceil(random(5));

On pourrait s'attendre à ce que ce code donne toujours un nombre entier compris entre 1 et 5, mais ce n'est pas toujours le cas, car il y a une très faible possibilité que la fonction aléatoire renvoie 0, et l'arrondi à 0 donne toujours 0. Il s'agit d'une faible mais doit être prise en compte lors de l'utilisation de cette fonction.

Syntaxe :

ceil(val);

ArgumentsType Description
x The number to change.

Les retours :

Exemple :

val = ceil( 3.4 );

La valeur de Val sera ainsi fixée à 4.
