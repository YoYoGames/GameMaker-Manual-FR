ln

Le logarithme naturel ln(n) est la quantité de temps nécessaire pour atteindre un certain niveau de croissance continue, où n est le niveau atteint. Ainsi, si nous voulons savoir de combien d'unités de temps nous avons besoin pour obtenir une croissance de 20, nous utiliserons ln(20). qui renvoie 2,99 unités de temps pour obtenir cette quantité de croissance.

Syntaxe :

ln(n);

ArgumentsType Description
n The input value.

Les retours :

Exemple :

alarm[0] = ln(age) * room_speed;

Le code ci-dessus utilise le logarithme naturel de la valeur stockée dans la variable "age" pour déclencher une alarme.
