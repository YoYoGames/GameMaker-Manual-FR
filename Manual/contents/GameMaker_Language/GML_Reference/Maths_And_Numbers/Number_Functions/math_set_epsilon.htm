math_set_epsilon

Epsilon est une valeur utilisée pour déterminer si deux nombres sujets à une erreur d'arrondi sont suffisamment proches pour être considérés comme "égaux". Elle est utile lorsqu'on traite des mathématiques à virgule flottante, car elle permet de réduire les "erreurs d'arrondi" qui font que certaines opérations renvoient des valeurs qui semblent incorrectes ou contraires à ce que l'on attend. Par exemple, nous pouvons avoir ajouté une valeur à l'index de l'image d'un site sprite et nous attendre à ce que le résultat soit un entier unique, mais en raison de la nature des mathématiques à virgule flottante, la valeur finale réelle finit par être quelque chose comme 5.0000002, donc lorsque nous avons le contrôle suivant :

if image_index == 5
{
    //do something
}

Le code ne se comporte pas comme prévu et renvoie false. Toutefois, si nous définissons la valeur d'epsilon à 0,000001, la valeur image_index sera arrondie au nombre réel le plus proche de +/- 0,000001 de la valeur d'origine, ce qui fait que la comparaison ci-dessus renvoie vrai.

La valeur epsilon sera utilisée pour tous les opérateurs entiers suivants :

  • < : Moins de
  • > : Supérieur à
  • == : Égal à
  • <= : Inférieur ou égal à
  • >= : Supérieur ou égal à
  • != : Non égal à

Notez que la définition d'une valeur d'epsilon de 0 désactivera tous les arrondis, et l'utilisation d'une valeur de 1 donnera une erreur.

Syntaxe :

math_set_epsilon(epsilon);

Arguments Type Description
epsilon Real The new epsilon value (from 0 to 0.999999999).

Les retours :

Real

Exemple :

math_set_epsilon(0.0001);

Ceci définira la valeur d'epsilon pour toutes les autres opérations en virgule flottante.
