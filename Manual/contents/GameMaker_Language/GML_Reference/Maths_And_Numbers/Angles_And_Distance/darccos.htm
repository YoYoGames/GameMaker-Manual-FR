darccos

Renvoie le cosinus inverse de x, c'est-à-dire que si dcos(val)=n, darccos(n)=val, et le nombre résultant sera compris entre 180 et 0.

REMARQUE: Cette fonction n'accepte qu'un nombre compris entre -1 et 1 (toute autre valeur entraînera une erreur).

NOTE: Cette fonction renvoie une valeur en degrés, et non en radians.

Syntaxe :

darccos(val)

ArgumentsType Description
x The value to return the inverse cosine of.

Les retours :

Exemple :

val = arccos(-1);

La valeur de "val" sera ainsi fixée à 180.
