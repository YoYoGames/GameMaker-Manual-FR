arctan

Renvoie la tangente inverse de x. Cette fonction accepte n'importe quel nombre car, contrairement à tan(), les asymptotes de arctan() sont sur l'axe des y, ce qui signifie simplement que vous ne recevrez jamais un nombre supérieur à pi/2 ou inférieur à -pi/2.

NOTE: Cette fonction prend une valeur en radians, et non en degrés.

Syntaxe :

arctan(x);

ArgumentsType Description
x The angle (in radians) to return the inverse tangent of.

Les retours :

Exemple :

val = arctan( 0 );

Ceci mettra val à 0.
