radtodeg

Une fois que vous avez effectué vos calculs en utilisant sin, ou cos etc... le résultat est en radians. Ce n'est pas toujours ce que vous voulez et donc pour transformer les radians en degrés, nous utilisons cette fonction. Par exemple, radtodeg(sin(180)) renverra -45 degrés. Cette fonction convertit les radians en degrés en utilisant la formule :

angle_degrees = angle_radians * 180 / pi;

Syntaxe :

radtodeg(rad);

ArgumentsType Description
rad The radians to convert.

Les retours :

Exemple :

val = radtodeg( pi );

La valeur de Val sera ainsi fixée à 180.
