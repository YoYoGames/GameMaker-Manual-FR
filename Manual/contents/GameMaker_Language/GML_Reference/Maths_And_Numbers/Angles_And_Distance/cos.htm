cos

Dans un triangle à angle droit, cos(val) = Adjacent / Hypoténuse où val est l'un des trois angles. Cette fonction renvoie toujours un nombre compris entre 1 et -1.

NOTE: Cette fonction prend une valeur en radians et non en degrés.

Syntaxe :

cos(val);

ArgumentsType Description
val The angle (in radians) to return the cosine of.

Les retours :

Exemple :

val = cos( 0 );

Ceci mettra val à 1.
