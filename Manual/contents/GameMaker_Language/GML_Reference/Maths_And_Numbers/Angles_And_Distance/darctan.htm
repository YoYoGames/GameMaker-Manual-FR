darctan

Renvoie la tangente inverse de x. Cette fonction acceptera n'importe quel nombre car, contrairement à dtan() darctan(), les asymptotes sont sur l'axe des y, ce qui signifie simplement que vous ne recevrez jamais un nombre supérieur à 90 ou inférieur à -90.

NOTE: Cette fonction renvoie une valeur en degrés, et non en radians.

Syntaxe :

darctan(x);

ArgumentsType Description
val The value to return the inverse tangent of.

Les retours :

Exemple :

val = darctan(1);

La valeur de "val" sera ainsi fixée à -45.
