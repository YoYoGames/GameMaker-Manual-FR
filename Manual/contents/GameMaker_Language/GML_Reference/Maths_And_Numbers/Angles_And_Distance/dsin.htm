dsin

Dans un triangle rectangle, dsin(val) = Opposé / Hypoténuse où "val" est l'un des trois angles.

NOTE: Cette fonction prend une valeur en degrés et non en radians.

Syntaxe :

dsin(val);

ArgumentsType Description
val The angle (in degrees) to return the sine of.

Les retours :

Exemple :

val = dsin(90);

Ceci mettra "val" à 1.
