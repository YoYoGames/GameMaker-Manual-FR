dcos

Dans un triangle à angle droit, cos(val) = Adjacent / Hypoténuse où val est l'un des trois angles. Cette fonction renvoie toujours un nombre compris entre 1 et -1.

NOTE: Cette fonction prend une valeur en degrés, pas en radians.

Syntaxe :

dcos(val);

ArgumentsType Description
val The angle (in degrees) to return the cosine of.

Les retours :

Exemple :

val = dcos(45);

La valeur de "val" sera ainsi fixée à 0,71.

Retour : Angles et distances
