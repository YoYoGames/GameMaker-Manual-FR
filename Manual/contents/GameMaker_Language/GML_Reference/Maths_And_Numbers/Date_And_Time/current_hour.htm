heure_courante

Cette variable en lecture seule renverra l'heure qui correspond au moment actuel sur la base du fuseau horaire par défaut du système (c'est-à-dire l'heure locale). Vous pouvez modifier le fuseau horaire de base à utiliser avec la fonction date_set_timezone().

Syntaxe :

current_hour;

Les retours :

Exemple :

draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

Le code ci-dessus affiche l'heure internationale actuelle à l'écran.
