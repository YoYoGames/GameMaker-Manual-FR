mois_courant

Cette variable en lecture seule renvoie le mois en cours sous forme de valeur numérique où 1 correspond à janvier et 12 à décembre.

Syntaxe :

current_month;

Les retours :

Exemple :

draw_text((32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

Le code ci-dessus affichera le jour, le mois et l'année.
