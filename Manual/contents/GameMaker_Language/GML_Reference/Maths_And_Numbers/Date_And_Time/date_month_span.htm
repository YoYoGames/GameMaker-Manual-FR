date_month_span

Cette fonction permet d'obtenir le nombre de mois entre deux dates. Cette valeur est toujours positive, et les mois incomplets seront retournés sous forme de fraction.

Syntaxe :

date_month_span(date1, date2);

ArgumentsType Description
date1 The first datetime.
date2 The datetime to compare it to.

Les retours :

Exemple :

diff = date_month_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Ainsi, "diff" correspondrait au nombre de mois entre le 15 septembre 2011, 11:04.0 et la date et l'heure actuelles.
