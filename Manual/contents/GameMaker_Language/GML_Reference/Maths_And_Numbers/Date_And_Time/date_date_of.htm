date_date_de

Renvoie la valeur de la date de l'heure donnée.

Syntaxe :

date_date_of(date);

ArgumentsType Description
date The datetime to extract the date from.

Les retours :

Exemple :

today = date_date_of(date_current_datetime());

Cela renvoie uniquement la date du jour et stocke la valeur dans la variable "today".
