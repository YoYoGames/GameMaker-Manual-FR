date_date_chaîne

Avec cette fonction, vous pouvez créer une chaîne de caractères contenant la date donnée, formatée en jour/mois/année.

Syntaxe :

date_date_string(date);

ArgumentsType Description
date The date to use.

Les retours :

Exemple :

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

Ainsi, "str" contient une chaîne formatée de la date et de l'heure actuelles telles qu'elles sont affichées par le système.
