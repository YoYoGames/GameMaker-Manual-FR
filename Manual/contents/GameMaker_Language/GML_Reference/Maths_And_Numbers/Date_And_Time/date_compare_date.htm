date_compare_date

Avec cette fonction, vous pouvez vérifier deux dates pour voir laquelle est antérieure ou postérieure à l'autre. La fonction renvoie -1 si la date1 est antérieure, 0 si les deux dates sont identiques, et 1 si la date1 est postérieure.

Syntaxe :

date_compare_date( date1, date2 );

Arguments Type Description
date1 Datetime The first date.
date2 Datetime The date to compare it to.

Les retours :

Real

Exemple :

d = date_compare_date(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime());

Cela donnerait à "d" la valeur correspondante en fonction de la date la plus ancienne, probablement 1 puisque la date actuelle est plus éloignée que le 15 septembre 2011.
