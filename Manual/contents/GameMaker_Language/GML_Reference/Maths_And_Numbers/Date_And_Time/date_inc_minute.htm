date_inc_minute

Avec cette fonction, vous pouvez incrémenter une valeur de date donnée d'un nombre spécifique de minutes, et elle renverra la nouvelle valeur de date.

Syntaxe :

date_inc_minute( date, amount );

ArgumentsType Description
date The datetime to add to.
amount The number of minutes (must be an integer) to add.

Les retours :

Exemple :

mynewdatetime = date_inc_minute(date_current_datetime(), 60);

Cela définirait "mynewdatetime" à la date actuelle mais avec 60 minutes ajoutées.
