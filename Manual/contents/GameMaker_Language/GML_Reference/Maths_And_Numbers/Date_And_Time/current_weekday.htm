jour de la semaine en cours

Cette variable en lecture seule renvoie le jour de la semaine sous forme de valeur, où le dimanche est égal à 0 et le samedi à 6.

Syntaxe :

current_weekday;

Les retours :

Exemple :

var day;
switch(current_weekday)
{
    case 0: day = "Sunday"; break;
    case 1: day = "Monday"; break;
    case 2: day = "Tuesday"; break;
    case 3: day = "Wednesday"; break;
    case 4: day = "Thursday"; break;
    case 5: day = "Friday"; break;
    case 6: day = "Saturday"; break;
}
draw_text(32, 32, "Today is " + day +".");

Le code ci-dessus utilise la valeur current_weekday pour définir une variable avec le jour correct en texte, puis le dessine pour que l'utilisateur puisse le voir.
