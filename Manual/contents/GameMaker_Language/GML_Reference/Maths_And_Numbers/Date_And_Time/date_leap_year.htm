date_année bissextile

Cette fonction renvoie true si le composant année de la valeur datetime donnée est une année bissextile ou false sinon. Cette fonction peut s'avérer pratique pour des choses comme les œufs de Pâques dans vos jeux, ou pour débloquer du contenu spécial.

Syntaxe :

date_leap_year(date);

ArgumentsType Description
date The datetime to use.

Les retours :

Exemple :

if date_leap_year(date_current_datetime())
{
    if (!global.ExtraContent)
    {
        global.ExtraContent = true;
    }
}

Le code ci-dessus va vérifier la date actuelle pour voir si l'année est bissextile ou non. Si c'est le cas, il définit une variable globale.
