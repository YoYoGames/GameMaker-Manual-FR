Facebook

Facebook est l'un des plus importants sites de médias sociaux sur Internet et de nombreux jeux et développeurs ont actuellement leurs propres pages pour publier des jeux et des médias. GameMaker possède une extension qui vous permet de vous connecter à Facebook et d'effectuer certaines actions et tâches que leur API permet. Il est recommandé de visiter la page des développeurs Facebook avant de commencer à utiliser l'extension, car une bonne compréhension du fonctionnement du graphique Facebook et des appels et autorisations disponibles est essentielle pour tirer le meilleur parti de ces fonctions.

Vous pouvez découvrir comment configurer et utiliser l'extension Facebook pour toutes les plateformes disponibles à partir du lien suivant :

  • Android, iOS et HTML5 : Intégration de Facebook

Les fonctions suivantes sont obsolètes et n'étaient valables que pour l'ancienne API Facebook. Vous ne pouvez donc plus utiliser ces fonctions dans vos projets. Elles sont répertoriées ici au cas où vous auriez importé un ancien projet les contenant. Pour la fonctionnalité Facebook actuelle, vous devez avoir activé l'API Facebook dans les options de jeu appropriées pour la plate-forme cible, ce qui vous invitera à télécharger et à installer une extension. L'extension est fournie avec un "mini-manuel" au format *.pdf expliquant comment l'utiliser et montrant toutes les définitions de fonctions et les exemples. Toutes les références aux anciennes fonctions facebook_*() devront être remplacées (et mises à jour) pour utiliser les équivalents de l'extension fb_*().

Les fonctions énumérées ci-dessous ont permis d'intégrer votre jeu à Facebook :

  • facebook_init
  • facebook_login
  • facebook_request_publish_permissions
  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
