shop_leave_rating

Cette fonction ouvre une boîte de dialogue dépendant du système d'exploitation dans laquelle vous pouvez demander à l'utilisateur de publier une évaluation ou un commentaire sur une page particulière. Vous pouvez définir le texte qui doit apparaître dans la boîte de dialogue, ainsi que le texte que vous souhaitez voir apparaître sur les deux boutons et l'URL où le commentaire doit être posté.

REMARQUE: Cette fonction ne concerne que les cibles Android et iOS.

IMPORTANT ! Sur iOS, lorsque vous utilisez TestFlight, l'invite s'affiche pendant le développement mais vous ne pouvez rien faire. Une fois que votre application est dans l'App Store, ce bouton devient actif et l'utilisateur a la possibilité d'écrire une critique ou de laisser une note. Notez que si vous distribuez le jeu par le biais de TestFlight, cette fenêtre peut même ne pas s'afficher.

Syntaxe :

shop_leave_rating(text, yes_string, no_string, url)

Arguments Type Description
text Chaîne de caractères Text that appears on the dialog.
yes_string Chaîne de caractères Text that appears on the "yes" button.
no_string Chaîne de caractères Text that appears on the "no" button.
url Chaîne de caractères The full URL where the comment has to be sent.

Les retours :

N/A

Exemple :

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://macsweeneygames.com/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

Le code ci-dessus demandera à l'utilisateur de laisser un commentaire si la variable "timer" a décompté jusqu'à 0.
