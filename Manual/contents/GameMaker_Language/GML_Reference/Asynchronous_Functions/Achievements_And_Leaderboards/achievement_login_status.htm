statut d'identification de l'accomplissement

Cette fonction n'est actuellement utile que pour Google Play sur la plateforme Android. Elle vérifiera si l'utilisateur est déjà connecté aux services Google, auquel cas elle renverra true, ou non, renvoyant false. Si elle renvoie vrai, il n'est pas nécessaire d'appeler la achievement_login() fonction.

Syntaxe :

achievement_login_status();

Les retours :

Boolean

Exemple :

if !achievement_login_status()
    {
    achievement_login();
    }

Le code ci-dessus vous permettra de vous connecter au service de classement et d'accomplissement choisi.
