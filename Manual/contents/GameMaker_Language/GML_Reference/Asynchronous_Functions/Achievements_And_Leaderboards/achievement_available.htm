disponible

Cette fonction renvoie un message vrai si l'utilisateur est actuellement connecté à Internet et si le système de classement et de réussite choisi est disponible, sinon elle renvoie un message faux.

Syntaxe :

achievement_available()

Les retours :

Boolean

Exemple :

if achievement_available()
    {
    achievement_post("ach_Level1", 100);
    }

Le code ci-dessus vérifiera que le service de classement est disponible et, si c'est le cas, il l'affichera.
