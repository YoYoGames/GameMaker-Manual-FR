gamepad_is_connected

Cette fonction indique si une manette de jeu est connectée au "slot" donné (renvoie true) ou non (renvoie false). Vous devez normalement utiliser cette fonction en conjonction avec la fonction gamepad_get_device_count() pour obtenir le nombre correct de manettes de jeu et/ou de "slots" disponibles. Notez qu'il peut y avoir un léger délai entre le moment où l'utilisateur connecte la manette de jeu et celui où GameMaker la détecte comme étant connectée (c'est particulièrement le cas lorsqu'il s'agit de manettes connectées par Bluetooth).

REMARQUE: Sur certaines plates-formes - en particulier les consoles - cette fonction peut renvoyer un faux immédiatement après qu'un gamepad ait été connecté/sélectionné et peut devoir être vérifiée dans une alarme quelques images (étapes) plus tard pour qu'elle détecte correctement le gamepad.

Syntaxe :

gamepad_is_connected(numb);

ArgumentsType Description
device Which gamepad "slot" to check.

Les retours :

Exemple :

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++)
{
    if gamepad_is_connected(i)
        global.gp[i] = true
    else
        global.gp[i] = false;
}

Le code ci-dessus passe en boucle les game pads (ou emplacements de gamepad) disponibles, puis vérifie pour chacun d'entre eux s'il existe un gamepad connecté. La valeur renvoyée est ensuite utilisée pour définir un tableau global à true ou false, qui sera utilisé lors des prochaines vérifications.
