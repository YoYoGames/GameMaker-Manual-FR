gamepad_axis_value

Vous pouvez utiliser cette fonction pour obtenir la valeur des différents axes d'un slot de gamepad donné. Vous fournissez l'index du slot de la manette à vérifier, ainsi qu'une constante d'index d'axe (telle que listée ici), ou une valeur entière entre 0 et gamepad_axis_count()-1. La valeur renvoyée sera comprise entre -1 et 1 pour chacun des axes horizontaux et verticaux disponibles.

Notez que la valeur retournée est normalisée pour prendre en compte la valeur définie pour la zone morte de l'axe (en utilisant la fonction gamepad_set_axis_deadzone()), et sera donc toujours comprise entre -1 et 0, et 0 et 1. Par exemple, en définissant la zone morte à 0.2 signifie que le fait de pousser le manche vers la droite ne commencera à renvoyer une valeur comprise entre 0 et 1 que lorsque la valeur brute de l'axe sera supérieure à 0,2. Par exemple, lorsque la valeur brute est de 0,2, la valeur renvoyée sera 0, lorsque la valeur brute est de 0,5, la valeur renvoyée sera 0,375, ou si la valeur brute est de 0,9, la valeur renvoyée sera 0,875.

Syntaxe :

gamepad_axis_value(device, axisIndex);

Arguments Type Description
device Real Which gamepad device "slot" to check.
axisIndex Constante d'axe de la manette de jeu The axis index to check (see the constants list).

Les retours :

Real

Exemple :

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Le code ci-dessus définit la vitesse et la direction de l'instance en fonction des valeurs renvoyées par le contrôleur analogique gauche de la manette de jeu branchée sur le périphérique "slot" 0.
