gamepad_get_button_threshold

Cette fonction peut être utilisée pour détecter le paramètre de seuil actuel des boutons analogiques pour un appareil donné. Le seuil par défaut pour tous les boutons analogiques est de 0,5, avec une plage de 0 à 1. Le seuil définit à quel moment le bouton est considéré comme "enfoncé" pour les jeux qui exigent qu'il agisse comme un bouton numérique.

Syntaxe :

gamepad_get_button_threshold(device);

ArgumentsType Description
device Which gamepad device "slot" to check.

Les retours :

Exemple :

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Le code ci-dessus vérifie le seuil du bouton analogique de la manette de jeu connectée au périphérique "slot" 0 et si ce n'est pas la valeur par défaut de 0,5, il est réglé sur cette valeur.
