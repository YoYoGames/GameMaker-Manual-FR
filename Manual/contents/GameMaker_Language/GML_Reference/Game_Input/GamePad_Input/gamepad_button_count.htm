nombre de boutons de la manette

Cette fonction renvoie le nombre total de boutons disponibles pour la manette de jeu connectée au périphérique "slot" donné.

Syntaxe :

gamepad_button_count(device);

ArgumentsType Description
device Which gamepad device "slot" to check.

Les retours :

Exemple :

b_num = gamepad_button_count(0);

Le code ci-dessus renvoie le nombre de boutons disponibles sur la manette de jeu branchée sur le périphérique "slot" 0 et stocke la valeur dans la variable "b_num".
