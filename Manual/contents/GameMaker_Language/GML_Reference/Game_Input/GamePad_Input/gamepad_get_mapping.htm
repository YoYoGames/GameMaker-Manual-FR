gamepad_get_mapping

Cette fonction peut être utilisée pour récupérer la chaîne de mappage pour le gamepad. Cette chaîne sera soit :

  • La chaîne de la carte actuelle pour le gamepad, par exemple :

    "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

  • Une chaîne avec "no mapping" s'il n'y a pas de mapping défini (ce n' est pas la même chose que lorsqu'aucun mapping n'est disponible)
  • Une chaîne avec "device index out of range" si l'indice de l'emplacement n'est pas un indice de manette de jeu valide.
  • Une chaîne vide "" si aucun mappage n'est disponible pour la manette de jeu connectée.

Pour plus d'informations sur le formatage de la chaîne de caractères retournée, veuillez consulter la fonction gamepad_test_mapping().

Syntaxe :

gamepad_get_mapping(index);

ArgumentsType Description
index Which gamepad index "slot" to get the mapping from.

Les retours :

Exemple :

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

Le code ci-dessus récupère la chaîne de caractères de la carte pour le slot de gamepad donné, puis l'envoie à la console pour debugging.

Retour : Entrée de la manette de jeu
