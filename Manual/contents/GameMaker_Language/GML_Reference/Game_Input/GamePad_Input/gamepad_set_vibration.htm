gamepad_set_vibration

Cette fonction permet de régler la vibration des moteurs du gamepad, chaque moteur ayant une valeur comprise entre 0 (aucune vibration) et 1 (vibration complète). Notez qu'il n'y a pas de limite de temps pour cette fonction, vous devrez donc utiliser une variable ou une alarme pour arrêter la vibration (mettre les moteurs à 0) après un certain temps, sinon le gamepad continuera à vibrer indéfiniment.

REMARQUE: Cette fonction n'est actuellement disponible que pour les modules cibles standard Windows, PS4 et Xbox One.

Syntaxe :

gamepad_set_vibration(device, left_motor, right_motor);

ArgumentsType Description
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

Les retours :

Exemple :

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

Le code ci-dessus serait utilisé (par exemple) dans un événement de collision pour faire vibrer pendant une demi-seconde la manette de jeu branchée sur le "slot" 0, l'alarme définie étant utilisée pour la désactiver une fois ce temps écoulé.
