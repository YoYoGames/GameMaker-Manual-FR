gamepad_get_axis_deadzone

Cette fonction peut être utilisée pour obtenir la valeur de la "zone morte" de l'axe du joystick. Vous spécifiez l'emplacement du dispositif à obtenir, et la fonction renvoie une valeur comprise entre 0 et 1, où la valeur reflète le seuil en dessous duquel l'axe du joystick est considéré comme étant à 0.

Syntaxe :

gamepad_get_axis_deadzone(device);

ArgumentsType Description
device Which gamepad device "slot" to check.

Les retours :

Exemple :

if gamepad_get_axis_deadzone(global.PadId) != 0.5
{
    gamepad_set_axis_deadzone(global.PadId, 0.5);
}

Le code ci-dessus vérifie le seuil de l'axe analogique de la manette de jeu connectée au périphérique "slot" stocké dans la variable globale et s'il ne s'agit pas de la valeur par défaut de 0,5, il est défini à cette valeur.
