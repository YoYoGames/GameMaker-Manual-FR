gamepad_set_colour

Cette fonction peut être utilisée pour définir la couleur des LEDs dans un contrôleur PlayStation. Vous spécifiez l'emplacement du périphérique à définir, puis vous donnez une couleur, qui peut être l'une des constantes de couleur ou une valeur de couleur créée à l'aide des fonctions de couleur spécifiques ou une valeur HEX (comme $FFFFFFF).

Syntaxe :

gamepad_set_colour(device, colour);

ArgumentsType Description
device Which gamepad device "slot" to set.
colour The colour to use.

Les retours :

Exemple :

if health < 10
{
    gamepad_set_colour(0, c_red);
}

Le code ci-dessus fera passer les voyants du contrôleur PlayStation au rouge si la variable health descend en dessous de 10.
