gamepad_button_check_released

Cette fonction renvoie true ou false selon que le bouton du gamepad est détecté comme ayant été relâché ou non. Notez que cette fonction ne se déclenchera qu'une seule fois pour le bouton au moment où il a été relâché. Pour qu'elle se déclenche à nouveau, le bouton doit d'abord être enfoncé, puis relâché une nouvelle fois. Si vous vérifiez un bouton analogique, la vérification prendra en compte le paramètre de seuil et ne renverra true que si la valeur d'entrée brute est inférieure au seuil donné (vous pouvez obtenir cette valeur brute à l'aide de la fonction gamepad_button_value()).

Syntaxe :

gamepad_button_check_released(device, button);

Arguments Type Description
device Real Which gamepad device "slot" to check.
button Bouton de la manette de jeu constant Which gamepad button constant to check for.

Les retours :

Boolean

Exemple :

if gamepad_button_check_released(0, gp_select)
{
    audio_play_sound(snd_Button, 0, false);
    global.Pause = !global.Pause;
}

Le code ci-dessus détectera si le bouton "select" de la manette de jeu connectée au périphérique "slot" 0 a été pressé ou non et basculera la variable globale "Pause".
