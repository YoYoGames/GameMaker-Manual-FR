valeur_bouton_de_jeu

Avec cette fonction, vous pouvez obtenir la valeur actuelle d'un bouton analogique, de 0 à 1, où 0 correspond à une pression nulle et 1 à une pression totale. Vous fournissez l'index du slot du gamepad à vérifier, ainsi qu'une constante de bouton (telle que listée ici), ou une valeur entière entre 0 et gamepad_button_count() - 1.

Notez que cela renvoie la valeur brute du bouton, et ne prend pas en compte le réglage du seuil (voir ici pour plus d'informations sur les seuils).

Syntaxe :

gamepad_button_value(device, button);

Arguments Type Description
device Real Which gamepad device "slot" to check.
button Bouton de la manette de jeu constant Which gamepad button constant to check for.

Les retours :

Real

Exemple :

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

Le code ci-dessus utilise la valeur du trigger analogique de la manette de jeu branchée sur le périphérique "slot" 0 pour définir la vitesse de l'instance.
