vérification du bouton du gamepad

Cette fonction renvoie true ou false selon que le bouton de la manette de jeu est détecté comme étant maintenu enfoncé ou non. Si vous vérifiez un bouton analogique, la vérification prendra en compte le paramètre de seuil et ne retournera true que si la valeur d'entrée brute est supérieure au seuil donné (vous pouvez obtenir cette valeur brute à l'aide de la fonction gamepad_button_value()).

Syntaxe :

gamepad_button_check(device, button);

Arguments Type Description
device Real Which gamepad device "slot" to check.
button Bouton de la manette de jeu constant Which gamepad button constant to check for.

Les retours :

Boolean

Exemple :

if gamepad_button_check(0, gp_face1)
{
    if (canshoot = true)
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

Le code ci-dessus détectera si le bouton inférieur de la face supérieure (le "X" sur une manette ps3) est maintenu enfoncé et si c'est le cas, il tirera une instance "balle" et déclenchera une alarme.
