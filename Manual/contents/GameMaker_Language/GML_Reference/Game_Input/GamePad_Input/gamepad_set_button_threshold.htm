gamepad_set_button_threshold

Cette fonction permet de définir le réglage actuel du seuil des boutons analogiques pour un dispositif donné. Le seuil par défaut pour tous les boutons analogiques est de 0,5, la plage étant de 0 à 1.

Le seuil définit à quel moment le bouton est considéré comme "enfoncé" pour les jeux qui exigent qu'ils agissent comme un bouton numérique.

Cette fonction affectera les états de vérification, d'appui et de relâchement des boutons analogiques, mais n' affectera pas la valeur renvoyée par la fonction gamepad_button_value()qui renvoie toujours la valeur brute du bouton.

Syntaxe :

gamepad_set_button_threshold(device, threshold);

Arguments Type Description
device Real Which gamepad device "slot" to check.
threshold Real The new threshold value (from 0 - 1, default 0.5).

Les retours :

N/A

Exemple :

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Le code ci-dessus vérifie le seuil du bouton analogique de la manette de jeu connectée au périphérique "slot" 0 et si ce n'est pas la valeur par défaut de 0,5, il est réglé sur cette valeur.
