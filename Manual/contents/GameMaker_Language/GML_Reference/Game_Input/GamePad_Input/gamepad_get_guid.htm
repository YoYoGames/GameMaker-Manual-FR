gamepad_get_guid

Avec cette fonction, vous pouvez récupérer le GUID de la manette de jeu connectée à l'index de slot donné. Vous fournissez le slot du gamepad à vérifier (de 0 à 11), et la fonction retournera l'une des chaînes suivantes :

  • "none" si aucun GUID n'est disponible ou si aucun gamepad n'est connecté
  • "device index out of range" si l'index du slot de la manette de jeu est hors de portée.
  • La chaîne GUID, quelque chose comme "050000005e040000fd020000ffff3f00", si la fente a une manette de jeu assignée et peut obtenir le GUID pour celle-ci.

Cette fonction est généralement utilisée en conjonction avec gamepad_get_description() pour générer une partie de la chaîne SDL nécessaire au remappage des contrôleurs sur Windows, Android et macOS.

Notes spécifiques à la plate-forme
Plate-forme Note
Opéra GX Sur la plate-forme cible Opera GX, cette fonction renverra toujours "none" pour les périphériques sans entrée directe.

Syntaxe :

gamepad_get_guid(index);

ArgumentsType Description
index Which gamepad "slot" index to check

Les retours :

Exemple :

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Le code ci-dessus récupère les chaînes de description du GUID et du nom, puis les concatène et stocke la chaîne finale dans une variable globale pour référence ultérieure.
