touche_clavier_press

Cette fonction permet de simuler l'enfoncement de n'importe quelle touche du clavier. Lorsqu'elle est utilisée, la touche sera signalée comme étant enfoncée jusqu'à ce que la fonction de libération correspondante soit appelée (cf. keyboard_key_release()). La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes vk_* listées sur la page principale de saisie du clavier.

Notez qu'après l'appel de cette fonction, si la touche est physiquement pressée sur le clavier, cet événement de pression ne sera pas réenregistré tant que la touche n'aura pas été physiquement relâchée (ce qui déclenche l'événement de libération et arrête cette fonction), ou que la fonction correspondante de relâchement de la touche ait été appelée, et que la touche ait été pressée à nouveau.

Syntaxe :

keyboard_key_press(key);

ArgumentsType Description
key The key to simulate a press of.

Les retours :

Exemple :

keyboard_key_press(vk_space);

Cela simulera une pression sur la barre d'espace.
