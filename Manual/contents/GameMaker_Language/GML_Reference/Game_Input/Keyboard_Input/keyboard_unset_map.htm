clavier_désactiver_map

Avec cette fonction, vous pouvez effacer toutes les touches réaffectées afin qu'elles reviennent à leur état par défaut, c'est-à-dire que toutes les touches s'affectent à elles-mêmes.

Syntaxe :

keyboard_get_map();

Les retours :

N/A

Exemple :

if keyboard_check_pressed(vk_escape)
{
    keyboard_unset_map();
}

L'exemple de code ci-dessus réinitialisera toutes les touches mappées à leurs paramètres par défaut si l'utilisateur appuie sur la touche "escape".
