keyboard_get_map

Parfois, vous pouvez souhaiter obtenir le code ascii d'une clé mappée (pour voir si elle est déjà mappée, par exemple), ce que cette fonction retourne. La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes vk_* listées sur la page principale de saisie du clavier.

Syntaxe :

keyboard_get_map(key);

ArgumentsType Description
key This is the key that you wish to get the mapped value from

Les retours :

Exemple :

if keyboard_get_map(ord("A")) = ord("A")
{
    keyboard_set_map(_keycode, keyboard_lastkey);
}

L'exemple de code ci-dessus vérifie d'abord si "A" a été associé à une autre touche, et si ce n'est pas le cas, il l'associe à la dernière touche sur laquelle l'utilisateur a appuyé.
