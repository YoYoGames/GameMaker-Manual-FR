keyboard_clear

Avec cette fonction, vous pouvez effacer l'état actuel du clavier, ce qui signifie essentiellement que si la touche est maintenue enfoncée, elle ne sera plus reconnue jusqu'à ce qu'elle soit relâchée (ce qui ne déclenchera pas non plus l'événement Keyboard Key Released à cette occasion) et appuyée à nouveau. occasion) et appuyée à nouveau. La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes de vk_* listées sur la page principale de saisie du clavier.

Syntaxe :

keyboard_clear(key);

ArgumentsType Description
key The key to clear.

Les retours :

Exemple :

keyboard_clear(vk_space);

Le code ci-dessus efface l'état de la barre d'espace.

Retour : Entrée du clavier
