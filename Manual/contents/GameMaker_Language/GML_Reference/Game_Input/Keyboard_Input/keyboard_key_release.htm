keyboard_key_release

Avec cette fonction, vous pouvez simuler le relâchement de n'importe quelle touche du clavier. La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes vk_* listées sur la page principale de saisie du clavier.

Syntaxe :

keyboard_key_release(key);

ArgumentsType Description
key The key to simulate a release of.

Les retours :

Exemple :

keyboard_key_release(vk_space);

Cela simulera le relâchement de la barre d'espacement.

Retour : Entrée du clavier
