contrôle du clavier

Cette fonction vous permet de vérifier si une clé a été libérée ou non. Contrairement à la fonction keyboard_check() cette fonction ne s'exécute qu'une fois pour chaque fois que la touche est relâchée. Pour qu'elle se déclenche à nouveau, la touche doit d'abord être enfoncée puis relâchée à nouveau. La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes vk_* listées sur la page principale de saisie du clavier.

Syntaxe :

keyboard_check_released(key);

ArgumentsType Description
key The key to check the released state of.

Les retours :

Exemple :

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

Le code ci-dessus vérifiera si la touche "P" a été relâchée et, si c'est le cas, créera une instance de "obj_Pause".
