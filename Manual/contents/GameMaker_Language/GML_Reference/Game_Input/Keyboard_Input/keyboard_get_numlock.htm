clavier_get_numlock

Vous pouvez utiliser cette fonction pour connaître l'état du verrouillage du numéro du clavier. La valeur true est renvoyée pour l'activation et la valeur false pour la désactivation.

NOTE Cette fonctionnalité n'est disponible que dans les builds de Windows exe et ne fonctionnera sur aucun autre appareil.

Syntaxe :

keyboard_get_numlock();

Les retours :

Exemple :

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

L'exemple de code ci-dessus récupère l'état de la touche de verrouillage numérique et si elle est activée (vrai), elle est désactivée (faux) et vice-versa.
