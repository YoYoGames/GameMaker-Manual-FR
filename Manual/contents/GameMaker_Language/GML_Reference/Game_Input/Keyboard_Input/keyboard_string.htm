clavier_chaîne

Cette variable contient une chaîne de caractères contenant les 1024 derniers caractères (au maximum) tapés sur le clavier. Cette chaîne ne contiendra que les caractères imprimables tapés, mais elle répondra correctement à l'appui sur la touche de retour en arrière en effaçant le dernier caractère. Cette variable n' est pas en lecture seule et vous pouvez la modifier, par exemple pour lui donner la valeur "" (une chaîne vide) si vous l'avez déjà manipulée, et vous pouvez utiliser les fonctions String pour la manipuler. Notez que lorsque vous utilisez le clavier virtuel à l'écran, seule cette variable sera mise à jour avec l'entrée au clavier.

Syntaxe :

keyboard_string;

Les retours :

Exemple :

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

Le code ci-dessus limitera la longueur de la chaîne du clavier à 15 caractères, et supprimera ceux qui dépassent cette limite en recopiant les quinze premiers caractères dans la variable.
