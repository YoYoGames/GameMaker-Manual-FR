clavier_dernier caractère

Cette variable stocke une chaîne de caractères de la dernière touche pressée. Cette variable n' est pas en lecture seule et vous pouvez la modifier, par exemple pour la mettre à "" (une chaîne vide) si vous l'avez déjà manipulée.

Syntaxe :

keyboard_lastchar;

Les retours :

Exemple :

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = vk_nokey;
}

Le code ci-dessus vérifie si la variable lastkey est supérieure à -1, et si c'est le cas, il ajoute la dernière touche sous forme de chaîne à la variable "str".
