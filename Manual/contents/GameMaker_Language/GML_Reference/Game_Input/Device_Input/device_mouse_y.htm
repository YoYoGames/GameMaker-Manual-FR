périphérique_souris_y

Cette fonction renvoie la position y d'une touche sur le périphérique. Si vous exécutez cette fonction sur un module HTML5 ou PC et Mac, cette valeur est mise à jour en permanence, tant que le dispositif (généralement une souris) est branché, mais pour les appareils mobiles, elle ne sera mise à jour que lorsque l'écran est touché. Notez que le nombre maximum de touchers pouvant être détectés dépend beaucoup de l'appareil utilisé et du système d'exploitation qu'il exécute.

Syntaxe :

device_mouse_y(device);

ArgumentsType Description
device The device (from 0 - n) that is being checked.

Les retours :

Exemple :

if device_mouse_check_button(0, mb_left)
{
    if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

Le code ci-dessus vérifie si le dispositif est pressé et si c'est le cas, il interroge la position y du dispositif pour voir si elle est dans les paramètres. Si c'est le cas, il donne à la variable "pressed" la valeur true, sinon, il lui donne la valeur false.
