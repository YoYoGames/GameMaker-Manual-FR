dispositif_est_clavier_ouvert

Cette fonction vérifie la présence d'un clavier dans l'appareil et, s'il y en a un de disponible, elle renvoie true sinon elle renvoie false. Veuillez noter que cette fonction est principalement destinée à être utilisée avec les dispositifs Android.

Pour les utilisateurs du Sony Xperia Play, il existe une configuration des boutons et des touches dans GameMaker. Vous pouvez donc utiliser les constantes du clavier vk_up, vk_down, vk_left, vk_right pour les touches du joypad et le triangle est ord("T"), le carré est ord("S"), le cercle est vk_alt + vk_backspace tandis que le bouton retour est simplement vk_backspace, la croix est vk_space, Select est vk_return et Start est vk_rshift, la gâchette L est ord("L") et la gâchette R est ord("R").

Syntaxe :

device_is_keypad_open

Les retours :

Exemple :

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

Le code ci-dessus vérifie la présence d'un clavier et modifie une variable globale en fonction de la valeur renvoyée.
