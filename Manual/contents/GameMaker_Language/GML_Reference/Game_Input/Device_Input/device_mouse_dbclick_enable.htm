dispositif_mouse_dbclick_enable

Cette fonction permet de configurer l'appareil pour qu'il détecte un double tapotement du mb_left (bouton gauche de la souris) comme une entrée mb_right (bouton droit de la souris).

Par défaut, cette option est définie sur true, ce qui signifie que chaque fois que l'utilisateur touche l'écran du périphérique deux fois rapidement et consécutivement, la valeur de retour est la même que si le bouton droit de la souris avait été cliqué. Lorsque cette option est activée, la première tape sera détectée comme mb_left, et la seconde comme mb_right, assurez-vous donc que le code que vous utilisez en tient compte.

Syntaxe :

device_mouse_dbclick_enable(bool);

Arguments Type Description
bool Booléen Set double-click detection on (true) ou éteint (false).

Les retours :

Boolean

Exemple :

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

Le code ci-dessus vérifie si le périphérique qui exécute le jeu est un PC Windows ou un Mac et s'il s'agit de l'un ou l'autre, il désactive la fonction de double-clic.
