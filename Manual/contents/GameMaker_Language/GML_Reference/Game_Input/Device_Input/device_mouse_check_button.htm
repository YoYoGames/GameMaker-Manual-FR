Bouton de contrôle de la souris du périphérique

Cette fonction renvoie true ou false selon que le bouton de la souris est maintenu enfoncé ou non sur le périphérique donné. Pour le bouton de la souris, vous pouvez fournir une des constantes listées sur cette page.

Cette fonction peut être utilisée pour les écrans tactiles, et l'argument du dispositif peut être compris entre 0 et n, où n est le nombre maximum de "doigts" pouvant toucher l'écran en même temps (le nombre maximum de touchers pouvant être détectés dépend beaucoup du dispositif utilisé et du système d'exploitation qu'il exécute. détecter dépendra beaucoup du périphérique utilisé et du système d'exploitation qu'il exécute, mais la plupart des périphériques devraient détecter au moins jusqu'à 4). Notez que mb_right ne sera détecté que si un double contact est détecté (ce comportement peut être désactivé en utilisant la fonction la fonction device_mouse_dbclick_enable()).

Syntaxe :

device_mouse_check_button(device, button);

ArgumentsType Description
device The device (from 0 - n) that is being checked
button The button of the device that is being checked.

Les retours :

Exemple :

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    room_goto(rm_Menu);
}

Le code ci-dessus vérifie si les deux touches sont maintenues enfoncées en même temps et si c'est le cas, il passe à une autre pièce.
