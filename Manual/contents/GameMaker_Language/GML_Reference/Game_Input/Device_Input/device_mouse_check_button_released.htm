device_mouse_check_button_released

Cette fonction renvoie true ou false selon que le toucher du périphérique que vous spécifiez a été libéré ou non. L'argument du périphérique fait référence au numéro de touche du périphérique, qui peut être compris entre 0 et n. Notez que le nombre maximum de touches pouvant être détectées dépend beaucoup du périphérique utilisé et du système d'exploitation qu'il exécute (la plupart des périphériques supportent au moins 4 touches simultanées). Cette fonction n'est déclenchée qu'une seule fois par l'action de relâchement réelle, et les constantes listées sur cette page peuvent être utilisées pour vérifier les boutons de la souris. Notez que mb_right n'aura été détecté que si un double toucher est détecté et maintenu sur le deuxième toucher avant le relâchement (ce comportement peut être désactivé en utilisant la fonction device_mouse_dbclick_enable()).

Syntaxe :

device_mouse_check_button_released(device, button);

ArgumentsType Description
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Les retours :

Exemple :

if device_mouse_check_button_released(0, mb_left)
{
    press = false;
}

Le code ci-dessus vérifie si le dispositif a été pressé et définit une variable à false si c'est le cas.
