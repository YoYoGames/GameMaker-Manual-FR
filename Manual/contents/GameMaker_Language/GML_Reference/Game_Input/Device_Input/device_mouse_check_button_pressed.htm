device_mouse_check_button_pressed

Cette fonction renvoie true ou false selon que le dispositif que vous spécifiez a été "touché" (cliqué) ou non. L'argument du dispositif fait référence au numéro de touche, qui peut être compris entre 0 et n. Le nombre maximum de touches pouvant être détectées dépend beaucoup du dispositif utilisé et du système d'exploitation qu'il exécute (la plupart des dispositifs supportent au moins 4 touches simultanées). Cette fonction n'est déclenchée qu'une seule fois par l'action de pression réelle, et les constantes listées sur cette page peuvent être utilisées pour vérifier les boutons de la souris. Notez que mb_right ne sera détecté que si une double pression est détectée et maintenue lors de la deuxième pression (ce comportement peut être désactivé à l'aide de la fonction device_mouse_dbclick_enable()).

Syntaxe :

device_mouse_check_button_pressed(device, button);

ArgumentsType Description
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Les retours :

Exemple :

if device_mouse_check_button_pressed(0, mb_left)
{
    press = true;
}

Le code ci-dessus vérifie si le dispositif a été pressé et met une variable à true si c'est le cas.
