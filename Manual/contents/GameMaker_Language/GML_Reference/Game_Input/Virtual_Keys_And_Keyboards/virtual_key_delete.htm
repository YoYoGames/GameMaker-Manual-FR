clé_virtuelle_suppression

Si votre jeu comporte différentes salles ou instances pour contrôler les menus, le gameplay et d'autres choses, il est probable que vous deviez changer la position et les cartes de touches de vos clés virtuelles à un moment donné. Pour cela, vous pouvez utiliser cette fonction pour supprimer les anciennes clés avant de créer les nouvelles (si nécessaire). La fonction nécessite que vous fournissiez l'ID de la clé virtuelle à supprimer - tel que renvoyé par la fonction virtual_key_add().

Notez que toutes les clés virtuelles seront automatiquement retirées de la pièce lorsque celle-ci sera modifiée. Cette fonction n' est donc nécessaire que si vous souhaitez retirer manuellement les clés avant la fin de la pièce elle-même.

Syntaxe

virtual_key_delete(index);

Arguments Type Description
index ID de la clé virtuelle The index of the virtual key to delete.

Les retours :

N/A

Exemple :

virtual_key_delete(global.Left);

Le code ci-dessus supprime la clé virtuelle indexée dans la variable globale "global.Left".
