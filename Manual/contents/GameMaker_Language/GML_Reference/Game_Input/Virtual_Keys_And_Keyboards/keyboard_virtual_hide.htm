keyboard_virtual_hide

Cette fonction peut être utilisée pour masquer le clavier virtuel sur le périphérique exécutant le jeu. L'appel de cette fonction génère un événement asynchrone du système, dans lequel la carte DS async_load est remplie avec les paires clé/valeur suivantes :

  • "event_type" - ce sera "l'état du clavier virtuel" lorsqu'il sera déclenché par les fonctions du clavier virtuel.
  • "screen_height" - la hauteur du clavier virtuel (en pixels). Elle sera de 0 si le clavier est invisible.
  • "keyboard_status" - l'état actuel du clavier, renvoyé sous la forme d'une des chaînes suivantes :
    • "caché"
    • "caché"
    • "visible"
    • "visible"

Syntaxe

keyboard_virtual_hide();

Les retours :

Exemple :

if input == true
{
    input = false;
    keyboard_virtual_hide();
}

Le code ci-dessus masquera le clavier virtuel de l'OS si la variable donnée n'est pas définie sur false.
