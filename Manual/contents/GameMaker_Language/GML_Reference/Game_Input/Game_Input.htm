Entrée du jeu

L'élément fondamental qui fait d'un jeu un jeu est l'interaction avec le joueur. Celle-ci s'effectue par divers moyens, avec le clavier et la souris sur les ordinateurs, par l'écran tactile et l'inclinaison de l'appareil sur les mobiles, voire par une manette de jeu sur toutes les plateformes !

Pour commencer, dans GameMaker, vous utiliserez les événements dédiés à la souris ou au clavier sur les PC ou les événements gestuels sur les mobiles, mais il arrive que vous ayez besoin de plus de contrôle que ce que ces événements fournissent, ou que vous souhaitiez inclure des commandes de manette de jeu ou plusieurs options de contrôle différentes. Pour cela, vous pouvez utiliser les différentes fonctions d'entrée de jeu pour obtenir les boutons ou les touches qui ont été enfoncés ou relâchés, ainsi que pour récupérer les touches d'un écran ou le déplacement d'une manette de jeu.

Il existe un grand nombre de fonctions de saisie et de contrôle dans vos jeux. Elles ont donc été divisées en sections liées à des méthodes de saisie spécifiques afin de faciliter la recherche de ce dont vous avez besoin. Les sections disponibles sont les suivantes :

  • Entrée du clavier
  • Entrée de la souris
  • Entrée du dispositif
  • Saisie gestuelle
  • Entrée du GamePad
  • Clés virtuelles et claviers virtuels
