mouse_check_button_pressed

Cette fonction renvoie true si le bouton de la souris vérifié a été enfoncé ou false dans le cas contraire. Cette fonction ne sera déclenchée qu'une seule fois pour chaque bouton de la souris lorsqu'il est pressé pour la première fois et pour la déclencher à nouveau, le bouton devra avoir été relâché et pressé à nouveau. Notez qu'elle sera considérée comme déclenchée pendant toute la durée de l'étape, et pour toutes les instances qui ont des événements souris ou qui utilisent cette même fonction.

Vous fournissez le bouton de la souris à vérifier à partir de l'une des constantes suivantes :

Syntaxe :

mouse_check_button_pressed(numb);

Arguments Type Description
numb Bouton de la souris constant Which mouse button constant to check for.

Les retours :

Boolean

Exemple :

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

Le code ci-dessus vérifie si le bouton gauche de la souris a été enfoncé et si c'est le cas, il ajoute 50 au score.
