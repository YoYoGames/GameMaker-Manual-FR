souris_y

Cette variable en lecture seule renvoie la position actuelle de l'axe y de la souris dans la pièce.

Syntaxe :

mouse_y;

Les retours :

Exemple :

y = median(64, mouse_y, room_height - 64);

Le code ci-dessus maintiendra l'instance à la position y de la souris tant qu'elle se trouve dans les limites de 64 pixels du haut et du bas de la pièce.
