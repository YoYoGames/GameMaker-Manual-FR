bouton de souris

Cette variable en lecture seule renvoie le bouton de la souris qui est actuellement pressé (actuellement, comme dans cette étape) et peut renvoyer n'importe quelle constante spéciale de la souris, sauf mb_any.

Syntaxe :

mouse_button;

Les retours :

Mouse Button Constant (except mb_any)

Exemple :

if mouse_button = mb_left
{
    x -= 1;
}

Ceci déplace l'instance actuelle vers la gauche si le bouton gauche de la souris est enfoncé.
