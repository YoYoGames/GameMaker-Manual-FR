Entrée de la souris

La saisie de la souris est acceptée sur toutes les plates-formes (sur les appareils mobiles, elle est acceptée comme un simple toucher d'écran - si vous devez utiliser le multi-touch, vous devez utiliser les fonctions spécifiques à l'appareil) et possède quelques constantes qui sont utilisées pour spécifier les boutons pressés. Ces constantes sont présentées dans le tableau suivant :

Les fonctions suivantes existent pour les commandes standard des boutons de la souris :

  • mouse_button
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • mouse_lastbutton
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

Notez qu'il existe également un ensemble de fonctions de fenêtre liées à l'utilisation de la souris sur les cibles du bureau :

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
