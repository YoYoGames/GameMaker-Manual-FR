mouse_check_button_released

Cette fonction renvoie true si le bouton de la souris vérifié a été relâché ou false dans le cas contraire. Cette fonction ne sera déclenchée qu'une seule fois pour chaque bouton de la souris lorsqu'il est relâché et pour la déclencher à nouveau, le bouton devra avoir été pressé et relâché à nouveau. Vous fournissez le bouton de la souris à vérifier à partir de l'une des constantes suivantes :

Syntaxe :

mouse_check_button_released(numb);

Arguments Type Description
numb Bouton de la souris constant Which mouse button constant to check for.

Les retours :

Boolean

Exemple :

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

Le code ci-dessus vérifie si le bouton droit de la souris a été relâché et si c'est le cas, il règle la vitesse de l'instance à un dixième de la distance entre la position x/y actuelle et la position x/y de la souris.
