mouse_check_button

Cette fonction renvoie true si le bouton de la souris à vérifier est maintenu enfoncé ou false si ce n'est pas le cas. Vous fournissez le bouton de la souris à vérifier à partir de l'une des constantes suivantes :

Syntaxe :

mouse_check_button(numb);

Arguments Type Description
numb Bouton de la souris constant Which mouse button constant to check for.

Les retours :

Boolean

Exemple :

if mouse_check_button(mb_left)
{
    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

Le code ci-dessus vérifiera la présence du bouton gauche de la souris et chaque fois qu'il sera maintenu enfoncé, il créera une instance de l'objet indexé dans "obj_Star".
