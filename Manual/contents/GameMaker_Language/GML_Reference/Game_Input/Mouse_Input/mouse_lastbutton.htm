souris_bouton gauche

Cette variable renvoie le dernier bouton de la souris qui a été pressé et peut renvoyer n'importe laquelle des constantes spéciales de la souris, à l'exception de mb_any (vous pouvez également définir cette variable sur l'une des constantes).

Syntaxe :

mouse_lastbutton;

Les retours :

Mouse Button Constant (except mb_any)

Exemple :

if mouse_lastbutton = mb_left
{
    x -= 1;
}

Cela déplace l'instance actuelle vers la gauche si le dernier bouton enfoncé était le bouton gauche de la souris.

Retour : Entrée de la souris
