mouse_clear

Cette fonction efface l'état actuel du bouton de la souris. Cela signifie que la vérification du maintien du bouton enfoncé ne renverra pas true tant que le joueur n'aura pas relâché le bouton et appuyé à nouveau dessus (mais l'état de relâchement sera toujours détecté si l'effacement est effectué alors que le bouton de la souris est maintenu enfoncé).

Vous pouvez fournir une des constantes de bouton de souris listées sur cette page.

Syntaxe :

mouse_clear(button);

Arguments Type Description
button Bouton de la souris constant Which mouse button constant to check for.

Les retours :

Boolean

Exemple :

mouse_clear(mb_any);

Le code ci-dessus effacera l'état d'activation de tous les boutons de la souris.
