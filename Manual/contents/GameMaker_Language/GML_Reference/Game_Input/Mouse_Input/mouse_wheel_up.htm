mouse_wheel_up

Cette fonction renvoie true si la molette de la souris est tournée vers le haut et false sinon.

Syntaxe :

mouse_wheel_up();

Les retours :

Exemple :

if mouse_wheel_up()
{
    y -= 10;
}

Cela déplace l'instance actuelle vers le haut de l'écran si la molette de la souris est tournée vers le haut.

Retour : Entrée de la souris
