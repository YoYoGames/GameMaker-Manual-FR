Angle de rotation du geste

Cette fonction permet de définir l'angle qu'une paire de touchers doit dépasser pour déclencher un geste de démarrage de rotation. L'angle est mesuré en degrés et sa valeur par défaut est de 5°.

Syntaxe :

gesture_rotate_angle();

Les retours :

(inches)

Exemple :

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

Le code ci-dessus vérifie si l'angle de rotation pour les gestes est fixé à 5° et si ce n'est pas le cas, il le fixe à cette valeur.
