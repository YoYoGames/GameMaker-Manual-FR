données de l'événement

Cette variable a une portée globale et est utilisée pour contenir une carte DS lorsqu'elle est utilisée dans l'événement approprié (par exemple, les événements de gestes et les messages de diffusion), et -1 à tout autre moment. Le contenu réel de la carte DS dépendra du type d'événement qui l'a déclenché, il faut donc se référer aux sections individuelles pour ces événements.

Syntaxe :

event_data;

Les retours :

DS Map ID

Exemple :

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

Le code ci-dessus est extrait de l'événement du geste de fin de glisser et vérifie si l'évent est un événement "flick". Si c'est le cas, il extrait les données requises de la carte DS event_data et les utilise pour définir certaines variables. Si un événement "flick" n'est pas détecté, alors les mêmes variables sont mises à 0.
