geste_get_pinch_angle_towards

Cette fonction est utilisée pour obtenir l'angle dans lequel une touche doit se déplacer vers une autre touche avant qu'un geste de pincement ne puisse être lancé. L'angle est mesuré en degrés et sa valeur par défaut est de 45°.

Syntaxe :

gesture_get_pinch_angle_towards();

Les retours :

(inches)

Exemple :

if gesture_get_pinch_angle_towards() != 45
{
    gesture_pinch_angle_towards(45);
}

Le code ci-dessus vérifie si l'angle de pincement pour les gestes est réglé sur 45° et, si ce n'est pas le cas, il le règle sur cette valeur.
