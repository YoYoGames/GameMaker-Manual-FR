geste_get_rotate_time

Cette fonction est utilisée pour obtenir le temps pendant lequel une paire de touches doit tourner dans une direction cohérente pour qu'un geste de démarrage de rotation soit déclenché. Le temps est mesuré en secondes et sa valeur par défaut est de 0,16s.

IMPORTANT ! Actuellement, pour un jeu à 60 images par seconde, vous ne pouvez régler ce paramètre qu'à une seconde maximum, sinon aucune rotation ne sera détectée. Cette durée augmentera pour un framrate inférieur, par exemple un jeu à 30fps peut avoir une durée maximale de 2 secondes.

Syntaxe :

gesture_get_rotate_time();

Les retours :

(inches)

Exemple :

if gesture_get_rotate_time() != 0.1
{
    gesture_rotate_time(0.1);
}

Le code ci-dessus vérifie si le temps de rotation des gestes est fixé à 0,1 seconde et, si ce n'est pas le cas, il le fixe à cette valeur.
