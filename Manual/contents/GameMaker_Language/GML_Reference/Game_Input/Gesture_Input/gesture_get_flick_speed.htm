geste_get_flick_speed

Cette fonction est utilisée pour obtenir la vitesse requise pour qu'un événement Flick Gesture soit déclenché lorsqu'une touche ou un clic est relâché. La vitesse est mesurée en pouces par seconde et a une valeur par défaut de 2,0.

Syntaxe :

gesture_get_flick_speed();

Les retours :

(inches per second)

Exemple :

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

Le code ci-dessus vérifie si la vitesse de défilement des gestes est définie sur 2 pouces par seconde et si ce n'est pas le cas, il la définit sur cette valeur.
