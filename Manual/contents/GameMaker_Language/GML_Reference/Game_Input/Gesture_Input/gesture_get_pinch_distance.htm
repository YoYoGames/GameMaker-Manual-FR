geste_get_pinch_distance

Cette fonction est utilisée pour obtenir la distance dans laquelle vous devez toucher/cliquer l'écran et vous déplacer avec deux doigts avant de déclencher un geste de pincement. La distance est mesurée en pouces et sa valeur par défaut est de 0,1.

Syntaxe :

gesture_get_pinch_distance();

Les retours :

(inches)

Exemple :

if gesture_get_pinch_distance() != 0.1
{
    gesture_pinch_distance(0.1);
}

Le code ci-dessus vérifie si la distance de pincement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
