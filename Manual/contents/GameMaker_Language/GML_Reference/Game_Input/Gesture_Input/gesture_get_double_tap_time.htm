geste_get_double_tap_time

Cette fonction est utilisée pour obtenir le temps qu'il faut entre deux touchers/clics pour déclencher un événement Geste de Double Tap. Le temps est mesuré en secondes et sa valeur par défaut est de 0,10.

Syntaxe :

gesture_get_double_tap_time();

Les retours :

Real (seconds)

Exemple :

if gesture_get_double_tap_time() != 0.16
{
    gesture_double_tap_time(0.16);
}

Le code ci-dessus vérifie si la durée du double tapotement pour les gestes est fixée à 0,16 seconde et si ce n'est pas le cas, il la fixe à cette valeur.
