gesture_get_drag_distance

Cette fonction est utilisée pour obtenir la distance nécessaire au déclenchement d'un événement de geste de glissement par le mouvement d'une touche ou d'un clic. La distance est mesurée en pouces et sa valeur par défaut est de 0,1.

Syntaxe :

gesture_get_drag_distance();

Les retours :

(inches)

Exemple :

if gesture_get_drag_distance() != 0.1
{
    gesture_drag_distance(0.1);
}

Le code ci-dessus vérifie si la distance de déplacement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
