temps_de_glissement

Cette fonction est utilisée pour définir le temps nécessaire pour qu'un événement de Geste de Glissement soit déclenché par un toucher ou un clic. Ce temps affecte également la façon dont l'événement Tap est déclenché, car un toucher/clic et un relâchement avant la fin de ce temps seront considérés comme un Tap. Le temps est mesuré en secondes et sa valeur par défaut est de 0,16.

Syntaxe :

gesture_drag_time(time);

ArgumentsType Description
time The time (in seconds) to set for drag gesture event detection.

Les retours :

Exemple :

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

Le code ci-dessus vérifie si le temps de déplacement pour les gestes est fixé à 0,16 seconde et, si ce n'est pas le cas, il le fixe à cette valeur.
