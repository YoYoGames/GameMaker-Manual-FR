Référence ducode GML

Cette section du manuel est un guide de référence pour le logiciel GameMaker (GML). Vous trouverez toutes les fonctions disponibles documentées ici, avec les arguments requis et des exemples de code pour montrer comment les utiliser. Pour plus d'informations sur l'utilisation de GML, veuillez consulter le document suivant Présentation du GML du manuel.

REMARQUE Lorsque le manuel indique "N/A" comme valeur de retour d'une fonction, cela signifie que la fonction n'est pas censée retourner une valeur dans le cadre de son fonctionnement. Si vous essayez de vérifier la valeur de retour de ces fonctions, vous obtiendrez la valeur undefined.
