uwp_show_help

Cette fonction lance l'application d'aide de la Xbox One qui, à son tour, affichera le fichier d'aide de votre jeu (qui est un fichier au format HTML que vous avez préalablement téléchargé sur le portail des développeurs de la Xbox). Notez qu'un ID utilisateur est spécifié lors de l'appel de cette fonction afin que les informations de la locale de l'utilisateur puissent être utilisées lors de l'affichage. Vous pouvez récupérer l'ID utilisateur à l'aide des fonctions xboxlive_get_user(), xboxlive_get_activating_user(), ou uwp_license_trial_user().

Syntaxe :

uwp_show_help(user_id);

ArgumentsType Description
user_id The User ID pointer to open the helpfile for.

Les retours :

Exemple :

if gamepad_button_check_pressed(0, gp_start)
{
    if (uwp_license_trial_version())
    {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
    }
}

Le code ci-dessus vérifie si un bouton est pressé et vérifie ensuite si l'application fonctionne avec une licence d'essai. Si c'est le cas, il obtient l'ID utilisateur de la licence et ouvre le fichier d'aide pour cet utilisateur.
