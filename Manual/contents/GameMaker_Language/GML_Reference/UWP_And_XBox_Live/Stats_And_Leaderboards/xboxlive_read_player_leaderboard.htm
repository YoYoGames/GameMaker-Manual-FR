xboxlive_read_player_leaderboard

Cette fonction renvoie les informations du classement pour le nom d'utilisateur donné. Le classement doit avoir été créé précédemment sur le tableau de bord XDP de votre jeu, et l'appel de la fonction déclenchera un événement asynchrone social de rappel qui contient la carte async_load map remplie avec les paires clé/valeur pertinentes. La clé "id" de cette carte DS est utilisée pour identifier le callback correct (il peut y avoir plus d'une fonction de déclenchement pour un événement asynchrone donné), et elle sera associée à la constante achievement_leaderboard_info ainsi qu'à un certain nombre d'autres paires clé/valeur. Le contenu exact de la carte est présenté ci-dessous :

  • "id" - Pour cette fonction, il doit être achievement_leaderboard_info
  • "leaderboardid" - L'ID unique du leaderboard tel que défini dans le tableau de bord du fournisseur.
  • "numentries" - Le nombre d'entrées dans le classement que vous avez reçues.
  • "PlayerN" - Le nom du joueur, où "N" est une valeur entière correspondant à sa position dans la liste des entrées du classement.
  • "PlayeridN" - L'identifiant unique du joueur, "N".
  • "RankN" - Le rang du joueur "N" dans le classement.
  • "ScoreN" - Le score du joueur "N".


La fonction vous demande d'indiquer l'une des constantes suivantes pour définir les propriétés du filtre :

Constant Description
xboxlive_achievement_filter_all_players Obtenez tous les scores.
xboxlive_achievement_filter_friends_only Obtenez uniquement les scores des amis, dans l'ordre croissant.
xboxlive_achievement_filter_favorites_only Obtenez uniquement les scores des favoris, dans l'ordre croissant.
xboxlive_achievement_filter_friends_alt Obtenez uniquement les scores des amis par ordre décroissant.
xboxlive_achievement_filter_favorites_alt Obtenez uniquement les scores des favoris par ordre décroissant.

Syntaxe :

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

Arguments Type Description
leaderboard_name Chaîne de caractères The name of the leaderboard to read, as set up on XDP.
user_name ID utilisateur Xbox The ID of the user to read from.
num_items Real The number of items to retrieve.
friend_filter Filtre d'accomplissement Xbox Live constant One of the filter constants (see the description, above).

Les retours :

Real

Exemple :

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

Le code ci-dessus obtient l'ID de l'utilisateur et l'utilise pour demander toutes les données du joueur pour les 10 premières places du classement donné.
