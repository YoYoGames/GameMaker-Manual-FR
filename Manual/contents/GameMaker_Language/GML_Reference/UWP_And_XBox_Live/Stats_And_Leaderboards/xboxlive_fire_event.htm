xboxlive_fire_event

Cette fonction peut être utilisée pour déclencher un événement de statut. L'argument "event_name" est le nom de l'événement à déclencher tel que défini dans la console XDP de votre jeu, et les paramètres supplémentaires suivants dépendront également de ce que vous avez mis en place pour la stat. La fonction retournera 0 si l'événement a été envoyé (et devrait être reçu/traité par le serveur) ou -1 s'il y a eu une erreur (ie : votre événement n'a pas été configuré comme le fichier manifeste de l'événement inclus dans le projet dit un autre nombre).

Syntaxe :

xboxlive_fire_event(event_name, [stat_parameters...]);

Arguments Type Description
event_name Chaîne de caractères The name of the event to fire.
stat_parameters Variable OPTIONAL Paramètres supplémentaires à ajouter à l'événement.

Les retours :

Real

Exemple :

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

Le code ci-dessus obtient certaines informations sur l'utilisateur, puis déclenche un événement de statut avec ces données (et d'autres).
