xboxlive_stats_setup

Cette fonction doit être appelée avant que vous puissiez utiliser l'une des autres fonctions de statistiques Xbox Live, et initialise simplement les bibliothèques requises sur le système. L'argument "user_id" est l'identifiant brut de l'utilisateur tel que renvoyé par la fonction xboxlive_get_user()tandis que les arguments "service_config" et "title_id" correspondent à l'ID unique de votre jeu dans le centre de développement Xbox Live.

Cette fonction ne peut être utilisée qu'avec les statistiques basées sur les événements (2013).

Syntaxe :

xboxlive_stats_setup(user_id, service_config_id, title_id);

Arguments Type Description
user_id ID utilisateur Xbox The ID pointer of the user to check.
service_config_id Chaîne de caractères This is the config_id string for the game.
title_id Chaîne de caractères The unique ID for your game on the Xbox Dev Center.

Les retours :

N/A

Exemple :

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Le code ci-dessus initialise le système de statistiques pour l'ID utilisateur donné.
