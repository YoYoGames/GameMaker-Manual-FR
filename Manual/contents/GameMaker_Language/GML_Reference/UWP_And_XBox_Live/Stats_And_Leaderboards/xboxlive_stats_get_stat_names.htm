xboxlive_stats_get_stat_names

Cette fonction peut être utilisée pour récupérer toutes les statistiques définies dans le gestionnaire de statistiques pour un utilisateur donné. Vous fournissez l'ID de l'utilisateur tel que retourné par (par exemple) la fonction xboxlive_get_user()et la fonction retournera un tableau de chaînes de caractères contenant les statistiques de l'utilisateur. Si une erreur se produit ou si l'utilisateur n'a pas de statistiques, le tableau sera toujours retourné mais aura un nombre d'éléments de 0.

Syntaxe :

xboxlive_stats_get_stat_names(user_id);

ArgumentsType Description
user_id The user ID pointer to get the stat names of

Les retours :

Exemple :

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

Le code ci-dessus récupère toutes les statistiques définies pour le pointeur de l'ID utilisateur stocké dans la variable donnée, puis boucle dans le tableau et supprime la statistique spécifiée du gestionnaire de statistiques.
