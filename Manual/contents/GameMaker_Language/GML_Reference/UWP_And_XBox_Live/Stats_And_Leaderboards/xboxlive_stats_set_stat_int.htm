xboxlive_stats_set_stat_int

Cette fonction peut être utilisée pour définir la valeur d'une statistique pour l'ID utilisateur donné. Vous fournissez l'ID utilisateur tel que retourné par (par exemple) la fonction xboxlive_get_user()puis la chaîne de statuts à définir(si la chaîne de statuts n'existe pas encore, un nouveau statut sera créé et défini à la valeur donnée) et une valeur (un nombre entier) à définir pour le statut. Notez que le nom de la statue doit être uniquement alphanumérique, sans symboles ni espaces.

Lors de la définition de la valeur de la stat, toute valeur précédente sera remplacée, c'est donc à vous de déterminer si la valeur de la stat doit être mise à jour ou non (par exemple, vérifier que le meilleur score est bien le plus élevé) en comparant la valeur de la stat actuelle avec la nouvelle valeur avant de la définir. la nouvelle valeur avant de la définir.

La fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur est invalide, ou toute autre valeur si la fonction a été appelée avec succès.

Syntaxe :

xboxlive_stats_set_stat_int(user_id, stat, value);

ArgumentsType Description
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (an integer)

Les retours :

Exemple :

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

Le code ci-dessus récupère la valeur de la statistique "Jeux joués", puis définit cette même statistique à cette valeur plus 1.
