xboxlive_stats_remove_user

Cette fonction peut être utilisée pour supprimer (désenregistrer) un utilisateur donné du gestionnaire de statistiques, en effectuant un vidage des données de statistiques vers le serveur live. Selon la documentation de la XBox, le jeu n'a pas besoin de supprimer l'utilisateur du gestionnaire de statistiques, car le système d'exploitation de la XBox vide périodiquement les statistiques de toute façon. Vous fournissez l'ID de l'utilisateur tel que retourné par (par exemple) la fonction xboxlive_get_user()et la fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur n'est pas valide, ou toute autre valeur si la fonction a été appelée avec succès.

REMARQUE: la suppression de l'utilisateur peut renvoyer une erreur si les statistiques qui ont été définies sur l'utilisateur ne sont pas valides (par exemple, si les noms des statistiques contiennent des caractères numériques non alpha).

La fonction générera un callback qui déclenchera un événement asynchrone du système. Cet événement aura la carte spéciale DS async_load qui devra ensuite être analysée pour les clés suivantes :

  • "id" - Tiendra la constante achievement_stat_event
  • "Eventname" - Contient la chaîne "LocalUserRemoved".
  • "userid" - L'ID utilisateur associé à la demande
  • "error" - 0 en cas de succès, une autre valeur en cas d'erreur.
  • "errormessage" - Une chaîne contenant un message d'erreur, s'il en existe un.

Notez que si vous voulez vider les données de statistiques sur le serveur live à tout moment sans supprimer l'utilisateur, vous pouvez utiliser la fonction xboxlive_stats_flush_user().

Syntaxe :

xboxlive_stats_remove_user(user_id);

ArgumentsType Description
user_id The user ID (a pointer) to remove

Les retours :

Exemple :

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

Le code ci-dessus passe en boucle les identifiants de compte utilisateur stockés et supprime (désenregistre) chaque utilisateur du gestionnaire de statistiques.
