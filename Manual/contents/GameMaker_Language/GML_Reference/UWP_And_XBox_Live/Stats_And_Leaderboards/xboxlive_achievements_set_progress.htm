xboxlive_achievements_set_progress

Cette fonction peut être utilisée pour mettre à jour la progression d'une réalisation. Vous fournissez l'ID utilisateur tel que renvoyé, par exemple, par la fonction xboxlive_get_user()et ensuite la chaîne de caractères de la réalisation (il s'agit de l'identifiant de la réalisation tel qu'il a été attribué dans le centre de développement XDP/Windows lors de sa création). Enfin, vous définissez la valeur de progression sur une valeur comprise entre 0 et 100. Notez que le système de réalisation refusera les mises à jour qui sont inférieures à la valeur de progression actuelle. actuelle.

Syntaxe :

xboxlive_achievements_set_progress(user_id, achievement, progress);

ArgumentsType Description
user_id The user ID of the user to set the achievement for
achievement The achievement to set (a string)
progress The progress value to set (from 0 to 100)

Les retours :

Exemple étendu :

Voici un exemple détaillé de la façon dont cette fonction peut être utilisée. Pour commencer, vous l'appelez dans un événement comme Room Start ou Create:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Le code ci-dessus sera appelé pour obtenir une liste de toutes les positions du classement mondial pour le jeu, et générera un appel à l'événement social asynchrone que nous traiterons de la manière suivante :

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Le code ci-dessus crée une valeur en pourcentage basée sur certaines variables globales et l'utilise ensuite pour définir la progression de l'accomplissement "Game_Completed".
