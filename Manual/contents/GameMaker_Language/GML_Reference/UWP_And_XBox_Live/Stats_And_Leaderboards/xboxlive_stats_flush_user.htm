xboxlive_stats_flush_user

Cette fonction peut être utilisée pour transférer les données de statistiques d'un utilisateur donné du gestionnaire de statistiques vers le serveur en direct, afin de s'assurer que le serveur est à jour avec les valeurs actuelles. Selon la documentation de la XBox, les développeurs doivent veiller à ne pas appeler cette fonction trop souvent, car la XBox limitera le nombre de requêtes et le système d'exploitation de la XBox évacuera automatiquement les statistiques toutes les 5 minutes environ.

REMARQUE: la suppression de l'utilisateur peut renvoyer une erreur si les statistiques qui ont été définies sur l'utilisateur ne sont pas valides (par exemple, si les noms des statistiques contiennent des caractères numériques non alpha).

Pour utiliser la fonction, vous devez fournir l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user()et une valeur de priorité (qui sera de 1 pour une haute priorité et de 0 pour une basse priorité). La fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur est invalide, ou toute autre valeur si la fonction a été appelée avec succès. La fonction génère également un callback qui déclenche un événement asynchrone du système. Cet événement contiendra la carte spéciale DS async_load qui devra ensuite être analysée pour les clés suivantes :

  • "id" - Tiendra la constante achievement_stat_event
  • "Eventname" - Contient la chaîne "StatisticUpdateComplete".
  • "userid" - L'ID utilisateur associé à la demande
  • "error" - 0 en cas de succès, une autre valeur en cas d'erreur.
  • "errormessage" - Une chaîne contenant un message d'erreur, s'il en existe un.

Notez que les statistiques seront vidées automatiquement pour vous lorsque vous supprimez un utilisateur du gestionnaire de statistiques. Voir xboxlive_stats_remove_user() pour plus d'informations.

Syntaxe :

xboxlive_stats_flush_user(user_id, priority);

Arguments Type Description
user_id ID utilisateur Xbox The user ID (a pointer) to use
priority Real The prority value (0 or 1)

Les retours :

Real

Exemple :

for(var i = 0; i < array_length(user_id); ++i;)
{
    xboxlive_stats_flush_user(user_id[i], 0);
}

Le code ci-dessus passe en boucle les identifiants de compte utilisateur stockés, puis vide les données du gestionnaire de statuts pour chaque utilisateur.
