xboxlive_get_stats_for_user

Cette fonction peut être utilisée pour récupérer des données sur des statistiques spécifiques à partir des serveurs Xbox Live. Vous fournissez l'ID utilisateur tel que renvoyé par la fonction xboxlive_get_user()puis l'ID de configuration de service de vos jeux (tel qu'indiqué sur la console XDP), et enfin les statistiques requises. Vous pouvez demander jusqu'à un maximum de 14 statistiques, mais cela ne garantit pas que vous obtiendrez 14 résultats, car il y a une limite à la longueur totale de la demande et donc le nombre maximum de statistiques dépend de la longueur des noms des statistiques elles-mêmes.

La fonction renverra 0 si la demande de statut a été envoyée ou -1 s'il y a eu une erreur, et le callback déclenchera un événement asynchrone du système. Cet événement aura la carte spéciale DS async_load qui devra ensuite être analysée pour la clé suivante :

  • "event_type" - doit contenir la chaîne "stat_result" si l'événement a été déclenché par cette fonction.

Si le type d'événement se rapporte à cette fonction, il y aura également la clé supplémentaire suivante :

  • "utilisateur" - contient l'ID de l'utilisateur pour les statistiques demandées.

Il peut également y avoir un ensemble supplémentaire de paires clé-valeur, où la clé est le nom de la statistique demandée, et la valeur la valeur de cette statistique sur le serveur. Notez que si une statistique n'a jamais été créée pour cet utilisateur (parce qu'il n'a pas déclenché les événements qui la définissent), aucune valeur ne peut être renvoyée pour cette statistique (ds_map_exists() peut être utilisé pour vérifier les stats dans la carte async_load ).

Si la demande échoue parce que la longueur de la demande est trop importante, il devrait y avoir un message dans la sortie console indiquant le code d'erreur :

xboxlive_get_stats_for_user - exception occurred getting results - 0x80190190

Dans ce cas, la carte DS asynchrone devrait avoir une clé "succeeded" avec une valeur de "0", et dans ce cas, vous devriez essayer de récupérer moins de statistiques en un seul appel.

Syntaxe :

xboxlive_get_stats_for_user(user_id, serviceconfig_id, statname1, …);

Arguments Type Description
user_id ID utilisateur Xbox The user ID pointer.
serviceconfig_id Chaîne de caractères The service config file ID
statname1 (2, 3, etc...) Cordes The stat names to retrieve the information for.

Les retours :

Real

Exemple :

var _uid = xboxlive_get_user(0);
var _configid = "00000000-0000-0000-0000-000000000000";
xboxlive_get_stats_for_user(xbuser, _configid, "GameProgress", "CurrentMode");

Le code ci-dessus récupère l'ID de l'utilisateur et l'utilise pour demander des informations sur les statistiques spécifiques de cet utilisateur.
