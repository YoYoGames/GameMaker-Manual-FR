xboxlive_achievement_show_achievements

Cette fonction ouvrira la page des réalisations pour la plate-forme choisie. Veuillez noter qu'il s'agit d'une fonction asynchrone, c'est-à-dire que votre jeu continuera de fonctionner en arrière-plan pendant que la page des réalisations s'affiche. Vous devez donc faire attention à l'endroit où vous utilisez cette fonction et vous assurer de mettre le jeu en pause ou de ne l'autoriser à s'afficher que dans les zones de votre jeu où elle n'interfère pas avec le déroulement du jeu.

Syntaxe :

xboxlive_achievement_show_achievements()

Les retours :

Exemple :

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

Le code ci-dessus vérifie une variable personnalisée pour voir si les réalisations sont activées, et si c'est le cas, il définit une variable globale et crée une instance de contrôleur "pause" avant d'appeler la fonction pour afficher les réalisations à l'écran.
