xboxlive_achievement_load_leaderboard

Cette fonction envoie une requête au serveur pour obtenir des informations sur le classement donné. Le classement doit avoir été créé précédemment dans le tableau de bord des développeurs de votre jeu et l'ID donné est celui que vous devez fournir comme premier argument de la fonction. Elle déclenchera un événement social asynchrone de type callback qui contient la carte async_load map remplie avec les paires clé/valeur pertinentes. La clé id de ce DS Map est utilisée pour identifier le bon callback (il peut y avoir plus d'une fonction de déclenchement pour un événement asynchrone donné), et sera associée à la constante xboxlive_achievement_leaderboard_info ainsi qu'à un certain nombre d'autres paires clé/valeur pour chaque joueur. Le contenu exact de la carte est présenté ci-dessous :

  • "id" - Pour cette fonction, il doit être xboxlive_achievement_leaderboard_info
  • "leaderboardid" - L'ID unique du leaderboard tel que défini dans le tableau de bord du fournisseur.
  • "numentries" - Le nombre d'entrées dans le classement que vous avez reçues.
  • "PlayerN" - Le nom du joueur, où "N" est une valeur entière correspondant à sa position dans la liste des entrées du classement.
  • "PlayeridN" - L'identifiant unique du joueur, "N".
  • "RankN" - Le rang du joueur "N" dans le classement.
  • "ScoreN" - Le score du joueur "N".

Vous pouvez spécifier la partie exacte du classement à afficher en donnant une valeur d'indice minimale et maximale, et demander que les résultats soient filtrés pour récupérer uniquement les scores de vos amis ou ceux affichés par tous les joueurs en utilisant les constantes de filtre suivantes :

Filtre d'accomplissement Xbox Live constant
Constant Description
xboxlive_achievement_filter_friends_only  Obtenez uniquement les scores des amis des joueurs.
xboxlive_achievement_filter_all_players  Obtenez tous les scores de tous les joueurs.

Syntaxe :

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

Arguments Type Description
ident Chaîne de caractères The unique ID of the leaderboard as shown on the developer dashboard.
minindex Real The starting index value to get the leaderboard data from.
maxindex Real The maximum index value to get the leaderboard data from.
filter Filtre d'accomplissement Xbox Live constant Whether to filter the results to those that are on your friends list or to all players (see the Description above)

Les retours :

N/A

Exemple étendu :

Le code suivant serait probablement appelé après que le joueur se soit connecté à son compte de jeu pour obtenir une liste de toutes les positions des amis dans le classement du jeu :

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

Cela enverra une demande d'informations sur le leaderboard donné et générera un callback asynchrone avec la carte spéciale async_load DS contenant les données suivantes :

var ident = ds_map_find_value(async_load, "id");
if (ident == xboxlive_achievement_leaderboard_info)
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

Le code ci-dessus vérifie le retour de DS Map dans l'événement social asynchrone et si son "id" correspond à la constante vérifiée, il vérifie ensuite que le classement envoyé correspond à l'appel de fonction original avant de parcourir la carte en boucle et de stocker toutes les différentes valeurs dans un certain nombre de tableaux.
