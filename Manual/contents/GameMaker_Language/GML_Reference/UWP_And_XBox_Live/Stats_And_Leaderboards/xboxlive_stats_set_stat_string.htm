xboxlive_stats_set_stat_string

Cette fonction peut être utilisée pour définir un statut à une chaîne pour l'ID utilisateur donné. Vous fournissez l'ID utilisateur tel que retourné par (par exemple) la fonction xboxlive_get_user()puis la chaîne de statut à définir(si la chaîne de statut n'existe pas déjà, un nouveau statut sera créé et défini à la valeur donnée) et enfin la chaîne à laquelle définir le statut. Notez que le nom de la statue doit être uniquement alphanumérique, sans symboles ni espaces, mais que la valeur de la chaîne que vous définissez peut comporter des espaces et des caractères spéciaux.

Lors de la définition de la chaîne de statut, toute chaîne précédente sera remplacée. Il vous appartient donc de déterminer si la chaîne de statut doit être mise à jour ou non en comparant la chaîne de statut actuelle avec la nouvelle avant de la définir.

La fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur est invalide, ou toute autre valeur si la fonction a été appelée avec succès.

Syntaxe :

xboxlive_stats_set_stat_string(user_id, stat, value);

Arguments Type Description
user_id ID utilisateur Xbox The user ID pointer to set the stat for
stat Chaîne de caractères The statistic to set (a string)
value Chaîne de caractères The value to set the stat to (a string)

Les retours :

Real

Exemple :

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

Le code ci-dessus définit l'état "LevelName" à la chaîne stockée dans la variable globale donnée.
