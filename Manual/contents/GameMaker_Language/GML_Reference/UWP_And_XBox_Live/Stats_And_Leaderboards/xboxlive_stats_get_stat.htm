xboxlive_stats_get_stat

Cette fonction peut être utilisée pour récupérer une seule valeur de stat dans le gestionnaire de stat pour un utilisateur donné. Vous fournissez l'ID de l'utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user()et ensuite la chaîne de caractères du statut telle que définie lors de sa création à l'aide de l'une des fonctions xboxlive_stats_set_stat_*. La valeur de retour peut être soit une chaîne de caractères, soit un réel (selon le statut vérifié), soit la constante GML undefined si le statut n'existe pas ou s'il y a eu une erreur.

Syntaxe :

xboxlive_stats_get_stat(user_id, stat);

ArgumentsType Description
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

Les retours :

Real, String or undefined

Exemple :

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Le code ci-dessus vérifie une variable et si elle est true, il vérifie la valeur de la statistique "PercentDone". Si cette valeur est inférieure à 100, une valeur est alors générée et la stat est définie sur cette valeur.
