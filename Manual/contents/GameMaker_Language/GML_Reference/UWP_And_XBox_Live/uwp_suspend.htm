uwp_suspension

Cette fonction indique au système que l'application a fini de sauvegarder les données suite à la détection d'une demande de suspension. L'appel de cette fonction est obligatoire pour la soumission à la Xbox One, et elle doit toujours être utilisée en conjonction avec la fonction uwp_is_suspending()comme le montre l'exemple ci-dessous.

Syntaxe :

uwp_suspend();

Les retours :

Exemple :

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

Le code ci-dessus vérifie si l'application va être suspendue et si c'est le cas, il appelle un script pour sauvegarder les données du jeu avant de le suspendre.
