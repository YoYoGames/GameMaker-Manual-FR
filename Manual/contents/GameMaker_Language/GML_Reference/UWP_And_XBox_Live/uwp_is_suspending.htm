uwp_est_suspendu

Cette fonction retournera true pour une seule trame (et false sinon) de manière similaire à os_is_paused(). Le jeu a ensuite une seconde pour sauvegarder l'état du jeu avant d'être potentiellement interrompu par le système. Une fois que le jeu a fini de sauvegarder les données, il doit appeler uwp_suspend() pour suspendre proprement. Le jeu devrait probablement aussi entrer dans un état de pause afin que, s'il est repris par le système (s'il n'a pas été complètement arrêté), le joueur ne soit pas directement projeté dans une situation de jeu.

Syntaxe :

uwp_is_suspending();

Les retours :

Exemple :

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

Le code ci-dessus vérifie si l'application va être suspendue et si c'est le cas, il appelle un script pour sauvegarder les données du jeu avant de le suspendre.
