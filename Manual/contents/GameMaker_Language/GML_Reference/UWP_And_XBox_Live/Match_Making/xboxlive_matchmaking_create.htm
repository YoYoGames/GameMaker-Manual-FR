xboxlive_matchmaking_create

Cette fonction va créer une session multi-joueurs à l'aide du modèle de session, de la trémie de mise en relation et du modèle d'association de dispositifs sécurisés, que vous devriez avoir créés au préalable dans le tableau de bord XDP. L'ID utilisateur que vous pouvez récupérer pour une tablette de jeu particulière en appelant la fonction xboxlive_user_for_pad() et la visibilité sera l'une des constantes suivantes :

Visibilité de la correspondance Xbox constante
Constant Description
xboxlive_match_visibility_open spécifie que la session peut être rejointe par les autres.
xboxlive_match_visibility_private spécifie que la session n'est pas visible pour les utilisateurs qui ne sont pas membres de la session.
xboxlive_match_visibility_usetemplate spécifie que la valeur utilisée dans le modèle de session doit être utilisée (ceci devrait normalement être utilisé car vous ne pouvez pas remplacer la valeur d'un modèle si une valeur a été définie dans XDP).

La fonction renvoie une valeur unique d'identification de la demande, qui peut ensuite être utilisée pour identifier l'événement asynchrone social correct pour cette fonction. Cet événement sera déclenché lorsque la session aura été créée et contiendra un DS Map dans la variable async_load avec les paires clé/valeur suivantes :

  • "requestid" - contient l'ID de la demande tel que renvoyé par la fonction appelante.
  • "status" - contient la chaîne "session_created" pour vous informer du type d'événement qui a été déclenché.
  • "sessionid" - contient l'ID de la session, ou -1 en cas d'échec
  • "error" - contient 0 en cas de succès, ou -1 en cas d'échec
  • "correlationid" - le handle de session pour la corrélation avec la fonction xboxlive_matchmaking_join_session()

Syntaxe :

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

Arguments Type Description
user_id ID utilisateur Xbox The user ID pointer to use
visibility Visibilité de la correspondance Xbox constante One of the constants listed above
template Chaîne de caractères The name of the session template
hopper Chaîne de caractères The name of the matchmaking hopper
sdatemplate Chaîne de caractères The name of the secure device association template
matchattributes Real OPTIONAL Les attributs du match

Les retours :

Async Request ID

Exemple :

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

L'exemple ci-dessus récupère l'ID de l'utilisateur sur la manette donnée et initialise une session de matchmaking pour lui.
