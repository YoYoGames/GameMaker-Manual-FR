xboxlive_matchmaking_find

Cette fonction recherchera une session multi-joueurs à l'aide du modèle de session, de la trémie de mise en relation et du modèle d'association de dispositifs sécurisés, que vous devriez avoir créés au préalable dans le tableau de bord XDP. L'ID utilisateur que vous pouvez récupérer pour une tablette de jeu particulière en appelant la fonction xboxlive_user_for_pad() . La fonction renverra une valeur unique d'identification de la demande, qui pourra ensuite être utilisée pour identifier l'événement asynchrone social correct pour cette fonction. Cet événement sera déclenché lorsque la session aura été créée et contiendra une carte DS dans la variable async_load avec les paires clé/valeur suivantes :

  • "requestid" - contient l'ID de la demande tel que renvoyé par la fonction appelante.
  • "status" - contient la chaîne "session_created" pour vous informer du type d'événement qui a été déclenché.
  • "num_results" - contient 1 ou plus si une session a été trouvée, sinon contient 0.
  • "sessionid<index> " - contient l'ID de la session, ou -1 en cas d'échec(<index/gt ; est une valeur de 0 à "num_results" - 1).
  • "sessionOwner<index> " - contient l'ID de l'hôte de la session si une session a été trouvée(<index> est une valeur de 0 à "num_results" - 1).
  • "error" - contient 0 en cas de succès, ou -1 en cas d'échec

Syntaxe :

xboxlive_matchmaking_find(user_id, template, hopper, sdatemplate, [matchattributes]);

Arguments Type Description
user_id ID utilisateur Xbox The user ID pointer to use
template Chaîne de caractères The name of the session template
hopper Chaîne de caractères The name of the matchmaking hopper
sdatemplate Chaîne de caractères The name of the secure device association template
matchattributes Real OPTIONAL Les attributs du match

Les retours :

Async Request ID

Exemple :

var userID = xboxone_user_for_pad(global.PadIndex);
result = xboxone_matchmaking_find(userID, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

L'exemple ci-dessus récupère l'ID de l'utilisateur sur la manette de jeu donnée et essaie ensuite de trouver une session de jumelage pour lui.
