xboxlive_matchmaking_set_joinable_session

Cette fonction configure une session (précédemment créée) pour que d'autres utilisateurs puissent la rejoindre via l'interface utilisateur du système. Un utilisateur n'a qu'une seule session joignable à la fois, et lorsqu'il quitte la session (ou que la session se termine), celle-ci sera effacée, mais vous pouvez l'effacer manuellement en passant -1 comme argument de la session.

Syntaxe :

xboxlive_matchmaking_set_joinable_session(local_user, session_that_is_joinable);

ArgumentsType Description
local_user The local user ID pointer.
session_that_is_joinable The session ID to make joinable, or -1.

Les retours :

Exemple :

if (global.session_ID != -1)
{
    xboxlive_matchmaking_set_joinable_session(xboxlive_user_for_pad(0), global.session_ID);
}

Le code ci-dessus vérifie l'existence d'une valeur d'ID de session valide, et si une telle valeur est détectée, il définit la session comme étant joignable.
