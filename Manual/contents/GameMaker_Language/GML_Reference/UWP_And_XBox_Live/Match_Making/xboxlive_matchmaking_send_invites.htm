xboxlive_matchmaking_send_invites

Cette fonction fait apparaître la boîte de dialogue du système Xbox Live pour permettre à un utilisateur d'envoyer des invitations à sa partie. Vous devez spécifier l'utilisateur local qui doit contrôler la boîte de dialogue (et dont la liste d'amis sera utilisée par la boîte de dialogue), la valeur unique de l'ID de session pour la session à laquelle ils veulent inviter des gens, et un texte par défaut à afficher dans la boîte de dialogue. Vous obtenez l'ID de session à partir de la carte async_load DS map générée par la fonction xboxlive_matchmaking_create().

La fonction renvoie -1 si les arguments ne sont pas valides ou si l'utilisateur n'est pas connecté à Xbox Live, ou une valeur supérieure à 0 sinon.

Syntaxe :

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

ArgumentsType Description
from_user The local user ID for controlling the dialogue.
session_to_invite_into The unique session ID for the session to join.
text_to_add_to_invite A short text to add to the invitation.

Les retours :

Exemple :

if gamepad_button_check_pressed(0, gp_face1)
{
    var _uid = xboxlive_user_for_pad(0);
    xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
}

Le code ci-dessus vérifie s'il y a une pression sur un bouton de la manette de jeu et, si c'est le cas, la boîte de dialogue Xbox Live pour les invitations de session s'ouvre.
