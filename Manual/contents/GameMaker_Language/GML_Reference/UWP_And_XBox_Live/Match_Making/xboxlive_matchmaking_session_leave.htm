xboxlive_matchmaking_session_leave

Cette fonction fait en sorte que l'utilisateur qui a créé ou trouvé la session spécifiée la quitte. Vous pouvez obtenir le pointeur d'ID de session à partir de la carte async_load DS map qui est renvoyé dans l'événement social asynchrone lorsque vous avez créé ou trouvé une session (voir xboxlive_matchmaking_create() pour plus de détails).

Syntaxe :

xboxlive_matchmaking_session_leave(session_id);

ArgumentsType Description
session_id The Session ID pointer of the session to leave.

Les retours :

Exemple :

if gamepad_button_check_pressed(0, gp_start)
{
    xboxlive_matchmaking_session_leave(global.SessionID);
}

Le code ci-dessus vérifie s'il y a eu une pression sur un bouton de la manette de jeu et si c'est le cas, l'utilisateur sera déconnecté de la session de matchmaking donnée.
