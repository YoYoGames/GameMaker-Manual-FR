xboxlive_set_savedata_user

Cette fonction spécifie que les futures opérations sur les fichiers qui opèrent dans la zone de sauvegarde du jeu (c'est-à-dire toutes les écritures de fichiers, et les lectures de fichiers qui ne sont pas dans la zone de regroupement) seront associées au pointeur d'ID utilisateur spécifié. Cette fonction peut être appelée aussi souvent que nécessaire pour rediriger les données de sauvegarde vers l'utilisateur approprié, ou vous pouvez utiliser la constante pointer_null pour sauvegarder dans la zone de stockage générique de la machine.

Syntaxe :

xboxlive_set_savedata_user(user_id);

Arguments Type Description
user_id ID utilisateur Xbox ou pointeur_null The user ID (a pointer) to set for saving, or pointer_null

Les retours :

N/A

Exemple :

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

Le code ci-dessus vérifie si un utilisateur est actuellement affecté comme cible de sauvegarde, et s'il ne l'est pas, il est affecté.
