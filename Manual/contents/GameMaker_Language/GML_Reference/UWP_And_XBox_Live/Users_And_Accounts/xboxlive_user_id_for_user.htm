xboxlive_user_id_for_user

Cette fonction convertit le pointeur de l'ID utilisateur en une chaîne de caractères.

Syntaxe :

xboxlive_user_id_for_user(user_id);

ArgumentsType Description
user_id The ID pointer of the user to check.

Les retours :

Exemple :

var _uid = xboxlive_user_for_pad(0);
uid_string = xboxlive_user_id_for_user(_uid);

Le code ci-dessus récupère le pointeur de l'ID de l'utilisateur affecté à l'emplacement [0] de la manette de jeu, puis enregistre cet ID sous forme de chaîne dans une variable d'instance.
