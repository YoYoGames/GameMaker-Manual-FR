xboxlive_pad_for_user

Les utilisateurs de la Xbox peuvent avoir plusieurs pads assignés à leur ID utilisateur, chacun remplissant un "slot". Ces emplacements sont numérotés à partir de 0 et peuvent contenir n'importe quel index de pad. Ainsi, l'emplacement 0 peut être attribué au pad 3 et l'emplacement 1 au pad 2. Cette fonction peut être utilisée pour obtenir le pad qui est assigné à un slot donné, et est particulièrement utile si un utilisateur laisse un gamepad sans entrée pendant un certain temps, car il s'éteindra de lui-même. Vous pouvez donc vérifier n'importe quel ID utilisateur en utilisant cette fonction pour vous assurer qu'il a un gamepad actif, et s'il ne l'est pas, réagir en conséquence (par exemple) en mettant votre jeu en pause. Si le pad est actif, la valeur de son index sera renvoyée, mais si ce n'est pas le cas, la fonction renverra -1.

Notez que si un pad dans un slot devient inactif, les slots suivants "glisseront" dans la position vacante, donc si vous avez trois pads actifs assignés aux slots 0, 1 et 2, et que le pad dans le slot 0 est inactif, les slots 1 et 2 deviendront maintenant les slots 0 et 1, donc le slot 0 devrait toujours contenir un pad actif à moins que tous les pads soient éteints (dans ce cas il retournera -1).

Syntaxe :

xboxlive_pad_for_user(user_id, slot);

Arguments Type Description
user_id ID utilisateur Xbox The User ID (a pointer) to check
slot Real The index (an integer) of the gamepad slot to check

Les retours :

Xbox Pad ID

Exemple :

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
    for(var i = 0; i < gamepad_get_device_count(); i++;)
    {
        var u_id = xboxlive_user_for_pad(i);
        if (u_id == -1)
        {
            async_ID = xboxlive_pad_for_user(i, u_id);
        }
    }
}

Le code ci-dessus vérifie l'emplacement 0 pour le pointeur d'ID utilisateur donné pour voir si des manettes de jeu ont été attribuées, et si ce n'est pas le cas, une manette est attribuée.
