xboxlive_get_user_count

Avec cette fonction, vous pouvez trouver le nombre total d'utilisateurs actuellement connectés au système XBox. La valeur de retour sera un nombre entier, de 0 à N, où N est le nombre d'utilisateurs connectés -1.

Syntaxe :

xboxlive_get_user_count();

Les retours :

Exemple :

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

La boucle ci-dessus passe en revue tous les utilisateurs inscrits et stocke leur pointeur d'identification unique dans un tableau.
