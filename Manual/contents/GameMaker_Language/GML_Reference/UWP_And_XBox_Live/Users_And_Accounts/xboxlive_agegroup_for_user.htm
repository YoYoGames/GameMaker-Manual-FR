xboxlive_agegroup_for_user

Cette fonction renvoie l'une des quatre constantes (illustrées ci-dessous) pour indiquer la tranche d'âge attribuée au pointeur d'identification de l'utilisateur spécifié.

Syntaxe :

xboxlive_agegroup_for_user(user_id);

Arguments Type Description
user_id ID utilisateur Xbox The user ID (a pointer) to check

Les retours :

Xbox Age Group Constant

Xbox Groupe d'âge Constant
Constant Description
xboxlive_agegroup_unknown La tranche d'âge de l'utilisateur est inconnue
xboxlive_agegroup_child La tranche d'âge des utilisateurs est comprise entre 8 et 12 ans (inclus).
xboxlive_agegroup_teen La tranche d'âge des utilisateurs est comprise entre 13 et 17 ans (inclus, et entre 13 et 19 ans en Corée du Sud).
xboxlive_agegroup_adult Le groupe d'âge des utilisateurs est de 18 ans ou plus (ou 20 ans ou plus en Corée du Sud).

Exemple :

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if (_age != xboxlive_agegroup_adult)
{
    global.ContentControl = true;
}
else global.ContentControl = false;

Le code ci-dessus vérifie le groupe d'âge de l'utilisateur activant et définit une variable globale à true ou false en fonction de la constante retournée.
