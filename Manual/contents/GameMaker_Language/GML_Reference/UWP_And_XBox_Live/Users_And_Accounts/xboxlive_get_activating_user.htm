xboxlive_get_activating_user

Cette fonction permet de récupérer le pointeur d'identification de l'utilisateur qui a lancé le jeu depuis la console.

REMARQUE: la façon dont vous utiliserez ce pointeur dépendra de ce que vous voulez faire dans le jeu, mais vous ne devez pas compter sur lui pour être l'ID de l'utilisateur en cours de jeu, car l'utilisateur qui l'active peut se déconnecter pendant que le jeu est en cours.

Syntaxe :

xboxlive_get_activating_user();

Les retours :

Xbox User ID

Exemple :

global.activUser = xboxlive_get_activating_user();

Ce qui précède stocke le pointeur d'identification de l'utilisateur activant dans une variable globale
