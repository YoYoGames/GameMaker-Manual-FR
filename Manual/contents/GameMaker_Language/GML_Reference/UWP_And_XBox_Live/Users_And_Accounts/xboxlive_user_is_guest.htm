xboxlive_user_is_guest

Cette fonction retournera true si le pointeur d'ID utilisateur donné est un utilisateur invité et false si ce n'est pas le cas. Vous pouvez obtenir un pointeur d'ID utilisateur avec la fonction xboxlive_get_user().

Syntaxe :

xboxlive_user_is_guest(user_id);

ArgumentsType Description
user_id The ID pointer of the user to check.

Les retours :

Exemple :

if xboxlive_user_is_guest(user_id[0])
{
    global.user_name[0] = "GUEST";
}
else
{
    global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]);
}

L'exemple ci-dessus stocke le pointeur d'identification de l'utilisateur en cours d'activation dans une variable globale.
