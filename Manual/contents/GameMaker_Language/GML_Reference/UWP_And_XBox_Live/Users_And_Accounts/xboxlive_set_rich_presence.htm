xboxlive_set_rich_presence

Cette fonction définit la chaîne de présence riche pour l'utilisateur donné. Une chaîne de présence riche indique l'activité d'un utilisateur dans le jeu après le nom du jeu auquel il joue, séparé par un trait d'union. Cette chaîne est affichée sous le Gamertag d'un joueur dans la liste "Friends & Clubs" ainsi que dans le profil utilisateur Xbox Live du joueur.

Lorsque vous utilisez cette fonction, vous devez fournir le pointeur d'identification de l'utilisateur, puis vous pouvez indiquer si l'utilisateur est actif ou non dans le jeu (en utilisant true/false). Si la valeur est false, la chaîne de présence riche sera complétée par "/afk" ou quelque chose d'approprié. L'argument suivant est l'ID de la chaîne de présence riche à afficher, et enfin vous pouvez (facultativement) fournir une chaîne service_config_id. Cet argument est facultatif, et s'il n'est pas fourni, il utilise le SCID spécifié dans les options de jeu Xbox.

Syntaxe :

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

Arguments Type Description
user_id ID utilisateur Xbox The ID pointer of the user to check.
is_user_active Booléen Flag the user as active or not.
rich_presence_string Chaîne de caractères The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id Chaîne de caractères OPTIONAL Il s'agit de la chaîne config_id du jeu.

Les retours :

N/A

Exemple :

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

Le code ci-dessus récupère le pointeur d'identification de l'utilisateur affecté à l'emplacement [0] du gamepad, puis définit la chaîne de présence riche pour cet utilisateur.
