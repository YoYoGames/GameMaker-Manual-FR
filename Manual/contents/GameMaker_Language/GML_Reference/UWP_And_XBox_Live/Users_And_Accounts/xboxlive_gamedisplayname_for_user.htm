xboxlive_gamedisplayname_for_user

Avec cette fonction, vous pouvez récupérer le nom d'affichage - sous forme de chaîne - à partir du pointeur d'identification de l'utilisateur donné. Notez que si l'utilisateur est local, la valeur renvoyée est simplement une chaîne de son nom d'affichage, mais s'il s'agit d'un utilisateur distant, la chaîne sera son gamertag qui peut alors être utilisé, par exemple, pour le muting/unmuting dans le chat vocal.

Syntaxe :

xboxlive_gamedisplayname_for_user(user_id);

ArgumentsType Description
user_id The ID pointer of the user to check.

Les retours :

Exemple :

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

Le code ci-dessus récupère le pointeur d'identification de l'utilisateur pour chaque compte d'utilisateur disponible, puis stocke le pointeur ainsi que le nom d'affichage dans deux tableaux.
