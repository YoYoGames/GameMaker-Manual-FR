xboxlive_user_for_pad

Cette fonction renvoie le pointeur d'identification de l'utilisateur associé à la valeur d'index de gamepad donnée, ou pointer_null si aucun utilisateur n'y est associé. Notez que cette fonction ne doit être utilisée qu'avec des indices de gamepad, et non avec des indices xboxlive_get_user_count().

Syntaxe :

xboxlive_user_for_pad(pad_index);

ArgumentsType Description
pad_index The index (an integer) of the gamepad slot to check

Les retours :

Xbox User ID or pointer_null

Exemple :

for(var i = 0; i < 11; ++i;)
{
    var u_id = xboxlive_user_for_pad(i);
    if u_id != pointer_null
    {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
    }
    else
    {
        pad_name[i] = "Press Play";
    }
}

Le code ci-dessus boucle sur tous les emplacements de pad disponibles et vérifie si un pointeur d'identification de l'utilisateur lui est associé. Si c'est le cas, un tableau est défini avec le nom d'affichage de l'utilisateur, sinon le tableau est défini avec un texte par défaut.
