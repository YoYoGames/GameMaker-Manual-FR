xboxlive_get_user

Avec cette fonction, vous pouvez récupérer le pointeur d'identification de l'utilisateur indexé. Si l'utilisateur n'existe pas, la fonction renverra la constante pointer_null à la place. Vous pouvez trouver le nombre d'utilisateurs actuellement connectés avec la fonction xboxlive_get_user_count().

IMPORTANT: Cette fonction ne doit être utilisée qu'avec xboxlive_get_user_count() - ne l'utilisez pas avec les manettes de jeu. Utilisez plutôt xboxlive_user_for_pad().

Syntaxe :

xboxlive_get_user(index);

Arguments Type Description
index Real The index (an integer) to get the User ID from.

Les retours :

Xbox User ID or pointer_null

Exemple :

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

La boucle ci-dessus passe en revue tous les utilisateurs inscrits et stocke leur pointeur d'identification unique dans un tableau.
