xboxlive_gamertag_for_user

Cette fonction permet d'obtenir le Gamer Tag de l'utilisateur actuellement connecté. Notez que cette fonction n' est valable que si l'utilisateur est connecté et que vous devez donc effectuer une vérification à l'aide de la fonction xboxlive_user_is_signed_in() avant de demander le Gamer Tag. Si vous appelez cette fonction alors qu'aucun utilisateur n'est connecté, vous obtiendrez une chaîne vide "" retournée.

Syntaxe :

xboxlive_gamertag_for_user();

Les retours :

Exemple :

if !xboxlive_user_is_signed_in()
{
    if (!xboxlive_user_is_signing_in())
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

Le code ci-dessus vérifie si un utilisateur est connecté à XBox Live et s'il l'est, le code récupère le gamertag de l'utilisateur et le stocke dans une variable globale pour une utilisation ultérieure. Si l'utilisateur n'est pas connecté, le code vérifie s'il est en train de se connecter, auquel cas rien ne se passe, et s'il ne l'est pas, il ouvre le sélecteur de compte, l'invitant à se connecter.
