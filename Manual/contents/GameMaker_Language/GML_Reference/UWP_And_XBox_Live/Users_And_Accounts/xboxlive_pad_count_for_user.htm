xboxlive_pad_count_for_user

Cette fonction permet de savoir combien de "slots" de pads sont actifs pour l'utilisateur actuel (cf. xboxlive_pad_for_user() pour plus de détails).

Syntaxe :

xboxlive_pad_count_for_user(user_id, slot);

ArgumentsType Description
user_id The user ID (a pointer) to check

Les retours :

Exemple :

global.slots[0] = xboxlive_pad_count_for_user(user_id[0])

Le code ci-dessus stocke le nombre de pads associés au pointeur d'ID utilisateur donné.
