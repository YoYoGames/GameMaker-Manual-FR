période_source_get_temps

Cette fonction renvoie la valeur de la période définie pour la source de temps donnée.

La valeur de cette période sera exprimée dans les unités définies pour la source de temps (secondes ou images).

Syntaxe :

time_source_get_period(id);

Arguments Type Description
id ID de la source de temps The Time Source pour obtenir la valeur de la période pour

Les retours :

Real

Retour : Sources de temps
