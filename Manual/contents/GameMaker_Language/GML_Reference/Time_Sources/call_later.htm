appeler plus tard

Cette fonction déclenche une fonction de rappel après le temps spécifié ou après un certain intervalle. Pour ce faire, elle crée une source de temps ( voir Sources de temps).

Pour annuler la fonction de rappel, voir call_cancel.

IMPORTANT Bien que cette fonction renvoie un ID de source de temps réel, cet ID ne peut pas être utilisé avec les fonctions de source de temps, car les sources de temps utilisées par call_later et call_cancel sont "invisibles" - elles n'apparaîtront pas comme enfants d'une source de temps intégrée et ne peuvent pas être utilisées avec d'autres fonctions de source de temps. Elles sont toutefois incluses dans le compte des sources de temps renvoyé par debug_event("ResourceCounts").

Arguments

Voici la liste des arguments de cette fonction :

Période

Il s'agit d'une période de temps qui peut être exprimée en secondes ou en images, selon l'unité spécifiée dans l'argument suivant.

Lorsque l'unité utilisée est l'image, la période doit être un nombre entier. Les valeurs non entières sont arrondies à l'unité inférieure, à l'exception des valeurs inférieures à 1, qui sont arrondies à l'unité supérieure.

Unité

C'est l'unité dans laquelle la période est exprimée, et peut être time_source_units_seconds ou time_source_units_frames.

Voir : Unités de source de temps

Vous pouvez utiliser une valeur de battements par minute en utilisant time_bpm_to_seconds().

Rappel

Vous devez spécifier un "callback" (rappel) method qui sera appelé lorsque la source de temps expirera.

Boucle

Si la source de temps doit être mise en boucle.

Syntaxe :

call_later(period, unit, callback, [loop=false]);

Arguments Type Description
period Real The period that the Time Source fonctionne pour, dans les unités données
units Source de temps Unité Constante The units that the given period is in
callback Méthode ou Fonction de script The method or script function to call after the specified time
loop Booléen OPTIONAL Si la source de temps doit être mise en boucle (par défaut, false).

Les retours :

Time Source ID

Exemple 1 :

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

Dans cet exemple, nous créons une fonction de rappel qui affiche un message de débogage au bout de cinq secondes. Après cela, la source de temps sous-jacente est détruite.

Exemple 2 :

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

Dans cet exemple, nous créons une fonction de rappel qui crée un nouvel ennemi à un endroit aléatoire de la pièce. Cette fonction est ensuite exécutée toutes les 10 secondes en la passant à call_later. Le dernier argument est défini à true afin que la fonction soit appelée toutes les 10 secondes.

Utilisez call_cancel avec le handle retourné par call_later pour arrêter la source de temps et les appels à la callback.
