remise à zéro de la source de temps

Cette fonction réinitialise la source de temps donnée, en remettant à zéro sa minuterie et ses informations de répétition.

Après avoir appelé cette fonction, vous devrez redémarrer la source de temps pour qu'elle effectue le décompte.

Syntaxe :

time_source_reset(id);

Arguments Type Description
id ID de la source de temps The Time Source pour réinitialiser

Les retours :

N/A

Exemple :

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

Ce code réinitialisera la source de temps donnée chaque fois qu'une nouvelle pièce commencera, puis relancera la source de temps.
