time_seconds_to_bpm

Cette fonction prend la durée d'un battement en secondes, et renvoie une valeur de battements par minute.

Syntaxe :

time_seconds_to_bpm(seconds);

Arguments Type Description
seconds Real The seconds-per-beat value to convert into beats-per-minute

Les retours :

Real

Exemple :

var _period = time_source_get_period(time_source);
var _bpm = time_seconds_to_bpm(_period);

show_debug_message(_bpm);

Cette fonction récupère la période d'une source de temps, la convertit en BPM et imprime cette valeur dans le journal de sortie.

Ce code suit l'exemple de time_bpm_to_seconds().
