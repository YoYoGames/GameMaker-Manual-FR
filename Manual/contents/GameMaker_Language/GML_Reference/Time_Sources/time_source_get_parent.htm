time_source_get_parent

Cette fonction retourne le ID de la source de temps du parent de la source de temps donnée.

La valeur renvoyée peut être l'ID d'une source de temps personnalisée ou l'une des sources de temps intégrées, selon la façon dont la source de temps donnée a été créée.

Syntaxe :

time_source_get_parent(id);

Arguments Type Description
id ID de la source de temps The Time Source pour obtenir le parent de

Les retours :

Time Source ID

Exemple :

var _parent = time_source_get_parent(time_source);

Ce code récupère le parent d'une source de temps et le stocke dans une variable locale.
