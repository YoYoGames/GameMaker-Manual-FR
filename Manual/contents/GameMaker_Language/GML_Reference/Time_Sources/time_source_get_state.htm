time_source_get_state

Cette fonction renvoie l'état de la source de temps donnée.

Syntaxe :

time_source_get_state(id);

Arguments Type Description
id ID de la source de temps The Time Source pour obtenir l'état de

Les retours :

Time Source State Constant

Exemple :

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_start(time_source);
    }
}

Lorsque la touche Espace est pressée, ce code récupère l'état d'une source de temps.

Lorsque l'état est actif, il met en pause la source de temps, et lorsqu'elle est en pause, il la reprend.
