temps_source_arrêt

Cette fonction arrête la source de temps donnée et réinitialise sa minuterie. Vous ne pouvez pas arrêter une source de temps qui n'a pas démarré ou qui est dans un état terminé.

Syntaxe :

time_source_stop(id);

Arguments Type Description
id ID de la source de temps The Time Source d'arrêter

Les retours :

N/A

Exemple :

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

Le code ci-dessus arrêtera une source de temps si elle n'est pas déjà arrêtée.
