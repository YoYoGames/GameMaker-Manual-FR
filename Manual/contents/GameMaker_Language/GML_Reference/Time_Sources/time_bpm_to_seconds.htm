time_bpm_to_seconds

Cette fonction prend une valeur de battements par minute, et renvoie la durée de chaque battement en secondes.

Ceci peut être utilisé lors de la création d'une source de temps pour utiliser une valeur BPM pour la source de temps. Il est important qu'une telle source de temps utilise les secondes comme unité.

Syntaxe :

time_bpm_to_seconds(bpm);

Arguments Type Description
bpm Real The beats-per-minute value to convert into seconds-per-beat

Les retours :

Real

Exemple :

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

Ce code convertit une valeur de 90 BPM en secondes, et l'utilise pour créer une source de temps qui fonctionne indéfiniment.

À chaque battement, il imprime le message "BEAT !" dans le journal de sortie.
