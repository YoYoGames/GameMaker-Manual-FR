appel_annulation

Cette fonction annule une fonction de rappel qui a été lancée plus tôt par call_later.

Syntaxe :

call_cancel(handle);

Arguments Type Description
handle ID de la source de temps The handle to a Time Source retourné par call_later.

Les retours :

N/A

Exemple :

call_cancel(handle);

Retour : call_later
Suivant : time_source_create
