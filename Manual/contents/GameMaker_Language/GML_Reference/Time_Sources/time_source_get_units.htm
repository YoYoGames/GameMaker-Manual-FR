unités de temps_source_get

Cette fonction renvoie les unités définies pour la source de temps donnée. La valeur de la période pour la source de temps est exprimée dans ces unités.

Syntaxe :

time_source_get_units(id);

Arguments Type Description
id ID de la source de temps The Time Source pour obtenir les unités de

Les retours :

Time Source Unit Constant
