temps_source_destruction

Cette fonction va détruire la source de temps donnée.

Vous pouvez éventuellement spécifier si toutes les sources de temps enfants doivent être supprimées également.

Syntaxe :

time_source_destroy(id, [destroy_tree]);

Arguments Type Description
id ID de la source de temps The Time Source de détruire
destroy_tree Booléen OPTIONAL Destruction ou non des sources temporelles enfants, false par défaut.

Les retours :

N/A

Exemple :

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

Le code d'événement Create crée une nouvelle source de temps qui expire une fois par seconde et se répète indéfiniment.

Le code de l'événement Clean Up détruit cette source de temps, car elle ne sera plus nécessaire après la destruction de l'instance.
