début_source_temps

Cette fonction démarre la source de temps donnée, en changeant son état en time_source_state_active.

La source de temps spécifiée peut être une nouvelle source de temps qui n'a jamais été démarrée, qui a été mise en pause, arrêtée ou qui a expiré sans répétitions.

Cette fonction va "réinitialiser en douceur" la source de temps donnée, en réinitialisant son temps d'expiration et les reps restants aux valeurs qui ont été configurées pour elle.

Syntaxe :

time_source_start(id);

Arguments Type Description
id ID de la source de temps The Time Source pour commencer

Les retours :

N/A

Exemple :

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

Dans cet exemple, nous voulons que l'instance se détruise 300 images plus tard.

Le code crée d'abord une méthode qui appelle simplement instance_destroy().

Il crée ensuite une source de temps, héritant de la source de temps du jeu. Il fixe sa période à 300 images.

Enfin, il lance la source de temps.
