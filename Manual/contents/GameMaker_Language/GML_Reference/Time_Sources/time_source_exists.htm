temps_source_existe

Cette fonction indique si l'identifiant donné est une source de temps valide.

Si la source temporelle a été détruite, cette fonction renvoie faux, mais si la source temporelle a été arrêtée ou si elle a simplement expiré, elle continuera d'exister, ce qui signifie que cette fonction renvoie vrai.

Syntaxe :

time_source_exists(id);

Arguments Type Description
id ID de la source de temps The Time Source pour vérifier

Les retours :

Boolean

Exemple :

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

Ce code vérifie si une source de temps existe. Si c'est le cas, il vérifie si la touche Espace a été pressée.

Dans ce cas, il détruit la Source Temporelle.
