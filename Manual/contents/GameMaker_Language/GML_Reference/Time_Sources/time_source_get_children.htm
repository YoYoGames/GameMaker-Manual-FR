time_source_get_children

Pour la Time Source donnée, cette fonction retourne un tableau contenant les ID de la source de tempsde ses enfants.

Syntaxe :

time_source_get_children(id);

Arguments Type Description
id ID de la source de temps The Time Source pour obtenir les enfants de

Les retours :

Array of Time Source ID

Exemple :

var _children = time_source_get_children(time_source);
var _count = array_length(_children);

for (var i = 0; i < _count; i++)
{
    time_source_destroy(_children[i]);
}

time_source_destroy(time_source);

Ce code passe en revue tous les enfants d'une Source Temporelle, et les détruit un par un.

A la fin, il détruit la Source Temporelle mère.

Retour : Sources de temps
