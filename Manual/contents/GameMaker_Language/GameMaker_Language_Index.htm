GML Code

Le site GameMaker Language (également appelé simplement GML) est le langage de script propriétaire de GameMaker. Ce langage est structuré pour permettre aux utilisateurs de créer leurs jeux de manière intuitive et flexible tout en offrant toute la puissance de n'importe quel autre grand langage de programmation. Il constitue également la base de GML Visual et peut être utilisé conjointement avec celui-ci si nécessaire.

La section GML Code du manuel est divisée en plusieurs parties :

  • Aperçu du GML Code
  • Référence du GML Code
