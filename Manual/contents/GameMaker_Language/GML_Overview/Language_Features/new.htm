L'opérateur new peut être utilisé pour créer une struct avec une fonction qui est appelée avec les arguments donnés. La fonction peut alors remplir le struct avec les variables des arguments - de la même manière que l'événement de création d'une instance définira les variables initiales de l'instance - puis l'opérateur new retournera le struct. Avant d'utiliser cet opérateur, il est important de noter que la fonction donnée doit être marquée comme une fonction constructor , sinon l'opérateur new ne créera pas le struct (l'exemple de code ci-dessous le montre, et pour plus d'informations, veuillez consulter la page sur les Structs).

NOTE Vous pouvez vérifier un struct pour savoir quelle fonction a été utilisée pour le créer en utilisant la fonction runtime instanceof().

Cet opérateur a la syntaxe suivante :

<variable> = new <function> (<argument>, <argument>, ...);

struct Lorsque vous utilisez l'opérateur new avec une fonction, celle-ci renvoie une référence à access qui est stockée dans la variable donnée, ce qui vous permet d'accéder à struct et à toutes les variables qui y ont été créées par la fonction.

L'exemple suivant montre une fonction qui a été définie pour être utilisée comme constructeur :

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Cette fonction peut ensuite être utilisée avec l'opérateur new pour créer un struct et le remplir avec les variables définies par les valeurs des arguments utilisés dans la fonction, comme ceci :

mystruct = new init_struct(10, 100, "Hello World");

Il est important de noter que l'appel de new sur une fonction qui n'a pas été marquée comme un constructeur provoquera une exception d'exécution.
