L'opérateur delete est utilisé pour déréférencer une structure, et a la syntaxe suivante :

delete <variable>;

Cela signifie que vous appelez l'opérateur delete avec la variable qui contient un struct et qu'il supprimera la référence spécifique au struct stocké dans la variable donnée (une référence est simplement une valeur qui pointe vers la zone de mémoire qui stocke le et son contenu). qui stocke le struct et son contenu). Si cette référence était la dernière référence à struct dans votre jeu, le ramasseur d'ordures supprimera struct de la mémoire lors de l'itération suivante, généralement à la fin de l'année. une itération suivante, généralement au tout début de l'étape suivante.

REMARQUE: delete ne peut être utilisé qu'avec des variables et non des expressions.

Par défaut, structs, qui n'est plus référencé dans le code, sera automatiquement ramassé dans les étapes qui suivent son utilisation, mais il est bon d'utiliser cet opérateur pour le signaler explicitement afin que le ramasseur d'ordures le retire de la mémoire au tout début de l'étape suivante. au tout début de l'étape suivante, libérant ainsi la mémoire plus rapidement et plus efficacement.

L'exemple suivant montre la création d'une struct - par exemple dans l'événement de création d'une instance :

mystruct =
{
    a : 0,
    b : 100,
    c : "Hello World"
}

Ce site struct sera ensuite utilisé dans les événements d'instance comme d'habitude, avant d'être finalement marqué pour la collecte des déchets dans l'événement de nettoyage, comme ceci :

delete mystruct;
