throw

L'instruction throw runtime vous permet de générer vos propres exceptions d'exécution, en mettant fin au jeu et en affichant un message d'erreur, en utilisant la syntaxe suivante :

throw (<expression>);

L'expression utilisée peut être une valeur, une chaîne de caractères ou tout autre type de données. Elle génère alors une exception d'erreur qui est - par défaut - affichée à l'écran et le jeu se termine à la fermeture du message d'erreur. Par exemple, en appelant ceci :

throw ("Hello World!");

Vous pouvez toutefois reprendre ce message d'erreur et utiliser votre propre code de gestion en appelant la la fonction exception_unhandled_handler(). Cette fonction d'exécution vous permet de fournir un code de gestion personnalisé. method personnalisée qui sera appelée à chaque fois qu'une exception non gérée se produit dans votre jeu.
