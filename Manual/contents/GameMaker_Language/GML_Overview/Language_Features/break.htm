break

L'instruction break est utilisée pour terminer prématurément une for, repeat, while ou do / until ou une boucle quelconque, ou pour indiquer à une instruction switch de s'arrêter à ce point, ou pour terminer prématurément un appel with. Veuillez consulter les pages individuelles de ces différentes fonctions pour obtenir une explication plus approfondie de la manière dont elles peuvent être utilisées dans chaque circonstance. Notez que si break est utilisé en dehors de l'un de ces contextes, une erreur sera générée.

Vous pouvez voir ci-dessous quelques exemples de base de l'utilisation de break, dont la syntaxe est simple:

break;

break dans une boucle for:

for (var i = 0; i < 10; i += 1)
{
    if (array[i] == 234)
    {
        break;
    }
}

num = i;

break dans une boucle repeat:

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

break dans une boucle while:

var i = 0;
while (!place_free(x, y))
{
    x = random(room_width);
    y = random(room_height);
    if (i > 50)
    {
        break;
    }
    else
    {
        i+=1;
    }
}

break dans une boucle do / until:

var _id = noone;
do
{
    _id = list[| 0];
    if (instance_exists(_id))
    {
        break;
    }
    ds_list_delete(list, 0);
}
until (ds_list_empty(list));
target = _id;

break lorsque vous utilisez with:

var count = 0;
with (obj_Enemy)
{
    count++;
    if (count > 10)
    {
        break;
    }
    hp = 100;
}

break dans un switch:

switch (keyboard_key)
{
    case vk_left:
    case ord("A"):
        x -= 4;
    break;

    case vk_right:
    case ord("D"):
        x += 4;
    break;

    case vk_up:
    case ord("W"):
        y -= 4;
    break;

    case vk_down:
    case ord("S"):
        y += 4;
    break;
}
