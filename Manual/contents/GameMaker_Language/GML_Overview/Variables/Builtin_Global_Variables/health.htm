health

Cette variable a une portée globale et est utilisée pour contenir une valeur numérique qui est généralement utilisée pour la santé du joueur. Cette variable est uniquement conçue pour prendre en charge les projets hérités de versions antérieures de GameMaker et doit être utilisée dans les nouveaux projets. ne pas utiliser dans les nouveaux projets car elle pourrait être obsolète à l'avenir.

Syntaxe :

health

Les retours :

Real (single precision floating point value)

Exemple

if (health <= 0)
{
    global.state = "Game Over";
    instance_destroy();
}

Le code ci-dessus vérifie la variable health et si elle est inférieure ou égale à 0, une variable globale est définie et l'instance est détruite.
