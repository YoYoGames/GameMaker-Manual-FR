vies

Cette variable a une portée globale et est utilisée pour contenir une valeur numérique qui est généralement utilisée pour les vies des joueurs. Cette variable est uniquement conçue pour prendre en charge les projets hérités de versions antérieures de GameMaker et doit être conservée. ne pas utiliser dans les nouveaux projets car elle pourrait être obsolète à l'avenir.

Syntaxe :

lives;

Les retours :

Real (single precision floating point value)

Exemple

if (lives < 5)
{
    lives += 1;
}

Le code ci-dessus vérifie la variable lives et si elle est inférieure à 5, on lui ajoute 1.
