Signets

Une autre méthode de navigation dans l'espace de travail consiste à utiliser des signets. Vous pouvez ajouter jusqu'à 9 "signets" à IDE fenêtres ou sections de code/GML Visual dans des scripts ou des événements d'objets, et vous pouvez ensuite passer rapidement de l'un à l'autre en utilisant la bonne combinaison de touches.

Pour ajouter un signet, il suffit de sélectionner la fenêtre ou la ligne de code que vous souhaitez étiqueter, puis de cliquer sur + + NumberKey[1 - 9] sur Windows ou + + NumberKey[1 - 9] sur Mac, ce qui aura pour effet de définir le signet et d'étiqueter l'élément dans IDE comme suit :

Dans l'image ci-dessus, nous avons assigné les signets numérotés 1, 2 et 4 à différentes parties de IDE, et nous pouvons maintenant naviguer loin de ces fenêtres ou même les fermer complètement, puisque le signet nous permettra d'y revenir directement à tout moment par la suite. Pour ce faire, il suffit d'appuyer sur / + NumberKey[1 - 9], et si l'élément marqué d'un signet a été fermé, il sera ouvert, sinon l'espace de travail s'orientera vers le signet sélectionné.

IMPORTANT Les signets ne sont pas sauvegardés avec le projet ou lorsque le site IDE est fermé. Ils ne sont valables que pour la session dans laquelle ils ont été créés.

Si vous souhaitez supprimer un signet, il vous suffit de sélectionner la fenêtre ou la ligne de code concernée, puis de répéter les opérations + / + NumberKey[1 - 9] que vous avez utilisées pour le créer.
