Le menu des mises en page

Ce menu est lié à la sauvegarde et au chargement de différentes mises en page de IDE. Vous pouvez configurer le site IDE comme vous le souhaitez, avec certains éléments (comme le navigateur d'actifs, ou la sortie Windows) disposés différemment de la configuration par défaut. Cela vous permet d'adapter GameMaker à votre flux de travail et, grâce aux options de ce menu, vous pouvez sauvegarder les modèles du projet en cours dans un fichier *.xml qui peut ensuite être chargé dans n'importe quel projet pour que les modèles soient les mêmes. Les options de ce menu sont les suivantes :

  • Charger la mise en page - Cela ouvrira un explorateur de fichiers où vous pouvez sélectionner un fichier *.xml de mise en page précédemment enregistré pour le charger.
  • Enregistrer la mise en page - Cela ouvrira un explorateur de fichiers où vous pourrez nommer et enregistrer la mise en page actuelle de IDE en tant que fichier *.xml.
  • Réinitialiser la mise en page - Ceci réinitialisera la mise en page actuelle de IDE aux paramètres par défaut.
