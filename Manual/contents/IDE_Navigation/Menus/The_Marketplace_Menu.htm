Le menu Marketplace

access Outre la possibilité de créer vos propres ressources et votre propre code pour vos projets, vous pouvez également consulter le site Marketplace de GameMaker où vous pouvez télécharger des ressources gratuites et payantes à ajouter à vos projets. Vous pouvez en savoir plus sur le site Marketplace en consultant la section sur le Marché de GameMaker, et les options de ce menu sont indiquées ci-dessous :

  • Allez sur le marché - Ouvrez la fenêtre Marketplace où vous pouvez rechercher des actifs.
  • Ma bibliothèque - Affichez votre bibliothèque actuelle de contenu Marketplace.
  • Historique de mes achats - Cette option ouvrira un navigateur et vous amènera à la page du compte de l'utilisateur connecté afin que vous puissiez voir l'historique des achats pour ce compte.
  • Mise à jour d'un paquet existant - Cette option ouvrira l'assistant de mise à jour du paquet afin que vous puissiez mettre à jour un paquet Marketplace que vous avez précédemment créé.
  • Créer un package - Cette option vous permet d'ouvrir l'assistant de création de package pour créer un nouveau package d'actifs ou un tutoriel pour le site Marketplace.
