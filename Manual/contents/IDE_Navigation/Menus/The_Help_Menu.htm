Le menu d'aide

Ce menu permet d'obtenir de l'aide lors de l'utilisation de GameMaker. Il contient les éléments de menu suivants :

  • Open Manual - Ouvre le manuel d'aide. Notez qu'en cliquant sur le bouton d'aide dans la barre de menu, vous ouvrirez également le document, tout comme en appuyant sur .
  • Base de connaissances - Vous accéderez à la base de connaissances deGameMaker, où vous trouverez des articles sur la configuration de la plate-forme, les problèmes connus, l'utilisation de GameMaker et un certain nombre d'autres choses.
  • Signaler un bug - Si vous trouvez des bugs ou si vous rencontrez des problèmes en utilisant GameMaker, vous pouvez les signaler ici.
  • Ouvrir le journal dans l'explorateur - Cela ouvre le répertoire sur votre système où vous pouvez trouver le fichier ui.log, qui est généralement nécessaire pour soumettre des rapports de bogue liés à l'IDE.
  • GameMaker.io - Ouvre le site Web de GameMaker.
  • GameMaker Community - Ceci ouvrira les forums de GameMaker Community dans votre navigateur. Notez qu'il s'agit d'un forum géré par la communauté , donc si vous avez un rapport de bogue ou besoin d'une assistance technique, vous devez d'abord vous rendre sur le site de support officiel (voir ci-dessus).
  • Notes de publication - Ceci ouvrira les notes de publication de GameMaker dans votre navigateur. Si vous devez annuler une mise à jour, vous pouvez obtenir des programmes d'installation de la version précédente en cliquant sur le lien du numéro de version sur cette page.
  • Notes de mise à jour actuelles de Runtime - Comme le Runtime peut être mis à jour indépendamment de IDE, il possède son propre ensemble de notes de mise à jour du Runtime. La sélection de cette option les ouvrira dans votre navigateur, mais notez que le dernier runtime peut ne pas être celui que vous utilisez. Pour savoir sur quel site runtime vous vous trouvez, ou pour obtenir des informations sur la façon d'ajouter, de supprimer ou d'activer runtimes, veuillez consulter la section Préférences du Runtime.
  • SDK requis - En raison de la nature multiplateforme de GameMaker, un certain nombre de SDK tiers sont nécessaires. En cliquant sur ce lien, vous accéderez à un article de support qui détaille ce dont vous avez besoin pour chaque plateforme.
  • Ouvrir le projet dans l'explorateur - Cela ouvrira l'emplacement de sauvegarde du projet actuellement chargé dans l'explorateur de fichiers du système d'exploitation.
  • Rafraîchir les polices du système - Cette option permet de rafraîchir les polices du système que GameMaker est capable de voir. Elle est utile pour détecter les nouvelles polices que vous avez installées lorsque le site IDE était ouvert.
  • Licences - Cette option ouvre une fenêtre contenant les licences de toutes les bibliothèques tierces utilisées dans GameMaker.
  • À propos - Cette option ouvre la fenêtre "À propos de GameMaker" où vous pouvez obtenir des informations sur la version du produit (essentielles pour signaler tout bogue ou problème) et voir les crédits.
