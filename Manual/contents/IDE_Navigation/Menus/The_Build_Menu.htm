Le menu de construction

Le menu de construction est l'endroit où vous pouvez choisir de construire votre projet pour le tester, debugging ou comme exécutable final. Les options disponibles sont les suivantes :

  • Run - Exécutez votre projet actuel sur la plate-forme cible choisie pour le tester. Pour connaître les différentes options disponibles lors de la compilation, veuillez consulter la section Compilation. Notez que cette commande est la même que celle qui consiste à appuyer sur le bouton Run dans le site IDE.
  • Débogage - Exécutez votre projet actuel pour le tester avec le débogueur afin que vous puissiez voir des informations détaillées sur les performances et déboguer toute erreur ou tout problème que vous pourriez rencontrer. Pour plus d'informations sur debugging, veuillez consulter la section Débogage. Notez que cette commande est la même que celle qui consiste à appuyer sur le bouton Debug dans la fenêtre IDE.
  • Nettoyer - GameMaker conserve un cache de fichiers pour accélérer le temps de compilation lors des tests. Cependant, il arrive qu'un fichier cache "périmé" entraîne des erreurs bizarres et inattendues dans votre jeu. Dans ce cas, la première chose à faire est de nettoyer le fichier cache et de refaire le test. C'est aussi une bonne idée de nettoyer le cache avant de créer un exécutable final pour la plateforme cible. Notez que cette commande est la même que celle qui consiste à appuyer sur le bouton Clean Cache dans la fenêtre IDE.
  • Créer un exécutable - Cette opération compile le projet actuel et crée un ou plusieurs paquets exécutables pour la plate-forme cible. Elle ouvrira d'abord l'explorateur de fichiers afin que vous puissiez sélectionner l'endroit où vous souhaitez enregistrer le projet, puis compiler le jeu. Consultez la section sur la compilation pour plus d'informations sur la configuration des cibles de compilation. Notez que cette commande est la même que celle qui consiste à appuyer sur le bouton Create Executable dans le site IDE.
