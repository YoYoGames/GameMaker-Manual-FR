Le menu Fichier

Le menu Fichier vous permet de modifier, d'enregistrer ou d'importer des projets, ainsi que de définir les préférences de GameMaker IDE. Les options disponibles sont les suivantes :

  • Nouveau projet - Cette option permet de créer un nouveau projet à partir de zéro dans GameMaker. La boîte de dialogue "Enregistrer" s'ouvre et vous invite à choisir un emplacement pour enregistrer le nouveau projet. Une fois que vous avez donné un nom au projet et un emplacement pour l'enregistrer, vous accédez à un nouvel espace de travail vierge pour commencer à travailler dessus. Les fichiers de projet sont enregistrés avec l'extension de fichier *.yyp.
  • Ouvrir le projet - Ceci ouvrira un explorateur de fichiers pour vous permettre de sélectionner un fichier de projet *.yyz précédemment créé pour le chargement. Veuillez noter que lors de l'importation, si le projet a été créé à l'aide d'une version inférieure à 2.3.0, vous obtiendrez le message d'avertissement suivant : Si vous choisissez de poursuivre, un explorateur de fichiers s'ouvrira et vous pourrez donner au projet un nom et un emplacement de sauvegarde mis à jour, afin que le projet original ne soit pas écrasé. Une fois que le projet a été converti au nouveau format 2.3.0+, il s'ouvrira dans le site IDE et vous recevrez un message indiquant que la conversion a réussi. Vous devez vérifier le rapport de compatibilité qui a été généré par le processus de conversion et voir s'il y a quelque chose que vous devez corriger avant d'essayer d'exécuter ou de compiler le projet. Notez que si vous obtenez des erreurs répétées en essayant d'ouvrir un projet réalisé avec une version antérieure à 2.3.0, vous devez contacter YoYo Games (en utilisant l'option "Report a bug" du menu d'aide), en fournissant le fichier original *.yyp que vous essayez de convertir.
  • Importer un projet - Vous pouvez importer ici des projets qui ont été précédemment exportés sous la forme d'une archive *.yyz (voir "Exporter un projet", ci-dessous), ou - si vous avez un projet réalisé avec l'ancien GameMaker : Studio 1.4, vous pouvez sélectionner cette option pour importer soit le fichier *.gmx, soit le fichier *.gmz de cette gamme de produits. Comme pour l'ouverture d'un projet, si le fichier *.yyz a été créé avec une version de GameMaker antérieure à la mise à jour 2.3.0, il devra être converti car le format du fichier du projet a changé avec cette mise à jour.

REMARQUE: pour importer un projet GameMaker antérieur dans GameMaker, il doit avoir été créé à l'aide de la version 1.4, et même dans ce cas, il se peut qu'il ne fonctionne pas directement à l'importation. Un grand nombre de choses sont différentes entre GameMaker et GameMaker : Studio 1.4, et il se peut donc que vos anciens projets nécessitent quelques modifications pour fonctionner. Veuillez consulter l'article Portage d'un jeu GM:S 1.4 vers GameMaker sur la base de connaissances GameMaker et la section du manuel consacrée aux fonctions obsolètes et aux scripts de compatibilité.

IMPORTANT ! L'importation d'un projet *.yyz qui contient des ressources de polices peut vous obliger à "régénérer" les polices en les supprimant/ajoutant, ou en modifiant la police dans l'éditeur de polices, puis en la remettant en place. En effet, les informations sur les polices existantes sont conservées par le site *.yyz lorsqu'il est exporté et, lors d'une importation ultérieure ou sur une autre machine, il se peut que les polices requises ne soient pas déjà installées dans le projet (ou qu'une police différente porte le même nom). Cette opération ne doit être effectuée que si vous rencontrez des problèmes de rendu de la police après l'importation.

  • Projets récents - Vous pouvez utiliser cette option pour afficher les projets récemment utilisés. En la sélectionnant, vous rouvrirez la page de démarrage (si vous l'avez fermée) et vous pourrez sélectionner un projet récemment utilisé dans la liste qui s'y trouve.
  • Sauvegarder le projet - Ceci sauvegardera votre jeu à l'endroit que vous avez défini lorsque vous avez créé le projet (ou à partir duquel vous l'avez ouvert).
  • Enregistrer le projet sous - Idem que ci-dessus, mais vous spécifiez l'emplacement et le nouveau nom du projet.
  • Exporter le projet - Ceci vous permet d'exporter votre projet soit comme un fichier, soit comme un modèle :
    • YYZ - Cette option exporte le projet sous la forme d'un seul fichier *.yyz. Elle est très utile pour envoyer des copies à d'autres personnes ou pour conserver des sauvegardes.
    • Comme modèle - Ceci vous permettra d'exporter votre projet comme modèle, que vous pourrez utiliser plus tard comme base pour d'autres projets sur la Page de démarrage. En sélectionnant cette option, vous obtiendrez l'invite suivante :
      Vous devez d'abord cliquer sur "Edit Template Info" pour vous assurer que les informations de votre modèle sont correctes, puis sélectionner "Export" lorsque vous ouvrez cette invite une deuxième fois pour l'enregistrer comme modèle. Vous le verrez alors sous le menu Modèle de l'écran de démarrage.
  • Préférences - En sélectionnant cette option, vous accédez à la fenêtre des préférences dans laquelle vous pouvez régler l'apparence et la convivialité de GameMaker.
  • New IDE - Lorsque vous sélectionnez cette option, une nouvelle instance du site IDE s'ouvre à vous (elle s'ouvre sur la page de démarrage).
  • Mon compte - Cette option ouvrira votre navigateur web par défaut et vous amènera à la page de votre compte YoYo.
  • Déconnexion - En sélectionnant cette option, vous vous déconnecterez de GameMaker. Vous devrez vous reconnecter avec la même licence (ou une autre) pour pouvoir utiliser le produit ultérieurement.

IMPORTANT ! Si vous avez utilisé une licence gratuite pour GameMaker et que vous achetez ensuite l'une des autres licences, vous devrez peut-être vous déconnecter de l'adresse IDE, puis redémarrer GameMaker et vous reconnecter pour vous assurer que la nouvelle licence est appliquée correctement (bien que cela doive se faire automatiquement).

  • Exit - Quittez GameMaker sur le bureau.
