Menus

La partie supérieure du site GameMaker IDE est occupée par la barre de menu. Vous y trouverez un certain nombre de menus déroulants qui peuvent être utilisés pour naviguer sur le site IDE, effectuer des tâches ou ouvrir des fenêtres supplémentaires contenant des outils pour vous aider à créer vos jeux. Notez que nous listons ici les menus par défaut disponibles, mais certains éditeurs d'actifs ajouteront des menus supplémentaires en fonction du contexte. Ces menus supplémentaires Ces menus supplémentaires sont expliqués sur les pages relatives à l'éditeur qui les a créés.

Les menus disponibles par défaut sont les suivants :

  • Le menu Fichier
  • Le menu d'édition
  • Le menu de construction
  • Le menu Windows
  • Le menu Outils
  • Le menu Marketplace
  • Le menu des mises en page
  • Le menu d'aide
