Entrée et navigation IDE

L'IDE GameMaker accepte les entrées par souris et par clavier, et de nombreuses opérations peuvent être effectuées en utilisant l'une ou l'autre ou les deux. En général, vous pouvez cliquer sur le bouton gauche de la souris pour sélectionner quoi que ce soit, utiliser / + pour sélectionner plusieurs éléments, maintenir pour faire glisser des éléments dans différents docks ou sur les espaces de travail, et aussi pour ouvrir des menus contextuels spécifiques. Notez que si vous exécutez GameMaker sur un système macOS et que vous utilisez une souris à un seul bouton, veuillez utiliser + pour obtenir un clic droit de la souris .

Il existe également un grand nombre de raccourcis clavier qui peuvent être utilisés pour naviguer dans les différents éléments de l'espace de travail et les éditeurs de ressources. Nous allons passer rapidement en revue les plus importants d'entre eux (vous pouvez trouver une liste complète ici) :

  • / + "Z" : Ceci annule l'action précédente et fonctionne dans la plupart des éditeurs. Vous disposez également de plusieurs niveaux d'annulation, de sorte que vous pouvez appuyer plusieurs fois sur cette touche pour "annuler" les modifications.
  • / + "Y" : Cette commande permet de rétablir une action précédente qui a été annulée et fonctionne dans la plupart des éditeurs. Comme pour l'annulation, il existe plusieurs niveaux de rétablissement.
  • / + "S" : Ceci forcera GameMaker à sauvegarder votre projet.
  • / + "T" : Ceci ouvrira la fenêtre Goto qui vous permettra de rechercher et d'accéder rapidement à n'importe quelle ressource de votre jeu, ainsi que de rechercher dans les préférences et les options du jeu. Une liste de tous les éléments vous est présentée et, au fur et à mesure que vous tapez le terme de recherche, cette liste est réduite en conséquence. Ainsi, si toutes vos pièces sont préfixées par "rm_" par exemple, le fait de taper ce terme réduira la liste pour n'afficher que les éléments qui commencent par ce nom.

  • / + : Cela ouvrira la fenêtre Aperçu de l'espace de travail qui peut être utilisée pour naviguer rapidement entre les éléments qui sont ouverts dans les différents espaces de travail. La fenêtre restera ouverte tant que ou sera maintenu, et en appuyant une fois sur, vous passerez à l'élément suivant dans la fenêtre. En relâchant le raccourci, vous accédez à l'élément sélectionné.

    REMARQUE: si vous remappez la liaison pour cette fenêtre afin d'utiliser plusieurs modificateurs (tels que + + B) ou une seule touche (telle que B uniquement), la fenêtre restera ouverte après avoir appuyé sur le raccourci, qu'elle soit maintenue ou non, jusqu'à ce que vous la fermiez manuellement ou que vous choisissiez un élément dans la fenêtre.

  • : Ceci ouvrira le manuel. Notez que lorsque vous utilisez le code GML Visual ou GML, vous pouvez également sélectionner une action ou une fonction (ou un mot-clé ou autre), puis appuyer sur ou cliquer sur le bouton central de la souris pour ouvrir le manuel à la page correspondante.
  • : Cette fonction peut être utilisée pour réduire ou développer toutes les différentes fenêtres ancrées dans le site IDE.

Outre ces méthodes de saisie, il existe également une prise en charge limitée des périphériques à stylet, ainsi qu'un mode spécial pour les personnes travaillant sur des projets à l'aide d'un ordinateur portable. Ces deux modes sont expliqués ci-dessous :

Mode Ordinateur portable

Si vous utilisez GameMaker sur un ordinateur portable, vous disposez d'une option supplémentaire en haut de la page IDE pour le mode ordinateur portable :

Cette fonction est activée par défaut, mais vous pouvez la désactiver en cliquant sur ce bouton. bouton. Le mode portable se combine avec certains outils du site IDE pour offrir une bien meilleure expérience lors de l'utilisation d'un pavé tactile, en simplifiant les 3 principales interactions de la souris : Panoramique, zoom et défilement. Pour ce faire, il utilise deux touches de modification : Gauche / et Gauche . Lorsque le mode ordinateur portable est activé, Gauche / et Gauche sont "réservés" pour les interactions avec la souris, comme le panoramique de l'éditeur de pièce, où Gauche / est le zoom, Gauche pour le défilement et le panoramique. Le défilement en mode ordinateur portable se comporte comme le panoramique, il doit donc être intuitive et suffisamment précise pour que vous n'ayez pas besoin d'utiliser la barre de défilement. Notez que lorsque vous maintenez ces boutons enfoncés, il vous suffit de déplacer la souris - pas de cliquer ni de faire glisser, juste de déplacer - et que s'il existe un raccourci général qui nécessite normalement l'utilisation du bouton / ou vous devez utiliser les touches touches de droite du clavier à la place.



Support de l'écran tactile

Une dernière chose que nous mentionnerons ici est que le site IDE pour GameMaker offre également une prise en charge minimale des écrans tactiles. Sur tous les systèmes d'exploitation, vous pouvez utiliser l'écran tactile pour cliquer et faire glisser des éléments dans l'espace de travail principal, et nous prenons en charge deux pointeurs simultanés. simultanés, où une deuxième pression permet d'effectuer un clic droit. Notez que sur Windows 8 et plus, GameMaker IDE prend également en charge les périphériques à stylet.
