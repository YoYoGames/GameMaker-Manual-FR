Notes de mise à jour

Chaque version de GameMaker est accompagnée de son propre ensemble de notes de mise à jour. Celles-ci contiennent des informations détaillées sur les modifications apportées ainsi qu'une liste complète de tous les bogues corrigés. C'est un outil inestimable pour décider de mettre à jour votre version actuelle, ou lorsque vous trouvez quelque chose qui pourrait être un bogue après une version.

Les notes de mise à jour sont divisées en deux ensembles différents du fait que le site IDE et le site Runtimes peuvent être mis à jour indépendamment. Vous pouvez trouver les notes de mise à jour actuelles pour le site IDE en cliquant sur le lien ci-dessous :

  • Notes de mise à jour de l'IDE

Vous pouvez trouver les notes de version actuelles pour le Runtimes à partir du lien ci-dessous :

  • Notes de version Runtime
