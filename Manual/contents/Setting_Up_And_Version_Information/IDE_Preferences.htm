Préférences de l'IDE

La fenêtre des préférences est une partie essentielle de GameMaker, car c'est là que vous pouvez configurer et personnaliser votre expérience en utilisant le site IDE, ainsi que configurer tout SDK de plateforme spécifique, etc... Dans cette section, nous couvrons les zones spécifiques de la fenêtre de préférences de IDE. Pour les plateformes cibles individuelles, veuillez consulter la section sur les préférences des plateformes cibles. Notez qu'après avoir modifié les préférences dans cette fenêtre, vous devez cliquer sur le bouton Appliquer pour qu'elles prennent effet.

Si, après avoir modifié les options ici pour personnaliser GameMaker à votre goût, vous souhaitez sauvegarder les préférences que vous avez définies, vous devez vous rendre dans ce dossier sur Windows :

%AppData%/GameMaker/<your_account_name>/

Ou ce dossier sur Mac :

~/.config/GameMaker

Puis copiez les deux fichiers suivants dans un endroit sûr :

  • local_settings.json
  • cloud_settings.json

Toutefois, si vous souhaitez réinitialiser les préférences que vous avez modifiées, vous devez cliquer sur le bouton Restaurer les valeurs par défaut en haut de la fenêtre. Notez que cette opération ne rétablira les valeurs par défaut que pour la section en cours d'édition.

Vous pouvez ouvrir les préférences de GameMaker à partir du menu Fichier situé en haut à gauche de l'IDE :

Une fois que la fenêtre des préférences est ouverte, vous disposez des différentes options suivantes (cette liste ne comprend pas toutes les préférences spécifiques à la plate-forme, pour celles-ci, veuillez vous reporter ici) :

  • Général
  • Écran de démarrage
  • Éditeur d'images
  • Éditeur de sprites
  • Éditeur d'objets
  • Éditeur de polices
  • Éditeur de salle
  • Editeur de spline
  • Éditeur de jeux de tuiles
  • Lignes du temps
  • Éditeur d'extensions
  • Éditeurs de texte
  • Préférences linguistiques
  • Réglage des plumes
  • Place du marché
  • Plugins
  • Flux d'exécution
  • Redéfinir les touches
  • Editeur de courbes d'animation
  • Navigateur d'actifs
  • Editeur de séquences
