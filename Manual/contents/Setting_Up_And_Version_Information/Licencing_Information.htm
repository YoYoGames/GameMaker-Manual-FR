Informations sur les licences

Une fois que vous avez téléchargé et installé GameMaker, la première fois que vous l'utiliserez, il vous sera demandé de saisir les détails de votre licence :

Compte YoYo

Votre licence est liée à un compte YoYo (YYA) et vous devez en avoir créé un pour pouvoir utiliser GameMaker. Ce compte contient tous les détails de votre licence pour les produits YoYo Games que vous possédez, et est également utilisé pour le marché GameMaker. Vous trouverez de plus amples informations sur le compte YYA à la page suivante :

  • Création d'un compte YoYo

Compte Opera

Vous pouvez utiliser le bouton "Opera Single Sign-In" pour vous connecter à l'aide de votre compte Opera, mais cela nécessite que vous ayez un compte YoYo et que votre compte Opera soit d'abord lié à votre compte YoYo (ce que vous pouvez faire sur cette page).

IMPORTANT Après avoir acheté une nouvelle licence pour GameMaker, nous vous recommandons de vous déconnecter du site IDE, de redémarrer GameMaker et de vous reconnecter afin de vous assurer que la nouvelle licence est correctement appliquée. Vous pouvez vous déconnecter à partir du menu Fichier.

Support Proxy

Si vous avez besoin du support Proxy pour votre installation de GameMaker, vous pouvez l'ajouter en cliquant simplement sur le Proxy. La fenêtre suivante apparaît alors :

Vous pouvez maintenant ajouter les détails de votre procuration. Certains des champs indiqués ci-dessus sont facultatifs et si vous n'en avez pas besoin, laissez simplement le champ vide. Avant de continuer, il est bon de noter que si vous avez défini un proxy dans les options Internet de Windows et qu'il ne nécessite pas de nom d'utilisateur/mot de passe, alors il doit rester tel quel et vous n'aurez probablement pas à le configurer dans GameMaker.

Si votre proxy nécessite un nom d'utilisateur et/ou un mot de passe, vous devrez les ajouter dans les champs indiqués dans la fenêtre ci-dessus. Notez que dans ces cas, les détails du nom d'utilisateur/mot de passe/proxy seront toujours requis, et que l'IP du proxy doit également avoir le port ajouté, par exemple : xx.xx.xx.xx:port.

Une fois que vous avez ajouté les informations pertinentes, cliquez sur OK et connectez-vous à GameMaker comme d'habitude.
