Préférences du navigateur d'actifs

Les préférences de l'explorateur d'actifs permettent de définir certaines propriétés pour le fonctionnement de l'explorateur d'actifs et de ses éléments. Les principales options sont les suivantes :

  • Afficher l'accès rapide: Lorsqu'elle est activée, cette option permet d'afficher la section "Accès rapide" en haut de l'Asset Browser. Cette option est activée par défaut.
  • Créer des dossiers par défaut dans un nouveau projet: Lorsqu'elle est activée, cette option fera en sorte que tous les nouveaux projets démarrent avec les dossiers d'actifs par défaut déjà créés, et lorsqu'elle est désactivée, seul un dossier de salle sera créé avec la salle par défaut requise par tous les projets. Cette option est activée par défaut.
  • Nombre maximal d'actifs à ouvrir simultanément sans avertissement: Vous pouvez sélectionner plusieurs ressources à ouvrir en même temps en utilisant / + pour les sélectionner, puis en faisant un clic droit sur et en choisissant d'ouvrir tout. Si vous avez sélectionné plus que le nombre spécifié dans cette option, un avertissement s'affichera, car le fait d'avoir trop de ressources ouvertes en même temps peut entraîner des problèmes de performance sur les machines de bas niveau. La valeur par défaut est de 10.
  • Nombre maximal d'éléments récemment modifiés: Ce paramètre régit le nombre d'actifs répertoriés dans la section "Récent" du navigateur d'actifs. La valeur par défaut est de 10.
  • Taille de la palette de couleurs: vous pouvez définir ici le nombre de couleurs disponibles dans la palette du navigateur d'actifs pour les groupes d'actifs. La valeur par défaut est de 16, avec une valeur minimale de 16 et une valeur maximale de 100.
  • Appliquer les couleurs des ressources dans l'accès rapide: Lorsqu'elle est cochée, cette option utilise les couleurs définies pour un type de ressource donné pour colorer la ressource dans la section Quick Access du navigateur. Si vous décochez cette option, la couleur de la ressource ne sera pas utilisée. Cette option est activée par défaut.
  • Les couleurs du groupe remplacent les couleurs des enfants: Lorsque cette option est cochée, la couleur du groupe de niveau supérieur est utilisée pour colorer les ressources du groupe, sans tenir compte de la couleur définie pour la ressource individuelle. Cette option est désactivée par défaut.
  • Créer un style de dialogue d'actif: Lorsque vous cliquez sur le bouton "Add Asset" dans le navigateur d'actifs, une fenêtre vous présentant les différents actifs pouvant être ajoutés à votre projet s'affiche. Cette fenêtre peut être présentée sous la forme d'une liste ou d'une grille, et ce paramètre détermine ce qui est affiché par défaut. La valeur par défaut est la grille.
  • Ordre de tri par défaut: Cette option peut être utilisée pour définir l'ordre de tri par défaut des actifs dans le navigateur, soit alphabétique, soit de A à Z, soit de Z à A. L'ordre par défaut est de A à Z.
  • Par défaut, les dossiers sont triés en premier: Lorsque cette option est activée, le navigateur de biens triera d'abord les dossiers, puis les biens "libres", tandis que si elle est désactivée, tous les biens et les dossiers seront triés ensemble. Cette option est activée par défaut.
  • Un double-clic sur le sprite ouvre l'éditeur d'images: par défaut, un double-clic sur sur n'importe quel actif ouvrira cet actif pour l'éditer, mais souvent, lorsque vous utilisez sprites, après la configuration initiale lorsque vous créez le nouvel actif, vous l'ouvrez pour éditer l'image plutôt que les propriétés de sprite. En sélectionnant cette option, vous ouvrirez l'éditeur d'image de l'actif sprite au lieu de l'actif sprite lui-même. Vous pouvez toujours ouvrir l'éditeur sprite en faisant un clic droit sur et en sélectionnant Edit. Cette option est désactivée par défaut.
  • Undouble clic sur un objet ouvre tous les événements de l'objet: Par défaut, un double clic sur sur n'importe quel objet ouvrira cet objet pour l'éditer, mais souvent, lorsque vous traitez des objets, après la configuration initiale lorsque vous créez le nouvel objet, vous l'ouvrez pour éditer uniquement les événements plutôt que les propriétés de l'objet. En sélectionnant cette option, vous ouvrirez les différents événements de l'objet dans un nouvel espace de travail, chaque événement ayant son propre onglet dans le nouvel espace de travail. Vous pouvez toujours ouvrir l'éditeur d'objets en faisant un clic droit sur et en sélectionnant Editer. Cette option est désactivée par défaut.
  • Ledégroupement de l'ordre des pièces supprime le groupe vide: Lorsqu'il est activé, le dégroupement de toutes les pièces qui appartiennent à un dossier de groupe supprime également le dossier de groupe. Lorsqu'elle est désactivée, la même action permet de dégrouper les pièces mais laisse le dossier de groupe vide dans le navigateur d'actifs. Cette option est activée par défaut.
  • Zoom par défaut (%): Ceci définit le pourcentage de zoom par défaut pour le navigateur d'actifs lorsque vous démarrez un nouveau projet. La valeur par défaut est de 100 %.
  • Zoom minimum (%): Ce paramètre définit la valeur de zoom minimale autorisée pour l'explorateur d'actifs. La valeur par défaut est de 50 %.
  • Zoom maximum (%): Ce paramètre définit la valeur de zoom maximale autorisée pour l'explorateur d'actifs. La valeur par défaut est de 250 %.
  • Afficher les lignes d'arbre: Vous pouvez activer/désactiver les lignes d'arbre qui sont affichées sur le côté des actifs dans le navigateur d'actifs. Cette option est cochée par défaut pour afficher les lignes de l'arbre.
  • Développer l'arbre lors du changement de filtre: Cette option force l'Asset Browser à développer les nœuds de l'arborescence lorsque vous modifiez les options du filtre. La désactivation de cette option empêchera cette opération. L'option est activée par défaut.

Dialogues de confirmation

Vous pouvez sélectionner ici le comportement des boîtes de dialogue de confirmation lorsque vous travaillez avec un actif dans le navigateur d'actifs. Les options disponibles sont les suivantes :

  • Réponse automatique à la suppression d'un poste: Lorsque vous supprimez un actif dans le navigateur d'actifs, vous recevez normalement un avertissement vous demandant si vous êtes sûr de vouloir continuer. Si vous réglez cette option sur "OK", le message sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Show Message".
  • Réponse automatique à la suppression d'un groupe de commandes de salles: Lorsque vous supprimez un dossier de groupe contenant des salles à partir de l'Asset Browser, vous recevez normalement un avertissement vous demandant si vous êtes sûr et si vous voulez poursuivre (qui liste toutes les salles du groupe qui seront supprimées). En réglant cette option sur "OK", le message sera supprimé comme si vous aviez cliqué sur le bouton OK pour poursuivre et supprimer toutes les pièces du groupe ainsi que le dossier de groupe lui-même. La valeur par défaut est "Show Message".
  • Réponse automatique à l'ouverture de plusieurs actifs: Comme mentionné ci-dessus dans les préférences générales du navigateur d'actifs, lorsque vous ouvrez plusieurs actifs à la fois depuis le navigateur d'actifs, vous pouvez recevoir un avertissement si le nombre dépasse le nombre spécifié. En réglant cette option sur "OK", l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Show Message".
  • Réponse automatique à la conversion de GML Visual en langage GameMaker: Lorsque vous sélectionnez l'option de conversion d'un nœud visuel GML en GML, vous serez invité à confirmer le changement, car il ne peut être annulé. En réglant cette option sur "OK l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Show Message".
  • Réponse automatique à la définition du parent d'une pièce par le biais d'un déplacement de nœud: Dans la section "Héritage" de la fenêtre Room Manager, vous pouvez faire glisser une pièce sur une autre pièce pour la définir comme enfant de la pièce sur laquelle elle est glissée. En faisant cela vous recevrez un avertissement vous demandant si vous êtes sûr et si vous voulez continuer. En réglant cette option sur "OK", le message sera supprimé comme si vous aviez cliqué sur le bouton OK pour continuer. La valeur par défaut est "Afficher le message".
  • Réponse automatique à l'erreur de nom de groupe dupliqué: Lorsque vous créez un dossier de groupe pour organiser vos ressources et que vous lui donnez le même nom qu'un autre dossier dans le navigateur, un message s'affiche vous demandant si vous souhaitez continuer et avoir deux dossiers portant le même nom. En réglant cette option sur "OK", l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Afficher le message".
  • Réponse automatique à la définition du parent d'une pièce par le biais d'un déplacement de nœud: Lorsque vous faites glisser une pièce sur une autre pièce dans le Room Manager et que vous la relâchez, un message s'affiche vous demandant si vous souhaitez faire de la pièce un "enfant" de la pièce sur laquelle elle est déposée. pièce sur laquelle vous avez glissé. En réglant cette option sur "OK", l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Afficher le message".
