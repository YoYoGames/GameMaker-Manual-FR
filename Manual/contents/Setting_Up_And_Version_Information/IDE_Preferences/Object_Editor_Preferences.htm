Préférences de l'éditeur d'objets

Les préférences de l'éditeur d'objets permettent de définir certaines propriétés de la fenêtre de l'éditeur d'objets. La section principale contient l'option suivante :

  • Cliquez pour ouvrir la méthode de l'événement: Cette option contrôle si vous devez effectuer un double clic ou un simple clic sur un événement d'objet pour l'ouvrir. Par défaut, cette option est définie sur double-clic.
  • Alpha des événements hérités: Cette option vous permet de modifier l'alpha avec lequel les événements ¡hérités seront dessinés dans la liste des événements de l'objet. Par défaut, cette valeur est fixée à 0,5 (les valeurs doivent être comprises entre 0 et 1).
  • Format de l'étiquette des événements: Ici, vous pouvez activer ou désactiver les différents textes possibles qui peuvent être appliqués à l'étiquette de l'événement dans la liste des événements de l'éditeur d'objets. Par défaut, seule la description de l'événement est activée.
  • Ajouter le contenu de l'événement par défaut: Cette option est activée par défaut. Elle permet d'ajouter un simple commentaire dans l'éditeur de code pour tous les nouveaux événements afin de vous rappeler que vous pouvez y ajouter votre code.
  • Contenu par défaut des événements: Vous pouvez modifier ici les commentaires qui sont ajoutés par défaut dans la fenêtre de code pour les nouveaux événements.

La sous-catégorie suivante existe également pour l'éditeur d'objets :

Dialogues de confirmation

Cette section traite des boîtes de dialogue qui peuvent être affichées lors de l'utilisation de l'éditeur d'objets, avec l'option suivante disponible :

  • Réponse automatique à la conversion de GML Visual en Gamemaker Language: Vous pouvez utiliser le bouton droit de la souris dans l'espace de travail visuel GML pour sélectionner l'option permettant de convertir tout le visuel GML de cet événement en code GML. L'action par défaut est l'affichage d'un message d'avertissement vous demandant si vous voulez continuer, mais vous pouvez le régler sur "Okay" et le dialogue ne sera pas affiché avant la conversion.
  • Réponse automatique au dépassement de la limite de vertex physique: cette option peut être utilisée pour contrôler le message qui s'affiche lorsque vous dépassez le nombre de vertex autorisé dans l'éditeur de forme physique. L'action par défaut est d'afficher un message d'avertissement, mais vous pouvez régler cette option sur "Okay" et la boîte de dialogue ne sera pas affichée.
  • Choix automatique lors de la suppression d'un événement: Cette option peut être utilisée pour contrôler le message qui s'affiche lorsque vous supprimez un événement dans la liste des événements de l'éditeur d'objets. L'action par défaut est l'affichage d'un message d'avertissement, mais vous pouvez régler cette option sur "OK" et la boîte de dialogue ne sera pas affichée.
