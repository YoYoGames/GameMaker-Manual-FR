Séquences Préférences

Les préférences des séquences permettent de définir certaines propriétés pour le fonctionnement de l'éditeur de séquences et de ses éléments. Les principales options sont les suivantes :

  • Afficher le cadre du canevas sur les séquences nouvellement créées: Lorsque cette option est activée, le guide intelligent du cadre du canevas sera affiché dans le canevas de l'éditeur de séquence pour toutes les nouvelles séquences. Cette option peut être activée/désactivée selon les besoins à partir des propriétés du canevas dans l'éditeur lui-même. La valeur par défaut est activée.
  • Largeur d'image du canevas sur les séquences nouvellement créées: Vous pouvez définir ici la largeur de cadre par défaut pour toutes les nouvelles séquences. La valeur par défaut est de 1366px.
  • Hauteur d'image du canevas sur les séquences nouvellement créées: Vous pouvez définir ici la hauteur de cadre par défaut pour toutes les nouvelles séquences. La valeur par défaut est de 768px.
  • Pause lors d'un clic sur le canevas: Lorsque cette option est cochée, le fait de cliquer n'importe où dans le canevas de l'éditeur de séquence met en pause la lecture de la séquence. Cette option est activée par défaut.
  • Délai de sélection de chevauchement en millisecondes: Ce paramètre contrôle le temps d'attente avant que la boîte à outils des éléments ne s'affiche pour les éléments qui se chevauchent dans le canevas de l'éditeur de séquence. La valeur par défaut est de 1000 ms.
  • Marge d'insertion du point de trajectoire de translation: Dans le canevas de l'éditeur de séquence, lorsque vous avez ajouté un poste et créé une piste de paramètres de position, des lignes en pointillé sont tracées pour illustrer la trajectoire que le poste suivra lors de son déplacement. Si vous placez votre souris à proximité de cette ligne, vous pouvez y ajouter un nouveau point de piste de paramètre. Cette option définit la distance de la ligne de translation à laquelle la souris doit se trouver avant que le marqueur de point ne s'affiche. La valeur par défaut est de 6px.
  • Créer automatiquement des pistes de paramètres par défaut: Lorsqu'elle est activée, cette option ajoute automatiquement des pistes de paramètres pour la position, l'échelle, la rotation et l'origine à toute nouvelle piste créée dans le panneau Piste de la séquence. L'option par défaut est désactivée.
  • Étirer automatiquement les actifs nouvellement créés: Cette option étire toutes les clés de poste nouvellement créées pour qu'elles correspondent à la longueur de la séquence. La valeur par défaut est désactivée.
  • Hauteur de la piste de courbe animée dans la feuille de dopage: Cette option peut être utilisée pour définir combien de lignes de piste les pistes de paramètres qui utilisent des courbes prendront dans la feuille de dopw lorsque dans la vue des images clés. La valeur par défaut est 2, mais vous pouvez également sélectionner 1 ou 3.
  • Texte par défaut pour une nouvelle piste de texte: La chaîne de texte utilisée lorsque vous créez une nouvelle piste de texte dans une séquence.
  • Couleur de sélection de l'inspecteur: couleur utilisée pour mettre en évidence la piste sélectionnée dans la liste multi-sélection de l'inspecteur.

Dialogues de confirmation

Les préférences Dialogues de confirmation permettent de modifier les messages qui s'affichent dans l'éditeur de séquence lorsque vous effectuez certaines actions. Les messages que vous pouvez modifier sont les suivants :

  • Choix automatique d'intégrer une courbe modifiée: Si vous essayez d'intégrer une courbe d'animation que vous avez modifiée, un message s'affiche par défaut pour confirmer l'action. Toutefois, si vous réglez cette option sur "Oui", le message ne s'affichera pas et la courbe sera intégrée à chaque fois. Si vous mettez cette option sur "Non", l'intégration sera impossible à chaque fois.
  • Choix automatique de convertir la courbe en clés: Si vous essayez de convertir une courbe d'animation en images clés, un message s'affiche par défaut pour vous demander si vous voulez continuer. Toutefois, si vous réglez cette option sur "Ok", le message ne s'affichera pas et vous ne pourrez pas convertir la courbe.

Guides intelligents

Vous pouvez sélectionner ici le comportement des guides intelligents lorsque vous travaillez dans l'éditeur de séquences. Les options disponibles sont les suivantes :

  • Activer les guides intelligents: Vous pouvez ici activer ou désactiver les guides intelligents dans le canevas de l'éditeur de séquences. Cette option est activée par défaut.
  • Activer les règles: Vous pouvez ici activer ou désactiver les guides intelligents dans le canevas de l'éditeur de séquences. Cette option est activée par défaut.
  • Accrocher les règles aux bords: Cette option permet aux règles de s'adapter aux bords des actifs dans le canevas des séquences (et vice versa). L'option par défaut est activée.
  • Distance d'accrochage: Il s'agit de la distance (en pixels) à laquelle un élément doit se trouver par rapport à une règle (ou une règle par rapport à un élément) pour qu'il s'accroche à la position. La valeur par défaut est 6px.
