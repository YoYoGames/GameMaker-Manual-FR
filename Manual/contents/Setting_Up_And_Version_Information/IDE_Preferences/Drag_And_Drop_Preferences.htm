Préférences de Drag And Drop™

Les préférences de Drag and Drop™ sont utilisées pour définir l'aspect et la convivialité de l'éditeur de script Drag and Drop™. Les différentes options disponibles sont listées ci-dessous avec une description de ce que fait chacune d'entre elles.

  • Emplacement par défaut de la boîte à outils DnD™: Par défaut, la boîte à outils DnD™ (où se trouvent toutes les différentes bibliothèques d'actions) est placée à droite de l'éditeur, mais vous pouvez utiliser cette option pour qu'elle soit placée par défaut à gauche.
  • Mode de visibilité de l'aperçu des nœuds DnD™: Un "aperçu" de nœud est le court texte descriptif qui apparaît à côté de certaines actions dans l'espace de travail des actions, par exemple cette action Destroy Instance :

    Vous pouvez choisir de les activer toujours, de les activer uniquement si l'action est réduite (le paramètre par défaut), ou de les désactiver complètement.
  • Réponse automatique à la conversion de DnD™ en langage GameMaker: Lorsque vous convertissez un événement ou un script Drag and Drop en code GML, vous obtenez par défaut un message vous avertissant de ce que vous êtes sur le point de faire et vous demandant si vous voulez continuer. Si vous réglez cette option sur "OK", le changement sera simplement validé sans afficher le message.
  • Réponse automatique à la suppression d'un nœud DnD™: Lorsque vous supprimez un nœud d'événement Drag and Drop, vous recevez par défaut un message vous avertissant de ce que vous êtes sur le point de faire et vous demandant si vous voulez continuer. En réglant cette option sur "OK", vous permet d'accepter la suppression sans afficher le message.
  • Taille maximale pour les vignettes d'action de la boîte à outils DnD™: En bas de la boîte à outils d'action, vous disposez d'un curseur permettant de modifier la taille des icônes Drag and Drop. Ce paramètre sera utilisé pour définir la taille maximale autorisée (64px par défaut).
  • Taille minimale pour les vignettes d'action de la boîte à outils DnD™: En bas de la boîte à outils d'action, vous disposez d'un curseur permettant de modifier la taille des icônes Drag and Drop. Ce paramètre sera utilisé pour définir la taille minimale autorisée (16px par défaut).
  • Taille par défaut des vignettes d'action de la boîte à outils DnD™: Ce paramètre sera utilisé pour définir la taille par défaut de toutes les icônes d'action de la boîte à outils (32px par défaut).
  • Nombre maximal d'actions récemment utilisées à afficher: Dans la boîte à outils des actions, il y a une bibliothèque en haut qui contient les actions que vous avez utilisées le plus récemment. Vous pouvez utiliser cette option pour modifier le nombre maximum d'actions récentes qui seront affichées (par défaut 5). qui seront affichées (5 par défaut).
  • Épaisseur de la chaîne qui relie les actions DnD™: Ce paramètre définit l'épaisseur du graphique de la chaîne qui relie les différentes actions dans un bloc de code DnD (3px par défaut).
  • Temps de pression de la souris pour commencer à faire glisser les nœuds (ms): Lorsque vous cliquez sur un nœud et que vous le faites glisser, vous repositionnez le nœud dans le code d'action. Cependant, si vous maintenez la souris enfoncée pendant un moment et que vous bougez, vous déplacerez la position du nœud dans l'espace de travail mais pas dans l'ordre du code d'action. position du nœud dans l'espace de travail mais pas dans l'ordre du code. Ce paramètre définit le nombre de millisecondes à attendre avec le bouton de la souris enfoncé avant que le nœud puisse être déplacé (pas repositionné). La valeur par défaut est 350.
  • Masquer le panneau de présentation des nœuds DnD™: Vous pouvez choisir ici de masquer le panneau de synthèse de l'éditeur de code d'action. Par défaut, celui-ci est désactivé et son activation supprime la liste en texte brut des chaînes d'action de l'éditeur.

Les préférences Commentaires contrôlent les différentes options de couleur et de formatage pour tous les commentaires qui ont été ajoutés à vos scripts d'action Drag&Drop™. ajoutés à vos scripts d'action Drag and Drop™. Ces optons sont :

  • Couleur: La couleur par défaut du texte écrit dans la fenêtre (par défaut ARGB #FF53E270).
  • Nom de la police: Le nom de la police à utiliser (les polices disponibles dépendent de celles installées sur votre système).
  • Style: Le style de police à utiliser, comme le gras ou l'italique (notez que toutes les fenêtres ne permettent pas les mêmes styles et que cela dépend également de la police utilisée).
  • Taille: La taille à laquelle la police doit être définie pour la fenêtre de sortie donnée.
  • Anti-crénelage: Vous pouvez définir ici le mode d'anticrénelage pour le rendu des polices de la fenêtre de sortie. Vous pouvez choisir de laisser le site IDE décider en sélectionnant Use Default, ou vous pouvez l'activer ou le désactiver en sélectionnant l'une ou l'autre de ces options.
