Préférences de l'écran de démarrage

Les préférences de l'écran de démarrage vous permettent de modifier le comportement de la page de démarrage. Les principales options sont les suivantes :

  • Afficher la visite: Cette option vous permet de choisir si la visite d'introduction est affichée au démarrage de IDE. Cette option est automatiquement désactivée la première fois que vous ignorez la visite.
  • Afficher l'assistant de configuration: Cette option vous permet de choisir si l'assistant de configuration s'affiche au démarrage de IDE. L'assistant de configuration vous guide essentiellement à travers les étapes nécessaires à la création d'un nouveau projet.
  • Style de la liste des projets récents: Ceci vous permet de changer l'affichage de la liste des projets récents sur la page de démarrage entre "Liste" et "Grille".
