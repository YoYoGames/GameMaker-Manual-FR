Préférences de l'éditeur de sprites

Les préférences de l'éditeur de sprites sont utilisées pour définir l'apparence et le fonctionnement de l'éditeur de sprites. Les options suivantes existent :

  • Origine par défaut de sprite : Cette option vous permet de définir la position d'origine par défaut pour toutes les nouvelles ressources sprite que vous créez. La valeur par défaut est le haut à gauche.
  • Type de masque de collision par défaut de sprite : Cette option vous permet de définir le type de masque de collision par défaut à utiliser pour les nouvelles sprites. Les choix sont les suivants : rectangle, rectangle pivoté, ellipse, diamant ou précis / précis par image. La valeur par défaut est rectangle.
  • Mode d'animation par défaut de sprite : Cette option vous permet de définir le mode d'animation par défaut pour sprites dans l'éditeur sprite. Vous pouvez choisir entre " Images par seconde" et " Images par trame de jeu", le paramètre par défaut étant " Frames per second".
  • Vitesse d'animation par défaut de sprite : cette option vous permet de définir la valeur de la vitesse d'animation par défaut lorsque vous créez une nouvelle ressource sprite. Notez que cette valeur est liée au mode d'animation et que - par exemple - 3 images par seconde n'est pas la même chose que 30 images par trame de jeu, veillez donc à définir cette valeur correctement en référence au mode préféré. La valeur par défaut est 30.
  • Largeur par défaut de sprite : Cette option vous permet de définir la largeur par défaut pour tous les nouveaux sprites créés. La valeur par défaut est 64.
  • Hauteur par défaut de sprite : Cette option vous permet de définir la hauteur par défaut pour tous les nouveaux sprites. La valeur par défaut est 64.
  • Appliquer le redimensionnement et l'échelle à l'origine de sprite : Lorsque cette option est cochée, l'origine de sprite est modifiée lorsque sprite est redimensionné ou mis à l'échelle, la nouvelle origine étant déplacée/mis à l'échelle pour correspondre aux nouvelles valeurs de sprite. Notez que ceci n'est applicable qu'aux origines personnalisées de sprite qui ne sont pas verrouillées dans l'éditeur de sprite. Si l'origine est verrouillée ou si le site sprite utilise l'une des positions par défaut (par exemple, "Centre" ou "Haut gauche", etc.), cette option sera grisée et non disponible. cette option sera grisée et ne pourra pas être modifiée. La valeur par défaut est activée.

Dialogues de confirmation

Les options disponibles ici contrôlent les messages qui s'affichent lorsque vous devez confirmer une action. Vous pouvez modifier ces paramètres pour que le message s'affiche toujours ou qu'il soit ignoré par une action spécifique (Oui ou Non) :

  • Réponse automatique à l'importation de fichiers : Lorsque vous importez une nouvelle image et qu'il en existe déjà une, vous êtes invité par défaut à poursuivre l'importation et à écraser l'image existante ou à annuler l'opération. Cette option vous permet de supprimer ce message et d'annuler l'importation en sélectionnant "Non", ou d'écraser l'image actuelle sprite en sélectionnant "Oui". La valeur par défaut est "Afficher le message".
  • Réponse automatique à l'échec du chargement d'une image : Lorsque vous importez une nouvelle image et que l'importation échoue, un message d'avertissement vous informe par défaut qu'il y a eu un échec. Cette option vous permet de supprimer ce message en sélectionnant "OK". La valeur par défaut est "Afficher le message".
