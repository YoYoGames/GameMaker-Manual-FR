Préférences d'entrée

Les préférences desaisie contrôlent certains aspects de la saisie à la souris et au clavier sur le site IDE. Les options disponibles sont les suivantes :

  • Temps de double-clic lent (ms): Certaines zones du site IDE nécessitent un double-clic plus lent pour fonctionner correctement, par exemple lors du renommage d'une ressource (un double-clic rapide ouvrira l'éditeur de ressources, tandis qu'un double-clic lent vous permettra de la renommer). Cette option vous permet de définir la vitesse de détection, avec une valeur par défaut de 500ms.
  • Scroll lock time (ms) : Cette option concerne le temps que GameMaker doit attendre entre les fenêtres de défilement qui se chevauchent. En gros, lorsque vous faites défiler la fenêtre "A", le curseur peut survoler une autre zone de défilement dans la fenêtre "B". Cette préférence définit donc le temps d'attente après l'arrêt du défilement de la fenêtre "A" avant de pouvoir faire défiler la fenêtre "B", afin d'éviter de faire défiler accidentellement des panneaux différents. La valeur par défaut est de 500 ms.
  • Enrouler le curseur autour de l'écran lors d'un panoramique: Lorsque cette option est activée, le curseur s'enroule autour des limites de l'écran lors du déplacement des espaces de travail. La valeur par défaut est activée.
  • Vitesse de défilement: permet de contrôler la vitesse de défilement dans les différents espaces de travail. La valeur par défaut est 1.
  • Distance de déplacement de l'onglet pour dé-maximiser le bureau (px): Lorsque vous faites glisser un onglet d'espace de travail à partir de sa position par défaut sur le site IDE, s'il n'y a qu'un seul espace de travail actif à ce moment-là, la fenêtre GameMaker s'agrandit et se déplace avec la souris (si vous avez plus d'un espace de travail actif, une nouvelle instance du site IDE est créée pour contenir l'espace de travail). Ce paramètre contrôle la distance à laquelle vous devez faire glisser l'onglet avant que l'espace de travail ne le reconnaisse. La valeur par défaut est de 50px.
  • Niveau de zoom minimum pour interagir avec le contenu des fenêtres (%): Cette option contrôle le point de coupure du zoom à partir duquel la souris ne pourra plus interagir avec les fenêtres de l'espace de travail (c'est-à-dire : ajouter des nœuds, faire glisser les fenêtres, etc...). La valeur par défaut est 40.
  • Le zoom avec la molette de la souris nécessite la touche Ctrl: Lorsque cette option est cochée, le zoom de tous les éditeurs (et de l'espace de travail principal) sera activé avec / + la molette de la souris , tandis que si elle n'est pas cochée, le zoom sera activé uniquement avec la molette de la souris Icône MMB.
  • Ctrl + MMB pour ouvrir l'événement hérité au lieu du manuel: Lorsque cette option est cochée, si vous maintenez la touche / + le bouton central de la souris dans un objet, l'événement hérité de l'objet (s'il en a un) s'ouvrira plutôt que le manuel.
  • Ajoute un biais à la sélection de texte pour éviter les sauts de ligne: Lorsque cette option est cochée, la hauteur de la ligne conçue lors de la sélection d'une seule ligne dans n'importe quelle fenêtre de texte sera pondérée, ce qui la rendra moins sujette à l'imprécision verticale. Cette option est activée par défaut.
  • Activez le mode ordinateur portable sur tous les appareils: Si vous cochez cette case, le bouton Mode Ordinateur portable s'affiche sur le site IDE, ce qui vous permet de l'activer ou de le désactiver, que l'appareil qui exécute GameMaker soit un ordinateur portable ou non. Cette option n'est pas cochée par défaut.
  • Forcer le mode Ordinateur portable: Ce paramètre permet de déterminer si le mode ordinateur portable est utilisable ou non, et il est activé par défaut. Vous pouvez le désactiver pour vous assurer que le mode Ordinateur portable n'est pas automatiquement activé lorsque vous démarrez le site IDE sur un ordinateur portable.
