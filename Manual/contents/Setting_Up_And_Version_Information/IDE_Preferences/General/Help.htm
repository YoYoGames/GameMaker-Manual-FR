Préférences d'aide

Le manuel d'aide est conçu pour fonctionner comme une page Web et même si les fichiers peuvent être stockés localement (en fonction de vos paramètres), il faut utiliser un navigateur externe pour les visualiser. Les paramètres suivants relatifs au manuel d'aide peuvent être modifiés ici :

  • Langue du manuel: Ce paramètre définit la langue du manuel. La valeur par défaut est "English".
  • Utiliser le manuel en ligne : Activez cette option pour utiliser le site Web du manuel en ligne chaque fois que le manuel est ouvert, ou désactivez-la pour exécuter le manuel localement. Cette option est désactivée par défaut.
  • Port du manuel d'aide: Ce paramètre définit le numéro de port du serveur web micro manuel à utiliser, si "Utiliser le manuel en ligne" est désactivé. La valeur par défaut est 51290.
