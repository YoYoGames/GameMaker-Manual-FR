Préférences d'économie d'énergie

Les préférences d'économie d'énergie affecteront la quantité d'énergie du CPU et du GPU utilisée pour générer GameMaker IDE (essentiellement, cela réduira la fréquence d'images de base à laquelle le programme fonctionne et donc la consommation d'énergie).

Les options sont les suivantes :

  • Taux de rafraîchissement de l'alimentation CA (%) : Le modificateur de framerate de base IDE lorsqu'il est connecté à une entrée d'alimentation. La valeur par défaut est 100%.
  • Taux de rafraîchissement de l'alimentation par batterie (%) : Le modificateur de framerate de base IDE lorsque la machine est alimentée par batterie. La valeur par défaut est de 35%.
  • Taux de rafraîchissement en arrière-plan (%): Le modificateur de framerate de base de IDE lorsque GameMaker n'est pas au point (indépendamment de l'alimentation). La valeur par défaut est de 15 %.
