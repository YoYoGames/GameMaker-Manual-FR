Préférences de la ligne de temps

Les préférences de la timeline sont utilisées pour définir certaines propriétés lorsque vous travaillez avec l'éditeur de timeline. Les options disponibles sont les suivantes :

  • Réponse automatique à la conversion de GML Visual en langage GameMaker: Lorsque vous utilisez l'option de menu du bouton droit de la souris à un moment de la ligne de temps pour convertir les actions de GML Visual en script (code), vous obtiendrez un message vous demandant si vous souhaitez continuer, car il s'agit d'une conversion à sens unique. Si vous changez cette option pour "Okay", le message sera supprimé et la conversion se poursuivra comme si vous aviez cliqué sur "Okay" dans la boîte de dialogue. Le paramètre par défaut est "Afficher le message".
  • Une réponse automatique à l'avertissement d'action permanente: Certaines actions dans une ligne de temps ne peuvent pas être annulées, et lorsque cela se produit, vous recevrez un message d'avertissement vous demandant si vous voulez continuer. Si vous changez cette option pour "Ok", le message sera supprimé et l'action se poursuivra comme si vous aviez cliqué sur "Ok" dans la boîte de dialogue. Le paramètre par défaut est "Afficher le message".
  • Ouvrir les nouveaux moments à l'édition lors de la création: Normalement, lorsque vous ajoutez un nouveau moment à une timeline, GameMaker ouvre automatiquement l'éditeur pour que vous puissiez y ajouter votre code ou GML Visual. Cependant, si vous décochez cette option, le moment sera ajouté sans ouvrir l'éditeur (vous pouvez l'ouvrir en double cliquant sur sur le moment). Par défaut, cette option est activée.
