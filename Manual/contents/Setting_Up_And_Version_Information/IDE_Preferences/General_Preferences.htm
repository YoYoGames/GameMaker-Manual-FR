Préférences générales

Les préférences générales sont utilisées pour définir le comportement général de l'IDE GameMaker. Les plus générales sont listées sur la page principale tandis que de nombreuses autres sont divisées en plusieurs sous-catégories, expliquées plus bas sur cette page.

  • Langue de l'IDE: Cette option vous permet de sélectionner la langue du site IDE dans la liste des options fournies. La valeur par défaut est l'anglais.
  • Police de l'IDE: Cette option vous permet de changer la police utilisée dans l'ensemble du site IDE. La police par défaut est Open Sans.
  • Fréquence d'images maximale de l'éditeur GameMaker: Cette option vous permet de définir la fréquence d'images maximale à laquelle GameMaker IDE se mettra à jour, ce qui vous permet de régler la fréquence d'images pour qu'elle corresponde à la vitesse de mise à jour de votre moniteur. La valeur par défaut est 60.
  • Skin IDE: Vous pouvez choisir ici parmi les différents habillages que le site IDE peut utiliser. Par défaut, GameMaker est livré avec deux habillages, un clair et un foncé, et la configuration initiale utilise l'habillage foncé.
  • Périphérique audio par défaut: Vous pouvez choisir ici le périphérique audio par défaut à utiliser par GameMaker IDE. Vous pouvez actualiser la liste en cliquant sur le bouton Actualiser .
  • Réponse automatique à la confirmation de sortie: Cette option vous permet de définir si un message de confirmation doit être affiché lorsque vous quittez GameMaker. Par défaut, cette option est réglée sur "Afficher le message", mais elle peut être réglée pour ignorer le message comme si vous aviez cliqué sur OK.
  • Automatic Response to The System Font Changed Dialog (Réponse automatique à la boîte de dialogue de modification de la police système) : Cette option vous permet de définir si un message de confirmation doit s'afficher lorsqu'une police système a été ajoutée, supprimée ou modifiée. Par défaut, cette option est réglée sur "Afficher le message", mais elle peut être réglée sur l'omission du message comme si vous aviez cliqué sur OK.
  • Activer les notifications: Si vous cochez cette case, vous permettrez à IDE d'afficher des notifications sur les nouvelles runtimes, etc... Cette option est activée par défaut.
  • Délai d'affichage de la notification (ms): Définit la durée (en millisecondes) d'affichage d'une notification. La valeur par défaut est 6000.
  • Déconnexion à la sortie: Si cette option est cochée, le site IDE déconnectera automatiquement la session de l'utilisateur lorsque GameMaker sera fermé. Cette option est principalement destinée aux utilisateurs du secteur de l'éducation, car, si elle est cochée, cela signifie qu'à chaque fois que le programme s'arrête, l'utilisateur est déconnecté, ce qui libère le siège de la session pour d'autres utilisateurs. Cette option est désactivée par défaut.
  • Afficher l'écran de bienvenue au démarrage: Cette option est activée par défaut et indique simplement à GameMaker d'afficher un bref écran d'accueil lorsque vous démarrez le programme. Si vous décochez cette option, l'écran d'accueil ne s'affichera pas.
  • Autoriser la fermeture des onglets vides via un raccourci: vous pouvez fermer tous les onglets ouverts à l'aide du raccourci clavier / + W, mais si vous désactivez cette option, ce raccourci ne fonctionnera plus. Cette option est activée par défaut.
  • Ouvrir de nouvelles ressources à éditer en faisant glisser des fichiers dans l'IDE: Certaines ressources (comme sprites ou les sons) peuvent être glissées sur l'espace de travail principal dans GameMaker IDE et seront détectées et ajoutées comme une nouvelle ressource dans le navigateur de ressources. Par défaut, l'éditeur de ressources correspondant sera également ouvert pour modification lorsque cela se produit, mais si vous décochez cette option, les éditeurs ne seront pas ouverts et vous devrez les ouvrir manuellement.
  • Ouvrir les nouvelles ressources pour l'édition à la création: Si cette option est cochée, chaque fois que vous créez une nouvelle ressource dans le navigateur d'actifs, l'éditeur de ressources correspondant sera ouvert automatiquement. Cette option est cochée par défaut et si vous la décochez, vous devrez ouvrir manuellement toutes les ressources nouvellement créées.
  • Recharger automatiquement les fichiers modifiés: Si cette case est cochée, le site IDE analysera et vérifiera automatiquement les fichiers pour voir s'ils ont été modifiés en dehors de GameMaker et les rechargera si c'est le cas.
  • Désactiver les animations de transition de IDE : Le site GameMaker IDE comporte un certain nombre de transitions animées pour des choses comme l'ouverture de fenêtres, etc... Cependant, elles ne sont pas du goût de tout le monde et vous pouvez constater une augmentation des performances si elles sont désactivées sur des machines plus anciennes ou sous-puissantes. Vous pouvez les désactiver ici, et elles sont activées par défaut.
  • Accepter de participer aux enquêtes de satisfaction: Lorsque cette option est cochée, vous indiquez à GameMaker que vous souhaitez participer à des enquêtes de satisfaction, ce qui signifie qu'à l'occasion, pendant que vous travaillez, un message s'affichera sur la zone sur laquelle vous travaillez et demandera une évaluation et des commentaires. Cette option est activée par défaut.
  • Activer le remplacement du DPI: Cette option vous permet de remplacer le paramètre DPI par défaut par le vôtre. Vous devez d'abord activer l'option, puis définir la valeur Raw DPi Override (la valeur par défaut est 96) ou définir le pourcentage du DPi natif (valeur par défaut 100 %). Une fois la modification effectuée, vous serez invité à redémarrer le site IDE, ce qui peut vous obliger à sauvegarder tout travail effectué sur le projet actuellement ouvert. Notez que le DPi a un pourcentage minimum de 50% et maximum de 500%, et si vous passez la souris sur l'icône d'information, vous pouvez voir le paramètre maximum recommandé pour l'affichage utilisé :

    Notez que si vous dépassez le réglage recommandé du DPi pour le moniteur, vous verrez apparaître une notification qui vous permettra de réinitialiser le DPi.

Les options ci-dessus sont les principales options disponibles dans les préférences générales , mais vous disposez d'un certain nombre d'options supplémentaires dans les sous-catégories énumérées ci-dessous :

  • Contexte
  • Compiler
  • Dialogues de confirmation
  • Débogage
  • Observateur de fichiers
  • GoTo
  • Aide
  • Entrée
  • Fenêtres de sortie
  • Chemins d'accès
  • Économie d'énergie
  • Fenêtres récentes
  • Ressources
  • Espaces de travail
