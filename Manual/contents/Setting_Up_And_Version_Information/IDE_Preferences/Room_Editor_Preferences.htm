Préférences de l'éditeur de salle

Les préférences de l'éditeur de pièce sont utilisées pour définir l'apparence et le comportement de l'éditeur de pièce. Les paramètres généraux sont les suivants :

  • Distance de départ du fondu enchaîné (px): Définit la distance de la fenêtre parent à partir de laquelle le lien de la chaîne doit s'estomper. La valeur par défaut est 10px.
  • Distance d'arrêt du fondu enchaîné (px): Définit la distance de la fenêtre parent à laquelle le lien de la chaîne doit s'arrêter en fondu. La valeur par défaut est 100px.
  • Alpha minimum de la chaîne: Définit la valeur alpha finale après le point d'affaiblissement du fondu pour l'alpha de la chaîne. La valeur par défaut est 0,3.
  • Espacement par défaut de la profondeur des calques: Lorsque vous créez plusieurs couches dans l'éditeur de pièce, une profondeur leur est automatiquement attribuée en fonction de leur position dans la liste des couches. Cette profondeur est incrémentée de la quantité indiquée ici. La valeur par défaut est de 100, donc chaque couche sera la couche précédente + 100 en profondeur.
  • Largeur par défaut de la pièce: Définit la largeur initiale qui sera sélectionnée par défaut pour une nouvelle pièce. La valeur par défaut est 1024px.
  • Hauteur par défaut de la pièce: Définit la hauteur initiale qui sera sélectionnée par défaut pour une nouvelle pièce. La valeur par défaut est 768px.
  • Ajouter des calques de salle par défaut: Lorsque vous créez une nouvelle ressource de pièce, GameMaker génère quelques calques de "base" pour vous : un calque d'arrière-plan et un calque d'instance. Vous pouvez désactiver ce comportement ici afin que les nouvelles pièces n'aient pas de calques prédéfinis. Cette option est activée par défaut.
  • Afficher les vues de la pièce: Si vous avez des vues actives dans une pièce, cela est indiqué par un contour où les limites de la vue seraient dans la pièce, et la visibilité de cette zone peut être basculée en utilisant le bouton dédié dans l'éditeur lui-même. Cependant, vous pouvez choisir de rendre les limites de vue invisibles lorsque vous ouvrez une pièce en décochant cette option (le bouton peut toujours être utilisé pour les afficher à nouveau). La valeur par défaut est activée, ce qui rend les limites de vue visibles lorsque vous ouvrez une pièce.
  • Couleur de sélection de l'inspecteur: couleur utilisée pour mettre en évidence l'élément sélectionné dans la liste multi-sélection de l'inspecteur.
  • Autres options de couleur: Elles sont liées aux différentes couleurs utilisées dans l'interface de l'éditeur de pièce. Si vous double-cliquez sur l'un des échantillons, vous pouvez modifier la couleur de ce paramètre.
  • Réglage du volume principal: Le réglage du volume principal pour la lecture de l'aperçu dans le Room Editor, une valeur entre 0 et 1. La valeur par défaut est 1.

Dialogues de confirmation

Les préférences Dialogues de confirmation permettent de modifier les messages qui s'affichent dans l'éditeur de pièce lorsque vous effectuez certaines actions. Les messages que vous pouvez modifier sont les suivants :

  • Réponse automatique à la mise à jour des profondeurs de calque: lorsque vous modifiez les profondeurs de calque, un message s'affiche pour vous avertir que cette opération entraîne la mise à jour automatique de toutes les autres profondeurs de calque. Avec cette option, vous pouvez choisir d'afficher le message, de mettre à jour les couches comme si vous aviez cliqué sur "Ok", ou de ne rien mettre à jour comme si vous aviez cliqué sur "Non".
  • Réponse automatique à l'avertissement de couche verrouillée: Si vous essayez de supprimer une couche dans la pièce et qu'elle est verrouillée, un message s'affiche par défaut. Cependant, si vous réglez cette option sur "Ok", le message ne s'affichera pas et vous ne pourrez pas supprimer la couche.
  • Réponse automatique à l'avertissement d'élément verrouillé: Si vous essayez de supprimer un élément qui est verrouillé, un message s'affiche par défaut. Toutefois, si vous réglez cette option sur "Ok", le message ne s'affichera pas et vous ne pourrez pas supprimer l'élément.
  • Réponse automatique à l'avertissement de verrouillage de la couche du propriétaire de l'élément: Si vous essayez de supprimer un élément sur un calque et que le calque est verrouillé (même si l'élément ne l'est pas), un message s'affichera par défaut pour vous informer que cela ne peut pas être fait. Toutefois, si vous réglez cette option sur "Ok", le message ne s'affichera pas et vous ne pourrez pas supprimer l'élément.
  • Création d'un nouveau calque d'actif lorsque vous faites glisser des séquences/sprites sur des calques inadaptés: Si vous essayez d'ajouter un sprite (ou une séquence) à un calque qui n'est pas un calque d'actif, un message s'affichera par défaut pour vous informer que cela ne peut pas être fait et vous demandera si vous voulez créer un nouveau calque d'actif pour l'élément qui est glissé. Toutefois, si vous réglez cette option sur "Ok", le message ne s'affichera pas et une nouvelle couche sera automatiquement créée pour sprites (et les séquences).
  • Réponse automatique à la suppression d'un élément d'une pièce parent: Ici vous pouvez choisir ce qui se passe lorsque vous supprimez un élément qui fait partie d'une pièce parent (et donc présent dans toutes les pièces enfants). Par défaut, un message d'avertissement s'affiche et vous laisse décider de ce que vous voulez faire, mais vous pouvez choisir "Supprimer tout" pour supprimer le message et poursuivre la suppression de l'élément dans la pièce parent et dans toutes ses pièces enfants, ou vous pouvez sélectionner "Garder tout" pour supprimer le message et annuler la suppression.
  • Réponse automatique à la suppression d'une couche d'une pièce parent: Ici, vous pouvez choisir ce qui se passe lorsque vous supprimez une couche qui fait partie d'une pièce parent (et donc présente dans toutes les pièces enfants). Le défaut est de vous montrer un message d'avertissement et de vous laisser décider quoi faire, mais vous pouvez le régler sur "Supprimer tout" pour supprimer le message et aller de l'avant avec la suppression de la couche de la salle parent et de tous ses enfants, ou vous pouvez sélectionner "Garder tout" pour supprimer le message et annuler la suppression.
  • Réponse automatique à l'erreur de déplacement d'un actif non valide: Si vous essayez d'ajouter un actif à une couche qui n'est pas désignée pour ce type d'actif (comme l'ajout d'un chemin à une couche d'instance), un message s'affichera par défaut pour vous informer que cela ne peut pas être fait. Toutefois, si vous réglez cette option sur "Ok", le message ne s'affichera pas et vous ne pourrez pas ajouter l'actif. Notez que cette option ne s'applique pas aux instances, sprites ou aux séquences.
  • Création d'un nouveau calque d'instance lorsque l'on fait glisser des objets sur des calques inadaptés: Si vous essayez d'ajouter une instance d'objet à un calque qui n'est pas un calque d'instance, un message s'affichera par défaut pour vous informer que cela ne peut pas être fait et vous demander si vous voulez créer un nouveau calque d'instance pour l'élément que vous faites glisser. Cependant, si vous réglez cette option sur "Ok", le message ne s'affichera pas et une nouvelle couche sera automatiquement créée pour l'instance.
  • Réponse automatique à une erreur de sélection de calque invalide: Vous pouvez choisir ici ce qui se passe lorsque vous faites glisser une ressource sur l'éditeur de pièce alors que vous avez sélectionné plusieurs couches dans l'éditeur de couches (ou que vous n'avez sélectionné aucune couche). Par défaut, un message d'avertissement est affiché, mais vous pouvez régler cette option sur "OK" pour supprimer le message et continuer, sans ajouter de ressources.
  • Réponse automatique en cas d'erreur de déplacement d'une couche verrouillée ou invisible: Vous pouvez choisir ici ce qui se passe lorsque vous faites glisser une ressource sur une couche qui a été verrouillée ou qui est signalée comme invisible. Par défaut, un message d'avertissement est affiché, mais vous pouvez définir cette option sur "OK" pour supprimer le message et continuer, sans ajouter de ressources.

Paramètres de la grille

Les préférences Paramètres de la grille permettent de modifier le fonctionnement de la superposition de la grille, avec les options suivantes :

  • Espacement horizontal de la grille (px): Ceci définit l'espace horizontal des cellules de la grille de la pièce en pixels. La valeur par défaut est de 32px.
  • Espacement vertical de la grille (px): Ceci définit l'espacement vertical des cellules de la grille de la pièce en pixels. La valeur par défaut est de 32px.
  • Afficher la grille: Cette option permet d'activer ou de désactiver la grille de la pièce lorsque vous ouvrez une pièce pour la première fois. Elle est activée par défaut, mais même si elle est désactivée, vous pouvez la réactiver en utilisant le bouton approprié dans l'éditeur de pièce lui-même.
  • Snap to grid: Cette option vous permet de déterminer si les actifs d'une pièce doivent s'adapter à la grille de la pièce par défaut ou non lors de l'ouverture initiale de la pièce. Cette option est activée par défaut, mais même si elle est désactivée, vous pouvez la réactiver en utilisant le bouton approprié dans l'éditeur de salle lui-même.
  • Couleur des lignes de la grille: cette option vous permet de définir la couleur (y compris la composante alpha) des lignes de la grille de la pièce. La valeur par défaut est ( ARGB) #30FFFFFF.


Guides intelligents

Cette section détaille les préférences des guides intelligents pour l'éditeur de pièce. Les options disponibles sont les suivantes :

  • Activer les guides intelligents: Ceci est utilisé pour activer/désactiver l'utilisation des guides intelligents dans l'éditeur de pièce. Par défaut, l'option est activée.
  • Activer les règles: Cette option permet d'activer/désactiver les règles latérales dans l'éditeur de pièce. L'option par défaut est activée.
  • Accrocher les règles aux bords: Lorsqu'elle est activée, cette option permet d'accrocher les guides glissés dans l'éditeur de pièce à partir des règles à tous les éléments placés dans la pièce. L'option par défaut est activée.
  • Snap Distance: Ce paramètre indique à l'éditeur de salle à quelle distance un guide doit se trouver d'un élément avant qu'il ne s'accroche au côté de l'élément. La valeur par défaut est de 6px.
