Informations sur les licences et les versions

Il existe différentes versions des licences que vous pouvez obtenir pour GameMaker, dont le choix dépendra largement du public que vous souhaitez cibler et du fait que vous souhaitiez simplement vous amuser en créant des jeux ou que vous souhaitiez également les vendre et les distribuer.

Pour des informations actualisées sur les licences GameMaker, veuillez consulter la page suivante :

  • Quelle licence me convient le mieux ?

Quelle que soit la version de GameMaker que vous utilisez, elles nécessitent toutes la même configuration de base avant d'être utilisées. Les sections ci-dessous traitent toutes de ce sujet et comprennent des informations sur la mise à jour des sites IDE et Runtimes, les kits SDK requis pour des modules cibles spécifiques et les préférences de IDE pour adapter le fonctionnement de GameMaker à votre propre style de fonctionnement.
