Préférences de la plate-forme

Cette section du manuel couvre les différentes préférences spécifiques à la plate-forme qui doivent être configurées avant que vous puissiez compiler vos jeux vers certaines cibles. Ces préférences ne doivent généralement être configurées qu'une seule fois pour commencer (bien que les futures mises à jour des moteurs d'exécution puissent vous obliger à modifier certaines choses, qui seront mentionnées dans les notes de version). Elles sont utilisées pour indiquer les SDKnécessaires et définir les options de compilation de base pour que GameMaker compile vos projets de jeu pour une plate-forme donnée. Notez que les plates-formes disponibles dépendent de la licence que vous possédez et qu'elles ne sont pas toutes disponibles.

Vous trouverez ci-dessous une liste de toutes les plateformes cibles disponibles pour les licences standard :

  • Android
  • HTML5
  • Opéra GX
  • iOS
  • tvOS
  • macOS
  • Windows
  • Windows UWP

Notez qu'en plus de configurer ces préférences, vous devrez également configurer les périphériques ciblés par certaines de ces plates-formes, sinon vous ne pourrez pas compiler ni tester vos projets. Voir la section sur la compilation et la section sur le gestionnaire de périphériques pour plus d'informations.

Le manuel n'indique pas les préférences des différentes consoles (Xbox, PlayStation et Nintendo Switch) en raison des différentes restrictions légales maintenues par les différentes sociétés. Si vous disposez d'une licence de console, vous pouvez obtenir ces informations auprès du service d'assistance de YoYo Games, mais vous devez d'abord contacter YoYo Games avec l'adresse électronique de votre console et demander la permission de consulter les différentes sections.
