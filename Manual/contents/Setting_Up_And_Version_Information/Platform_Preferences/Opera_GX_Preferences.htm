Préférences d'Opera GX

Les préférences d'Opera GX comportent les options suivantes :

  • Emplacement du SDK Emscripten: Il s'agit de l'emplacement du SDK Emscripten, requis pour les constructions YYC faites pour Opera GX. Lisez Configuration pour Opera GX pour obtenir des informations sur la configuration du SDK.
  • Port par défaut du serveur Web: permet de modifier le port par défaut du serveur Web d'Opera GX. La valeur par défaut est 51264.
  • Adresse du serveur Web par défaut: Vous pouvez modifier ici l'adresse du serveur Web par défaut du serveur Web du Opera GX. La valeur par défaut est 127.0.0.1.
  • IPs autorisées par défaut du serveur Web: Vous pouvez ajouter ici toutes les IP autorisées pour le serveur Web et elles seront définies par défaut pour tous les dispositifs ajoutés via le Target Manager.
