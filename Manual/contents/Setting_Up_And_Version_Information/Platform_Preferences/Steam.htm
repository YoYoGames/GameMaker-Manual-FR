Préférences de Steam

Le SDK Steam est utilisé pour les constructions de Windows, macOS et Ubuntu (Linux) qui doivent être distribuées par la plateforme Steam. Cette section contient les options de configuration nécessaires pour le faire fonctionner dans vos projets :

  • Chemin d'accès au SDK Steam : Le chemin du système vers l'endroit où vous avez installé le SDK Steam. Une fois que vous avez défini ce chemin d'accès au SDK, vous devez activer Steam dans les options du jeu, puis ajouter l'App ID pour chaque plate-forme. pour chaque plateforme, ajouter l'App ID.
