Préférences HTML5

Les préférences HTML5 comportent les options suivantes :

  • Type d'emballage par défaut: Lors de la création d'un paquet final exécutable HTML5, vous pouvez choisir entre Package as loose files - qui génère simplement tous les fichiers requis dans un dossier - et Package as Zip - qui crée un fichier compressé ZIP contenant le jeu et tous les fichiers supplémentaires. Par défaut, vous pourrez choisir laquelle de ces deux options vous souhaitez choisir lors de la compilation, mais vous pouvez définir cette préférence pour toujours utiliser l'une ou l'autre. Notez que lorsque cette option est réglée sur "Afficher le dialogue", dans la fenêtre de dialogue qui s'affiche lors de la compilation, le fait de choisir "Se souvenir de l'option d'emballage" réglera cette préférence sur l'option que vous avez sélectionnée.
  • Port du serveur Web par défaut: permet de modifier le port par défaut du serveur Web HTML5. La valeur par défaut est 51264.
  • Adresse du serveur Web par défaut: Vous pouvez modifier ici l'adresse du serveur Web par défaut pour le serveur Web HTML5. La valeur par défaut est 127.0.0.1.
  • IPs autorisées par défaut du serveur Web: Vous pouvez ajouter ici toutes les IP autorisées pour le serveur Web et elles seront définies par défaut pour tous les dispositifs ajoutés via le Target Manager.

IMPORTANT Lorsque vous utilisez un périphérique cible particulier (c'est-à-dire un navigateur), les valeurs d'IP et de port définies pour ce périphérique particulier sont utilisées à la place de ces préférences. Consultez le Gestionnaire de périphériques pour voir une liste de vos périphériques avec leurs détails.
