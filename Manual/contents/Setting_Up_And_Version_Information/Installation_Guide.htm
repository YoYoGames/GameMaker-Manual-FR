Guide d'installation

GameMaker est conçu pour être utilisé sur des ordinateurs de bureau macOS et Windows présentant les spécifications minimales mentionnées ici.

Vous pouvez télécharger le programme d'installation de GameMaker depuis la page de votre compte YoYo sur le site Web de YoYo Games. Une fois installé, la première fois que vous lancerez GameMaker, vous serez peut-être invité à effectuer une mise à jour avant de pouvoir utiliser le produit. Après l'installation des mises à jour, vous devrez fournir les détails de la licence de votre compte YoYo Games (voir ici pour plus d'informations) avant de pouvoir utiliser le produit, ce qui signifie que la première fois que vous utilisez GameMaker , vous devez disposer d'une connexion Internet. Après la connexion initiale, vous devriez pouvoir utiliser le produit hors connexion, mais il vous demandera périodiquement de vous connecter pour vérifier votre compte de connexion.

Il convient de noter que, selon le type de licence dont vous disposez, vous devrez peut-être installer des composants supplémentaires ou suivre des étapes supplémentaires pour configurer et utiliser GameMaker sur des plates-formes spécifiques. Pour obtenir des informations complètes sur les exigences de chaque plate-forme, consultez cet article sur le site YoYo Games Helpdesk : GameMaker Required SDKs.

Versions stables et bêta

La version que vous téléchargez depuis la page web YoYo Games devrait être la version stable actuelle de GameMaker, mais il se peut qu'une autre version " Beta" soit disponible. Elle est utilisée pour présenter ou tester de nouvelles fonctionnalités avant qu'elles ne soient intégrées à la version stable finale. Les détails complets de la version sont les suivants :

  • Stable: Cette version est celle que vous utiliserez pour le développement général et est recommandée pour une utilisation avec tout produit commercial. Il se peut que certaines fonctionnalités disponibles dans les versions bêta soient absentes, mais c'est parce qu'elles n'ont pas été entièrement testées ou qu'elles nécessitent un travail supplémentaire.
  • Bêta: La version bêta de GameMaker est destinée à ceux qui souhaitent essayer de nouvelles fonctionnalités avant qu'elles ne soient ajoutées à la version stable, ou à ceux qui ont besoin d'une correction prochaine pour un problème. Il n'est pas recommandé d'utiliser la version bêta pour la production du jeu final, à moins qu'elle ne contienne une correction importante d'un bug qui ralentit votre projet. Vous pouvez obtenir la version bêta de GameMaker à partir de la page "Téléchargements" de votre compte YoYo. Elle peut être installée et exécutée en parallèle avec la version stable. access Veuillez noter que si vous exécutez GameMaker via Steam , vous n'aurez pas accès à la version bêta du produit en utilisant Steam, mais vous pourrez toujours télécharger et utiliser la version de bureau.
