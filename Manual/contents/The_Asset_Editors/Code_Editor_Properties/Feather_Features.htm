Caractéristiques des plumes

Feather fournit une complétion de code intelligente et une vérification améliorée de la syntaxe dans vos scripts de code GML, ainsi que des options de refactoring intelligentes.

Cette page énumère les fonctionnalités que Feather offre dans l'éditeur de code GML pour les scripts et les événements d'objets. Ces fonctionnalités ne seront disponibles que si Feather est enabled.

Informations sur le survol

En passant la souris sur une fonction, sprite, une couleur ou toute autre valeur spéciale, vous obtiendrez des informations à son sujet. Vous pouvez voir quelques exemples ci-dessous.

Refonte des actifs

Si vous renommez un asset dans votre Asset Browser, Feather mettra automatiquement à jour toutes les mentions de cet asset dans le code de votre projet, afin que ces références ne soient pas rompues. Ceci peut être activé ou désactivé dans les paramètres de Feather.

Vous pouvez également renommer une ressource dans l'éditeur de code. Placez votre curseur texte sur la référence de l'actif, puis appuyez sur CTRL/CMD + SHIFT + R.

Saisissez un nouveau nom et cliquez sur "Apply". La ressource elle-même, et toutes les références à celle-ci dans votre projet, seront renommées.

Réparations rapides

Lorsque vous obtenez une erreur ou un avertissement dans votre code, placez votre curseur texte sur la ligne signalée et appuyez sur CTRL/CMD + Q pour faire apparaître le menu des corrections rapides. Vous pouvez également cliquer sur l'icône de la plume dans la gouttière.

Ce menu vous indique une solution rapide si possible, et vous permet de désactiver la règle pour cette ligne, ou de modifier sa sévérité.

L'exemple ci-dessus montre une solution rapide utilisée pour corriger un événement où la cible de surface n'était pas réinitialisée après avoir été modifiée.

Lors de la création d'une fonction, le menu Correction rapide vous permet de générer automatiquement des commentaires JSDoc pour la fonction.

Trouver toutes les références

Feather vous permet de trouver toutes les références à une variable dans votre projet.

Placez votre curseur texte sur une variable, et appuyez sur F3. Vous verrez apparaître une petite barre de recherche en haut de votre espace de travail :

Naviguez vers la référence précédente ou suivante en appuyant sur la flèche correspondante, ou fermez-la.

Cette recherche est intelligente. Au lieu de rechercher simplement une chaîne de texte, elle recherche les références de la variable sélectionnée dans sa portée déclarée. Cela signifie que deux variables portant le même nom, mais existant séparément dans deux objets différents, ne correspondront pas.

Liste de toutes les références

Placez votre curseur texte sur une variable, et appuyez sur SHIFT + F3. Cela ouvrira la fenêtre Résultats de la recherche et affichera toutes les références de la variable sélectionnée dans une liste.
