Extraits de code

Un outil très pratique dont vous disposez lorsque vous modifiez votre code est l'utilisation des extraits de code. Lorsque vous utilisez l'éditeur de script, vous pouvez appuyer sur F4 pour ouvrir la fenêtre contextuelle des extraits de code, qui vous permet de sélectionner dans une liste l'une des méthodes de code les plus couramment utilisées :

Dans cette fenêtre, vous pouvez sélectionner avec la souris le snippet à utiliser, ou vous pouvez appuyer sur le raccourci clavier associé listé à droite. Ceci ajoutera le snippet à votre code pour l'éditer.

Vous pouvez également définir vos propres extraits de code si vous le souhaitez. Avant de faire cela, vous devez créer un fichier appelé "snippets.txt" dans l'un des répertoires suivants :

  • Les fenêtres : %ProgramData%\GameMaker\User\
  • macOS : /Users/Shared/GameMaker/User/

Ce dossier n'est pas modifié lors d'une mise à jour de GameMaker, votre fichier édité restera donc intact (mais toute modification apportée au fichier des extraits de base dans le répertoire d'installation sera annulée).

Une fois le fichier créé dans le répertoire de l'utilisateur, vous pouvez le modifier avec n'importe quel éditeur de texte en respectant les règles suivantes :

  • Chaque extrait doit être sur une ligne distincte (il ne doit pas y avoir de lignes vides).
  • Chaque snippet doit commencer par la touche de raccourci à utiliser, suivie de "-", puis du nom du snippet (qui est celui qui est affiché dans le menu), suivi de deux points ":", par exemple :

I - Instance Create :

Après les deux points, vous ajoutez le bout de code suivant

I - Création d'instance:instance_create_layer(x, y, |layer|, object) ;

Le code que vous ajoutez doit également respecter un format spécifique où :

  • Tout le code est sur une seule ligne
  • Vous utilisez "$" pour une nouvelle ligne.
    • REMARQUE Lorsque vous créez plusieurs lignes, il doit y avoir un espace entre chaque signe "$". Par exemple, vous écrirez $ $ $ pour ajouter trois traits de suspension.
  • Vous placez le texte qui doit être mis en évidence entre les symboles "|". L'éditeur de code sélectionnera automatiquement ce texte pour le modifier lorsque le snippet sera ajouté.

Consultez les exemples qui se trouvent déjà dans le fichier de base des snippets pour voir comment il est mis en place en suivant les règles ci-dessus. Vous pouvez trouver le fichier de base dans le répertoire d'installation de GameMaker:

  • Windows : GameMaker\TextEditor\snippets.txt
  • macOS : /Applications/GameMaker.app/Contents/MonoBundle/TextEditor/snippets.txt
