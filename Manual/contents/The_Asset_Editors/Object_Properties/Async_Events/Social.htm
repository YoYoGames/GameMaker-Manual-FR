Social

Cet événement ne peut être déclenché que par les rappelsde diverses fonctions sociales et fonctions UWP Xbox Live spécifiques et renverra une carte DS Map stockée dans la variable async_loadcontenant différentes paires clé/valeur en fonction du callback de la fonction qui a déclenché l'événement.

Cet événement peut également être déclenché par la lecture vidéo.

La carte DS contient un certain nombre de clés, dont la plus importante est la clé "id". Elle renvoie une constante qui peut ensuite être vérifiée dans le code pour déterminer lequel des nombreux rappels l'événement a reçu. Une fois que vous avez analysé cette clé et comparé la valeur renvoyée avec les constantes disponibles, vous pouvez continuer à extraire le reste des informations de la carte.

Les pages de fonctions sur les réalisations et les classements dans la section Référence détaillent les fonctions et les rappels ainsi que les constantes utilisées pour déclencher cet événement, mais il y a aussi certaines circonstances dans lesquelles il sera déclenché sans appel de fonction du jeu en cours d'exécution :

  • lorsque votre jeu est lancé depuis le tableau de bord de l'OS
  • lorsque vous terminez un défi
  • lorsqu'un autre joueur termine un défi
  • quand le jeu est lancé à partir du tableau de bord de l'OS en sélectionnant un défi

N'importe lequel des rappels ci-dessus déclenchera l'événement social et une carte async_load sera générée avec les détails suivants (notez qu'il y a une valeur de clé "id" différente pour définir chacune des différentes raisons pour lesquelles l'événement a été appelé, mais tous les autres contenus de la carte sont les mêmes) :

  • "id" - La valeur de cette clé dépendra du type de rappel déclenchant l'événement (principalement utilisé par les extensions).
  • "playerid" - L'identifiant du joueur pour le défi.
  • "issuerid- L'identifiant de la personne qui a lancé le défi.
  • "state- L'état du défi, qui aura une valeur de 0 à 3 (sous forme de chaîne) pour invalide, en attente, terminé ou refusé.
  • "message" - Le message texte pour le défi.
  • "issueddate- La date d'émission du défi.
  • "completeddate" - La date d'achèvement du défi.
  • "type" - Le type de défi lancé.
  • "identifier" - La chaîne d'identification du défi.
  • "score" - Le score est lié au défi.
