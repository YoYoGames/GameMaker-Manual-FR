Comment désactiver les extensions ?

GameMaker possède des extensions pour certaines plateformes cibles, par exemple Steamworks et GDK.

Si vous utilisez ces deux extensions, il est préférable de désactiver l'extension GDK lors de la compilation pour Steam, et vice-versa.

Ce problème n'est pas limité à ces deux extensions, et s'applique à toute extension destinée à une cible spécifique.

Désactiver les extensions de manière sélective

Tout d'abord, établissez une configuration pour chacune de vos plates-formes cibles, si vous ne l'avez pas déjà fait.

Cet exemple n'utilise que Steamworks et GDK.

Sélectionnez la configuration pour une plate-forme cible. Supposons que vous ayez sélectionné la configuration GDK.

Ouvrez une extension qui ne doit pas être utilisée pour GDK. Pour cet exemple, ouvrez l'extension Steamworks.

Désactivez toutes les cibles sous Copies vers. Cela permettra de s'assurer que l'extension Steamworks n'est pas exécutée lors de l'utilisation de la configuration GDK.

Maintenant, passez à votre configuration Steamworks. Ouvrez l'extension GDK, et désactivez toutes ses cibles.

C'est fait !

Maintenant, lorsque vous construisez pour GDK, utilisez la configuration GDK ; lorsque vous construisez pour Steam, utilisez la configuration Steamworks, et ainsi de suite.

En utilisant les configurations de cette manière, vous pouvez contrôler quelles extensions sont utilisées pour construire quelles plateformes.

Si vous avez d'autres extensions, appliquez-leur le même traitement : désactivez leurs exportations dans les configurations qui n'en ont pas besoin.

Si vous avez d'autres plates-formes cibles, ajoutez des configurations pour elles, et modifiez les extensions qui sont utilisées pour elles.
