L'éditeur d'images

L'éditeur d'images est l'endroit où vous pouvez créer vos propres sprites ou modifier ceux que vous avez importés ou créés précédemment. Si vous avez utilisé d'autres outils de création d'images, cela devrait vous être familier, mais il y a quelques points à noter lors de l'utilisation de l'éditeur d'images de GameMaker qui ne sont pas forcément évidents :

  • Lorsque vous créez plusieurs cadres, vous pouvez dessiner sur ceux-ci pendant qu'ils s'animent ! Ainsi, si vous appuyez sur le bouton de lecture dans la barre de contrôle des images, puis sélectionnez un pinceau et commencez à dessiner dans la fenêtre de l'éditeur, vous verrez que les images s'animeront dans l'éditeur et que ce que vous dessinerez sera ajouté à l'image correspondante à ce moment-là. sprites Vous pouvez expérimenter avec différentes fréquences d'images jusqu'à ce que vous obteniez quelque chose qui vous plaise, mais une fois que vous aurez compris comment cela fonctionne, vous pourrez créer des effets et des sites dynamiques en un rien de temps.
  • Vous pouvez retirer l'onglet Éditeur d'images de la fenêtre principale de l'espace de travail et le placer dans sa propre fenêtre. Cela peut s'avérer pratique lorsque vous utilisez plusieurs moniteurs, par exemple.
  • Lorsque vous utilisez les fonctions de copie pour, par exemple, copier une sélection d'une image, la sélection copiée est ajoutée à votre liste de pinceaux. Cela signifie que vous pouvez sélectionner une partie d'une image, puis l'utiliser avec les autres outils de peinture pour tracer des lignes, des polygones ou simplement peindre librement. Le pinceau peut également être mis à l'échelle et mélangé avec une couleur. Notez que la fonctionnalité Coller n'est pas destinée à être utilisée en interne dans l'éditeur, car tout ce que vous copiez est ajouté à la liste des pinceaux et peut donc être sélectionné pour être "collé" de cette façon. Au lieu de cela, le collage vous permet simplement de capturer quelque chose à partir d'un programme externe et de le coller dans l'éditeur d'images.
  • Vous pouvez disposer de plusieurs calques pour chaque sprite grâce à l'éditeur de calques très pratique. Ces calques peuvent être ordonnés et même placés dans des dossiers pour faciliter la création d'images plus complexes.
  • Vous pouvez passer rapidement d'une sous-image (cadre) à l'autre dans l'éditeur en utilisant les touches "1" et "2" du clavier.

Ce ne sont là que quelques-unes des fonctions les plus avancées de l'éditeur d'images. Pour découvrir l'ensemble des fonctionnalités, veuillez lire les articles ci-dessous qui détaillent les fonctions de chaque partie de l'éditeur :

Vue du cadre

Cette partie de la fenêtre montre les images dans l'ordre dans lequel elles seront animées. Vous pouvez cliquer sur le bouton à gauche pour ajouter d'autres images - ou utiliser le menu Image - et vous pouvez également cliquer et faire glisser n'importe quelle image pour modifier sa position sur la ligne de temps. En haut à gauche de la vue des images, vous pouvez activer l'option Onion Skinning , ce qui signifie que dans une seule image de n'importe quelle animation sprite, vous pouvez choisir de voir n'importe quelle image précédente ou suivante comme une couche semi-transparente sous l'image que vous êtes en train de modifier. L'avantage de cette méthode est que chaque image est dessinée légèrement plus claire que la précédente, ce qui vous permet de voir la progression de l'animation d'une image à l'autre tout en dessinant la suivante.

En activant cette option, des contrôles supplémentaires s'affichent sur les cadres. Vous pouvez cliquer sur et faire glisser pour définir le nombre de cadres précédents à afficher ainsi que le nombre de cadres suivants.

Vous pouvez définir certaines propriétés pour la pelure d'oignon à partir du menu View conext, qui est expliqué plus bas dans cette page.

Sous le bouton Onion Skin se trouve le bouton Loop Control qui permet de définir si le site sprite doit tourner en boucle (revenir à l'image 1 lorsque la dernière image est atteinte) ou en ping-pong (revenir en arrière dans les images lorsque la dernière image est atteinte). Le bouton Play , situé en bas, permet de démarrer/arrêter l'aperçu de l'animation.

Vous pouvez cliquer sur n'importe quel cadre pour le sélectionner et le modifier, et un cadre sélectionné aura un bouton Supprimer dans le coin supérieur sur lequel vous pouvez cliquer pour supprimer le cadre de sprite. Vous pouvez également cliquer sur / + sur plusieurs cadres pour les sélectionner ensemble, auquel cas cliquer sur le bouton Supprimer supprimera tous les cadres sélectionnés. Enfin, vous pouvez utiliser les raccourcis clavier standard pour Couper, Copier et Coller pour modifier les cadres sélectionnés : / + "X", / + "C", et / + "V".

Informations sur le cadre

Cette barre contient les informations sur les images. Vous pouvez y définir la vitesse à laquelle les images de l'animation doivent être lues à l'aide du champ de saisie situé à gauche. La vitesse des images peut être basée sur "Images par seconde" ou "Images par image de jeu". Le nombre total de cadres disponibles est affiché à droite avec le cadre actuel qui a été sélectionné, et vous pouvez réduire cette barre (la cacher) en cliquant sur l'icône à gauche. Vous pouvez régler cette barre pour qu'elle soit toujours ouverte ou toujours fermée dans les préférences de l'éditeur d'images.

Notez que la valeur utilisée ici pour la vitesse d'animation définira également la manière dont le site sprite sera animé lorsqu'il sera utilisé dans un jeu, soit en tant qu'actif sprite sur un calque, soit en tant que sprite assigné à une instance, bien que vous puissiez modifier cela à runtime en utilisant les fonctions et variables appropriées.

Contrôles du canevas

Les contrôles du canevas traitent de divers aspects de l'affichage du canevas de dessin principal et de son contenu. Les options sont les suivantes :

  • Toggle Canvas Grid : Cette option permet d'activer ou de désactiver la grille du canevas. Il s'agit d'une grille que GameMaker dessine sur le canevas principal pour le diviser en sections, et dont la taille par défaut est de 1px. Cependant, si vous cliquez sur l'icône Menu Grille , vous ouvrirez les options de la grille :
    Ces options vous permettent de définir la couleur et l'alpha de la grille, ainsi que les valeurs des cellules de la grille le long des axes X et Y. Vous avez également la possibilité d'activer ou de désactiver la grille. Vous avez également la possibilité d'activer ou de désactiver l'accrochage de la grille (désactivé par défaut).
  • Contrôles du zoom du canevas : Ces boutons contrôlent le niveau de zoom actuel du canevas. Vous pouvez effectuer un zoom avant ou arrière et un clic sur le bouton réinitialisera le canevas pour qu'il soit 1:1 avec l'image en cours d'édition. Vous pouvez également cliquer sur le bouton Window Fit pour que l'ensemble du canevas de l'image s'adapte à l'espace de travail actuel de l'éditeur (un zoom avant/arrière sera effectué pour l'adapter). Notez que vous pouvez également effectuer un zoom avant et arrière à l'aide des touches / et de la molette de la souris , et que si vous appuyez sur / + , le canevas de la pièce sera à l'échelle 1:1 de l'image.
  • Diviser le canevas : En cliquant sur cette option, vous pouvez diviser le canevas horizontalement ou verticalement, tout en conservant un seul canevas. Lorsque vous êtes en mode d'affichage divisé, chaque canevas dispose de son propre ensemble de commandes, ce qui vous permet de zoomer sur l'un et sur l'autre, par exemple, et de modifier l'un ou l'autre canevas sans que les modifications ne soient répercutées sur l'autre. Pour modifier le basculement de l'affichage du canevas, vous pouvez cliquer sur le bouton Options de fractionnement et sélectionner celui que vous souhaitez utiliser ou simplement cliquer sur le bouton Fractionner le canevas jusqu'à ce qu'il passe à celui que vous souhaitez.

Dessiner la toile

Il s'agit de la principale fenêtre d'édition de vos images. Vous pouvez y appliquer n'importe lequel des outils situés à droite pour peindre sur le calque actuellement sélectionné. Vous pouvez peindre avec le bouton gauche de la souris ou le bouton droit de la souris (et attribuer une couleur à chacun d'eux à partir du sélecteur de couleur) et utiliser l'une des brosses définies en haut à droite de l'éditeur. Vous pouvez également utiliser / et la molette de la souris pour effectuer un zoom avant ou arrière, et en appuyant sur / + , le canevas sera réglé à l'échelle 1:1 avec l'image (ou vous pouvez utiliser les commandes du canevas comme expliqué ci-dessus). Notez également que vous pouvez effectuer un panoramique dans le canevas en utilisant le bouton central de la souris ou + .

Valeurs de la toile

Cette petite barre en bas montre les différentes valeurs pour la position de la souris et la largeur et la hauteur des outils de sélection dans l'éditeur.

Brosses

Vous pouvez ici sélectionner la brosse avec laquelle vous souhaitez peindre. Par défaut, vous disposez d'une sélection de pinceaux carrés et ronds de différentes tailles, mais vous pouvez facilement créer votre propre pinceau en peignant sur le calque de l'image, puis en sélectionnant la partie que vous souhaitez utiliser pour le nouveau pinceau et en utilisant / + "C" pour la copier. La section copiée sera alors ajoutée à vos brosses et vous pourrez peindre avec comme avec n'importe quelle autre brosse, ce qui inclut l'utilisation des différents outils de peinture disponibles comme le tracé d'une ligne.

Notez que la couleur de mélange par défaut de la nouvelle brosse sera le blanc, mais si vous changez la couleur avec le sélecteur de couleur et que vous peignez avec la brosse, elle sera mélangée avec cette nouvelle couleur. Si vous souhaitez supprimer une brosse de la liste, il suffit de cliquer dessus et d'appuyer sur la touche pour qu'elle soit supprimée.

REMARQUE: les brosses ne sont pas conservées entre les sessions, la fermeture de votre projet supprimera toutes les brosses que vous avez créées.

Sélecteur de couleurs

Dans la partie supérieure de cette section, vous pouvez choisir parmi un certain nombre de nuanciers de couleurs prédéfinis, qui peuvent être attribués respectivement aux boutons gauche et droit de la souris en cliquant dessus avec l'un ou l'autre des boutons (vous pouvez également permuter les couleurs entre chaque bouton en cliquant sur l'icône de la flèche à côté des nuanciers). Les couleurs sélectionnées s'affichent ci-dessous sous forme d'échantillons à côté de l'image de la souris, et vous pouvez cliquer dessus pour ouvrir la fenêtre du sélecteur de couleurs:

Dans cette fenêtre, vous pouvez définir la couleur à l'aide de l'un des curseurs et gadgets disponibles, ou vous pouvez saisir directement les valeurs ARGB ou HSV.

Si vous maintenez la touche CTRL/CMD enfoncée et que vous cliquez sur sur l'un des nuanciers par défaut, l'outil de sélection des couleurs s'affiche et vous permet de modifier la palette par défaut de sprite. En cliquant sur "OK" dans le sélecteur de couleurs, la palette prend la nouvelle couleur et est enregistrée avec le bien sprite en cours de modification.

Ces palettes de couleurs peuvent ensuite être permutées et appliquées à différents sprites à l'aide du menu Options de la palette de couleurs :

La ligne unique de couleurs située sous la palette présente l'historique de vos couleurs, la dernière couleur sélectionnée se trouvant à l'extrême gauche. Cette section n' est pas copiée lorsque vous utilisez l'option "Importer des couleurs" ou "Copier des couleurs", car elle ne s'applique qu'à la palette principale située au-dessus.

Ainsi, vous pouvez créer une palette de couleurs personnalisée dans une image sprite et l'utiliser pour définir la palette de n'importe quelle autre sprite dans le navigateur d'actifs, ou vous pouvez ouvrir une image sprite et importer la palette de couleurs de n'importe quelle autre sprite dans le navigateur d'actifs. Outils

Vous trouverez ci-dessous la liste des outils disponibles par défaut dans GameMaker. Notez que tous les outils peuvent être utilisés avec le bouton gauche de la souris et le bouton droit , et que la plupart d'entre eux sont associés à des raccourcis clavier (les plus importants sont listés ici avec l'outil, mais pour une liste complète, veuillez consulter la page sur les raccourcis). La plupart des outils ont également des options supplémentaires qui sont visibles en haut de l'éditeur d'images, comme indiqué ci-dessous :

Crayon Il s'agit de l'outil crayon. Il utilise le pinceau sélectionné pour peindre dans la couleur choisie pour le bouton de la souris, et vous pouvez définir la taille à partir de la barre supérieure de l'éditeur d'images ainsi qu'activer/désactiver le lissage (interpolation). Notez que vous pouvez également redimensionner et mettre à l'échelle n'importe quel pinceau en maintenant la touche + "Z" enfoncée, puis en cliquant sur et en faisant glisser la souris pour modifier les proportions du pinceau, ou encore en sélectionnant l'un des pinceaux prédéfinis en haut à droite.
Gomme à effacer Il s'agit de l'outil gomme. Il utilise la brosse sélectionnée pour effacer une zone en cliquant sur l'un des boutons de la souris. En sélectionnant cet outil, vous ajouterez également des outils supplémentaires à la barre supérieure de l'éditeur pour vous permettre de mettre la brosse à l'échelle ou d'activer/désactiver le lissage(interpolation linéaire).
Peinture / Remplissage Il s'agit de l'outil godet de peinture (ou remplissage). Il remplit une zone de l'image en fonction du paramètre de tolérance alpha choisi, que vous pouvez modifier à l'aide du curseur affiché dans la barre supérieure de l'éditeur.
Couleur Supprimer Il s'agit de l'outil de suppression des couleurs. Lorsque vous le sélectionnez, vous pouvez ensuite cliquer sur ou sur n'importe quel pixel de l'image et tous les pixels de la même couleur seront supprimés (ils sont techniquement remplacés par la couleur noire avec un alpha de 0). La barre supérieure comporte un curseur de tolérance dans lequel vous pouvez définir la tolérance de la couleur du pixel - plus cette valeur est faible, plus la couleur doit correspondre exactement à celle qui a été cliquée, tandis qu'une valeur plus élevée élargira cette tolérance pour inclure les couleurs similaires à celle qui a été cliquée. Si vous avez déjà effectué une sélection à l'aide de l'un des outils de sélection, le fait de cliquer sur les pixels de la sélection aura un effet et seuls les pixels de la sélection seront supprimés.
Remplacement de la couleur Il s'agit de l'outil de remplacement des couleurs. Lorsque vous le sélectionnez, vous pouvez ensuite cliquer sur ou sur n'importe quel pixel de l'image et tous les pixels de la même couleur seront remplacés par la couleur attribuée au bouton de la souris utilisé. La barre supérieure comporte un curseur de tolérance qui vous permet de définir la tolérance de la couleur du pixel - plus cette valeur est faible, plus la couleur doit correspondre exactement à celle qui a été cliquée, tandis qu'une valeur plus élevée élargira cette tolérance pour inclure les couleurs similaires à celle qui a été cliquée. Si vous avez préalablement effectué une sélection à l'aide de l'un des outils de sélection, seul le fait de cliquer sur les pixels de la sélection aura un effet, et seuls les pixels de la sélection seront affectés par le changement de couleur.
Ligne Cet outil permet de tracer une ligne entre deux points. Cliquez une fois avec l'un des boutons pour créer le premier point, puis cliquez à nouveau ailleurs pour créer un autre point et les joindre. La ligne sera tracée en utilisant la brosse sélectionnée pour créer le contour, et vous pouvez modifier certaines des propriétés de la brosse en utilisant la barre supérieure de l'éditeur (taille, interpolation linéaire, etc...).
Rectangle Cet outil permet de dessiner un rectangle rempli ou délimité. Si vous cliquez en haut à gauche de l'icône, le rectangle sera tracé, tandis qu'en cliquant en bas à droite, il sera rempli. Si vous souhaitez les deux, vous pouvez sélectionner cette option dans la barre supérieure de l'éditeur d'images (ou utiliser + pour sélectionner l'une puis l'autre sur l'icône de la boîte à outils), ainsi que définir la taille et le lissage de la brosse utilisée. Le rectangle sera ensuite dessiné à l'aide de la brosse sélectionnée. Notez que pendant le dessin, vous pouvez maintenir la touche enfoncée et cliquer pour dessiner à partir du centre de la position de la souris.
Ellipse Cet outil permet de dessiner une ellipse remplie ou délimitée. Si vous cliquez en haut à gauche de l'icône, l'ellipse sera tracée, tandis que si vous cliquez en bas à droite, elle sera remplie. Si vous souhaitez les deux, vous pouvez sélectionner cette option dans la barre supérieure de l'éditeur d'images (ou utiliser + pour sélectionner l'une puis l'autre sur l'icône de la boîte à outils), ainsi que définir la taille et le lissage de la brosse utilisée. L'ellipse sera dessinée en utilisant la brosse sélectionnée pour créer le contour. Notez que pendant le dessin, vous pouvez maintenir la touche enfoncée et cliquer pour dessiner à partir du centre de la position de la souris.
Polygone Cet outil permet de dessiner un polygone rempli ou délimité. Si vous cliquez en haut à gauche de l'icône, il sera tracé, tandis que si vous cliquez en bas à droite, il sera rempli. Si vous souhaitez les deux, vous pouvez sélectionner cette option dans la barre supérieure de l'éditeur d'image (ou utiliser + pour sélectionner l'une puis l'autre sur l'icône de la boîte à outils), ainsi que définir la taille et le lissage de la brosse utilisée. Vous cliquez sur ou pour ajouter des points au polygone à l'aide de la brosse sélectionnée et, lorsque vous êtes satisfait des résultats, vous devez sélectionner un autre outil pour valider les modifications et générer l'image. Le polygone sera dessiné à l'aide de la brosse sélectionnée. Vous pouvez appuyer sur à tout moment pour annuler la définition du polygone.
Cannelure Il s'agit de l'outil spline. Il utilise la brosse sélectionnée pour créer une courbe. Cliquez sur l'un des boutons de la souris et maintenez-le enfoncé pour définir un point, puis faites glisser la souris pour générer la courbe, relâchez la souris et répétez l'opération pour ajouter d'autres points et courbes. La courbe est dessinée à l'aide du pinceau sélectionné, qui peut être modifié à l'aide de la barre supérieure de l'éditeur pour définir la taille et le lissage. La barre supérieure comporte également un paramètre pour les itérations qui définira le degré de lissage de la courbe résultante (les valeurs les plus faibles seront plus rugueuses, les plus élevées plus lisses, et la valeur par défaut est 10).
Texte
Il s'agit de l'outil de saisie de texte. Il crée un nouveau calque de texte dans lequel vous pouvez taper ce dont vous avez besoin. Vous pouvez définir la police à utiliser ainsi que le formatage à partir de la barre supérieure de l'éditeur d'images.
Compte-gouttes pour les yeux L'outil pipette vous permet d'échantillonner une couleur de l'image en cours d'édition.
Sélection du rectangle Avec cet outil, vous pouvez générer une sélection rectangulaire. Vous pouvez annuler une sélection à partir du menu Image ou en appuyant sur .
Choix de la peinture Avec cet outil, vous pouvez "peindre" une zone sélectionnée en utilisant n'importe quelle brosse, y compris les brosses personnalisées. Les mêmes raccourcis clavier que pour l'outil Crayon s'appliquent. Vous pouvez annuler une sélection à partir du menu Image ou en appuyant sur
Baguette magique L'outil de sélection de la baguette magique sélectionne les zones de même couleur en fonction de la valeur de tolérance définie dans la barre supérieure de l'éditeur. Si vous cochez "Couleur uniquement", la valeur alpha ne sera pas prise en compte. Vous pouvez annuler une sélection à partir du menu Image ou en appuyant sur .
Rotation Il s'agit de l'outil de rotation. Si vous avez sélectionné une zone à l'aide de l'un des outils de sélection, cliquez sur cet outil pour faire pivoter la zone sélectionnée d'une valeur arbitraire comprise entre 0° et 360°. Si aucune zone de l'image n'est sélectionnée, l'outil vous permettra de faire pivoter la brosse actuelle. Pour effectuer une rotation, il suffit de maintenir le bouton gauche de la souris enfoncé et de faire glisser la souris vers la gauche ou la droite ou de définir la valeur directement dans la barre supérieure. Pour accepter les changements, vous pouvez appuyer sur ou sélectionner un autre outil. Notez que l'acceptation de la modification ne changera pas le matériau source, mais créera plutôt une nouvelle ressource Brosse que vous pourrez utiliser avec l'image sélectionnée ou une brosse pivotée selon la valeur que vous avez définie. Notez également que le raccourci clavier "Z" peut être utilisé pour faire pivoter toute sélection ou brosse de 90° et que cette option ne crée pas une nouvelle brosse et n'affecte que la façon dont la brosse actuelle est dessinée.
Miroir Il s'agit de l'outil miroir. En sélectionnant cette option, la brosse actuelle sera reflétée le long de l'axe vertical. Cette option ne crée pas de nouvelle brosse et n'affecte que la façon dont la brosse actuelle est dessinée.
Flip Il s'agit de l'outil d'inversion. En sélectionnant cette option, la brosse actuelle sera inversée le long de l'axe horizontal. Cette option ne crée pas de nouvelle brosse et n'affecte que la façon dont la brosse actuelle est dessinée.
Pan Il s'agit de l'outil panoramique. Lorsque vous le sélectionnez, vous pouvez cliquer sur et faire glisser la souris pour déplacer le contenu du calque à l'intérieur de celui-ci. Notez que tous les pixels situés en dehors du canevas de dessin ne seront pas visibles et que l'image sera recadrée à la taille du canevas de dessin lorsque vous changerez à nouveau d'outil. Notez également que si vous avez effectué une sélection, ce sont les pixels sélectionnés qui seront déplacés et non le contenu du calque.

Éditeur de couches

Vous pouvez créer des couches dans l'éditeur d'images pour différentes parties de votre image, et ces couches peuvent être manipulées et dessinées à l'aide des différents pinceaux et outils. Pour créer un nouveau calque, il suffit de cliquer sur l'icône , et pour le supprimer, sur l'icône . Les calques peuvent également être ordonnés en cliquant sur sur un calque, puis en faisant glisser le calque vers le haut ou vers le bas à la position souhaitée. Vous pouvez également organiser les calques en créant des dossiers à l'aide du bouton , puis en faisant glisser les calques dans ces dossiers. Si vous double-cliquez sur sur un calque (ou utilisez et sélectionnez Editer les propriétés du calque), vous obtiendrez access à la fenêtre des propriétés du calque :

Ici, vous pouvez nommer le calque et définir le mode de fusion (la façon dont il doit être "mélangé" avec les calques inférieurs), en choisissant parmi les mélanges normaux, additifs, soustractifs ou multiples . Enfin, vous pouvez définir l'opacité (transparence) du calque à l'aide d'un curseur, mais notez qu'elle est cumulée avec la valeur alpha réelle des pixels individuels - par exemple, si vous avez des pixels opaques à 50 % et que vous définissez l'opacité du calque à 50 %, les pixels finaux seront dessinés avec une opacité de 25 %.

Image

Le menu Image comporte les options suivantes (certaines d'entre elles peuvent être disponibles ou non en fonction de l'orientation actuelle de l'éditeur) :

  • Couper le cadre / la sélection - Coupe le cadre ou la sélection de l'image, en fonction de la focalisation de l'éditeur. Si vous coupez une sélection, cela l'ajoutera aux brosses.
  • Copier le cadre / la sélection - Copie le cadre ou la sélection de l'image, en fonction de la focalisation de l'éditeur. Si vous copiez une sélection, cela l'ajoutera aux brosses.
  • Coller le cadre / la sélection - Collez le cadre à partir du presse-papiers, ou collez la sélection du presse-papiers dans l'image, en fonction du focus actuel de IDE. Lorsque vous collez dans une image, vous devez noter que cette option est conçue uniquement pour être utilisée avec les données du presse-papiers provenant de programmes externes. Lorsque vous coupez ou copiez dans le presse-papiers à partir d'une image dans l'éditeur d'images, ces données sont ajoutées aux brosses en haut à droite de l'éditeur, et vous devez sélectionner une brosse pour dessiner avec la sélection coupée/copiée plutôt que d'essayer de la coller dans l'image. Notez également qu'un certain nombre de programmes auront des difficultés à coller les transparences du presse-papiers dans l'éditeur d'images de GameMaker en raison du format utilisé par le presse-papiers.
  • Select All - Sélectionnez l'image entière.
  • Annuler la sélection - Efface toute sélection de l'image.
  • Inverser la sélection - Inverse la sélection actuelle.
  • Ajouter un cadre - Ajouter un nouveau cadre à l'adresse sprite.
  • Insérer un cadre - Insère un nouveau cadre après celui qui est actuellement affiché.
  • Supprimer les cadres sélectionnés - Supprime le cadre sélectionné du site sprite (vous pouvez utiliser / + et/ou + pour sélectionner plusieurs cadres).
  • Importer une ou plusieurs images - Chargez une image à utiliser comme cadre à partir du disque. Notez que vous pouvez charger plusieurs cadres à la fois en sélectionnant plusieurs fichiers individuels à importer dans la fenêtre de l'explorateur. Une fois que vous avez sélectionné votre ou vos fichiers, la fenêtre d'importation suivante s'affiche :
    Dans cette fenêtre, vous pouvez choisir comment l'image importée doit être ajoutée à l'adresse sprite, ce qui vous permet de redimensionner l'ensemble du canevas de dessin, de le recadrer ou de le rembourrer, et de définir la position d'"ancrage" pour placer la ou les nouvelles images.
  • Convertir en cadres - Cette option permet de convertir une image unique en un certain nombre de cadres séparés. Vous devez d'abord avoir créé un cadre unique contenant toutes les parties de l'animation (ou importé une seule image en bande) comme celui illustré ci-dessous :
    Comme il s'agit d'une image unique et que nous voulons la convertir en cadres individuels, nous pouvons utiliser cette option qui ouvrira alors la fenêtre suivante :
    Ici, vous pouvez définir la façon dont l'image unique doit être divisée pour créer les cadres, en définissant le nombre de cadres, la largeur et la hauteur des cadres et toute valeur de décalage.
  • Importer une image en bande - Cette option ouvre un explorateur de fichiers et vous invite à rechercher un fichier d'image en bande unique à utiliser pour créer des cadres. Une fois l'image sélectionnée, la fenêtre Importer une bande s'ouvre, exactement comme pour l'option Convertir en cadres présentée ci-dessus.
  • Exporter en PNG - En sélectionnant cette option, le site sprite sera exporté sous la forme d'une seule image PNG. Si le site sprite comporte plus d'une image, une " bande " sprite sera créée avec toutes les images, l'une après l'autre, et le nom du fichier d'enregistrement sera automatiquement complété par "_stripX", où "X" est le nombre d'images exportées. Par exemple, une bande de cinq images sprite sera exportée sous le nom de "sprite_name_strip5.png".
  • Redimensionner toutes les images - Lorsque vous sélectionnez cette option, la fenêtre suivante s'ouvre et vous permet de définir la manière dont vous souhaitez redimensionner toutes les images d'animation qui composent le sprite :
    Vous pouvez choisir soit de mettre l'image à l'échelle , soit de redimensionner le canevas. Si vous choisissez de mettre l'image à l'échelle, il vous suffit de sélectionner la nouvelle largeur et/ou hauteur de chaque image, d'indiquer si vous souhaitez utiliser le même rapport d'aspect et si vous souhaitez mettre à l'échelle en pixels ou en pourcentage. Notez que lorsque vous mettez l'image à l'échelle, vous pouvez activer ou désactiver l'interpolation linéaire, car si vous l'activez, les pixels seront flous ou "tachés" lorsque l'image sera mise à l'échelle (en général, pour l'art en haute résolution, il est préférable de l'activer, mais pour l'art en pixels, il est préférable de le désactiver).

    L'option de redimensionnement du canevas ne modifiera pas les pixels de l'image, mais augmentera ou diminuera la taille du canevas sur lequel les images sont dessinées. Vous pouvez définir l'emplacement de l'"ancre" pour le redimensionnement à l'aide des icônes en forme de flèche, puis définir la nouvelle taille (en pixels ou en pourcentage) et déterminer si le rapport hauteur/largeur des cadres d'origine doit être conservé. Notez qu'en rendant le cadre plus petit que les images originales sprite, les pixels seront rognés.
  • Recadrer toutes les images sur la sélection - Cette option n'est disponible que si vous avez sélectionné une zone de l'image dans le canevas de dessin principal. Lorsque vous choisissez cette option, l'image en cours d'édition (et toutes les autres images de l'animation) sera recadrée aux limites de la sélection.
  • Découpage automatique de toutes les images - Cette option vous permet de découper tous les pixels alpha zéro sur les bords d'un site sprite. Cette opération prend en compte toutes les images du site sprite, de sorte que chaque côté est découpé jusqu'au bord de l'image dont le pixel alpha non nul est le plus éloigné.
  • Inverser les images - Cette option permet d'inverser l'ordre des images sur le site sprite.
  • Miroir - Cette option vous permet de mettre en miroir soit le cadre actuel, soit tous les cadres qui composent le site sprite le long de l'axe vertical.
  • Retourner - Cette option vous permet de retourner soit le cadre actuel, soit tous les cadres qui composent le site sprite le long de l'axe horizontal.
  • Rotation de toutes les images (90° dans le sens des aiguilles d'une montre) - Cette option permet de faire pivoter toutes les images de l'animation de 90° dans le sens des aiguilles d'une montre.
  • Faire pivoter toutes les images (90° dans le sens inverse des aiguilles d'une montre) - Cette option permet de faire pivoter toutes les images de l'animation de 90° dans le sens inverse des aiguilles d'une montre.

Voir

Le menu Affichage comporte les options suivantes :

  • 1:1 - Mettez l'image à l'échelle de la résolution (c'est-à-dire à la taille réelle) et centrez-la dans la fenêtre d'édition de l'image.
  • Adapter à l'écran - Met l'image à l'échelle pour qu'elle remplisse la fenêtre d'édition de l'image.
  • Afficher la grille - Affiche la grille de pixels. Il s'agit du même bouton que celui de la grille qui se trouve dans les contrôles du canevas (expliqué plus haut sur cette page).
  • Onion Skin Settings - Modifiez les paramètres de la pelure d'oignon. Cela ouvrira la fenêtre suivante, dans laquelle vous pouvez définir les cadres affichés par le bouton de la pelure d'oignon, ainsi que l'opacité et la couleur des cadres pelés.

Effets

Le menu Effets est l'endroit où vous pouvez ajouter vos propres plugins d'effets personnalisés, et dispose des effets suivants par défaut :

  • Blur - Applique un effet de flou à l'image. Vous pouvez définir l'intensité ainsi que le type de flou à utiliser (boîte ou gaussien) et déterminer si le flou doit être appliqué uniquement sur l'axe horizontal ou vertical, ou sur les deux. Vous pouvez ensuite choisir de l'appliquer uniquement au calque actuel, aux calques visibles ou à tous les calques.
  • Niveaux de gris - Désaturer l'image d'une quantité déterminée. Vous pouvez choisir de l'appliquer uniquement au calque actuel, aux calques visibles ou à tous les calques.
