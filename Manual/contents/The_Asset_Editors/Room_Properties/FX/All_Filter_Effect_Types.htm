Types et paramètres de FX

Cette page répertorie tous les types de filtres et d'effets fournis par GameMaker, ainsi que les informations suivantes sur chaque FX :

  • Nom interne (par exemple "_filter_tintfilter") qui peut être transmis à fx_create() pour créer un FX au moment de l'exécution.
  • Description
  • Exemple d'image
  • Paramètres de l'IDE, qui peuvent être modifiés par le biais des propriétés de la couche.
  • Paramètres d'exécution, qui peuvent être utilisés pour modifier les FX sur runtime, à l'aide des fonctions de filtre et d'effet.
    • Les paramètres Runtime mentionnent également leurs types, afin que vous sachiez quel type de données appliquer à quel paramètre.
